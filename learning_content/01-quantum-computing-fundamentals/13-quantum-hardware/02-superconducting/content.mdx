# Superconducting Qubits

## Introduction

Superconducting qubits are artificial atoms made from electrical circuits operating at millikelvin temperatures. They are the leading platform for commercial quantum computers from IBM, Google, and others.

## The Josephson Junction

```
The Heart of Superconducting Qubits
═══════════════════════════════════

JOSEPHSON JUNCTION:
    Two superconductors separated by thin barrier

    ┌─────────────┐
    │ Supercond.  │
    │    (Al)     │
    ├─────────────┤ ← Barrier (~1nm oxide)
    │ Supercond.  │
    │    (Al)     │
    └─────────────┘


KEY PHYSICS:
    Cooper pairs tunnel through barrier
    Creates nonlinear inductance
    I = Ic sin(φ)  (Josephson relation)

    Energy: U = -EJ cos(φ)

    This nonlinearity is ESSENTIAL
    Makes energy levels non-uniform


WHY NEEDED:
    Linear circuits (LC oscillator):
    Equal spacing → can't address single transition

    Josephson junction adds anharmonicity:
    ω₀₁ ≠ ω₁₂  → can isolate |0⟩↔|1⟩ transition
```

## Operating Conditions

```
The Cryogenic Environment
═════════════════════════

TEMPERATURE:
    Operating: 10-20 mK (millikelvin!)
    Room temperature: 300 K
    Ratio: 10,000× colder than room

    Why so cold?
    kT << ℏω  (thermal energy << qubit energy)

    For ω = 5 GHz:
    ℏω/k ≈ 240 mK
    At 15 mK: thermal population ~0.5%


DILUTION REFRIGERATOR:
    ┌────────────────────┐  300 K  (Room temp)
    │    Electronics     │
    ├────────────────────┤  50 K   (Pulse tube)

    ├────────────────────┤  4 K    (He-4)

    ├────────────────────┤  700 mK (Still)

    │    QUBITS          │
    └────────────────────┘


CHALLENGES:
    - Limited cooling power (~10 μW at 15mK)
    - Must filter all wiring
    - Vibration isolation
    - Magnetic shielding
```

## Single-Qubit Gates

```
Controlling Superconducting Qubits
══════════════════════════════════

MICROWAVE PULSES:
    Apply ~5 GHz microwave signal
    Duration ~10-50 ns
    Amplitude and phase control rotation

    H = ℏω₀|1⟩⟨1| + ℏΩ(t)cos(ωt + φ)(|0⟩⟨1| + |1⟩⟨0|)

    Rotating frame → Rabi oscillations


PULSE SHAPING:
    Simple square pulses → leakage to |2⟩
    DRAG pulses reduce leakage
    Derivative removes spurious coupling

    ┌──────────────────────┐
    │   ╭──╮               │
    │  ╱    ╲   Gaussian   │
    │ ╱      ╲  envelope   │
    │╱        ╲            │
    └──────────────────────┘


GATE SET:
    X gate: π pulse on I channel
    Y gate: π pulse on Q channel
    Z gate: virtual (change frame) or flux pulse
    Arbitrary: Adjust amplitude, phase, duration

TYPICAL FIDELITIES:
    Single-qubit: 99.9%+
    Gate time: 20-50 ns
```

## Two-Qubit Gates

```
Entangling Gates
════════════════

FIXED COUPLING:
    Qubits permanently coupled via capacitor/resonator
    Always-on interaction
    Use frequency tuning for gate

    Cross-resonance (CR) gate:
    Drive qubit 1 at qubit 2's frequency
    Creates ZX interaction
    Fidelity: ~99%

TUNABLE COUPLING:
    Adjust interaction strength dynamically
    Turn off coupling when idle
    Better isolation, more complex

    ┌─────┐     ┌─────────┐     ┌─────┐
    │ Q₁  ├─────┤ Coupler ├─────┤ Q₂  │
    └──┬──┘     └────┬────┘     └──┬──┘
       │             │             │
    Flux         Flux          Flux


iSWAP GATE:
    Tune qubits into resonance
    Exchange excitation
    20-60 ns gate time

CZ GATE:
    Tune |11⟩ near |02⟩ or |20⟩
    Phase accumulation
    ~30-50 ns

FIDELITIES:
    Best demonstrated: 99.5%+
    Typical systems: 98-99%
```

## Current Challenges

```
Superconducting Qubit Challenges
════════════════════════════════

1. COHERENCE
   T₁, T₂ limited by material defects
   Two-level systems (TLS) in oxides
   Improving but still ~100 μs

2. CONNECTIVITY
   Fixed 2D planar layout
   Limited to nearest neighbors
   SWAP gates add overhead

3. CROSSTALK
   Qubits affect each other
   Frequency collisions
   Requires careful calibration

4. WIRING
   Many microwave lines per qubit
   Scaling to 1M qubits?
   Cryogenic multiplexing needed

5. ERROR RATES
   Two-qubit gates ~0.5-1% error
   Need <0.1% for error correction
   Improving steadily


COMPANIES WORKING ON THIS:
    IBM: Quantum System Two (1000+ qubits)
    Google: Sycamore, future plans
    Rigetti: Superconducting systems
    IQM: European systems
    Oxford Quantum Circuits
```

## Summary

Superconducting qubits are the current leaders in qubit count:

1. **Josephson junction**: Provides essential nonlinearity
2. **Transmon**: Most common design (EJ >> EC)
3. **Cryogenic**: Operates at 15 mK in dilution fridge
4. **Fast gates**: 20-50 ns single-qubit, 30-60 ns two-qubit
5. **Challenges**: Coherence, connectivity, scaling wiring

This platform powers most commercial quantum computers today.
