# Trapped Ion Qubits

## Introduction

Trapped ion quantum computers use individual atoms suspended in electromagnetic traps as qubits. This platform holds records for gate fidelity and coherence times, making it a leading contender for fault-tolerant quantum computing.

## Ion Trapping

```
Paul Trap (Linear RF Trap)
══════════════════════════

PRINCIPLE:
    Oscillating electric field creates pseudopotential
    Ions confined in 3D by time-averaged force

    ┌─────────────────────────────────────┐
    │     RF electrodes (oscillating)     │
    │      ┌───┐           ┌───┐          │
    │      │   │   ●──●──● │   │  → ions  │
    │      │   │   (chain) │   │          │
    │      └───┘           └───┘          │
    │     DC endcaps (static)             │
    │      ○                    ○         │
    └─────────────────────────────────────┘


TYPICAL PARAMETERS:
    RF frequency: 10-50 MHz
    Trap depth: ~1 eV
    Ion spacing: 2-5 μm
    Secular frequencies: 1-5 MHz


IONS USED:
    Species     Mass    Qubit transition
    ─────────────────────────────────────
    ⁹Be⁺       9       Hyperfine
    ²⁵Mg⁺      25      Hyperfine
    ⁴⁰Ca⁺      40      Optical (S-D)
    ⁸⁸Sr⁺      88      Optical
    ¹³³Ba⁺     133     Hyperfine
    ¹⁷¹Yb⁺     171     Hyperfine (most common)
```

## State Preparation and Readout

```
Initialization and Measurement
══════════════════════════════

OPTICAL PUMPING (Initialization):
    Shine laser on cycling transition
    Ion decays to desired ground state
    Fidelity: >99.9%

    369 nm laser → pumps to F=0 ground state


FLUORESCENCE DETECTION (Readout):
    Apply detection laser
```

## Single-Qubit Gates

```
Laser-Driven Rotations
══════════════════════

MECHANISM:
    Two-photon Raman transition (hyperfine)
    Or direct optical transition

    Laser beam(s) drive Rabi oscillations
    ℏΩ = ℏd·E (Rabi frequency)


RAMAN TRANSITIONS (hyperfine):
    Two lasers with frequency difference = 12.6 GHz
    Virtual intermediate state
    Stimulated Raman adiabatic passage (STIRAP)

    ┌────────────────────────────────┐
    │         |e⟩ (virtual)         │
    │        ╱    ╲                  │
    │    ω₁╱      ╲ω₂               │
    │     ╱        ╲                 │
    │  |0⟩ ←──Δ──→ |1⟩              │
    │     (ground states)            │
    └────────────────────────────────┘


GATE FIDELITIES:
    Single-qubit: 99.99%+
    Fastest: ~1 μs
    Typical: ~10 μs

    Record: 99.9999% (6 nines!)
```

## Two-Qubit Gates

```
Entangling via Shared Motion
════════════════════════════

KEY INSIGHT:
    Ions share motional modes (phonons)
    Motion mediates interaction

    Ion 1 ↔ Motion ↔ Ion 2


MØLMER-SØRENSEN (MS) GATE:
    Bichromatic laser field
    ω₀ ± δ (red and blue sidebands)

    Creates state-dependent force
    |↑↑⟩ and |↓↓⟩ → different phases

    Result: XX interaction
    e^(-iθ X⊗X)


GATE SEQUENCE:
    1. Laser excites motion conditionally
    2. Motion evolves
    3. Second pulse returns motion to ground
    4. Geometric phase accumulated

    Phase ∝ enclosed area in phase space


PERFORMANCE:
    Fidelity: 99.9%+ achieved
    Gate time: 50-200 μs (slower than SC)
    All-to-all connectivity natural


CIRAC-ZOLLER GATE (historical):
    Sequential: ion1 → motion → ion2
    Lower fidelity than MS
    Important conceptually
```

## Challenges and Scaling

```
Trapped Ion Challenges
══════════════════════

1. SPEED
   Gates: 1-200 μs (vs 20 ns SC)
   Readout: 100+ μs
   Clock speed limited

2. SCALING
   Linear chains: ~50 ions max
   Heating increases with more ions
   Sympathetic cooling needed

3. OPTICAL COMPLEXITY
   Many laser beams needed
   Precise alignment critical
   Integrated photonics emerging

4. VACUUM REQUIREMENTS
   Ultra-high vacuum (~10⁻¹¹ torr)
   Single ion lifetime: hours to days


SCALING APPROACHES:

    QCCD (Quantum CCD):
    Shuttle ions between zones
    Demonstrated by Quantinuum

    ┌────────────────────────────────────┐
    │  Zone 1    ←→    Zone 2           │
    │  (compute)       (memory)         │
    │     ●●●    ──→    ●●●            │
    └────────────────────────────────────┘

    PHOTONIC INTERCONNECTS:
    Connect separate traps optically
    Entanglement distribution


COMPANIES:
    Quantinuum (Honeywell + Cambridge)
    IonQ
    Alpine Quantum Technologies
    Oxford Ionics
```

## Summary

Trapped ions offer the highest-quality qubits available:

1. **Trapping**: Paul trap confines ions using RF fields
2. **Encoding**: Hyperfine or optical transitions
3. **Gates**: Laser-driven, using shared motional modes
4. **Fidelities**: Record-breaking (99.9%+ two-qubit)
5. **Challenges**: Speed and scaling beyond ~50 qubits

This platform leads in quality, with active work on scaling solutions.
