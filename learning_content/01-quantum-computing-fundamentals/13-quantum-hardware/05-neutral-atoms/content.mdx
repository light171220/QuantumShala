# Neutral Atom Quantum Computing

## Introduction

Neutral atom quantum computers trap individual atoms using focused laser beams (optical tweezers) and use Rydberg states for entangling interactions. This platform combines long coherence with reconfigurable connectivity.

## Rydberg Interactions

```
The Rydberg Blockade
════════════════════

KEY INSIGHT:
    Rydberg atoms interact strongly
    Two nearby atoms can't both be in |r⟩

    V(R) = C₆/R⁶  (van der Waals)

    C₆ scales as n¹¹!


BLOCKADE MECHANISM:
    If one atom in |r⟩, neighbor's |r⟩ shifts
    Shift larger than laser linewidth → blocked

    ┌─────────────────────────────────────┐
    │                                     │
    │  Energy                             │
    │    │                                │
    │    │  ───── |rr⟩ (shifted up!)      │
    │    │     │                          │
    │    │     │ Δ (blockade shift)       │
    │    │     │                          │
    │    │  ───── |r⟩ laser resonance     │
    │    │                                │
    │    │  ───── |g⟩ ground state        │
    │    │                                │
    └─────────────────────────────────────┘


BLOCKADE RADIUS:
    R_b = (C₆/ℏΩ)^(1/6)

    Typically: 5-15 μm
    Atoms within R_b share one excitation
```

## Two-Qubit Gates

```
Rydberg-Mediated Entanglement
═════════════════════════════

CZ GATE PROTOCOL:

    1. Both atoms in ground state |gg⟩
    2. π pulse on atom 1: |gg⟩ → |rg⟩
    3. 2π pulse attempt on atom 2
       - If atom 1 in |r⟩: blocked, phase π
       - If atom 1 in |g⟩: completes, phase 0
    4. π pulse on atom 1: return to ground

    Result: |11⟩ → -|11⟩ (CZ gate!)


ALTERNATIVE: GLOBAL ADDRESSING
    Simultaneous Rydberg pulse on pair
    Collective behavior from blockade

    Faster but requires careful calibration


FIDELITIES ACHIEVED:
    Two-qubit: 99.5%+ demonstrated
    Improving rapidly
    Gate time: 0.5-1 μs


CONNECTIVITY:
    Can rearrange atoms during computation!
    Non-local gates without SWAP
    Zoned architecture possible
```

## Challenges and Progress

```
Current Challenges
══════════════════

1. ATOM LOSS
   Finite trap lifetime
   Loss during Rydberg excitation
   Need replacement or error correction

2. RYDBERG DECAY
   Short Rydberg lifetime (~100 μs)
   Limits gate time
   Some population leakage

3. READOUT
   Imaging takes time (~10 ms)
   Atom can be lost during readout
   Non-destructive methods developing

4. TEMPERATURE
   Atoms heated by operations
   Need recooling
   Mid-circuit cooling challenging


RECENT MILESTONES:

    2023: 48 logical qubits (error corrected!)
          Harvard/QuEra

    2024: 1000+ qubit arrays
          Multiple groups

    Fidelities: Rapidly approaching ions


COMPANIES:
    QuEra Computing
    Pasqal
    Atom Computing
    ColdQuanta (Infleqtion)
```

## Summary

Neutral atom quantum computing offers unique advantages:

1. **Optical tweezers**: Individual atom trapping and rearrangement
2. **Rydberg blockade**: Strong interactions for entangling gates
3. **Reconfigurable**: Can change connectivity dynamically
4. **Scalable**: 1000+ qubit arrays demonstrated
5. **Challenges**: Atom loss, readout speed, cooling

This platform is advancing rapidly and has demonstrated error-corrected logical qubits.
