# Photonic Quantum Computing

## Introduction

Photons are natural carriers of quantum information. They don't decohere in free space, travel at the speed of light, and are essential for quantum networks. However, making photons interact is challenging.

## Photonic Qubit Encodings

```
How to Encode Qubits in Light
═════════════════════════════

1. POLARIZATION ENCODING
   |0⟩ = |H⟩ (horizontal)
   |1⟩ = |V⟩ (vertical)

   Easy single-qubit gates (waveplates)
   Hard two-qubit gates

2. PATH ENCODING (DUAL-RAIL)
   |0⟩ = photon in path A
   |1⟩ = photon in path B

   ──●────── path A (|0⟩)
     │
   ──○────── path B (|1⟩)

   Beamsplitters for gates

3. TIME-BIN ENCODING
   |0⟩ = early time bin
   |1⟩ = late time bin

   Good for fiber communication
   Requires interferometric stability

4. CONTINUOUS VARIABLE (CV)
   Encode in quadratures X and P
   Squeezed states, cat states
   Different paradigm


PHOTON SOURCES:
    - Spontaneous parametric down-conversion
    - Quantum dots
    - Integrated photon sources
    Production rate: challenging for scaling
```

## The Hong-Ou-Mandel Effect

```
Photon Bunching
═══════════════

SETUP:
    Two identical photons enter 50:50 beamsplitter
    from different ports

    Input: |1,1⟩ (one photon each port)

         ↓
    ─────┬─────
         │
    ─────┴─────
         ↑

OUTPUT:
    |1,1⟩ → (|2,0⟩ + |0,2⟩)/√2

    NEVER |1,1⟩!

    Photons always exit together
    Destructive interference of |1,1⟩ paths


WHY IT MATTERS:

    1. Certifies photon indistinguishability
       HOM dip visibility measures quality

    2. Enables entanglement
       Post-select on |1,1⟩ output → entangled

    3. Foundation for linear optical gates
       Many protocols use HOM interference


VISIBILITY:
    V = (N_classical - N_HOM) / N_classical

    Perfect: V = 1.0
    Typical: V > 0.95 for good sources
```

## Boson Sampling

```
Boson Sampling
══════════════

A SIMPLER TASK (but still hard):

    Input: n photons into m >> n modes
    Apply: Random linear optical network
    Output: Sample from output distribution


WHY INTERESTING:

    1. COMPUTATIONALLY HARD
       Best classical: O(n² 2^n) to simulate
       Quantum: Just run the experiment!

    2. NOT UNIVERSAL
       Can't do all computations
       But still shows quantum advantage

    3. DEMONSTRATIONS:
       2020: Jiuzhang (China) - 76 photons
       2023: Borealis (Xanadu) - 216 modes


CIRCUIT:

    ○──┬──┬──┬──┬──○
       │  │  │  │
    ○──┴──┬──┬──┴──○
          │  │
    ○──┬──┴──┴──┬──○
       │        │
    ○──┴────────┴──○

    Beamsplitters create complex interference
    Output distribution → permanent of matrix
    Permanent is #P-hard to compute!
```

## Measurement-Based Quantum Computing

```
One-Way Quantum Computing
═════════════════════════

PARADIGM SHIFT:
    Instead of gates, prepare entangled state
    Compute by measuring qubits adaptively

CLUSTER STATE:
    Highly entangled multi-qubit state
    Create using CZ gates on grid of |+⟩

    ●──●──●──●──●
    │  │  │  │  │
    ●──●──●──●──●
    │  │  │  │  │
    ●──●──●──●──●


COMPUTATION:
    1. Prepare cluster state
    2. Measure qubits left-to-right
    3. Measurement basis depends on previous results
    4. Output on right edge

    Information "flows" through the cluster


WHY FOR PHOTONICS?
    - Creating cluster state is probabilistic
       but can be done offline
    - Once created, computation is deterministic
    - Fits well with photon loss model


FUSION-BASED QC:
    Modern approach (PsiQuantum)
    Small entangled groups ("resource states")
    Fuse together probabilistically
    Build up cluster for computation
```

## Strengths and Challenges

```
Photonic QC: Summary
════════════════════

STRENGTHS:
    ✓ No decoherence in transmission
    ✓ Room temperature operation
    ✓ Natural for quantum networks
    ✓ High-bandwidth operations
    ✓ Mature fabrication technology

CHALLENGES:
    ✗ Photons hard to generate deterministically
    ✗ Photons don't interact (nonlinearity issue)
    ✗ Detection losses are critical
    ✗ Probabilistic gates require resources


CURRENT STATE:
    Boson sampling: 50-200+ modes
    Programmable chips: 12-20 modes
    Gate fidelities: 99%+ for linear ops
    Photon loss: main limiting factor


APPLICATIONS:
    1. Quantum communication (QKD)
    2. Boson sampling advantage
    3. Gaussian boson sampling (chemistry)
    4. Future: Fault-tolerant computing?
```

## Summary

Photonic quantum computing leverages light's quantum properties:

1. **Encodings**: Polarization, path, time-bin
2. **Linear optics**: Beamsplitters + measurement enables universality
3. **Boson sampling**: Demonstrated quantum computational advantage
4. **Integration**: PICs enable scalable, stable systems
5. **Challenges**: Deterministic sources, photon loss

Photons excel at communication and may scale to fault-tolerant computing.
