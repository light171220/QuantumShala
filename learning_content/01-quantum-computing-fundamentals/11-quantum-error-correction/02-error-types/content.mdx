# Quantum Error Types

## Introduction

Understanding the types of errors that affect qubits is fundamental to designing error correction codes. Remarkably, any quantum error can be decomposed into combinations of just three basic error types.

## Pauli Errors

```
The Pauli Error Basis
═════════════════════

Any single-qubit error can be written as a combination of:

    I (Identity):   No error
    X (Bit flip):   |0⟩ ↔ |1⟩
    Z (Phase flip): |+⟩ ↔ |-⟩
    Y = iXZ:        Combined bit and phase flip


EFFECT ON STATES:

    X|0⟩ = |1⟩       X|1⟩ = |0⟩
    X(α|0⟩ + β|1⟩) = α|1⟩ + β|0⟩  (swaps amplitudes)

    Z|0⟩ = |0⟩       Z|1⟩ = -|1⟩
    Z(α|0⟩ + β|1⟩) = α|0⟩ - β|1⟩  (flips phase of |1⟩)

    Y|0⟩ = i|1⟩      Y|1⟩ = -i|0⟩
    Y(α|0⟩ + β|1⟩) = iα|1⟩ - iβ|0⟩


BLOCH SPHERE VISUALIZATION:

         Z error: Rotation around Z axis
              │
              ▼
         ┌────●────┐
         │    │    │
    X ◀──│────●────│──▶ X error: Rotation around X
         │    │    │
         └────●────┘
              │
              ▼
              Z

    Y error: Rotation around Y axis (out of page)
```

## Bit Flip Error

```python
def bit_flip_analysis():
    """
    Analyze the bit flip (X) error.
    """
    print("""
Bit Flip Error (X)
══════════════════

PHYSICAL CAUSE:
    - Electromagnetic interference
    - Control pulse errors
    - Thermal excitation/relaxation (T1)

EFFECT:
    |0⟩ → |1⟩
    |1⟩ → |0⟩
    α|0⟩ + β|1⟩ → α|1⟩ + β|0⟩

    Swaps the computational basis amplitudes


EXAMPLE:

    Original: |ψ⟩ = 0.6|0⟩ + 0.8|1⟩

    After X error: |ψ'⟩ = 0.6|1⟩ + 0.8|0⟩
                       = 0.8|0⟩ + 0.6|1⟩

    The state is different! (unless it was |+⟩ or |-⟩)


IN DENSITY MATRIX:
    ρ → XρX†

    For pure state:
    |ψ⟩⟨ψ| → X|ψ⟩⟨ψ|X


ANALOGOUS TO:
    Classical bit flip: 0 ↔ 1
    But acts on superpositions!
    """)

bit_flip_analysis()
```

## Phase Flip Error

```
Phase Flip Error (Z)
════════════════════

PHYSICAL CAUSE:
    - Dephasing from environment (T2)
    - Frequency fluctuations
    - Z-axis noise

EFFECT:
    |0⟩ → |0⟩
    |1⟩ → -|1⟩
    α|0⟩ + β|1⟩ → α|0⟩ - β|1⟩

    Flips the relative phase!


EXAMPLE:

    Original: |ψ⟩ = (|0⟩ + |1⟩)/√2 = |+⟩

    After Z error: |ψ'⟩ = (|0⟩ - |1⟩)/√2 = |-⟩

    Completely different state in X basis!


IN DENSITY MATRIX:
    ρ → ZρZ†


WHY IT'S SUBTLE:
    Measurement in Z basis: Same probabilities!
    P(0) = |α|²    P(1) = |β|²

    But measurement in X basis: Totally different
    |+⟩ becomes |-⟩

    Phase information is crucial for interference


NO CLASSICAL ANALOG:
    Classical bits don't have phase
    This is a uniquely quantum error type
```

## Combined and Continuous Errors

```
Combined Errors
═══════════════

Y ERROR (Bit + Phase flip):

    Y = iXZ

    Y|0⟩ = i|1⟩
    Y|1⟩ = -i|0⟩

    Both amplitude swap AND phase change


ARBITRARY ERROR:

    Any 2×2 error matrix E can be written:

    E = c₀I + c₁X + c₂Y + c₃Z

    where c₀, c₁, c₂, c₃ are complex coefficients


THE MAGIC OF DISCRETIZATION:

    Suppose error is small rotation Rz(θ):

    Rz(θ) = cos(θ/2)I - i·sin(θ/2)Z

    After error correction measurement:
    - State collapses to "no error" (I) with prob cos²(θ/2)
    - State collapses to "Z error" with prob sin²(θ/2)

    Small θ → mostly no error detected
    When Z detected → apply Z to fix

    ┌─────────────────────────────────────────────────┐
    │  Continuous error                              │
    │       ↓                                        │
    │  Syndrome measurement                          │
    │       ↓                                        │
    │  Collapses to discrete Pauli (I, X, Y, or Z)  │
    │       ↓                                        │
    │  Apply correction if needed                    │
    │       ↓                                        │
    │  Error corrected!                              │
    └─────────────────────────────────────────────────┘

    This is why correcting just I, X, Y, Z suffices!
```

## Common Noise Channels

```python
def noise_channels():
    """
    Common quantum noise models.
    """
    print("""
Common Noise Channels
═════════════════════

1. DEPOLARIZING CHANNEL
    With probability p, apply random Pauli:
    ρ → (1-p)ρ + (p/3)(XρX + YρY + ZρZ)

    Uniform random errors


2. AMPLITUDE DAMPING (T1)
    Models energy relaxation: |1⟩ → |0⟩
    E₀ = [[1, 0], [0, √(1-γ)]]
    E₁ = [[0, √γ], [0, 0]]

    Asymmetric: favors |0⟩


3. DEPHASING (T2)
    Pure phase randomization:
    ρ → (1-p)ρ + p·ZρZ

    Phase flip without bit flip


4. BIT FLIP CHANNEL
    With probability p, apply X:
    ρ → (1-p)ρ + p·XρX


ERROR RATES:

    ┌────────────────────────────────────────────┐
    │  Error Type      │  Typical Rate (NISQ)   │
    ├────────────────────────────────────────────┤
    │  Single-qubit    │  0.01 - 0.1%           │
    │  Two-qubit       │  0.1 - 1%              │
    │  Measurement     │  0.5 - 2%              │
    │  Idle (per μs)   │  0.01 - 0.1%           │
    └────────────────────────────────────────────┘
    """)

noise_channels()
```

## Summary

Quantum errors fall into three basic types:

1. **Bit flip (X)**: Swaps |0⟩ and |1⟩ amplitudes
2. **Phase flip (Z)**: Flips relative phase
3. **Bit-phase flip (Y)**: Both effects combined

Any error can be decomposed into these Pauli errors, enabling discrete correction of continuous errors through syndrome measurement.
