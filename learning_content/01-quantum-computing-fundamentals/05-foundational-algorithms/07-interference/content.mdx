## Learning Objectives

## 1. Types of Interference

<Definition term="Constructive Interference">
When amplitudes add together, increasing probability.
</Definition>

<Definition term="Destructive Interference">
When amplitudes cancel, decreasing probability.
</Definition>

```python
import numpy as np

# Constructive: amplitudes add
a1 = 0.5
a2 = 0.5
constructive = a1 + a2
print(f"Constructive: {a1} + {a2} = {constructive}")
print(f"Probability: {constructive**2}")

# Destructive: amplitudes cancel
a1 = 0.5
a2 = -0.5
destructive = a1 + a2
print(f"\nDestructive: {a1} + ({a2}) = {destructive}")
print(f"Probability: {destructive**2}")
```

## 2. Interference in Algorithms

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

# Start with |0⟩
ket_0 = np.array([1, 0])

# H creates superposition
after_H = H @ ket_0
print(f"After H: {after_H}")

# Another H causes interference
after_HH = H @ after_H
print(f"After HH: {after_HH}")
print("Back to |0⟩! Constructive interference at |0⟩")
```

## 3. Phase-Based Interference

In algorithms like Deutsch-Jozsa, phase differences create interference patterns:

```python
import numpy as np

# If f is constant: all phases same → constructive at |0⟩
# If f is balanced: phases differ → destructive at |0⟩

print("Deutsch-Jozsa interference:")
print("  Constant f: all (-1)^f(x) = +1 → |0...0⟩")
print("  Balanced f: half +1, half -1 → not |0...0⟩")
```

## Summary

- Constructive interference increases amplitude/probability
- Destructive interference decreases amplitude/probability
- Quantum algorithms engineer interference patterns
- Phase information determines interference outcome
