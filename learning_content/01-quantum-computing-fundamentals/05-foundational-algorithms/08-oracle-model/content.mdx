## Learning Objectives

## 1. What is an Oracle?

<Definition term="Oracle">
A black-box unitary that computes a function f(x) without revealing its internal structure. We measure algorithm complexity by counting oracle queries.
</Definition>

## 2. Types of Oracles

### Bit Oracle (Standard Oracle)

<Formula>
$$U_f|x\rangle|y\rangle = |x\rangle|y \oplus f(x)\rangle$$
</Formula>

```python
import numpy as np

def bit_oracle(f, n):
    """Create bit oracle for f: {0,1}^n -> {0,1}"""
    N = 2**(n+1)
    oracle = np.zeros((N, N))
    
    for x in range(2**n):
        for y in range(2):
            input_idx = 2*x + y
            output_y = y ^ f(x)
            output_idx = 2*x + output_y
            oracle[output_idx, input_idx] = 1
    
    return oracle

# Example: f(x) = x mod 2 (parity)
f = lambda x: x % 2
oracle = bit_oracle(f, 2)
print("Bit oracle for parity function")
```

### Phase Oracle

<Formula>
$$O_f|x\rangle = (-1)^{f(x)}|x\rangle$$
</Formula>

```python
import numpy as np

def phase_oracle(f, n):
    """Create phase oracle for f: {0,1}^n -> {0,1}"""
    N = 2**n
    oracle = np.zeros((N, N))
    
    for x in range(N):
        oracle[x, x] = (-1)**f(x)
    
    return oracle

# Example: mark x=3
f = lambda x: 1 if x == 3 else 0
oracle = phase_oracle(f, 3)
print("Phase oracle marking x=3:")
print(np.diag(oracle))
```

## 3. Converting Between Oracle Types

Phase oracle = Bit oracle with ancilla in |-⟩:

```python
import numpy as np

H = np.array([[1,1],[1,-1]])/np.sqrt(2)

# Bit oracle on |x⟩|-⟩ gives phase oracle behavior
# Uf|x⟩|-⟩ = |x⟩|(-1)^f(x)⟩|-⟩ = (-1)^f(x)|x⟩|-⟩

print("Phase kickback: Bit oracle + |-⟩ ancilla = Phase oracle")
```

## Summary

- Oracle: black-box computing f(x)
- Bit oracle: XORs f(x) into ancilla
- Phase oracle: applies (-1)^f(x) phase
- Phase kickback converts between types
