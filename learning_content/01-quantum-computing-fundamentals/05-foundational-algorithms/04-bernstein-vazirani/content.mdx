## 1. The Hidden String Problem

<Definition term="Bernstein-Vazirani Problem">
Given a function f(x) = s·x mod 2, where s is a hidden n-bit string and · denotes bitwise inner product, find s using as few queries to f as possible.
</Definition>

**Classical complexity**: n queries (one per bit of s)
**Quantum complexity**: 1 query!

## 2. The Oracle

The oracle computes f(x) = s·x mod 2:

```python
import numpy as np

def create_bv_oracle(s):
    """
    Create oracle for f(x) = s·x mod 2.
    s: hidden string as list of bits
    """
    n = len(s)
    N = 2**(n + 1)  # n input qubits + 1 ancilla
    oracle = np.zeros((N, N))
    
    for x in range(2**n):
        for y in range(2):
            # Compute s·x mod 2
            inner_product = sum(s[i] * ((x >> i) & 1) for i in range(n)) % 2
            
            input_idx = 2*x + y
            output_y = y ^ inner_product
            output_idx = 2*x + output_y
            oracle[output_idx, input_idx] = 1
    
    return oracle

# Example: s = "101" (binary)
s = [1, 0, 1]
oracle = create_bv_oracle(s)
print(f"Oracle for s = {''.join(map(str, s))}")
```

## 3. The Algorithm

The circuit is identical to Deutsch-Jozsa:

```python
import numpy as np

def bernstein_vazirani(oracle, n):
    """
    Run BV algorithm.
    oracle: (2^(n+1)) × (2^(n+1)) unitary
    n: number of input qubits
    Returns: the hidden string s
    """
    N = 2**(n + 1)
    
    # Initial state |0...0⟩|1⟩
    state = np.zeros(N)
    state[1] = 1
    
    # H^⊗(n+1)
    H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
    H_all = H
    for _ in range(n):
        H_all = np.kron(H_all, H)
    
    state = H_all @ state
    state = oracle @ state
    
    # H^⊗n on first n qubits
    H_n = np.eye(1)
    for _ in range(n):
        H_n = np.kron(H_n, H)
    H_n_I = np.kron(H_n, np.eye(2))
    state = H_n_I @ state
    
    # The measurement reveals s directly!
    # Find the basis state with highest probability
    probs = np.abs(state)**2
    max_idx = np.argmax(probs)
    s = format(max_idx // 2, f'0{n}b')
    
    return s

# Test
s_hidden = [1, 0, 1]
oracle = create_bv_oracle(s_hidden)
s_found = bernstein_vazirani(oracle, len(s_hidden))
print(f"Hidden: {''.join(map(str, s_hidden))}")
print(f"Found:  {s_found}")
```

## 4. Why It Works

After applying H^⊗n to the input register and querying the oracle:

<Formula>
$$|x\rangle \rightarrow (-1)^{s \cdot x}|x\rangle$$
</Formula>

The final Hadamard transform converts phase information to amplitude:

<Formula>
$$H^{\otimes n} \sum_x (-1)^{s \cdot x}|x\rangle = |s\rangle$$
</Formula>

## Summary

- BV finds n-bit hidden string in 1 query (vs n classical)
- Same circuit structure as Deutsch-Jozsa
- Demonstrates power of phase kickback
- Linear to constant query improvement
