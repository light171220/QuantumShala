## Learning Objectives

- Extend Deutsch's algorithm to n bits
- Implement the Deutsch-Jozsa circuit
- Understand exponential quantum speedup

## 1. The Problem

Given f: {0,1}^n → {0,1}, promised to be either constant or balanced, determine which.

<Definition term="Balanced Function">
A function where exactly half of inputs map to 0 and half to 1.
</Definition>

Classically: Need 2^(n-1) + 1 queries (worst case)
Quantum: Need exactly 1 query!

## 2. The Algorithm

```python
import numpy as np

def deutsch_jozsa(oracle, n):
    """
    Deutsch-Jozsa algorithm.
    oracle: (2^(n+1)) × (2^(n+1)) unitary
    n: number of input qubits
    """
    dim = 2**(n+1)
    
    # |0...0⟩|1⟩
    state = np.zeros(dim)
    state[1] = 1  # |0...01⟩
    
    # H^⊗(n+1)
    H = np.array([[1,1],[1,-1]])/np.sqrt(2)
    H_all = H
    for _ in range(n):
        H_all = np.kron(H, H_all)
    
    state = H_all @ state
    state = oracle @ state
    
    # H^⊗n on first n qubits
    H_n = np.eye(1)
    for _ in range(n):
        H_n = np.kron(H, H_n)
    H_n_I = np.kron(H_n, np.eye(2))
    state = H_n_I @ state
    
    # Measure first n qubits
    # Constant → |0...0⟩, Balanced → not |0...0⟩
    prob_zero = sum(np.abs(state[i])**2 for i in range(2))  # |0...0,0⟩ and |0...0,1⟩
    
    return "constant" if prob_zero > 0.5 else "balanced"

print("Deutsch-Jozsa: Exponential speedup!")
print("Classical: O(2^n) queries")
print("Quantum: O(1) query")
```

## 3. Why It Works

After applying H^⊗n and the oracle:

<Formula>
$$|x\rangle \rightarrow (-1)^{f(x)}|x\rangle$$
</Formula>

The final Hadamards create constructive interference at |0...0⟩ only if f is constant.

## Summary

- Deutsch-Jozsa: exponential speedup
- 1 query vs 2^(n-1)+1 classical
- Uses interference to detect global property
