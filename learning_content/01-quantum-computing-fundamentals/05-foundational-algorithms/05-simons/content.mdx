## 1. Simon's Problem

<Definition term="Simon's Problem">
Given a function f: {0,1}^n → {0,1}^n with the promise that there exists a hidden string s such that f(x) = f(y) if and only if x ⊕ y ∈ {0, s}, find s.
</Definition>

**Classical complexity**: O(2^(n/2)) queries (birthday paradox bound)
**Quantum complexity**: O(n) queries

This is the first algorithm showing exponential quantum speedup for a specific problem.

## 2. The Key Insight

Simon's algorithm inspired Shor's algorithm. Both use:
1. Quantum parallelism to query all inputs
2. Interference to extract hidden structure
3. Classical post-processing to find the answer

```python
import numpy as np

def complexity_comparison(n):
    """Compare classical vs quantum complexity."""
    classical = int(2**(n/2))  # Birthday bound
    quantum = n
    speedup = classical / quantum
    
    print(f"n = {n} bits:")
    print(f"  Classical: ~{classical:,} queries")
    print(f"  Quantum: {quantum} queries")
    print(f"  Speedup: ~{speedup:,.0f}x")
    print()

for n in [10, 20, 40, 100]:
    complexity_comparison(n)
```

## 3. Algorithm Overview

```
1. Initialize: |0⟩^n |0⟩^n
2. Apply H^⊗n to first register
3. Query oracle: |x⟩|0⟩ → |x⟩|f(x)⟩
4. Apply H^⊗n to first register
5. Measure first register → get y where y·s = 0
6. Repeat O(n) times to get n-1 independent equations
7. Solve linear system to find s
```

## 4. The Quantum Part

After querying the oracle:

<Formula>
$$\frac{1}{\sqrt{2^n}} \sum_x |x\rangle|f(x)\rangle$$
</Formula>

Measuring the second register collapses the first to a superposition of x and x⊕s:

<Formula>
$$\frac{1}{\sqrt{2}}(|x_0\rangle + |x_0 \oplus s\rangle)|f(x_0)\rangle$$
</Formula>

## 5. Classical Post-Processing

Each measurement gives y where y·s = 0 mod 2. After n-1 independent measurements, solve the linear system to find s.

```python
import numpy as np

def solve_for_s(measurements, n):
    """
    Given measurements y_i where y_i · s = 0,
    solve for s using Gaussian elimination.
    """
    # Build matrix A where each row is a measurement
    A = np.array(measurements) % 2
    
    # Add all possible values and check which satisfies all equations
    for candidate in range(1, 2**n):
        s = np.array([int(b) for b in format(candidate, f'0{n}b')])
        if all((A @ s) % 2 == 0):
            return s
    
    return np.zeros(n)  # s = 0 (trivial case)

# Example: measurements from Simon's algorithm
measurements = [
    [1, 0, 1, 1],
    [0, 1, 1, 0],
    [1, 1, 0, 1],
]
s = solve_for_s(measurements, 4)
print(f"Found s = {s}")
```

## 6. Historical Significance

Simon's algorithm (1994) was the first to show exponential speedup over classical algorithms for a non-trivial problem. It directly inspired Peter Shor to develop his famous factoring algorithm.
