## 1. What is a Quantum Algorithm?

<Definition term="Quantum Algorithm">
A quantum algorithm is a step-by-step procedure that runs on a quantum computer, leveraging quantum mechanical phenomena like superposition and entanglement to solve problems.
</Definition>

Unlike classical algorithms that process definite bits (0 or 1), quantum algorithms manipulate qubits that can exist in superposition, enabling parallel exploration of solution spaces.

## 2. Types of Quantum Speedup

Quantum algorithms provide different levels of speedup over classical algorithms:

```python
# Speedup comparison
speedups = {
    "Exponential": {
        "example": "Shor's Algorithm (factoring)",
        "classical": "O(exp(n^(1/3)))",
        "quantum": "O(n³)"
    },
    "Quadratic": {
        "example": "Grover's Algorithm (search)",
        "classical": "O(N)",
        "quantum": "O(√N)"
    },
    "Polynomial": {
        "example": "Quantum simulation",
        "classical": "O(exp(n))",
        "quantum": "O(poly(n))"
    }
}

for speedup_type, data in speedups.items():
    print(f"{speedup_type} Speedup:")
    print(f"  Example: {data['example']}")
    print(f"  Classical: {data['classical']}")
    print(f"  Quantum: {data['quantum']}")
    print()
```

## 3. Key Quantum Algorithms Overview

| Algorithm | Problem | Speedup |
|-----------|---------|---------|
| Deutsch-Jozsa | Constant vs balanced function | Exponential |
| Bernstein-Vazirani | Hidden string finding | Linear → 1 query |
| Simon's | Period finding | Exponential |
| Grover's | Unstructured search | Quadratic |
| Shor's | Integer factoring | Exponential |
| QFT | Fourier transform | Exponential |

## 4. The Power of Interference

Quantum algorithms work by:

1. **Superposition**: Create all possible states simultaneously
2. **Oracle Query**: Mark or phase-flip desired solutions
3. **Interference**: Amplify correct answers, cancel wrong ones
4. **Measurement**: Extract the answer

```python
print("Quantum Algorithm Pattern:")
print("1. Initialize: |0⟩^n")
print("2. Superposition: H^n|0⟩^n = equal superposition")
print("3. Oracle: marks solutions")
print("4. Interference: amplifies solutions")
print("5. Measure: get answer with high probability")
```

## Summary

- Quantum algorithms leverage superposition and entanglement
- Speedups range from quadratic to exponential
- Key techniques: superposition, oracles, interference
- Not all problems have quantum speedup
