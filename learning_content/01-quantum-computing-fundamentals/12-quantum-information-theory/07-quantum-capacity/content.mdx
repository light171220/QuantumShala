# Quantum Channel Capacity

## Introduction

Just as classical channels have capacity limits, quantum channels have capacities for transmitting both classical and quantum information. These capacities reveal deep aspects of quantum information theory.

## Types of Channel Capacity

```
Channel Capacities Overview
═══════════════════════════

CLASSICAL CAPACITY (C):
    Max rate of classical bits per channel use
    Using quantum states and measurements

QUANTUM CAPACITY (Q):
    Max rate of qubits per channel use
    Faithfully transmit quantum states

PRIVATE CAPACITY (P):
    Classical bits secure from eavesdropper

ENTANGLEMENT-ASSISTED (CE):
    Classical capacity with shared entanglement


HIERARCHY:
    Q ≤ P ≤ C ≤ CE

    Quantum info hardest to transmit
    Entanglement helps most
```

```python
def classical_capacity_concepts():
    """
    Classical capacity of quantum channels.
    """
    print("Classical Capacity")
    print("=" * 50)
    print("""
HOLEVO QUANTITY:
    χ(ε) = max S(ε(ρ)) - ΣpᵢS(ε(ρᵢ))
           {pᵢ,ρᵢ}

    where ρ = Σpᵢρᵢ (ensemble average)


HSW THEOREM (Holevo-Schumacher-Westmoreland):
    C(ε) = lim (1/n) χ(ε⊗ⁿ)
           n→∞

    For most channels: C(ε) = χ(ε)
    But sometimes need entangled inputs!

    ε(ρ) = (1-p)ρ + pI/2

    χ(ε) = 1 - H(p)  (using optimal input)

    where H(p) = -p log₂(p) - (1-p) log₂(1-p)
""")

classical_capacity_concepts()
```

## The Holevo Bound

```
Holevo Bound Details
════════════════════

BOUND ON ACCESSIBLE INFORMATION:
    I(X;Y) ≤ χ = S(ρ) - Σpₓ S(ρₓ)

    where:
    X = classical input variable
    Y = measurement outcome
    ρₓ = states encoding x
    ρ = Σpₓρₓ = average state


ACHIEVING THE BOUND:
    Use product states: ρ^⊗n
    Measure collectively (joint measurement)
    Asymptotically achieves χ

    Single-letter: One channel use
    Multi-letter: Need n uses together


EXAMPLE:
    Encoding in {|0⟩, |+⟩}:
    χ < 1 bit (non-orthogonal states)

    Encoding in {|0⟩, |1⟩}:
    χ = 1 bit (orthogonal = optimal)
```

## Quantum Capacity

```
Quantum Channel Capacity
════════════════════════

GOAL: Transmit qubits through noisy channel

COHERENT INFORMATION:
    Ic(ρ, ε) = S(ε(ρ)) - S((I⊗ε)(|ψ⟩⟨ψ|))

    where |ψ⟩ is purification of ρ

    Equivalently:
    Ic = S(B) - S(E)

    (Bob's entropy minus environment's)


LSD THEOREM (Lloyd-Shor-Devetak):
    Q(ε) = lim (1/n) max Ic(ρ, ε⊗ⁿ)
           n→∞        ρ

    Regularized coherent information


KEY INSIGHT:
    Ic can be NEGATIVE!
    (Environment learns more than Bob)

    This means: some channels have Q = 0
    (Can't transmit quantum info reliably)
```

## Superadditivity

```
Superadditivity of Capacity
═══════════════════════════

SHOCKING DISCOVERY (Hastings 2009):
    C(ε₁ ⊗ ε₂) > C(ε₁) + C(ε₂)

    Using two channels together can send
    MORE than sum of individual capacities!


FOR CLASSICAL CAPACITY:
    Need entangled input states
    Violates additivity conjecture

    Implication:
    C(ε) ≠ χ(ε) for some channels
    Need multi-letter formula


FOR QUANTUM CAPACITY:
    Q is NOT additive
    Q(ε₁ ⊗ ε₂) can exceed Q(ε₁) + Q(ε₂)

    Even stranger:
    Q(ε) = 0 and Q(ε') = 0
    but Q(ε ⊗ ε') > 0  possible!

    "Superactivation"


IMPLICATIONS:
    - Capacity formulas require regularization
    - Computing capacity is hard (undecidable!)
    - Quantum information is weird
```

## Entanglement-Assisted Capacity

```
With Shared Entanglement
════════════════════════

ENTANGLEMENT-ASSISTED CLASSICAL CAPACITY:
    CE(ε) = max I(A;B)ρ
            ρ

    where I(A;B) = S(A) + S(B) - S(AB)

    Uses pre-shared entanglement
    Sender keeps half, sends other half through channel


BENNETT-SHOR-SMOLIN-THAPLIYAL:
    CE(ε) = max S(ρ) + S(ε(ρ)) - S((I⊗ε)|ψ⟩⟨ψ|)
            ρ

    Single-letter formula!
    No regularization needed


COMPARISON:
    CE ≥ C always
    CE can be much larger

    For depolarizing at p = 0.5:
    C = 0 (useless classically)
    CE > 0 (entanglement helps!)


SUPERDENSE CODING CONNECTION:
    With perfect channel: CE = 2
    (2 bits per qubit with entanglement)
```

## Computing Capacities

```python
def capacity_computation():
    """
    Practical capacity calculation.
    """
    print("Computing Channel Capacities")
    print("=" * 50)
    print("""
DEPOLARIZING CHANNEL ε_p:
    ε(ρ) = (1-p)ρ + (p/3)(XρX + YρY + ZρZ)

    Classical capacity (single-letter):
    C ≈ 1 - H(p) - p·log₂(3)

    Quantum capacity:
    Q = max(0, 1 - H(p) - p·log₂(3))

    Zero capacity threshold: p ≈ 0.25


AMPLITUDE DAMPING ε_γ:
    Kraus: E₀ = |0⟩⟨0| + √(1-γ)|1⟩⟨1|
           E₁ = √γ |0⟩⟨1|

    Both C and Q known analytically
    Q > 0 for all γ < 1


ERASURE CHANNEL:
    ε(ρ) = (1-p)ρ + p|e⟩⟨e|

    C = 1 - p
    Q = 1 - 2p  (for p < 0.5)
    Q = 0       (for p ≥ 0.5)
""")

capacity_computation()
```

## Summary

Channel capacities quantify fundamental limits of quantum communication:

1. **Classical capacity C**: Classical bits through quantum channel
2. **Quantum capacity Q**: Qubits through quantum channel
3. **Coherent information**: Key quantity for Q
4. **Superadditivity**: Capacities can be non-additive
5. **Entanglement-assisted**: CE ≥ C always

Understanding capacity is essential for quantum networks and error correction.
