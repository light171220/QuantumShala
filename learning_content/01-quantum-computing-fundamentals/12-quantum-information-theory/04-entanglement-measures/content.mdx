# Entanglement Measures

## Introduction

Quantifying "how entangled" a state is requires careful mathematical measures. Different measures capture different aspects of entanglement and have different operational meanings.

## Why Measure Entanglement?

```
Motivation for Entanglement Measures
════════════════════════════════════

QUESTIONS WE WANT TO ANSWER:

1. Is this state entangled at all?
2. How much entanglement does it have?
3. Can we compare two entangled states?
4. How much entanglement is needed for a task?
5. How much entanglement can we distill?


REQUIREMENTS FOR A GOOD MEASURE E(ρ):

1. E(ρ) ≥ 0
   Non-negative

2. E(ρ) = 0 iff ρ is separable
   Zero for product states

3. E cannot increase under LOCC
   Local operations + classical communication

4. E(|Φ⁺⟩) = 1 (often)
   Normalized to Bell state = 1 ebit
```

## Entanglement Entropy (Pure States)

```python
import numpy as np

def entanglement_entropy(state_vector, dims):
    """
    For pure bipartite state |ψ⟩AB:
    E(|ψ⟩) = S(ρA) = S(ρB)

    Measured in ebits (log base 2).
    """
    # Reshape state vector
    psi = state_vector.reshape(dims)

    # Compute reduced density matrix via SVD
    # |ψ⟩ = Σᵢ λᵢ|iA⟩|iB⟩ (Schmidt decomposition)
    U, S, Vh = np.linalg.svd(psi, full_matrices=False)

    # Schmidt coefficients
    schmidt = S ** 2  # λᵢ²

    # Entropy
    schmidt = schmidt[schmidt > 1e-10]
    return -np.sum(schmidt * np.log2(schmidt))

# Examples
print("Entanglement Entropy")
print("=" * 40)

# Product state |00⟩
product = np.array([1, 0, 0, 0])
print(f"|00⟩: E = {entanglement_entropy(product, (2,2)):.3f} ebits")

# Bell state |Φ⁺⟩ = (|00⟩ + |11⟩)/√2
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
print(f"|Φ⁺⟩: E = {entanglement_entropy(bell, (2,2)):.3f} ebit")

# Partially entangled state
partial = np.array([np.sqrt(0.8), 0, 0, np.sqrt(0.2)])
print(f"|ψ⟩:  E = {entanglement_entropy(partial, (2,2)):.3f} ebits")
```

## Schmidt Decomposition

```
Schmidt Decomposition
═════════════════════

Any bipartite pure state can be written:

    |ψ⟩AB = Σᵢ λᵢ |iA⟩|iB⟩

where:
    λᵢ ≥ 0 are Schmidt coefficients
    Σᵢ λᵢ² = 1 (normalization)
    {|iA⟩}, {|iB⟩} are orthonormal bases

ENTANGLEMENT FROM SCHMIDT:

    E(|ψ⟩) = -Σᵢ λᵢ² log₂(λᵢ²)

    (Shannon entropy of Schmidt probabilities)


EXAMPLES:

    |00⟩:  λ = (1)         E = 0 ebits
    |Φ⁺⟩:  λ = (1/√2, 1/√2)  E = 1 ebit
    |GHZ⟩: More complex for >2 parties
```

## Concurrence

```python
def concurrence(rho):
    """
    Concurrence for two-qubit mixed states.

    C(ρ) = max(0, λ₁ - λ₂ - λ₃ - λ₄)

    where λᵢ are square roots of eigenvalues of
    ρ(σy⊗σy)ρ*(σy⊗σy), in decreasing order.
    """
    # Pauli Y
    sy = np.array([[0, -1j], [1j, 0]])
    sysy = np.kron(sy, sy)

    # R matrix
    R = rho @ sysy @ rho.conj() @ sysy

    # Eigenvalues
    eigenvalues = np.sqrt(np.abs(np.linalg.eigvals(R)))
    eigenvalues = np.sort(eigenvalues)[::-1]  # Decreasing order

    return max(0, eigenvalues[0] - eigenvalues[1] -
               eigenvalues[2] - eigenvalues[3])

print("\nConcurrence")
print("=" * 40)
print("""
PROPERTIES:
    C(ρ) ∈ [0, 1]
    C = 0 for separable states
    C = 1 for Bell states
    Works for mixed states!

ENTANGLEMENT OF FORMATION:
    EF(ρ) = h((1 + √(1-C²))/2)

    where h(x) = -x log₂(x) - (1-x) log₂(1-x)

    Gives entanglement cost to create ρ
""")
```

## Negativity

```python
def negativity(rho, dims):
    """
    Negativity: sum of negative eigenvalues of ρ^TA

    N(ρ) = (||ρ^TA||₁ - 1) / 2

    where ρ^TA is partial transpose.
    """
    # Partial transpose on first subsystem
    d1, d2 = dims
    rho_reshaped = rho.reshape(d1, d2, d1, d2)
    rho_pt = rho_reshaped.transpose(2, 1, 0, 3).reshape(d1*d2, d1*d2)

    # Eigenvalues
    eigenvalues = np.linalg.eigvalsh(rho_pt)

    # Sum of negative eigenvalues
    return -np.sum(eigenvalues[eigenvalues < 0])

def log_negativity(rho, dims):
    """
    Log-negativity: EN = log₂(2N + 1)

    Additive under tensor product.
    Upper bound on distillable entanglement.
    """
    N = negativity(rho, dims)
    return np.log2(2*N + 1)

print("\nNegativity and Log-Negativity")
print("=" * 40)
print("""
PPT CRITERION (Peres-Horodecki):
    ρ^TA has negative eigenvalue → entangled

    For 2×2 and 2×3 systems:
    Negative eigenvalue ⟺ entangled

NEGATIVITY:
    N(ρ) = Σᵢ |λᵢ⁻| (sum of negative eigenvalues)

ADVANTAGES:
    - Computable for any mixed state
    - Additive log-negativity
    - Physical interpretation via PPT

LIMITATION:
    - Misses bound entanglement (PPT entangled states)
""")
```

## Comparison of Measures

```
Entanglement Measure Comparison
═══════════════════════════════

MEASURE          PURE STATES    MIXED STATES    COMPUTABLE
────────────────────────────────────────────────────────────
Entropy E        Exact          N/A*            Yes
Concurrence C    Exact          2 qubits only   Yes
Negativity N     Related to E   All states      Yes
EoF              = E            Convex roof     Hard
Distillable Ed   = E            ≤ EoF           Very hard

*For mixed states, need EoF or other measures


OPERATIONAL MEANINGS:

    Entanglement Entropy:
        Ebits needed to create |ψ⟩
        Ebits extractable from |ψ⟩

    Entanglement of Formation:
        Ebits needed to create ρ

    Distillable Entanglement:
        Ebits extractable from ρ

    Negativity:
        Related to teleportation fidelity
```

## Bound Entanglement

```
Bound Entanglement
══════════════════

A surprising phenomenon: some states are
entangled but cannot be distilled!

PPT ENTANGLED STATES:
    - Partial transpose is positive (PPT)
    - But the state is NOT separable
    - Cannot distill Bell pairs
    - "Locked" entanglement

EXAMPLE: 3×3 Bound Entangled State
    Exists! First found by Horodecki family

IMPLICATIONS:
    - Distillable entanglement can be zero
    - EoF > Ed possible
    - Negativity misses these states

    ┌─────────────────────────────────┐
    │  All states                     │
    │  ┌───────────────────────────┐  │
    │  │  Entangled               │  │
    │  │  ┌─────────────────────┐ │  │
    │  │  │  NPT (distillable)  │ │  │
    │  │  └─────────────────────┘ │  │
    │  │  PPT entangled (bound)   │  │
    │  └───────────────────────────┘  │
    │  Separable (PPT)                │
    └─────────────────────────────────┘
```

## Multipartite Entanglement

```
Beyond Bipartite Entanglement
═════════════════════════════

THREE OR MORE PARTIES:

Much more complex! Different "types" of entanglement.

GHZ STATE: |GHZ⟩ = (|000⟩ + |111⟩)/√2
    - Maximally entangled all together
    - Trace out one → separable
    - Fragile entanglement

W STATE: |W⟩ = (|001⟩ + |010⟩ + |100⟩)/√3
    - Different type of tripartite entanglement
    - Trace out one → still entangled
    - More robust

STOCHASTIC LOCC (SLOCC):
    Two states equivalent if convertible
    with nonzero probability

    GHZ and W are SLOCC inequivalent!
    Cannot convert one to other by LOCC.


MEASURES MORE COMPLEX:
    - Geometric entanglement
    - Localizable entanglement
    - Various monotones
    - Area law (many-body physics)
```

## Summary

Key entanglement measures and their properties:

1. **Entanglement entropy**: Fundamental for pure states (S(ρA))
2. **Concurrence**: Practical for two-qubit mixed states
3. **Negativity**: Computable for any state, based on partial transpose
4. **Entanglement of formation**: Operational meaning as creation cost
5. **Bound entanglement**: Undistillable but still entangled

Understanding entanglement quantification is essential for quantum information processing.
