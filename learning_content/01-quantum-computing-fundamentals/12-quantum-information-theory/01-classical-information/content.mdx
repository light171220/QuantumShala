# Classical Information Theory

## Introduction

Before understanding quantum information, we must master classical information theory. Claude Shannon's revolutionary 1948 paper laid the foundation for how we measure and communicate information.

## What Is Information?

```
Shannon's Key Insight
════════════════════

Information = Reduction of Uncertainty

EXAMPLE: Coin flip
    Before: Could be heads or tails (uncertain)
    After:  You know the result (certainty)
    Information gained: 1 bit

KEY PRINCIPLE:
    Rare events carry MORE information
    Common events carry LESS information

    "The sun rose today" → Little information (expected)
    "The sun didn't rise" → Huge information (unexpected)
```

## Shannon Entropy

```python
import numpy as np

def shannon_entropy(probabilities):
    """
    Calculate Shannon entropy H(X).

    H(X) = -Σ p(x) log₂ p(x)

    Measured in bits.
    """
    p = np.array(probabilities)
    # Avoid log(0) by filtering zeros
    p = p[p > 0]
    return -np.sum(p * np.log2(p))

# Examples
print("Shannon Entropy Examples")
print("=" * 40)

# Fair coin: maximum uncertainty
fair_coin = [0.5, 0.5]
print(f"Fair coin: {shannon_entropy(fair_coin):.3f} bits")

# Biased coin: less uncertainty
biased_coin = [0.9, 0.1]
print(f"Biased coin (90/10): {shannon_entropy(biased_coin):.3f} bits")

# Certain outcome: no uncertainty
certain = [1.0, 0.0]
print(f"Certain outcome: {shannon_entropy(certain):.3f} bits")

# Fair die: 6 outcomes
fair_die = [1/6] * 6
print(f"Fair 6-sided die: {shannon_entropy(fair_die):.3f} bits")
```

## Properties of Entropy

```
Entropy Properties
══════════════════

1. NON-NEGATIVITY
   H(X) ≥ 0
   Entropy is never negative

2. MAXIMUM ENTROPY
   H(X) ≤ log₂(n)  for n outcomes
   Achieved when all outcomes equally likely

3. ADDITIVITY (Independent Variables)
   H(X,Y) = H(X) + H(Y)  if X,Y independent

4. CONCAVITY
   H(λp + (1-λ)q) ≥ λH(p) + (1-λ)H(q)
   Mixing increases entropy


INTUITION:

    Entropy (bits)
    │
    1 ├────────●────────┤  Maximum at p=0.5
      │       /·\       │
      │      / · \      │
      │     /  ·  \     │
      │    /   ·   \    │
    0 ├───●────┼────●───┤
      0       0.5      1
              p
```

## Conditional Entropy

```python
def conditional_entropy(joint_prob, marginal_prob):
    """
    H(Y|X) = H(X,Y) - H(X)

    Uncertainty in Y given we know X.
    """
    h_xy = shannon_entropy(joint_prob.flatten())
    h_x = shannon_entropy(marginal_prob)
    return h_xy - h_x

# Example: Correlated variables
print("\nConditional Entropy")
print("=" * 40)
print("""
If X and Y are perfectly correlated:
    H(Y|X) = 0  (knowing X tells you Y)

If X and Y are independent:
    H(Y|X) = H(Y)  (X tells you nothing about Y)

General case:
    0 ≤ H(Y|X) ≤ H(Y)

Chain rule:
    H(X,Y) = H(X) + H(Y|X)
           = H(Y) + H(X|Y)
""")
```

## Mutual Information

```
Mutual Information I(X;Y)
═════════════════════════

How much information X and Y share.

FORMULA:
    I(X;Y) = H(X) + H(Y) - H(X,Y)
           = H(X) - H(X|Y)
           = H(Y) - H(Y|X)

VENN DIAGRAM:

    ┌─────────────────────────────────┐
    │         ┌─────────┐             │
    │   H(X│Y)│  I(X;Y) │ H(Y│X)     │
    │         │         │             │
    │    ─────┴─────────┴─────        │
    │         H(X,Y)                  │
    └─────────────────────────────────┘
         H(X)       H(Y)

PROPERTIES:
    I(X;Y) ≥ 0           (always non-negative)
    I(X;Y) = I(Y;X)      (symmetric)
    I(X;Y) ≤ min(H(X), H(Y))
    I(X;X) = H(X)        (self-information)
```

## Channel Capacity

```
Noisy Channel Coding
════════════════════

PROBLEM:
    Send data reliably over noisy channel

    Sender → [Noisy Channel] → Receiver
              (errors occur)

SHANNON'S NOISY CHANNEL THEOREM:

    Channel capacity C = max I(X;Y)
                          p(x)

    Can transmit at any rate R < C with
    arbitrarily small error probability!

BINARY SYMMETRIC CHANNEL:

    Input     Output
      0 ──┬── 0    with prob (1-p)
          └── 1    with prob p (error)
      1 ──┬── 1    with prob (1-p)
          └── 0    with prob p (error)

    Capacity: C = 1 - H(p, 1-p)

    When p = 0:   C = 1 bit (perfect channel)
    When p = 0.5: C = 0 bits (useless channel)
```

## From Classical to Quantum

```
Bridge to Quantum Information
═════════════════════════════

CLASSICAL                  QUANTUM
─────────────────────────────────────────
Bits (0 or 1)        →   Qubits (|0⟩, |1⟩, superposition)
Shannon entropy      →   Von Neumann entropy
Mutual information   →   Quantum mutual information
Data compression     →   Schumacher compression
Channel capacity     →   Quantum channel capacity

KEY DIFFERENCES:
    1. Quantum has no-cloning (can't copy qubits)
    2. Measurement disturbs quantum states
    3. Entanglement is a new resource
    4. Quantum channels can transmit more

QUANTUM ADVANTAGES:
    - Superdense coding: 2 classical bits per qubit
    - Quantum teleportation: state transfer
    - Quantum key distribution: provable security
```

## Summary

Classical information theory provides the foundation:

1. **Shannon entropy**: Measures uncertainty in bits
2. **Mutual information**: Quantifies shared information
3. **Source coding**: Compression limit is entropy
4. **Channel capacity**: Maximum reliable transmission rate
5. **Foundation**: Extends naturally to quantum setting

These concepts will be generalized to quantum systems in the following lessons.
