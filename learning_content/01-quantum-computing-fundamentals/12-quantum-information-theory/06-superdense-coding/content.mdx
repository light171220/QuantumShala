# Superdense Coding

## Introduction

Superdense coding is the inverse of quantum teleportation. While teleportation uses 2 classical bits to send 1 qubit, superdense coding uses 1 qubit to send 2 classical bits.

## The Superdense Coding Protocol

```
Superdense Coding Overview
══════════════════════════

GOAL: Send 2 classical bits using 1 qubit

RESOURCES:
    - Shared Bell pair (pre-distributed)
    - 1 qubit transmission
    - Bell measurement

RESOURCE EQUATION:

    1 ebit + 1 qubit → 2 classical bits

Compare to teleportation:
    1 ebit + 2 cbits → 1 qubit


PROTOCOL SKETCH:

    Alice                           Bob
    ┌───────┐                      ┌───────┐
    │ 2 bits│                      │       │
    │  ij   │  Encode              │       │
    └───┬───┘    │                 │       │
        │        ▼                 │       │
    ┌───┴───┐  ┌───┐   Send    ┌───┴───┐
    │Alice's│──│ U ├──────────▶│Bob's  │
    │ qubit │  │ij │   qubit   │qubits │
    └───────┘  └───┘           └───┬───┘
                                   │
    Bell pair                      ▼
    ═══════════════════════    Bell Measure
                               ┌───────┐
                               │  ij   │
                               │ 2 bits│
                               └───────┘
```

## Step-by-Step Protocol

```python
import numpy as np

def superdense_coding():
    """
    Detailed superdense coding steps.
    """
    print("Superdense Coding Protocol")
    print("=" * 50)
    print("""
SETUP: Alice and Bob share |Φ⁺⟩ = (|00⟩ + |11⟩)/√2

STEP 1: Alice wants to send two bits ij

    Bits  Operation  Result on Bell pair
    ──────────────────────────────────────
    00    I          (|00⟩ + |11⟩)/√2 = |Φ⁺⟩
    01    X          (|10⟩ + |01⟩)/√2 = |Ψ⁺⟩
    10    Z          (|00⟩ - |11⟩)/√2 = |Φ⁻⟩
    11    XZ         (|10⟩ - |01⟩)/√2 = |Ψ⁻⟩

    Alice applies Uij to HER qubit only!

STEP 2: Alice sends her qubit to Bob

STEP 3: Bob now has both qubits of a Bell state
    He performs Bell measurement:
    - CNOT (first qubit controls second)
    - Hadamard on first qubit
    - Measure both

    Result: Exactly recovers ij!
""")

superdense_coding()
```

## Why It Works

```
The Mathematics
═══════════════

PAULI OPERATIONS ON BELL STATES:

    I ⊗ I |Φ⁺⟩ = |Φ⁺⟩
    X ⊗ I |Φ⁺⟩ = |Ψ⁺⟩
    Z ⊗ I |Φ⁺⟩ = |Φ⁻⟩
    Y ⊗ I |Φ⁺⟩ = i|Ψ⁻⟩  (or use XZ)


KEY INSIGHT:
    Local operations on ONE qubit can
    transform between ALL FOUR Bell states!

    The four Bell states are orthogonal:
    ⟨Φ⁺|Ψ⁺⟩ = 0, etc.

    So they can be perfectly distinguished
    by measurement → 2 bits of information


BELL MEASUREMENT:

    |Φ⁺⟩ ─[CNOT]─[H]─ |00⟩
    |Ψ⁺⟩ ─[CNOT]─[H]─ |01⟩
    |Φ⁻⟩ ─[CNOT]─[H]─ |10⟩
    |Ψ⁻⟩ ─[CNOT]─[H]─ |11⟩
```

## The Circuit

```
Superdense Coding Circuit
═════════════════════════

                    Alice           Bob
    |0⟩ ──[H]──●────[Uij]────────●────[H]────[M]─→ i
               │                  │
    |0⟩ ───────X──────────────────X──────────[M]─→ j

    ├─────────────┤├────────┤├───────────────────┤
      Bell pair     Encode       Decode
     preparation


ENCODING GATES:
    Bits    Gate on Alice's qubit
    00      I (identity)
    01      X (bit flip)
    10      Z (phase flip)
    11      XZ or iY


DECODING:
    CNOT: Undoes entanglement
    H: Returns to computational basis
    Measure: Read out bits
```

## Comparison: Teleportation vs Superdense

```
Duality Between Protocols
═════════════════════════

TELEPORTATION:
    1 ebit + 2 cbits → 1 qubit (transmitted)
    "Quantum over classical"

SUPERDENSE CODING:
    1 ebit + 1 qubit → 2 cbits (transmitted)
    "Classical over quantum"


RESOURCE TABLE:

    Protocol          Uses           Transmits
    ─────────────────────────────────────────
    Teleportation     1 ebit         1 qubit
                      2 cbits
    Superdense        1 ebit         2 cbits
                      1 qubit


HOLEVO BOUND:
    Normally: 1 qubit carries at most 1 classical bit
    With entanglement: 1 qubit carries 2 classical bits!

    Entanglement doubles classical capacity!
```

## The Holevo Bound

```
Information Capacity
════════════════════

HOLEVO BOUND (no prior entanglement):
    1 qubit can carry at most 1 bit

    χ = S(Σpᵢρᵢ) - ΣpᵢS(ρᵢ)

    For pure states: χ ≤ 1 bit

SUPERDENSE CODING BYPASSES THIS:
    Pre-shared entanglement changes the rules

    Alice's qubit + shared entanglement
    = 2 bits transmitted


WHY NOT MORE?
    Four Bell states = maximum
    log₂(4) = 2 bits

    Higher dimensions:
    d-dimensional system + d-dimensional entanglement
    → 2 log₂(d) classical bits
```

## Practical Considerations

```
Experimental Implementation
═══════════════════════════

REQUIREMENTS:
    1. High-fidelity Bell pairs
    2. Good single-qubit gates (I, X, Z)
    3. Bell measurement capability
    4. Quantum channel for qubit transmission

CHALLENGES:
    - Entanglement distribution over distance
    - Maintaining coherence during transmission
    - Bell measurement efficiency


DEMONSTRATIONS:
    1996: First demonstration (photons)
    Many since: Various platforms

APPLICATIONS:
    - Quantum networks
    - Quantum key distribution variants
    - Quantum communication complexity
    - Dense coding for quantum computation


LIMITATIONS:
    - Need pre-shared entanglement
    - One-time use (Bell pair consumed)
    - Still limited by qubit transmission rate
```

```python
CAPACITY:
    1 qudit + 1 edit → 2 log₂(d) bits

    |Φᵢⱼ⟩ = (Uᵢⱼ ⊗ I)|Φ⟩

higher_dimensional_superdense()
```

## Summary

Superdense coding demonstrates the power of entanglement:

1. **Protocol**: Encode 2 bits in 1 qubit using shared Bell pair
2. **Operations**: I, X, Z, XZ create four orthogonal Bell states
3. **Decoding**: Bell measurement perfectly distinguishes states
4. **Duality**: Inverse of teleportation's resource trade-off
5. **Capacity**: Doubles classical capacity with entanglement

This protocol proves that entanglement is a genuine resource for communication.
