# Quantum Teleportation

## Introduction

Quantum teleportation is one of the most striking protocols in quantum information. It allows the transfer of a quantum state from one location to another using entanglement and classical communication.

## The Teleportation Protocol

```
Quantum Teleportation Overview
══════════════════════════════

GOAL: Transfer |ψ⟩ from Alice to Bob
      without sending the qubit directly

RESOURCES NEEDED:
    1. Shared entanglement (1 Bell pair)
    2. Classical communication (2 bits)
    3. Local quantum operations

KEY INSIGHT:
    State is destroyed at source,
    recreated at destination.
    No cloning violation!

    Alice                      Bob
    ┌───┐                     ┌───┐
    │|ψ⟩│──┐                  │   │
    └───┘  │   Bell pair      │   │
    ┌───┐  │  ┌─────────┐    │   │
    │|A⟩├──┴──┤  |Φ⁺⟩   ├────┤|B⟩│
    └───┘     └─────────┘    └───┘
       │                        │
       ▼ Measure                │
    ╔═══╗    2 bits            │
    ║ij ║━━━━━━━━━━━━━━━━━━━━▶│
    ╚═══╝   Classical          ▼
                             Apply Uij
                              ┌───┐
                              │|ψ⟩│
                              └───┘
```

## Step-by-Step Protocol

```python
import numpy as np

def teleportation_protocol():
    """
    Detailed teleportation steps.
    """
    print("Teleportation Protocol")
    print("=" * 50)
    print("""
INITIAL STATE:
    |ψ⟩ = α|0⟩ + β|1⟩  (state to teleport)
    |Φ⁺⟩ = (|00⟩ + |11⟩)/√2  (shared Bell pair)

    Full state: |ψ⟩ ⊗ |Φ⁺⟩ = (α|0⟩ + β|1⟩) ⊗ (|00⟩ + |11⟩)/√2

STEP 1: Alice applies CNOT (control: |ψ⟩, target: her Bell qubit)

    α|0⟩(|00⟩ + |11⟩)/√2 + β|1⟩(|10⟩ + |01⟩)/√2

STEP 2: Alice applies Hadamard to |ψ⟩

    Rewrite in Bell basis:
    (|00⟩ + |11⟩)/2 ⊗ (α|0⟩ + β|1⟩)  → outcome 00
    (|01⟩ + |10⟩)/2 ⊗ (α|1⟩ + β|0⟩)  → outcome 01
    (|00⟩ - |11⟩)/2 ⊗ (α|0⟩ - β|1⟩)  → outcome 10
    (|01⟩ - |10⟩)/2 ⊗ (α|1⟩ - β|0⟩)  → outcome 11

STEP 3: Alice measures both qubits → gets ij ∈ {00, 01, 10, 11}

STEP 4: Alice sends ij to Bob (2 classical bits)

STEP 5: Bob applies correction
    00 → I (no correction)
    01 → X
    10 → Z
    11 → XZ

RESULT: Bob has |ψ⟩ = α|0⟩ + β|1⟩  ✓
""")

teleportation_protocol()
```

## The Math Behind Teleportation

```
Mathematical Analysis
═════════════════════

BELL STATE REWRITING:
    |ψ⟩|Φ⁺⟩ = (α|0⟩ + β|1⟩) ⊗ (|00⟩ + |11⟩)/√2

After CNOT and H, this equals:

    ½|Φ⁺⟩(α|0⟩ + β|1⟩) +
    ½|Ψ⁺⟩(α|1⟩ + β|0⟩) +
    ½|Φ⁻⟩(α|0⟩ - β|1⟩) +
    ½|Ψ⁻⟩(α|1⟩ - β|0⟩)

where:
    |Φ±⟩ = (|00⟩ ± |11⟩)/√2
    |Ψ±⟩ = (|01⟩ ± |10⟩)/√2


CORRECTION OPERATORS:

    Outcome    Bob's state        Correction
    ────────────────────────────────────────
    |Φ⁺⟩ (00)  α|0⟩ + β|1⟩       I
    |Ψ⁺⟩ (01)  α|1⟩ + β|0⟩       X
    |Φ⁻⟩ (10)  α|0⟩ - β|1⟩       Z
    |Ψ⁻⟩ (11)  α|1⟩ - β|0⟩       ZX = iY
```

## Why It Works

```
Key Insights
════════════

1. NO FASTER-THAN-LIGHT COMMUNICATION
   Bob's qubit is random until correction
   Classical bits must be sent (speed of light)

2. NO CLONING
   Alice's qubit is destroyed by measurement
   |ψ⟩ doesn't exist in two places

3. ENTANGLEMENT AS RESOURCE
   Bell pair is consumed (one-time use)
   Entanglement "spent" to send quantum info

4. CLASSICAL + QUANTUM = MORE POWERFUL
   Neither alone can do this
   Demonstrates complementarity


RESOURCE EQUATION:

    1 ebit + 2 cbits → 1 qubit teleported

    (ebit = shared Bell pair)
    (cbit = classical bit)
    (qubit = quantum bit of information)
```

## Teleportation Circuit

```
Quantum Circuit
═══════════════

    |ψ⟩ ────●────[H]────[M]──══╗
            │               ║
    |A⟩ ────X─────────[M]──╬═╗
                           ║ ║
    |B⟩ ════════════════[Z][X]═══ |ψ⟩

    ├─────────────────┤    ├───┤
    Alice's side           Bob's side
    (Entangled)           (Classical
                           bits control)


GATE SEQUENCE:
    1. CNOT(ψ → A)
    2. H(ψ)
    3. Measure ψ → controls Z
    4. Measure A → controls X
    5. Conditioned X and Z on Bob
```

## Teleportation as a Primitive

```
Teleportation in Quantum Computing
══════════════════════════════════

GATE TELEPORTATION:
    Can teleport gates, not just states!

    |ψ⟩ → [U] → U|ψ⟩

    Using entanglement:
    - Prepare U|Φ⁺⟩ (encoded gate)
    - Teleport through it
    - Apply corrections

    Basis for:
    - One-way quantum computing
    - Fault-tolerant gates


MEASUREMENT-BASED QC:
    Prepare cluster state (highly entangled)
    Compute by measuring qubits
    Teleportation powers the computation!


QUANTUM NETWORKS:
    Teleportation = quantum data link
    Send qubits between nodes
    Quantum internet architecture
```

## Summary

Quantum teleportation demonstrates fundamental quantum information principles:

1. **Protocol**: CNOT + H + Measure + Correct
2. **Resources**: 1 ebit + 2 cbits → 1 qubit
3. **No cloning**: Original state destroyed
4. **No FTL**: Classical communication required
5. **Foundation**: For quantum networks and computation

Teleportation shows entanglement's power as a communication resource.
