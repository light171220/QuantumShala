# Von Neumann Entropy

## Introduction

The von Neumann entropy is the quantum analog of Shannon entropy. It measures the uncertainty or mixedness of a quantum state, playing a central role in quantum information theory.

## Definition

```python
import numpy as np
from scipy.linalg import logm

def von_neumann_entropy(rho):
    """
    Calculate von Neumann entropy S(ρ).

    S(ρ) = -Tr(ρ log₂ ρ)

    Equivalently, using eigenvalues:
    S(ρ) = -Σ λᵢ log₂(λᵢ)
    """
    # Get eigenvalues
    eigenvalues = np.linalg.eigvalsh(rho)
    # Filter out zeros (0*log(0) = 0)
    eigenvalues = eigenvalues[eigenvalues > 1e-10]
    # Calculate entropy
    return -np.sum(eigenvalues * np.log2(eigenvalues))

# Examples
print("Von Neumann Entropy Examples")
print("=" * 40)

# Pure state |0⟩ (minimum entropy)
pure_state = np.array([[1, 0], [0, 0]])
print(f"Pure state |0⟩: S = {von_neumann_entropy(pure_state):.3f} bits")

# Maximally mixed state (maximum entropy)
mixed_state = np.array([[0.5, 0], [0, 0.5]])
print(f"Maximally mixed I/2: S = {von_neumann_entropy(mixed_state):.3f} bit")

# Partially mixed state
partial = np.array([[0.8, 0], [0, 0.2]])
print(f"Partially mixed: S = {von_neumann_entropy(partial):.3f} bits")
```

## Pure vs Mixed States

```
Entropy and Purity
══════════════════

PURE STATES: S(ρ) = 0
    ρ = |ψ⟩⟨ψ|
    Only one eigenvalue (λ = 1)
    No uncertainty about the state
    Examples: |0⟩, |+⟩, any |ψ⟩

MAXIMALLY MIXED: S(ρ) = log₂(d)
    ρ = I/d  (d = dimension)
    All eigenvalues equal (λᵢ = 1/d)
    Maximum uncertainty
    Example: I/2 for qubit

GENERAL MIXED: 0 < S(ρ) < log₂(d)
    ρ = Σᵢ pᵢ|ψᵢ⟩⟨ψᵢ|
    Intermediate uncertainty


ENTROPY SPECTRUM:

    S(ρ)
    │
  1 ├───────────────●  Maximally mixed (I/2)
    │            ·
    │         ·
    │      ·
    │   ·
  0 ├──●               Pure state
    └──────────────────
      Pure      Mixed
```

""")

## Key Properties

```
Properties of Von Neumann Entropy
═════════════════════════════════

1. NON-NEGATIVITY
   S(ρ) ≥ 0
   Zero iff ρ is pure

2. BOUNDED ABOVE
   S(ρ) ≤ log₂(d)
   Maximum for maximally mixed state

3. UNITARY INVARIANCE
   S(UρU†) = S(ρ)
   Unitary operations preserve entropy

4. CONCAVITY
   S(Σᵢ pᵢρᵢ) ≥ Σᵢ pᵢS(ρᵢ)
   Mixing increases entropy

5. ADDITIVITY (product states)
   S(ρ ⊗ σ) = S(ρ) + S(σ)
   For uncorrelated systems


UNLIKE CLASSICAL:
   - Subadditivity can be tight for entanglement
   - Conditional entropy can be NEGATIVE
   - Strong subadditivity holds
```

## Subadditivity

```python
def demonstrate_subadditivity():
    """
    S(ρAB) ≤ S(ρA) + S(ρB)

    Equality holds iff ρAB = ρA ⊗ ρB
    """
    print("Subadditivity of Entropy")
    print("=" * 40)

    # Maximally entangled state |Φ⁺⟩ = (|00⟩ + |11⟩)/√2
    bell_state = np.array([
        [0.5, 0, 0, 0.5],
        [0, 0, 0, 0],
        [0, 0, 0, 0],
        [0.5, 0, 0, 0.5]
    ])

    # Partial trace to get ρA
    rho_A = np.array([[0.5, 0], [0, 0.5]])  # I/2

    print(f"Bell state |Φ⁺⟩:")
    print(f"  S(ρAB) = {von_neumann_entropy(bell_state):.3f} bits (pure!)")
    print(f"  S(ρA)  = {von_neumann_entropy(rho_A):.3f} bit")
    print(f"  S(ρB)  = {von_neumann_entropy(rho_A):.3f} bit")
    print(f"\nSubadditivity: 0 ≤ 1 + 1 = 2  ✓")
    print(f"\nNote: S(ρAB) < S(ρA)!")
    print("This is impossible classically - signals entanglement!")

demonstrate_subadditivity()
```

## Strong Subadditivity

```
Strong Subadditivity
════════════════════

The deepest inequality in quantum information:

    S(ρABC) + S(ρB) ≤ S(ρAB) + S(ρBC)

EQUIVALENT FORMS:

    S(A|BC) ≤ S(A|B)
    "Conditioning reduces entropy"

    I(A;C|B) ≥ 0
    "Conditional mutual information non-negative"


CONSEQUENCES:

    1. Data processing inequality
       Information never increases through channels

    2. Quantum Markov chains
       Equality iff quantum Markov condition

    3. Entanglement measures
       Many properties follow from SSA


HISTORY:
    - Conjectured: Lanford & Robinson, 1968
    - Proved: Lieb & Ruskai, 1973
    - Essential for quantum information theory
```

## Relative Entropy

```python
def quantum_relative_entropy(rho, sigma):
    """
    S(ρ||σ) = Tr(ρ log ρ) - Tr(ρ log σ)

    Measures distinguishability of quantum states.
    """
    # Simplified: use eigenvalue decomposition
    print("Quantum Relative Entropy S(ρ||σ)")
    print("=" * 40)
    print("""
DEFINITION:
    S(ρ||σ) = Tr(ρ log ρ) - Tr(ρ log σ)

PROPERTIES:
    S(ρ||σ) ≥ 0           (always non-negative)
    S(ρ||σ) = 0  iff ρ = σ
    S(ρ||σ) ≠ S(σ||ρ)     (asymmetric!)

APPLICATIONS:
    - Hypothesis testing
    - Channel capacity
    - Entanglement measures
    - Thermodynamics (free energy)

MONOTONICITY:
    S(ε(ρ)||ε(σ)) ≤ S(ρ||σ)

    Quantum operations cannot increase
    distinguishability.
""")

quantum_relative_entropy(None, None)
```

## Entropy of Entanglement

```
Entanglement Entropy
════════════════════

For pure bipartite state |ψ⟩AB:

    E(|ψ⟩) = S(ρA) = S(ρB)

    where ρA = TrB(|ψ⟩⟨ψ|)


EXAMPLES:

    |00⟩:
        ρA = |0⟩⟨0|
        E = 0 (product state)

    |Φ⁺⟩ = (|00⟩ + |11⟩)/√2:
        ρA = I/2
        E = 1 ebit (maximally entangled)


SCHMIDT DECOMPOSITION:
    |ψ⟩ = Σᵢ λᵢ|iA⟩|iB⟩

    E = -Σᵢ λᵢ² log₂(λᵢ²)

    (entropy of Schmidt coefficients)
```

## Summary

Von Neumann entropy is the fundamental measure of quantum uncertainty:

1. **Definition**: S(ρ) = -Tr(ρ log ρ) using eigenvalues
2. **Pure states**: Zero entropy (no uncertainty)
3. **Mixed states**: Positive entropy (quantum uncertainty)
4. **Subadditivity**: Entangled states violate classical intuition
5. **Applications**: Entanglement, channels, thermodynamics

This entropy underlies all of quantum information theory.
