## 1. The Core Problem

<Definition term="Phase Estimation Problem">
Given a unitary operator U and one of its eigenvectors |ψ⟩, find the eigenvalue phase φ where:

$$U|ψ⟩ = e^{2πiφ}|ψ⟩$$
</Definition>

The eigenvalue is always of the form e^(2πiφ) because unitary operators preserve norm.

```python
import numpy as np

# Example: Phase gate as a unitary with known eigenvalue
def phase_gate(theta):
    """Create phase gate with eigenvalue e^(i*theta)."""
    return np.array([[1, 0], [0, np.exp(1j * theta)]])

# |1⟩ is an eigenvector with eigenvalue e^(i*theta)
theta = np.pi / 4  # 45 degrees
U = phase_gate(theta)
ket_1 = np.array([0, 1])

result = U @ ket_1
eigenvalue = result[1] / ket_1[1]

print("Phase Estimation Example")
print("=" * 40)
print(f"U|1⟩ = e^(iπ/4)|1⟩")
print(f"Eigenvalue: {eigenvalue:.4f}")
print(f"Expected: {np.exp(1j * theta):.4f}")
print(f"Phase φ = θ/(2π) = {theta / (2 * np.pi):.4f}")
```

## 2. Why Phases Matter

Quantum algorithms extract information through phases:


## 3. Unitary Eigenvalue Structure

All eigenvalues of unitary matrices have magnitude 1:

<Formula>
$$U|ψ⟩ = λ|ψ⟩ \implies |λ| = 1 \implies λ = e^{2πiφ}$$
</Formula>

```python
import numpy as np

def analyze_unitary_eigenvalues(U, name="U"):
    """Analyze eigenvalues of a unitary matrix."""
    eigenvalues, eigenvectors = np.linalg.eig(U)
    
    print(f"\nEigenvalue analysis of {name}:")
    print("-" * 40)
    
    for i, (val, vec) in enumerate(zip(eigenvalues, eigenvectors.T)):
        magnitude = np.abs(val)
        phase = np.angle(val) / (2 * np.pi)  # Normalized to [0, 1)
        if phase < 0:
            phase += 1
        
        print(f"  λ_{i} = {val:.4f}")
        print(f"       |λ| = {magnitude:.4f} (should be 1)")
        print(f"       φ = {phase:.4f}")

# Examples
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
X = np.array([[0, 1], [1, 0]])
T = np.array([[1, 0], [0, np.exp(1j * np.pi / 4)]])

analyze_unitary_eigenvalues(H, "Hadamard")
analyze_unitary_eigenvalues(X, "Pauli-X")
analyze_unitary_eigenvalues(T, "T-gate")
```

## 4. The Classical Difficulty

Classically, estimating φ to n bits of precision requires exponential resources:

```python
import numpy as np

def classical_phase_estimation_difficulty():
    """
    Explain why classical phase estimation is hard.
    """
    print("\nClassical Difficulty")
    print("=" * 50)
    print("""
To estimate φ to n bits classically:

1. Apply U many times to build up phase
2. Interference effects cancel without quantum coherence
3. Need ~2^n applications of U

Quantum Phase Estimation:
- Uses superposition to query all powers of U simultaneously
- QFT extracts phase from interference pattern
- Only O(n) controlled-U operations needed!
    """)
    
    print("\nResource comparison:")
    print(f"{'Bits':>6} {'Classical':>15} {'Quantum':>15}")
    print("-" * 38)
    for n in [4, 8, 12, 16, 20]:
        classical = 2**n
        quantum = n
        print(f"{n:>6} {classical:>15,} {quantum:>15}")

classical_phase_estimation_difficulty()
```

## 5. Problem Setup Summary

<Tip>
**Given:**
- Unitary operator U (as a quantum circuit)
- Eigenvector |ψ⟩ (or ability to prepare it)

**Goal:**
- Find φ such that U|ψ⟩ = e^(2πiφ)|ψ⟩
- Estimate φ to n bits: φ ≈ φ̃ = 0.φ₁φ₂...φₙ

**Resources:**
- n "counting" qubits for precision
- Controlled-U, controlled-U², ..., controlled-U^(2^(n-1)) operations
- QFT circuit
</Tip>

## Summary

In this lesson, you learned:

- QPE finds eigenvalue phases of unitary operators
- Unitary eigenvalues have form e^(2πiφ) with |eigenvalue| = 1
- Phase estimation is crucial for Shor's, quantum chemistry, HHL
- Classical phase estimation requires exponential resources
- Quantum approach: polynomial resources via superposition + QFT

## Key Formula

<Formula>
$$U|ψ⟩ = e^{2πiφ}|ψ⟩, \quad φ \in [0, 1)$$
</Formula>
