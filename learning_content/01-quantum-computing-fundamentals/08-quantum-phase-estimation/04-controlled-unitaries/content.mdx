## 1. The Role of Controlled-U in QPE

In QPE, each counting qubit k controls the application of U^(2^k) to the eigenstate register:

<Formula>
$$\text{C-}U^{2^k}|c\rangle|\psi\rangle = |c\rangle U^{c \cdot 2^k}|\psi\rangle$$
</Formula>

When the control qubit is |0⟩, nothing happens. When |1⟩, U^(2^k) is applied.

```python
import numpy as np

def controlled_unitary(U):
    """
    Create controlled version of unitary U.
    CU = |0⟩⟨0| ⊗ I + |1⟩⟨1| ⊗ U
    """
    n = U.shape[0]
    I = np.eye(n)
    
    # Projectors
    P0 = np.array([[1, 0], [0, 0]])  # |0⟩⟨0|
    P1 = np.array([[0, 0], [0, 1]])  # |1⟩⟨1|
    
    CU = np.kron(P0, I) + np.kron(P1, U)
    return CU

# Example: Controlled-Z
Z = np.array([[1, 0], [0, -1]])
CZ = controlled_unitary(Z)
print("Controlled-Z matrix:")
print(CZ)
```

## 2. Computing U^(2^k) Efficiently

Naive approach: Apply U 2^k times (exponential).
Smart approach: Repeated squaring (linear in k).

```python
import numpy as np

def unitary_power(U, k):
    """
    Compute U^(2^k) using repeated squaring.
    
    U^(2^k) = (U^(2^(k-1)))^2
    """
    result = U.copy()
    for _ in range(k):
        result = result @ result
    return result

# Example: Phase gate P(θ) = diag(1, e^(iθ))
def phase_gate(theta):
    return np.array([[1, 0], [0, np.exp(1j * theta)]])

theta = np.pi / 8
P = phase_gate(theta)

print(f"P(π/8) eigenvalue: e^(i·π/8)")
print()

for k in range(5):
    Pk = unitary_power(P, k)
    phase = np.angle(Pk[1, 1])
    print(f"P^(2^{k}) = P^{2**k}: phase = {phase:.4f} = {phase/np.pi:.4f}π")
```

## 3. Multi-Qubit Controlled Operations

For n-qubit unitaries U, the controlled version acts on n+1 qubits:

```python
import numpy as np

def multi_controlled_U(U, num_controls=1):
    """
    Create multiply-controlled version of U.
    
    For single control: CU = |0⟩⟨0| ⊗ I + |1⟩⟨1| ⊗ U
    For two controls: CCU = |00⟩⟨00|⊗I + |01⟩⟨01|⊗I + |10⟩⟨10|⊗I + |11⟩⟨11|⊗U
    """
    n = U.shape[0]
    total_dim = n * (2 ** num_controls)
    result = np.zeros((total_dim, total_dim), dtype=complex)
    
    # All control patterns except all-1s get identity
    for c in range(2**num_controls - 1):
        start = c * n
        result[start:start+n, start:start+n] = np.eye(n)
    
    # All-1s pattern gets U
    start = (2**num_controls - 1) * n
    result[start:start+n, start:start+n] = U
    
    return result

# Example: Toffoli-like controlled unitary
X = np.array([[0, 1], [1, 0]])
CCX = multi_controlled_U(X, num_controls=2)
print("Doubly-controlled X (Toffoli-like):")
print(CCX.astype(int))
```

## 4. QPE Circuit Structure

```python
import numpy as np

def qpe_controlled_layers(U, eigenstate, t):
    """
    Simulate the controlled-U layers in QPE.
    
    Returns state after all controlled operations.
    """
    # Start: counting register in |+⟩^t, eigenstate in |ψ⟩
    # Each control k applies U^(2^k) when control is |1⟩
    
    # For eigenstate with eigenvalue e^(2πiφ):
    # After k-th control: |control_k⟩ picks up phase 2^k·φ
    
    # Simulate phases picked up
    n_counting = t
    phases = []
    
    for k in range(t):
        # This qubit controls U^(2^k)
        # Phase kickback: e^(2πi · 2^k · φ)
        power = 2**k
        phases.append(power)
    
    return phases

# Example
t = 4
phases = qpe_controlled_layers(None, None, t)
print(f"QPE with t={t} counting qubits:")
print(f"Controlled operations: U^1, U^2, U^4, U^8, ...")
print(f"Phase factors: 2^k for k = 0, 1, ..., {t-1}")
print(f"Powers: {phases}")
```

## 5. Practical Considerations

Key insights for implementing QPE:

1. **Gate decomposition**: Break U into native gates
2. **Repeated squaring**: Compute U^(2^k) = (U^(2^(k-1)))²
3. **Approximate unitaries**: For large powers, approximation may be needed
4. **Circuit depth**: Scales with t (counting qubits) and complexity of U

```python
import numpy as np

def estimate_qpe_depth(U_depth, t):
    """
    Estimate total circuit depth for QPE.
    
    Args:
        U_depth: Depth of one U application
        t: Number of counting qubits
    
    Returns:
        Estimated total depth
    """
    # Controlled-U^(2^k) depth ≈ 2^k × U_depth (naive)
    # With repeated squaring: k × U_depth
    
    total_depth_naive = sum(2**k * U_depth for k in range(t))
    total_depth_smart = sum((k+1) * U_depth for k in range(t))
    
    # Add inverse QFT depth: O(t²)
    iqft_depth = t * (t + 1) // 2
    
    return {
        'naive': total_depth_naive + iqft_depth,
        'optimized': total_depth_smart + iqft_depth,
        'iqft': iqft_depth
    }

U_depth = 10
for t in [4, 8, 12]:
    depths = estimate_qpe_depth(U_depth, t)
    print(f"t={t}: naive={depths['naive']}, optimized={depths['optimized']}")
```

## Summary

Key takeaways:

1. **Controlled-U^(2^k)** applies U^(2^k) when control is |1⟩
2. **Repeated squaring** computes U^(2^k) in O(k) operations
3. **Multi-controlled gates** extend to multiple control qubits
4. **Circuit depth** depends on U complexity and qubit count
5. **Phase kickback** is what makes QPE work

These controlled operations are the heart of QPE and essential for algorithms like Shor's.
