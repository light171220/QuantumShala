## 1. Formal Problem Definition

<Definition term="Unstructured Search Problem">
Given a boolean function f: {0,1}^n → {0,1} with exactly one x* where f(x*) = 1 (the "marked item"), find x* using the minimum number of queries to f.
</Definition>

The search space has N = 2^n items. We can only learn about f by querying it.

```python
import numpy as np

def example_oracle(x, marked_item):
    """Simple oracle: returns 1 if x is the marked item."""
    return 1 if x == marked_item else 0

# Example: searching in 8 items (n=3 qubits)
n = 3
N = 2**n
marked = 5  # Our secret marked item

print(f"Search space: {N} items")
print(f"Marked item: {marked}")
print()

# Classical search
for x in range(N):
    result = example_oracle(x, marked)
    print(f"f({x}) = {result}", "← Found!" if result == 1 else "")
```

## 2. The Oracle Model

In quantum computing, we use a **phase oracle** that marks the solution with a phase flip:

<Formula>
$$O_f|x\rangle = (-1)^{f(x)}|x\rangle$$
</Formula>

```python
import numpy as np

def phase_oracle(marked_item, n_qubits):
    """Create phase oracle as a diagonal matrix."""
    N = 2**n_qubits
    # Identity with -1 at the marked position
    oracle = np.eye(N)
    oracle[marked_item, marked_item] = -1
    return oracle

# Example for 3 qubits, marked item = 5
oracle = phase_oracle(5, 3)
print("Phase Oracle (diagonal entries):")
print(np.diag(oracle))
print("\nOracle flips phase of |5⟩ = |101⟩")
```

## 3. Initial State Preparation

Grover's algorithm starts with a uniform superposition over all states:

<Formula>
$$|s\rangle = H^{\otimes n}|0\rangle^{\otimes n} = \frac{1}{\sqrt{N}}\sum_{x=0}^{N-1}|x\rangle$$
</Formula>

```python
import numpy as np

def create_uniform_superposition(n_qubits):
    """Create uniform superposition |s⟩."""
    N = 2**n_qubits
    return np.ones(N) / np.sqrt(N)

n = 3
s = create_uniform_superposition(n)

print(f"Uniform superposition over {2**n} states:")
print(f"|s⟩ = {np.round(s, 4)}")
print(f"\nEach amplitude: 1/√{2**n} = {1/np.sqrt(2**n):.4f}")
print(f"Probability of each state: {1/(2**n):.4f}")
```

## 4. Geometric Interpretation

The key insight is to view the problem geometrically in a 2D plane:

- **|w⟩**: The marked state (what we're looking for)
- **|s'⟩**: Uniform superposition over unmarked states
- **|s⟩**: Initial state (lies mostly along |s'⟩)

```python
import numpy as np

def analyze_geometry(n_qubits, marked_item):
    """Analyze the geometric structure."""
    N = 2**n_qubits
    
    # |w⟩: marked state
    w = np.zeros(N)
    w[marked_item] = 1
    
    # |s⟩: uniform superposition
    s = np.ones(N) / np.sqrt(N)
    
    # Angle between |s⟩ and |w⟩
    sin_theta = np.abs(np.dot(s, w))  # = 1/√N
    theta = np.arcsin(sin_theta)
    
    print(f"N = {N} items")
    print(f"sin(θ) = 1/√N = {sin_theta:.4f}")
    print(f"θ = {np.degrees(theta):.2f}°")
    print(f"Initial probability of marked item: {sin_theta**2:.4f}")

analyze_geometry(3, 5)
print()
analyze_geometry(10, 42)
```

## Summary

- Search problem: find marked item in N items
- Phase oracle: O|x⟩ = (-1)^f(x)|x⟩
- Initial state: uniform superposition |s⟩ = H^⊗n|0⟩
- Geometry: |s⟩ is nearly perpendicular to |w⟩ (angle θ ≈ 1/√N)
