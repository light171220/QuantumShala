## 1. Cryptographic Applications

Grover's halves the effective key length of symmetric encryption:

```python
print("Impact on Symmetric Cryptography:")
print(f"{'Algorithm':<15} {'Key Size':<10} {'Effective':>10}")
print("-" * 35)

for name, bits in [("DES", 56), ("AES-128", 128), ("AES-256", 256)]:
    effective = bits // 2
    status = "BROKEN" if effective < 64 else "OK" if effective < 128 else "SECURE"
    print(f"{name:<15} {bits:<10} {effective:>10} ({status})")
```

## 2. Database Search

Search unsorted data in O(√N) time:
- Faster than any classical algorithm
- Requires quantum RAM (QRAM) for practical use

## 3. Optimization

Speed up brute-force optimization:
- SAT solving
- Constraint satisfaction
- Combinatorial optimization

## 4. Limitations

- Quadratic (not exponential) speedup
- Need quantum oracle access
- QRAM requirements for database search

## Summary

- Cryptography: doubles required key lengths
- Database: √N speedup (if QRAM available)
- Optimization: quadratic speedup for NP problems
- Practical challenges remain (QRAM, error correction)
