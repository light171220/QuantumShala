## 1. From Bit Oracle to Phase Oracle

A classical oracle computes f(x) into an auxiliary bit. We convert this to a phase oracle using phase kickback.

<Formula>
$$U_f|x\rangle|y\rangle = |x\rangle|y \oplus f(x)\rangle$$
</Formula>

With ancilla in |−⟩ state:

<Formula>
$$U_f|x\rangle|-\rangle = (-1)^{f(x)}|x\rangle|-\rangle$$
</Formula>

```python
import numpy as np

def bit_oracle_to_phase(f, n_qubits):
    """
    Convert function f to phase oracle.
    f: function that returns 0 or 1
    """
    N = 2**n_qubits
    oracle = np.eye(N)
    for x in range(N):
        if f(x) == 1:
            oracle[x, x] = -1
    return oracle

# Example: mark states where x mod 3 == 0
def f(x):
    return 1 if x % 3 == 0 else 0

oracle = bit_oracle_to_phase(f, 3)
print("Oracle marks: x mod 3 == 0")
print("Diagonal:", np.diag(oracle))
```

## 2. Multi-Solution Oracles

When multiple items are marked:

```python
import numpy as np

def multi_solution_oracle(marked_items, n_qubits):
    """Create oracle marking multiple items."""
    N = 2**n_qubits
    oracle = np.eye(N)
    for item in marked_items:
        oracle[item, item] = -1
    return oracle

# Mark items 2, 5, and 7 in 3-qubit system
marked = [2, 5, 7]
oracle = multi_solution_oracle(marked, 3)

print("Multi-solution oracle (k=3 solutions):")
print("Diagonal:", np.diag(oracle))
```

## 3. Oracle from Boolean Function

For SAT-like problems, build oracle from boolean formula:

```python
import numpy as np

def sat_oracle(n_qubits, clause_function):
    """
    Oracle for SAT problem.
    clause_function(x): returns True if x satisfies all clauses
    """
    N = 2**n_qubits
    oracle = np.eye(N)
    
    for x in range(N):
        # Convert integer to bit array
        bits = [(x >> i) & 1 for i in range(n_qubits)]
        if clause_function(bits):
            oracle[x, x] = -1
    
    return oracle

# Example: (x0 OR x1) AND (NOT x1 OR x2)
def example_sat(bits):
    x0, x1, x2 = bits
    clause1 = x0 or x1
    clause2 = (not x1) or x2
    return clause1 and clause2

oracle = sat_oracle(3, example_sat)
satisfying = [i for i in range(8) if oracle[i,i] == -1]
print(f"Satisfying assignments: {satisfying}")
```

## Summary

- Phase oracle: O|x⟩ = (-1)^f(x)|x⟩
- Use phase kickback with |−⟩ ancilla
- Multi-solution oracles mark multiple items
- SAT oracles from boolean functions
