## 1. Reflection About the Mean

The diffusion operator reflects amplitudes about their mean:

<Formula>
$$D = 2|s\rangle\langle s| - I$$
</Formula>

where |s⟩ is the uniform superposition.

```python
import numpy as np

def diffusion_operator(n_qubits):
    """Create diffusion operator D = 2|s⟩⟨s| - I"""
    N = 2**n_qubits
    
    # |s⟩ = uniform superposition
    s = np.ones(N) / np.sqrt(N)
    
    # D = 2|s⟩⟨s| - I
    D = 2 * np.outer(s, s) - np.eye(N)
    return D

D = diffusion_operator(2)
print("Diffusion operator (2 qubits):")
print(np.round(D, 3))
```

## 2. How Reflection Works

For a state with amplitudes [a₀, a₁, ..., aₙ₋₁]:
1. Calculate mean: μ = (a₀ + a₁ + ... + aₙ₋₁)/N
2. New amplitudes: aᵢ → 2μ - aᵢ

```python
import numpy as np

def visualize_reflection(amplitudes):
    """Show how reflection about mean works."""
    mean = np.mean(amplitudes)
    reflected = 2 * mean - amplitudes
    
    print(f"Original amplitudes: {np.round(amplitudes, 4)}")
    print(f"Mean: {mean:.4f}")
    print(f"After reflection: {np.round(reflected, 4)}")
    return reflected

# Example: after oracle marks item 0
N = 4
state = np.ones(N) / np.sqrt(N)
state[0] = -state[0]  # Oracle flipped phase

print("After oracle:")
visualize_reflection(state)
```

## 3. Circuit Implementation

D = H^⊗n · (2|0⟩⟨0| - I) · H^⊗n

```python
import numpy as np

def diffusion_circuit(n_qubits):
    """Build D from Hadamards and conditional phase."""
    N = 2**n_qubits
    
    # Hadamard on all qubits
    H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
    H_all = H
    for _ in range(n_qubits - 1):
        H_all = np.kron(H_all, H)
    
    # 2|0⟩⟨0| - I (phase flip all states except |0⟩)
    phase_flip = -np.eye(N)
    phase_flip[0, 0] = 1
    
    # D = H · phase_flip · H
    D = H_all @ phase_flip @ H_all
    return D

D = diffusion_circuit(2)
print("Diffusion from circuit:")
print(np.round(D, 3))
```

## 4. Geometric Interpretation

The diffusion operator reflects the state about |s⟩ in the 2D plane spanned by |w⟩ and |s⟩.

```python
import numpy as np

def grover_step(state, oracle, diffusion):
    """One Grover iteration: Oracle then Diffusion."""
    state = oracle @ state
    state = diffusion @ state
    return state

# Track angle to solution
n = 3
marked = 5
N = 2**n

oracle = np.eye(N)
oracle[marked, marked] = -1
D = 2 * np.outer(np.ones(N)/np.sqrt(N), np.ones(N)/np.sqrt(N)) - np.eye(N)

state = np.ones(N) / np.sqrt(N)

print("Amplitude of marked item after each iteration:")
for i in range(5):
    prob = np.abs(state[marked])**2
    print(f"  Iteration {i}: amplitude={state[marked]:.4f}, prob={prob:.4f}")
    state = grover_step(state, oracle, D)
```

## Summary

- Diffusion: D = 2|s⟩⟨s| - I
- Reflects amplitudes about their mean
- Circuit: H^⊗n · (2|0⟩⟨0| - I) · H^⊗n
- Each iteration increases marked amplitude
