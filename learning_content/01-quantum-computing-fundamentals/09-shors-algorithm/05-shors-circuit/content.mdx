# Shor's Algorithm Circuit

## Introduction

Shor's algorithm requires a carefully constructed quantum circuit that performs modular exponentiation in superposition, followed by quantum Fourier transform to extract the period. This lesson details the complete circuit structure.

## Circuit Overview

```
Shor's Algorithm Circuit Structure
══════════════════════════════════

INPUT: N (number to factor), a (random coprime to N)
OUTPUT: Period r of f(x) = a^x mod N

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  |0⟩ ─────[H]─────────●─────────────────────[QFT†]───[M]──→ x  │
│  |0⟩ ─────[H]─────────┼────●────────────────[QFT†]───[M]──→    │
│  |0⟩ ─────[H]─────────┼────┼────●───────────[QFT†]───[M]──→    │
│   ⋮                   │    │    │                              │
│  |0⟩ ─────[H]─────────┼────┼────┼────●──────[QFT†]───[M]──→    │
│                       │    │    │    │                         │
│  ════════════════════════════════════════════════════════════  │
│                       │    │    │    │                         │
│  |0⟩ ─────────────────┼────┼────┼────┼──────────────────────   │
│  |0⟩ ─────────────────┼────┼────┼────┼──────────────────────   │
│   ⋮        WORK      [a¹] [a²] [a⁴] [a^2ⁿ]  mod N             │
│  |1⟩ ─────REGISTER────┴────┴────┴────┴──────────────────────   │
│                                                                 │
│  CONTROL REGISTER          MODULAR              TARGET         │
│  (2n qubits)              EXPONENTIATION        REGISTER       │
│                                                 (n qubits)     │
└─────────────────────────────────────────────────────────────────┘

STAGES:
1. Initialize: Control in |0⟩, Target in |1⟩
2. Hadamard: Create superposition in control register
3. Modular Exp: Compute a^x mod N controlled on x
4. Inverse QFT: Extract period information
5. Measure: Get value related to period
```

## Register Requirements

```python
def calculate_register_sizes(N):
    """
    Determine register sizes for factoring N.

    Control register: 2n qubits where N < 2^n
    Target register: n qubits to hold values mod N
    """
    import math

    n = math.ceil(math.log2(N))
    control_qubits = 2 * n  # Need 2n for sufficient precision
    target_qubits = n       # Hold values up to N-1

    print(f"""
Register Sizes for Factoring N = {N}
════════════════════════════════════

n = ⌈log₂(N)⌉ = {n}

CONTROL REGISTER: {control_qubits} qubits
    - Holds superposition of x values
    - Range: 0 to 2^{control_qubits} - 1 = {2**control_qubits - 1}
    - Precision for period extraction

TARGET REGISTER: {target_qubits} qubits
    - Holds a^x mod N values
    - Range: 0 to N-1 = {N-1}
    - Initialized to |1⟩

TOTAL QUBITS: {control_qubits + target_qubits}
    (Plus ancilla for modular arithmetic)
    """)

    return control_qubits, target_qubits

# Example: Factor 15
calculate_register_sizes(15)
```

## Step-by-Step Circuit Construction

```
Step 1: Initialization
══════════════════════

Control Register: |0⟩^⊗2n
Target Register:  |1⟩ (encoded in n qubits as |00...01⟩)

    |0⟩──────
    |0⟩──────     Control: Will hold x values
    |0⟩──────
    |0⟩──────
    ─────────
    |0⟩──────
    |0⟩──────     Target: Initialized to |1⟩
    |0⟩──────
    |1⟩──────


Step 2: Hadamard on Control Register
════════════════════════════════════

Apply H to each control qubit:

    |0⟩──[H]──|+⟩
    |0⟩──[H]──|+⟩
    |0⟩──[H]──|+⟩     Creates uniform superposition:
    |0⟩──[H]──|+⟩     |ψ⟩ = (1/√2^2n) Σ|x⟩
    ──────────────
    |1⟩───────|1⟩     Target unchanged


Step 3: Controlled Modular Exponentiation
═════════════════════════════════════════

For each control qubit j (from 0 to 2n-1):
Apply controlled-U^(2^j) where U|y⟩ = |ay mod N⟩

    |x₀⟩────●─────────────────────
            │
    |x₁⟩────┼────●────────────────
            │    │
    |x₂⟩────┼────┼────●───────────
            │    │    │
    |x₃⟩────┼────┼────┼────●──────
            │    │    │    │
    ═══════════════════════════════
            │    │    │    │
    |1⟩────[a¹]─[a²]─[a⁴]─[a⁸]────|a^x mod N⟩

Result: |x⟩|1⟩ → |x⟩|a^x mod N⟩


Step 4: Inverse Quantum Fourier Transform
═════════════════════════════════════════

Apply QFT† to control register only:

    |x⟩──[QFT†]──|φ⟩

This transforms the periodic structure into
peaks at multiples of 2^2n/r


Step 5: Measurement
═══════════════════

Measure control register to get value m
Then: m/2^2n ≈ k/r for some integer k
Use continued fractions to extract r
```

## Complete Circuit Example: Factoring 15

```python
def shors_circuit_example():
    """
    Trace through Shor's circuit for N=15, a=7.
    """
    print("""
Complete Example: Factor N=15 with a=7
══════════════════════════════════════

SETUP:
    N = 15, a = 7 (coprime to 15)
    n = 4 (since 15 < 2^4 = 16)
    Control register: 8 qubits
    Target register: 4 qubits

STEP 1: Initialize
    |ψ₀⟩ = |00000000⟩|0001⟩
           (control)  (target = 1)

STEP 2: Hadamard on control
    |ψ₁⟩ = (1/16) Σ_{x=0}^{255} |x⟩|1⟩

STEP 3: Modular exponentiation
    |ψ₂⟩ = (1/16) Σ_{x=0}^{255} |x⟩|7^x mod 15⟩

    Period of 7^x mod 15:
    7^0 = 1
    7^1 = 7
    7^2 = 49 mod 15 = 4
    7^3 = 28 mod 15 = 13
    7^4 = 91 mod 15 = 1  ← Period r = 4!

    Values cycle: 1, 7, 4, 13, 1, 7, 4, 13, ...

STEP 4: State before QFT†
    |ψ₂⟩ = (1/16)[|0⟩|1⟩ + |1⟩|7⟩ + |2⟩|4⟩ + |3⟩|13⟩ +
                  |4⟩|1⟩ + |5⟩|7⟩ + |6⟩|4⟩ + |7⟩|13⟩ + ...]

    Grouping by target value:
    |1⟩: |0⟩ + |4⟩ + |8⟩ + |12⟩ + ...  (spacing = 4 = r)
    |7⟩: |1⟩ + |5⟩ + |9⟩ + |13⟩ + ...
    etc.

STEP 5: After QFT†
    Peaks at m where m/256 ≈ k/4 for k = 0,1,2,3

    m = 0:   m/256 = 0 = 0/4
    m = 64:  m/256 = 1/4
    m = 128: m/256 = 2/4 = 1/2
    m = 192: m/256 = 3/4

STEP 6: Measurement and classical post-processing
    If we measure m = 64:
    64/256 = 1/4 → r = 4 ✓

    Check: r = 4 is even, and 7^2 = 49 ≢ -1 (mod 15)

    Factors: gcd(7^2 ± 1, 15) = gcd(48, 15) = 3
             gcd(7^2 + 1, 15) = gcd(50, 15) = 5

    Result: 15 = 3 × 5 ✓
    """)

shors_circuit_example()
```

## Circuit Depth Analysis

```
Circuit Complexity
══════════════════

For factoring an n-bit number N:

GATE COUNTS:
    Hadamard gates:     O(n)     (2n gates)
    QFT gates:          O(n²)    (n² controlled rotations)
    Modular exp:        O(n³)    (dominant cost)

CIRCUIT DEPTH:
    Total depth: O(n³) with standard implementation
    Can be reduced with parallel techniques

BREAKDOWN:
┌─────────────────────────────────────────────────┐
│  Component              │  Gates    │  Depth   │
├─────────────────────────────────────────────────┤
│  Initialization         │  O(n)     │  O(n)    │
│  Hadamard layer         │  O(n)     │  O(1)    │
│  Modular exponentiation │  O(n³)    │  O(n³)   │
│  Inverse QFT            │  O(n²)    │  O(n)    │
│  Measurement            │  O(n)     │  O(1)    │
├─────────────────────────────────────────────────┤
│  TOTAL                  │  O(n³)    │  O(n³)   │
└─────────────────────────────────────────────────┘

The modular exponentiation dominates:
    - 2n controlled multiplications
    - Each multiplication: O(n²) gates
    - Total: O(n³) gates
```

## Summary

The Shor's algorithm circuit consists of:

1. **Two registers**: Control (2n qubits) and target (n qubits)
2. **Hadamard layer**: Creates superposition of all x values
3. **Modular exponentiation**: Most complex component, O(n³) gates
4. **Inverse QFT**: Extracts period information
5. **Measurement**: Yields value for classical post-processing

Understanding this circuit structure is essential for implementing and optimizing Shor's algorithm.
