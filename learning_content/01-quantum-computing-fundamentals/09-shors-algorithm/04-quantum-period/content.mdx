## 1. The Quantum Approach

The key insight: period finding is equivalent to phase estimation on a carefully constructed unitary.

<Definition term="Modular Exponentiation Unitary">
Define unitary operator U_a acting on computational basis:

$$U_a|y\rangle = |ay \mod N\rangle$$

The eigenvalues of U_a encode the period r!
</Definition>

```python
import numpy as np

def modular_mult_unitary(a, N):
    """
    Create unitary matrix for |y⟩ → |a*y mod N⟩.
    Only defined for y coprime to N.
    """
    # For simplicity, use full N-dimensional space
    U = np.zeros((N, N))
    for y in range(N):
        if np.gcd(y, N) == 1:  # y coprime to N
            U[(a * y) % N, y] = 1
        else:
            U[y, y] = 1  # Leave non-coprime unchanged
    return U

# Example for N=15, a=7
N = 15
a = 7
U = modular_mult_unitary(a, N)

print(f"Modular multiplication unitary U_a for a={a}, N={N}")
print("Action on computational basis:")
for y in [1, 2, 4, 7, 8, 11, 13, 14]:
    if np.gcd(y, N) == 1:
        result = np.argmax(U @ np.eye(N)[y])
        print(f"  U|{y}⟩ = |{a}×{y} mod {N}⟩ = |{result}⟩")
```

## 2. Eigenvalues Encode the Period

The eigenstates of U_a are:

<Formula>
$$|u_s\rangle = \frac{1}{\sqrt{r}} \sum_{k=0}^{r-1} e^{-2\pi i s k/r} |a^k \mod N\rangle$$
</Formula>

with eigenvalue:

<Formula>
$$U_a|u_s\rangle = e^{2\pi i s/r}|u_s\rangle$$
</Formula>

The phase s/r encodes the period r!

```python
import numpy as np

def eigenvalue_analysis(a, N, r):
    """
    Analyze eigenvalues of modular multiplication unitary.
    The eigenvalues are e^(2πis/r) for s = 0, 1, ..., r-1.
    """
    print(f"Eigenvalue Analysis: a={a}, N={N}, r={r}")
    print("=" * 50)
    print(f"\nEigenvalues of U_a are e^(2πis/r) for s = 0,...,{r-1}:")
    print(f"{'s':>4} {'Phase s/r':>12} {'Eigenvalue':>25}")
    print("-" * 45)
    
    for s in range(r):
        phase = s / r
        eigenvalue = np.exp(2j * np.pi * s / r)
        print(f"{s:>4} {phase:>12.4f} {eigenvalue.real:>10.4f} + {eigenvalue.imag:>8.4f}i")
    
    print(f"\nQPE measures phase ≈ s/r, from which we extract r!")

# For a=7 mod 15, period r=4
eigenvalue_analysis(7, 15, 4)
```

## 3. The Quantum Circuit

The quantum period finding circuit:

```
|0⟩^⊗n ─────H^⊗n────┬─────┬──────┬───────QFT†───── Measure → s/r
                     │     │      │
|1⟩    ──────────────U────U²────U^(2^(n-1))────────
```

```python
import numpy as np

def quantum_period_finding_simulation(a, N, n_precision_bits):
    """
    Simulate quantum period finding.
    
    Args:
        a: Base for a^x mod N
        N: Modulus
        n_precision_bits: Number of qubits in counting register
    
    Returns:
        Measured phase (approximation to s/r)
    """
    # The key insight: after QPE, we measure a value close to s/r
    # for some s in {0, 1, ..., r-1}
    
    # Find actual period (for simulation)
    r = 1
    current = a % N
    while current != 1:
        current = (current * a) % N
        r += 1
    
    # Simulate measurement: get random s, compute s/r approximation
    M = 2 ** n_precision_bits
    s = np.random.randint(0, r)
    
    # The measured value is close to M * s / r
    measured_int = round(M * s / r)
    measured_phase = measured_int / M
    
    return measured_phase, s, r

# Simulation
print("Quantum Period Finding Simulation")
print("=" * 50)

a, N = 7, 15
for trial in range(5):
    phase, s, r = quantum_period_finding_simulation(a, N, n_precision_bits=8)
    print(f"Trial {trial+1}: measured phase ≈ {phase:.6f} (true: s={s}, r={r}, s/r={s/r:.6f})")
```

## 4. Extracting the Period

From the measured phase ≈ s/r, we extract r using continued fractions:

```python
from fractions import Fraction

def extract_period_from_phase(measured_phase, N):
    """
    Use continued fractions to extract r from measured phase ≈ s/r.
    """
    # Convert to fraction with denominator at most N
    frac = Fraction(measured_phase).limit_denominator(N)
    
    return frac.denominator, frac.numerator

def continued_fraction_demo(phase, max_denom):
    """Demonstrate continued fraction convergence."""
    print(f"\nContinued Fraction Expansion of {phase}:")
    print(f"{'Denominator limit':>20} {'Fraction':>15} {'Decimal':>12}")
    print("-" * 50)
    
    for limit in [2, 4, 8, 16, 32, max_denom]:
        frac = Fraction(phase).limit_denominator(limit)
        decimal = float(frac)
        print(f"{limit:>20} {str(frac):>15} {decimal:>12.6f}")

# Example: measured phase close to 1/4
true_phase = 1/4  # s=1, r=4
noisy_phase = 0.2498  # Simulated measurement with noise

print("Extracting Period from Measured Phase")
print("=" * 50)
print(f"True phase: s/r = {true_phase}")
print(f"Measured:   {noisy_phase}")

continued_fraction_demo(noisy_phase, 15)

r, s = extract_period_from_phase(noisy_phase, 15)
print(f"\n→ Extracted r = {r} (from fraction {s}/{r})")
```

## 5. Why Quantum is Faster

```python

    """Compare classical vs quantum period finding complexity."""

    ─────────────────────────────────────────────────────────

    • 1024-bit N: Classical ~10^154 ops, Quantum ~10^9 ops

    """)
```

```python
def shors_algorithm_overview():
    """Overview of complete Shor's algorithm."""
    print("\nShor's Algorithm: Complete Overview")
    print("=" * 60)
    print("""
    INPUT: Integer N to factor
    
    CLASSICAL PRE-PROCESSING:
    1. Check if N is even → factor = 2
    2. Check if N = a^k → factor = a
    3. Choose random a, check gcd(a, N)
    
    QUANTUM PART (Period Finding):
    4. Prepare superposition |0⟩|1⟩ → 1/√M Σ|x⟩|1⟩
    5. Apply controlled-U^x: 1/√M Σ|x⟩|a^x mod N⟩
    6. Apply inverse QFT to first register
    7. Measure → get approximation to s/r
    
    CLASSICAL POST-PROCESSING:
    8. Use continued fractions to find r
    9. Check if r is even and a^(r/2) ≢ ±1 (mod N)
    10. Compute gcd(a^(r/2) ± 1, N) → factors!
    
    REPEAT: If unsuccessful, go to step 3 with new a
    
    SUCCESS PROBABILITY: ≥ 1 - 1/2^k after k attempts
    """)
```

## Summary
