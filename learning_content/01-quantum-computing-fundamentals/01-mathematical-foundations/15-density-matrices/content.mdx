## Learning Objectives

After completing this lesson, you will be able to:

- Distinguish pure states from mixed states
- Construct density matrices from state ensembles
- Calculate expectation values using Tr(ρA)
- Determine purity and von Neumann entropy
- Work with reduced density matrices from partial traces

## 1. Why Density Matrices?

State vectors |ψ⟩ can't describe all quantum situations.

<Definition term="Mixed State">
A mixed state is a statistical ensemble of pure states. We have state $|\psi_i\rangle$ with probability $p_i$, but we don't know which one. This is different from superposition!
</Definition>

**When do mixed states arise?**
- Classical uncertainty about which state we have
- After measurement of an entangled system
- Decoherence due to environment
- Thermal equilibrium states

## 2. The Density Matrix

<Definition term="Density Matrix">
The density matrix (or density operator) is:
$$\rho = \sum_i p_i |\psi_i\rangle\langle\psi_i|$$
where $\{|\psi_i\rangle\}$ are states with probabilities $\{p_i\}$ (with $\sum_i p_i = 1$).
</Definition>

**Properties of valid density matrices:**
1. **Hermitian**: $\rho = \rho^\dagger$
2. **Positive semidefinite**: $\langle\phi|\rho|\phi\rangle \geq 0$ for all $|\phi\rangle$
3. **Unit trace**: $\text{Tr}(\rho) = 1$

<Example>
```python
import numpy as np

def create_density_matrix(states: list, probabilities: list) -> np.ndarray:
    """Create density matrix from ensemble."""
    dim = states[0].shape[0]
    rho = np.zeros((dim, dim), dtype=complex)
    
    for state, prob in zip(states, probabilities):
        rho += prob * (state @ state.conj().T)
    
    return rho

# Pure state |0⟩
ket_0 = np.array([[1], [0]])
rho_pure = create_density_matrix([ket_0], [1.0])
print("Pure state |0⟩:")
print(rho_pure)

# Mixed state: 50% |0⟩, 50% |1⟩
ket_1 = np.array([[0], [1]])
rho_mixed = create_density_matrix([ket_0, ket_1], [0.5, 0.5])
print("\nMixed state (50% |0⟩ + 50% |1⟩):")
print(rho_mixed)

# Verify properties
print(f"\nTrace: {np.trace(rho_mixed)}")
print(f"Hermitian: {np.allclose(rho_mixed, rho_mixed.conj().T)}")
eigenvalues = np.linalg.eigvalsh(rho_mixed)
print(f"Eigenvalues (≥0): {eigenvalues}")
```
</Example>

## 3. Pure States vs Mixed States

<Definition term="Pure State">
A pure state has density matrix $\rho = |\psi\rangle\langle\psi|$. Equivalently, $\rho^2 = \rho$ (projector property).
</Definition>

<Formula>
**Purity**: $\gamma = \text{Tr}(\rho^2)$
- Pure state: $\gamma = 1$
- Mixed state: $\gamma < 1$
- Maximally mixed: $\gamma = 1/d$ (d = dimension)
</Formula>

<Example>
```python
import numpy as np

def purity(rho: np.ndarray) -> float:
    """Calculate purity Tr(ρ²)."""
    return np.real(np.trace(rho @ rho))

def is_pure(rho: np.ndarray, tol: float = 1e-10) -> bool:
    """Check if state is pure."""
    return np.isclose(purity(rho), 1, atol=tol)

# Pure state |+⟩
ket_plus = np.array([[1], [1]]) / np.sqrt(2)
rho_plus = ket_plus @ ket_plus.conj().T

# Maximally mixed state I/2
rho_max_mixed = np.eye(2) / 2

# Partially mixed
ket_0 = np.array([[1], [0]])
rho_partial = 0.7 * (ket_0 @ ket_0.conj().T) + 0.3 * rho_max_mixed

print("State analysis:")
for name, rho in [('|+⟩ (pure)', rho_plus), 
                   ('I/2 (max mixed)', rho_max_mixed),
                   ('70% |0⟩ + 30% mixed', rho_partial)]:
    p = purity(rho)
    print(f"\n{name}:")
    print(f"  Purity: {p:.4f}")
    print(f"  Is pure: {is_pure(rho)}")
```
</Example>

<Warning>
Don't confuse mixed states with superpositions! 
- $|\psi\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)$ is a PURE state (superposition)
- $\rho = \frac{1}{2}|0\rangle\langle 0| + \frac{1}{2}|1\rangle\langle 1|$ is a MIXED state (classical uncertainty)
</Warning>

## 4. Expectation Values

Expectation values use the trace formula.

<Formula>
For observable $A$ and state $\rho$:
$$\langle A \rangle = \text{Tr}(\rho A) = \text{Tr}(A\rho)$$
</Formula>

<Example>
```python
import numpy as np

def expectation_value(rho: np.ndarray, observable: np.ndarray) -> float:
    """Calculate ⟨A⟩ = Tr(ρA)."""
    return np.real(np.trace(rho @ observable))

# Pauli matrices
X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]], dtype=complex)

# Pure state |0⟩
ket_0 = np.array([[1], [0]], dtype=complex)
rho_0 = ket_0 @ ket_0.conj().T

# Mixed state
rho_mixed = np.eye(2, dtype=complex) / 2

print("Expectation values for |0⟩:")
for name, obs in [('X', X), ('Y', Y), ('Z', Z)]:
    ev = expectation_value(rho_0, obs)
    print(f"  ⟨{name}⟩ = {ev:.4f}")

print("\nExpectation values for maximally mixed:")
for name, obs in [('X', X), ('Y', Y), ('Z', Z)]:
    ev = expectation_value(rho_mixed, obs)
    print(f"  ⟨{name}⟩ = {ev:.4f}")
```
</Example>

## 5. Von Neumann Entropy

Entropy measures the "mixedness" of a state.

<Definition term="Von Neumann Entropy">
The von Neumann entropy is:
$$S(\rho) = -\text{Tr}(\rho \log_2 \rho) = -\sum_i \lambda_i \log_2 \lambda_i$$
where $\lambda_i$ are the eigenvalues of $\rho$.
</Definition>

**Properties:**
- Pure state: $S = 0$
- Maximally mixed: $S = \log_2 d$ (maximum)
- Always non-negative: $S \geq 0$

<Example>
```python
import numpy as np

def von_neumann_entropy(rho: np.ndarray) -> float:
    """Calculate S(ρ) = -Tr(ρ log₂ ρ)."""
    eigenvalues = np.linalg.eigvalsh(rho)
    # Filter out zero eigenvalues (0 log 0 = 0)
    eigenvalues = eigenvalues[eigenvalues > 1e-15]
    return -np.sum(eigenvalues * np.log2(eigenvalues))

# Various states
ket_0 = np.array([[1], [0]], dtype=complex)
rho_pure = ket_0 @ ket_0.conj().T
rho_max_mixed = np.eye(2, dtype=complex) / 2

# Partially mixed
for p in [0, 0.1, 0.3, 0.5]:
    rho = (1-p) * rho_pure + p * (np.eye(2) / 2)
    S = von_neumann_entropy(rho)
    gamma = purity(rho)
    print(f"p={p}: S={S:.4f}, purity={gamma:.4f}")

print(f"\nMaximum entropy (d=2): {np.log2(2):.4f}")
```
</Example>

## 6. Bloch Sphere Representation

Single-qubit density matrices have a nice geometric representation.

<Formula>
Any single-qubit density matrix can be written:
$$\rho = \frac{1}{2}(I + \vec{r} \cdot \vec{\sigma}) = \frac{1}{2}(I + r_x X + r_y Y + r_z Z)$$
where $\vec{r} = (r_x, r_y, r_z)$ is the Bloch vector with $|\vec{r}| \leq 1$.
</Formula>

**Properties:**
- Pure states: $|\vec{r}| = 1$ (on the surface)
- Mixed states: $|\vec{r}| < 1$ (inside the sphere)
- Maximally mixed: $\vec{r} = 0$ (center)

<Example>
```python
import numpy as np

def bloch_vector(rho: np.ndarray) -> np.ndarray:
    """Extract Bloch vector from density matrix."""
    X = np.array([[0, 1], [1, 0]], dtype=complex)
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]], dtype=complex)
    
    rx = np.real(np.trace(rho @ X))
    ry = np.real(np.trace(rho @ Y))
    rz = np.real(np.trace(rho @ Z))
    
    return np.array([rx, ry, rz])

def density_from_bloch(r: np.ndarray) -> np.ndarray:
    """Create density matrix from Bloch vector."""
    X = np.array([[0, 1], [1, 0]], dtype=complex)
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]], dtype=complex)
    I = np.eye(2, dtype=complex)
    
    return 0.5 * (I + r[0]*X + r[1]*Y + r[2]*Z)

# Test various states
states = {
    '|0⟩': np.array([[1, 0], [0, 0]], dtype=complex),
    '|1⟩': np.array([[0, 0], [0, 1]], dtype=complex),
    '|+⟩': np.array([[0.5, 0.5], [0.5, 0.5]], dtype=complex),
    'mixed': np.eye(2, dtype=complex) / 2
}

print("Bloch vectors:")
for name, rho in states.items():
    r = bloch_vector(rho)
    length = np.linalg.norm(r)
    print(f"  {name}: r={np.round(r, 4)}, |r|={length:.4f}")
```
</Example>

## 7. Reduced Density Matrices

For composite systems, we can "trace out" one subsystem.

<Definition term="Reduced Density Matrix">
For a bipartite system with density matrix $\rho_{AB}$, the reduced density matrix of A is:
$$\rho_A = \text{Tr}_B(\rho_{AB})$$
This describes the state of A alone, ignoring B.
</Definition>

<Example>
```python
import numpy as np

def partial_trace_B(rho_AB: np.ndarray, dim_A: int, dim_B: int) -> np.ndarray:
    """Trace out system B."""
    reshaped = rho_AB.reshape(dim_A, dim_B, dim_A, dim_B)
    return np.trace(reshaped, axis1=1, axis2=3)

# Product state |00⟩
ket_00 = np.array([[1], [0], [0], [0]], dtype=complex)
rho_00 = ket_00 @ ket_00.conj().T

print("Product state |00⟩:")
rho_A = partial_trace_B(rho_00, 2, 2)
print(f"ρ_A =\n{rho_A}")
print(f"Purity: {purity(rho_A):.4f}")

# Bell state (|00⟩ + |11⟩)/√2
ket_11 = np.array([[0], [0], [0], [1]], dtype=complex)
bell = (ket_00 + ket_11) / np.sqrt(2)
rho_bell = bell @ bell.conj().T

print("\nBell state (|00⟩ + |11⟩)/√2:")
rho_A_bell = partial_trace_B(rho_bell, 2, 2)
print(f"ρ_A =\n{rho_A_bell}")
print(f"Purity: {purity(rho_A_bell):.4f}")
print(f"Entropy: {von_neumann_entropy(rho_A_bell):.4f}")
print("(Maximally entangled → maximally mixed reduced state)")
```
</Example>

<Tip>
The reduced density matrix of an entangled state is always mixed! This is a signature of quantum entanglement - you cannot describe the parts independently.
</Tip>

## 8. Density Matrix Evolution

Density matrices evolve under unitary operations.

<Formula>
Under unitary $U$:
$$\rho \mapsto U\rho U^\dagger$$
</Formula>

<Example>
```python
import numpy as np

def evolve_density_matrix(rho: np.ndarray, U: np.ndarray) -> np.ndarray:
    """Apply unitary evolution ρ → UρU†."""
    return U @ rho @ U.conj().T

# Hadamard gate
H = np.array([[1, 1], [1, -1]], dtype=complex) / np.sqrt(2)

# Start with |0⟩
ket_0 = np.array([[1], [0]], dtype=complex)
rho_0 = ket_0 @ ket_0.conj().T

print("Initial state |0⟩:")
print(rho_0)

rho_after_H = evolve_density_matrix(rho_0, H)
print("\nAfter Hadamard:")
print(np.round(rho_after_H, 4))

# Verify it's |+⟩⟨+|
ket_plus = np.array([[1], [1]], dtype=complex) / np.sqrt(2)
rho_plus = ket_plus @ ket_plus.conj().T
print(f"\nMatches |+⟩⟨+|: {np.allclose(rho_after_H, rho_plus)}")
```
</Example>

## 9. Summary: Pure vs Mixed States

| Property | Pure State | Mixed State |
|----------|------------|-------------|
| Description | Single $\|\psi\rangle$ | Ensemble $\{p_i, \|\psi_i\rangle\}$ |
| Density matrix | $\rho = \|\psi\rangle\langle\psi\|$ | $\rho = \sum_i p_i \|\psi_i\rangle\langle\psi_i\|$ |
| Purity Tr(ρ²) | 1 | < 1 |
| Entropy S(ρ) | 0 | > 0 |
| Bloch vector | $\|\vec{r}\| = 1$ | $\|\vec{r}\| < 1$ |
| $\rho^2 = \rho$ | Yes | No |

## Summary

In this lesson, you learned about density matrices:

1. **Density matrices** $\rho = \sum_i p_i |\psi_i\rangle\langle\psi_i|$ describe general quantum states
2. **Properties**: Hermitian, positive semidefinite, unit trace
3. **Purity** $\text{Tr}(\rho^2)$ distinguishes pure ($=1$) from mixed ($<1$)
4. **Expectation values**: $\langle A\rangle = \text{Tr}(\rho A)$
5. **Von Neumann entropy** measures mixedness
6. **Bloch sphere** visualizes single-qubit states
7. **Reduced density matrices** from partial trace describe subsystems

<Tip>
Density matrices are essential for quantum computing because real quantum computers experience decoherence, which creates mixed states!
</Tip>
