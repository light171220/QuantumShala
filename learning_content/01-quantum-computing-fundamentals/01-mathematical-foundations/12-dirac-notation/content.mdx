## Learning Objectives

After completing this lesson, you will be able to:

- Write quantum states using ket notation |ψ⟩
- Represent dual vectors using bra notation ⟨φ|
- Calculate inner products as bra-kets ⟨φ|ψ⟩
- Construct outer products and projectors |ψ⟩⟨φ|
- Express operators and measurements in Dirac notation

## 1. Introduction to Dirac Notation

Dirac notation (also called bra-ket notation) was invented by physicist Paul Dirac. It's the standard language of quantum mechanics and quantum computing.

<Definition term="Dirac Notation">
Dirac notation provides a concise way to represent quantum states, their duals, and operations on them. The notation gets its name from "bra-ket" - the combination of ⟨ | (bra) and | ⟩ (ket) forming a "bracket" ⟨|⟩.
</Definition>

Why Dirac notation?
- **Concise**: $|0\rangle$ is cleaner than $\begin{pmatrix}1\\0\end{pmatrix}$
- **Intuitive**: The notation naturally expresses quantum operations
- **Basis-independent**: Can represent abstract states without choosing a basis
- **Universal**: Used in all quantum computing papers and textbooks

## 2. Ket Notation: |ψ⟩

A **ket** represents a quantum state vector.

<Definition term="Ket">
A ket $|\psi\rangle$ represents a column vector in a Hilbert space. The label inside (ψ, 0, 1, +, etc.) identifies the state.
</Definition>

<Example>
Standard basis states for a qubit:

$$|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}, \quad |1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$$

Superposition states:
$$|+\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}, \quad |-\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ -1 \end{pmatrix}$$

General qubit state:
$$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle = \begin{pmatrix} \alpha \\ \beta \end{pmatrix}$$

```python
import numpy as np

# Define kets
ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])
ket_plus = np.array([[1], [1]]) / np.sqrt(2)
ket_minus = np.array([[1], [-1]]) / np.sqrt(2)

print("|0⟩ =", ket_0.flatten())
print("|1⟩ =", ket_1.flatten())
print("|+⟩ =", ket_plus.flatten())
print("|−⟩ =", ket_minus.flatten())

# General state |ψ⟩ = (3|0⟩ + 4i|1⟩)/5
alpha, beta = 3/5, 4j/5
psi = alpha * ket_0 + beta * ket_1
print(f"|ψ⟩ = {psi.flatten()}")
```
</Example>

## 3. Bra Notation: ⟨φ|

A **bra** is the conjugate transpose of a ket.

<Definition term="Bra">
A bra $\langle\phi|$ is the conjugate transpose (Hermitian adjoint) of the corresponding ket:
$$\langle\phi| = (|\phi\rangle)^\dagger$$
If $|\phi\rangle$ is a column vector, $\langle\phi|$ is a row vector with complex conjugated entries.
</Definition>

<Formula>
If $|\phi\rangle = \begin{pmatrix} a \\ b \end{pmatrix}$, then $\langle\phi| = \begin{pmatrix} a^* & b^* \end{pmatrix}$
</Formula>

<Example>
Converting kets to bras:

$$|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix} \Rightarrow \langle 0| = \begin{pmatrix} 1 & 0 \end{pmatrix}$$

$$|\psi\rangle = \begin{pmatrix} \frac{1}{\sqrt{2}} \\ \frac{i}{\sqrt{2}} \end{pmatrix} \Rightarrow \langle\psi| = \begin{pmatrix} \frac{1}{\sqrt{2}} & \frac{-i}{\sqrt{2}} \end{pmatrix}$$

```python
import numpy as np

def ket_to_bra(ket):
    """Convert a ket to its corresponding bra."""
    return ket.conj().T

ket_0 = np.array([[1], [0]])
ket_psi = np.array([[1], [1j]]) / np.sqrt(2)

bra_0 = ket_to_bra(ket_0)
bra_psi = ket_to_bra(ket_psi)

print("|0⟩ =", ket_0.flatten())
print("⟨0| =", bra_0.flatten())
print()
print("|ψ⟩ =", ket_psi.flatten())
print("⟨ψ| =", bra_psi.flatten())
```
</Example>

<Tip>
The names "bra" and "ket" come from "bracket" - when you put a bra and ket together, you get a bracket: ⟨bra|ket⟩ = ⟨φ|ψ⟩.
</Tip>

## 4. Inner Products: ⟨φ|ψ⟩

The bra-ket combination gives the inner product.

<Definition term="Bra-Ket Inner Product">
The inner product of two states is written as a bra-ket:
$$\langle\phi|\psi\rangle = \sum_i \phi_i^* \psi_i$$
This is a complex number representing the "overlap" between the states.
</Definition>

<Example>
Computing inner products:

$$\langle 0|0\rangle = \begin{pmatrix} 1 & 0 \end{pmatrix} \begin{pmatrix} 1 \\ 0 \end{pmatrix} = 1$$

$$\langle 0|1\rangle = \begin{pmatrix} 1 & 0 \end{pmatrix} \begin{pmatrix} 0 \\ 1 \end{pmatrix} = 0$$

$$\langle +|0\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \end{pmatrix} \begin{pmatrix} 1 \\ 0 \end{pmatrix} = \frac{1}{\sqrt{2}}$$

```python
import numpy as np

def braket(bra_state, ket_state):
    """Compute ⟨bra|ket⟩ inner product."""
    bra = bra_state.conj().T if bra_state.shape[0] > 1 else bra_state
    return (bra @ ket_state)[0, 0]

ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])
ket_plus = np.array([[1], [1]]) / np.sqrt(2)

print(f"⟨0|0⟩ = {braket(ket_0, ket_0)}")
print(f"⟨0|1⟩ = {braket(ket_0, ket_1)}")
print(f"⟨1|1⟩ = {braket(ket_1, ket_1)}")
print(f"⟨+|0⟩ = {braket(ket_plus, ket_0):.4f}")
print(f"⟨+|1⟩ = {braket(ket_plus, ket_1):.4f}")
```
</Example>

Key properties:
- $\langle\psi|\psi\rangle = 1$ for normalized states
- $\langle\phi|\psi\rangle = 0$ means orthogonal states
- $|\langle\phi|\psi\rangle|^2$ = probability of measuring $|\psi\rangle$ to be $|\phi\rangle$

## 5. Outer Products: |ψ⟩⟨φ|

Reversing the order gives an outer product - a matrix!

<Definition term="Outer Product">
The outer product $|\psi\rangle\langle\phi|$ is a matrix formed by multiplying a column vector by a row vector:
$$|\psi\rangle\langle\phi| = \begin{pmatrix} \psi_1 \\ \psi_2 \\ \vdots \end{pmatrix} \begin{pmatrix} \phi_1^* & \phi_2^* & \cdots \end{pmatrix}$$
</Definition>

<Example>
$$|0\rangle\langle 0| = \begin{pmatrix} 1 \\ 0 \end{pmatrix} \begin{pmatrix} 1 & 0 \end{pmatrix} = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}$$

$$|1\rangle\langle 1| = \begin{pmatrix} 0 \\ 1 \end{pmatrix} \begin{pmatrix} 0 & 1 \end{pmatrix} = \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}$$

$$|0\rangle\langle 1| = \begin{pmatrix} 1 \\ 0 \end{pmatrix} \begin{pmatrix} 0 & 1 \end{pmatrix} = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$$

```python
import numpy as np

def outer_product(ket1, ket2):
    """Compute |ket1⟩⟨ket2| outer product."""
    return ket1 @ ket2.conj().T

ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])

print("|0⟩⟨0| =")
print(outer_product(ket_0, ket_0))

print("\n|1⟩⟨1| =")
print(outer_product(ket_1, ket_1))

print("\n|0⟩⟨1| =")
print(outer_product(ket_0, ket_1))

print("\n|1⟩⟨0| =")
print(outer_product(ket_1, ket_0))
```
</Example>

## 6. Projectors

Outer products of a state with itself create projection operators.

<Definition term="Projector">
The projector onto state $|\psi\rangle$ is:
$$P_\psi = |\psi\rangle\langle\psi|$$
It projects any state onto the direction of $|\psi\rangle$.
</Definition>

Properties of projectors:
- $P^2 = P$ (idempotent)
- $P^\dagger = P$ (Hermitian)
- $P|\psi\rangle = |\psi\rangle$ (fixed point)
- $\text{Tr}(P) = 1$ for rank-1 projectors

<Example>
Projection onto $|+\rangle$:

$$P_+ = |+\rangle\langle +| = \frac{1}{2}\begin{pmatrix} 1 & 1 \\ 1 & 1 \end{pmatrix}$$

Applying to $|0\rangle$:
$$P_+|0\rangle = |+\rangle\langle +|0\rangle = |+\rangle \cdot \frac{1}{\sqrt{2}} = \frac{1}{\sqrt{2}}|+\rangle$$

```python
import numpy as np

ket_plus = np.array([[1], [1]]) / np.sqrt(2)
ket_0 = np.array([[1], [0]])

# Projector onto |+⟩
P_plus = outer_product(ket_plus, ket_plus)
print("P_+ = |+⟩⟨+| =")
print(P_plus)

# Project |0⟩
result = P_plus @ ket_0
print(f"\nP_+|0⟩ = {result.flatten()}")
print(f"= (1/√2)|+⟩ since |+⟩ = {ket_plus.flatten()}")

# Verify P² = P
print(f"\nP² = P? {np.allclose(P_plus @ P_plus, P_plus)}")
```
</Example>

## 7. Completeness Relations

Orthonormal bases satisfy a completeness relation.

<Definition term="Completeness Relation">
For an orthonormal basis $\{|e_i\rangle\}$:
$$\sum_i |e_i\rangle\langle e_i| = I$$
The sum of all projectors equals the identity operator.
</Definition>

<Example>
Computational basis completeness:
$$|0\rangle\langle 0| + |1\rangle\langle 1| = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix} + \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix} = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix} = I$$

Hadamard basis completeness:
$$|+\rangle\langle +| + |-\rangle\langle -| = I$$

```python
import numpy as np

ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])
ket_plus = np.array([[1], [1]]) / np.sqrt(2)
ket_minus = np.array([[1], [-1]]) / np.sqrt(2)

def outer_product(k1, k2):
    return k1 @ k2.conj().T

# Computational basis
P0 = outer_product(ket_0, ket_0)
P1 = outer_product(ket_1, ket_1)
print("Computational basis completeness:")
print("|0⟩⟨0| + |1⟩⟨1| =")
print(P0 + P1)

# Hadamard basis
Pplus = outer_product(ket_plus, ket_plus)
Pminus = outer_product(ket_minus, ket_minus)
print("\nHadamard basis completeness:")
print("|+⟩⟨+| + |−⟩⟨−| =")
print(np.round(Pplus + Pminus, 10))
```
</Example>

## 8. Operators in Dirac Notation

Quantum gates can be expressed using outer products.

<Example>
**Pauli X gate** (NOT gate):
$$X = |0\rangle\langle 1| + |1\rangle\langle 0| = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$$

**Pauli Z gate**:
$$Z = |0\rangle\langle 0| - |1\rangle\langle 1| = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$

**Hadamard gate**:
$$H = |+\rangle\langle 0| + |-\rangle\langle 1| = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$$

```python
import numpy as np

ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])
ket_plus = np.array([[1], [1]]) / np.sqrt(2)
ket_minus = np.array([[1], [-1]]) / np.sqrt(2)

def outer(k1, k2):
    return k1 @ k2.conj().T

# Pauli X
X = outer(ket_0, ket_1) + outer(ket_1, ket_0)
print("X = |0⟩⟨1| + |1⟩⟨0| =")
print(X)

# Pauli Z
Z = outer(ket_0, ket_0) - outer(ket_1, ket_1)
print("\nZ = |0⟩⟨0| - |1⟩⟨1| =")
print(Z)

# Hadamard
H = outer(ket_plus, ket_0) + outer(ket_minus, ket_1)
print("\nH = |+⟩⟨0| + |−⟩⟨1| =")
print(np.round(H, 4))
```
</Example>

## 9. Spectral Decomposition

Hermitian operators can be written in terms of their eigenvalues and eigenstates.

<Definition term="Spectral Decomposition">
A Hermitian operator $A$ with eigenvalues $\lambda_i$ and orthonormal eigenstates $|a_i\rangle$ can be written:
$$A = \sum_i \lambda_i |a_i\rangle\langle a_i|$$
</Definition>

<Example>
The Pauli Z operator has eigenvalues +1 and -1:
$$Z = (+1)|0\rangle\langle 0| + (-1)|1\rangle\langle 1| = |0\rangle\langle 0| - |1\rangle\langle 1|$$

```python
import numpy as np

# Z eigenvalues and eigenvectors
eigenvalues = [1, -1]
eigenvectors = [np.array([[1], [0]]), np.array([[0], [1]])]

# Reconstruct Z from spectral decomposition
Z_reconstructed = np.zeros((2, 2))
for val, vec in zip(eigenvalues, eigenvectors):
    Z_reconstructed += val * (vec @ vec.conj().T)

print("Z from spectral decomposition:")
print(Z_reconstructed)

# Verify
Z_standard = np.array([[1, 0], [0, -1]])
print(f"\nMatches standard Z? {np.allclose(Z_reconstructed, Z_standard)}")
```
</Example>

## 10. Measurement in Dirac Notation

Quantum measurement is elegantly expressed using projectors.

<Definition term="Projective Measurement">
Measuring observable $A = \sum_i \lambda_i |a_i\rangle\langle a_i|$ on state $|\psi\rangle$:
- **Probability** of outcome $\lambda_i$: $P(i) = |\langle a_i|\psi\rangle|^2 = \langle\psi|a_i\rangle\langle a_i|\psi\rangle$
- **Post-measurement state**: $\frac{|a_i\rangle\langle a_i|\psi\rangle}{\sqrt{P(i)}} = |a_i\rangle$
</Definition>

<Example>
Measuring $|+\rangle$ in the computational basis:

Probability of $|0\rangle$:
$$P(0) = |\langle 0|+\rangle|^2 = \left|\frac{1}{\sqrt{2}}\right|^2 = \frac{1}{2}$$

Post-measurement state if we get $|0\rangle$:
$$\frac{|0\rangle\langle 0|+\rangle}{\sqrt{1/2}} = \frac{|0\rangle \cdot \frac{1}{\sqrt{2}}}{\frac{1}{\sqrt{2}}} = |0\rangle$$

```python
import numpy as np

ket_plus = np.array([[1], [1]]) / np.sqrt(2)
ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])

# Measurement probabilities
P0 = np.abs(np.vdot(ket_0, ket_plus))**2
P1 = np.abs(np.vdot(ket_1, ket_plus))**2

print("Measuring |+⟩ in computational basis:")
print(f"P(|0⟩) = {P0}")
print(f"P(|1⟩) = {P1}")

# Post-measurement state calculation
projection_0 = (ket_0 @ ket_0.conj().T) @ ket_plus
post_state_0 = projection_0 / np.linalg.norm(projection_0)
print(f"\nPost-measurement state if |0⟩: {post_state_0.flatten()}")
```
</Example>

## 11. Multi-Qubit Notation

Dirac notation extends naturally to multiple qubits.

<Formula>
$$|01\rangle = |0\rangle \otimes |1\rangle = |0\rangle|1\rangle$$

$$\langle 01| = \langle 0| \otimes \langle 1| = \langle 0|\langle 1|$$

$$\langle 01|10\rangle = \langle 0|1\rangle\langle 1|0\rangle = 0 \cdot 0 = 0$$
</Formula>

<Example>
The Bell state:
$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

Probability of measuring $|00\rangle$:
$$P(00) = |\langle 00|\Phi^+\rangle|^2 = \left|\frac{1}{\sqrt{2}}\right|^2 = \frac{1}{2}$$

```python
import numpy as np

# Two-qubit basis states
ket_00 = np.array([[1], [0], [0], [0]])
ket_01 = np.array([[0], [1], [0], [0]])
ket_10 = np.array([[0], [0], [1], [0]])
ket_11 = np.array([[0], [0], [0], [1]])

# Bell state
bell = (ket_00 + ket_11) / np.sqrt(2)

print("Bell state |Φ+⟩ =", bell.flatten())

# Measurement probabilities
for label, state in [("00", ket_00), ("01", ket_01), 
                      ("10", ket_10), ("11", ket_11)]:
    prob = np.abs(np.vdot(state, bell))**2
    print(f"P({label}) = {prob}")
```
</Example>

## Summary

In this lesson, you mastered Dirac notation:

| Notation | Name | Type | Example |
|----------|------|------|---------|
| $\|ψ\rangle$ | Ket | Column vector | $\|0\rangle = (1,0)^T$ |
| $\langle φ\|$ | Bra | Row vector | $\langle 0\| = (1,0)$ |
| $\langle φ\|ψ\rangle$ | Bra-ket | Scalar (inner product) | $\langle 0\|1\rangle = 0$ |
| $\|ψ\rangle\langle φ\|$ | Ket-bra | Matrix (outer product) | $\|0\rangle\langle 0\|$ = projector |

Key concepts:
1. **Kets** represent quantum states as column vectors
2. **Bras** are conjugate transpose row vectors
3. **Inner products** give overlap and probability amplitudes
4. **Outer products** create operators and projectors
5. **Completeness**: $\sum_i |e_i\rangle\langle e_i| = I$
6. **Spectral decomposition**: $A = \sum_i \lambda_i |a_i\rangle\langle a_i|$

<Tip>
Read ⟨φ|A|ψ⟩ from right to left: "apply A to ψ, then take inner product with φ."
</Tip>
