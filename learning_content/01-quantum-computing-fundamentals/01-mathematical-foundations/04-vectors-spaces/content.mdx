## 1. What is a Vector?

The concept of a "vector" has multiple interpretations, each valuable for different purposes:

### Geometric View: Arrows in Space

Intuitively, a vector is an arrow with magnitude (length) and direction. In 2D or 3D space, we can visualize vectors as arrows pointing from the origin to a point.

### Algebraic View: Ordered Lists of Numbers

More abstractly, a vector is an ordered list of numbers:

<Formula>
$$\vec{v} = \begin{pmatrix} v_1 \\ v_2 \\ \vdots \\ v_n \end{pmatrix}$$
</Formula>

### Quantum View: States in Hilbert Space

In quantum computing, vectors represent **quantum states**. A qubit state is a vector in a 2-dimensional complex vector space:

<Formula>
$$|\psi\rangle = \begin{pmatrix} \alpha \\ \beta \end{pmatrix} = \alpha|0\rangle + \beta|1\rangle$$
</Formula>

where $\alpha, \beta \in \mathbb{C}$ and $|\alpha|^2 + |\beta|^2 = 1$.

<Definition term="Vector">
A vector is an element of a vector space - a mathematical structure that supports addition and scalar multiplication while satisfying certain axioms. In quantum computing, vectors represent quantum states.
</Definition>

<Tip>
Don't think of quantum state vectors as "arrows pointing somewhere in physical space." They're abstract mathematical objects living in Hilbert space - a complex vector space with additional structure. The geometric intuition helps, but the algebra is what matters for computation.
</Tip>

## 2. Vector Spaces: The Formal Definition

A **vector space** over a field $F$ (for us, $F = \mathbb{C}$, the complex numbers) is a set $V$ with two operations:

1. **Vector Addition**: $+: V \times V \rightarrow V$
2. **Scalar Multiplication**: $\cdot: F \times V \rightarrow V$

These operations must satisfy the following axioms:

### Addition Axioms

| Axiom | Statement | Meaning |
|-------|-----------|---------|
| Closure | $\vec{u} + \vec{v} \in V$ | Sum of vectors is a vector |
| Commutativity | $\vec{u} + \vec{v} = \vec{v} + \vec{u}$ | Order doesn't matter |
| Associativity | $(\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})$ | Grouping doesn't matter |
| Identity | $\exists \vec{0}: \vec{v} + \vec{0} = \vec{v}$ | Zero vector exists |
| Inverse | $\exists (-\vec{v}): \vec{v} + (-\vec{v}) = \vec{0}$ | Negatives exist |

### Scalar Multiplication Axioms

| Axiom | Statement | Meaning |
|-------|-----------|---------|
| Closure | $c\vec{v} \in V$ | Scaled vector is a vector |
| Identity | $1\vec{v} = \vec{v}$ | Multiplying by 1 does nothing |
| Compatibility | $(ab)\vec{v} = a(b\vec{v})$ | Scalars associate |
| Distributivity (scalar) | $a(\vec{u} + \vec{v}) = a\vec{u} + a\vec{v}$ | Distributes over vector sum |
| Distributivity (vector) | $(a + b)\vec{v} = a\vec{v} + b\vec{v}$ | Distributes over scalar sum |

<Warning>
These axioms might seem abstract, but they're precisely what makes quantum mechanics mathematically consistent. Every quantum operation must respect these rules, ensuring that quantum states remain valid vectors after any transformation.
</Warning>

## 3. The Vector Space ℂⁿ

The most important vector space for quantum computing is $\mathbb{C}^n$: the set of all $n$-tuples of complex numbers.

<Definition term="ℂⁿ">
$\mathbb{C}^n$ is the set of all column vectors with $n$ complex entries:
$$\mathbb{C}^n = \left\{ \begin{pmatrix} z_1 \\ z_2 \\ \vdots \\ z_n \end{pmatrix} : z_i \in \mathbb{C} \right\}$$
</Definition>

### Operations in ℂⁿ

**Vector Addition** (component-wise):
<Formula>
$$\begin{pmatrix} u_1 \\ u_2 \\ \vdots \\ u_n \end{pmatrix} + \begin{pmatrix} v_1 \\ v_2 \\ \vdots \\ v_n \end{pmatrix} = \begin{pmatrix} u_1 + v_1 \\ u_2 + v_2 \\ \vdots \\ u_n + v_n \end{pmatrix}$$
</Formula>

**Scalar Multiplication**:
<Formula>
$$c \begin{pmatrix} v_1 \\ v_2 \\ \vdots \\ v_n \end{pmatrix} = \begin{pmatrix} cv_1 \\ cv_2 \\ \vdots \\ cv_n \end{pmatrix}$$
</Formula>

```python
import numpy as np

# Vectors in C²
u = np.array([[1 + 2j], [3 - 1j]])
v = np.array([[2 - 1j], [1 + 1j]])

# Vector addition
print("Vector Addition:")
print(f"u = \n{u}")
print(f"v = \n{v}")
print(f"u + v = \n{u + v}")

# Scalar multiplication
c = 2 + 1j
print(f"\nScalar Multiplication:")
print(f"c = {c}")
print(f"c * u = \n{c * u}")
```

## 4. Quantum States as Vectors

### Single Qubit: Vectors in ℂ²

A single qubit lives in $\mathbb{C}^2$. The **computational basis** consists of two special vectors:

<Formula>
$$|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}, \quad |1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$$
</Formula>

Any qubit state is a **linear combination** (superposition) of these basis vectors:

<Formula>
$$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle = \alpha\begin{pmatrix} 1 \\ 0 \end{pmatrix} + \beta\begin{pmatrix} 0 \\ 1 \end{pmatrix} = \begin{pmatrix} \alpha \\ \beta \end{pmatrix}$$
</Formula>

<Example>
The **plus state** $|+\rangle$ is an equal superposition:

$$|+\rangle = \frac{1}{\sqrt{2}}|0\rangle + \frac{1}{\sqrt{2}}|1\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}$$

```python
import numpy as np

# Basis states
ket_0 = np.array([[1], [0]], dtype=complex)
ket_1 = np.array([[0], [1]], dtype=complex)

# Plus state as linear combination
alpha = 1/np.sqrt(2)
beta = 1/np.sqrt(2)
ket_plus = alpha * ket_0 + beta * ket_1

print("|0⟩ =", ket_0.flatten())
print("|1⟩ =", ket_1.flatten())
print("|+⟩ = (1/√2)|0⟩ + (1/√2)|1⟩ =", ket_plus.flatten())

# Verify normalization
norm_squared = np.sum(np.abs(ket_plus)**2)
print(f"\n|α|² + |β|² = {norm_squared} (normalized)")
```
</Example>

### Multiple Qubits: Vectors in ℂ^(2ⁿ)

For $n$ qubits, the state space is $\mathbb{C}^{2^n}$:

| Qubits | Dimension | Basis States |
|--------|-----------|--------------|
| 1 | 2 | $\|0\rangle, \|1\rangle$ |
| 2 | 4 | $\|00\rangle, \|01\rangle, \|10\rangle, \|11\rangle$ |
| 3 | 8 | $\|000\rangle, \ldots, \|111\rangle$ |
| n | $2^n$ | $\|0\ldots0\rangle, \ldots, \|1\ldots1\rangle$ |

<Example>
A 2-qubit state lives in $\mathbb{C}^4$:

$$|\psi\rangle = \alpha_{00}|00\rangle + \alpha_{01}|01\rangle + \alpha_{10}|10\rangle + \alpha_{11}|11\rangle = \begin{pmatrix} \alpha_{00} \\ \alpha_{01} \\ \alpha_{10} \\ \alpha_{11} \end{pmatrix}$$

```python
import numpy as np

# 2-qubit basis states
ket_00 = np.array([[1], [0], [0], [0]], dtype=complex)
ket_01 = np.array([[0], [1], [0], [0]], dtype=complex)
ket_10 = np.array([[0], [0], [1], [0]], dtype=complex)
ket_11 = np.array([[0], [0], [0], [1]], dtype=complex)

# Bell state |Φ+⟩ = (|00⟩ + |11⟩)/√2
bell_phi_plus = (ket_00 + ket_11) / np.sqrt(2)

print("Bell state |Φ+⟩ = (|00⟩ + |11⟩)/√2:")
print(bell_phi_plus.flatten())
print(f"\nNormalized: {np.isclose(np.sum(np.abs(bell_phi_plus)**2), 1)}")
```
</Example>

<Tip>
The exponential growth of dimension ($2^n$) is both the power and the challenge of quantum computing. 50 qubits require $2^{50} \approx 10^{15}$ complex amplitudes - impossible to store classically, but naturally represented by a quantum system!
</Tip>

## 5. Linear Combinations and Span

### Linear Combinations

A **linear combination** of vectors $\vec{v}_1, \vec{v}_2, \ldots, \vec{v}_k$ is:

<Formula>
$$c_1\vec{v}_1 + c_2\vec{v}_2 + \cdots + c_k\vec{v}_k$$
</Formula>

where $c_1, c_2, \ldots, c_k$ are scalars.

<Definition term="Span">
The **span** of a set of vectors is the set of all possible linear combinations:
$$\text{span}\{\vec{v}_1, \ldots, \vec{v}_k\} = \{c_1\vec{v}_1 + \cdots + c_k\vec{v}_k : c_i \in F\}$$
</Definition>

### Quantum Interpretation

Every quantum state is a linear combination of basis states. The span of the computational basis is the entire state space:

$$\text{span}\{|0\rangle, |1\rangle\} = \mathbb{C}^2$$

This means any single-qubit state can be written as $\alpha|0\rangle + \beta|1\rangle$.

```python
import numpy as np

# Show that any state can be expressed as linear combination of basis
def decompose_state(state):
    """Decompose a qubit state into computational basis."""
    alpha = state[0, 0]
    beta = state[1, 0]
    return alpha, beta

# Random normalized state
theta = np.pi/3
phi = np.pi/4
psi = np.array([
    [np.cos(theta/2)],
    [np.exp(1j*phi) * np.sin(theta/2)]
], dtype=complex)

alpha, beta = decompose_state(psi)
print(f"State |ψ⟩ = \n{psi}")
print(f"\nDecomposition: |ψ⟩ = ({alpha:.4f})|0⟩ + ({beta:.4f})|1⟩")

# Verify reconstruction
ket_0 = np.array([[1], [0]], dtype=complex)
ket_1 = np.array([[0], [1]], dtype=complex)
reconstructed = alpha * ket_0 + beta * ket_1
print(f"\nReconstructed: {reconstructed.flatten()}")
print(f"Original:      {psi.flatten()}")
print(f"Match: {np.allclose(psi, reconstructed)}")
```

## 6. The Zero Vector and Vector Subtraction

### Zero Vector

Every vector space has a unique **zero vector** $\vec{0}$ such that:

<Formula>
$$\vec{v} + \vec{0} = \vec{v} \quad \text{for all } \vec{v}$$
</Formula>

In $\mathbb{C}^n$:
<Formula>
$$\vec{0} = \begin{pmatrix} 0 \\ 0 \\ \vdots \\ 0 \end{pmatrix}$$
</Formula>

<Warning>
The zero vector is NOT a valid quantum state! Quantum states must be normalized ($|\alpha|^2 + |\beta|^2 = 1$), so the zero vector with norm 0 doesn't represent any physical state. This is a subtle but important distinction between general vector spaces and the space of quantum states.
</Warning>

### Vector Subtraction

Subtraction is defined using the additive inverse:

<Formula>
$$\vec{u} - \vec{v} = \vec{u} + (-\vec{v}) = \vec{u} + (-1)\vec{v}$$
</Formula>

```python
import numpy as np

# Zero vector
zero_vec = np.zeros((2, 1), dtype=complex)
print(f"Zero vector: {zero_vec.flatten()}")

# Vector subtraction
u = np.array([[3 + 1j], [2 - 1j]])
v = np.array([[1 + 2j], [1 + 1j]])

diff = u - v
print(f"\nu = {u.flatten()}")
print(f"v = {v.flatten()}")
print(f"u - v = {diff.flatten()}")

# Verify: u - v = u + (-1)*v
diff_alt = u + (-1) * v
print(f"u + (-1)*v = {diff_alt.flatten()}")
print(f"Same result: {np.allclose(diff, diff_alt)}")
```

## 7. Important Subspaces

A **subspace** is a subset of a vector space that is itself a vector space (closed under addition and scalar multiplication).

### Examples in Quantum Computing

1. **Single-qubit states** ($\mathbb{C}^2$) form a subspace of 2-qubit states ($\mathbb{C}^4$) when tensored with a fixed state.

2. **Symmetric states** of multiple qubits form a subspace (e.g., Dicke states).

3. **Error-corrected code spaces** are subspaces protected from certain errors.

<Example>
The **even parity** subspace of 2 qubits:

$$V_{\text{even}} = \text{span}\{|00\rangle, |11\rangle\}$$

This is a 2-dimensional subspace of $\mathbb{C}^4$. The Bell state $|Φ^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$ lives in this subspace.

```python
import numpy as np

# Computational basis for 2 qubits
ket_00 = np.array([[1], [0], [0], [0]], dtype=complex)
ket_01 = np.array([[0], [1], [0], [0]], dtype=complex)
ket_10 = np.array([[0], [0], [1], [0]], dtype=complex)
ket_11 = np.array([[0], [0], [0], [1]], dtype=complex)

# Even parity subspace basis
even_basis = [ket_00, ket_11]
print("Even parity subspace basis:")
print(f"|00⟩ = {ket_00.flatten()}")
print(f"|11⟩ = {ket_11.flatten()}")

# Any state in this subspace
alpha, beta = 1/np.sqrt(2), 1/np.sqrt(2)
state_in_even = alpha * ket_00 + beta * ket_11
print(f"\nBell state |Φ+⟩ in even subspace: {state_in_even.flatten()}")

# Odd parity subspace
odd_basis = [ket_01, ket_10]
print("\nOdd parity subspace basis:")
print(f"|01⟩ = {ket_01.flatten()}")
print(f"|10⟩ = {ket_10.flatten()}")
```
</Example>

## 8. Column Vectors vs Row Vectors

In quantum computing, we distinguish between:

- **Ket** $|\psi\rangle$: Column vector (state)
- **Bra** $\langle\psi|$: Row vector (conjugate transpose of ket)

<Formula>
$$|\psi\rangle = \begin{pmatrix} \alpha \\ \beta \end{pmatrix}, \quad \langle\psi| = \begin{pmatrix} \alpha^* & \beta^* \end{pmatrix}$$
</Formula>

This distinction is crucial for inner products (next lesson) and quantum measurements.

```python
import numpy as np

# Ket (column vector)
ket_psi = np.array([[1/np.sqrt(2)], [1j/np.sqrt(2)]], dtype=complex)
print("Ket |ψ⟩ (column vector):")
print(ket_psi)

# Bra (row vector) - conjugate transpose
bra_psi = ket_psi.conj().T
print("\nBra ⟨ψ| (row vector = conjugate transpose):")
print(bra_psi)

# The transpose alone is NOT the bra!
wrong_bra = ket_psi.T
print("\nWRONG - transpose without conjugate:")
print(wrong_bra)
print("Notice the imaginary part should flip sign!")
```

## 9. Computational Implementation

### NumPy for Quantum Vectors

```python
import numpy as np

class QuantumState:
    """A simple class for single-qubit quantum states."""
    
    def __init__(self, amplitudes):
        """Initialize from a list or array of amplitudes."""
        self.state = np.array(amplitudes, dtype=complex).reshape(-1, 1)
        self._normalize()
    
    def _normalize(self):
        """Normalize the state vector."""
        norm = np.sqrt(np.sum(np.abs(self.state)**2))
        if norm > 0:
            self.state = self.state / norm
    
    @classmethod
    def zero(cls):
        """Return |0⟩ state."""
        return cls([1, 0])
    
    @classmethod
    def one(cls):
        """Return |1⟩ state."""
        return cls([0, 1])
    
    @classmethod
    def plus(cls):
        """Return |+⟩ state."""
        return cls([1, 1])
    
    @classmethod
    def minus(cls):
        """Return |−⟩ state."""
        return cls([1, -1])
    
    def probabilities(self):
        """Return measurement probabilities."""
        return np.abs(self.state.flatten())**2
    
    def __repr__(self):
        alpha, beta = self.state.flatten()
        return f"|ψ⟩ = ({alpha:.4f})|0⟩ + ({beta:.4f})|1⟩"
    
    def __add__(self, other):
        """Add two states (result needs renormalization)."""
        new_state = self.state + other.state
        return QuantumState(new_state.flatten())
    
    def __mul__(self, scalar):
        """Scalar multiplication."""
        new_state = scalar * self.state
        return QuantumState(new_state.flatten())
    
    __rmul__ = __mul__

# Usage examples
print("=== Quantum State Class Demo ===\n")

ket_0 = QuantumState.zero()
ket_1 = QuantumState.one()
ket_plus = QuantumState.plus()

print(f"|0⟩: {ket_0}")
print(f"|1⟩: {ket_1}")
print(f"|+⟩: {ket_plus}")

print(f"\n|+⟩ probabilities: {ket_plus.probabilities()}")
print("50% chance of |0⟩, 50% chance of |1⟩")
```

## Summary

In this lesson, you've built a solid foundation in vector spaces for quantum computing:

1. **Vectors** are abstract objects supporting addition and scalar multiplication
2. **Vector spaces** satisfy 10 axioms ensuring mathematical consistency
3. **ℂⁿ** is the space of n-dimensional complex vectors where quantum states live
4. **Quantum states** are normalized vectors: $|\psi\rangle$ with $\langle\psi|\psi\rangle = 1$
5. **Linear combinations** express any state in terms of basis states (superposition)
6. **Subspaces** are vector spaces within vector spaces (e.g., error-corrected code spaces)
7. **Bras and kets** distinguish row and column vectors via conjugate transpose

The vector space structure is what enables quantum superposition - the ability for quantum states to be "multiple things at once." This mathematical framework, combined with the rules we'll learn for combining and transforming vectors, forms the complete language of quantum computation.
