## 1. Review and Extension of Basic Operations

In the previous lesson, we introduced complex number operations. Now we'll deepen our understanding and explore their algebraic properties.

<Definition term="Field">
A field is a set of numbers where addition, subtraction, multiplication, and division (except by zero) are all defined and satisfy certain properties. Complex numbers form a field, denoted ℂ.
</Definition>

The complex numbers ℂ extend the real numbers ℝ, which extend the rationals ℚ, which extend the integers ℤ:

<Formula>
$$\mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R} \subset \mathbb{C}$$
</Formula>

This hierarchy matters in quantum computing because we need the full power of complex numbers to describe quantum phenomena—real numbers alone are insufficient!

## 2. Properties of Complex Addition

Complex addition inherits all the nice properties of real addition:

### Closure
The sum of two complex numbers is always a complex number.

<Formula>
$$(a + bi) + (c + di) = (a + c) + (b + d)i \in \mathbb{C}$$
</Formula>

### Commutativity
Order doesn't matter in addition.

<Formula>
$$z_1 + z_2 = z_2 + z_1$$
</Formula>

### Associativity
Grouping doesn't matter in addition.

<Formula>
$$(z_1 + z_2) + z_3 = z_1 + (z_2 + z_3)$$
</Formula>

### Identity Element
Zero is the additive identity.

<Formula>
$$z + 0 = z$$
</Formula>

### Inverse Element
Every complex number has an additive inverse.

<Formula>
$$z + (-z) = 0$$
</Formula>

```python
import numpy as np

# Demonstrating addition properties
z1 = 2 + 3j
z2 = 1 - 4j
z3 = -1 + 2j

# Commutativity
print(f"z1 + z2 = {z1 + z2}")
print(f"z2 + z1 = {z2 + z1}")
print(f"Commutative: {z1 + z2 == z2 + z1}")

# Associativity
print(f"\n(z1 + z2) + z3 = {(z1 + z2) + z3}")
print(f"z1 + (z2 + z3) = {z1 + (z2 + z3)}")
print(f"Associative: {(z1 + z2) + z3 == z1 + (z2 + z3)}")

# Additive inverse
print(f"\nz1 + (-z1) = {z1 + (-z1)}")
```

## 3. Properties of Complex Multiplication

Multiplication has similar algebraic properties:

### Closure
<Formula>
$$(a + bi)(c + di) = (ac - bd) + (ad + bc)i \in \mathbb{C}$$
</Formula>

### Commutativity
<Formula>
$$z_1 \cdot z_2 = z_2 \cdot z_1$$
</Formula>

### Associativity
<Formula>
$$(z_1 \cdot z_2) \cdot z_3 = z_1 \cdot (z_2 \cdot z_3)$$
</Formula>

### Identity Element
One is the multiplicative identity.
<Formula>
$$z \cdot 1 = z$$
</Formula>

### Inverse Element
Every non-zero complex number has a multiplicative inverse.
<Formula>
$$z \cdot z^{-1} = 1 \quad \text{where} \quad z^{-1} = \frac{z^*}{|z|^2}$$
</Formula>

<Example>
Finding the multiplicative inverse of $z = 3 + 4i$:

$$z^{-1} = \frac{z^*}{|z|^2} = \frac{3 - 4i}{25} = \frac{3}{25} - \frac{4}{25}i = 0.12 - 0.16i$$

Verification: $z \cdot z^{-1} = (3 + 4i)(0.12 - 0.16i) = 0.36 - 0.48i + 0.48i - 0.64i^2 = 0.36 + 0.64 = 1$ ✓

```python
z = 3 + 4j
z_inv = z.conjugate() / abs(z)**2
print(f"z = {z}")
print(f"z⁻¹ = {z_inv}")
print(f"z × z⁻¹ = {z * z_inv}")  # Should be 1
```
</Example>

### Distributivity
Multiplication distributes over addition.
<Formula>
$$z_1 \cdot (z_2 + z_3) = z_1 \cdot z_2 + z_1 \cdot z_3$$
</Formula>

<Tip>
The distributive property is crucial in quantum computing! When we apply a quantum gate to a superposition state, we distribute the operation across all basis states.
</Tip>

## 4. Advanced Multiplication Techniques

### Multiplying Multiple Complex Numbers

When multiplying several complex numbers, we can use associativity to group them conveniently:

<Example>
Calculate $(1 + i)(2 - i)(1 + 2i)$:

First, multiply the first two:
$(1 + i)(2 - i) = 2 - i + 2i - i^2 = 2 + i + 1 = 3 + i$

Then multiply by the third:
$(3 + i)(1 + 2i) = 3 + 6i + i + 2i^2 = 3 + 7i - 2 = 1 + 7i$

```python
z1 = 1 + 1j
z2 = 2 - 1j
z3 = 1 + 2j

result = z1 * z2 * z3
print(f"({z1})({z2})({z3}) = {result}")
```
</Example>

### Modulus of a Product

An extremely useful property:

<Formula>
$$|z_1 \cdot z_2| = |z_1| \cdot |z_2|$$
</Formula>

The modulus of a product equals the product of the moduli!

<Example>
For $z_1 = 3 + 4i$ and $z_2 = 5 + 12i$:
- $|z_1| = 5$, $|z_2| = 13$
- $|z_1 \cdot z_2| = |z_1| \cdot |z_2| = 5 \times 13 = 65$

Verify: $z_1 \cdot z_2 = (3 + 4i)(5 + 12i) = 15 + 36i + 20i + 48i^2 = 15 + 56i - 48 = -33 + 56i$
$|-33 + 56i| = \sqrt{33^2 + 56^2} = \sqrt{1089 + 3136} = \sqrt{4225} = 65$ ✓
</Example>

<Warning>
In quantum computing, this property ensures that applying unitary operations preserves the normalization of quantum states!
</Warning>

## 5. Complex Conjugate Properties

The complex conjugate has beautiful algebraic properties:

### Sum of Conjugates
<Formula>
$$(z_1 + z_2)^* = z_1^* + z_2^*$$
</Formula>

### Product of Conjugates
<Formula>
$$(z_1 \cdot z_2)^* = z_1^* \cdot z_2^*$$
</Formula>

### Conjugate of Quotient
<Formula>
$$\left(\frac{z_1}{z_2}\right)^* = \frac{z_1^*}{z_2^*}$$
</Formula>

### Conjugate of Conjugate
<Formula>
$$(z^*)^* = z$$
</Formula>

### Real and Imaginary Parts via Conjugates
<Formula>
$$\text{Re}(z) = \frac{z + z^*}{2}, \quad \text{Im}(z) = \frac{z - z^*}{2i}$$
</Formula>

```python
import numpy as np

z1 = 3 + 4j
z2 = 1 - 2j

# Verify (z1 * z2)* = z1* * z2*
product = z1 * z2
left_side = product.conjugate()
right_side = z1.conjugate() * z2.conjugate()
print(f"(z1 × z2)* = {left_side}")
print(f"z1* × z2* = {right_side}")
print(f"Equal: {left_side == right_side}")

# Extract real and imaginary parts using conjugates
z = 5 + 7j
real_part = (z + z.conjugate()) / 2
imag_part = (z - z.conjugate()) / (2j)
print(f"\nFor z = {z}:")
print(f"Re(z) via conjugate formula: {real_part}")
print(f"Im(z) via conjugate formula: {imag_part}")
```

## 6. Division in Depth

### The Division Algorithm

Division by a complex number $z_2 = c + di$ involves multiplying by its conjugate:

<Formula>
$$\frac{z_1}{z_2} = \frac{z_1 \cdot z_2^*}{z_2 \cdot z_2^*} = \frac{z_1 \cdot z_2^*}{|z_2|^2}$$
</Formula>

### Modulus of a Quotient

<Formula>
$$\left|\frac{z_1}{z_2}\right| = \frac{|z_1|}{|z_2|}$$
</Formula>

<Example>
Divide $\frac{2 + 5i}{1 - 3i}$:

Step 1: Multiply by conjugate
$$\frac{2 + 5i}{1 - 3i} \cdot \frac{1 + 3i}{1 + 3i} = \frac{(2 + 5i)(1 + 3i)}{(1 - 3i)(1 + 3i)}$$

Step 2: Expand numerator
$(2 + 5i)(1 + 3i) = 2 + 6i + 5i + 15i^2 = 2 + 11i - 15 = -13 + 11i$

Step 3: Expand denominator
$(1 - 3i)(1 + 3i) = 1 - 9i^2 = 1 + 9 = 10$

Step 4: Combine
$$\frac{-13 + 11i}{10} = -1.3 + 1.1i$$

```python
z1 = 2 + 5j
z2 = 1 - 3j
result = z1 / z2
print(f"({z1}) / ({z2}) = {result}")
```
</Example>

## 7. Solving Equations with Complex Numbers

### Linear Equations

Solve $z \cdot (2 + i) = 5 - 3i$ for $z$:

<Formula>
$$z = \frac{5 - 3i}{2 + i}$$
</Formula>

<Example>
$$z = \frac{5 - 3i}{2 + i} \cdot \frac{2 - i}{2 - i} = \frac{(5 - 3i)(2 - i)}{(2 + i)(2 - i)} = \frac{10 - 5i - 6i + 3i^2}{4 + 1}$$
$$= \frac{10 - 11i - 3}{5} = \frac{7 - 11i}{5} = 1.4 - 2.2i$$

```python
# Solve z * (2 + i) = 5 - 3i
rhs = 5 - 3j
coeff = 2 + 1j
z = rhs / coeff
print(f"z = {z}")

# Verify
print(f"z × (2 + i) = {z * coeff}")  # Should equal 5 - 3i
```
</Example>

### Quadratic Equations with Complex Coefficients

Solve $z^2 + 2iz - 3 = 0$:

Using the quadratic formula:
<Formula>
$$z = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
</Formula>

where $a = 1$, $b = 2i$, $c = -3$.

<Example>
$$b^2 - 4ac = (2i)^2 - 4(1)(-3) = -4 + 12 = 8$$
$$z = \frac{-2i \pm \sqrt{8}}{2} = \frac{-2i \pm 2\sqrt{2}}{2} = -i \pm \sqrt{2}$$

So $z_1 = \sqrt{2} - i$ and $z_2 = -\sqrt{2} - i$.

```python
import numpy as np

# Coefficients
a = 1
b = 2j
c = -3

# Discriminant
discriminant = b**2 - 4*a*c
print(f"Discriminant: {discriminant}")

# Solutions
z1 = (-b + np.sqrt(discriminant)) / (2*a)
z2 = (-b - np.sqrt(discriminant)) / (2*a)
print(f"z1 = {z1}")
print(f"z2 = {z2}")

# Verify
print(f"\nVerification:")
print(f"z1² + 2i·z1 - 3 = {z1**2 + 2j*z1 - 3}")
print(f"z2² + 2i·z2 - 3 = {z2**2 + 2j*z2 - 3}")
```
</Example>

## 8. Application: Quantum Gate Operations

In quantum computing, quantum gates are represented by matrices with complex entries. Let's see how complex operations apply:

<Example>
The Hadamard gate has matrix:
$$H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$$

Applying H to $|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$:

$$H|0\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}\begin{pmatrix} 1 \\ 0 \end{pmatrix} = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix} = |+\rangle$$

```python
import numpy as np

# Hadamard gate
H = (1/np.sqrt(2)) * np.array([[1, 1], [1, -1]], dtype=complex)

# |0⟩ state
ket_0 = np.array([[1], [0]], dtype=complex)

# Apply Hadamard
result = H @ ket_0
print("H|0⟩ =")
print(result)
print(f"\nAmplitudes: α = {result[0,0]:.4f}, β = {result[1,0]:.4f}")
print(f"Probabilities: P(0) = {abs(result[0,0])**2:.4f}, P(1) = {abs(result[1,0])**2:.4f}")
```
</Example>

<Tip>
The complex operations you're learning now are exactly the operations that quantum computers perform billions of times! Mastering them is essential for understanding quantum algorithms.
</Tip>

## 9. Common Identities and Shortcuts

Here are useful identities to memorize:

### Product Identities
<Formula>
$$z \cdot z^* = |z|^2$$
$$|z^n| = |z|^n$$
$$(z^n)^* = (z^*)^n$$
</Formula>

### Sum and Difference Patterns
<Formula>
$$|z_1 + z_2|^2 = |z_1|^2 + |z_2|^2 + 2\text{Re}(z_1 z_2^*)$$
$$|z_1 - z_2|^2 = |z_1|^2 + |z_2|^2 - 2\text{Re}(z_1 z_2^*)$$
</Formula>

### Triangle Inequality
<Formula>
$$|z_1 + z_2| \leq |z_1| + |z_2|$$
</Formula>

The "distance" interpretation: the direct path is never longer than the detour!
