## 1. Introduction: Why Linear Independence Matters

In quantum computing, we represent quantum states as vectors in a complex vector space. But not all sets of vectors are equally useful for describing states. The key concept that distinguishes useful sets is **linear independence**.

<Definition term="Linear Independence">
A set of vectors {v₁, v₂, ..., vₙ} is linearly independent if the only solution to:
$$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_n\mathbf{v}_n = \mathbf{0}$$
is the trivial solution $c_1 = c_2 = \cdots = c_n = 0$.
</Definition>

If non-trivial solutions exist (some $c_i \neq 0$), the vectors are **linearly dependent**.

<Tip>
Think of linear independence as "no redundancy" - none of the vectors can be written as a combination of the others. In quantum computing, we need linearly independent basis states to uniquely represent any quantum state.
</Tip>

## 2. Geometric Intuition

### In 2D Space

Two vectors are linearly independent if and only if they don't point in the same (or opposite) direction:

- **Independent**: $\mathbf{u} = [1, 0]^T$ and $\mathbf{v} = [0, 1]^T$ (perpendicular)
- **Independent**: $\mathbf{u} = [1, 0]^T$ and $\mathbf{v} = [1, 1]^T$ (different directions)
- **Dependent**: $\mathbf{u} = [1, 2]^T$ and $\mathbf{v} = [2, 4]^T$ (same direction, $\mathbf{v} = 2\mathbf{u}$)

### In 3D Space

Three vectors are linearly independent if they don't all lie in the same plane:

- **Independent**: The standard basis $\mathbf{e}_1, \mathbf{e}_2, \mathbf{e}_3$
- **Dependent**: Any three vectors in the xy-plane

```python
import numpy as np

# Example: Testing linear independence geometrically
u = np.array([1, 0])
v1 = np.array([0, 1])  # Independent with u
v2 = np.array([2, 0])  # Dependent with u (same direction)

print("u =", u)
print("v1 =", v1, "- Different direction from u")
print("v2 =", v2, "- Same direction as u (v2 = 2u)")
```

## 3. Testing Linear Independence

### Method 1: Definition-Based Test

Set up the equation $c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots = \mathbf{0}$ and solve for the coefficients.

<Example>
Test if $\mathbf{v}_1 = [1, 2]^T$ and $\mathbf{v}_2 = [3, 6]^T$ are linearly independent.

Set up: $c_1[1, 2]^T + c_2[3, 6]^T = [0, 0]^T$

This gives:
- $c_1 + 3c_2 = 0$
- $2c_1 + 6c_2 = 0$

From the first equation: $c_1 = -3c_2$

Substituting: $2(-3c_2) + 6c_2 = -6c_2 + 6c_2 = 0$ ✓

This works for ANY $c_2$! For example, $c_2 = 1$, $c_1 = -3$:
$$-3[1, 2]^T + 1[3, 6]^T = [-3, -6]^T + [3, 6]^T = [0, 0]^T$$

Since non-trivial solutions exist, the vectors are **linearly dependent**.
</Example>

### Method 2: Matrix Rank Test

Form a matrix with vectors as columns. Vectors are linearly independent if and only if the matrix has full column rank.

<Formula>
$$\text{rank}([\mathbf{v}_1 | \mathbf{v}_2 | \cdots | \mathbf{v}_n]) = n \Leftrightarrow \text{linearly independent}$$
</Formula>

```python
import numpy as np

def test_independence(vectors):
    """Test if a list of vectors is linearly independent."""
    # Stack vectors as columns
    matrix = np.column_stack(vectors)
    rank = np.linalg.matrix_rank(matrix)
    num_vectors = len(vectors)
    
    is_independent = (rank == num_vectors)
    
    return {
        'matrix': matrix,
        'rank': rank,
        'num_vectors': num_vectors,
        'independent': is_independent
    }

# Test cases
v1 = np.array([1, 2])
v2 = np.array([3, 6])  # Dependent (v2 = 3*v1)
v3 = np.array([1, 1])  # Independent of v1

print("Test 1: v1=[1,2], v2=[3,6]")
result = test_independence([v1, v2])
print(f"  Rank: {result['rank']}, Independent: {result['independent']}")

print("\nTest 2: v1=[1,2], v3=[1,1]")
result = test_independence([v1, v3])
print(f"  Rank: {result['rank']}, Independent: {result['independent']}")
```

### Method 3: Determinant Test (Square Matrices)

If you have exactly n vectors in ℝⁿ (or ℂⁿ), you can use the determinant:

<Formula>
$$\det([\mathbf{v}_1 | \mathbf{v}_2 | \cdots | \mathbf{v}_n]) \neq 0 \Leftrightarrow \text{linearly independent}$$
</Formula>

```python
import numpy as np

# Determinant test for 2 vectors in 2D
v1 = np.array([1, 0])
v2 = np.array([0, 1])

matrix = np.column_stack([v1, v2])
det = np.linalg.det(matrix)

print(f"Vectors: {v1}, {v2}")
print(f"Matrix:\n{matrix}")
print(f"Determinant: {det}")
print(f"Independent: {abs(det) > 1e-10}")
```

## 4. Span of Vectors

<Definition term="Span">
The span of a set of vectors {v₁, v₂, ..., vₙ} is the set of all possible linear combinations:
$$\text{span}\{\mathbf{v}_1, \ldots, \mathbf{v}_n\} = \{c_1\mathbf{v}_1 + \cdots + c_n\mathbf{v}_n : c_i \in \mathbb{C}\}$$
</Definition>

The span is always a subspace (as we saw in the previous lesson).

<Example>
In ℝ²:
- span{[1,0]ᵀ} = the x-axis (1D subspace)
- span{[1,0]ᵀ, [0,1]ᵀ} = all of ℝ² (2D, full space)
- span{[1,0]ᵀ, [2,0]ᵀ} = the x-axis (still 1D, redundant vector)

```python
import numpy as np
import matplotlib.pyplot as plt

# Visualize span in 2D
fig, axes = plt.subplots(1, 3, figsize=(15, 5))

# Span of single vector
ax = axes[0]
ax.quiver(0, 0, 1, 0, angles='xy', scale_units='xy', scale=1, color='blue')
t = np.linspace(-3, 3, 100)
ax.plot(t, np.zeros_like(t), 'b--', alpha=0.5, label='span{[1,0]}')
ax.set_xlim(-3, 3)
ax.set_ylim(-3, 3)
ax.set_title('span{[1,0]ᵀ} = x-axis')
ax.grid(True, alpha=0.3)
ax.set_aspect('equal')

# Span of two independent vectors
ax = axes[1]
ax.quiver(0, 0, 1, 0, angles='xy', scale_units='xy', scale=1, color='blue')
ax.quiver(0, 0, 0, 1, angles='xy', scale_units='xy', scale=1, color='red')
ax.fill([-3, 3, 3, -3], [-3, -3, 3, 3], alpha=0.2, color='green')
ax.set_xlim(-3, 3)
ax.set_ylim(-3, 3)
ax.set_title('span{[1,0]ᵀ, [0,1]ᵀ} = ℝ²')
ax.grid(True, alpha=0.3)
ax.set_aspect('equal')

# Span of dependent vectors
ax = axes[2]
ax.quiver(0, 0, 1, 0, angles='xy', scale_units='xy', scale=1, color='blue')
ax.quiver(0, 0, 2, 0, angles='xy', scale_units='xy', scale=1, color='red')
ax.plot(t, np.zeros_like(t), 'g--', alpha=0.5)
ax.set_xlim(-3, 3)
ax.set_ylim(-3, 3)
ax.set_title('span{[1,0]ᵀ, [2,0]ᵀ} = x-axis (redundant)')
ax.grid(True, alpha=0.3)
ax.set_aspect('equal')

plt.tight_layout()
plt.show()
```

## 5. Basis and Dimension

<Definition term="Basis">
A basis for a vector space V is a set of vectors that is:
1. **Linearly independent**: No redundancy
2. **Spanning**: Every vector in V can be written as a linear combination

A basis is a "minimal spanning set" or equivalently a "maximal independent set."
</Definition>

<Definition term="Dimension">
The dimension of a vector space is the number of vectors in any basis. All bases have the same number of vectors.
</Definition>

<Example>
For ℂ² (single qubit state space):

**Standard/Computational Basis:**
$$|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}, \quad |1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$$

**Hadamard Basis:**
$$|+\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}, \quad |-\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ -1 \end{pmatrix}$$

Both are valid bases for ℂ². The dimension is 2.
</Example>

<Warning>
Every finite-dimensional vector space has infinitely many different bases, but they all have the same number of vectors (the dimension).
</Warning>

## 6. Quantum Computing Applications

### Single Qubit: 2D Complex Space

The state space of a single qubit is ℂ², with dimension 2.

```python
import numpy as np

# Computational basis
ket_0 = np.array([[1], [0]], dtype=complex)
ket_1 = np.array([[0], [1]], dtype=complex)

# Hadamard basis
ket_plus = np.array([[1], [1]], dtype=complex) / np.sqrt(2)
ket_minus = np.array([[1], [-1]], dtype=complex) / np.sqrt(2)

# Verify both are bases (linearly independent)
def check_basis(v1, v2, name):
    matrix = np.hstack([v1, v2])
    det = np.linalg.det(matrix)
    print(f"{name}: det = {det:.4f}, is basis: {abs(det) > 1e-10}")

check_basis(ket_0, ket_1, "Computational basis {|0⟩, |1⟩}")
check_basis(ket_plus, ket_minus, "Hadamard basis {|+⟩, |-⟩}")
```

### Multiple Qubits: Exponential Growth

For n qubits, the state space is ℂ^(2ⁿ) with dimension 2ⁿ.

| Qubits | Dimension | Basis States |
|--------|-----------|--------------|
| 1 | 2 | \|0⟩, \|1⟩ |
| 2 | 4 | \|00⟩, \|01⟩, \|10⟩, \|11⟩ |
| 3 | 8 | \|000⟩, ..., \|111⟩ |
| n | 2ⁿ | All n-bit strings |

<Tip>
This exponential growth is both the power and the challenge of quantum computing! 50 qubits require 2⁵⁰ ≈ 10¹⁵ complex numbers to describe classically.
</Tip>

## 7. Properties of Linear Independence

### Key Theorems

1. **Any set containing the zero vector is linearly dependent**
   - Because $1 \cdot \mathbf{0} = \mathbf{0}$ (non-trivial combination)

2. **A single non-zero vector is linearly independent**
   - The only solution to $c\mathbf{v} = \mathbf{0}$ is $c = 0$

3. **More vectors than dimension ⟹ dependent**
   - In ℂⁿ, any set of n+1 or more vectors is dependent

4. **Subset of independent set is independent**
   - If {v₁, v₂, v₃} is independent, so is {v₁, v₂}

```python
import numpy as np

# Theorem 3: More vectors than dimension
# In 2D, any 3 vectors must be dependent

v1 = np.array([1, 0])
v2 = np.array([0, 1])
v3 = np.array([1, 1])  # Any third vector in 2D

matrix = np.column_stack([v1, v2, v3])
rank = np.linalg.matrix_rank(matrix)

print("Three vectors in 2D space:")
print(f"v1 = {v1}, v2 = {v2}, v3 = {v3}")
print(f"Rank of [v1|v2|v3] = {rank}")
print(f"Number of vectors = 3")
print(f"Independent: {rank == 3}")  # False - can't have 3 independent vectors in 2D
```

## 8. Change of Basis

Any vector can be expressed in different bases. Converting between bases is crucial in quantum computing.

<Formula>
If $|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$ in the computational basis, then in the Hadamard basis:
$$|\psi\rangle = \frac{\alpha + \beta}{\sqrt{2}}|+\rangle + \frac{\alpha - \beta}{\sqrt{2}}|-\rangle$$
</Formula>

<Example>
Express $|0\rangle$ in the Hadamard basis:

$|0\rangle = \alpha|0\rangle + \beta|1\rangle$ with $\alpha = 1, \beta = 0$

In Hadamard basis:
- Coefficient of $|+\rangle$: $\frac{1 + 0}{\sqrt{2}} = \frac{1}{\sqrt{2}}$
- Coefficient of $|-\rangle$: $\frac{1 - 0}{\sqrt{2}} = \frac{1}{\sqrt{2}}$

So $|0\rangle = \frac{1}{\sqrt{2}}|+\rangle + \frac{1}{\sqrt{2}}|-\rangle$

```python
import numpy as np

# Computational basis
ket_0 = np.array([[1], [0]], dtype=complex)
ket_1 = np.array([[0], [1]], dtype=complex)

# Hadamard basis
ket_plus = (ket_0 + ket_1) / np.sqrt(2)
ket_minus = (ket_0 - ket_1) / np.sqrt(2)

# Express |0⟩ in Hadamard basis
# |0⟩ = c₊|+⟩ + c₋|-⟩
# Solve: [|+⟩ | |-⟩] @ [c₊, c₋]ᵀ = |0⟩

H_matrix = np.hstack([ket_plus, ket_minus])
coeffs = np.linalg.solve(H_matrix, ket_0)

print("|0⟩ in Hadamard basis:")
print(f"c₊ = {coeffs[0,0]:.4f}")
print(f"c₋ = {coeffs[1,0]:.4f}")
print(f"|0⟩ = {coeffs[0,0]:.4f}|+⟩ + {coeffs[1,0]:.4f}|-⟩")

# Verify
reconstructed = coeffs[0,0] * ket_plus + coeffs[1,0] * ket_minus
print(f"\nVerification: {np.allclose(reconstructed, ket_0)}")
```
</Example>

## Summary

Linear independence is fundamental to understanding quantum state spaces:

1. **Linear Independence**: No vector is a combination of others; tested via rank or determinant
2. **Span**: All possible linear combinations; always forms a subspace
3. **Basis**: Linearly independent and spanning; uniquely represents every vector
4. **Dimension**: Number of basis vectors; for n qubits, dimension is 2ⁿ
5. **Change of Basis**: Same vector, different representations; crucial for quantum algorithms

These concepts explain why:
- A qubit has exactly 2 basis states
- n qubits have 2ⁿ basis states (exponential growth)
- Quantum states can be represented in different bases (computational, Hadamard, etc.)
- Quantum algorithms can exploit different basis representations
