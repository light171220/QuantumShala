## 1. Key Properties

### Unitarity
QFT† · QFT = I

### Symmetry
QFT = QFTᵀ (symmetric matrix)

### Periodicity
QFT|j + N⟩ = QFT|j⟩

```python
import numpy as np

def qft(n):
    N = 2**n
    omega = np.exp(2j*np.pi/N)
    return np.array([[omega**(j*k) for k in range(N)] for j in range(N)])/np.sqrt(N)

QFT = qft(2)
print("Unitary:", np.allclose(QFT @ QFT.conj().T, np.eye(4)))
print("Symmetric:", np.allclose(QFT, QFT.T))
```

## 2. QFT on Uniform Superposition

```python
import numpy as np

def qft(n):
    N = 2**n
    omega = np.exp(2j*np.pi/N)
    return np.array([[omega**(j*k) for k in range(N)] for j in range(N)])/np.sqrt(N)

# Uniform superposition |+...+⟩
uniform = np.ones(4) / 2
result = qft(2) @ uniform
print("QFT|+⟩⊗|+⟩ =", np.round(result, 3))
print("Result is |00⟩!")
```
