## Learning Objectives

## 1. What is the Fourier Transform?

The Fourier Transform decomposes a signal into its frequency components.

<Definition term="Discrete Fourier Transform">
The DFT transforms a sequence of N complex numbers into another sequence representing frequency amplitudes.
</Definition>

## 2. The DFT Formula

<Formula>
$$X_k = \sum_{n=0}^{N-1} x_n \cdot e^{-2\pi i kn/N}$$
</Formula>

Where ω = e^(2πi/N) is the N-th root of unity.

```python
import numpy as np

def dft(x):
    """Compute the Discrete Fourier Transform."""
    N = len(x)
    X = np.zeros(N, dtype=complex)
    for k in range(N):
        for n in range(N):
            X[k] += x[n] * np.exp(-2j * np.pi * k * n / N)
    return X

# Example: Transform a simple signal
signal = [1, 0, 1, 0]
spectrum = dft(signal)
print("Signal:", signal)
print("DFT:", np.round(spectrum, 2))
```

## 3. Matrix Form of DFT

The DFT can be written as matrix multiplication:

```python
import numpy as np

def dft_matrix(N):
    """Create the N×N DFT matrix."""
    omega = np.exp(-2j * np.pi / N)
    return np.array([[omega**(j*k) for k in range(N)] for j in range(N)])

F4 = dft_matrix(4)
print("4×4 DFT Matrix:")
print(np.round(F4, 2))

# Verify with NumPy's FFT
signal = [1, 2, 3, 4]
print("\nManual DFT:", np.round(F4 @ signal, 2))
print("NumPy FFT:", np.round(np.fft.fft(signal), 2))
```

## 4. Computational Complexity

<Definition term="DFT Complexity">
- Naive DFT: O(N²) operations
- Fast Fourier Transform (FFT): O(N log N) operations
</Definition>

```python
import numpy as np

# Compare computation times conceptually
def count_operations(N, method):
    if method == "naive":
        return N * N
    elif method == "fft":
        return N * np.log2(N)

for N in [8, 64, 1024]:
    naive = count_operations(N, "naive")
    fft = count_operations(N, "fft")
    print(f"N={N}: Naive={naive:.0f}, FFT={fft:.0f}, Speedup={naive/fft:.1f}x")
```

## 5. Properties of DFT

Key properties:
- **Linearity**: DFT(ax + by) = a·DFT(x) + b·DFT(y)
- **Periodicity**: The DFT is periodic with period N
- **Inverse**: The inverse DFT uses conjugate of the matrix

```python
import numpy as np

def inverse_dft(X):
    """Compute inverse DFT."""
    N = len(X)
    x = np.zeros(N, dtype=complex)
    for n in range(N):
        for k in range(N):
            x[n] += X[k] * np.exp(2j * np.pi * k * n / N)
    return x / N

# Verify inverse
signal = [1, 2, 3, 4]
spectrum = dft(signal)
recovered = inverse_dft(spectrum)
print("Original:", signal)
print("Recovered:", np.round(recovered.real, 2))
```

## Summary

- DFT transforms signals from time to frequency domain
- Formula: Xₖ = Σₙ xₙ·ω^(kn) where ω = e^(-2πi/N)
- Complexity: O(N²) naive, O(N log N) with FFT
- Matrix form enables clear understanding of the transformation
- Quantum version (QFT) will achieve O(log² N) complexity!
