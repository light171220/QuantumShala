## 1. QFT Definition

The Quantum Fourier Transform is the quantum analog of the DFT.

<Definition term="Quantum Fourier Transform">
QFT transforms computational basis states to Fourier basis states:
QFT|j⟩ = (1/√N) Σₖ e^(2πijk/N)|k⟩
</Definition>

<Formula>
$$\text{QFT}|j\rangle = \frac{1}{\sqrt{N}} \sum_{k=0}^{N-1} e^{2\pi i jk/N} |k\rangle$$
</Formula>

## 2. QFT Matrix

```python
import numpy as np

def qft_matrix(n_qubits):
    """Create the QFT matrix for n qubits."""
    N = 2**n_qubits
    omega = np.exp(2j * np.pi / N)
    F = np.array([[omega**(j*k) for k in range(N)] for j in range(N)])
    return F / np.sqrt(N)

# 2-qubit QFT
QFT_2 = qft_matrix(2)
print("2-qubit QFT matrix:")
print(np.round(QFT_2, 3))

# Verify unitarity
print("\nQFT is unitary:", np.allclose(QFT_2 @ QFT_2.conj().T, np.eye(4)))
```

## 3. QFT vs DFT

Key differences:

| Property | Classical DFT | Quantum QFT |
|----------|---------------|-------------|
| Input | N complex numbers | log₂(N) qubits |
| Output | N complex numbers | log₂(N) qubits |
| Complexity | O(N log N) with FFT | O(log² N) |
| Readout | Direct | Measurement collapses |

```python
import numpy as np

def compare_complexity(n_qubits):
    N = 2**n_qubits
    fft = N * np.log2(N)
    qft = n_qubits**2
    return fft, qft

print("Complexity Comparison:")
print(f"{'Qubits':<8} {'N':<12} {'FFT':<15} {'QFT':<15} {'Speedup'}")
for n in [4, 8, 12, 16, 20]:
    N = 2**n
    fft, qft = compare_complexity(n)
    print(f"{n:<8} {N:<12} {fft:<15.0f} {qft:<15.0f} {fft/qft:.0f}x")
```

## 4. Product Representation

QFT has a beautiful product form:

<Formula>
$$\text{QFT}|j\rangle = \frac{1}{\sqrt{N}} \bigotimes_{l=1}^{n} \left(|0\rangle + e^{2\pi i j/2^l}|1\rangle\right)$$
</Formula>

```python
import numpy as np

def qft_product_form(j, n_qubits):
    """Compute QFT|j⟩ using product form."""
    N = 2**n_qubits
    state = np.array([1.0])
    
    for l in range(1, n_qubits + 1):
        phase = np.exp(2j * np.pi * j / (2**l))
        qubit_state = np.array([1, phase]) / np.sqrt(2)
        state = np.kron(qubit_state, state)
    
    return state

# Verify against matrix form
n = 3
j = 5
QFT = qft_matrix(n)
basis_j = np.zeros(2**n)
basis_j[j] = 1

matrix_result = QFT @ basis_j
product_result = qft_product_form(j, n)

print(f"QFT|{j}⟩ (matrix):", np.round(matrix_result, 3))
print(f"QFT|{j}⟩ (product):", np.round(product_result, 3))
print("Match:", np.allclose(matrix_result, product_result))
```

## 5. Key Properties

1. **Unitary**: QFT† QFT = I
2. **Symmetric**: QFT = QFT^T
3. **Periodic**: QFT|j+N⟩ = QFT|j⟩

## Summary

- QFT|j⟩ = (1/√N) Σₖ ω^(jk)|k⟩ where ω = e^(2πi/N)
- QFT is unitary and has O(log² N) gate complexity
- Product form enables efficient circuit implementation
- Exponential speedup over classical FFT in terms of gates
