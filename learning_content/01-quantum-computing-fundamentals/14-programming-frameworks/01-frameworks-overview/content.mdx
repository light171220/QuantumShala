# Quantum Programming Frameworks Overview

## Introduction

Quantum programming frameworks provide the tools to design, simulate, and execute quantum algorithms. They bridge the gap between quantum theory and practical implementation on real hardware.

## Why Frameworks?

```
The Need for Quantum Frameworks
═══════════════════════════════

ABSTRACTION:
    Hide low-level hardware details
    Focus on algorithm design
    Portable across backends

    Algorithm → Circuit → Pulses → Hardware

SIMULATION:
    Test on classical computers first
    Debug without hardware access
    Verify correctness

COMPILATION:
    Translate ideal circuits to hardware
    Optimize gate sequences
    Route for connectivity

EXECUTION:
    Submit to quantum hardware
    Manage job queues
    Retrieve results


WITHOUT FRAMEWORKS:
    - Write raw pulses by hand
    - Handle every hardware quirk
    - No debugging tools
    - No portability
```

## Common Concepts

```
Shared Concepts Across Frameworks
═════════════════════════════════

1. QUANTUM CIRCUITS
   Sequence of gates on qubits
   Standard representation

   ┌───┐     ┌───┐
   ─┤ H ├──●──┤ M ├─
   └───┘  │  └───┘
         ┌┴┐ ┌───┐
   ──────┤X├─┤ M ├─
         └─┘ └───┘

2. QUANTUM REGISTERS
   Named collections of qubits
   Classical bits for measurement

3. GATES
   Single-qubit: H, X, Y, Z, Rx, Ry, Rz
   Two-qubit: CNOT, CZ, SWAP
   Multi-qubit: Toffoli, custom

4. BACKENDS
   Simulators: Classical emulation
   Hardware: Real quantum computers

5. TRANSPILATION
   Circuit optimization
   Hardware-specific compilation
   Gate decomposition

6. EXECUTION
   Shots: Number of runs
   Results: Measurement counts
   Post-processing
```

## First Circuit Example

```python
# Same circuit in different frameworks

# QISKIT:
"""
from qiskit import QuantumCircuit
qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()
"""

# CIRQ:
"""
import cirq
q0, q1 = cirq.LineQubit.range(2)
circuit = cirq.Circuit([
    cirq.H(q0),
    cirq.CNOT(q0, q1),
    cirq.measure(q0, q1)
])
"""

# PENNYLANE:
"""
import pennylane as qml
dev = qml.device('default.qubit', wires=2)

@qml.qnode(dev)
def circuit():
    qml.Hadamard(wires=0)
    qml.CNOT(wires=[0, 1])
    return qml.probs(wires=[0, 1])
"""

print("""
All create the same Bell state circuit:

    |0⟩ ──H──●─── |0⟩ + |1⟩
             │        ↓
    |0⟩ ─────X─── |00⟩ + |11⟩
                   (Bell state)
""")
```
