## 1. The Superposition Creator

The **Hadamard gate** (H) is arguably the most important single-qubit gate in quantum computing. It transforms computational basis states into equal superpositions, unlocking the power of quantum parallelism.

<Definition term="Hadamard Gate">
The Hadamard gate creates equal superpositions from basis states. It maps |0⟩ to |+⟩ and |1⟩ to |-⟩, transforming between the computational (Z) basis and the Hadamard (X) basis.
</Definition>

<Formula>
$$H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$$
</Formula>

```python
import numpy as np

# Define the Hadamard gate
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

print("Hadamard Gate Matrix:")
print(H)
print(f"\nNormalization factor: 1/√2 ≈ {1/np.sqrt(2):.4f}")
```

## 2. Action on Basis States

The Hadamard gate transforms computational basis states into superpositions:

<Formula>
$$H|0\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle) = |+\rangle$$
$$H|1\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle) = |-\rangle$$
</Formula>

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

# Basis states
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

# Apply Hadamard
plus = H @ ket_0   # |+⟩
minus = H @ ket_1  # |-⟩

print("Hadamard on Basis States:")
print(f"H|0⟩ = {np.round(plus, 4)} = |+⟩ = (|0⟩ + |1⟩)/√2")
print(f"H|1⟩ = {np.round(minus, 4)} = |-⟩ = (|0⟩ - |1⟩)/√2")

# Verify normalization
print(f"\n||H|0⟩|| = {np.linalg.norm(plus):.4f}")
print(f"||H|1⟩|| = {np.linalg.norm(minus):.4f}")
```

<Tip>
After applying H to |0⟩, measuring gives |0⟩ or |1⟩ with equal probability (50% each). This is the essence of superposition - the qubit is in both states simultaneously until measured!
</Tip>

## 3. The Hadamard Basis

The states |+⟩ and |-⟩ form an alternative orthonormal basis called the **Hadamard basis** or **X basis**:

<Formula>
$$|+\rangle = \frac{|0\rangle + |1\rangle}{\sqrt{2}}, \quad |-\rangle = \frac{|0\rangle - |1\rangle}{\sqrt{2}}$$
</Formula>

```python
import numpy as np

# Define both bases
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])
plus = np.array([1, 1]) / np.sqrt(2)
minus = np.array([1, -1]) / np.sqrt(2)

# Verify orthonormality of {|+⟩, |-⟩}
print("Hadamard Basis Orthonormality:")
print(f"⟨+|+⟩ = {np.vdot(plus, plus):.4f}")
print(f"⟨-|-⟩ = {np.vdot(minus, minus):.4f}")
print(f"⟨+|-⟩ = {np.vdot(plus, minus):.4f}")

# Express |0⟩ and |1⟩ in Hadamard basis
print("\nComputational basis in Hadamard basis:")
print(f"|0⟩ = (|+⟩ + |-⟩)/√2")
print(f"|1⟩ = (|+⟩ - |-⟩)/√2")

# Verify
ket_0_reconstructed = (plus + minus) / np.sqrt(2)
ket_1_reconstructed = (plus - minus) / np.sqrt(2)
print(f"\nVerification:")
print(f"|0⟩ reconstructed: {np.round(ket_0_reconstructed, 4)}")
print(f"|1⟩ reconstructed: {np.round(ket_1_reconstructed, 4)}")
```

## 4. Hadamard is Self-Inverse: H² = I

One of the most important properties of the Hadamard gate is that it's its own inverse:

<Formula>
$$H^2 = HH = I$$
$$H = H^{-1} = H^\dagger$$
</Formula>

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
I = np.eye(2)

# Verify H² = I
H_squared = H @ H

print("H² = I Verification:")
print(f"H² =\n{np.round(H_squared, 10)}")
print(f"\nH² = I: {np.allclose(H_squared, I)}")

# Demonstrate: applying H twice returns to original state
ket_0 = np.array([1, 0])
after_H = H @ ket_0
after_HH = H @ after_H

print(f"\n|0⟩ → H → {np.round(after_H, 4)} → H → {np.round(after_HH, 4)}")
print("We're back to |0⟩!")
```

<Warning>
This property is crucial for quantum algorithms! The Hadamard gate creates superposition, and applying it again "undoes" the superposition, returning to a definite state.
</Warning>

## 5. Bloch Sphere Interpretation

On the Bloch sphere, the Hadamard gate performs a **rotation of 180° around the axis (X+Z)/√2**:

- |0⟩ (north pole) → |+⟩ (positive X equator)
- |1⟩ (south pole) → |-⟩ (negative X equator)
- |+⟩ → |0⟩
- |-⟩ → |1⟩

```python
import numpy as np

def state_to_bloch(state):
    """Convert state to Bloch coordinates (x, y, z)"""
    X = np.array([[0, 1], [1, 0]])
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]])
    
    state = state / np.linalg.norm(state)
    x = np.real(state.conj() @ X @ state)
    y = np.real(state.conj() @ Y @ state)
    z = np.real(state.conj() @ Z @ state)
    return (x, y, z)

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

states = {
    '|0⟩': np.array([1, 0]),
    '|1⟩': np.array([0, 1]),
    '|+⟩': np.array([1, 1]) / np.sqrt(2),
    '|-⟩': np.array([1, -1]) / np.sqrt(2),
}

print("Hadamard on Bloch Sphere:")
print(f"{'State':<6} {'Before (x,y,z)':<20} {'After H (x,y,z)':<20}")
print("-" * 50)

for name, state in states.items():
    before = state_to_bloch(state)
    after = state_to_bloch(H @ state)
    print(f"{name:<6} ({before[0]:>5.2f},{before[1]:>5.2f},{before[2]:>5.2f})   "
          f"({after[0]:>5.2f},{after[1]:>5.2f},{after[2]:>5.2f})")
```

## 6. Hadamard as Basis Change

The Hadamard gate can be understood as a **change of basis** between the computational (Z) and Hadamard (X) bases:

<Formula>
$$H = |+\rangle\langle 0| + |-\rangle\langle 1| = |0\rangle\langle +| + |1\rangle\langle -|$$
</Formula>

```python
import numpy as np

# Basis states
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])
plus = np.array([1, 1]) / np.sqrt(2)
minus = np.array([1, -1]) / np.sqrt(2)

# Construct H from outer products
# H = |+⟩⟨0| + |-⟩⟨1|
H_constructed = np.outer(plus, ket_0) + np.outer(minus, ket_1)

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

print("Hadamard as Basis Change:")
print(f"H = |+⟩⟨0| + |-⟩⟨1|")
print(f"\nConstructed H =\n{np.round(H_constructed, 4)}")
print(f"\nStandard H =\n{np.round(H, 4)}")
print(f"\nMatch: {np.allclose(H_constructed, H)}")
```

## 7. Eigenvalues and Eigenvectors

The Hadamard gate has eigenvalues +1 and -1:

<Formula>
$$H|h_+\rangle = +|h_+\rangle, \quad H|h_-\rangle = -|h_-\rangle$$
$$|h_+\rangle = \cos(\pi/8)|0\rangle + \sin(\pi/8)|1\rangle$$
$$|h_-\rangle = \sin(\pi/8)|0\rangle - \cos(\pi/8)|1\rangle$$
</Formula>

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

# Compute eigenvalues and eigenvectors
eigenvalues, eigenvectors = np.linalg.eig(H)

print("Hadamard Eigendecomposition:")
print(f"Eigenvalues: {eigenvalues}")

for i, (val, vec) in enumerate(zip(eigenvalues, eigenvectors.T)):
    print(f"\nEigenvalue λ_{i+1} = {val:.4f}")
    print(f"Eigenvector |h_{'+' if val > 0 else '-'}⟩ = {np.round(vec, 4)}")
    
    # Verify: H|v⟩ = λ|v⟩
    Hv = H @ vec
    lambda_v = val * vec
    print(f"Verification H|v⟩ = λ|v⟩: {np.allclose(Hv, lambda_v)}")

# The eigenvectors are at angles π/8 and 5π/8 from |0⟩
print(f"\nAngle of |h_+⟩ from |0⟩: π/8 = {np.pi/8:.4f} rad = 22.5°")
```

## 8. Hadamard in Quantum Algorithms

The Hadamard gate is essential for quantum algorithms:

### Creating Uniform Superposition

Apply H to each qubit in |0...0⟩ to create uniform superposition over all basis states:

<Formula>
$$H^{\otimes n}|0\rangle^{\otimes n} = \frac{1}{\sqrt{2^n}}\sum_{x=0}^{2^n-1}|x\rangle$$
</Formula>

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

def multi_qubit_hadamard(n):
    """Create n-qubit Hadamard H⊗n"""
    result = H
    for _ in range(n - 1):
        result = np.kron(result, H)
    return result

# Example: 3-qubit uniform superposition
n = 3
H_3 = multi_qubit_hadamard(n)
ket_000 = np.zeros(2**n)
ket_000[0] = 1

uniform_superposition = H_3 @ ket_000

print(f"{n}-Qubit Uniform Superposition:")
print(f"H⊗{n}|000⟩ creates equal superposition of {2**n} states")
print(f"\nAmplitudes: {np.round(uniform_superposition, 4)}")
print(f"Each amplitude = 1/√{2**n} = {1/np.sqrt(2**n):.4f}")
print(f"\nProbability of each state = 1/{2**n} = {1/2**n:.4f}")
```

### Interference Pattern

The Hadamard gate enables quantum interference:

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
Z = np.array([[1, 0], [0, -1]])

ket_0 = np.array([1, 0])

# Create superposition
superposition = H @ ket_0
print(f"After H: {np.round(superposition, 4)} = |+⟩")

# Apply phase (Z gate)
with_phase = Z @ superposition
print(f"After Z: {np.round(with_phase, 4)} = |-⟩")

# Apply H again (interference!)
final = H @ with_phase
print(f"After H: {np.round(final, 4)} = |1⟩")

print("\nThe phase information affected the final measurement!")
print("This is quantum interference - the core of quantum speedup.")
```

<Tip>
The pattern H → computation → H is fundamental to quantum algorithms. The first H creates superposition (quantum parallelism), computation adds phases, and the final H creates interference that amplifies correct answers.
</Tip>

## 9. Relationship with Other Gates

### Hadamard and Pauli Gates

<Formula>
$$HXH = Z, \quad HYH = -Y, \quad HZH = X$$
</Formula>

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])

print("Hadamard Conjugation of Pauli Gates:")
print(f"HXH = Z: {np.allclose(H @ X @ H, Z)}")
print(f"HYH = -Y: {np.allclose(H @ Y @ H, -Y)}")
print(f"HZH = X: {np.allclose(H @ Z @ H, X)}")

print("\nThis means H 'swaps' the X and Z bases!")
```

### Hadamard Decomposition

<Formula>
$$H = \frac{1}{\sqrt{2}}(X + Z) = e^{i\pi/2} R_y(\pi/2) R_z(\pi)$$
</Formula>

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
X = np.array([[0, 1], [1, 0]])
Z = np.array([[1, 0], [0, -1]])

# H as linear combination of X and Z
H_decomposed = (X + Z) / np.sqrt(2)

print("Hadamard Decomposition:")
print(f"H = (X + Z)/√2")
print(f"Verification: {np.allclose(H, H_decomposed)}")
```
