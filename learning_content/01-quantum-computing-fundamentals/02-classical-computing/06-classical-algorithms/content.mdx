## 2. Search Algorithms

### Linear Search

The simplest search: check each element one by one.

<Example>
```python
def linear_search(arr: list, target) -> int:
    """
    Linear search: O(N) worst case, O(1) best case.
    Returns index of target, or -1 if not found.
    """
    for i, item in enumerate(arr):
        if item == target:
            return i
    return -1

# Test
arr = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3]
print(f"Search for 5: index {linear_search(arr, 5)}")
print(f"Search for 7: index {linear_search(arr, 7)}")
```
</Example>

### Binary Search

For sorted arrays: eliminate half the search space each step.

<Formula>
Time complexity: O(log N) - exponentially faster than linear search!
</Formula>

<Example>
```python
def binary_search(arr: list, target) -> int:
    """
    Binary search on sorted array: O(log N).
    """
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1

# Test
arr = sorted([3, 1, 4, 1, 5, 9, 2, 6, 5, 3])
print(f"Sorted array: {arr}")
print(f"Search for 5: index {binary_search(arr, 5)}")
print(f"Search for 7: index {binary_search(arr, 7)}")

# Compare operations
import math
n = 1000000
print(f"\nFor N = {n:,}:")
print(f"  Linear search: up to {n:,} comparisons")
print(f"  Binary search: up to {int(math.log2(n))} comparisons")
```
</Example>

<Tip>
Grover's quantum algorithm searches unstructured data in O(√N) - a quadratic speedup over linear search's O(N), but not as fast as binary search's O(log N) for sorted data!
</Tip>

## 3. Sorting Algorithms

| Algorithm | Best | Average | Worst | Space |
|-----------|------|---------|-------|-------|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) | O(1) |

<Example>
```python
def merge_sort(arr: list) -> list:
    """
    Merge sort: O(n log n) guaranteed.
    Divide and conquer approach.
    """
    if len(arr) <= 1:
        return arr
    
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    return merge(left, right)

def merge(left: list, right: list) -> list:
    """Merge two sorted lists."""
    result = []
    i = j = 0
    
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    
    result.extend(left[i:])
    result.extend(right[j:])
    return result

# Test
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original: {arr}")
print(f"Sorted:   {merge_sort(arr)}")
```
</Example>

## 4. Graph Algorithms

Graphs represent connections between objects - essential for networking, social media, and optimization.

### Breadth-First Search (BFS)

Explores neighbors level by level.

<Example>
```python
from collections import deque

def bfs(graph: dict, start) -> list:
    """
    BFS traversal: O(V + E).
    Returns nodes in order visited.
    """
    visited = set()
    queue = deque([start])
    order = []
    
    while queue:
        node = queue.popleft()
        if node not in visited:
            visited.add(node)
            order.append(node)
            for neighbor in graph.get(node, []):
                if neighbor not in visited:
                    queue.append(neighbor)
    
    return order

# Test
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}

print(f"BFS from A: {bfs(graph, 'A')}")
```
</Example>

### Depth-First Search (DFS)

Explores as deep as possible before backtracking.

<Example>
```python
def dfs(graph: dict, start) -> list:
    """
    DFS traversal: O(V + E).
    Returns nodes in order visited.
    """
    visited = set()
    order = []
    
    def explore(node):
        if node in visited:
            return
        visited.add(node)
        order.append(node)
        for neighbor in graph.get(node, []):
            explore(neighbor)
    
    explore(start)
    return order

print(f"DFS from A: {dfs(graph, 'A')}")
```
</Example>

### Dijkstra's Shortest Path

Find shortest paths from a source to all other nodes.

<Example>
```python
import heapq

def dijkstra(graph: dict, start) -> dict:
    """
    Dijkstra's algorithm: O((V + E) log V) with heap.
    graph: {node: [(neighbor, weight), ...]}
    Returns: {node: shortest_distance}
    """
    distances = {node: float('inf') for node in graph}
    distances[start] = 0
    heap = [(0, start)]
    
    while heap:
        dist, node = heapq.heappop(heap)
        
        if dist > distances[node]:
            continue
        
        for neighbor, weight in graph.get(node, []):
            new_dist = dist + weight
            if new_dist < distances[neighbor]:
                distances[neighbor] = new_dist
                heapq.heappush(heap, (new_dist, neighbor))
    
    return distances

# Test
weighted_graph = {
    'A': [('B', 1), ('C', 4)],
    'B': [('A', 1), ('C', 2), ('D', 5)],
    'C': [('A', 4), ('B', 2), ('D', 1)],
    'D': [('B', 5), ('C', 1)]
}

distances = dijkstra(weighted_graph, 'A')
print("Shortest distances from A:")
for node, dist in sorted(distances.items()):
    print(f"  {node}: {dist}")
```
</Example>

## 5. Randomized Algorithms

Randomness can simplify algorithms and improve average performance.

### Randomized QuickSort

<Example>
```python
import random

def quicksort_randomized(arr: list) -> list:
    """
    Randomized quicksort: O(n log n) expected.
    Random pivot avoids worst-case on sorted input.
    """
    if len(arr) <= 1:
        return arr
    
    pivot = random.choice(arr)
    less = [x for x in arr if x < pivot]
    equal = [x for x in arr if x == pivot]
    greater = [x for x in arr if x > pivot]
    
    return quicksort_randomized(less) + equal + quicksort_randomized(greater)

arr = list(range(20, 0, -1))  # Worst case for standard quicksort
print(f"Sorted: {quicksort_randomized(arr)}")
```
</Example>

### Monte Carlo Algorithms

Use random sampling to estimate results.

<Example>
```python
import random

def estimate_pi(n_samples: int) -> float:
    """
    Estimate π using Monte Carlo: throw darts at unit square.
    """
    inside_circle = 0
    
    for _ in range(n_samples):
        x, y = random.random(), random.random()
        if x*x + y*y <= 1:
            inside_circle += 1
    
    # Area of quarter circle = π/4
    return 4 * inside_circle / n_samples

for n in [100, 1000, 10000, 100000]:
    pi_estimate = estimate_pi(n)
    error = abs(pi_estimate - 3.14159265) / 3.14159265 * 100
    print(f"n={n:>6}: π ≈ {pi_estimate:.4f} (error: {error:.2f}%)")
```
</Example>

## 6. Algorithms Quantum Can Improve

| Classical Algorithm | Complexity | Quantum Algorithm | Complexity | Speedup |
|---------------------|------------|-------------------|------------|---------|
| Linear search | O(N) | Grover's search | O(√N) | Quadratic |
| Integer factoring | O(exp(n^⅓)) | Shor's algorithm | O(n³) | Exponential |
| Simulating physics | O(2^n) | Quantum simulation | O(n) | Exponential |
| Optimization | Varies | Quantum annealing | Varies | Problem-dependent |

<Warning>
Quantum doesn't speed up everything! Sorting, shortest path, and many graph algorithms have no known quantum speedup beyond polynomial factors.
</Warning>
