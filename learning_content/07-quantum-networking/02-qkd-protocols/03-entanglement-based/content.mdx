# Entanglement-Based Protocols

Entanglement-based QKD protocols leverage the unique correlations of entangled quantum states to establish secure keys. Unlike prepare-and-measure protocols where one party creates states, here both parties receive halves of entangled pairs and perform measurements. This approach offers unique security guarantees and enables device-independent security proofs.

## Entanglement-Based Architecture

The fundamental difference from prepare-and-measure protocols lies in the source of randomness and correlation.

### Architectural Overview

### Bell States and Perfect Correlations

```python
import numpy as np
from dataclasses import dataclass
from typing import List, Tuple, Dict, Optional
from enum import Enum
import math

class BellState(Enum):
    """The four maximally entangled Bell states."""
    PHI_PLUS = 0   # |Φ+⟩ = (|00⟩ + |11⟩)/√2
    PHI_MINUS = 1  # |Φ-⟩ = (|00⟩ - |11⟩)/√2
    PSI_PLUS = 2   # |Ψ+⟩ = (|01⟩ + |10⟩)/√2
    PSI_MINUS = 3  # |Ψ-⟩ = (|01⟩ - |10⟩)/√2

@dataclass
class EntangledPair:
    """Represents an entangled photon pair."""
    state: BellState
    index: int
    alice_measured: bool = False
    bob_measured: bool = False
    alice_result: Optional[int] = None
    bob_result: Optional[int] = None

class EntanglementSource:
    """Entangled photon pair source."""

    def __init__(self, fidelity: float = 1.0,
                 state: BellState = BellState.PHI_PLUS):
        """
        Args:
            fidelity: Probability of producing correct Bell state
            state: Target Bell state to produce
        """
        self.fidelity = fidelity
        self.target_state = state
        self.pairs_generated = 0

    def generate_pair(self) -> EntangledPair:
        """Generate an entangled pair."""
        if np.random.random() < self.fidelity:
            state = self.target_state
        else:
            # Mixed with white noise
            state = np.random.choice(list(BellState))

        self.pairs_generated += 1
        return EntangledPair(state=state, index=self.pairs_generated)

    def measure_correlation(self, pair: EntangledPair,
                           alice_basis: float,
                           bob_basis: float) -> Tuple[int, int]:
        """
        Measure entangled pair in given bases.

        Args:
            pair: The entangled pair to measure
            alice_basis: Alice's measurement angle (radians)
            bob_basis: Bob's measurement angle (radians)

        Returns:
            Tuple of (alice_result, bob_result)
        """
        # Correlation depends on Bell state and measurement angles
        if pair.state == BellState.PHI_PLUS:
            # Perfect correlation when same basis
            correlation = np.cos(alice_basis - bob_basis) ** 2
        elif pair.state == BellState.PHI_MINUS:
            correlation = np.cos(alice_basis - bob_basis) ** 2
        elif pair.state == BellState.PSI_PLUS:
            # Anti-correlation
            correlation = np.sin(alice_basis - bob_basis) ** 2
        else:  # PSI_MINUS
            correlation = np.sin(alice_basis - bob_basis) ** 2

        # Generate correlated outcomes
        alice_result = np.random.randint(0, 2)

        if np.random.random() < correlation:
            if pair.state in [BellState.PHI_PLUS, BellState.PHI_MINUS]:
                bob_result = alice_result  # Same
            else:
                bob_result = 1 - alice_result  # Opposite
        else:
            if pair.state in [BellState.PHI_PLUS, BellState.PHI_MINUS]:
                bob_result = 1 - alice_result
            else:
                bob_result = alice_result

        pair.alice_measured = True
        pair.bob_measured = True
        pair.alice_result = alice_result
        pair.bob_result = bob_result

        return alice_result, bob_result
```

## E91 Protocol

The E91 protocol, proposed by Artur Ekert in 1991, uses Bell inequality violations to verify the absence of eavesdropping.

### E91 Measurement Settings

```
┌─────────────────────────────────────────────────────────────────┐
│                   E91 Measurement Angles                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Alice's bases:              Bob's bases:                      │
│   a₁ = 0°                     b₁ = 22.5°                       │
│   a₂ = 45°                    b₂ = 67.5°                       │
│   a₃ = 90°                    b₃ = 112.5°                      │
│                                                                 │
│                    Angle diagram:                               │
│                                                                 │
│                        90° (a₃)                                │
│                          │                                      │
│                    b₃ ╲  │  ╱ b₂                               │
│                       ╲ │ ╱                                     │
│             a₂ ─────────●───────── 0° (a₁)                     │
│                       ╱ │ ╲                                     │
│                      ╱  │  ╲ b₁                                │
│                          │                                      │
│                                                                 │
│   Key generation: a₁-b₁ or a₃-b₃ (45° apart each)              │
│   Bell test: a₁-b₂, a₂-b₁, a₂-b₂, a₂-b₃, a₃-b₂                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### E91 Implementation

```python
class E91Protocol:
    """Complete E91 protocol implementation."""

    def __init__(self, n_pairs: int, source_fidelity: float = 1.0):
        self.n_pairs = n_pairs
        self.source = EntanglementSource(fidelity=source_fidelity)

        # E91 measurement angles (in radians)
        self.alice_angles = [0, np.pi/4, np.pi/2]  # 0°, 45°, 90°
        self.bob_angles = [np.pi/8, 3*np.pi/8, 5*np.pi/8]  # 22.5°, 67.5°, 112.5°

        # Results storage
        self.results = []
        self.key_pairs = []
        self.bell_test_pairs = []

    def run_protocol(self) -> Dict:
        """Execute the E91 protocol."""
        # Step 1: Generate and measure entangled pairs
        for i in range(self.n_pairs):
            pair = self.source.generate_pair()

            # Random basis selection
            alice_basis_idx = np.random.randint(0, 3)
            bob_basis_idx = np.random.randint(0, 3)

            alice_angle = self.alice_angles[alice_basis_idx]
            bob_angle = self.bob_angles[bob_basis_idx]

            # Measure
            alice_result, bob_result = self.source.measure_correlation(
                pair, alice_angle, bob_angle
            )

            self.results.append({
                'index': i,
                'alice_basis': alice_basis_idx,
                'bob_basis': bob_basis_idx,
                'alice_result': alice_result,
                'bob_result': bob_result
            })

        # Step 2: Basis reconciliation (over classical channel)
        self._classify_results()

        # Step 3: Bell test on subset
        bell_value = self._perform_bell_test()

        # Step 4: Extract key from matching bases
        key_alice, key_bob = self._extract_key()

        return {
            'total_pairs': self.n_pairs,
            'key_pairs': len(self.key_pairs),
            'bell_test_pairs': len(self.bell_test_pairs),
            'bell_value': bell_value,
            'classical_bound': 2.0,
            'quantum_max': 2 * np.sqrt(2),
            'bell_violation': bell_value > 2.0,
            'key_alice': key_alice,
            'key_bob': key_bob,
            'qber': self._calculate_qber(key_alice, key_bob)
        }

    def _classify_results(self):
        """Classify results into key generation and Bell test."""
        for r in self.results:
            # Key generation: a₁-b₁ (indices 0,0) or a₃-b₃ (indices 2,2)
            # Actually in E91: a₁-b₃ or a₃-b₁ give 45° difference for key
            if (r['alice_basis'] == 0 and r['bob_basis'] == 0):
                # a₁ and b₁ are 22.5° apart - good for key (strong correlation)
                self.key_pairs.append(r)
            elif (r['alice_basis'] == 2 and r['bob_basis'] == 2):
                # a₃ and b₃ are also 22.5° apart
                self.key_pairs.append(r)
            else:
                # Use other combinations for Bell test
                self.bell_test_pairs.append(r)

    def _perform_bell_test(self) -> float:
        """Calculate CHSH Bell inequality value."""
        # Count correlations for each basis combination
        correlators = {}

        for r in self.bell_test_pairs:
            key = (r['alice_basis'], r['bob_basis'])
            if key not in correlators:
                correlators[key] = {'same': 0, 'diff': 0}

            if r['alice_result'] == r['bob_result']:
                correlators[key]['same'] += 1
            else:
                correlators[key]['diff'] += 1

        # Calculate E(a,b) = (N_same - N_diff) / (N_same + N_diff)
        def E(a, b):
            if (a, b) not in correlators:
                return 0
            c = correlators[(a, b)]
            total = c['same'] + c['diff']
            if total == 0:
                return 0
            return (c['same'] - c['diff']) / total

        # CHSH: S = E(a₁,b₁) - E(a₁,b₃) + E(a₃,b₁) + E(a₃,b₃)
        # Using our indices: a₁=0, a₂=1, a₃=2, b₁=0, b₂=1, b₃=2
        S = abs(E(0, 1) - E(0, 2) + E(1, 1) + E(1, 2))

        return S

    def _extract_key(self) -> Tuple[List[int], List[int]]:
        """Extract raw key from matching-basis measurements."""
        alice_key = []
        bob_key = []

        for r in self.key_pairs:
            alice_key.append(r['alice_result'])
            # Bob may need to flip for PSI states
            bob_key.append(r['bob_result'])

        return alice_key, bob_key

    def _calculate_qber(self, key_a: List[int], key_b: List[int]) -> float:
        """Calculate quantum bit error rate."""
        if not key_a:
            return 0.0
        errors = sum(a != b for a, b in zip(key_a, key_b))
        return errors / len(key_a)

# Run E91 simulation
e91 = E91Protocol(n_pairs=10000)
result = e91.run_protocol()
print(f"Bell value S = {result['bell_value']:.3f}")
print(f"Classical bound: {result['classical_bound']}")
print(f"Quantum maximum: {result['quantum_max']:.3f}")
print(f"Bell violation: {result['bell_violation']}")
print(f"QBER: {result['qber']:.3f}")
```

### Bell Inequality and Security

```
┌─────────────────────────────────────────────────────────────────┐
│                 CHSH Bell Inequality                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   CHSH inequality: |S| ≤ 2  (classical bound)                  │
│                                                                 │
│   S = |E(a,b) - E(a,b') + E(a',b) + E(a',b')|                  │
│                                                                 │
│   Where E(a,b) is the correlation coefficient:                  │
│   E(a,b) = P(same) - P(different)                              │
│                                                                 │
│                                                                 │
│   Value Interpretation:                                         │
│   ├─────────────────────────────────────────────────────────┤  │
│   │   S = 0      │  No correlation (noise)                  │  │
│   │   S ≤ 2      │  Classical - LHV possible                │  │
│   │   2 < S ≤ 2√2│  Quantum correlations                    │  │
│   │   S = 2√2    │  Maximum quantum violation               │  │
│   ├─────────────────────────────────────────────────────────┤  │
│                                                                 │
│   Security connection:                                          │
│   • If S > 2, correlations cannot be reproduced classically    │
│   • Eve cannot have predetermined the outcomes                  │
│   • Higher S → more quantum → more secure                      │
│   • S < 2 → abort protocol (possible attack)                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## BBM92 Protocol

BBM92 (Bennett-Brassard-Mermin 1992) is the entanglement-based equivalent of BB84, simpler than E91.

### BBM92 Implementation

```python
class BBM92Protocol:
    """
    BBM92 Protocol - Entanglement-based equivalent of BB84.

    Uses only two measurement bases (like BB84) and relies on
    correlation statistics rather than Bell inequality.
    """

    def __init__(self, n_pairs: int, source_fidelity: float = 1.0):
        self.n_pairs = n_pairs
        self.source = EntanglementSource(
            fidelity=source_fidelity,
            state=BellState.PHI_PLUS
        )

        # Two bases: rectilinear (0°) and diagonal (45°)
        self.bases = [0, np.pi/4]

        self.results = []

    def run_protocol(self) -> Dict:
        """Execute BBM92 protocol."""
        # Step 1: Generate pairs and measure
        for i in range(self.n_pairs):
            pair = self.source.generate_pair()

            # Random basis selection (2 choices each)
            alice_basis = np.random.choice(self.bases)
            bob_basis = np.random.choice(self.bases)

            # Measure
            alice_result, bob_result = self.source.measure_correlation(
                pair, alice_basis, bob_basis
            )

            self.results.append({
                'alice_basis': alice_basis,
                'bob_basis': bob_basis,
                'alice_result': alice_result,
                'bob_result': bob_result
            })

        # Step 2: Sift - keep matching bases
        sifted = [r for r in self.results
                  if r['alice_basis'] == r['bob_basis']]

        alice_key = [r['alice_result'] for r in sifted]
        bob_key = [r['bob_result'] for r in sifted]

        # For |Φ+⟩, Alice and Bob get same results
        qber = sum(a != b for a, b in zip(alice_key, bob_key)) / len(alice_key) if alice_key else 0

        return {
            'total_pairs': self.n_pairs,
            'sifted_pairs': len(sifted),
            'sifting_efficiency': len(sifted) / self.n_pairs,
            'alice_key': alice_key,
            'bob_key': bob_key,
            'qber': qber,
            'secure': qber < 0.11
        }
```

### DI-QKD Framework

```python
class DeviceIndependentQKD:
    """
    Device-Independent QKD framework.

    Security relies ONLY on:
    1. Observed Bell inequality violation
    2. No information leakage between devices
    3. Proper random number generation for inputs
    """

    def __init__(self, n_rounds: int):
        self.n_rounds = n_rounds
        self.test_fraction = 0.1  # Fraction used for Bell test
        self.results = []

    def abstract_device_interaction(self,
                                    input_x: int,
                                    input_y: int) -> Tuple[int, int]:
        """
        Abstract device interaction - we treat devices as black boxes.

        Args:
            input_x: Alice's input (measurement setting choice)
            input_y: Bob's input (measurement setting choice)

        Returns:
            Tuple of (output_a, output_b) - measurement outcomes
        """
        # In real DI-QKD, this would be physical devices
        # Here we simulate ideal quantum behavior
        # Assuming |Φ+⟩ state and CHSH-optimal angles

        angles = {
            0: 0,           # a₁ = 0
            1: np.pi/4,     # a₂ = π/4
        }

        bob_angles = {
            0: np.pi/8,     # b₁ = π/8
            1: -np.pi/8,    # b₂ = -π/8
        }

        alice_angle = angles[input_x]
        bob_angle = bob_angles[input_y]

        # Quantum correlation for |Φ+⟩
        correlation_prob = np.cos((alice_angle - bob_angle)) ** 2

        output_a = np.random.randint(0, 2)
        if np.random.random() < correlation_prob:
            output_b = output_a
        else:
            output_b = 1 - output_a

        return output_a, output_b

    def run_protocol(self) -> Dict:
        """Execute DI-QKD protocol."""
        key_bits_alice = []
        key_bits_bob = []
        test_results = []

        for i in range(self.n_rounds):
            # Random input selection
            x = np.random.randint(0, 2)
            y = np.random.randint(0, 2)

            # Get outputs from devices
            a, b = self.abstract_device_interaction(x, y)

            # Decide: test round or key round
            if np.random.random() < self.test_fraction:
                # Test round - record for Bell test
                test_results.append({
                    'x': x, 'y': y, 'a': a, 'b': b
                })
            else:
                # Key round - use only when x=y=0
                if x == 0 and y == 0:
                    key_bits_alice.append(a)
                    key_bits_bob.append(b)

        # Compute Bell value from test rounds
        bell_value = self._compute_bell_value(test_results)

        # Security threshold
        min_bell_for_security = 2.0 + 0.01  # Just above classical

        return {
            'total_rounds': self.n_rounds,
            'test_rounds': len(test_results),
            'key_rounds': len(key_bits_alice),
            'bell_value': bell_value,
            'secure': bell_value > min_bell_for_security,
            'key_alice': key_bits_alice[:100],  # Sample
            'key_bob': key_bits_bob[:100],
            'raw_key_rate': len(key_bits_alice) / self.n_rounds
        }

    def _compute_bell_value(self, test_results: List[Dict]) -> float:
        """Compute CHSH value from test results."""
        correlators = {}

        for r in test_results:
            key = (r['x'], r['y'])
            if key not in correlators:
                correlators[key] = {'same': 0, 'diff': 0}

            if r['a'] == r['b']:
                correlators[key]['same'] += 1
            else:
                correlators[key]['diff'] += 1

        def E(x, y):
            if (x, y) not in correlators:
                return 0
            c = correlators[(x, y)]
            total = c['same'] + c['diff']
            if total == 0:
                return 0
            return (c['same'] - c['diff']) / total

        # CHSH: S = E(0,0) - E(0,1) + E(1,0) + E(1,1)
        S = abs(E(0, 0) - E(0, 1) + E(1, 0) + E(1, 1))

        return S

    @staticmethod
    def di_key_rate(bell_violation: float, detection_efficiency: float) -> float:
        """
        Calculate DI-QKD key rate from Bell violation.

        The key rate depends on how much the Bell value exceeds 2.
        """
        if bell_violation <= 2:
            return 0

        # Simplified key rate formula
        # Full formula involves min-entropy bounds
        excess = bell_violation - 2
        max_excess = 2 * np.sqrt(2) - 2

        # Rough estimate of secret fraction
        secret_fraction = (excess / max_excess) ** 2

        return detection_efficiency * secret_fraction * 0.5
```

```python
    def compare_key_rate(self, distance_km: float,
                        fiber_loss_db_per_km: float = 0.2) -> Dict[str, float]:
        """Compare key rates at given distance."""
        total_loss_db = fiber_loss_db_per_km * distance_km
        transmission = 10 ** (-total_loss_db / 10)

        rates = {}
        base_rate = 1e9  # 1 GHz source

        for name, props in self.protocols.items():
            if props['loss_scaling'] == 'linear':
                detected = base_rate * transmission
            else:  # quadratic
                detected = base_rate * transmission ** 2

            sifted = detected * props['sifting']
            # Assume 3% QBER for comparison
            secret_fraction = 0.5 if 0.03 < props['qber_threshold'] else 0
            rates[name] = sifted * secret_fraction

        return rates

    def recommend_protocol(self, scenario: Dict) -> str:
        """Recommend protocol based on scenario."""
        distance = scenario.get('distance_km', 50)
        trust_level = scenario.get('trust_level', 'normal')
        priority = scenario.get('priority', 'rate')

        if trust_level == 'none':
            return 'DI-QKD'
        elif trust_level == 'source_untrusted':
            return 'E91'
        elif priority == 'rate':
            if distance > 100:
                return 'BB84'
            else:
                return 'BB84'  # Higher rate
        elif priority == 'security':
            return 'E91'
        else:
            return 'BB84'
```

## Summary

Entanglement-based QKD protocols offer unique security advantages through Bell inequality testing:

- **E91**: Uses Bell violation to verify quantum correlations and detect eavesdropping
- **BBM92**: Simpler entanglement protocol, equivalent to BB84 in key generation
- **Device-Independent**: Ultimate security - even untrusted devices cannot compromise the key
- **Tradeoffs**: Lower key rates due to quadratic loss scaling, but stronger security guarantees

The choice between entanglement and prepare-measure depends on trust assumptions, distance, and required security level.
