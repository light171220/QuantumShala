# BB84 Protocol Basics

The BB84 protocol, introduced by Charles Bennett and Gilles Brassard in 1984, is the cornerstone of quantum key distribution. It was the first QKD protocol ever proposed and remains the most widely implemented today. This lesson provides a deep dive into the fundamental principles and operations of BB84.

## The Four BB84 States

BB84 uses four quantum states organized in two conjugate (mutually unbiased) bases.

### State Definitions

```
┌─────────────────────────────────────────────────────────────────┐
│                   BB84 State Space                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   RECTILINEAR BASIS (Z-basis, ⊕)                               │
│   ┌─────────────────────────────────────────┐                  │
│   │                                         │                  │
│   │     |0⟩ = (1)     |1⟩ = (0)            │                  │
│   │          (0)          (1)              │                  │
│   │                                         │                  │
│   │     ↑              ↓                   │                  │
│   │   bit 0          bit 1                 │                  │
│   │                                         │                  │
│   │   Polarization: Horizontal (H) / Vertical (V)             │
│   │   or: 0°/90° linear polarization       │                  │
│   └─────────────────────────────────────────┘                  │
│                                                                 │
│   DIAGONAL BASIS (X-basis, ⊗)                                  │
│   ┌─────────────────────────────────────────┐                  │
│   │                                         │                  │
│   │     |+⟩ = 1/√2 (1)    |-⟩ = 1/√2 ( 1)  │                  │
│   │              (1)               (-1)    │                  │
│   │                                         │                  │
│   │     ↗                ↘                 │                  │
│   │   bit 0            bit 1               │                  │
│   │                                         │                  │
│   │   Polarization: +45° / -45° linear     │                  │
│   └─────────────────────────────────────────┘                  │
│                                                                 │
│   Key relationships:                                            │
│   |+⟩ = (|0⟩ + |1⟩)/√2                                         │
│   |-⟩ = (|0⟩ - |1⟩)/√2                                         │
│                                                                 │
│   ⟨0|+⟩ = ⟨0|-⟩ = ⟨1|+⟩ = 1/√2                                │
│   ⟨1|-⟩ = -1/√2                                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Mathematical Representation

```python
import numpy as np
from dataclasses import dataclass
from typing import List, Tuple, Dict, Optional
from enum import Enum

class Basis(Enum):
    """BB84 measurement bases."""
    Z = 0  # Rectilinear (computational)
    X = 1  # Diagonal (Hadamard)

class BB84State:
    """
    Represents a BB84 quantum state.

    The four states are:
    |0⟩, |1⟩ (Z-basis)
    |+⟩, |-⟩ (X-basis)
    """

    def __init__(self, bit: int, basis: Basis):
        """
        Create a BB84 state.

        Args:
            bit: The encoded bit value (0 or 1)
            basis: The encoding basis (Z or X)
        """
        self.bit = bit
        self.basis = basis
        self.state_vector = self._compute_state_vector()

    def _compute_state_vector(self) -> np.ndarray:
        """Compute the quantum state vector."""
        if self.basis == Basis.Z:
            if self.bit == 0:
                return np.array([1, 0], dtype=complex)  # |0⟩
            else:
                return np.array([0, 1], dtype=complex)  # |1⟩
        else:  # X basis
            if self.bit == 0:
                return np.array([1, 1], dtype=complex) / np.sqrt(2)  # |+⟩
            else:
                return np.array([1, -1], dtype=complex) / np.sqrt(2)  # |-⟩

    def measure(self, measurement_basis: Basis) -> int:
        """
        Measure the state in given basis.

        Returns:
            Measured bit value (0 or 1)
        """
        if measurement_basis == self.basis:
            # Matching basis → deterministic result
            return self.bit

        # Mismatched basis → random result
        # Probability of each outcome is 50%
        return np.random.randint(0, 2)

    def overlap_probability(self, other: 'BB84State') -> float:
        """
        Calculate |⟨ψ|φ⟩|² - probability of measuring other state.
        """
        inner_product = np.vdot(self.state_vector, other.state_vector)
        return abs(inner_product) ** 2

    @property
    def label(self) -> str:
        """Human-readable state label."""
        labels = {
            (0, Basis.Z): '|0⟩',
            (1, Basis.Z): '|1⟩',
            (0, Basis.X): '|+⟩',
            (1, Basis.X): '|-⟩'
        }
        return labels[(self.bit, self.basis)]

    def bloch_coordinates(self) -> Tuple[float, float, float]:
        """
        Return Bloch sphere coordinates (x, y, z).
        """
        coords = {
            (0, Basis.Z): (0, 0, 1),   # North pole
            (1, Basis.Z): (0, 0, -1),  # South pole
            (0, Basis.X): (1, 0, 0),   # +X
            (1, Basis.X): (-1, 0, 0)   # -X
        }
        return coords[(self.bit, self.basis)]

# Demonstrate state relationships
states = [
    BB84State(0, Basis.Z),  # |0⟩
    BB84State(1, Basis.Z),  # |1⟩
    BB84State(0, Basis.X),  # |+⟩
    BB84State(1, Basis.X)   # |-⟩
]

print("BB84 States and Overlaps:")
print("-" * 40)
for s1 in states:
    for s2 in states:
        prob = s1.overlap_probability(s2)
        print(f"⟨{s1.label}|{s2.label}⟩|² = {prob:.3f}")
```

## Complete Protocol Workflow

The BB84 protocol consists of several distinct phases.

### Protocol Overview

```
┌─────────────────────────────────────────────────────────────────┐
│                  BB84 Protocol Phases                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Phase 1: QUANTUM TRANSMISSION                                 │
│   ─────────────────────────────                                 │
│   Alice                              Bob                        │
│   ┌───────────┐                     ┌───────────┐              │
│   │ Random    │                     │ Random    │              │
│   │ bits: b_i │                     │ bases: β_j│              │
│   │ bases: α_i│                     │           │              │
│   └─────┬─────┘                     └─────┬─────┘              │
│         │                                 │                     │
│         │    Prepare |ψ(b_i, α_i)⟩       │                     │
│         │ ════════════════════════════►  │                     │
│         │         Quantum Channel         │ Measure in β_j     │
│         │                                 │ → result r_j       │
│                                                                 │
│   Phase 2: SIFTING                                              │
│   ────────────────                                              │
│         │     "I used bases: α_1, α_2, ..."                    │
│         │ ◄───────────────────────────────                     │
│         │     "Keep indices where α_i = β_j"                   │
│         │ ───────────────────────────────►                     │
│         │                                 │                     │
│   Keep bits where bases matched                                 │
│   → Sifted key (~50% of original)                              │
│                                                                 │
│   Phase 3: PARAMETER ESTIMATION                                 │
│   ─────────────────────────────                                 │
│   Sacrifice subset to estimate QBER                            │
│   If QBER > threshold → ABORT                                  │
│                                                                 │
│   Phase 4: ERROR CORRECTION                                     │
│   ─────────────────────────                                     │
│   Correct remaining errors                                      │
│   (CASCADE, LDPC, etc.)                                        │
│                                                                 │
│   Phase 5: PRIVACY AMPLIFICATION                                │
│   ────────────────────────────                                  │
│   Hash to remove Eve's information                             │
│   → Final secure key                                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Complete Implementation

```python
class BB84Protocol:
    """
    Complete BB84 protocol implementation.
    """

    def __init__(self, n_qubits: int, noise_level: float = 0.0):
        """
        Initialize BB84 protocol.

        Args:
            n_qubits: Number of qubits to transmit
            noise_level: Probability of bit flip (simulates channel noise)
        """
        self.n_qubits = n_qubits
        self.noise_level = noise_level

        # Protocol data
        self.alice_bits = None
        self.alice_bases = None
        self.bob_bases = None
        self.bob_results = None
        self.sifted_alice = None
        self.sifted_bob = None

    def run_full_protocol(self) -> Dict:
        """Execute complete BB84 protocol."""
        # Phase 1: Quantum transmission
        self._alice_prepare()
        self._bob_measure()

        # Phase 2: Sifting
        self._sift_keys()

        # Phase 3: Parameter estimation
        qber, sample_size = self._estimate_qber()

        # Check security threshold
        secure = qber < 0.11

        return {
            'n_transmitted': self.n_qubits,
            'n_sifted': len(self.sifted_alice),
            'sifting_efficiency': len(self.sifted_alice) / self.n_qubits,
            'qber': qber,
            'sample_size': sample_size,
            'secure': secure,
            'raw_key_alice': self.sifted_alice[sample_size:],
            'raw_key_bob': self.sifted_bob[sample_size:]
        }

    def _alice_prepare(self):
        """Alice prepares random bits with random bases."""
        # Random bits
        self.alice_bits = np.random.randint(0, 2, self.n_qubits)

        # Random bases
        self.alice_bases = np.random.choice(
            [Basis.Z, Basis.X], self.n_qubits
        )

        # Prepare quantum states
        self.alice_states = [
            BB84State(bit, basis)
            for bit, basis in zip(self.alice_bits, self.alice_bases)
        ]

    def _bob_measure(self):
        """Bob measures in random bases."""
        # Random measurement bases
        self.bob_bases = np.random.choice(
            [Basis.Z, Basis.X], self.n_qubits
        )

        # Measure each state
        self.bob_results = []
        for state, basis in zip(self.alice_states, self.bob_bases):
            result = state.measure(basis)

            # Apply channel noise
            if np.random.random() < self.noise_level:
                result = 1 - result

            self.bob_results.append(result)

        self.bob_results = np.array(self.bob_results)

    def _sift_keys(self):
        """Keep only bits where bases matched."""
        matching_indices = [
            i for i in range(self.n_qubits)
            if self.alice_bases[i] == self.bob_bases[i]
        ]

        self.sifted_alice = [self.alice_bits[i] for i in matching_indices]
        self.sifted_bob = [self.bob_results[i] for i in matching_indices]

    def _estimate_qber(self, sample_fraction: float = 0.1) -> Tuple[float, int]:
        """
        Estimate QBER using a sample of sifted bits.

        Returns:
            Tuple of (estimated QBER, sample size used)
        """
        n_sifted = len(self.sifted_alice)
        sample_size = max(1, int(n_sifted * sample_fraction))

        # Compare sample
        errors = sum(
            self.sifted_alice[i] != self.sifted_bob[i]
            for i in range(sample_size)
        )

        qber = errors / sample_size if sample_size > 0 else 0

        return qber, sample_size
```

## Why BB84 Works

The security of BB84 relies on fundamental quantum properties.

### Security Intuition

```
┌─────────────────────────────────────────────────────────────────┐
│                  BB84 Security Foundation                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. NO-CLONING THEOREM                                         │
│   ──────────────────────                                        │
│   Eve cannot perfectly copy an unknown quantum state.           │
│                                                                 │
│   If Eve tries to copy: |ψ⟩|blank⟩ → |ψ⟩|ψ⟩                   │
│   This is IMPOSSIBLE for arbitrary unknown |ψ⟩                 │
│                                                                 │
│   Implication: Eve must measure to gain information             │
│                                                                 │
│   2. MEASUREMENT DISTURBANCE                                    │
│   ───────────────────────────                                   │
│   Measuring in the wrong basis disturbs the state.              │
│                                                                 │
│   Example: |+⟩ measured in Z-basis                             │
│   ├── 50% chance: collapse to |0⟩ → Wrong if Bob uses X       │
│   └── 50% chance: collapse to |1⟩ → Wrong if Bob uses X       │
│                                                                 │
│   3. CONJUGATE BASES                                            │
│   ─────────────────────                                         │
│   Z and X bases are "maximally complementary"                   │
│                                                                 │
│   Knowing Z-value gives NO information about X-value            │
│   ⟨0|+⟩ = ⟨0|-⟩ = ⟨1|+⟩ = ⟨1|-⟩ (in absolute value)          │
│                                                                 │
│   4. EVE'S DILEMMA                                              │
│   ─────────────────                                             │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ Eve doesn't know which basis Alice used                 │  │
│   │                                                         │  │
│   │ If Eve measures in Z:                                   │  │
│   │   • Correct for Z-encoded → learns bit, no disturbance  │  │
│   │   • Wrong for X-encoded → 50% error introduced         │  │
│   │                                                         │  │
│   │ Since bases are random, Eve causes ~25% QBER            │  │
│   │ if she intercepts all qubits                            │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   DETECTION: If QBER > 11%, Alice and Bob abort                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Information-Disturbance Analysis

```python
class SecurityAnalysis:
    """Analyze BB84 security properties."""

    @staticmethod
    def eve_intercept_resend(intercept_fraction: float) -> Dict:
        """
        Analyze intercept-resend attack.

        When Eve intercepts a qubit:
        - 50% chance she uses correct basis → no error
        - 50% chance wrong basis → 50% error when Bob measures in Alice's basis
        - Overall: 25% error per intercepted qubit
        """
        # QBER introduced
        qber = intercept_fraction * 0.25

        # Eve's information (fraction of key she knows)
        # She learns bit when guessing basis correctly (50% of intercepts)
        eve_info = intercept_fraction * 0.5

        # Detection probability (for n sample bits)
        def detection_prob(n_sample: int) -> float:
            if qber == 0:
                return 0
            # Probability of detecting at least one error
            return 1 - (1 - qber) ** n_sample

        return {
            'intercept_fraction': intercept_fraction,
            'qber': qber,
            'eve_information': eve_info,
            'detection_100_samples': detection_prob(100),
            'detection_1000_samples': detection_prob(1000)
        }

    @staticmethod
    def no_cloning_proof():
        """
        Demonstrate no-cloning theorem.

        Suppose cloning were possible:
        U|ψ⟩|blank⟩ = |ψ⟩|ψ⟩

        For |0⟩: U|0⟩|0⟩ = |0⟩|0⟩
        For |1⟩: U|1⟩|0⟩ = |1⟩|1⟩
        For |+⟩: U|+⟩|0⟩ = |+⟩|+⟩

        But |+⟩ = (|0⟩ + |1⟩)/√2, so by linearity:
        U|+⟩|0⟩ = (|0⟩|0⟩ + |1⟩|1⟩)/√2 ≠ |+⟩|+⟩

        Contradiction! Cloning is impossible.
        """
        explanation = """
        NO-CLONING THEOREM PROOF SKETCH

        1. Assume a universal cloning operation U exists

        2. For computational basis states:
           U|0⟩|b⟩ = |0⟩|0⟩
           U|1⟩|b⟩ = |1⟩|1⟩

        3. For superposition |+⟩ = (|0⟩ + |1⟩)/√2:
           By linearity: U|+⟩|b⟩ = (|00⟩ + |11⟩)/√2

        4. But correct cloning requires:
           U|+⟩|b⟩ = |+⟩|+⟩ = (|00⟩ + |01⟩ + |10⟩ + |11⟩)/2

        5. These are different states! Contradiction.

        6. Therefore, no universal cloning operation exists.

        IMPLICATION FOR BB84:
        Eve cannot copy qubits to measure later.
        She must measure immediately, disturbing the state.
        """
        return explanation

# Demonstrate security analysis
for fraction in [0.0, 0.1, 0.25, 0.5, 1.0]:
    result = SecurityAnalysis.eve_intercept_resend(fraction)
    print(f"Intercept {fraction*100:.0f}%: QBER={result['qber']*100:.1f}%, "
          f"Eve knows {result['eve_information']*100:.1f}%, "
          f"Detect(100)={result['detection_100_samples']*100:.1f}%")
```

## Summary

BB84 is the foundational QKD protocol with these key features:

- **Four states**: |0⟩, |1⟩ (Z-basis) and |+⟩, |-⟩ (X-basis)
- **Random bases**: Both Alice and Bob choose bases randomly
- **Sifting**: Keep ~50% where bases matched
- **Security**: Based on no-cloning and measurement disturbance
- **Threshold**: QBER < 11% for secure key generation

Understanding BB84 is essential for all advanced QKD concepts.
