# Key Sifting

Sifting is the process of extracting the raw key from quantum measurements by identifying which bits can be reliably correlated between Alice and Bob. This critical phase bridges quantum transmission and classical post-processing.

## Sifting Protocol

The sifting process extracts usable key bits from quantum measurements.

### Protocol Steps

```
┌─────────────────────────────────────────────────────────────────┐
│                    Sifting Protocol                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   STEP 1: Quantum Transmission Complete                        │
│   ─────────────────────────────────────                         │
│   Alice has: bits b₁,...,bₙ and bases α₁,...,αₙ               │
│   Bob has: results r₁,...,rₙ and bases β₁,...,βₙ              │
│                                                                 │
│   STEP 2: Basis Announcement                                    │
│   ───────────────────────────                                   │
│   Alice ──────────────────────────────────────────► Bob         │
│          "My bases: α₁, α₂, α₃, ..."                           │
│          (publicly broadcast, authenticated)                    │
│                                                                 │
│   STEP 3: Basis Comparison                                      │
│   ────────────────────────                                      │
│   Bob compares αᵢ with βᵢ for all i                            │
│                                                                 │
│   Bob ──────────────────────────────────────────── Alice       │
│          "Keep positions where αᵢ = βᵢ: 1, 4, 5, 8, ..."       │
│                                                                 │
│   STEP 4: Key Extraction                                        │
│   ──────────────────────                                        │
│   Both keep only matching-basis bits                            │
│                                                                 │
│   Alice's sifted key: b₁, b₄, b₅, b₈, ...                      │
│   Bob's sifted key: r₁, r₄, r₅, r₈, ...                        │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ SECURITY NOTE:                                          │  │
│   │ • Bases announced, NOT bit values                       │  │
│   │ • Eve learns which basis, not which state              │  │
│   │ • |0⟩ and |1⟩ both use Z-basis - indistinguishable     │  │
│   │ • Classical channel must be authenticated              │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Implementation

```python
import numpy as np
from typing import List, Tuple, Dict, Optional
from dataclasses import dataclass
from enum import Enum

class Basis(Enum):
    Z = 0
    X = 1

@dataclass
class SiftingResult:
    """Result of sifting protocol."""
    alice_sifted: np.ndarray
    bob_sifted: np.ndarray
    matching_indices: np.ndarray
    n_transmitted: int
    n_sifted: int
    efficiency: float

class SiftingProtocol:
    """
    Complete implementation of BB84 sifting.
    """

    def __init__(self):
        self.alice_bits = None
        self.alice_bases = None
        self.bob_results = None
        self.bob_bases = None

    def set_alice_data(self, bits: np.ndarray, bases: np.ndarray):
        """Set Alice's transmission data."""
        assert len(bits) == len(bases)
        self.alice_bits = bits
        self.alice_bases = bases

    def set_bob_data(self, results: np.ndarray, bases: np.ndarray):
        """Set Bob's measurement data."""
        assert len(results) == len(bases)
        self.bob_results = results
        self.bob_bases = bases

    def announce_bases(self) -> Tuple[np.ndarray, np.ndarray]:
        """
        Simulate classical channel basis announcement.

        In practice, Alice broadcasts her bases.
        Bob receives and compares with his own.
        """
        # Alice announces (public broadcast)
        alice_announcement = self.alice_bases.copy()

        # Bob receives and notes his bases
        bob_announcement = self.bob_bases.copy()

        return alice_announcement, bob_announcement

    def compare_bases(self) -> np.ndarray:
        """
        Find positions where bases match.

        Returns array of matching indices.
        """
        matching = (self.alice_bases == self.bob_bases)
        return np.where(matching)[0]

    def sift(self) -> SiftingResult:
        """
        Execute complete sifting protocol.
        """
        # Step 1: Basis announcement (already done in practice)
        # Step 2: Find matching positions
        matching_indices = self.compare_bases()

        # Step 3: Extract sifted keys
        alice_sifted = self.alice_bits[matching_indices]
        bob_sifted = self.bob_results[matching_indices]

        # Calculate statistics
        n_transmitted = len(self.alice_bits)
        n_sifted = len(matching_indices)
        efficiency = n_sifted / n_transmitted

        return SiftingResult(
            alice_sifted=alice_sifted,
            bob_sifted=bob_sifted,
            matching_indices=matching_indices,
            n_transmitted=n_transmitted,
            n_sifted=n_sifted,
            efficiency=efficiency
        )

    def visualize_sifting(self, n_show: int = 15) -> str:
        """Create visualization of sifting process."""
        lines = []
        lines.append("Sifting Process Visualization")
        lines.append("=" * 70)
        lines.append(f"{'Pos':<5}{'A.Bit':<8}{'A.Base':<10}{'B.Base':<10}{'B.Res':<8}{'Match':<8}{'Keep':<6}")
        lines.append("-" * 70)

        matching_indices = set(self.compare_bases())

        for i in range(min(n_show, len(self.alice_bits))):
            a_bit = self.alice_bits[i]
            a_base = 'Z' if self.alice_bases[i] == Basis.Z.value else 'X'
            b_base = 'Z' if self.bob_bases[i] == Basis.Z.value else 'X'
            b_res = self.bob_results[i]
            match = '✓' if a_base == b_base else '✗'
            keep = 'Yes' if i in matching_indices else 'No'

            lines.append(f"{i:<5}{a_bit:<8}{a_base:<10}{b_base:<10}{b_res:<8}{match:<8}{keep:<6}")

        result = self.sift()
        lines.append("-" * 70)
        lines.append(f"Sifting efficiency: {result.efficiency*100:.1f}%")
        lines.append(f"Sifted bits: {result.n_sifted} / {result.n_transmitted}")

        return "\n".join(lines)


def simulate_sifting(n_qubits: int = 1000) -> SiftingResult:
    """Simulate complete sifting process."""
    # Alice prepares
    alice_bits = np.random.randint(0, 2, n_qubits)
    alice_bases = np.random.randint(0, 2, n_qubits)

    # Bob measures (random bases, matching basis → correct result)
    bob_bases = np.random.randint(0, 2, n_qubits)
    bob_results = np.where(
        alice_bases == bob_bases,
        alice_bits,  # Matching basis → correct
        np.random.randint(0, 2, n_qubits)  # Mismatched → random
    )

    # Execute sifting
    protocol = SiftingProtocol()
    protocol.set_alice_data(alice_bits, alice_bases)
    protocol.set_bob_data(bob_results, bob_bases)

    return protocol.sift()

# Demonstrate
result = simulate_sifting(10000)
print(f"Sifting efficiency: {result.efficiency*100:.1f}%")
print(f"Expected: ~50%")
```

## Sifting Efficiency

Understanding and optimizing sifting efficiency is crucial for QKD performance.

### Efficiency Analysis

```
┌─────────────────────────────────────────────────────────────────┐
│                  Sifting Efficiency Analysis                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   THEORETICAL EFFICIENCY:                                       │
│                                                                 │
│   For BB84 with 2 equally-probable bases:                      │
│   P(match) = P(Alice=Z, Bob=Z) + P(Alice=X, Bob=X)             │
│            = (1/2)(1/2) + (1/2)(1/2)                           │
│            = 1/4 + 1/4                                          │
│            = 1/2 = 50%                                         │
│                                                                 │
│   For 6-state with 3 equally-probable bases:                   │
│   P(match) = 3 × (1/3)(1/3) = 1/3 ≈ 33%                       │
│                                                                 │
│   EFFICIENCY VARIATIONS:                                        │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                         │  │
│   │   Protocol      Bases   P(match)   Efficiency          │  │
│   │   ─────────────────────────────────────────────────     │  │
│   │   BB84           2       1/2        50%                 │  │
│   │   B92            2       1/4        25% (conclusive)    │  │
│   │   6-state        3       1/3        33%                 │  │
│   │   BBM92 (EB)     2       1/2        50%                 │  │
│   │                                                         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   FACTORS AFFECTING EFFICIENCY:                                 │
│   • Number of bases used                                        │
│   • Detection probability (optical losses)                     │
│   • Detector dark counts (false matches)                       │
│   • Multi-photon pulses (may not be detected)                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Efficiency Calculator

```python
class SiftingEfficiencyAnalyzer:
    """Analyze sifting efficiency under various conditions."""

    def __init__(self):
        self.theoretical = {
            'BB84': 0.5,
            'B92': 0.25,
            'Six-State': 1/3,
            'BBM92': 0.5
        }

    def calculate_effective_efficiency(self,
                                       protocol: str,
                                       detection_prob: float,
                                       dark_count_prob: float = 0) -> Dict:
        """
        Calculate effective sifting efficiency including losses.

        Args:
            protocol: Protocol name
            detection_prob: Probability of detecting a transmitted photon
            dark_count_prob: Probability of dark count per detector gate
        """
        base_efficiency = self.theoretical.get(protocol, 0.5)

        # Detected fraction
        signal_detected = detection_prob * base_efficiency

        # Dark counts add noise but also "detections"
        noise_detected = dark_count_prob

        # Effective efficiency
        total_detected = signal_detected + noise_detected
        signal_fraction = signal_detected / total_detected if total_detected > 0 else 0

        return {
            'protocol': protocol,
            'base_efficiency': base_efficiency,
            'detection_probability': detection_prob,
            'effective_efficiency': total_detected,
            'signal_fraction': signal_fraction,
            'noise_fraction': 1 - signal_fraction
        }

    def analyze_distance_dependence(self, protocol: str,
                                    distances_km: List[float],
                                    fiber_loss_db_km: float = 0.2) -> Dict:
        """Analyze how sifting efficiency changes with distance."""
        efficiencies = []
        signal_fractions = []

        for d in distances_km:
            loss_db = fiber_loss_db_km * d
            detection_prob = 10 ** (-loss_db / 10)

            result = self.calculate_effective_efficiency(
                protocol, detection_prob, dark_count_prob=1e-6
            )
            efficiencies.append(result['effective_efficiency'])
            signal_fractions.append(result['signal_fraction'])

        return {
            'distances': distances_km,
            'efficiencies': efficiencies,
            'signal_fractions': signal_fractions
        }

    def optimize_basis_probability(self, qber_estimate: float) -> Dict:
        """
        Some protocols can optimize basis probabilities.

        For standard BB84, equal probabilities are optimal.
        For some scenarios, asymmetric bases can help.
        """
        # Standard equal probability
        equal = {'p_Z': 0.5, 'p_X': 0.5, 'efficiency': 0.5}

        # Note: Asymmetric BB84 uses more Z-basis for key
        # and less X-basis for parameter estimation
        # This doesn't change sifting but affects key rate

        return {
            'standard': equal,
            'asymmetric_available': True,
            'note': 'Asymmetric basis choice optimizes key rate, not sifting'
        }
```

## Parameter Estimation

A fraction of sifted bits is sacrificed to estimate the quantum bit error rate.

### QBER Estimation

```
┌─────────────────────────────────────────────────────────────────┐
│                    Parameter Estimation                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   PURPOSE:                                                      │
│   Estimate QBER to determine if eavesdropping occurred         │
│                                                                 │
│   PROCESS:                                                      │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                         │  │
│   │   1. Select random subset of sifted key                 │  │
│   │      (typically 10-20% of sifted bits)                 │  │
│   │                                                         │  │
│   │   2. Alice reveals her bit values for subset           │  │
│   │      (publicly, over authenticated channel)             │  │
│   │                                                         │  │
│   │   3. Bob compares with his values                       │  │
│   │                                                         │  │
│   │   4. Count errors: QBER = errors / sample_size          │  │
│   │                                                         │  │
│   │   5. Discard revealed bits (Eve now knows them)         │  │
│   │                                                         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   DECISION:                                                     │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                         │  │
│   │   If QBER < threshold (~11% for BB84):                 │  │
│   │      → Proceed to error correction and privacy amp     │  │
│   │                                                         │  │
│   │   If QBER ≥ threshold:                                 │  │
│   │      → ABORT protocol (possible eavesdropping)         │  │
│   │                                                         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   STATISTICAL CONSIDERATIONS:                                   │
│   • Larger sample → more accurate estimate                     │
│   • Smaller sample → more key bits remain                      │
│   • Finite-size effects require confidence intervals           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### QBER Calculator

```python
@dataclass
class QBEREstimate:
    """Result of QBER estimation."""
    observed_qber: float
    sample_size: int
    confidence_interval: Tuple[float, float]
    confidence_level: float
    secure: bool
    threshold: float

class ParameterEstimation:
    """Parameter estimation for BB84."""

    def __init__(self, security_threshold: float = 0.11):
        self.threshold = security_threshold

    def estimate_qber(self, alice_sample: np.ndarray,
                      bob_sample: np.ndarray,
                      confidence: float = 0.99) -> QBEREstimate:
        """
        Estimate QBER with confidence interval.

        Uses Clopper-Pearson exact binomial confidence interval.
        """
        from scipy import stats

        n = len(alice_sample)
        errors = np.sum(alice_sample != bob_sample)
        qber = errors / n

        # Clopper-Pearson confidence interval
        alpha = 1 - confidence
        lower = stats.beta.ppf(alpha/2, errors, n - errors + 1)
        upper = stats.beta.ppf(1 - alpha/2, errors + 1, n - errors)

        # Handle edge cases
        if errors == 0:
            lower = 0
        if errors == n:
            upper = 1

        return QBEREstimate(
            observed_qber=qber,
            sample_size=n,
            confidence_interval=(lower, upper),
            confidence_level=confidence,
            secure=upper < self.threshold,
            threshold=self.threshold
        )

    def optimal_sample_size(self, n_sifted: int,
                            expected_qber: float = 0.03,
                            margin: float = 0.02) -> int:
        """
        Determine optimal sample size for parameter estimation.

        Balance between estimation accuracy and key length.
        """
        from scipy import stats

        # We want: upper_bound of CI < threshold - margin
        target_upper = self.threshold - margin

        for sample_frac in np.arange(0.05, 0.5, 0.01):
            n_sample = int(n_sifted * sample_frac)
            expected_errors = int(n_sample * expected_qber)

            # Upper bound of 99% CI
            upper = stats.beta.ppf(0.995, expected_errors + 1,
                                  n_sample - expected_errors)

            if upper < target_upper:
                return n_sample

        # Default to 20% if nothing works
        return int(n_sifted * 0.2)

    def perform_estimation(self, alice_sifted: np.ndarray,
                          bob_sifted: np.ndarray,
                          sample_fraction: float = 0.1) -> Tuple[QBEREstimate, np.ndarray, np.ndarray]:
        """
        Perform parameter estimation and return remaining key.

        Returns:
            (QBER estimate, remaining Alice key, remaining Bob key)
        """
        n = len(alice_sifted)
        sample_size = max(1, int(n * sample_fraction))

        # Random sample selection
        indices = np.random.permutation(n)
        sample_indices = indices[:sample_size]
        keep_indices = indices[sample_size:]

        # Estimate QBER
        alice_sample = alice_sifted[sample_indices]
        bob_sample = bob_sifted[sample_indices]
        estimate = self.estimate_qber(alice_sample, bob_sample)

        # Remaining key
        alice_remaining = alice_sifted[keep_indices]
        bob_remaining = bob_sifted[keep_indices]

        return estimate, alice_remaining, bob_remaining


class FiniteSizeAnalysis:
    """Analyze finite-size effects on parameter estimation."""

    @staticmethod
    def estimation_uncertainty(n_sample: int, qber: float) -> float:
        """
        Calculate uncertainty in QBER estimate.

        Standard error = sqrt(qber * (1-qber) / n)
        """
        if n_sample == 0:
            return float('inf')
        return np.sqrt(qber * (1 - qber) / n_sample)

    @staticmethod
    def detection_probability(qber: float, n_sample: int,
                              threshold: float = 0.11) -> float:
        """
        Probability of detecting QBER above threshold.

        If true QBER is above threshold, what's probability
        our sample correctly identifies this?
        """
        from scipy import stats

        if qber <= threshold:
            return 0  # Nothing to detect

        # Probability that sample QBER > threshold
        # Using normal approximation for large n
        std_err = np.sqrt(qber * (1 - qber) / n_sample)
        z = (threshold - qber) / std_err

        return 1 - stats.norm.cdf(z)

    @staticmethod
    def required_sample_size(qber: float, threshold: float,
                            detection_prob: float = 0.99) -> int:
        """
        Calculate sample size to detect QBER with given probability.
        """
        from scipy import stats

        if qber <= threshold:
            return float('inf')

        # Required z-score for detection probability
        z_alpha = stats.norm.ppf(detection_prob)

        # Solve for n: (threshold - qber) / sqrt(q(1-q)/n) = -z_alpha
        diff = threshold - qber  # negative
        variance = qber * (1 - qber)

        n = (z_alpha ** 2 * variance) / (diff ** 2)

        return int(np.ceil(n))
```

## Complete Sifting Pipeline

```python
class CompleteSiftingPipeline:
    """Complete sifting pipeline including parameter estimation."""

    def __init__(self, security_threshold: float = 0.11):
        self.sifter = SiftingProtocol()
        self.estimator = ParameterEstimation(security_threshold)
        self.threshold = security_threshold

    def process(self, alice_bits: np.ndarray, alice_bases: np.ndarray,
                bob_results: np.ndarray, bob_bases: np.ndarray,
                pe_fraction: float = 0.1) -> Dict:
        """
        Execute complete sifting pipeline.

        Args:
            alice_bits: Alice's prepared bit values
            alice_bases: Alice's basis choices
            bob_results: Bob's measurement results
            bob_bases: Bob's basis choices
            pe_fraction: Fraction of sifted bits for parameter estimation

        Returns:
            Dictionary with complete pipeline results
        """
        # Set up sifting
        self.sifter.set_alice_data(alice_bits, alice_bases)
        self.sifter.set_bob_data(bob_results, bob_bases)

        # Sift
        sift_result = self.sifter.sift()

        # Parameter estimation
        qber_estimate, alice_key, bob_key = self.estimator.perform_estimation(
            sift_result.alice_sifted,
            sift_result.bob_sifted,
            pe_fraction
        )

        # Decision
        proceed = qber_estimate.secure

        return {
            'n_transmitted': sift_result.n_transmitted,
            'n_sifted': sift_result.n_sifted,
            'sifting_efficiency': sift_result.efficiency,
            'qber': qber_estimate.observed_qber,
            'qber_upper_bound': qber_estimate.confidence_interval[1],
            'pe_sample_size': qber_estimate.sample_size,
            'n_remaining': len(alice_key),
            'proceed': proceed,
            'alice_raw_key': alice_key if proceed else None,
            'bob_raw_key': bob_key if proceed else None,
            'abort_reason': None if proceed else f"QBER {qber_estimate.observed_qber:.3f} too high"
        }

# Full demonstration
np.random.seed(42)
n_qubits = 100000
noise = 0.02  # 2% noise

# Simulate quantum transmission
alice_bits = np.random.randint(0, 2, n_qubits)

bob_bases = np.random.randint(0, 2, n_qubits)

# Bob's results (with noise)
bob_results = np.where(
    alice_bases == bob_bases,
    np.where(np.random.random(n_qubits) < noise,
             1 - alice_bits,  # Error
             alice_bits),     # Correct
    np.random.randint(0, 2, n_qubits)  # Random for mismatched
)

# Run pipeline
pipeline = CompleteSiftingPipeline()
result = pipeline.process(alice_bits, alice_bases, bob_results, bob_bases)

print("Sifting Pipeline Results:")
print(f"  Transmitted: {result['n_transmitted']}")
print(f"  Sifted: {result['n_sifted']} ({result['sifting_efficiency']*100:.1f}%)")
print(f"  QBER: {result['qber']*100:.2f}%")
print(f"  Proceed: {result['proceed']}")
print(f"  Raw key length: {result['n_remaining']}")
```

## Summary

Key sifting is the bridge between quantum transmission and classical post-processing:

- **Process**: Identify matching-basis measurements and extract raw key
- **Efficiency**: ~50% for BB84 (half the bits match)
- **Parameter estimation**: Sacrifice ~10% of sifted bits to estimate QBER
- **Decision**: Abort if QBER exceeds security threshold (~11%)
- **Output**: Raw key ready for error correction and privacy amplification

Proper sifting is essential for extracting secure keys from quantum measurements.
