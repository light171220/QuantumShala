## 1. The Many-Time Signature Problem

One-time signatures are elegant but impractical for most applications.

```python
    REAL-WORLD REQUIREMENTS:

    • Code Signing:        Signs every software update

    • Cryptocurrency:      Signs every transaction
    
    ONE-TIME SIGNATURE PROBLEM:
    
    1. Generate new key pair for EVERY signature
    2. Distribute new public key each time
    3. Trust establishment nightmare

    ┌─────────────────────────────────────────────────────────────────┐
    │  1. Pre-generate 2^h OTS key pairs (leaves)                     │
    │  2. Build binary tree of hashes                                 │
    │  3. Root hash = single public key                               │
    │  4. Sign with OTS, prove membership via authentication path     │

    └─────────────────────────────────────────────────────────────────┘

    """
```

## 2. Merkle Tree Construction

<Definition term="Merkle Tree">
A binary tree where:
- Each **leaf** contains a hash of data (here: OTS public key)
- Each **internal node** contains the hash of its two children
- The **root** is the single hash representing all leaves
</Definition>

```python
class MerkleTree:
    """
    Merkle tree for hash-based signatures.
    
    Tree structure:
                    root
                   /    \
                 n0      n1
                / \     / \
              l0  l1  l2  l3   ← leaves (hashes of OTS public keys)
    """
    
    def __init__(self, height: int, hash_bytes: int = 32):
        """
        Initialize Merkle tree.
        
        Args:
            height: Tree height (supports 2^height signatures)
            hash_bytes: Hash output size in bytes
        """
        self.height = height
        self.num_leaves = 2 ** height
        self.hash_bytes = hash_bytes
        
        # Tree stored as array: [leaves..., level h-1..., ..., root]
        self.tree = [None] * (2 * self.num_leaves - 1)
    
    def _hash(self, data: bytes) -> bytes:
        """Hash function."""
        return hashlib.sha256(data).digest()[:self.hash_bytes]
    
    def _parent_index(self, child_index: int) -> int:
        """Get parent node index."""
        return self.num_leaves - 1 + (child_index - self.num_leaves + 1) // 2
    
    def _sibling_index(self, index: int) -> int:
        """Get sibling node index."""
        if index % 2 == 0:
            return index + 1
        return index - 1
    
    def build(self, leaves: List[bytes]) -> bytes:
        """
        Build Merkle tree from leaf values.
        
        Args:
            leaves: List of 2^h leaf values (hashed OTS public keys)
        
        Returns:
            Root hash (the public key)
        """
        if len(leaves) != self.num_leaves:
            raise ValueError(f"Expected {self.num_leaves} leaves, got {len(leaves)}")
        
        # Set leaves (first 2^h positions)
        for i, leaf in enumerate(leaves):
            self.tree[i] = leaf
        
        # Build tree bottom-up
        # For each level from bottom to root
        for level in range(self.height - 1, -1, -1):
            level_start = 2 ** level - 1
            level_size = 2 ** level
            
            for i in range(level_size):
                node_idx = level_start + i
                left_child = 2 * (node_idx - level_start + 1) + level_start - 1
                
                # For internal nodes, children are at specific positions
                # Simpler: just compute from leaves
                pass
        
        # Alternative: simpler bottom-up construction
        # Level 0 = leaves
        current_level = list(leaves)
        all_levels = [current_level]
        
        # Build each level
        for _ in range(self.height):
            next_level = []
            for i in range(0, len(current_level), 2):
                combined = current_level[i] + current_level[i + 1]
                next_level.append(self._hash(combined))
            current_level = next_level
            all_levels.append(current_level)
        
        # Store in tree array
        idx = 0
        for level in all_levels:
            for node in level:
                self.tree[idx] = node
                idx += 1
        
        self.levels = all_levels
        return all_levels[-1][0]  # Root
    
    def get_authentication_path(self, leaf_index: int) -> List[bytes]:
        """
        Get authentication path for a leaf.
        
        The authentication path contains sibling nodes from leaf to root.
        
        Args:
            leaf_index: Index of leaf (0 to 2^h - 1)
        
        Returns:
            List of h sibling hashes
        """
        path = []
        idx = leaf_index
        
        for level in range(self.height):
            # Get sibling at this level
            sibling_idx = idx ^ 1  # Flip last bit to get sibling
            path.append(self.levels[level][sibling_idx])
            idx //= 2  # Move to parent
        
        return path
    
    def verify_path(self, leaf: bytes, leaf_index: int, 
                    path: List[bytes], root: bytes) -> bool:
        """
        Verify that a leaf is in the tree using authentication path.
        
        Args:
            leaf: The leaf value to verify
            leaf_index: Index of the leaf
            path: Authentication path (h sibling hashes)
            root: Expected root hash
        
        Returns:
            True if verification succeeds
        """
        current = leaf
        idx = leaf_index
        
        for sibling in path:
            if idx % 2 == 0:
                # Current is left child
                current = self._hash(current + sibling)
            else:
                # Current is right child
                current = self._hash(sibling + current)
            idx //= 2
        
        return current == root
```

## 3. Merkle Signature Scheme

```python
class MerkleSignatureScheme:
    """
    Merkle Signature Scheme (MSS).
    
    Combines:
    - One-time signatures (e.g., WOTS) for individual signatures
    - Merkle tree for public key aggregation
    """
    
    def __init__(self, height: int = 10, hash_bytes: int = 32):
        """
        Initialize MSS.
        
        Args:
            height: Tree height (allows 2^height signatures)
            hash_bytes: Security parameter
        """
        self.height = height
        self.num_signatures = 2 ** height
        self.hash_bytes = hash_bytes
        
        # State: track used leaves
        self.next_leaf = 0
        
        # Storage for OTS keys
        self.ots_secret_keys = []
        self.ots_public_keys = []
        
        # Merkle tree
        self.tree = None
        self.root = None
    
    def _hash(self, data: bytes) -> bytes:
        return hashlib.sha256(data).digest()[:self.hash_bytes]
    
    def _generate_ots_keypair(self, seed: bytes) -> Tuple[List[bytes], bytes]:
        """
        Generate simplified OTS keypair from seed.
        
        Returns (secret_key, public_key_hash)
        """
        # Simplified: just random values for demo
        # Real implementation would use WOTS
        sk = [self._hash(seed + i.to_bytes(4, 'big')) for i in range(32)]
        pk_values = [self._hash(s) for s in sk]
        pk_hash = self._hash(b''.join(pk_values))
        return sk, pk_hash
    
    def _ots_sign(self, sk: List[bytes], message: bytes) -> List[bytes]:
        """Simplified OTS sign."""
        msg_hash = self._hash(message)
        signature = []
        for i in range(min(32, len(sk))):
            bit = (msg_hash[i // 8] >> (7 - i % 8)) & 1
            # Simplified: just reveal sk or hash of sk
            signature.append(sk[i] if bit == 1 else self._hash(sk[i]))
        return signature
    
    def _ots_verify(self, pk_hash: bytes, message: bytes, 
                    signature: List[bytes]) -> bool:
        """Simplified OTS verify."""
        msg_hash = self._hash(message)
        pk_values = []
        for i in range(min(32, len(signature))):
            bit = (msg_hash[i // 8] >> (7 - i % 8)) & 1
            if bit == 1:
                pk_values.append(self._hash(signature[i]))
            else:
                pk_values.append(signature[i])
        computed_pk = self._hash(b''.join(pk_values))
        return computed_pk == pk_hash
    
    def keygen(self, seed: bytes = None) -> bytes:
        """
        Generate MSS key pair.
        
        Returns:
            Public key (Merkle root)
        """
        if seed is None:
            seed = secrets.token_bytes(32)
        
        # Generate all OTS key pairs
        print(f"Generating {self.num_signatures} OTS key pairs...")
        
        for i in range(self.num_signatures):
            ots_seed = self._hash(seed + i.to_bytes(4, 'big'))
            sk, pk_hash = self._generate_ots_keypair(ots_seed)
            self.ots_secret_keys.append(sk)
            self.ots_public_keys.append(pk_hash)
        
        # Build Merkle tree
        print("Building Merkle tree...")
        self.tree = MerkleTree(self.height, self.hash_bytes)
        self.root = self.tree.build(self.ots_public_keys)
        
        return self.root
    
    def sign(self, message: bytes) -> Optional[dict]:
        """
        Sign a message.
        
        Returns:
            Signature containing OTS signature, leaf index, and auth path
            Or None if all keys are used
        """
        if self.next_leaf >= self.num_signatures:
            return None  # No more keys available
        
        leaf_idx = self.next_leaf
        self.next_leaf += 1  # Update state
        
        # Get OTS key for this leaf
        ots_sk = self.ots_secret_keys[leaf_idx]
        ots_pk = self.ots_public_keys[leaf_idx]
        
        # Sign with OTS
        ots_sig = self._ots_sign(ots_sk, message)
        
        # Get authentication path
        auth_path = self.tree.get_authentication_path(leaf_idx)
        
        return {
            'leaf_index': leaf_idx,
            'ots_signature': ots_sig,
            'ots_public_key': ots_pk,
            'auth_path': auth_path
        }
    
    def verify(self, public_key: bytes, message: bytes, 
               signature: dict) -> bool:
        """
        Verify an MSS signature.
        
        Args:
            public_key: Merkle root
            message: Signed message
            signature: MSS signature
        
        Returns:
            True if valid
        """
        leaf_idx = signature['leaf_index']
        ots_sig = signature['ots_signature']
        ots_pk = signature['ots_public_key']
        auth_path = signature['auth_path']
        
        # Verify OTS signature
        if not self._ots_verify(ots_pk, message, ots_sig):
            return False
        
        # Verify Merkle path
        return self.tree.verify_path(ots_pk, leaf_idx, auth_path, public_key)


def demonstrate_mss():
    """Demonstrate Merkle Signature Scheme."""
    print("\nMERKLE SIGNATURE SCHEME DEMONSTRATION")
    print("=" * 65)
    
    # Small tree for demo
    mss = MerkleSignatureScheme(height=4)  # 16 signatures
    
    # Key generation
    print("\n1. KEY GENERATION")
    root = mss.keygen()
    print(f"   Public key (root): {root[:16].hex()}...")
    print(f"   Can sign up to {mss.num_signatures} messages")
    
    # Sign multiple messages
    print("\n2. SIGNING MESSAGES")
    messages = [f"Message {i}".encode() for i in range(3)]
    signatures = []
    
    for msg in messages:
        sig = mss.sign(msg)
        signatures.append((msg, sig))
        print(f"   Signed: {msg.decode()}, leaf index: {sig['leaf_index']}")
    
    # Verify
    print("\n3. VERIFICATION")
    for msg, sig in signatures:
        valid = mss.verify(root, msg, sig)
        print(f"   {msg.decode()}: {'✓ Valid' if valid else '✗ Invalid'}")
    
    # Show state
    print(f"\n4. STATE")
    print(f"   Signatures used: {mss.next_leaf}/{mss.num_signatures}")
    print(f"   Remaining: {mss.num_signatures - mss.next_leaf}")

demonstrate_mss()
```

## 4. State Management: The Critical Challenge

```python
def explain_state_management():
    """Explain state management challenges."""
    print("\nSTATE MANAGEMENT IN MSS")
    print("=" * 65)
    
    explanation = """
    THE STATE PROBLEM:
    
    MSS MUST track which OTS keys have been used.
    
    If the same OTS key is used twice:
    ┌─────────────────────────────────────────────────────────────────┐
    │  Two signatures reveal different parts of the OTS secret key   │
    │  Attacker can combine them to forge signatures                 │
    │  COMPLETE SECURITY BREAKDOWN                                   │
    └─────────────────────────────────────────────────────────────────┘
    
    STATE MANAGEMENT REQUIREMENTS:
    
    1. PERSISTENCE: State must survive crashes/reboots
       - Write state to non-volatile storage BEFORE signing
       - Use atomic operations if possible
    
    2. BACKUP PREVENTION: Never restore from backup
       - Old state + new signatures = potential reuse
       - State must be FORWARD-ONLY
    
    3. SYNCHRONIZATION: Multi-device signing is dangerous
       - Same key on two devices = certain reuse
       - Need distributed state management
    
    4. RECOVERY: What if state is lost?
       - Conservative: Assume all keys used, generate new tree
       - Risky: Try to determine which were used (error-prone)
    
    SOLUTIONS:
    
    1. Hardware Security Modules (HSMs)
       - Dedicated hardware manages state
       - Tamper-resistant storage
    
    2. Stateless Signatures (SPHINCS+)
       - No state to manage
       - Larger signatures, but no reuse risk
    
    3. Hierarchical Schemes (XMSS^MT)
       - Multiple smaller trees
       - State only for currently active tree
    """
    print(explanation)
    
    # Demonstrate state problem
    print("\nSTATE REUSE ATTACK DEMONSTRATION:")
    print("-" * 50)
    
    # Simplified OTS for demo
    def simple_ots_sign(sk, msg_hash):
        """Return sk[i] if bit i is 1, else hash(sk[i])."""
        return [(sk[i] if (msg_hash[i//8] >> (7-i%8)) & 1 else 
                hashlib.sha256(sk[i]).digest()[:32]) for i in range(32)]
    
    # Generate OTS key
    sk = [secrets.token_bytes(32) for _ in range(32)]
    
    # Sign two messages (REUSE - BAD!)
    msg1 = b"Transfer $100 to Alice"
    msg2 = b"Transfer $999 to Eve"
    
    h1 = hashlib.sha256(msg1).digest()
    h2 = hashlib.sha256(msg2).digest()
    
    sig1 = simple_ots_sign(sk, h1)
    sig2 = simple_ots_sign(sk, h2)
    
    # Attacker analysis
    both_known = 0
    for i in range(32):
        bit1 = (h1[i//8] >> (7-i%8)) & 1
        bit2 = (h2[i//8] >> (7-i%8)) & 1
        if bit1 != bit2:
            both_known += 1
    
    print(f"After signing two messages:")
    print(f"  Positions where attacker knows BOTH sk values: {both_known}/32")
    print(f"  Positions needing luck for forgery: {32 - both_known}/32")
    
    # After many reuses
    print(f"\nAfter 10 signatures on random messages:")
    print(f"  Expected positions fully exposed: ~32 (1 - 2*(0.5)^10) = ~31.9")
    print(f"  Attacker can forge almost any message!")

explain_state_management()
```

## Summary

In this lesson, you learned about Merkle trees for signatures:

1. **Problem**: OTS can only sign once, need many-time capability
2. **Solution**: Merkle tree over OTS public keys
3. **Construction**: Binary tree of hashes with root as public key
4. **Authentication Path**: Log(n) hashes prove leaf membership
5. **MSS**: Combines OTS + Merkle tree for practical signatures
6. **State Management**: Critical requirement to prevent reuse
7. **Signature Size**: OTS signature + authentication path
8. **Limitations**: Finite signatures (2^h), state management burden
