## 2. Key Metrics and Milestones

```python
def key_metrics():
    """Understanding quantum computing metrics."""
    print("\nQUANTUM COMPUTING METRICS")
    print("=" * 70)
    
    print("""
    IMPORTANT METRICS:
    ━━━━━━━━━━━━━━━━━━
    
    1. NUMBER OF QUBITS
       • Physical qubit count
       • NOT a complete picture of capability
    
    2. QUANTUM VOLUME
       • Combines qubits, connectivity, error rates
       • Higher = more capable
       • IBM's metric, widely adopted
    
    3. ERROR RATES
       • Single-qubit gate error: ~0.1%
       • Two-qubit gate error: ~1%
       • Need: ~0.001% for useful error correction
    
    4. COHERENCE TIME
       • How long qubits maintain quantum state
       • Current: microseconds to milliseconds
       • Need: longer for complex algorithms
    
    5. CIRCUIT DEPTH
       • How many operations before errors dominate
       • Current: hundreds to low thousands
       • Shor's for RSA-2048: millions of gates
    
    
    MILESTONES ACHIEVED:
    ━━━━━━━━━━━━━━━━━━━━━
    
    ┌─────────────────────────────────────────────────────────────────┐
    │ Year │ Milestone                                               │

    │ 2020 │ China photonic quantum advantage                        │

    │ 2022 │ IBM 433-qubit Osprey processor                          │

    │ 2023 │ Google logical qubit breakthrough                       │
    │ 2024 │ Error correction improvements continue                  │

    MILESTONES NEEDED FOR CRYPTO THREAT:
```

```python
def mosca_calculator(migration_years: int, secrecy_years: int, 
                     crqc_years: int) -> str:
    """Calculate if migration should start now."""
    if migration_years + secrecy_years > crqc_years:
        return "⚠️  START MIGRATION NOW!"
    else:
        return "✓ Time available, but plan ahead"

print("\nMOSCA INEQUALITY EXAMPLES:")
print(f"Enterprise (5+15 vs 20): {mosca_calculator(5, 15, 20)}")
print(f"Government (10+30 vs 25): {mosca_calculator(10, 30, 25)}")
print(f"Healthcare (5+50 vs 30): {mosca_calculator(5, 50, 30)}")
```

## Summary

Quantum computing progress update:

1. **Current State**: ~1,000 physical qubits, far from cryptographic threat
2. **Timeline**: Expert estimates range from 2030-2050+
3. **Key Gap**: Need millions of qubits with low error rates
4. **Mosca Inequality**: If migration + secrecy > CRQC timeline, start now
5. **Monitoring**: Track key indicators, adjust plans accordingly
6. **Recommendation**: Begin PQC migration regardless of uncertainty

The prudent approach is to migrate now while there's time.
