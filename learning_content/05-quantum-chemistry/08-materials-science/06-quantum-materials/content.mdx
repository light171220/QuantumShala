## 1. Topological Insulators

<Definition term="Topological Insulator">
A material that is an insulator in its bulk but has conducting surface/edge states protected by time-reversal symmetry, characterized by topological invariants of the bulk band structure.
</Definition>


## 2. Berry Phase and Topology

```python
def berry_phase():
    """Berry phase calculations for topological properties."""

    print("Berry Phase and Topological Invariants")
    print("=" * 60)

    print("\nBerry Connection and Phase:")
    print("-" * 45)
    print("  A_n(k) = i⟨u_nk|∇_k|u_nk⟩  (Berry connection)")
    print()
    print("  γ = ∮ A(k)·dk  (Berry phase around loop)")
    print()
    print("  F(k) = ∇ × A(k)  (Berry curvature)")

    print("\nChern Number:")
    print("-" * 45)
    print("  C = (1/2π) ∫_BZ F(k) d²k")
    print()
    print("  Integer topological invariant")
    print("  C ≠ 0 → Quantum Hall effect")
    print("  C determines number of edge states")

    print("\nWilson Loop Method:")
    print("-" * 45)
    print("  W(k_∥) = P exp[i∮ A_⊥ dk_⊥]")
    print()
    print("  Eigenvalues give Wannier centers")
    print("  Winding = Z₂ invariant")
    print()
    print("  Implemented in: Z2Pack, WannierTools")

    print("\nSpin-Orbit Coupling:")
    print("-" * 45)
    print("  Essential for time-reversal TIs")
    print("  Strong in heavy elements (Bi, Pb, W)")
    print()
    print("  DFT with SOC:")
    print("    • Full relativistic pseudopotentials")
    print("    • Spin-orbit splitting of bands")
    print("    • Required for accurate topology")

berry_phase()
```

<Formula name="Z₂ Invariant from Parity">
$$(-1)^{\nu} = \prod_{i=1}^{N_{TRIM}} \prod_{n=1}^{N_{occ}/2} \xi_{2n}(\Gamma_i)$$

Where $\xi_{2n}$ are parity eigenvalues of occupied Kramers pairs at time-reversal invariant momenta $\Gamma_i$.
</Formula>

## 5. Strongly Correlated Systems

```python
def correlated_systems():
    """Strongly correlated electron systems."""

    print("Strongly Correlated Systems")
    print("=" * 60)

    print("\nWhy DFT Fails:")
    print("-" * 45)
    print("  • Localized d/f electrons")
    print("  • Strong Coulomb repulsion U")
    print("  • Dynamic correlation effects")
    print("  • Metal-insulator transitions")

    print("\nDFT+U Method:")
    print("-" * 45)
    print("  Add Hubbard correction to localized orbitals")
    print()
    print("  E_U = (U-J)/2 Σᵢ nᵢ(1-nᵢ)")
    print()
    print("  Typical U values:")
    print("    3d metals: 3-5 eV")
    print("    4f lanthanides: 5-8 eV")
    print()
    print("  Limitations:")
    print("    • Static mean-field")
    print("    • U is empirical")

    print("\nDMFT (Dynamical Mean Field Theory):")
    print("-" * 45)
    print("  Map lattice → impurity problem")
    print("  Include local dynamics exactly")
    print()
    print("  Self-consistency:")
    print("    G_loc(ω) = ∫ G(k,ω) dk")
    print("    Solve impurity → Σ(ω)")
    print("    Update G(k,ω) = [ω - ε_k - Σ(ω)]⁻¹")

    print("\nApplications:")
    print("-" * 45)
    apps = [
        ("Mott insulators", "NiO, V₂O₃", "Metal-insulator"),
        ("Heavy fermions", "CeCoIn₅", "f-electron"),
        ("Cuprates", "YBCO", "High-Tc SC"),
        ("Nickelates", "NdNiO₂", "Superconducting"),
    ]
    for app, example, phenomenon in apps:
        print(f"  • {app}: {example}")
        print(f"    {phenomenon}")

correlated_systems()
```

<Warning>
DFT+U is a starting point for correlated systems. For quantitative predictions of spectral functions or phase transitions, DMFT or beyond is often necessary but computationally expensive.
</Warning>
