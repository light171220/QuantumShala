## 1. Crystal Structures

<Definition term="Crystal Structure">
A periodic arrangement of atoms in three dimensions, characterized by a lattice (geometric arrangement of points) and a basis (atoms associated with each lattice point).
</Definition>

## 2. Reciprocal Space

```python
def reciprocal_space():
    """Reciprocal lattice and k-space concepts."""

    print("Reciprocal Space")
    print("=" * 60)

    print("\nReciprocal Lattice Vectors:")
    print("-" * 45)
    print("  b₁ = 2π (a₂ × a₃) / (a₁ · (a₂ × a₃))")
    print("  b₂ = 2π (a₃ × a₁) / (a₁ · (a₂ × a₃))")
    print("  b₃ = 2π (a₁ × a₂) / (a₁ · (a₂ × a₃))")
    print()
    print("  Key property: aᵢ · bⱼ = 2πδᵢⱼ")

    print("\nPhysical Meaning:")
    print("-" * 45)
    print("  • Reciprocal space = momentum/wavevector space")
    print("  • Brillouin zone = primitive cell of reciprocal lattice")
    print("  • Crystal momentum: p = ℏk")
    print("  • Diffraction condition: G = k' - k")

    print("\nBrillouin Zone:")
    print("-" * 45)
    print("  First BZ: Wigner-Seitz cell of reciprocal lattice")
    print()
    print("  High-symmetry points:")
    points = [
        ("Γ", "(0, 0, 0)", "Zone center"),
        ("X", "(π/a, 0, 0)", "Face center"),
        ("L", "(π/a, π/a, π/a)", "Diagonal"),
        ("K", "(3π/2a, 3π/2a, 0)", "Edge (FCC)"),
        ("M", "(π/a, π/a, 0)", "Edge"),
    ]
    for symbol, coords, meaning in points:
        print(f"    {symbol}: {coords} - {meaning}")

    # Calculate reciprocal vectors for cubic lattice
    print("\nExample: Simple Cubic")
    print("-" * 45)
    a = 5.0  # Angstrom
    print(f"  Direct lattice constant: a = {a} Å")
    b = 2 * np.pi / a
    print(f"  Reciprocal lattice constant: b = 2π/a = {b:.3f} Å⁻¹")
    print(f"  BZ volume: (2π/a)³ = {b**3:.3f} Å⁻³")

reciprocal_space()
```

<Formula name="Reciprocal Lattice">
$$\mathbf{b}_1 = 2\pi \frac{\mathbf{a}_2 \times \mathbf{a}_3}{\mathbf{a}_1 \cdot (\mathbf{a}_2 \times \mathbf{a}_3)}$$

The reciprocal lattice vector $\mathbf{G} = h\mathbf{b}_1 + k\mathbf{b}_2 + l\mathbf{b}_3$ satisfies $e^{i\mathbf{G}\cdot\mathbf{R}} = 1$ for all lattice vectors $\mathbf{R}$.
</Formula>

## 3. Bloch's Theorem

```python
def bloch_theorem():
    """Bloch's theorem for periodic systems."""

    print("Bloch's Theorem")
    print("=" * 60)

    print("\nStatement:")
    print("-" * 45)
    print("  For a periodic potential V(r + R) = V(r):")
    print()
    print("  ψ_nk(r) = e^(ik·r) u_nk(r)")
    print()
    print("  where u_nk(r + R) = u_nk(r)")
    print()
    print("  n = band index, k = crystal momentum")

    print("\nImplications:")
    print("-" * 45)
    implications = [
        ("Discrete bands", "Continuous E(k) in each band"),
        ("Band structure", "E_n(k) determines properties"),
        ("k-point sampling", "BZ integral approximation"),
        ("Periodicity", "E(k + G) = E(k)"),
    ]
    for impl, meaning in implications:
        print(f"  • {impl}: {meaning}")

    print("\nBloch Wave Properties:")
    print("-" * 45)
    print("  • ψ extends over entire crystal")
    print("  • Probability |ψ|² has lattice periodicity")
    print("  • k labels the state (within 1st BZ)")
    print("  • Crystal momentum ≠ true momentum")

    print("\nFrom Molecules to Solids:")
    print("-" * 45)
    print("  Molecule: Discrete orbitals")
    print("  Cluster:  Many levels → quasi-continuous")
    print("  Solid:    Continuous bands E(k)")
    print()
    print("  N atoms → N states per band")
    print("  Spacing → 0 as N → ∞")

bloch_theorem()
```

## 5. k-Point Sampling

```python
def kpoint_sampling():
    """k-point sampling for Brillouin zone integration."""

    print("k-Point Sampling")
    print("=" * 60)

    print("\nBZ Integration:")
    print("-" * 45)
    print("  Many properties involve BZ integrals:")
    print()
    print("  n = (1/Ω_BZ) ∫_BZ f(E(k)) dk")
    print()
    print("  Numerical: Replace integral with sum")
    print("  n ≈ Σ_k w_k f(E(k))")

    print("\nMonkhorst-Pack Grid:")
    print("-" * 45)
    print("  Uniform grid of k-points")
    print()
    print("  k = (n₁/N₁)b₁ + (n₂/N₂)b₂ + (n₃/N₃)b₃")
    print()
    print("  Grid size: N₁ × N₂ × N₃")
    print("  Symmetry reduces unique k-points")

    print("\nConvergence:")
    print("-" * 45)
    convergence = [
        ("Metal", "Dense grid (12×12×12+)", "Fermi surface sampling"),
        ("Semiconductor", "Medium (6×6×6)", "Smooth bands"),
        ("Insulator", "Coarse (4×4×4)", "Wide gap"),
        ("Surface", "Dense in-plane, 1 normal", "2D periodicity"),
    ]
    print(f"  {'System':<14} {'Typical Grid':<18} {'Reason'}")
    print("  " + "-" * 50)
    for system, grid, reason in convergence:
        print(f"  {system:<14} {grid:<18} {reason}")

    print("\nSmearing Methods:")
    print("-" * 45)
    print("  For metals, sharp Fermi function problematic")
    print()
    print("  • Gaussian smearing: Broadened occupation")
    print("  • Methfessel-Paxton: Polynomial correction")
    print("  • Fermi-Dirac: Physical temperature")
    print()
    print("  Extrapolate σ → 0 for ground state")

kpoint_sampling()
```

<Tip>
Always test k-point convergence! Increase the grid until total energy changes by less than 1 meV/atom. Metals require denser grids than insulators.
</Tip>

## 6. Connecting to Molecular QM

```python
def molecular_to_solid():
    """Connection between molecular QM and solid state."""

    print("From Molecules to Solids")
    print("=" * 60)

    print("\nConceptual Bridge:")
    print("-" * 45)
    mapping = [
        ("MO (LCAO)", "→", "Crystal orbital / Band"),
        ("HOMO/LUMO", "→", "Valence/Conduction band"),
        ("HOMO-LUMO gap", "→", "Band gap"),
        ("Basis set", "→", "Plane waves / Local basis"),
        ("SCF", "→", "DFT-SCF (periodic)"),
    ]
    for mol, arrow, solid in mapping:
        print(f"  {mol:<18} {arrow} {solid}")

    print("\nBasis Set Choices:")
    print("-" * 45)
    print("  Plane waves:")
    print("    ψ = Σ_G c_G e^(i(k+G)·r)")
    print("    + Systematic, unbiased")
    print("    + Easily controllable (cutoff)")
    print("    - Many basis functions needed")
    print()
    print("  Local orbitals:")
    print("    Similar to molecular GTO/STO")
    print("    + Compact, efficient")
    print("    + Chemical intuition")
    print("    - Basis set superposition")

    print("\nPseudopotentials:")
    print("-" * 45)
    print("  Replace core electrons with effective potential")
    print()
    print("  Why?")
    print("    • Core electrons don't participate in bonding")
    print("    • Rapid oscillations near nucleus")
    print("    • Would need huge plane wave cutoff")
    print()
    print("  Types: Norm-conserving, Ultrasoft, PAW")

molecular_to_solid()
```
