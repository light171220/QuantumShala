## 1. Expectation Values in HF

For a converged HF wavefunction, molecular properties are calculated as:

<Formula>
$$\langle\hat{O}\rangle = \langle\Psi_{HF}|\hat{O}|\Psi_{HF}\rangle$$
</Formula>

```python
def expectation_value_concept():
    """Explain property calculation in HF."""

    print("Molecular Properties as Expectation Values")
    print("=" * 60)

    print("\nGeneral Formula:")
    print("  ⟨Ô⟩ = ⟨Ψ_HF|Ô|Ψ_HF⟩")

    print("\nFor One-Electron Operators:")
    print("  ⟨Ô₁⟩ = Σᵢ ⟨χᵢ|Ô|χᵢ⟩ = Tr(P @ O)")
    print("  where O is the operator matrix in AO basis")

    print("\nExamples of One-Electron Properties:")
    properties = [
        ('Kinetic energy', 'T = -½∇²'),
        ('Nuclear attraction', 'V_ne = -Σₐ Zₐ/rₐ'),
        ('Dipole moment', 'μ = -er = -e(x, y, z)'),
        ('Electron density', 'ρ(r) = Σᵢ |χᵢ(r)|²'),
    ]

    for prop, operator in properties:
        print(f"  • {prop}: {operator}")

    print("\nTwo-Electron Properties:")
    print("  ⟨Ô₂⟩ = ½Σᵢⱼ(⟨χᵢχⱼ|Ô|χᵢχⱼ⟩ - ⟨χᵢχⱼ|Ô|χⱼχᵢ⟩)")

expectation_value_concept()
```

## 2. Dipole Moment

<Definition term="Electric Dipole Moment">
A measure of the separation of positive and negative charge in a molecule. Calculated as μ = ⟨Ψ|(-e)r|Ψ⟩ for electrons plus the nuclear contribution.
</Definition>

```python
import numpy as np

def dipole_moment():
    """Calculate molecular dipole moment."""

    print("Dipole Moment Calculation")
    print("=" * 60)

    print("\nFormula:")
    print("  μ = μ_nuclear + μ_electronic")
    print("  μ_nuclear = Σₐ Zₐ Rₐ  (nuclear charges × positions)")
    print("  μ_electronic = -Tr(P @ D)  (density × dipole integrals)")

    print("\nIn AO Basis:")
    print("  Dμν = ⟨χμ|r|χν⟩  (dipole integrals, 3 components)")
    print("  μ = Σₐ Zₐ Rₐ - Σμν Pμν Dμν")

    # Example: H2O
    print("\nExample: H₂O")
    print("-" * 40)

    # Simplified example (actual values)
    mu_nuclear = np.array([0.0, 0.0, 1.2])  # Debye
    mu_electronic = np.array([0.0, 0.0, -2.9])  # Debye

    mu_total = mu_nuclear + mu_electronic
    mu_magnitude = np.linalg.norm(mu_total)

    print(f"  Nuclear contribution: {mu_nuclear}")
    print(f"  Electronic contribution: {mu_electronic}")
    print(f"  Total dipole: {mu_total}")
    print(f"  |μ| = {mu_magnitude:.2f} Debye")
    print(f"  (Experimental: 1.85 Debye)")

dipole_moment()
```

<Tip>
Dipole moments are among the properties HF calculates most accurately—typically within 5-10% of experiment for polar molecules.
</Tip>

## 3. Mulliken Population Analysis

<Definition term="Mulliken Population Analysis">
A method to assign partial charges to atoms by partitioning the electron density according to the overlap between basis functions.
</Definition>

```python
def mulliken_analysis():
    """Explain Mulliken population analysis."""

    print("Mulliken Population Analysis")
    print("=" * 60)

    print("\nTotal Electrons:")
    print("  N = Tr(P @ S) = Σμν Pμν Sμν")

    print("\nPartitioning:")
    print("  Pμν Sμν is shared between atoms A and B")
    print("  Assign half to A and half to B")

    print("\nGross Population on Atom A:")
    print("  qₐ = Σμ∈A (PS)μμ = Σμ∈A Σν Pμν Sνμ")

    print("\nMulliken Charge:")
    print("  Qₐ = Zₐ - qₐ")
    print("  (Nuclear charge minus electron population)")

    # Example
    print("\nExample: H₂O")
    print("-" * 40)
    print("  Gross populations:")
    print("    O: 8.8 electrons")
    print("    H: 0.6 electrons each")
    print("  Mulliken charges:")
    print("    O: -0.8 e")
    print("    H: +0.4 e each")

    print("\nLimitations:")
    print("  • Basis set dependent")
    print("  • Can give unphysical results (negative populations)")
    print("  • Better alternatives: Löwdin, NPA, Hirshfeld")

mulliken_analysis()
```

## 4. Geometry Optimization

Finding equilibrium structures by minimizing energy:

<Formula>
$$\frac{\partial E}{\partial R_A} = 0 \quad \text{for all nuclear coordinates } R_A$$
</Formula>

```python
def geometry_optimization():
    """Explain geometry optimization with HF."""

    print("Geometry Optimization")
    print("=" * 60)

    print("\nGoal: Find geometry where forces are zero")
    print("  ∂E/∂Rₐ = 0 for all coordinates")

    print("\nAnalytical Gradients:")
    print("-" * 45)
    print("  Modern codes compute ∂E/∂R analytically")
    print("  Much faster than numerical (finite difference)")
    print()
    print("  Hellmann-Feynman + Pulay terms:")
    print("  ∂E/∂R = ⟨Ψ|∂Ĥ/∂R|Ψ⟩ + (basis set derivative terms)")

    print("\nOptimization Algorithms:")
    algorithms = [
        ('Steepest Descent', 'Simple but slow', 'Rarely used'),
        ('Conjugate Gradient', 'No Hessian needed', 'Medium speed'),
        ('BFGS', 'Quasi-Newton, builds Hessian', 'Most common'),
        ('Newton-Raphson', 'Exact Hessian', 'Fast near minimum'),
    ]

    print(f"\n{'Algorithm':<20} {'Features':<25} {'Use':<20}")
    print("-" * 65)
    for alg, feat, use in algorithms:
        print(f"{alg:<20} {feat:<25} {use:<20}")

    print("\nTypical Workflow:")
    print("  1. Start from guess geometry")
    print("  2. Compute energy and gradient")
    print("  3. Take step (BFGS direction)")
    print("  4. Check convergence criteria")
    print("  5. Repeat until converged")

geometry_optimization()
```

## 5. Vibrational Frequencies

<Definition term="Harmonic Frequencies">
The normal mode frequencies obtained from the eigenvalues of the mass-weighted Hessian matrix at a stationary point. Positive frequencies indicate a minimum, imaginary frequencies indicate a transition state.
</Definition>

```python
def vibrational_analysis():
    """Explain vibrational frequency calculation."""

    print("Vibrational Frequency Analysis")
    print("=" * 60)

    print("\nHessian Matrix:")
    print("  Hᵢⱼ = ∂²E / ∂Rᵢ∂Rⱼ")
    print("  Second derivatives of energy w.r.t. coordinates")

    print("\nMass-Weighted Hessian:")
    print("  H̃ᵢⱼ = Hᵢⱼ / √(Mᵢ Mⱼ)")
    print("  M is atomic mass")

    print("\nDiagonalization → Normal Modes:")
    print("  H̃ L = L Λ")
    print("  Λ = diagonal matrix of ω²")
    print("  L = normal mode vectors")

    print("\nFrequencies:")
    print("  ω (cm⁻¹) = √(eigenvalue) × conversion factor")
    print("  Real ω → stable minimum")
    print("  Imaginary ω → saddle point / transition state")

    print("\nHF Frequency Accuracy:")
    print("-" * 45)
    print("  • Typically 5-10% too high")
    print("  • Scale factors often applied: ~0.89-0.93")
    print("  • Anharmonic corrections improve accuracy")

vibrational_analysis()
```

## 6. Response Properties

<Definition term="Response Property">
A molecular property that describes how the system responds to an external perturbation, such as an electric field. Examples include polarizability, magnetic susceptibility, and NMR shieldings.
</Definition>


## 8. Population Analysis Methods

```python
def population_methods():
    """Compare different population analysis methods."""

    print("Population Analysis Methods")
    print("=" * 60)

    methods = {
        'Mulliken': {
            'idea': 'Partition PS matrix by atoms',
            'pros': 'Simple, fast',
            'cons': 'Basis set dependent, can be negative'
        },
        'Löwdin': {
            'idea': 'Symmetric orthogonalization first',
            'pros': 'Less basis set sensitive',
            'cons': 'Still can give odd results'
        },
        'NPA (Natural)': {
            'idea': 'Based on natural atomic orbitals',
            'pros': 'Physical, basis-set stable',
            'cons': 'More complex, patented'
        },
        'Hirshfeld': {
            'idea': 'Partition density by atomic reference',
            'pros': 'Always positive, chemically reasonable',
            'cons': 'Depends on reference atoms'
        },
        'Bader (AIM)': {
            'idea': 'Topological partitioning of ρ(r)',
            'pros': 'Uniquely defined, rigorous',
            'cons': 'Computationally expensive'
        }
    }

    for name, info in methods.items():
        print(f"\n{name}:")
        print(f"  Idea: {info['idea']}")
        print(f"  Pros: {info['pros']}")
        print(f"  Cons: {info['cons']}")

population_methods()
```
