## 1. Spectroscopy and Quantum Mechanics

Spectroscopy probes the quantized energy levels of molecules by measuring their interaction with electromagnetic radiation.

<Definition term="Spectroscopy">
The study of the interaction between matter and electromagnetic radiation. Absorption or emission of photons occurs when the photon energy matches the difference between two quantum states.
</Definition>

<Formula>
$$\Delta E = h\nu = \frac{hc}{\lambda}$$
</Formula>

where:
- $h$ = Planck's constant (6.626 × 10⁻³⁴ J·s)
- $\nu$ = frequency (Hz)
- $c$ = speed of light (3 × 10⁸ m/s)
- $\lambda$ = wavelength

```python
import numpy as np

def energy_to_wavenumber(energy_eV):
    """Convert energy (eV) to wavenumber (cm⁻¹)."""
    return energy_eV * 8065.54  # cm⁻¹ per eV

def wavenumber_to_wavelength(wavenumber):
    """Convert wavenumber (cm⁻¹) to wavelength (nm)."""
    return 1e7 / wavenumber  # nm

def wavelength_to_frequency(wavelength_nm):
    """Convert wavelength (nm) to frequency (Hz)."""
    c = 3e8  # m/s
    return c / (wavelength_nm * 1e-9)

# Energy scale comparison

print("=" * 60)

print("-" * 60)

    ('Rotational', 0.001, 'Microwave'),

    ('Electronic', 3.0, 'UV-Visible'),
    ('Core electron', 100, 'X-ray')
]

for name, energy, region in transitions:
    wn = energy_to_wavenumber(energy)
```

## 2. Rotational Spectroscopy

Molecules can rotate about their center of mass, with quantized angular momentum.

<Definition term="Rotational Spectroscopy">
Spectroscopy in the microwave region that probes the quantized rotational energy levels of molecules. It provides precise information about bond lengths and molecular geometry.
</Definition>

For a rigid rotor (diatomic molecule):

<Formula>
$$E_J = BJ(J+1)$$
$$B = \frac{\hbar^2}{2I} = \frac{h}{8\pi^2 I}$$
</Formula>

where:
- $J$ = rotational quantum number (0, 1, 2, ...)
- $B$ = rotational constant
- $I$ = moment of inertia = $\mu r^2$

```python
import numpy as np

def rotational_constant(reduced_mass_amu, bond_length_angstrom):
    """
    Calculate rotational constant B in cm⁻¹.

    Args:
        reduced_mass_amu: Reduced mass in atomic mass units
        bond_length_angstrom: Bond length in Angstroms
    """
    # Convert to SI units
    amu_to_kg = 1.66054e-27
    angstrom_to_m = 1e-10
    h = 6.62607e-34  # J·s
    c = 2.998e10  # cm/s

    mu = reduced_mass_amu * amu_to_kg
    r = bond_length_angstrom * angstrom_to_m

    I = mu * r**2  # Moment of inertia
    B = h / (8 * np.pi**2 * I * c)  # in cm⁻¹

    return B

def rotational_spectrum(B, J_max=5):
    """Generate rotational energy levels and transitions."""
    levels = []
    for J in range(J_max + 1):
        E = B * J * (J + 1)
        levels.append((J, E))

    transitions = []
    for J in range(J_max):
        # Selection rule: ΔJ = ±1
        delta_E = B * (J + 1) * (J + 2) - B * J * (J + 1)
        transitions.append((J, J+1, delta_E))

    return levels, transitions

# Example: CO molecule
m_C = 12.0
m_O = 16.0
mu_CO = (m_C * m_O) / (m_C + m_O)
r_CO = 1.128  # Angstrom

B_CO = rotational_constant(mu_CO, r_CO)
print(f"CO Molecule Rotational Spectroscopy")
print("=" * 50)
print(f"Bond length: {r_CO} Å")
print(f"Reduced mass: {mu_CO:.2f} amu")
print(f"Rotational constant B: {B_CO:.4f} cm⁻¹")

levels, transitions = rotational_spectrum(B_CO)
print(f"\nEnergy levels:")
for J, E in levels:
    print(f"  J = {J}: E = {E:.4f} cm⁻¹")

print(f"\nTransitions (ΔJ = +1):")
for J_low, J_high, delta_E in transitions[:4]:
    print(f"  J = {J_low} → {J_high}: ν = {delta_E:.4f} cm⁻¹")
```

## 3. Vibrational Spectroscopy

Molecular vibrations are quantized, with energy levels given by the harmonic oscillator model (plus anharmonic corrections).

<Definition term="Vibrational Spectroscopy">
Spectroscopy in the infrared region that probes molecular vibrations. It reveals information about bond strengths, functional groups, and molecular structure.
</Definition>

<Formula>
$$E_v = \hbar\omega\left(v + \frac{1}{2}\right) = h\nu\left(v + \frac{1}{2}\right)$$
$$\nu = \frac{1}{2\pi}\sqrt{\frac{k}{\mu}}$$
</Formula>

where:
- $v$ = vibrational quantum number (0, 1, 2, ...)
- $k$ = force constant
- $\mu$ = reduced mass

```python
import numpy as np

def vibrational_frequency(k_mdyne_per_angstrom, reduced_mass_amu):
    """
    Calculate vibrational frequency in cm⁻¹.

    Args:
        k_mdyne_per_angstrom: Force constant (mdyn/Å ≈ 10² N/m)
        reduced_mass_amu: Reduced mass in amu
    """
    # Convert units
    k_SI = k_mdyne_per_angstrom * 100  # N/m
    mu_SI = reduced_mass_amu * 1.66054e-27  # kg
    c = 2.998e10  # cm/s

    # ν = (1/2π)√(k/μ)
    nu_Hz = (1 / (2 * np.pi)) * np.sqrt(k_SI / mu_SI)
    nu_cm = nu_Hz / c  # Convert to wavenumber

    return nu_cm

# Common functional group frequencies
molecules = [
    ('H-H', 5.75, 0.504, 'H₂ stretch'),
    ('C-H', 5.1, 0.923, 'Alkane C-H stretch'),
    ('C=O', 12.1, 6.86, 'Carbonyl stretch'),
    ('O-H', 7.8, 0.948, 'Alcohol O-H stretch'),
    ('C≡C', 15.6, 6.0, 'Alkyne C≡C stretch'),
    ('C-C', 4.5, 6.0, 'Alkane C-C stretch')
]

print("Vibrational Frequencies of Common Bonds")
print("=" * 65)
print(f"{'Bond':<8} {'k (mdyn/Å)':<12} {'μ (amu)':<10} {'ν (cm⁻¹)':<12} {'Region'}")
print("-" * 65)

for bond, k, mu, description in molecules:
    freq = vibrational_frequency(k, mu)
    region = 'fingerprint' if freq < 1500 else ('mid-IR' if freq < 3000 else 'high freq')
    print(f"{bond:<8} {k:<12.2f} {mu:<10.3f} {freq:<12.0f} {description}")
```

Output:
```

=================================================================

-----------------------------------------------------------------

C-H      5.10         0.923      3030         Alkane C-H stretch
C=O      12.10        6.860      1715         Carbonyl stretch
```

<Warning>
Real molecules have 3N-6 vibrational modes (3N-5 for linear molecules). Each mode has its own frequency. This is why IR spectra have many peaks!
</Warning>

## 4. Electronic Spectroscopy

Electronic transitions involve promoting electrons between molecular orbitals.

<Definition term="Electronic Spectroscopy">
Spectroscopy in the UV-visible region that probes transitions between electronic states. It reveals information about conjugation, chromophores, and electronic structure.
</Definition>

```python
def electronic_transition_energy(E_HOMO, E_LUMO):
    """
    Estimate electronic transition energy (eV).
    Simple approximation: ΔE ≈ E_LUMO - E_HOMO
    """
    return E_LUMO - E_HOMO

def energy_to_color(wavelength_nm):
    """Approximate color from wavelength."""
    if wavelength_nm < 380:
        return "ultraviolet"
    elif wavelength_nm < 450:
        return "violet"
    elif wavelength_nm < 495:
        return "blue"
    elif wavelength_nm < 570:
        return "green"
    elif wavelength_nm < 590:
        return "yellow"
    elif wavelength_nm < 620:
        return "orange"
    elif wavelength_nm < 750:
        return "red"
    else:
        return "infrared"

# Example chromophores
chromophores = [
    ('Benzene', 4.9, 'π→π*'),
    ('Naphthalene', 4.0, 'π→π*'),
    ('Anthracene', 3.3, 'π→π*'),
    ('β-Carotene', 2.4, 'π→π* (conjugated)'),
    ('Chlorophyll', 1.9, 'π→π* (Soret)'),
]

print("Electronic Transitions in Chromophores")
print("=" * 60)
print(f"{'Molecule':<15} {'ΔE (eV)':<10} {'λ (nm)':<10} {'Color':<15} {'Type'}")
print("-" * 60)

for mol, delta_E, trans_type in chromophores:
    wavelength = 1240 / delta_E  # nm (from E = hc/λ)
    color = energy_to_color(wavelength)
    print(f"{mol:<15} {delta_E:<10.1f} {wavelength:<10.0f} {color:<15} {trans_type}")
```

## 5. Selection Rules

Not all transitions are allowed. Selection rules determine which transitions can occur.

<Definition term="Selection Rule">
A quantum mechanical rule that determines whether a transition between two states is allowed or forbidden. Allowed transitions have non-zero transition dipole moment.
</Definition>

<Formula>
$$\langle \psi_f | \hat{\mu} | \psi_i \rangle \neq 0 \quad \text{(allowed)}$$
</Formula>

| Spectroscopy | Selection Rules |
|--------------|-----------------|
| Rotational | ΔJ = ±1, molecule must have permanent dipole |
| Vibrational | Δv = ±1 (harmonic), dipole must change during vibration |
| Electronic | Spin must be conserved, symmetry considerations |

```python
def check_ir_active(vibration_type, molecule_symmetry):
    """
    Determine if a vibration is IR active.
    IR active = dipole moment changes during vibration
    """
    # Simplified rules for common cases
    ir_active_vibrations = {
        'asymmetric_stretch': True,  # Dipole changes
        'symmetric_stretch_hetero': True,  # e.g., H₂O
        'symmetric_stretch_homo': False,  # e.g., CO₂
        'bending': True,  # Usually IR active
    }

    return ir_active_vibrations.get(vibration_type, False)

def check_raman_active(vibration_type):
    """
    Determine if a vibration is Raman active.
    Raman active = polarizability changes during vibration
    """
    raman_active_vibrations = {
        'asymmetric_stretch': True,
        'symmetric_stretch_hetero': True,
        'symmetric_stretch_homo': True,  # Raman active even if IR inactive
        'bending': True,
    }

    return raman_active_vibrations.get(vibration_type, False)

# CO₂ example
print("CO₂ Vibrational Modes")
print("=" * 50)
modes = [
    ('Symmetric stretch', 'symmetric_stretch_homo', 1388),
    ('Asymmetric stretch', 'asymmetric_stretch', 2349),
    ('Bending (2 modes)', 'bending', 667)
]

for mode, vib_type, freq in modes:
    ir = "Yes" if check_ir_active(vib_type, 'linear') else "No"
    raman = "Yes" if check_raman_active(vib_type) else "No"
    print(f"{mode:<25} {freq:>6} cm⁻¹  IR: {ir:<4} Raman: {raman}")

print("\nNote: For centrosymmetric molecules (like CO₂),")
print("modes that are IR active are Raman inactive and vice versa.")
print("This is the 'rule of mutual exclusion'.")
```

## 6. Quantum Chemistry and Spectroscopy

Quantum chemistry calculations can predict spectroscopic properties.

```python
def predict_ir_spectrum(force_constants, reduced_masses, intensities):
    """
    Predict IR spectrum from quantum chemistry calculations.

    Args:
        force_constants: List of force constants (mdyn/Å)
        reduced_masses: List of reduced masses (amu)
        intensities: List of IR intensities (km/mol)
    """
    print("Predicted IR Spectrum")
    print("=" * 50)
    print(f"{'Mode':<6} {'ν (cm⁻¹)':<12} {'Intensity':<12} {'Strength'}")
    print("-" * 50)

    for i, (k, mu, I) in enumerate(zip(force_constants, reduced_masses, intensities)):
        freq = vibrational_frequency(k, mu)

        if I < 10:
            strength = 'weak'
        elif I < 50:
            strength = 'medium'
        else:
            strength = 'strong'

        print(f"{i+1:<6} {freq:<12.0f} {I:<12.1f} {strength}")

# Example: Water molecule (simplified)
force_constants = [7.8, 7.8, 0.7]  # O-H stretches and bend
reduced_masses = [0.948, 0.948, 1.08]
intensities = [44, 100, 65]  # km/mol

predict_ir_spectrum(force_constants, reduced_masses, intensities)
```

## Summary

In this lesson, you learned the quantum mechanical foundations of molecular spectroscopy:

1. **Spectroscopy** probes quantized energy levels through photon absorption/emission
2. **Rotational spectroscopy** reveals bond lengths and geometry (microwave region)
3. **Vibrational spectroscopy** provides functional group information (IR region)
4. **Electronic spectroscopy** probes orbital transitions (UV-Vis region)
5. **Selection rules** determine which transitions are allowed
6. **Quantum chemistry** can predict spectroscopic properties from first principles

Spectroscopy is one of the most important applications of quantum chemistry, connecting theoretical calculations to experimental observations.
