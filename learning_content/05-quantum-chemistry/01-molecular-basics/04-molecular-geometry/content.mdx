## 1. Why Geometry Matters

Molecular geometry—the three-dimensional arrangement of atoms—determines nearly every property of a molecule:

<Definition term="Molecular Geometry">
The spatial arrangement of atoms in a molecule, characterized by bond lengths, bond angles, and dihedral angles. The equilibrium geometry corresponds to the minimum on the potential energy surface.
</Definition>

- **Reactivity**: Which sites are accessible for reactions
- **Polarity**: Whether bond dipoles cancel or add
- **Biological activity**: Lock-and-key enzyme recognition
- **Spectroscopy**: IR and Raman active modes

<Tip>
Drug molecules must have the correct 3D shape to fit into receptor binding sites. Even mirror-image molecules (enantiomers) can have drastically different biological effects!
</Tip>

## 2. VSEPR Theory

The Valence Shell Electron Pair Repulsion (VSEPR) theory provides a simple way to predict molecular geometry.

<Definition term="VSEPR Theory">
A model predicting molecular geometry based on the principle that electron pairs (bonding and lone pairs) around a central atom repel each other and arrange to minimize this repulsion.
</Definition>

### Basic Geometries

| Electron Domains | Arrangement | Bond Angle | Example |
|-----------------|-------------|------------|---------|
| 2 | Linear | 180° | CO₂, BeH₂ |
| 3 | Trigonal planar | 120° | BF₃, SO₃ |
| 4 | Tetrahedral | 109.5° | CH₄, NH₄⁺ |
| 5 | Trigonal bipyramidal | 90°, 120° | PCl₅ |
| 6 | Octahedral | 90° | SF₆ |

```python
import numpy as np

def vsepr_geometry(n_bonding: int, n_lone_pairs: int) -> dict:
    """
    Predict molecular geometry using VSEPR theory.

    Args:
        n_bonding: Number of bonding domains (atoms bonded)
        n_lone_pairs: Number of lone pairs on central atom

    Returns:
        Dictionary with geometry information
    """
    total_domains = n_bonding + n_lone_pairs

    # Base arrangements
    arrangements = {
        2: {'name': 'linear', 'base_angle': 180.0},
        3: {'name': 'trigonal planar', 'base_angle': 120.0},
        4: {'name': 'tetrahedral', 'base_angle': 109.5},
        5: {'name': 'trigonal bipyramidal', 'base_angle': (90, 120)},
        6: {'name': 'octahedral', 'base_angle': 90.0}
    }

    # Molecular shapes based on (total_domains, lone_pairs)
    shapes = {
        (2, 0): 'linear',
        (3, 0): 'trigonal planar',
        (3, 1): 'bent',
        (4, 0): 'tetrahedral',
        (4, 1): 'trigonal pyramidal',
        (4, 2): 'bent',
        (5, 0): 'trigonal bipyramidal',
        (5, 1): 'seesaw',
        (5, 2): 'T-shaped',
        (5, 3): 'linear',
        (6, 0): 'octahedral',
        (6, 1): 'square pyramidal',
        (6, 2): 'square planar'
    }

    key = (total_domains, n_lone_pairs)
    shape = shapes.get(key, 'unknown')

    return {
        'total_domains': total_domains,
        'electron_geometry': arrangements.get(total_domains, {}).get('name', 'unknown'),
        'molecular_shape': shape,
        'n_bonding': n_bonding,
        'n_lone_pairs': n_lone_pairs
    }

# Test examples
molecules = [
    ('CO₂', 2, 0),
    ('H₂O', 2, 2),
    ('NH₃', 3, 1),
    ('CH₄', 4, 0),
    ('SF₄', 4, 1),
    ('XeF₂', 2, 3),
    ('SF₆', 6, 0)
]

print("VSEPR Geometry Predictions:")
print("-" * 70)
print(f"{'Molecule':<10} {'Bonding':<8} {'LP':<4} {'E⁻ Geometry':<20} {'Molecular Shape'}")
print("-" * 70)
for mol, bonds, lp in molecules:
    result = vsepr_geometry(bonds, lp)
    print(f"{mol:<10} {bonds:<8} {lp:<4} {result['electron_geometry']:<20} {result['molecular_shape']}")
```

## 3. Lone Pair Effects

Lone pairs occupy more space than bonding pairs, causing deviations from ideal angles.

<Formula>
$$\text{Repulsion Order: Lone Pair-Lone Pair} > \text{Lone Pair-Bond Pair} > \text{Bond Pair-Bond Pair}$$
</Formula>

<Example>
**Water (H₂O)**:
- Electron geometry: Tetrahedral (4 domains)
- Molecular shape: Bent
- Ideal angle: 109.5°
- Actual H-O-H angle: 104.5° (compressed by lone pairs)

**Ammonia (NH₃)**:
- Electron geometry: Tetrahedral
- Molecular shape: Trigonal pyramidal
- Actual H-N-H angle: 107° (less compression, only 1 lone pair)
</Example>

```python
def estimate_bond_angle(n_bonding: int, n_lone_pairs: int, base_angle: float = 109.5) -> float:
    """
    Estimate bond angle with lone pair compression.
    Each lone pair reduces bond angle by approximately 2-2.5°.
    """
    compression_per_lp = 2.5  # degrees
    adjusted_angle = base_angle - (n_lone_pairs * compression_per_lp)
    return max(adjusted_angle, 90.0)  # Don't go below 90°

examples = [
    ('CH₄', 4, 0, 109.5),
    ('NH₃', 3, 1, 107.0),
    ('H₂O', 2, 2, 104.5),
    ('PH₃', 3, 1, 93.3),
    ('H₂S', 2, 2, 92.1)
]

print("Bond Angle Analysis:")
print("-" * 55)
print(f"{'Molecule':<10} {'Bonds':<6} {'LP':<4} {'Predicted':<12} {'Experimental'}")
print("-" * 55)
for mol, bonds, lp, experimental in examples:
    predicted = estimate_bond_angle(bonds, lp)
    print(f"{mol:<10} {bonds:<6} {lp:<4} {predicted:<12.1f} {experimental}°")
```

## 4. Computational Geometry Optimization

In quantum chemistry, molecular geometry is determined by minimizing the total energy.

<Definition term="Geometry Optimization">
An iterative computational procedure that adjusts nuclear positions to find the configuration with minimum energy on the potential energy surface.
</Definition>

### The Optimization Algorithm

<Formula>
$$\mathbf{R}_{n+1} = \mathbf{R}_n - \mathbf{H}^{-1} \cdot \nabla E(\mathbf{R}_n)$$
</Formula>

where:
- $\mathbf{R}$ are the nuclear coordinates
- $\nabla E$ is the energy gradient (forces)
- $\mathbf{H}$ is the Hessian matrix (second derivatives)

```python
import numpy as np

def geometry_optimization_demo(initial_r, force_constant, r_eq, max_steps=20, tolerance=1e-6):
    """
    Simple 1D geometry optimization for a diatomic molecule.
    Uses steepest descent algorithm.
    """
    def energy(r, k, r_eq):
        """Harmonic potential: E = 0.5 * k * (r - r_eq)²"""
        return 0.5 * k * (r - r_eq)**2

    def gradient(r, k, r_eq):
        """dE/dr = k * (r - r_eq)"""
        return k * (r - r_eq)

    r = initial_r
    step_size = 0.1

    print("Geometry Optimization (Steepest Descent)")
    print("-" * 50)
    print(f"{'Step':<6} {'r':<12} {'E':<15} {'Gradient':<12}")
    print("-" * 50)

    for step in range(max_steps):
        E = energy(r, force_constant, r_eq)
        grad = gradient(r, force_constant, r_eq)

        print(f"{step:<6} {r:<12.6f} {E:<15.8f} {grad:<12.6f}")

        if abs(grad) < tolerance:
            print(f"\nConverged after {step} steps!")
            break

        # Update position (steepest descent)
        r = r - step_size * grad

    return r, energy(r, force_constant, r_eq)

# Run optimization
r_initial = 2.5  # Start away from equilibrium
r_equilibrium = 1.4
k = 0.5

r_final, E_final = geometry_optimization_demo(r_initial, k, r_equilibrium)
print(f"\nFinal geometry: r = {r_final:.4f} (target: {r_equilibrium})")
```

## 5. Describing Molecular Geometry

Molecular geometry is described using internal coordinates:

<Definition term="Internal Coordinates">
A set of coordinates describing molecular geometry in terms of bond lengths, bond angles, and dihedral angles rather than Cartesian coordinates.
</Definition>

```python
import numpy as np

def calculate_bond_length(coord1, coord2):
    """Calculate distance between two atoms."""
    return np.linalg.norm(np.array(coord1) - np.array(coord2))

def calculate_bond_angle(coord1, coord2, coord3):
    """
    Calculate angle at atom 2 (vertex).
    Returns angle in degrees.
    """
    v1 = np.array(coord1) - np.array(coord2)
    v2 = np.array(coord3) - np.array(coord2)

    cos_angle = np.dot(v1, v2) / (np.linalg.norm(v1) * np.linalg.norm(v2))
    angle = np.arccos(np.clip(cos_angle, -1, 1))
    return np.degrees(angle)

def calculate_dihedral(coord1, coord2, coord3, coord4):
    """
    Calculate dihedral angle between planes (1,2,3) and (2,3,4).
    Returns angle in degrees.
    """
    b1 = np.array(coord2) - np.array(coord1)
    b2 = np.array(coord3) - np.array(coord2)
    b3 = np.array(coord4) - np.array(coord3)

    n1 = np.cross(b1, b2)
    n2 = np.cross(b2, b3)

    n1 = n1 / np.linalg.norm(n1)
    n2 = n2 / np.linalg.norm(n2)

    cos_dihedral = np.dot(n1, n2)
    dihedral = np.arccos(np.clip(cos_dihedral, -1, 1))
    return np.degrees(dihedral)

# Example: Water molecule
# Coordinates in Angstroms
O = [0.0, 0.0, 0.0]
H1 = [0.757, 0.587, 0.0]
H2 = [-0.757, 0.587, 0.0]

r_OH1 = calculate_bond_length(O, H1)
r_OH2 = calculate_bond_length(O, H2)
angle_HOH = calculate_bond_angle(H1, O, H2)

print("Water Molecule Geometry:")
print("-" * 35)
print(f"O-H bond length: {r_OH1:.3f} Å")
print(f"H-O-H angle: {angle_HOH:.1f}°")

# Example: Ethane (eclipsed vs staggered)
print("\nEthane Dihedral Angles:")

# Staggered conformation
C1 = [0.0, 0.0, 0.0]
C2 = [1.54, 0.0, 0.0]
H1_stag = [-0.36, 1.03, 0.0]
H4_stag = [1.90, 1.03, 0.51]

dihedral_staggered = calculate_dihedral(H1_stag, C1, C2, H4_stag)
print(f"Staggered: {dihedral_staggered:.1f}° (most stable)")

# Eclipsed would be 0° or 120°
print(f"Eclipsed: 0° (higher energy)")
```

## 6. Geometry and Reactivity

Molecular geometry directly influences chemical reactivity:

### Steric Effects

<Definition term="Steric Hindrance">
The slowing or prevention of chemical reactions due to the physical size of substituent groups blocking access to reactive sites.
</Definition>

### Electronic Effects

The geometry determines orbital overlap and thus reaction rates.

```python
def assess_nucleophilic_attack(geometry_type: str) -> dict:
    """
    Assess susceptibility to nucleophilic attack based on geometry.
    """
    assessments = {
        'tetrahedral': {
            'susceptibility': 'moderate',
            'mechanism': 'SN2 backside attack, or SN1',
            'example': 'Alkyl halides (CH₃Br)'
        },
        'trigonal planar': {
            'susceptibility': 'high',
            'mechanism': 'Direct approach to central atom',
            'example': 'Carbonyl compounds (C=O)'
        },
        'octahedral': {
            'susceptibility': 'moderate',
            'mechanism': 'Associative or dissociative',
            'example': 'Transition metal complexes'
        },
        'linear': {
            'susceptibility': 'high',
            'mechanism': 'Approach along axis',
            'example': 'CO₂, acetylene'
        }
    }

    return assessments.get(geometry_type, {'susceptibility': 'unknown'})

# Display examples
geometries = ['tetrahedral', 'trigonal planar', 'octahedral', 'linear']
print("Geometry and Nucleophilic Attack:")
print("=" * 60)
for geom in geometries:
    info = assess_nucleophilic_attack(geom)
    print(f"\n{geom.upper()}:")
    print(f"  Susceptibility: {info['susceptibility']}")
    print(f"  Mechanism: {info['mechanism']}")
    print(f"  Example: {info['example']}")
```
