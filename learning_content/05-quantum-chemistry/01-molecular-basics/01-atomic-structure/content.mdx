## 1. From Classical to Quantum Atoms

The story of atomic structure is one of humanity's greatest intellectual journeys. Classical physics, with its billiard-ball model of atoms, completely failed to explain experimental observations like atomic spectra and the photoelectric effect.

<Definition term="Quantum Mechanical Atom">
A quantum mechanical description of the atom where electrons exist not as point particles in fixed orbits, but as probability distributions (orbitals) governed by the Schrödinger equation.
</Definition>

### The Hydrogen Atom: Our Starting Point

The hydrogen atom—one proton and one electron—is the only atom we can solve exactly. Its Schrödinger equation is:

<Formula>
$$\hat{H}\psi = E\psi$$
$$\left[-\frac{\hbar^2}{2m_e}\nabla^2 - \frac{e^2}{4\pi\epsilon_0 r}\right]\psi = E\psi$$
</Formula>

where:
- $\hat{H}$ is the Hamiltonian operator
- $\psi$ is the wavefunction
- $E$ is the energy eigenvalue
- $m_e$ is the electron mass
- $r$ is the distance from the nucleus

<Tip>
The hydrogen atom is to quantum chemistry what "Hello World" is to programming—the essential starting point that teaches fundamental concepts applicable everywhere else.
</Tip>

## 2. The Four Quantum Numbers

Every electron in an atom is completely described by four quantum numbers. These emerge directly from solving the Schrödinger equation in spherical coordinates.

<Definition term="Quantum Numbers">
A set of four integers or half-integers that uniquely specify the quantum state of an electron in an atom: principal (n), angular momentum (l), magnetic (m_l), and spin (m_s).
</Definition>

### Principal Quantum Number (n)

The principal quantum number determines the energy level and size of the orbital:

<Formula>
$$n = 1, 2, 3, 4, \ldots$$
$$E_n = -\frac{13.6 \text{ eV}}{n^2}$$
</Formula>

```python
import numpy as np

def hydrogen_energy(n):
    """Calculate hydrogen atom energy level in eV."""
    return -13.6 / n**2

# Display first 5 energy levels
print("Hydrogen Atom Energy Levels:")
print("-" * 30)
for n in range(1, 6):
    energy = hydrogen_energy(n)
    print(f"n = {n}: E = {energy:.4f} eV")
```

Output:
```
Hydrogen Atom Energy Levels:
------------------------------
n = 1: E = -13.6000 eV
n = 2: E = -3.4000 eV
n = 3: E = -1.5111 eV
n = 4: E = -0.8500 eV
n = 5: E = -0.5440 eV
```

### Angular Momentum Quantum Number (l)

The angular momentum quantum number determines the orbital shape:

<Formula>
$$l = 0, 1, 2, \ldots, (n-1)$$
</Formula>

| l value | Letter | Orbital Shape |
|---------|--------|---------------|
| 0 | s | Spherical |
| 1 | p | Dumbbell |
| 2 | d | Cloverleaf |
| 3 | f | Complex |

### Magnetic Quantum Number (m_l)

The magnetic quantum number determines orbital orientation in space:

<Formula>
$$m_l = -l, -l+1, \ldots, 0, \ldots, l-1, l$$
</Formula>

For example, p orbitals (l=1) have three orientations: $m_l = -1, 0, +1$ corresponding to $p_x$, $p_y$, $p_z$.

### Spin Quantum Number (m_s)

The spin quantum number describes the intrinsic angular momentum of the electron:

<Formula>
$$m_s = +\frac{1}{2} \text{ (spin up, } \uparrow \text{)} \quad \text{or} \quad m_s = -\frac{1}{2} \text{ (spin down, } \downarrow \text{)}$$
</Formula>

<Warning>
Electron spin has no classical analog—it's a purely quantum mechanical property. The electron isn't actually spinning; rather, it possesses an intrinsic angular momentum that can only take two values.
</Warning>

## 3. Atomic Orbitals

Atomic orbitals are wavefunctions that describe the quantum state of an electron. The probability of finding an electron at a given location is $|\psi|^2$.

<Definition term="Atomic Orbital">
A mathematical function describing the wave-like behavior of an electron in an atom. The square of the orbital (|\psi|²) gives the probability density for finding the electron at any point in space.
</Definition>

### The Radial and Angular Components

The hydrogen wavefunction can be separated into radial and angular parts:

<Formula>
$$\psi_{nlm}(r, \theta, \phi) = R_{nl}(r) \cdot Y_l^m(\theta, \phi)$$
</Formula>

where:
- $R_{nl}(r)$ is the radial wavefunction
- $Y_l^m(\theta, \phi)$ are the spherical harmonics

```python
import numpy as np
from scipy import special

def radial_wavefunction_1s(r, a0=1):
    """1s orbital radial wavefunction (in units of Bohr radius)."""
    return 2 * np.exp(-r / a0) / a0**(3/2)

def radial_wavefunction_2s(r, a0=1):
    """2s orbital radial wavefunction."""
    rho = r / a0
    return (1 / (2 * np.sqrt(2) * a0**(3/2))) * (2 - rho) * np.exp(-rho / 2)

def radial_probability(R, r):
    """Radial probability density: P(r) = r² |R(r)|²"""
    return r**2 * np.abs(R)**2

# Calculate radial probability for 1s and 2s orbitals
r = np.linspace(0, 15, 200)
R_1s = radial_wavefunction_1s(r)
R_2s = radial_wavefunction_2s(r)

P_1s = radial_probability(R_1s, r)
P_2s = radial_probability(R_2s, r)

# Find the most probable radius
r_max_1s = r[np.argmax(P_1s)]
r_max_2s = r[np.argmax(P_2s)]

print(f"Most probable radius for 1s: {r_max_1s:.2f} a₀ (theory: 1.00 a₀)")
print(f"Most probable radius for 2s: {r_max_2s:.2f} a₀ (theory: 5.24 a₀)")
```

Output:
```
Most probable radius for 1s: 1.00 a₀ (theory: 1.00 a₀)
Most probable radius for 2s: 5.26 a₀ (theory: 5.24 a₀)
```

## 4. Electron Configuration Rules

For multi-electron atoms, we must follow specific rules to determine how electrons fill orbitals.

### Aufbau Principle

Electrons fill orbitals starting from the lowest energy:

<Formula>
$$1s \rightarrow 2s \rightarrow 2p \rightarrow 3s \rightarrow 3p \rightarrow 4s \rightarrow 3d \rightarrow 4p \rightarrow \ldots$$
</Formula>

<Example>
Carbon (6 electrons): $1s^2 2s^2 2p^2$

The electron configuration shows:
- 2 electrons in the 1s orbital
- 2 electrons in the 2s orbital
- 2 electrons in the 2p subshell
</Example>

### Pauli Exclusion Principle

<Definition term="Pauli Exclusion Principle">
No two electrons in an atom can have the same set of four quantum numbers. This means each orbital can hold at most 2 electrons with opposite spins.
</Definition>

This principle is fundamental to understanding chemistry and is why atoms have structure rather than all electrons collapsing into the lowest orbital.

### Hund's Rule

When filling degenerate orbitals (orbitals with the same energy), electrons spread out with parallel spins before pairing:

<Example>
Nitrogen (7 electrons): $1s^2 2s^2 2p^3$

The three 2p electrons occupy separate $p_x$, $p_y$, and $p_z$ orbitals with parallel spins (all spin-up or all spin-down):
$$\uparrow \quad \uparrow \quad \uparrow$$
rather than:
$$\uparrow\downarrow \quad \uparrow \quad \text{empty}$$
</Example>

```python
def electron_configuration(atomic_number):
    """
    Generate electron configuration for an element.
    Simplified version following aufbau principle.
    """
    orbitals = [
        ('1s', 2), ('2s', 2), ('2p', 6), ('3s', 2), ('3p', 6),
        ('4s', 2), ('3d', 10), ('4p', 6), ('5s', 2), ('4d', 10),
        ('5p', 6), ('6s', 2), ('4f', 14), ('5d', 10), ('6p', 6),
        ('7s', 2), ('5f', 14), ('6d', 10), ('7p', 6)
    ]

    config = []
    electrons_left = atomic_number

    for orbital, max_electrons in orbitals:
        if electrons_left <= 0:
            break
        electrons_in_orbital = min(electrons_left, max_electrons)
        config.append(f"{orbital}{electrons_in_orbital}")
        electrons_left -= electrons_in_orbital

    return ' '.join(config)

# Test with some elements
elements = [('H', 1), ('C', 6), ('N', 7), ('O', 8), ('Fe', 26), ('Cu', 29)]
print("Electron Configurations:")
print("-" * 40)
for name, Z in elements:
    print(f"{name:2} (Z={Z:2}): {electron_configuration(Z)}")
```

Output:
```
Electron Configurations:
----------------------------------------
H  (Z= 1): 1s1
C  (Z= 6): 1s2 2s2 2p2
N  (Z= 7): 1s2 2s2 2p3
O  (Z= 8): 1s2 2s2 2p4
Fe (Z=26): 1s2 2s2 2p6 3s2 3p6 4s2 3d6
Cu (Z=29): 1s2 2s2 2p6 3s2 3p6 4s2 3d9
```

<Warning>
Some elements like copper (Cu) and chromium (Cr) have exceptions to the aufbau principle due to the extra stability of half-filled and fully-filled d subshells.
</Warning>

## 5. Why Quantum Computers for Atomic Simulation?

Classical computers struggle with atomic simulations because the wavefunction grows exponentially with the number of electrons:

<Formula>
$$\psi(r_1, r_2, \ldots, r_N) \in \mathbb{C}^{3N}$$
</Formula>

For N electrons, we need to track a function of 3N spatial coordinates (ignoring spin). The number of parameters to describe this function accurately grows exponentially.

<Definition term="Curse of Dimensionality">
The exponential growth of computational resources needed to simulate quantum systems as the number of particles increases. For N electrons, the Hilbert space dimension grows as O(M^N) where M is the number of basis functions.
</Definition>

```python
import numpy as np

def classical_memory_estimate(n_electrons, n_basis=10):
    """
    Estimate memory needed for full CI calculation.
    Assumes 16 bytes per complex coefficient.
    """
    # Number of determinants grows combinatorially
    from math import comb
    n_spin_orbitals = 2 * n_basis
    n_determinants = comb(n_spin_orbitals, n_electrons)
    memory_bytes = n_determinants * 16

    # Convert to appropriate units
    if memory_bytes < 1e3:
        return f"{memory_bytes:.0f} bytes"
    elif memory_bytes < 1e6:
        return f"{memory_bytes/1e3:.1f} KB"
    elif memory_bytes < 1e9:
        return f"{memory_bytes/1e6:.1f} MB"
    elif memory_bytes < 1e12:
        return f"{memory_bytes/1e9:.1f} GB"
    elif memory_bytes < 1e15:
        return f"{memory_bytes/1e12:.1f} TB"
    else:
        return f"{memory_bytes/1e15:.1f} PB"

print("Memory Required for Full CI (10 basis functions):")
print("-" * 50)
for n_electrons in [2, 4, 6, 8, 10, 12, 14]:
    memory = classical_memory_estimate(n_electrons)
    print(f"{n_electrons:2} electrons: {memory}")
```

Output:
```
Memory Required for Full CI (10 basis functions):
--------------------------------------------------
 2 electrons: 5.9 KB
 4 electrons: 1.5 MB
 6 electrons: 194.3 MB
 8 electrons: 14.7 GB
10 electrons: 703.7 GB
12 electrons: 22.4 TB
14 electrons: 496.0 TB
```

<Tip>
Quantum computers can represent quantum states naturally using qubits, potentially achieving exponential speedup for certain atomic and molecular simulations. This is one of the most promising near-term applications of quantum computing!
</Tip>

## Summary

In this lesson, you learned the foundational concepts of atomic structure:

1. **Quantum numbers** (n, l, m_l, m_s) completely describe each electron's state
2. **Atomic orbitals** are wavefunctions whose square gives probability densities
3. **Electron configurations** follow the aufbau principle, Pauli exclusion, and Hund's rule
4. **Multi-electron atoms** cannot be solved exactly, requiring approximations
5. **Quantum computers** offer potential exponential speedup for atomic simulations

These concepts form the essential foundation for understanding molecular structure, chemical bonding, and ultimately how quantum computers can revolutionize computational chemistry.
