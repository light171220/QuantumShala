## 1. From Atomic to Molecular Orbitals

When atoms approach each other to form molecules, their atomic orbitals interact and combine to create new orbitals that extend over the entire molecule—these are **molecular orbitals**.

<Definition term="Molecular Orbital (MO)">
A wavefunction that describes the spatial distribution of an electron in a molecule, extending over two or more atomic nuclei. Electrons in molecular orbitals belong to the whole molecule, not individual atoms.
</Definition>

### Why Molecular Orbitals Matter

Classical Lewis structures show bonds as electron pairs between atoms, but this picture fails to explain:
- Why O₂ is paramagnetic (has unpaired electrons)
- Delocalized electrons in benzene and conducting polymers
- The colors and optical properties of molecules

Molecular orbital theory provides the quantum mechanical explanation.

<Tip>
Think of molecular orbitals like standing waves on a guitar string stretched between two atoms. The electron's wavefunction can have different shapes depending on how the atomic orbitals constructively or destructively interfere.
</Tip>

## 2. Linear Combination of Atomic Orbitals (LCAO)

The simplest way to construct molecular orbitals is to combine atomic orbitals mathematically:

<Formula>
$$\psi_{MO} = c_1\phi_A + c_2\phi_B$$
</Formula>

where:
- $\psi_{MO}$ is the molecular orbital
- $\phi_A, \phi_B$ are atomic orbitals on atoms A and B
- $c_1, c_2$ are coefficients determined by solving the Schrödinger equation

<Definition term="LCAO Approximation">
A method for constructing molecular orbitals as linear combinations of atomic orbitals from the constituent atoms. The coefficients are determined by minimizing the energy (variational principle).
</Definition>

### The Hydrogen Molecule Ion (H₂⁺)

The simplest molecule to analyze is H₂⁺—two protons and one electron:

```python
import numpy as np
import matplotlib.pyplot as plt

def atomic_1s(r, center, a0=1.0):
    """1s atomic orbital centered at 'center'."""
    distance = np.abs(r - center)
    return (1 / np.sqrt(np.pi * a0**3)) * np.exp(-distance / a0)

def bonding_orbital(r, R, a0=1.0):
    """Bonding molecular orbital: σ = φ_A + φ_B"""
    phi_A = atomic_1s(r, -R/2, a0)
    phi_B = atomic_1s(r, R/2, a0)
    psi = phi_A + phi_B
    # Normalize
    norm = np.sqrt(np.trapz(psi**2, r))
    return psi / norm

def antibonding_orbital(r, R, a0=1.0):
    """Antibonding molecular orbital: σ* = φ_A - φ_B"""
    phi_A = atomic_1s(r, -R/2, a0)
    phi_B = atomic_1s(r, R/2, a0)
    psi = phi_A - phi_B
    # Normalize
    norm = np.sqrt(np.trapz(psi**2, r))
    return psi / norm if norm > 0 else psi

# Calculate orbitals
R = 2.0  # Internuclear distance
r = np.linspace(-6, 6, 500)

psi_bonding = bonding_orbital(r, R)
psi_antibonding = antibonding_orbital(r, R)

print("Bonding orbital (σ): Electron density concentrated BETWEEN nuclei")
print("Antibonding orbital (σ*): Node between nuclei, density OUTSIDE")
print(f"\nAt r=0 (midpoint):")
print(f"  Bonding |ψ|² = {psi_bonding[250]**2:.4f}")
print(f"  Antibonding |ψ|² = {psi_antibonding[250]**2:.6f} (near node)")
```

Output:
```
Bonding orbital (σ): Electron density concentrated BETWEEN nuclei
Antibonding orbital (σ*): Node between nuclei, density OUTSIDE

At r=0 (midpoint):
  Bonding |ψ|² = 0.0847
  Antibonding |ψ|² = 0.000000 (near node)
```

## 3. Bonding and Antibonding Orbitals

When two atomic orbitals combine, they form two molecular orbitals:

<Definition term="Bonding Orbital">
A molecular orbital with lower energy than the original atomic orbitals, formed when atomic orbitals combine with the same phase. Electron density is concentrated between the nuclei, stabilizing the molecule.
</Definition>

<Definition term="Antibonding Orbital">
A molecular orbital with higher energy than the original atomic orbitals, formed when atomic orbitals combine with opposite phases. A node exists between nuclei, destabilizing the molecule. Denoted with an asterisk (*).
</Definition>

<Formula>
$$\sigma = \frac{1}{\sqrt{2(1+S)}}(\phi_A + \phi_B) \quad \text{(bonding)}$$
$$\sigma^* = \frac{1}{\sqrt{2(1-S)}}(\phi_A - \phi_B) \quad \text{(antibonding)}$$
</Formula>

where $S = \langle\phi_A|\phi_B\rangle$ is the overlap integral.

<Warning>
The number of molecular orbitals always equals the number of atomic orbitals combined. Two 1s orbitals give one σ bonding and one σ* antibonding orbital.
</Warning>

## 4. Sigma (σ) and Pi (π) Orbitals

Molecular orbitals are classified by their symmetry with respect to rotation around the bond axis:

<Definition term="Sigma (σ) Orbital">
A molecular orbital with cylindrical symmetry around the internuclear axis. Formed from head-on overlap of s orbitals or p orbitals pointing along the bond axis.
</Definition>

<Definition term="Pi (π) Orbital">
A molecular orbital with a nodal plane containing the internuclear axis. Formed from sideways overlap of p orbitals perpendicular to the bond axis.
</Definition>

| Overlap Type | Resulting MO | Symmetry |
|-------------|--------------|----------|
| s + s | σ, σ* | Cylindrical |
| pz + pz (head-on) | σ, σ* | Cylindrical |
| px + px (sideways) | π, π* | Nodal plane |
| py + py (sideways) | π, π* | Nodal plane |

```python
# Illustrating MO formation from atomic orbitals

print()
```

## 5. Molecular Orbital Diagrams

An MO diagram shows the relative energies of molecular orbitals and how electrons fill them.

<Example>
**Hydrogen (H₂)**: Two electrons, both in the σ1s bonding orbital.

```
Energy
  ↑
  |    σ*1s  ____       (antibonding, empty)
  |
  |
  |    σ1s   ↑↓        (bonding, filled)
  |
  └─────────────────
       H₂ molecule
```

Bond Order = (2 - 0) / 2 = 1 (single bond)
</Example>

```python
def molecular_orbital_filling(molecule_name, n_electrons, mo_list):
    """
    Fill molecular orbitals following aufbau principle.

    Args:
        molecule_name: Name of the molecule
        n_electrons: Total number of electrons
        mo_list: List of (MO name, degeneracy, bonding/antibonding)
    """
    print(f"\nMolecular Orbital Diagram: {molecule_name}")
    print("=" * 50)

    electrons_left = n_electrons
    bonding_electrons = 0
    antibonding_electrons = 0

    for mo_name, degeneracy, is_bonding in mo_list:
        max_electrons = 2 * degeneracy
        electrons_in_mo = min(electrons_left, max_electrons)
        electrons_left -= electrons_in_mo

        if is_bonding:
            bonding_electrons += electrons_in_mo
        else:
            antibonding_electrons += electrons_in_mo

        # Display orbital filling
        if degeneracy == 1:
            if electrons_in_mo == 2:
                filling = "↑↓"
            elif electrons_in_mo == 1:
                filling = "↑ "
            else:
                filling = "__"
        else:  # Degenerate orbitals (like π)
            if electrons_in_mo == 4:
                filling = "↑↓ ↑↓"
            elif electrons_in_mo == 3:
                filling = "↑↓ ↑ "
            elif electrons_in_mo == 2:
                filling = "↑  ↑ "  # Hund's rule
            elif electrons_in_mo == 1:
                filling = "↑  __"
            else:
                filling = "__ __"

        print(f"  {mo_name:8} [{filling}]  ({electrons_in_mo} e⁻)")

    bond_order = (bonding_electrons - antibonding_electrons) / 2
    print(f"\nBond Order = ({bonding_electrons} - {antibonding_electrons}) / 2 = {bond_order}")
    return bond_order

# Define MO ordering for second-period homonuclear diatomics
mo_order_O2_F2 = [
    ("σ1s", 1, True), ("σ*1s", 1, False),
    ("σ2s", 1, True), ("σ*2s", 1, False),
    ("σ2p", 1, True),
    ("π2p", 2, True),
    ("π*2p", 2, False),
    ("σ*2p", 1, False)
]

# Analyze O₂
molecular_orbital_filling("O₂ (16 electrons)", 16, mo_order_O2_F2)
print("\nNote: O₂ has 2 unpaired electrons in π*2p → PARAMAGNETIC!")
```

Output:
```
Molecular Orbital Diagram: O₂ (16 electrons)
==================================================
  σ1s      [↑↓]  (2 e⁻)
  σ*1s     [↑↓]  (2 e⁻)
  σ2s      [↑↓]  (2 e⁻)
  σ*2s     [↑↓]  (2 e⁻)
  σ2p      [↑↓]  (2 e⁻)
  π2p      [↑↓ ↑↓]  (4 e⁻)
  π*2p     [↑  ↑ ]  (2 e⁻)
  σ*2p     [__]  (0 e⁻)

Bond Order = (10 - 6) / 2 = 2.0

```

## 6. Bond Order and Molecular Properties

<Definition term="Bond Order">
A measure of bond strength calculated from molecular orbital occupancy: Bond Order = (bonding electrons - antibonding electrons) / 2
</Definition>

<Formula>
$$\text{Bond Order} = \frac{n_b - n_a}{2}$$
</Formula>

| Molecule | Electrons | Bond Order | Bond Length | Bond Energy | Magnetic |
|----------|-----------|------------|-------------|-------------|----------|
| H₂ | 2 | 1 | 74 pm | 436 kJ/mol | Diamagnetic |
| He₂ | 4 | 0 | - | - | Doesn't exist |
| N₂ | 14 | 3 | 110 pm | 945 kJ/mol | Diamagnetic |
| O₂ | 16 | 2 | 121 pm | 498 kJ/mol | Paramagnetic |
| F₂ | 18 | 1 | 141 pm | 159 kJ/mol | Diamagnetic |

<Tip>
Higher bond order correlates with shorter bond length and higher bond dissociation energy. A bond order of 0 means the molecule is unstable.
</Tip>

## 7. Heteronuclear Diatomics

When two different atoms combine, the molecular orbitals are no longer symmetric. The more electronegative atom contributes more to bonding orbitals:

<Formula>
$$\psi_{MO} = c_A\phi_A + c_B\phi_B$$
$$|c_B|^2 > |c_A|^2 \text{ if B is more electronegative}$$
</Formula>

```python
import numpy as np

def heteronuclear_mo(electronegativity_A, electronegativity_B):
    """
    Calculate MO coefficients for heteronuclear diatomic.
    More electronegative atom contributes more to bonding orbital.
    """
    # Normalize so |c_A|² + |c_B|² = 1
    total_en = electronegativity_A + electronegativity_B

    # Bonding orbital: more coefficient on more electronegative
    c_A_bonding = electronegativity_A / total_en
    c_B_bonding = electronegativity_B / total_en

    # Normalize
    norm = np.sqrt(c_A_bonding**2 + c_B_bonding**2)
    c_A_bonding /= norm
    c_B_bonding /= norm

    # Antibonding: more coefficient on less electronegative
    c_A_antibonding = electronegativity_B / total_en
    c_B_antibonding = -electronegativity_A / total_en

    norm = np.sqrt(c_A_antibonding**2 + c_B_antibonding**2)
    c_A_antibonding /= norm
    c_B_antibonding /= norm

    return {
        'bonding': (c_A_bonding, c_B_bonding),
        'antibonding': (c_A_antibonding, c_B_antibonding)
    }

# Example: HF molecule
en_H = 2.20
en_F = 3.98

coeffs = heteronuclear_mo(en_H, en_F)
print("HF Molecular Orbitals:")
print(f"  Electronegativity: H = {en_H}, F = {en_F}")
print(f"\n  Bonding MO: ψ = {coeffs['bonding'][0]:.3f}φ_H + {coeffs['bonding'][1]:.3f}φ_F")
print(f"  Antibonding MO: ψ* = {coeffs['antibonding'][0]:.3f}φ_H + {coeffs['antibonding'][1]:.3f}φ_F")
print(f"\n  Electron density in bonding MO:")
print(f"    On H: {coeffs['bonding'][0]**2 * 100:.1f}%")
print(f"    On F: {coeffs['bonding'][1]**2 * 100:.1f}%")
```

Output:
```
HF Molecular Orbitals:
  Electronegativity: H = 2.2, F = 3.98

  Bonding MO: ψ = 0.484φ_H + 0.875φ_F
  Antibonding MO: ψ* = 0.875φ_H + -0.484φ_F

  Electron density in bonding MO:
    On H: 23.4%
    On F: 76.6%
```

<Warning>
The unequal distribution of electron density in bonding orbitals leads to bond polarity and molecular dipole moments.
</Warning>

## 8. Molecular Orbitals on Quantum Computers

Molecular orbital theory provides the foundation for quantum chemistry simulations. On quantum computers, molecular orbitals are encoded as qubit states:

<Definition term="Second Quantization">
A formalism where molecular orbitals are represented by creation and annihilation operators acting on occupation number states. Each orbital can be occupied (|1⟩) or empty (|0⟩).
</Definition>

<Formula>
$$|1100\rangle = |\text{orbital 1 occupied, orbital 2 occupied, orbitals 3,4 empty}\rangle$$
</Formula>

```python
def mo_to_qubit_state(occupation_list, orbital_names=None):
    """
    Convert molecular orbital occupation to qubit state notation.

    Args:
        occupation_list: List of 0s and 1s indicating orbital occupation
        orbital_names: Optional list of orbital names
    """
    n_orbitals = len(occupation_list)

    if orbital_names is None:
        orbital_names = [f"MO{i+1}" for i in range(n_orbitals)]

    print(f"Molecular Orbital State → Qubit State Mapping")
    print("=" * 50)

    qubit_string = ''.join(map(str, occupation_list))
    print(f"Qubit state: |{qubit_string}⟩")
    print()

    print("Orbital occupation:")
    for i, (occ, name) in enumerate(zip(occupation_list, orbital_names)):
        status = "occupied ↑↓" if occ else "empty"
        print(f"  {name}: |{occ}⟩ → {status}")

    n_electrons = sum(occupation_list) * 2  # Assuming paired electrons
    print(f"\nTotal electrons: {n_electrons}")

    return qubit_string

# Example: H₂ ground state
print("H₂ Ground State Configuration:")
mo_to_qubit_state([1, 0], ["σ1s (bonding)", "σ*1s (antibonding)"])

print("\n" + "-"*50 + "\n")

# Example: O₂ with simplified orbitals
print("O₂ Valence Configuration (simplified):")
mo_to_qubit_state(
    [1, 1, 1, 1, 1, 0],
    ["σ2s", "σ*2s", "σ2p", "π2p_x", "π2p_y", "σ*2p"]
)
```

## Summary

In this lesson, you learned the fundamental concepts of molecular orbital theory:

1. **LCAO method** combines atomic orbitals to form molecular orbitals
2. **Bonding orbitals** concentrate electron density between nuclei and lower energy
3. **Antibonding orbitals** have nodes between nuclei and higher energy
4. **σ and π orbitals** differ in their symmetry around the bond axis
5. **Bond order** predicts bond strength and molecular stability
6. **Heteronuclear molecules** have asymmetric electron distribution
7. **Quantum computers** encode molecular orbitals as qubit occupation states

Molecular orbital theory is the bridge between atomic structure and the quantum chemistry simulations that make quantum computing so powerful for chemistry applications.
