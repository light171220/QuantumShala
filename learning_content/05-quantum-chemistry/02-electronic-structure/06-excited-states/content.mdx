## 1. What Are Electronic Excited States?

<Definition term="Electronic Excited State">
A higher-energy electronic configuration where one or more electrons have been promoted to orbitals that are unoccupied in the ground state. These states are accessed by absorbing light or other energy sources.
</Definition>

<Formula>
$$E_{excitation} = E_{excited} - E_{ground} = h\nu = \frac{hc}{\lambda}$$
</Formula>

```python
def excited_state_concept():
    """Illustrate electronic excitations."""

    print("Electronic Excited States")
    print("=" * 60)

    print("\nGround State Configuration (example: H₂):")
    print("  |σ↑ σ↓⟩  - Both electrons in bonding orbital")
    print("  └── Lowest energy configuration")

    print("\nExcited State Configurations:")
    excitations = [
        ('S₁ (singlet)', 'σ↑ → σ*↑', '1/√2(|σ↑σ*↓⟩ - |σ↓σ*↑⟩)', 'Allowed'),
        ('T₁ (triplet)', 'σ↑ → σ*↑', '|σ↑σ*↑⟩', 'Forbidden'),
        ('S₂', 'σ → higher orbital', '...', 'Allowed'),
    ]

    print(f"\n{'State':<15} {'Excitation':<15} {'Wavefunction':<25} {'Dipole':<10}")
    print("-" * 70)
    for state, exc, wf, allowed in excitations:
        print(f"{state:<15} {exc:<15} {wf:<25} {allowed:<10}")

    print("\nKey concepts:")
    print("  • S = Singlet (paired spins, ΔS=0)")
    print("  • T = Triplet (parallel spins, ΔS=1)")
    print("  • Selection rules govern which transitions are 'allowed'")

excited_state_concept()
```

## 3. Configuration Interaction Singles (CIS)

The simplest ab initio method for excited states:

<Formula>
$$|\Psi_{CIS}\rangle = \sum_{ia} c_i^a |\Phi_i^a\rangle$$
</Formula>

```python
def cis_method():
    """Explain CIS for excited states."""

    print("Configuration Interaction Singles (CIS)")
    print("=" * 60)

    print("\nMethod:")
    print("  Expand excited states as linear combinations of")
    print("  singly-excited determinants from HF reference")
    print()
    print("  |Ψ⟩ = Σᵢₐ cᵢᵃ |Φᵢᵃ⟩")
    print()
    print("  where i → a means electron from occupied i to virtual a")

    print("\nMatrix eigenvalue problem:")
    print("  A c = ω c")
    print()
    print("  Aᵢₐ,ⱼᵦ = (εₐ - εᵢ)δᵢⱼδₐᵦ + ⟨aj||ib⟩")

    print("\nCharacteristics:")
    properties = [
        ('Ground state correlation', 'No', 'Singles don\'t mix with HF ground state'),
        ('Excited state correlation', 'Partial', 'Only singles included'),
        ('Scaling', 'O(N⁴)', 'Same as HF'),
        ('Accuracy', '~1 eV error', 'Qualitative at best'),
        ('Oscillator strengths', 'Yes', 'Can predict intensities'),
    ]

    for prop, value, note in properties:
        print(f"\n  {prop}:")
        print(f"    Value: {value}")
        print(f"    Note: {note}")

cis_method()
```

## 4. Time-Dependent DFT (TDDFT)

The most widely used method for excited states:

<Definition term="Time-Dependent DFT (TDDFT)">
An extension of DFT to time-dependent systems that, in the linear response limit, provides excited state energies and properties. It's much more accurate than CIS at similar cost.
</Definition>

```python
def tddft_method():
    """Explain TDDFT for excited states."""

    print("Time-Dependent DFT (TDDFT)")
    print("=" * 60)

    print("\nLinear Response Formulation:")
    print("  Excited states from poles of frequency-dependent")
    print("  polarizability (response function)")

    print("\nCasida Equation:")
    print("  [A  B ] [X]     [1  0] [X]")
    print("  [B* A*] [Y] = ω [0 -1] [Y]")

    print("\nAdvantages:")
    advantages = [
        'Similar cost to ground state DFT: O(N³)',
        'Accuracy ~0.3 eV for valence states',
        'Oscillator strengths and properties',
        'Works for large molecules (100+ atoms)',
    ]
    for adv in advantages:
        print(f"  ✓ {adv}")

    print("\nLimitations:")
    limitations = [
        'Charge-transfer states underestimated',
        'Rydberg states require special functionals',
        'Double excitations missing',
        'Strong correlation cases fail',
    ]
    for lim in limitations:
        print(f"  ✗ {lim}")

    print("\nPopular Functionals for TDDFT:")
    functionals = [
        ('B3LYP', 'General purpose, ~0.3 eV error'),
        ('CAM-B3LYP', 'Better for charge-transfer'),
        ('ωB97X-D', 'Range-separated + dispersion'),
        ('PBE0', 'Good balance, no empirical params'),
    ]
    for func, description in functionals:
        print(f"  {func}: {description}")

tddft_method()
```

## 5. Equation-of-Motion Coupled Cluster (EOM-CC)

High-accuracy excited states from coupled cluster:

```python
def eom_cc_method():
    """Explain EOM-CC for excited states."""

    print("Equation-of-Motion Coupled Cluster (EOM-CC)")

    print("\nIdea: Add excitation operators to CC ground state")
    print()
    print("  |Ψ_excited⟩ = R̂ e^T̂ |Φ₀⟩")
    print()
    print("  R̂ = R₀ + R₁ + R₂ + ...  (excitation operators)")

    print("\nRelated methods:")
    methods = {
        'EOM-CCSD': {
            'scaling': 'O(N⁶)',
            'accuracy': '~0.1-0.2 eV for singles',

        },
        'EOM-CC3': {
            'scaling': 'O(N⁷)',
```

## 6. State Classification

Different types of excited states require different approaches:

```python
def state_classification():
    """Classify types of electronic excited states."""

    print("Classification of Electronic Excited States")
    print("=" * 65)

    state_types = {
        'Valence (ππ*)': {
            'character': 'Excitation within valence orbitals',
            'examples': 'Benzene π→π*, formaldehyde n→π*',
            'methods': 'TDDFT, EOM-CCSD work well',
            'challenge': 'Generally well-behaved'
        },
        'Rydberg': {
            'character': 'Electron promoted to diffuse orbital',
            'examples': 'Atomic-like states, high-lying states',
            'methods': 'Need diffuse basis, range-separated DFT',
            'challenge': 'Require augmented basis sets'
        },
        'Charge-Transfer (CT)': {
            'character': 'Electron moves from donor to acceptor',
            'examples': 'TICT states, solar cell dyes',
            'methods': 'Range-separated DFT, EOM-CC',
            'challenge': 'Pure DFT fails badly'
        },
        'Doubly Excited': {
            'character': 'Two electrons promoted simultaneously',
            'examples': 'Polyene dark states, some conical intersections',
            'methods': 'CASPT2, EOM-CCSD(T)',
            'challenge': 'CIS/TDDFT miss entirely'
        },
        'Core Excited (XAS)': {
            'character': 'Core electron promoted',
            'examples': 'X-ray spectroscopy',
            'methods': 'CVS-EOM-CC, TDDFT with special treatment',
            'challenge': 'Require special methods'
        }
    }

    for state_type, info in state_types.items():
        print(f"\n{state_type}:")
        print(f"  Character: {info['character']}")
        print(f"  Examples: {info['examples']}")
        print(f"  Methods: {info['methods']}")
        print(f"  Challenge: {info['challenge']}")

state_classification()
```

<Warning>
Different excited state types require different computational approaches. Using the wrong method can give errors of several eV or miss states entirely!
</Warning>

## 7. Oscillator Strengths and Selection Rules

```python
def oscillator_strengths():
    """Explain oscillator strengths and selection rules."""

    print("Oscillator Strengths and Selection Rules")
    print("=" * 60)

    print("\nOscillator Strength (f):")
    print("  f = (2/3) × ΔE × |⟨Ψ₀|μ̂|Ψₙ⟩|²")
    print()
    print("  Dimensionless measure of transition intensity")
    print("  f = 0 → Forbidden transition (dark state)")
    print("  f ~ 1 → Strongly allowed transition")

    print("\nSelection Rules:")
    rules = [
        ('Spin', 'ΔS = 0', 'Singlet↔Singlet, Triplet↔Triplet'),
        ('Orbital', 'Δℓ = ±1', 's↔p, p↔d allowed'),
        ('Symmetry', 'Direct product contains A₁', 'Group theory governs'),
        ('Laporte', 'g ↔ u in centrosymmetric', 'd-d forbidden in octahedral'),
    ]

    print(f"\n{'Rule':<12} {'Requirement':<20} {'Example':<30}")
    print("-" * 65)
    for rule, req, example in rules:
        print(f"{rule:<12} {req:<20} {example:<30}")

    print("\nSpectroscopic Consequences:")
    print("  • Allowed: Strong absorption, short lifetime")
    print("  • Forbidden: Weak/no absorption, long lifetime")
    print("  • Phosphorescence: Spin-forbidden, ms-s lifetime")
    print("  • Fluorescence: Spin-allowed, ns lifetime")

oscillator_strengths()
```

## 8. Quantum Computing for Excited States

Quantum computers offer advantages for certain excited state problems:

```python
def quantum_excited_states():
    """Quantum computing approaches for excited states."""

    print("Quantum Computing for Excited States")
    print("=" * 60)

    print("\nClassical Challenges:")
    challenges = [
        'Double excitations require expensive methods',
        'Conical intersections need multireference',
        'Core-excited states are difficult',
        'Large active spaces become intractable',
    ]
    for c in challenges:
        print(f"  • {c}")

    print("\nQuantum Approaches:")

    methods = {
        'VQE with excited states': {
            'approach': 'Use symmetry or orthogonality constraints',
            'advantage': 'Can access multiple states',
            'challenge': 'State preparation, overlap calculation'
        },
        'VQD (Variational Quantum Deflation)': {
            'approach': 'Penalize overlap with lower states',
            'advantage': 'Systematic excited state access',
            'challenge': 'Need good initial guesses'
        },
        'SSVQE (Subspace Search VQE)': {
            'approach': 'Diagonalize in subspace of states',
            'advantage': 'Multiple states simultaneously',
            'challenge': 'Requires overlap measurements'
        },
        'qEOM (Quantum EOM)': {
            'approach': 'EOM on quantum computer',
            'advantage': 'Systematic, connected to classical',
            'challenge': 'Deep circuits required'
        },
        'QPE for excited states': {
            'approach': 'Prepare excited state, measure energy',
            'advantage': 'Exact eigenvalue (with resources)',
            'challenge': 'State preparation is hard'
        }
    }

    for method, info in methods.items():
        print(f"\n  {method}:")
        print(f"    Approach: {info['approach']}")
        print(f"    Advantage: {info['advantage']}")
        print(f"    Challenge: {info['challenge']}")

    print("\nNear-term opportunities:")
    print("  • Small active spaces with strong correlation")
    print("  • Transition metal excited states")
    print("  • Conical intersection regions")
    print("  • Benchmarking classical approximations")

quantum_excited_states()
```
