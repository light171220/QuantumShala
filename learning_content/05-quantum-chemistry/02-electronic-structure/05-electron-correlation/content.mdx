## 1. The Missing Physics in Hartree-Fock

<Definition term="Electron Correlation">
The difference between the exact electronic energy and the Hartree-Fock limit energy. It arises from the instantaneous electron-electron repulsion that mean-field theory averages away.
</Definition>

<Formula>
$$E_{corr} = E_{exact} - E_{HF}$$
</Formula>

```python
def correlation_concept():
    """Explain what's missing in mean-field theory."""

    print("What is Electron Correlation?")
    print("=" * 60)

    print("\n1. HARTREE-FOCK APPROXIMATION:")
    print("   Each electron moves in the AVERAGE field of all others")
    print("   Wavefunction = Single Slater determinant")
    print()
    print("   Missing: Instantaneous electron-electron avoidance")

    print("\n2. REALITY:")
    print("   Electrons are correlated - they avoid each other")
    print("   When electron 1 is 'here', electron 2 is less likely 'here'")
    print()
    print("   This creates a 'correlation hole' around each electron")

    print("\n3. CONSEQUENCES:")
    print("   HF energy is always ABOVE the true energy")
    print("   E_HF ≥ E_exact")
    print()
    print("   The difference is the correlation energy:")
    print("   E_corr = E_exact - E_HF < 0")

    # Example correlation energies
    print("\nTypical correlation energies:")
    molecules = [
        ('He', -0.042, -2.904),
        ('H₂', -0.041, -1.174),
        ('H₂O', -0.367, -76.07),
        ('N₂', -0.546, -109.5),
        ('Benzene', -1.52, -232.0),
    ]

    print(f"{'Molecule':<12} {'E_corr (Eh)':<15} {'E_total (Eh)':<15} {'% of total':<10}")
    print("-" * 55)
    for mol, e_corr, e_total in molecules:
        pct = (e_corr / e_total) * 100
        print(f"{mol:<12} {e_corr:<15.3f} {e_total:<15.1f} {pct:<10.2f}%")

correlation_concept()
```

<Warning>
Although correlation energy is only 0.5-2% of total energy, it can be 100% of chemically relevant quantities like bond energies, reaction barriers, and relative conformational energies!
</Warning>

## 2. Types of Electron Correlation

<Definition term="Dynamic Correlation">
The correlation arising from instantaneous electron-electron avoidance. It's present in all systems and can be captured by perturbative or coupled cluster methods.
</Definition>

<Definition term="Static (Non-Dynamic) Correlation">
The correlation arising from near-degeneracy of electronic configurations. It requires multiple determinants even for a qualitatively correct description.
</Definition>

```python
def correlation_types():
    """Distinguish dynamic and static correlation."""

    print("Types of Electron Correlation")
    print("=" * 65)

    print("\n1. DYNAMIC CORRELATION")
    print("-" * 40)
    print("Physical origin: Electrons avoiding each other instantaneously")
    print("Present in: All systems")
    print("Characteristics:")
    print("  • Many small contributions from many configurations")
    print("  • HF is qualitatively correct")
    print("  • Well-described by perturbation theory or CC")
    print("Examples: Noble gases, closed-shell molecules at equilibrium")

    print("\n2. STATIC (NON-DYNAMIC) CORRELATION")
    print("-" * 40)
    print("Physical origin: Multiple configurations with similar importance")
    print("Present in: Stretched bonds, transition states, open shells")
    print("Characteristics:")
    print("  • Few configurations dominate")
    print("  • HF is qualitatively WRONG")
    print("  • Requires multireference methods")
    print("Examples:")
    print("  • H₂ at dissociation limit (needs both |σσ̄⟩ and |σ*σ̄*⟩)")
    print("  • Transition metals with d-orbital degeneracy")
    print("  • Biradicals and transition states")

    print("\n3. PRACTICAL DISTINCTION")
    print("-" * 40)
    print("Static: Need multiple determinants in reference")
    print("Dynamic: Single reference + excitations works")

correlation_types()
```

## 4. Configuration Interaction (CI)

The conceptually simplest post-HF method:

<Formula>
$$|\Psi\rangle = c_0|\Phi_0\rangle + \sum_{ia}c_i^a|\Phi_i^a\rangle + \sum_{ijab}c_{ij}^{ab}|\Phi_{ij}^{ab}\rangle + \cdots$$
</Formula>

```python
def ci_methods():
    """Describe configuration interaction methods."""

    print("Configuration Interaction (CI)")
    print("=" * 60)

    print("\nIdea: Linear combination of Slater determinants")
    print()
    print("Ψ = c₀|Φ₀⟩ + Σ cᵢᵃ|Φᵢᵃ⟩ + Σ cᵢⱼᵃᵇ|Φᵢⱼᵃᵇ⟩ + ...")
    print("         singles      doubles       ...")

    levels = {
        'CIS': {
            'description': 'Singles only',
            'use': 'Excited states',
            'correlation': 'None for ground state!',
            'scaling': 'O(N⁴)'
        },
        'CID': {
            'description': 'Doubles only',
            'use': 'Ground state correlation',
            'correlation': '~90% dynamic',
            'scaling': 'O(N⁶)'
        },
        'CISD': {
            'description': 'Singles and Doubles',
            'use': 'Ground and excited states',
            'correlation': '~90-95%',
            'scaling': 'O(N⁶)'
        },
        'CISDT': {
            'description': '+ Triples',
            'use': 'High accuracy',
            'correlation': '~99%',
            'scaling': 'O(N⁸)'
        },
        'Full CI': {
            'description': 'All excitations',
            'use': 'Exact in given basis',
            'correlation': '100%',
            'scaling': 'Exponential!'
        }
    }

    print("\nCI Hierarchy:")
    print(f"{'Method':<10} {'Excitations':<20} {'Scaling':<12}")
    print("-" * 45)
    for method, info in levels.items():
        print(f"{method:<10} {info['description']:<20} {info['scaling']:<12}")

    print("\nSize Consistency Problem:")
    print("  Truncated CI is NOT size-consistent")
    print("  E(A...B) ≠ E(A) + E(B) for separated fragments")
    print("  This limits CISD for large systems")

ci_methods()
```

## 5. Coupled Cluster Theory

The gold standard for dynamic correlation:

<Formula>
$$|\Psi_{CC}\rangle = e^{\hat{T}}|\Phi_0\rangle$$
</Formula>

where $\hat{T} = \hat{T}_1 + \hat{T}_2 + \cdots$

```python
def coupled_cluster():
    """Explain coupled cluster theory."""

    print("Coupled Cluster (CC) Theory")
    print("=" * 60)

    print("\nKey innovation: Exponential ansatz")
    print()
    print("  |Ψ⟩ = e^T̂ |Φ₀⟩")
    print()
    print("  where T̂ = T̂₁ + T̂₂ + T̂₃ + ...")

    print("\nExponential expansion:")
    print("  e^T̂ = 1 + T̂ + T̂²/2! + T̂³/3! + ...")

    print("\nWhy exponential?")
    print("  • Automatically size-consistent!")
    print("  • E(A...B) = E(A) + E(B) for separated fragments")
    print("  • Includes disconnected terms: T̂₂²/2 generates quadruples")

    methods = {
        'CCD': {
            'clusters': 'T₂',
            'scaling': 'O(N⁶)',
            'accuracy': '~95% correlation'
        },
        'CCSD': {
            'clusters': 'T₁ + T₂',
            'scaling': 'O(N⁶)',
            'accuracy': '~95-98%'
        },
        'CCSD(T)': {
            'clusters': 'T₁ + T₂ + (T₃)',
            'scaling': 'O(N⁷)',
            'accuracy': '~99% (gold standard)'
        },
        'CCSDT': {
            'clusters': 'T₁ + T₂ + T₃',
            'scaling': 'O(N⁸)',
            'accuracy': '~99.5%'
        },
        'CCSDTQ': {
            'clusters': 'T₁ + T₂ + T₃ + T₄',
            'scaling': 'O(N¹⁰)',
            'accuracy': '~99.9%'
        }
    }

    print("\nCC Hierarchy:")
    print(f"{'Method':<12} {'Clusters':<18} {'Scaling':<12}")
    print("-" * 45)
    for method, info in methods.items():
        print(f"{method:<12} {info['clusters']:<18} {info['scaling']:<12}")

coupled_cluster()
```

<Tip>
CCSD(T) is called the "gold standard" of quantum chemistry. The (T) indicates a perturbative triples correction, giving O(N⁷) scaling instead of O(N⁸) for full CCSDT.
</Tip>

## 6. Møller-Plesset Perturbation Theory

A systematic approach based on perturbation theory:

<Formula>
$$\hat{H} = \hat{H}_0 + \lambda\hat{V}$$
$$E = E^{(0)} + E^{(1)} + E^{(2)} + \cdots$$
</Formula>

```python
def moller_plesset():
    """Explain MP perturbation theory."""

    print("Møller-Plesset Perturbation Theory")
    print("=" * 60)

    print("\nIdea: Treat correlation as a perturbation")
    print()
    print("  Ĥ = Ĥ₀ + V̂")
    print("  Ĥ₀ = Fock operator (mean field)")
    print("  V̂  = Fluctuation potential (correlation)")

    print("\nPerturbation expansion:")
    print("  E = E⁽⁰⁾ + E⁽¹⁾ + E⁽²⁾ + E⁽³⁾ + ...")
    print()
    print("  E⁽⁰⁾ + E⁽¹⁾ = E_HF (no correlation yet)")
    print("  E⁽²⁾ = MP2 correlation energy (first correction)")

    orders = {
        'MP0': {'formula': 'Σ εᵢ', 'meaning': 'Sum of orbital energies'},
        'MP1': {'formula': 'E_HF - MP0', 'meaning': 'Corrects double counting'},
        'MP2': {'formula': 'Σᵢⱼᵃᵇ |⟨ij||ab⟩|²/(εᵢ+εⱼ-εₐ-εᵦ)', 'meaning': 'Doubles contribution'},
        'MP3': {'formula': 'More complex', 'meaning': 'Higher order correction'},
        'MP4': {'formula': 'Even more complex', 'meaning': 'Includes singles, triples'},
    }

    print("\n" + f"{'Order':<8} {'Scaling':<12} {'What it captures':<30}")
    print("-" * 55)
    print(f"{'MP2':<8} {'O(N⁵)':<12} {'~85-90% of correlation':<30}")
    print(f"{'MP3':<8} {'O(N⁶)':<12} {'Correction to MP2':<30}")
    print(f"{'MP4':<8} {'O(N⁷)':<12} {'Singles, triples contributions':<30}")

    print("\nMP2 is extremely popular:")
    print("  • O(N⁵) scaling - feasible for large molecules")
    print("  • ~85-90% of correlation energy")
    print("  • Size-consistent")
    print("  • Often good for geometries and frequencies")

moller_plesset()
```

Comparing accuracy and cost:


## Summary

In this lesson, you learned:

1. **Correlation energy** is the difference between exact and HF energies
2. **Dynamic correlation** arises from instantaneous electron avoidance
3. **Static correlation** requires multiple reference determinants
4. **CI methods** are conceptually simple but not size-consistent
5. **Coupled Cluster** is the gold standard for dynamic correlation
6. **MP2** offers a good cost/accuracy balance
7. **Quantum computers** can naturally represent correlated states

Understanding correlation is key to understanding why quantum chemistry is hard—and why quantum computers may help.
