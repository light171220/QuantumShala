## 1. The Pauli Exclusion Principle

<Definition term="Pauli Exclusion Principle">
No two identical fermions (like electrons) can occupy the same quantum state simultaneously. This is a consequence of the antisymmetry requirement for fermionic wavefunctions.
</Definition>

The wavefunction must change sign under particle exchange:

<Formula>
$$\Psi(x_1, x_2, ..., x_i, ..., x_j, ..., x_N) = -\Psi(x_1, x_2, ..., x_j, ..., x_i, ..., x_N)$$
</Formula>

where $x_i = (\mathbf{r}_i, \sigma_i)$ includes both spatial and spin coordinates.

```python
def demonstrate_antisymmetry():
    """Show how antisymmetry leads to the Pauli principle."""

    print("Fermionic Antisymmetry")
    print("=" * 55)

    print("\n1. ANTISYMMETRY REQUIREMENT:")
    print("   Ψ(x₁, x₂) = -Ψ(x₂, x₁)")

    print("\n2. IF TWO ELECTRONS ARE IN THE SAME STATE (x₁ = x₂):")
    print("   Ψ(x₁, x₁) = -Ψ(x₁, x₁)")
    print("   ⟹ 2Ψ(x₁, x₁) = 0")
    print("   ⟹ Ψ(x₁, x₁) = 0")

    print("\n3. INTERPRETATION:")
    print("   The probability of finding two electrons")
    print("   in the same state is ZERO!")

    print("\n4. CONTRAST WITH BOSONS:")
    print("   Bosons: Ψ(x₁, x₂) = +Ψ(x₂, x₁)  [symmetric]")
    print("   Multiple bosons CAN occupy the same state")
    print("   Example: Photons in a laser, BEC")

demonstrate_antisymmetry()
```

## 2. Spin-Orbitals

<Definition term="Spin-Orbital">
A single-particle wavefunction that includes both spatial and spin components: χ(x) = φ(r)σ(s), where φ is a spatial orbital and σ is a spin function (α or β).
</Definition>

```python
def spin_orbital_structure():

    print("Spin-Orbital Structure")
    print("=" * 55)

    print("\nSpin-orbital: χ(x) = φ(r) × σ(s)")
    print()
    print("Spatial part φ(r):")
    print("  • Depends on position (x, y, z)")
    print("  • Examples: 1s, 2s, 2p_x, 2p_y, 2p_z, 3d, ...")

    print("\nSpin part σ(s):")
    print("  • α(s) = spin up   = |↑⟩ = |0⟩")

    print("\nOrthonormality:")
    print("  ⟨α|α⟩ = 1,  ⟨β|β⟩ = 1,  ⟨α|β⟩ = 0")
```

## 3. The Slater Determinant

<Definition term="Slater Determinant">
An antisymmetrized product of spin-orbitals that automatically satisfies the Pauli exclusion principle. For N electrons in N spin-orbitals, it's a normalized determinant of the orbital matrix.
</Definition>

<Formula>
$$\Psi(x_1, x_2, ..., x_N) = \frac{1}{\sqrt{N!}} \begin{vmatrix} \chi_1(x_1) & \chi_2(x_1) & \cdots & \chi_N(x_1) \\ \chi_1(x_2) & \chi_2(x_2) & \cdots & \chi_N(x_2) \\ \vdots & \vdots & \ddots & \vdots \\ \chi_1(x_N) & \chi_2(x_N) & \cdots & \chi_N(x_N) \end{vmatrix}$$
</Formula>

```python
import numpy as np
from itertools import permutations

def slater_determinant_expansion(n_electrons):
    """Show the expansion of a Slater determinant."""

    print(f"Slater Determinant for {n_electrons} Electrons")
    print("=" * 60)

    # Normalization factor
    norm = 1.0 / np.sqrt(np.math.factorial(n_electrons))

    print(f"\nNormalization: 1/√{n_electrons}! = {norm:.4f}")

    if n_electrons == 2:
        print("\n2-electron Slater determinant:")
        print()
        print("       1  | χ₁(x₁)  χ₂(x₁) |")
        print("  Ψ = ─── |                 |")
        print("      √2  | χ₁(x₂)  χ₂(x₂) |")
        print()
        print("    = 1/√2 [χ₁(x₁)χ₂(x₂) - χ₂(x₁)χ₁(x₂)]")
        print()
        print("Note: Swapping electrons 1↔2 gives a minus sign!")

    elif n_electrons == 3:
        print("\n3-electron Slater determinant:")
        print()
        print("       1  | χ₁(x₁)  χ₂(x₁)  χ₃(x₁) |")
        print("  Ψ = ─── | χ₁(x₂)  χ₂(x₂)  χ₃(x₂) |")
        print("      √6  | χ₁(x₃)  χ₂(x₃)  χ₃(x₃) |")
        print()
        print("    = 1/√6 [χ₁(x₁)χ₂(x₂)χ₃(x₃) - χ₁(x₁)χ₃(x₂)χ₂(x₃)")
        print("           +χ₂(x₁)χ₃(x₂)χ₁(x₃) - χ₂(x₁)χ₁(x₂)χ₃(x₃)")
        print("           +χ₃(x₁)χ₁(x₂)χ₂(x₃) - χ₃(x₁)χ₂(x₂)χ₁(x₃)]")

    # Count terms
    n_terms = np.math.factorial(n_electrons)
    print(f"\nNumber of terms in expansion: {n_electrons}! = {n_terms}")

slater_determinant_expansion(2)
print()
slater_determinant_expansion(3)
```

## 4. Why Determinants Work

The determinant has key properties that ensure proper antisymmetry:

<Tip>
Two properties of determinants make them perfect for fermions: (1) swapping any two rows (or columns) changes the sign, and (2) if two rows (or columns) are identical, the determinant is zero.
</Tip>

```python
def determinant_properties():
    """Demonstrate properties of determinants relevant to fermions."""

    print("Determinant Properties for Fermions")
    print("=" * 55)

    print("\n1. ROW/COLUMN SWAP → SIGN CHANGE")
    print("   This ensures antisymmetry under particle exchange")
    print()

    # Example 2x2
    A = np.array([[1, 2], [3, 4]])
    A_swap = np.array([[3, 4], [1, 2]])

    print(f"   Original: det|1 2| = {np.linalg.det(A):.0f}")
    print(f"                |3 4|")
    print()
    print(f"   Swapped:  det|3 4| = {np.linalg.det(A_swap):.0f}")
    print(f"                |1 2|")

    print("\n2. IDENTICAL ROWS → ZERO DETERMINANT")
    print("   This enforces the Pauli exclusion principle")
    print()

    B = np.array([[1, 2], [1, 2]])
    print(f"   det|1 2| = {np.linalg.det(B):.0f}")
    print(f"      |1 2|")
    print()
    print("   If two electrons occupy the same orbital → Ψ = 0")

    print("\n3. MULTILINEARITY")
    print("   Each orbital appears exactly once in each term")
    print("   Each electron appears exactly once in each term")

determinant_properties()
```

## 5. Shorthand Notation

For convenience, Slater determinants are often written in shorthand:

<Formula>
$$|\Psi\rangle = |\chi_1 \chi_2 \cdots \chi_N\rangle = |\chi_1 \chi_2 \cdots \chi_N|$$
</Formula>

```python
def slater_notation():
    """Show different notations for Slater determinants."""

    print("Slater Determinant Notation")
    print("=" * 55)

    examples = [
        {
            'molecule': 'H₂ (ground state)',
            'full': '|1sα 1sβ⟩',
            'short': '|1s 1s̄⟩',
            'occupation': '(1s)²'
        },
        {
            'molecule': 'He',
            'full': '|1sα 1sβ⟩',
            'short': '|1s 1s̄⟩',
            'occupation': '(1s)²'
        },
        {
            'molecule': 'Li',
            'full': '|1sα 1sβ 2sα⟩',
            'short': '|1s 1s̄ 2s⟩',
            'occupation': '(1s)²(2s)¹'
        },
        {
            'molecule': 'O₂ (simplified)',
            'full': '|... 2p_xα 2p_yα⟩',
            'short': '|... πₓ πᵧ⟩',
            'occupation': 'Triplet: parallel spins'
        }
    ]

    print("\nNotation conventions:")
    print("  • χ or χ̄ : spin up or down (bar = down)")
    print("  • Order matters: |χ₁χ₂⟩ = -|χ₂χ₁⟩")
    print("  • Square brackets [ ] sometimes used for antisymmetrization")

    print("\nExamples:")
    for ex in examples:
        print(f"\n  {ex['molecule']}:")
        print(f"    Full: {ex['full']}")
        print(f"    Short: {ex['short']}")
        print(f"    Config: {ex['occupation']}")

slater_notation()
```

## 6. Matrix Elements Between Determinants

Computing expectation values requires evaluating matrix elements:

<Formula>
$$\langle\Psi|\hat{O}|\Psi\rangle = \int \Psi^*(x_1,...,x_N)\hat{O}\Psi(x_1,...,x_N)dx_1...dx_N$$
</Formula>

```python
def slater_condon_rules():
    """Describe Slater-Condon rules for matrix elements."""

    print("Slater-Condon Rules")
    print("=" * 60)

    rules = {
        'Same determinant': {
            'one_body': '⟨Ψ|Ô₁|Ψ⟩ = Σᵢ ⟨χᵢ|ĥ|χᵢ⟩',
            'two_body': '⟨Ψ|Ô₂|Ψ⟩ = ½ Σᵢⱼ [⟨χᵢχⱼ|χᵢχⱼ⟩ - ⟨χᵢχⱼ|χⱼχᵢ⟩]',
            'description': 'Sum over occupied orbitals'
        },
        'Single excitation': {
            'one_body': '⟨Ψᵢᵃ|Ô₁|Ψ⟩ = ⟨χₐ|ĥ|χᵢ⟩',
            'two_body': '⟨Ψᵢᵃ|Ô₂|Ψ⟩ = Σⱼ [⟨χₐχⱼ|χᵢχⱼ⟩ - ⟨χₐχⱼ|χⱼχᵢ⟩]',
            'description': 'One orbital different'
        },
        'Double excitation': {
            'one_body': '⟨Ψᵢⱼᵃᵇ|Ô₁|Ψ⟩ = 0',
            'two_body': '⟨Ψᵢⱼᵃᵇ|Ô₂|Ψ⟩ = ⟨χₐχᵦ|χᵢχⱼ⟩ - ⟨χₐχᵦ|χⱼχᵢ⟩',
            'description': 'Two orbitals different'
        },
        'Triple+ excitation': {
            'one_body': '0',
            'two_body': '0',
            'description': 'More than 2 orbitals different'
        }
    }

    for name, info in rules.items():
        print(f"\n{name}:")
        print(f"  {info['description']}")
        print(f"  One-body: {info['one_body']}")
        print(f"  Two-body: {info['two_body']}")

    print("\nKey insight: Most matrix elements are ZERO!")
    print("This sparsity is exploited in efficient algorithms.")

slater_condon_rules()
```

## 7. Multiple Determinants: Configuration Interaction

A single Slater determinant is an approximation. The exact wavefunction is a linear combination:

<Formula>
$$|\Psi_{exact}\rangle = c_0|\Phi_0\rangle + \sum_{ia}c_i^a|\Phi_i^a\rangle + \sum_{ijab}c_{ij}^{ab}|\Phi_{ij}^{ab}\rangle + \cdots$$
</Formula>

```python
def configuration_interaction():
    """Explain configuration interaction expansions."""

    print("Configuration Interaction (CI)")
    print("=" * 60)

    print("\nExact wavefunction = Linear combination of determinants")
    print()
    print("Ψ = c₀|Φ₀⟩ + Σᵢₐ cᵢᵃ|Φᵢᵃ⟩ + Σᵢⱼₐᵦ cᵢⱼᵃᵇ|Φᵢⱼᵃᵇ⟩ + ...")
    print()
    print("Where:")
    print("  |Φ₀⟩     = Reference (Hartree-Fock) determinant")
    print("  |Φᵢᵃ⟩    = Single excitation: i→a")
    print("  |Φᵢⱼᵃᵇ⟩  = Double excitation: i,j→a,b")

    levels = [
        ('HF', 1, 'Single determinant'),
        ('CIS', 'N×(M-N)', 'Singles only'),
        ('CISD', 'O(N²M²)', 'Singles and Doubles'),
        ('CISDT', 'O(N³M³)', '+ Triples'),
        ('Full CI', 'C(M,N)', 'All excitations - EXACT')
    ]

    print("\nCI Hierarchy:")
    print(f"{'Method':<10} {'# Determinants':<15} {'Description':<25}")
    print("-" * 50)
    for method, n_det, desc in levels:
        print(f"{method:<10} {str(n_det):<15} {desc:<25}")

    print("\nN = number of electrons, M = number of orbitals")
    print("C(M,N) = M!/(N!(M-N)!) grows factorially!")

configuration_interaction()
```

<Warning>
The number of Slater determinants grows combinatorially with system size. Full CI is exact but only feasible for small systems (~18 electrons classically). This is why we need approximations—and quantum computers.
</Warning>

## 8. Connection to Quantum Computing

Slater determinants map naturally to qubit states in quantum computing:

```python
def quantum_computing_mapping():
    """Show how determinants map to qubits."""

    print("Slater Determinants in Quantum Computing")
    print("=" * 60)

    print("\nJordan-Wigner Mapping:")
    print("  Each spin-orbital → One qubit")
    print("  |0⟩ = orbital empty")
    print("  |1⟩ = orbital occupied")

    print("\nExample: 4 spin-orbitals, 2 electrons")
    configs = [
        ('|1100⟩', '|χ₁χ₂⟩', 'Orbitals 1,2 occupied'),
        ('|1010⟩', '|χ₁χ₃⟩', 'Orbitals 1,3 occupied'),
        ('|1001⟩', '|χ₁χ₄⟩', 'Orbitals 1,4 occupied'),
        ('|0110⟩', '|χ₂χ₃⟩', 'Orbitals 2,3 occupied'),
        ('|0101⟩', '|χ₂χ₄⟩', 'Orbitals 2,4 occupied'),
        ('|0011⟩', '|χ₃χ₄⟩', 'Orbitals 3,4 occupied'),
    ]

    print(f"\n{'Qubit state':<12} {'Determinant':<12} {'Description':<25}")
    print("-" * 50)
    for qubit, det, desc in configs:
        print(f"{qubit:<12} {det:<12} {desc:<25}")

    print("\nAntisymmetry is encoded in the Jordan-Wigner transformation:")
    print("  a†ᵢ = Z₁⊗Z₂⊗...⊗Zᵢ₋₁⊗(X-iY)/2 ⊗ I ⊗...")
    print("  The Z string ensures proper fermionic signs!")

quantum_computing_mapping()
```
