## 1. The Variational Principle

<Definition term="Variational Principle">
For any trial wavefunction |ψ(θ)⟩, the expectation value of the Hamiltonian provides an upper bound to the true ground state energy: E(θ) = ⟨ψ(θ)|Ĥ|ψ(θ)⟩ ≥ E₀.
</Definition>

```python
def variational_principle():
    """Explain the variational principle for VQE."""

    print("The Variational Principle")
    print("=" * 60)

    print("\nStatement:")
    print("  E(θ) = ⟨ψ(θ)|Ĥ|ψ(θ)⟩ ≥ E₀")
    print()
    print("  For ANY normalized trial state |ψ(θ)⟩:")
    print("  The energy expectation value is at least E₀")

    print("\nProof Sketch:")
    print("-" * 45)
    print("  Expand |ψ(θ)⟩ in energy eigenstates:")
    print("  |ψ(θ)⟩ = Σₙ cₙ|Eₙ⟩")
    print()
    print("  E(θ) = Σₙ |cₙ|² Eₙ ≥ Σₙ |cₙ|² E₀ = E₀")
    print()
    print("  Equality when |ψ(θ)⟩ = |E₀⟩")

    print("\nKey Insight:")
    print("  • Minimum of E(θ) = ground state energy")
    print("  • Optimal θ* gives best approximation to |E₀⟩")
    print("  • No need to know E₀ beforehand!")

    print("\nFor Quantum Computing:")
    print("-" * 45)
    print("  • Prepare |ψ(θ)⟩ on quantum computer")
    print("  • Measure ⟨Ĥ⟩ by sampling")
    print("  • Optimize θ classically")
    print("  • Repeat until converged")

variational_principle()
```

## 2. VQE Algorithm Overview

<Formula>
$$E(\boldsymbol{\theta}) = \langle\psi(\boldsymbol{\theta})|\hat{H}|\psi(\boldsymbol{\theta})\rangle = \sum_k w_k \langle\psi(\boldsymbol{\theta})|P_k|\psi(\boldsymbol{\theta})\rangle$$
</Formula>

```python
def vqe_overview():
    """VQE algorithm structure."""

    print("VQE Algorithm Overview")
    print("=" * 60)

    print("\nAlgorithm Steps:")
    print("-" * 45)
    steps = [
        "1. Initialize parameters θ (random or informed guess)",
        "2. Prepare state |ψ(θ)⟩ on quantum computer",
        "3. Measure expectation values ⟨Pₖ⟩ for each Pauli term",
        "4. Compute energy E(θ) = Σₖ wₖ⟨Pₖ⟩",
        "5. Use classical optimizer to update θ",
        "6. Repeat steps 2-5 until converged",
        "7. Report final energy and optimal parameters",
    ]
    for step in steps:
        print(f"  {step}")

    print("\nHybrid Quantum-Classical Nature:")
    print("-" * 45)
    print("  Quantum Computer:")
    print("    • Prepare parameterized quantum state")
    print("    • Measure Pauli operators")
    print()
    print("  Classical Computer:")
    print("    • Store molecular integrals")
    print("    • Compute energy from measurements")
    print("    • Run optimization algorithm")
    print("    • Store best parameters")

    print("\nWhy Hybrid?")
    print("  • Keeps quantum circuits short")
    print("  • Tolerates some noise (variational)")
    print("  • Classical optimization is robust")

vqe_overview()
```

```python
    print("  |0⟩ ─[Init]─[Layer 1(θ₁)]─[Layer 2(θ₂)]─...─ Measure")
    print("  |0⟩ ─[Init]─[Layer 1(θ₁)]─[Layer 2(θ₂)]─...─ Measure")

    print("\nTypical Gates:")
    print("  • Single-qubit rotations: Rx(θ), Ry(θ), Rz(θ)")
    print("  • Entangling gates: CNOT, CZ")
    print()
    print("  Rotation: Ry(θ) = exp(-iθY/2)")

    print("\nNumber of Parameters:")
    print("  • Simple ansatz: O(n) parameters")
    print("  • Hardware-efficient: O(n × depth)")
    print("  • Chemistry-inspired (UCCSD): O(n⁴)")

    print("\nAnsatz Requirements:")
    print("-" * 45)
    print("  1. Expressibility: Can represent ground state")
    print("  2. Trainability: Gradients not vanishing")
    print("  3. Hardware-friendly: Short depth, native gates")

parameterized_circuit()
```

<Tip>
The choice of ansatz is crucial for VQE success. Too simple and it can't represent the ground state; too complex and optimization becomes intractable due to barren plateaus.
</Tip>

```python
    print()
```

## 7. Measurement Strategies

```python
def measurement_strategies():
    """Efficient measurement strategies for VQE."""

    print("Measurement Strategies for VQE")
    print("=" * 60)

    print("\nThe Measurement Problem:")
    print("  H = Σₖ wₖ Pₖ  with O(N⁴) terms")
    print("  Naive: measure each term separately")
    print("  Cost: O(N⁴) × N_shots → very expensive!")

    print("\nGrouping Commuting Paulis:")
    print("-" * 45)
    print("  Paulis that commute can be measured together!")
    print()
    print("  Example: ZI, IZ, ZZ all commute")
    print("  Can measure with single circuit")

    print("\nQubit-Wise Commuting (QWC):")
    print("  P₁ and P₂ QWC if same Pauli or I at each site")
    print()
    print("  XIZI, XZZZ → QWC (X,I in pos 0; I,Z in pos 1; etc.)")
    print("  XIZI, YIZI → Not QWC (X vs Y at position 0)")

    print("\nReduction from Grouping:")
    print("-" * 45)
    print("  H₂ (4 qubits):")
    print("    • 15 Pauli terms")
    print("    • 5-8 measurement groups")
    print("    • ~3x speedup")
    print()
    print("  Larger molecules:")
    print("    • O(N³) groups from O(N⁴) terms")
    print("    • Significant savings!")

    print("\nAdvanced Techniques:")
    print("  • Classical shadows")
    print("  • Derandomization")
    print("  • Unitary partitioning")

measurement_strategies()
```
