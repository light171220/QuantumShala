## 1. Lead Optimization Goals

<Definition term="Lead Optimization">
The process of systematically modifying a lead compound to improve its drug-like properties including potency, selectivity, pharmacokinetics, and safety while maintaining favorable characteristics.
</Definition>


<Formula name="Free-Wilson Activity Model">
$$\log(1/IC_{50}) = \mu + \sum_i a_i \cdot X_i$$

Where $\mu$ is the parent compound activity and $a_i$ are additive group contributions for substituents $X_i$.
</Formula>

## 4. Multi-Parameter Optimization (MPO)

```python
import numpy as np

def multi_parameter_optimization():
    """Multi-parameter optimization strategies."""

    print("Multi-Parameter Optimization")
    print("=" * 60)

    print("\nThe MPO Challenge:")
    print("-" * 45)
    print("  Must optimize simultaneously:")
    print("    • Potency (IC50)")
    print("    • Selectivity (vs off-targets)")
    print("    • Solubility")
    print("    • Permeability")
    print("    • Metabolic stability")
    print("    • Safety (hERG, CYP inhibition)")

    print("\nMPO Scoring:")
    print("-" * 45)
    print("  Combine multiple properties into single score")
    print()

    def mpo_score(potency, selectivity, solubility, stability):
        """Calculate multi-parameter optimization score."""
        # Each component scored 0-1
        scores = {
            'potency': min(1, -np.log10(potency) / 9),  # pIC50 scale
            'selectivity': min(1, np.log10(selectivity) / 2),  # >100x
            'solubility': min(1, np.log10(solubility + 1) / 2),
            'stability': min(1, stability / 100),  # % remaining
        }
        return sum(scores.values()) / len(scores), scores

    print("  Example MPO scoring:")
    compounds = [
        ("Cmpd A", 50e-9, 50, 10, 80),
        ("Cmpd B", 10e-9, 200, 5, 60),
        ("Cmpd C", 100e-9, 100, 50, 90),
    ]
    for name, pot, sel, sol, stab in compounds:
        score, components = mpo_score(pot, sel, sol, stab)
        print(f"    {name}: MPO = {score:.2f}")
        print(f"      (Pot:{components['potency']:.2f}, Sel:{components['selectivity']:.2f}, "
              f"Sol:{components['solubility']:.2f}, Stab:{components['stability']:.2f})")

    print("\nDesirability Functions:")
    print("-" * 45)
    print("  Transform each property to 0-1 scale")
    print("  D = (Πdᵢ)^(1/n)  (geometric mean)")
    print()
    print("  Advantages:")
    print("    • Single unacceptable property → D ≈ 0")
    print("    • Balances all properties equally")

multi_parameter_optimization()
```

<Tip>
Bioisosteric replacement can maintain potency while dramatically improving properties. For example, replacing COOH with tetrazole often improves metabolic stability and membrane permeability.
</Tip>

<Warning>
Don't optimize potency in isolation. A compound with IC50 = 1 nM but poor ADMET will fail in development. Track multiple parameters throughout optimization.
</Warning>
