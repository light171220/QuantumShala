# Quantum Chemistry Applications

Quantum chemistry is one of the most promising near-term applications for quantum computing. Simulating molecular systems is inherently quantum mechanical, making it a natural fit.

## Learning Objectives

After completing this lesson, you will be able to:

- Understand molecular Hamiltonians
- Implement VQE for molecular ground states
- Choose appropriate ansätze for chemistry
- Apply error mitigation to chemistry problems
- Evaluate accuracy vs classical methods
- Identify promising chemistry applications

## 1. Why Quantum Chemistry?

Quantum computers are naturally suited to simulating quantum systems like molecules.

<Definition term="Electronic Structure Problem">
Finding the ground state energy and wavefunction of electrons in a molecule, which determines chemical properties, reaction rates, and molecular stability.
</Definition>

## 2. Molecular Hamiltonians

The molecular Hamiltonian describes the energy of electrons in a molecule.

<Formula>
Second-quantized molecular Hamiltonian:
$$H = \sum_{pq} h_{pq} a_p^\dagger a_q + \frac{1}{2}\sum_{pqrs} h_{pqrs} a_p^\dagger a_q^\dagger a_r a_s$$
</Formula>

## 3. VQE for Chemistry

The Variational Quantum Eigensolver finds the ground state energy.

```python
import numpy as np

class ChemistryVQE:
    """
    VQE implementation for molecular Hamiltonians.
    """

    def __init__(self, hamiltonian, n_qubits):
        self.hamiltonian = hamiltonian
        self.n_qubits = n_qubits

    def compute_expectation(self, params, ansatz_type='hwe'):
        """
        Compute energy expectation value.
        """
        # Simulate circuit and compute expectation
        energy = 0.0

        for coeff, pauli in self.hamiltonian:
            exp_val = self._measure_pauli(params, pauli, ansatz_type)
            energy += coeff * exp_val

        return energy

    def _measure_pauli(self, params, pauli, ansatz_type):
        """
        Measure expectation of Pauli string.
        Simplified simulation.
        """
        if pauli == 'II':
            return 1.0

        # Simulate based on ansatz
        if ansatz_type == 'hwe':
            # Hardware efficient ansatz
            result = 0.0
            for i, p in enumerate(pauli):
                if p == 'Z':
                    result += np.cos(2 * params[i % len(params)])
                elif p == 'X':
                    result += np.sin(params[i % len(params)])
                elif p == 'Y':
                    result += np.sin(params[i % len(params)] + np.pi/4)
            return np.tanh(result / len(pauli))
        else:
            # UCCSD-like
            result = np.cos(params[0]) * np.cos(params[1])
            for i, p in enumerate(pauli):
                if p in ['X', 'Y']:
                    result *= np.sin(params[i % len(params)])
            return result

    def optimize(self, n_params, n_steps=100, lr=0.1):
        """
        Optimize VQE parameters.
        """
        params = np.random.randn(n_params) * 0.1
        energies = []

        for step in range(n_steps):
            # Gradient estimation
            grad = np.zeros(n_params)
            shift = np.pi / 2

            for i in range(n_params):
                params[i] += shift
                e_plus = self.compute_expectation(params)
                params[i] -= 2 * shift
                e_minus = self.compute_expectation(params)
                params[i] += shift

                grad[i] = (e_plus - e_minus) / 2

            params -= lr * grad
            energy = self.compute_expectation(params)
            energies.append(energy)

            if step % 20 == 0:
                print(f"  Step {step}: E = {energy:.6f} Ha")

        return params, energies

# Demonstration
print("VQE for H2:")
print("=" * 50)

np.random.seed(42)

# Use H2 Hamiltonian
vqe = ChemistryVQE(h2_ham, n_qubits=2)

print("\nOptimizing with hardware-efficient ansatz...")
```

## 5. Error Mitigation for Chemistry

Chemistry applications require high precision, making error mitigation essential.

```python
import numpy as np

def chemistry_error_mitigation():
    """
    Error mitigation strategies for chemistry.
    """
    print("Error Mitigation for Chemistry:")
    print("=" * 50)

    # Simulate noisy vs mitigated results
    np.random.seed(42)

    exact_energy = -1.1373  # H2 ground state

    # Different noise levels
    noise_levels = [0.01, 0.02, 0.05]

    print("\nZero-Noise Extrapolation (ZNE):")
    print("-" * 40)

    for noise in noise_levels:
        # Noisy measurements at different noise scales
        scales = [1, 2, 3]
        noisy_energies = []

        for scale in scales:
            # Energy with scaled noise
            noisy_e = exact_energy + noise * scale * np.random.randn()
            noisy_energies.append(noisy_e)

        # Linear extrapolation to zero noise
        coeffs = np.polyfit(scales, noisy_energies, 1)
        extrapolated = coeffs[1]  # y-intercept

        error_noisy = abs(noisy_energies[0] - exact_energy)
        error_mitigated = abs(extrapolated - exact_energy)

        print(f"\n  Noise level: {noise}")
        print(f"    Raw energy (1x): {noisy_energies[0]:.4f} Ha (error: {error_noisy:.4f})")
        print(f"    Extrapolated:    {extrapolated:.4f} Ha (error: {error_mitigated:.4f})")
        print(f"    Improvement: {error_noisy/error_mitigated:.1f}x")
```

## Summary

Quantum chemistry is a leading application for NISQ and near-term quantum computers:

1. **Natural Fit**: Quantum systems simulate quantum systems naturally
2. **Hamiltonians**: Molecules map to qubit operators via fermion-to-qubit transforms
3. **VQE**: Variational approach enables shallow circuits
4. **Ansätze**: Choose based on accuracy needs and device constraints
5. **Error Mitigation**: Essential for achieving chemical accuracy
6. **Current State**: Small molecules demonstrated, larger systems in progress

The path to practical quantum advantage in chemistry is clearer than most other applications.
