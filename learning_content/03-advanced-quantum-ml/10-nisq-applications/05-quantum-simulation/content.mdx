# Quantum Simulation Applications

Quantum simulation is the original motivation for quantum computing—using controllable quantum systems to simulate other quantum systems that are classically intractable.

## Learning Objectives

After completing this lesson, you will be able to:

- Understand quantum simulation fundamentals
- Implement Hamiltonian simulation
- Apply Trotterization techniques
- Simulate spin systems
- Model many-body dynamics
- Evaluate simulation accuracy

## 1. Why Quantum Simulation?

Simulating quantum systems classically is exponentially hard due to the exponential growth of Hilbert space dimension.

<Definition term="Quantum Simulation">
Using a controllable quantum system to mimic the dynamics of another quantum system, leveraging the natural quantum behavior of the simulator to avoid exponential classical overhead.
</Definition>

## 2. Hamiltonian Simulation

The core task is implementing time evolution under a Hamiltonian.

<Formula>
Time evolution under Hamiltonian H:
$$|\psi(t)\rangle = e^{-iHt}|\psi(0)\rangle$$
For local Hamiltonians $H = \sum_j H_j$, use Trotter decomposition.
</Formula>

```python
import numpy as np
from scipy.linalg import expm

class HamiltonianSimulator:
    """
    Simulate time evolution under a Hamiltonian.
    """

    # Pauli matrices
    I = np.eye(2)
    X = np.array([[0, 1], [1, 0]])
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]])

    def __init__(self, n_qubits):
        self.n_qubits = n_qubits
        self.dim = 2**n_qubits

    def pauli_string(self, paulis, positions):
        """
        Build tensor product of Paulis at specified positions.
        """
        ops = [self.I] * self.n_qubits

        for p, pos in zip(paulis, positions):
            if p == 'X':
                ops[pos] = self.X
            elif p == 'Y':
                ops[pos] = self.Y
            elif p == 'Z':
                ops[pos] = self.Z

        result = ops[0]
        for op in ops[1:]:
            result = np.kron(result, op)

        return result

    def exact_evolution(self, H, t, psi0):
        """
        Exact time evolution (for small systems).
        """
        U = expm(-1j * H * t)
        return U @ psi0

    def trotter_step(self, terms, coeffs, dt):
        """
        First-order Trotter step.

        exp(-i H dt) ≈ ∏_j exp(-i H_j dt)
        """
        U = np.eye(self.dim)

        for term, coeff in zip(terms, coeffs):
            U = expm(-1j * coeff * term * dt) @ U

        return U

    def trotter_evolution(self, terms, coeffs, t, n_steps, psi0):
        """
        Time evolution using Trotterization.
        """
        dt = t / n_steps
        psi = psi0.copy()

        for _ in range(n_steps):
            U_step = self.trotter_step(terms, coeffs, dt)
            psi = U_step @ psi

        return psi

# Demonstration
print("Hamiltonian Simulation:")
print("=" * 50)

sim = HamiltonianSimulator(n_qubits=2)

# Simple Hamiltonian: H = J(XX + YY) + h(Z⊗I + I⊗Z)
J, h = 1.0, 0.5

XX = sim.pauli_string(['X', 'X'], [0, 1])
YY = sim.pauli_string(['Y', 'Y'], [0, 1])
ZI = sim.pauli_string(['Z'], [0])
IZ = sim.pauli_string(['Z'], [1])

H = J * (XX + YY) + h * (ZI + IZ)
terms = [XX, YY, ZI, IZ]
coeffs = [J, J, h, h]

print("\nHamiltonian: H = J(XX + YY) + h(ZI + IZ)")
print(f"Parameters: J={J}, h={h}")

# Initial state |01⟩
psi0 = np.zeros(4, dtype=complex)
psi0[1] = 1.0

t = 2.0
print(f"\nTime evolution for t={t}:")

# Exact evolution
psi_exact = sim.exact_evolution(H, t, psi0)

# Trotter evolution
for n_steps in [1, 5, 10, 50]:
    psi_trotter = sim.trotter_evolution(terms, coeffs, t, n_steps, psi0)
    fidelity = np.abs(np.vdot(psi_exact, psi_trotter))**2
    print(f"  {n_steps:3d} Trotter steps: Fidelity = {fidelity:.6f}")
```

## 3. Spin System Simulation

Spin models are natural targets for quantum simulation.

```python
import numpy as np
from scipy.linalg import expm

class SpinChainSimulator:
    """
    Simulate spin chain models.
    """

    I = np.eye(2)
    X = np.array([[0, 1], [1, 0]])
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]])

    def __init__(self, n_spins):
        self.n_spins = n_spins
        self.dim = 2**n_spins

    def build_operator(self, ops):
        """Build tensor product of operators."""
        result = ops[0]
        for op in ops[1:]:
            result = np.kron(result, op)
        return result

    def heisenberg_hamiltonian(self, J=1.0, h=0.0):
        """
        Heisenberg XXX model with transverse field.

        H = J Σ_i (X_i X_{i+1} + Y_i Y_{i+1} + Z_i Z_{i+1}) + h Σ_i Z_i
        """
        H = np.zeros((self.dim, self.dim), dtype=complex)
        n = self.n_spins

        for i in range(n - 1):
            for pauli in [self.X, self.Y, self.Z]:
                ops = [self.I] * n
                ops[i] = pauli
                ops[i + 1] = pauli
                H += J * self.build_operator(ops)

        for i in range(n):
            ops = [self.I] * n
            ops[i] = self.Z
            H += h * self.build_operator(ops)

        return H

    def ising_hamiltonian(self, J=1.0, h=1.0):
        """
        Transverse-field Ising model.

        H = -J Σ_i Z_i Z_{i+1} - h Σ_i X_i
        """
        H = np.zeros((self.dim, self.dim), dtype=complex)
        n = self.n_spins

        for i in range(n - 1):
            ops = [self.I] * n
            ops[i] = self.Z
            ops[i + 1] = self.Z
            H += -J * self.build_operator(ops)

        for i in range(n):
            ops = [self.I] * n
            ops[i] = self.X
            H += -h * self.build_operator(ops)

        return H

    def compute_magnetization(self, psi):
        """Compute average magnetization in Z direction."""
        n = self.n_spins
        mag = 0.0

        for i in range(n):
            ops = [self.I] * n
            ops[i] = self.Z
            Z_i = self.build_operator(ops)
            mag += np.real(np.vdot(psi, Z_i @ psi))

        return mag / n

    def compute_correlations(self, psi, i, j):
        """Compute ZZ correlation between sites i and j."""
        ops = [self.I] * self.n_spins
        ops[i] = self.Z
        ops[j] = self.Z
        ZZ = self.build_operator(ops)
        return np.real(np.vdot(psi, ZZ @ psi))

# Demonstration
print("Spin Chain Simulation:")
print("=" * 50)

n_spins = 4
sim = SpinChainSimulator(n_spins)

# Transverse-field Ising model
print(f"\nTransverse-Field Ising Model ({n_spins} spins)")
print("-" * 40)

h_values = [0.1, 0.5, 1.0, 2.0, 5.0]
print("\nGround state magnetization vs transverse field:")
print(f"  h/J    <Z>/n")

for h in h_values:
    H = sim.ising_hamiltonian(J=1.0, h=h)
    eigenvalues, eigenvectors = np.linalg.eigh(H)
    ground_state = eigenvectors[:, 0]
    mag = sim.compute_magnetization(ground_state)
    print(f"  {h:.1f}    {mag:+.4f}")

# Time evolution
print("\n\nDynamical Simulation:")
print("-" * 40)

H = sim.ising_hamiltonian(J=1.0, h=0.5)

# Start from |↑↑↑↑⟩
psi0 = np.zeros(sim.dim, dtype=complex)
psi0[0] = 1.0

print("\nStarting from |↑↑↑↑⟩, evolving under Ising Hamiltonian:")
print(f"  Time    <Z>/n    <Z_0 Z_1>")

for t in [0, 0.5, 1.0, 2.0, 4.0]:
    U = expm(-1j * H * t)
    psi_t = U @ psi0
    mag = sim.compute_magnetization(psi_t)
    corr = sim.compute_correlations(psi_t, 0, 1)
    print(f"  {t:.1f}      {mag:+.4f}    {corr:+.4f}")
```

<Warning>
The Hilbert space dimension grows as $2^n$ for $n$ spins. Classical simulation is limited to approximately 40-50 spins, while quantum simulators can potentially handle much larger systems.
</Warning>

## 4. Trotter Error Analysis

Understanding and controlling Trotter errors is crucial for accurate simulation.

```python
import numpy as np
from scipy.linalg import expm

def trotter_error_analysis():
    """
    Analyze Trotter decomposition errors.
    """
    print("Trotter Error Analysis:")
    print("=" * 50)

    # Two non-commuting operators
    np.random.seed(42)
    dim = 4

    # Generate random Hermitian operators
    A_rand = np.random.randn(dim, dim) + 1j * np.random.randn(dim, dim)
    A = (A_rand + A_rand.conj().T) / 2

    B_rand = np.random.randn(dim, dim) + 1j * np.random.randn(dim, dim)
    B = (B_rand + B_rand.conj().T) / 2

    H = A + B

    # Commutator
    commutator = A @ B - B @ A
    comm_norm = np.linalg.norm(commutator, 'fro')
    print(f"\n||[A, B]||_F = {comm_norm:.4f}")

    t = 1.0

    # Exact evolution
    U_exact = expm(-1j * H * t)

    print(f"\nTrotter error for t={t}:")
    print("-" * 40)
    print(f"  n_steps   ||U_exact - U_trotter||   Error/n²")

    errors = []
    steps_list = [1, 2, 4, 8, 16, 32]

    for n_steps in steps_list:
        dt = t / n_steps

        # First-order Trotter
        U_A = expm(-1j * A * dt)
        U_B = expm(-1j * B * dt)
        U_step = U_A @ U_B

        U_trotter = np.eye(dim)
        for _ in range(n_steps):
            U_trotter = U_step @ U_trotter

        error = np.linalg.norm(U_exact - U_trotter, 'fro')
        errors.append(error)
        error_scaled = error * n_steps**2 / (t**2)

        print(f"  {n_steps:5d}     {error:.6f}              {error_scaled:.4f}")

    # Second-order Trotter (Strang splitting)
    print("\n\nSecond-order Trotter (Strang splitting):")
    print("-" * 40)
    print(f"  n_steps   ||U_exact - U_trotter||   Error/n³")

    for n_steps in steps_list:
        dt = t / n_steps

        # Second-order: exp(-iAdt/2) exp(-iBdt) exp(-iAdt/2)
        U_A_half = expm(-1j * A * dt / 2)
        U_B_full = expm(-1j * B * dt)
        U_step = U_A_half @ U_B_full @ U_A_half

        U_trotter = np.eye(dim)
        for _ in range(n_steps):
            U_trotter = U_step @ U_trotter

        error = np.linalg.norm(U_exact - U_trotter, 'fro')
        error_scaled = error * n_steps**3 / (t**3)

        print(f"  {n_steps:5d}     {error:.6f}              {error_scaled:.4f}")

trotter_error_analysis()
```

<Tip>
Second-order Trotter (Strang splitting) has error $O(t^3/n^2)$ compared to first-order's $O(t^2/n)$. For fixed total error, second-order requires $O(\sqrt{n})$ fewer steps.
</Tip>

## 5. Many-Body Dynamics

Simulating out-of-equilibrium many-body dynamics.

```python
import numpy as np
from scipy.linalg import expm

class ManyBodySimulator:
    """
    Simulate many-body quantum dynamics.
    """

    I = np.eye(2)
    X = np.array([[0, 1], [1, 0]])
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]])

    def __init__(self, n_sites):
        self.n_sites = n_sites
        self.dim = 2**n_sites

    def build_operator(self, ops):
        result = ops[0]
        for op in ops[1:]:
            result = np.kron(result, op)
        return result

    def create_neel_state(self):
        """Create Néel state |↑↓↑↓...⟩."""
        psi = np.zeros(self.dim, dtype=complex)
        # |↑↓↑↓...⟩ corresponds to index with alternating bits
        idx = sum(2**(self.n_sites - 1 - i) for i in range(0, self.n_sites, 2))
        psi[idx] = 1.0
        return psi

    def create_domain_wall(self):
        """Create domain wall state |↑↑...↓↓⟩."""
        psi = np.zeros(self.dim, dtype=complex)
        half = self.n_sites // 2
        idx = sum(2**(self.n_sites - 1 - i) for i in range(half))
        psi[idx] = 1.0
        return psi

    def site_magnetization(self, psi, site):
        """Compute ⟨Z⟩ at a specific site."""
        ops = [self.I] * self.n_sites
        ops[site] = self.Z
        Z_site = self.build_operator(ops)
        return np.real(np.vdot(psi, Z_site @ psi))

    def simulate_quench(self, H, psi0, times):
        """
        Simulate quantum quench dynamics.
        """
        results = {'times': times, 'magnetization': []}

        for t in times:
            if t == 0:
                psi_t = psi0
            else:
                U = expm(-1j * H * t)
                psi_t = U @ psi0

            mags = [self.site_magnetization(psi_t, i) for i in range(self.n_sites)]
            results['magnetization'].append(mags)

        return results

# Demonstration
print("Many-Body Dynamics Simulation:")
print("=" * 50)

n_sites = 6
sim = ManyBodySimulator(n_sites)

# XXZ Hamiltonian
def build_xxz_hamiltonian(sim, Jxy=1.0, Jz=0.5):
    H = np.zeros((sim.dim, sim.dim), dtype=complex)
    n = sim.n_sites

    for i in range(n - 1):
        for pauli in [sim.X, sim.Y]:
            ops = [sim.I] * n
            ops[i] = pauli
            ops[i + 1] = pauli
            H += Jxy * sim.build_operator(ops)

        ops = [sim.I] * n
        ops[i] = sim.Z
        ops[i + 1] = sim.Z
        H += Jz * sim.build_operator(ops)

    return H

H = build_xxz_hamiltonian(sim, Jxy=1.0, Jz=0.5)

# Quench from Néel state
print(f"\nQuantum Quench ({n_sites} sites):")
print("-" * 40)
print("Initial state: Néel |↑↓↑↓↑↓⟩")
print("Hamiltonian: XXZ with Jxy=1.0, Jz=0.5")

psi0 = sim.create_neel_state()
times = [0, 0.5, 1.0, 2.0, 4.0]
results = sim.simulate_quench(H, psi0, times)

print("\nSite magnetization ⟨Z_i⟩ over time:")
print(f"  Time  ", end="")
for i in range(n_sites):
    print(f"  Site {i}", end="")
print()

for t_idx, t in enumerate(times):
    print(f"  {t:.1f}   ", end="")
    for mag in results['magnetization'][t_idx]:
        print(f"  {mag:+.3f}", end="")
    print()

# Entropy dynamics
print("\n\nEntanglement Entropy Evolution:")
print("-" * 40)

def compute_entropy(psi, n_sites, cut):
    """Compute entanglement entropy across a cut."""
    dim = 2**n_sites
    dim_A = 2**cut
    dim_B = 2**(n_sites - cut)

    # Reshape to bipartite form
    psi_matrix = psi.reshape(dim_A, dim_B)

    # Compute singular values
    s = np.linalg.svd(psi_matrix, compute_uv=False)
    s = s[s > 1e-10]

    # von Neumann entropy
    entropy = -np.sum(s**2 * np.log2(s**2 + 1e-15))
    return entropy

print("Half-chain entanglement entropy:")
print(f"  Time    S(L/2)")

for t in [0, 0.5, 1.0, 2.0, 4.0, 8.0]:
    if t == 0:
        psi_t = psi0
    else:
        U = expm(-1j * H * t)
        psi_t = U @ psi0

    S = compute_entropy(psi_t, n_sites, n_sites // 2)
    print(f"  {t:.1f}      {S:.4f}")
```

## 6. NISQ Considerations for Simulation

Practical aspects of running simulations on near-term quantum hardware.

```python
import numpy as np

def nisq_simulation_analysis():
    """
    Analyze NISQ considerations for quantum simulation.
    """
    print("NISQ Simulation Considerations:")
    print("=" * 50)

    # Circuit depth analysis
    print("\n1. CIRCUIT DEPTH ANALYSIS")
    print("-" * 40)

    def estimate_trotter_depth(n_qubits, n_terms, n_steps, gates_per_term=4):
        """Estimate circuit depth for Trotter simulation."""
        return n_steps * n_terms * gates_per_term

    print("\nTrotter circuit depth (4 gates/term):")
    print(f"  Qubits  Terms   Steps   Depth")

    configs = [
        (4, 6, 10),
        (6, 10, 10),
        (8, 14, 10),
        (10, 18, 10),
        (4, 6, 50),
        (6, 10, 50),
    ]

    for n_q, n_t, n_s in configs:
        depth = estimate_trotter_depth(n_q, n_t, n_s)
        print(f"  {n_q:6d}  {n_t:5d}  {n_s:6d}  {depth:6d}")

    # Error accumulation
    print("\n\n2. ERROR ACCUMULATION")
    print("-" * 40)

    def simulation_fidelity(depth, error_per_gate=0.001):
        """Estimate simulation fidelity."""
        return (1 - error_per_gate)**depth

    print("\nExpected fidelity vs depth (error/gate = 0.1%):")
    print(f"  Depth    Fidelity")

    for depth in [10, 50, 100, 200, 500, 1000]:
        fid = simulation_fidelity(depth)
        print(f"  {depth:5d}    {fid:.4f}")

    # Strategies
    print("\n\n3. ERROR MITIGATION STRATEGIES")
    print("-" * 40)

    strategies = [
        {
            'name': 'Zero-noise extrapolation',
            'overhead': '3-5x circuit evaluations',
            'effectiveness': 'Reduces systematic errors'
        },
        {
            'name': 'Probabilistic error cancellation',
            'overhead': 'Exponential in error rate',
            'effectiveness': 'High accuracy but costly'
        },
        {
            'name': 'Symmetry verification',
            'overhead': 'Discards ~50% of shots',
            'effectiveness': 'Good for conserved quantities'
        },
        {
            'name': 'Randomized compiling',
            'overhead': 'Multiple circuit variants',
            'effectiveness': 'Converts coherent to incoherent'
        }
    ]

    for s in strategies:
        print(f"\n  {s['name']}:")
        print(f"    Overhead: {s['overhead']}")
        print(f"    Effectiveness: {s['effectiveness']}")

    # What's achievable
    print("\n\n4. CURRENT NISQ CAPABILITIES")
    print("-" * 40)

    achievements = [
        "Heisenberg model dynamics (10-20 qubits)",
        "Ising model phase transitions (50+ qubits noisy)",
        "Simple molecule dynamics (4-10 qubits)",
        "Lattice gauge theory basics (10-20 qubits)"
    ]

    print("\nDemonstrated simulations:")
    for a in achievements:
        print(f"  - {a}")

    limitations = [
        "Deep circuits (>1000 gates) fail",
        "Long-time dynamics limited",
        "Precise energies require error mitigation",
        "Classical verification often possible"
    ]

    print("\nCurrent limitations:")
    for l in limitations:
        print(f"  - {l}")

nisq_simulation_analysis()
```

<Warning>
For current NISQ devices, the circuit depth must typically stay below a few hundred gates to maintain reasonable fidelity. This limits the simulation time and system size that can be accurately studied.
</Warning>

## Summary

Quantum simulation leverages quantum mechanics to study quantum systems:

1. **Motivation**: Classical simulation of quantum systems is exponentially hard
2. **Hamiltonian Simulation**: Implement $e^{-iHt}$ using Trotter decomposition
3. **Spin Systems**: Natural models for quantum computers
4. **Trotter Errors**: Higher-order methods reduce gate count
5. **Many-Body Dynamics**: Quenches, thermalization, entanglement growth
6. **NISQ Reality**: Limited depth, error mitigation essential

Quantum simulation remains one of the most promising near-term applications.
