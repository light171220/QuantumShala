## 1. Introduction to Reservoir Computing

Reservoir computing (RC) is a computational framework for efficiently training recurrent neural networks by separating the network into a fixed, randomly-initialized "reservoir" and a simple, trainable readout layer.

<Definition term="Reservoir Computing">
Reservoir computing is a machine learning paradigm where a complex dynamical system (the reservoir) transforms input signals into a high-dimensional feature space, and only a simple linear readout layer is trained to produce the desired output.
</Definition>

### The Training Problem with RNNs

Traditional recurrent neural networks (RNNs) suffer from:

1. **Vanishing/exploding gradients**: Deep temporal dependencies are hard to learn
2. **Slow training**: Backpropagation through time is computationally expensive
3. **Local minima**: Complex loss landscapes lead to poor convergence

Reservoir computing sidesteps these issues by not training the recurrent connections at all!

<Formula>
$$\mathbf{x}(t+1) = f(W_{in}\mathbf{u}(t) + W\mathbf{x}(t))$$
</Formula>

where only the readout $\mathbf{y} = W_{out}\mathbf{x}$ is trained.

## 2. Architecture Overview

A reservoir computer consists of three components:

```
Input Layer → Reservoir (Fixed) → Readout Layer (Trained)
     ↓              ↓                    ↓
   u(t)         x(t+1) = f(·)        y(t) = Wout·x(t)
```

### Input Layer

Maps input signals to the reservoir:

<Formula>
$$\mathbf{x}_{input} = W_{in}\mathbf{u}(t)$$
</Formula>

### Reservoir

A high-dimensional nonlinear dynamical system:

```python
import numpy as np

class Reservoir:
    """Basic reservoir implementation."""

    def __init__(self, n_reservoir=100, spectral_radius=0.9,
                 input_scaling=1.0, sparsity=0.1):
        """
        Initialize reservoir.

        Args:
            n_reservoir: Number of reservoir nodes
            spectral_radius: Largest eigenvalue magnitude (< 1 for stability)
            input_scaling: Scale factor for inputs
            sparsity: Fraction of non-zero connections
        """
        self.N = n_reservoir
        self.spectral_radius = spectral_radius
        self.input_scaling = input_scaling

        # Initialize random reservoir weights
        W = np.random.randn(n_reservoir, n_reservoir)

        # Apply sparsity
        mask = np.random.random((n_reservoir, n_reservoir)) < sparsity
        W = W * mask

        # Scale to desired spectral radius
        eigenvalues = np.linalg.eigvals(W)
        W = W * (spectral_radius / np.max(np.abs(eigenvalues)))

        self.W = W
        self.state = np.zeros(n_reservoir)

    def step(self, u, W_in):
        """Single reservoir update step."""
        # x(t+1) = tanh(W_in * u + W * x(t))
        self.state = np.tanh(
            self.input_scaling * W_in @ u + self.W @ self.state
        )
        return self.state

    def reset(self):
        """Reset reservoir state."""
        self.state = np.zeros(self.N)
```

### Readout Layer

Simple linear transformation (trainable):

<Formula>
$$\mathbf{y}(t) = W_{out}\mathbf{x}(t)$$
</Formula>

<Tip>
The readout is typically trained using ridge regression (linear regression with L2 regularization), which is extremely fast compared to gradient-based training.
</Tip>

## 3. The Echo State Property

For a reservoir to function correctly, it must satisfy the **echo state property** (ESP).

<Definition term="Echo State Property">
A reservoir has the echo state property if, for any input sequence, the reservoir state depends only on the input history and not on the initial state. This ensures that the reservoir "forgets" its initial conditions.
</Definition>

### Sufficient Condition

If the spectral radius $\rho(W) < 1$, the ESP is guaranteed for any input:

<Formula>
$$\rho(W) = \max_i |\lambda_i| < 1$$
</Formula>

where $\lambda_i$ are eigenvalues of the reservoir weight matrix $W$.

```python
def check_echo_state_property(W):
    """Check if reservoir satisfies ESP (sufficient condition)."""
    eigenvalues = np.linalg.eigvals(W)
    spectral_radius = np.max(np.abs(eigenvalues))

    print(f"Spectral radius: {spectral_radius:.4f}")
    print(f"ESP guaranteed: {spectral_radius < 1}")

    return spectral_radius < 1
```

<Warning>
While ρ(W) < 1 guarantees ESP, it may not be the optimal setting. For some tasks, operating near the "edge of chaos" (ρ ≈ 1) provides better computational power.
</Warning>

## 4. Fading Memory

Related to ESP is the concept of fading memory.

<Definition term="Fading Memory">
A system has fading memory if its current output depends more strongly on recent inputs than on distant past inputs. The influence of past inputs decays over time.
</Definition>

The fading memory capacity can be measured:

<Formula>
$$MC = \sum_{k=1}^{\infty} MC_k = \sum_{k=1}^{\infty} \max_{f} \text{cor}^2(y(t), u(t-k))$$
</Formula>

```python
def measure_memory_capacity(reservoir, max_delay=50, n_samples=2000):
    """
    Measure memory capacity of a reservoir.

    Returns the ability to recall past inputs.
    """
    n_input = 1
    W_in = np.random.randn(reservoir.N, n_input) * 0.1

    # Generate random input
    u = np.random.uniform(-0.5, 0.5, n_samples)

    # Run reservoir
    states = []
    reservoir.reset()
    for t in range(n_samples):
        x = reservoir.step(u[t:t+1], W_in)
        states.append(x.copy())
    X = np.array(states)

    # Measure correlation with delayed inputs
    memory_capacities = []
    washout = 100

    for delay in range(1, max_delay + 1):
        # Target: u(t - delay)
        target = u[washout:-delay] if delay > 0 else u[washout:]
        features = X[washout + delay:]

        # Linear regression
        W_out = np.linalg.lstsq(features, target, rcond=None)[0]
        prediction = features @ W_out

        # Correlation squared
        correlation = np.corrcoef(prediction, target)[0, 1]
        mc = correlation**2 if not np.isnan(correlation) else 0

        memory_capacities.append(mc)

    total_mc = sum(memory_capacities)
    print(f"Total memory capacity: {total_mc:.2f}")
    return memory_capacities
```

## 5. Echo State Networks

Echo State Networks (ESNs) are the most common form of reservoir computing.

```python
import numpy as np
from scipy.linalg import solve

class EchoStateNetwork:
    """Complete Echo State Network implementation."""

    def __init__(self, n_input, n_reservoir, n_output,
                 spectral_radius=0.9, sparsity=0.1,
                 input_scaling=1.0, regularization=1e-6):
        self.n_input = n_input
        self.n_reservoir = n_reservoir
        self.n_output = n_output
        self.reg = regularization

        # Input weights
        self.W_in = np.random.randn(n_reservoir, n_input) * input_scaling

        # Reservoir weights
        W = np.random.randn(n_reservoir, n_reservoir)
        mask = np.random.random((n_reservoir, n_reservoir)) < sparsity
        W = W * mask
        rho = np.max(np.abs(np.linalg.eigvals(W)))
        self.W = W * (spectral_radius / rho)

        # Output weights (learned)
        self.W_out = None

        # State
        self.state = np.zeros(n_reservoir)

    def _update(self, u):
        """Update reservoir state."""
        self.state = np.tanh(self.W_in @ u + self.W @ self.state)
        return self.state

    def fit(self, inputs, targets, washout=100):
        """
        Train the readout layer.

        Args:
            inputs: Input time series (T, n_input)
            targets: Target outputs (T, n_output)
            washout: Initial steps to discard
        """
        T = len(inputs)
        states = np.zeros((T, self.n_reservoir))

        # Run reservoir
        self.state = np.zeros(self.n_reservoir)
        for t in range(T):
            states[t] = self._update(inputs[t])

        # Discard washout period
        X = states[washout:]
        Y = targets[washout:]

        # Ridge regression: W_out = (X^T X + λI)^{-1} X^T Y
        XTX = X.T @ X + self.reg * np.eye(self.n_reservoir)
        XTY = X.T @ Y
        self.W_out = solve(XTX, XTY)

        return self

    def predict(self, inputs, continuation=True):
        """
        Predict outputs for input sequence.

        Args:
            continuation: If True, continue from current state
        """
        if not continuation:
            self.state = np.zeros(self.n_reservoir)

        predictions = []
        for u in inputs:
            x = self._update(u)
            y = x @ self.W_out
            predictions.append(y)

        return np.array(predictions)

    def generate(self, n_steps, seed_input=None):
        """
        Autonomous generation (for time series forecasting).

        Uses own output as next input.
        """
        if seed_input is not None:
            for u in seed_input:
                self._update(u)

        outputs = []
        y = self.state @ self.W_out

        for _ in range(n_steps):
            x = self._update(y)
            y = x @ self.W_out
            outputs.append(y)

        return np.array(outputs)
```

## 6. Training the Readout

The readout is trained using simple linear regression:

<Formula>
$$W_{out} = (X^TX + \lambda I)^{-1}X^TY$$
</Formula>

This closed-form solution is:
- **Fast**: No iterative optimization needed
- **Optimal**: Globally optimal for linear readout
- **Regularized**: λ prevents overfitting

```python
def train_readout(states, targets, regularization=1e-6):
    """
    Train linear readout using ridge regression.

    Args:
        states: Reservoir states (T, N)
        targets: Target outputs (T, M)
        regularization: L2 regularization parameter

    Returns:
        W_out: Trained output weights (N, M)
    """
    N = states.shape[1]

    # Ridge regression
    XTX = states.T @ states + regularization * np.eye(N)
    XTY = states.T @ targets

    W_out = np.linalg.solve(XTX, XTY)

    return W_out
```

## 7. Example: Mackey-Glass Prediction

The Mackey-Glass system is a classic benchmark for time series prediction:

<Formula>
$$\frac{dx}{dt} = \beta \frac{x(t-\tau)}{1 + x(t-\tau)^n} - \gamma x(t)$$
</Formula>

```python
import numpy as np
import matplotlib.pyplot as plt

def mackey_glass(n_points, tau=17, beta=0.2, gamma=0.1, n=10):
    """Generate Mackey-Glass time series."""
    x = np.zeros(n_points + tau)
    x[:tau] = 1.2 + 0.1 * np.random.randn(tau)

    for t in range(tau, n_points + tau):
        x[t] = x[t-1] + beta * x[t-tau] / (1 + x[t-tau]**n) - gamma * x[t-1]

    return x[tau:]

# Generate data
data = mackey_glass(3000)

# Prepare training data
train_len = 2000
test_len = 500

inputs = data[:-1].reshape(-1, 1)
targets = data[1:].reshape(-1, 1)

train_inputs = inputs[:train_len]
train_targets = targets[:train_len]
test_inputs = inputs[train_len:train_len+test_len]
test_targets = targets[train_len:train_len+test_len]

# Create and train ESN
esn = EchoStateNetwork(
    n_input=1,
    n_reservoir=500,
    n_output=1,
    spectral_radius=0.9
)

esn.fit(train_inputs, train_targets, washout=100)

# Predict
predictions = esn.predict(test_inputs)

# Evaluate
mse = np.mean((predictions - test_targets)**2)
print(f"Test MSE: {mse:.6f}")
```

## 8. Advantages and Limitations

### Advantages

1. **Fast training**: Only linear readout is trained
2. **No gradient issues**: No backpropagation through time
3. **Universal approximation**: Can approximate any fading memory system
4. **Biological plausibility**: Similar to cortical microcircuits

### Limitations

1. **Task-specific tuning**: Hyperparameters depend on the task
2. **Memory-computation trade-off**: Hard to optimize both
3. **Fixed reservoir**: May not be optimal for all tasks
4. **Interpretability**: Hard to understand what reservoir learns

<Tip>
Reservoir computing shines when you need fast training, have limited computational resources, or when temporal processing is the primary challenge. It's especially powerful as a foundation for quantum implementations.
</Tip>

5. **Training** is simple linear regression on readout weights
