## 1. Quantum Control Applications

QRL is naturally suited for learning optimal control sequences for quantum systems.

<Definition term="Quantum Optimal Control">
Finding time-dependent control fields or gate sequences that drive a quantum system to a desired target state or implement a target operation, often under constraints like limited time or fidelity requirements.
</Definition>

### Pulse-Level Control

```python
import numpy as np

class QuantumPulseControl:
    """
    RL environment for learning optimal control pulses.

    Agent learns pulse amplitudes to implement target gates.
    """

    def __init__(self, n_time_steps=20, dt=0.1):
        self.n_time_steps = n_time_steps
        self.dt = dt  # Time step duration

        # Single qubit system
        self.dim = 2

        # Hamiltonian components
        # H = Ω_x(t) σ_x + Ω_y(t) σ_y + Δ σ_z
        self.sigma_x = np.array([[0, 1], [1, 0]], dtype=complex)
        self.sigma_y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.sigma_z = np.array([[1, 0], [0, -1]], dtype=complex)

        self.detuning = 0.0  # Static detuning Δ

        # State and target
        self.current_unitary = None
        self.target_unitary = None
        self.pulse_history = []
        self.step_count = 0

        # Action space: discretized pulse amplitudes
        self.n_amplitude_levels = 11  # -5 to 5
        self.max_amplitude = 5.0
        self.n_actions = self.n_amplitude_levels ** 2  # Ω_x and Ω_y

    def reset(self, target='X'):
        """Reset and set target gate."""
        self.current_unitary = np.eye(self.dim, dtype=complex)
        self.pulse_history = []
        self.step_count = 0

        # Set target unitary
        if target == 'X':
            self.target_unitary = self.sigma_x
        elif target == 'H':
            self.target_unitary = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
        elif target == 'T':
            self.target_unitary = np.array([[1, 0], [0, np.exp(1j * np.pi / 4)]])
        else:
            # Random target
            self.target_unitary = self._random_unitary()

        return self._get_observation()

    def _random_unitary(self):
        """Generate random SU(2) unitary."""
        theta = np.random.uniform(0, np.pi)
        phi = np.random.uniform(0, 2 * np.pi)
        lam = np.random.uniform(0, 2 * np.pi)

        c, s = np.cos(theta/2), np.sin(theta/2)
        return np.array([
            [c, -np.exp(1j * lam) * s],
            [np.exp(1j * phi) * s, np.exp(1j * (phi + lam)) * c]
        ])

    def step(self, action):
        """Apply pulse and evolve system."""
        # Decode action to pulse amplitudes
        omega_x_idx = action // self.n_amplitude_levels
        omega_y_idx = action % self.n_amplitude_levels

        omega_x = (omega_x_idx - 5) * self.max_amplitude / 5
        omega_y = (omega_y_idx - 5) * self.max_amplitude / 5

        self.pulse_history.append((omega_x, omega_y))

        # Construct Hamiltonian
        H = omega_x * self.sigma_x + omega_y * self.sigma_y + self.detuning * self.sigma_z

        # Time evolution U = exp(-i H dt)
        eigenvalues, eigenvectors = np.linalg.eigh(H)
        U_step = eigenvectors @ np.diag(np.exp(-1j * eigenvalues * self.dt)) @ eigenvectors.conj().T

        # Update accumulated unitary
        self.current_unitary = U_step @ self.current_unitary

        self.step_count += 1

        # Compute fidelity
        fidelity = self._compute_gate_fidelity()

        # Reward
        reward = fidelity
        if fidelity > 0.99:
            reward += 10.0

        # Penalty for high amplitude (energy cost)
        energy_penalty = 0.001 * (omega_x**2 + omega_y**2)
        reward -= energy_penalty

        done = self.step_count >= self.n_time_steps or fidelity > 0.999

        info = {
            'fidelity': fidelity,
            'pulse': (omega_x, omega_y),
            'energy': sum(p[0]**2 + p[1]**2 for p in self.pulse_history)
        }

        return self._get_observation(), reward, done, info

    def _compute_gate_fidelity(self):
        """Compute average gate fidelity."""
        # F_avg = (|Tr(U_target† U)|² + d) / (d(d+1))
        trace = np.trace(self.target_unitary.conj().T @ self.current_unitary)
        return (np.abs(trace)**2 + self.dim) / (self.dim * (self.dim + 1))

    def _get_observation(self):
        """Get observation for agent."""
        # Current and target unitary as flattened real/imag
        current_flat = np.concatenate([
            self.current_unitary.real.flatten(),
            self.current_unitary.imag.flatten()
        ])
        target_flat = np.concatenate([
            self.target_unitary.real.flatten(),
            self.target_unitary.imag.flatten()
        ])

        progress = self.step_count / self.n_time_steps
        fidelity = self._compute_gate_fidelity()

        return np.concatenate([current_flat, target_flat, [progress, fidelity]])

# Example: Learning X gate pulse
env = QuantumPulseControl(n_time_steps=10, dt=0.2)
obs = env.reset(target='X')

print("Learning X gate pulse sequence...")
print(f"Initial fidelity: {env._compute_gate_fidelity():.4f}")

# Simple policy: explore
for step in range(10):
    action = np.random.randint(env.n_actions)
    obs, reward, done, info = env.step(action)
    print(f"Step {step+1}: Ω=({info['pulse'][0]:.2f}, {info['pulse'][1]:.2f}), F={info['fidelity']:.4f}")
    if done:
        break

print(f"\nFinal fidelity: {info['fidelity']:.4f}")
```

<Tip>
For pulse control, include energy penalties in the reward to encourage finding low-power control sequences. This is important for hardware implementations where high-power pulses cause heating.
</Tip>

## 2. Circuit Compilation and Optimization

QRL can learn efficient circuit compilations and gate decompositions.

<Formula>
Circuit Optimization Objective:
$$\min_{\text{circuit}} \left( \text{depth} + \lambda \cdot \text{error} \right)$$
subject to: $F(\text{circuit}, U_\text{target}) \geq 1 - \epsilon$
</Formula>

### Implementation

```python
class CircuitOptimizer:
    """
    RL agent for learning optimal circuit compilations.

    Given a target unitary, find a short gate sequence from a native gate set.
    """

    def __init__(self, n_qubits=2, max_depth=30):
        self.n_qubits = n_qubits
        self.dim = 2 ** n_qubits
        self.max_depth = max_depth

        # Native gate set (e.g., for superconducting qubits)
        self.native_gates = self._build_native_gates()
        self.n_actions = len(self.native_gates)

        self.current_circuit = []
        self.current_unitary = None
        self.target_unitary = None

        # Track best solution
        self.best_fidelity = 0
        self.best_circuit = []

    def _build_native_gates(self):
        """Build native gate set."""
        gates = {}

        # Single-qubit gates
        sqrt_x = np.array([[0.5 + 0.5j, 0.5 - 0.5j],
                          [0.5 - 0.5j, 0.5 + 0.5j]])
        rz = lambda theta: np.array([[np.exp(-1j*theta/2), 0],
                                     [0, np.exp(1j*theta/2)]])

        for q in range(self.n_qubits):
            gates[f'SX_{q}'] = self._embed_single(sqrt_x, q)
            gates[f'RZ_pi4_{q}'] = self._embed_single(rz(np.pi/4), q)
            gates[f'RZ_pi2_{q}'] = self._embed_single(rz(np.pi/2), q)
            gates[f'RZ_pi_{q}'] = self._embed_single(rz(np.pi), q)

        # Two-qubit gates (CZ)
        if self.n_qubits >= 2:
            for i in range(self.n_qubits - 1):
                gates[f'CZ_{i}_{i+1}'] = self._build_cz(i, i+1)

        return gates

    def _embed_single(self, gate, qubit):
        """Embed single-qubit gate in full Hilbert space."""
        if self.n_qubits == 1:
            return gate

        result = np.eye(self.dim, dtype=complex)
        for i in range(self.dim):
            for j in range(self.dim):
                i_bit = (i >> qubit) & 1
                j_bit = (j >> qubit) & 1
                i_rest = i ^ (i_bit << qubit)
                j_rest = j ^ (j_bit << qubit)

                if i_rest == j_rest:
                    result[i, j] = gate[i_bit, j_bit]
                else:
                    result[i, j] = 0

        return result

    def _build_cz(self, q1, q2):
        """Build CZ gate matrix."""
        cz = np.eye(self.dim, dtype=complex)
        for i in range(self.dim):
            if ((i >> q1) & 1) and ((i >> q2) & 1):
                cz[i, i] = -1
        return cz

    def reset(self, target_unitary=None):
        """Reset with new target."""
        self.current_circuit = []
        self.current_unitary = np.eye(self.dim, dtype=complex)

        if target_unitary is not None:
            self.target_unitary = target_unitary
        else:
            self.target_unitary = self._random_unitary()

        self.best_fidelity = 0
        self.best_circuit = []

        return self._get_observation()

    def _random_unitary(self):
        """Generate random unitary."""
        real = np.random.randn(self.dim, self.dim)
        imag = np.random.randn(self.dim, self.dim)
        A = real + 1j * imag
        Q, R = np.linalg.qr(A)
        D = np.diag(np.diag(R) / np.abs(np.diag(R)))
        return Q @ D

    def step(self, action):
        """Add gate to circuit."""
        gate_name = list(self.native_gates.keys())[action]
        gate_matrix = self.native_gates[gate_name]

        # Add to circuit
        self.current_circuit.append(gate_name)
        self.current_unitary = gate_matrix @ self.current_unitary

        # Compute fidelity
        fidelity = self._compute_fidelity()

        # Track best
        if fidelity > self.best_fidelity:
            self.best_fidelity = fidelity
            self.best_circuit = self.current_circuit.copy()

        # Reward design
        reward = fidelity

        # Depth penalty
        depth_penalty = 0.005 * len(self.current_circuit)
        reward -= depth_penalty

        # Bonus for good fidelity with short circuit
        if fidelity > 0.99:
            efficiency_bonus = 10.0 / (1 + len(self.current_circuit))
            reward += efficiency_bonus

        done = len(self.current_circuit) >= self.max_depth or fidelity > 0.9999

        info = {
            'fidelity': fidelity,
            'depth': len(self.current_circuit),
            'gate': gate_name,
            'best_fidelity': self.best_fidelity,
            'best_depth': len(self.best_circuit)
        }

        return self._get_observation(), reward, done, info

    def _compute_fidelity(self):
        """Average gate fidelity."""
        trace = np.trace(self.target_unitary.conj().T @ self.current_unitary)
        return (np.abs(trace)**2 + self.dim) / (self.dim * (self.dim + 1))

    def _get_observation(self):
        """Observation for agent."""
        current_flat = np.concatenate([
            self.current_unitary.real.flatten(),
            self.current_unitary.imag.flatten()
        ])
        target_flat = np.concatenate([
            self.target_unitary.real.flatten(),
            self.target_unitary.imag.flatten()
        ])

        progress = len(self.current_circuit) / self.max_depth
        fidelity = self._compute_fidelity()

        return np.concatenate([current_flat, target_flat, [progress, fidelity]])

    def get_best_circuit(self):
        """Return the best circuit found."""
        return self.best_circuit, self.best_fidelity

# Example
optimizer = CircuitOptimizer(n_qubits=1, max_depth=20)

# Target: Hadamard gate
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
obs = optimizer.reset(target_unitary=H)

print("Optimizing circuit for Hadamard gate...")
print(f"Native gates: {list(optimizer.native_gates.keys())}")

for step in range(20):
    action = np.random.randint(optimizer.n_actions)
    obs, reward, done, info = optimizer.step(action)

    if step % 5 == 0 or done:
        print(f"Step {step+1}: depth={info['depth']}, fidelity={info['fidelity']:.4f}")

    if done:
        break

circuit, fidelity = optimizer.get_best_circuit()
print(f"\nBest circuit: {circuit}")
print(f"Best fidelity: {fidelity:.4f}")
```

<Warning>
Circuit optimization with RL can get stuck in local optima. Use exploration strategies like entropy bonuses or curiosity-driven exploration to discover novel gate sequences.
</Warning>

## 3. Variational Quantum Eigensolver Control

QRL can adaptively select ansatz operations for VQE, learning problem-specific circuit structures.

<Definition term="Adaptive VQE">
A variational approach where the circuit structure is learned alongside the parameters, potentially discovering more efficient ansätze than fixed designs.
</Definition>

### Implementation

```python
class AdaptiveVQE:
    """
    RL-guided adaptive VQE for finding ground states.

    Agent selects which gates to add to the ansatz based on gradient information.
    """

    def __init__(self, n_qubits=2, max_layers=10):
        self.n_qubits = n_qubits
        self.dim = 2 ** n_qubits
        self.max_layers = max_layers

        # Hamiltonian (example: H2 in minimal basis)
        self.hamiltonian = self._build_h2_hamiltonian()
        self.exact_ground_energy = self._exact_diagonalization()

        # Gate pool for ADAPT-VQE style selection
        self.gate_pool = self._build_gate_pool()
        self.n_actions = len(self.gate_pool)

        # Current circuit
        self.circuit_gates = []
        self.circuit_params = []

    def _build_h2_hamiltonian(self):
        """Build simplified H2 Hamiltonian."""
        I = np.eye(2)
        X = np.array([[0, 1], [1, 0]])
        Y = np.array([[0, -1j], [1j, 0]])
        Z = np.array([[1, 0], [0, -1]])

        # H = g0*I + g1*Z0 + g2*Z1 + g3*Z0Z1 + g4*X0X1 + g5*Y0Y1
        # Simplified coefficients
        g0, g1, g2, g3, g4, g5 = -0.5, 0.2, -0.2, 0.1, 0.05, 0.05

        II = np.kron(I, I)
        IZ = np.kron(I, Z)
        ZI = np.kron(Z, I)
        ZZ = np.kron(Z, Z)
        XX = np.kron(X, X)
        YY = np.kron(Y, Y)

        H = g0 * II + g1 * ZI + g2 * IZ + g3 * ZZ + g4 * XX + g5 * YY

        return H

    def _exact_diagonalization(self):
        """Find exact ground state energy."""
        eigenvalues = np.linalg.eigvalsh(self.hamiltonian)
        return np.min(eigenvalues)

    def _build_gate_pool(self):
        """Build pool of parameterized gates."""
        pool = []

        # Single-qubit rotations
        for q in range(self.n_qubits):
            pool.append(('RY', q))
            pool.append(('RZ', q))

        # Two-qubit excitations (ADAPT-VQE style)
        for i in range(self.n_qubits):
            for j in range(i + 1, self.n_qubits):
                pool.append(('CNOT_RY_CNOT', i, j))  # Entangling block

        return pool

    def reset(self):
        """Reset circuit."""
        self.circuit_gates = []
        self.circuit_params = []
        return self._get_observation()

    def step(self, action):
        """Add selected gate and optimize its parameter."""
        gate = self.gate_pool[action]

        # Add gate with initial parameter
        self.circuit_gates.append(gate)

        # Optimize new parameter via gradient descent
        initial_param = 0.0
        optimized_param = self._optimize_new_parameter(initial_param)
        self.circuit_params.append(optimized_param)

        # Compute energy
        energy = self._compute_energy()
        error = energy - self.exact_ground_energy

        # Reward: negative energy (want to minimize)
        reward = -energy

        # Bonus for approaching ground state
        if error < 0.1:
            reward += 1.0
        if error < 0.01:
            reward += 5.0

        done = len(self.circuit_gates) >= self.max_layers or error < 0.001

        info = {
            'energy': energy,
            'error': error,
            'exact_energy': self.exact_ground_energy,
            'depth': len(self.circuit_gates),
            'gate_added': gate
        }

        return self._get_observation(), reward, done, info

    def _build_circuit_unitary(self, params=None):
        """Build unitary from current circuit."""
        if params is None:
            params = self.circuit_params

        U = np.eye(self.dim, dtype=complex)

        for gate, param in zip(self.circuit_gates, params):
            if gate[0] == 'RY':
                qubit = gate[1]
                gate_matrix = self._ry_gate(param, qubit)
            elif gate[0] == 'RZ':
                qubit = gate[1]
                gate_matrix = self._rz_gate(param, qubit)
            elif gate[0] == 'CNOT_RY_CNOT':
                q1, q2 = gate[1], gate[2]
                gate_matrix = self._entangling_block(param, q1, q2)
            else:
                gate_matrix = np.eye(self.dim, dtype=complex)

            U = gate_matrix @ U

        return U

    def _ry_gate(self, theta, qubit):
        """RY gate embedded in full space."""
        c, s = np.cos(theta/2), np.sin(theta/2)
        ry = np.array([[c, -s], [s, c]])
        return self._embed_single(ry, qubit)

    def _rz_gate(self, theta, qubit):
        """RZ gate embedded in full space."""
        rz = np.array([[np.exp(-1j*theta/2), 0], [0, np.exp(1j*theta/2)]])
        return self._embed_single(rz, qubit)

    def _embed_single(self, gate, qubit):
        """Embed single-qubit gate."""
        result = np.eye(self.dim, dtype=complex)
        for i in range(self.dim):
            for j in range(self.dim):
                i_bit = (i >> qubit) & 1
                j_bit = (j >> qubit) & 1
                i_rest = i ^ (i_bit << qubit)
                j_rest = j ^ (j_bit << qubit)

                if i_rest == j_rest:
                    result[i, j] = gate[i_bit, j_bit]
                else:
                    result[i, j] = 0
        return result

    def _entangling_block(self, theta, q1, q2):
        """CNOT - RY - CNOT entangling block."""
        cnot = self._build_cnot(q1, q2)
        ry = self._ry_gate(theta, q2)
        return cnot @ ry @ cnot

    def _build_cnot(self, control, target):
        """Build CNOT gate."""
        cnot = np.eye(self.dim, dtype=complex)
        for i in range(self.dim):
            if (i >> control) & 1:
                j = i ^ (1 << target)
                cnot[i, i] = 0
                cnot[j, i] = 1
        return cnot

    def _compute_energy(self, params=None):
        """Compute expectation value of Hamiltonian."""
        U = self._build_circuit_unitary(params)

        # Initial state |00...0⟩
        psi0 = np.zeros(self.dim, dtype=complex)
        psi0[0] = 1.0

        psi = U @ psi0
        energy = np.real(psi.conj() @ self.hamiltonian @ psi)

        return energy

    def _optimize_new_parameter(self, initial, n_steps=20, lr=0.1):
        """Optimize newly added parameter."""
        param = initial

        for _ in range(n_steps):
            # Compute gradient via parameter shift
            params_plus = self.circuit_params + [param + np.pi/2]
            params_minus = self.circuit_params + [param - np.pi/2]

            E_plus = self._compute_energy(params_plus)
            E_minus = self._compute_energy(params_minus)

            grad = (E_plus - E_minus) / 2
            param -= lr * grad

        return param

    def _get_observation(self):
        """Get observation."""
        energy = self._compute_energy() if self.circuit_params else 0.0
        depth = len(self.circuit_gates)

        # One-hot encoding of recent gates
        recent_gates = np.zeros(self.n_actions)
        for gate in self.circuit_gates[-3:]:
            idx = self.gate_pool.index(gate)
            recent_gates[idx] = 1

        return np.concatenate([[energy, depth / self.max_layers], recent_gates])

# Example
vqe = AdaptiveVQE(n_qubits=2, max_layers=8)
obs = vqe.reset()

print(f"Exact ground energy: {vqe.exact_ground_energy:.6f}")
print(f"Gate pool: {vqe.gate_pool}")

print("\nAdaptive VQE optimization...")
for step in range(8):
    action = np.random.randint(vqe.n_actions)
    obs, reward, done, info = vqe.step(action)
    print(f"Step {step+1}: Added {info['gate_added']}, E={info['energy']:.6f}, error={info['error']:.6f}")

    if done:
        break

print(f"\nFinal energy: {info['energy']:.6f}")
print(f"Error from exact: {info['error']:.6f}")
```

## 4. Combinatorial Optimization

QRL can solve combinatorial optimization by learning to construct or improve solutions.

<Formula>
MaxCut Problem:
$$\max_{\mathbf{x} \in \{0,1\}^n} \sum_{(i,j) \in E} w_{ij} (x_i \oplus x_j)$$

QRL maps this to finding quantum states that encode optimal bit strings.
</Formula>

### Implementation

```python
class MaxCutQRL:
    """
    QRL environment for MaxCut optimization.

    Agent learns gate sequences that prepare states encoding good cuts.
    """

    def __init__(self, n_nodes=4, max_steps=20):
        self.n_nodes = n_nodes
        self.n_qubits = n_nodes
        self.dim = 2 ** n_qubits
        self.max_steps = max_steps

        # Generate random graph
        self.adjacency = self._generate_random_graph()
        self.optimal_cut = self._brute_force_optimal()

        # Quantum state
        self.state = None
        self.step_count = 0

        # Actions: gates on each qubit
        self.n_actions = 3 * n_nodes + (n_nodes - 1)  # RY, RZ, X per qubit + CNOTs

    def _generate_random_graph(self):
        """Generate random weighted graph."""
        adj = np.zeros((self.n_nodes, self.n_nodes))

        # Random edges with random weights
        for i in range(self.n_nodes):
            for j in range(i + 1, self.n_nodes):
                if np.random.random() < 0.5:  # 50% edge probability
                    weight = np.random.uniform(0.5, 2.0)
                    adj[i, j] = weight
                    adj[j, i] = weight

        return adj

    def _brute_force_optimal(self):
        """Find optimal cut by brute force."""
        best_cut = 0
        best_assignment = None

        for x in range(2 ** self.n_nodes):
            cut_value = self._compute_cut_value(x)
            if cut_value > best_cut:
                best_cut = cut_value
                best_assignment = x

        return best_cut

    def _compute_cut_value(self, assignment):
        """Compute cut value for binary assignment."""
        cut = 0
        for i in range(self.n_nodes):
            for j in range(i + 1, self.n_nodes):
                if self.adjacency[i, j] > 0:
                    bit_i = (assignment >> i) & 1
                    bit_j = (assignment >> j) & 1
                    if bit_i != bit_j:
                        cut += self.adjacency[i, j]
        return cut

    def reset(self):
        """Reset to uniform superposition."""
        self.state = np.ones(self.dim, dtype=complex) / np.sqrt(self.dim)
        self.step_count = 0
        return self._get_observation()

    def step(self, action):
        """Apply gate and measure expected cut value."""
        self._apply_action(action)
        self.step_count += 1

        # Compute expected cut value
        expected_cut = self._compute_expected_cut()

        # Reward: normalized cut value
        reward = expected_cut / max(self.optimal_cut, 1e-6)

        # Bonus for good approximation
        if expected_cut > 0.9 * self.optimal_cut:
            reward += 5.0

        done = self.step_count >= self.max_steps

        info = {
            'expected_cut': expected_cut,
            'optimal_cut': self.optimal_cut,
            'approximation_ratio': expected_cut / max(self.optimal_cut, 1e-6)
        }

        return self._get_observation(), reward, done, info

    def _apply_action(self, action):
        """Apply gate based on action."""
        if action < self.n_nodes:
            # RY on qubit
            qubit = action
            angle = np.pi / 4
            self._apply_ry(angle, qubit)
        elif action < 2 * self.n_nodes:
            # RZ on qubit
            qubit = action - self.n_nodes
            angle = np.pi / 4
            self._apply_rz(angle, qubit)
        elif action < 3 * self.n_nodes:
            # X on qubit
            qubit = action - 2 * self.n_nodes
            self._apply_x(qubit)
        else:
            # CNOT
            cnot_idx = action - 3 * self.n_nodes
            self._apply_cnot(cnot_idx, cnot_idx + 1)

    def _apply_ry(self, angle, qubit):
        """Apply RY gate."""
        c, s = np.cos(angle/2), np.sin(angle/2)
        new_state = np.zeros(self.dim, dtype=complex)

        for i in range(self.dim):
            bit = (i >> qubit) & 1
            i_f = i ^ (1 << qubit)
            if bit == 0:
                new_state[i] += c * self.state[i]
                new_state[i_f] += s * self.state[i]
            else:
                new_state[i] += c * self.state[i]
                new_state[i_f] -= s * self.state[i]

        self.state = new_state

    def _apply_rz(self, angle, qubit):
        """Apply RZ gate."""
        for i in range(self.dim):
            bit = (i >> qubit) & 1
            phase = np.exp(-1j * angle/2) if bit == 0 else np.exp(1j * angle/2)
            self.state[i] *= phase

    def _apply_x(self, qubit):
        """Apply X gate."""
        new_state = np.zeros(self.dim, dtype=complex)
        for i in range(self.dim):
            new_state[i ^ (1 << qubit)] = self.state[i]
        self.state = new_state

    def _apply_cnot(self, control, target):
        """Apply CNOT gate."""
        new_state = np.zeros(self.dim, dtype=complex)
        for i in range(self.dim):
            if (i >> control) & 1:
                new_state[i ^ (1 << target)] = self.state[i]
            else:
                new_state[i] = self.state[i]
        self.state = new_state

    def _compute_expected_cut(self):
        """Compute expected cut value from quantum state."""
        expected = 0

        for x in range(self.dim):
            prob = np.abs(self.state[x]) ** 2
            cut = self._compute_cut_value(x)
            expected += prob * cut

        return expected

    def _get_observation(self):
        """Get observation."""
        # Measurement probabilities for each qubit being |1⟩
        qubit_probs = []
        for q in range(self.n_qubits):
            prob_1 = sum(np.abs(self.state[i])**2
                        for i in range(self.dim) if (i >> q) & 1)
            qubit_probs.append(prob_1)

        expected_cut = self._compute_expected_cut()
        progress = self.step_count / self.max_steps

        return np.array(qubit_probs + [expected_cut / self.optimal_cut, progress])
```

## 5. Quantum Error Correction Control

QRL can learn adaptive error correction strategies.

<Definition term="Adaptive QEC">
Using reinforcement learning to decide when and how to apply error correction operations based on syndrome measurements, potentially achieving better protection than fixed protocols.
</Definition>

### Implementation

```python
class QECController:
    """
    RL controller for quantum error correction.

    Simple 3-qubit bit-flip code with adaptive correction.
    """

    def __init__(self, error_rate=0.05, max_cycles=20):
        self.n_data_qubits = 3
        self.n_ancilla = 2
        self.error_rate = error_rate
        self.max_cycles = max_cycles

        # Logical state (encoded in 3 qubits)
        self.logical_state = None  # |0_L⟩ = |000⟩, |1_L⟩ = |111⟩

        self.syndrome_history = []
        self.cycle_count = 0

        # Actions: no correction, correct qubit 0, 1, or 2
        self.n_actions = 4

    def reset(self, logical_value=0):
        """Initialize logical qubit."""
        self.logical_state = np.zeros(8, dtype=complex)

        if logical_value == 0:
            self.logical_state[0] = 1.0  # |000⟩
        else:
            self.logical_state[7] = 1.0  # |111⟩

        self.syndrome_history = []
        self.cycle_count = 0
        self.initial_logical = logical_value

        return self._get_observation()

    def step(self, action):
        """
        One QEC cycle:
        1. Apply correction based on action
        2. Apply random errors
        3. Measure syndromes
        """
        # Apply correction
        if action > 0:
            qubit = action - 1
            self._apply_x(qubit)

        # Apply random errors
        self._apply_errors()

        # Measure syndromes
        syndrome = self._measure_syndrome()
        self.syndrome_history.append(syndrome)

        self.cycle_count += 1

        # Compute logical fidelity
        fidelity = self._compute_logical_fidelity()

        # Reward: maintain fidelity
        reward = fidelity

        # Penalty for unnecessary corrections
        if action > 0 and syndrome == (0, 0):
            reward -= 0.1  # Correcting when no error

        done = self.cycle_count >= self.max_cycles or fidelity < 0.5

        info = {
            'fidelity': fidelity,
            'syndrome': syndrome,
            'action': action,
            'errors_occurred': self._count_errors()
        }

        return self._get_observation(), reward, done, info

    def _apply_x(self, qubit):
        """Apply X gate (bit flip) to qubit."""
        new_state = np.zeros(8, dtype=complex)
        for i in range(8):
            new_state[i ^ (1 << qubit)] = self.logical_state[i]
        self.logical_state = new_state

    def _apply_errors(self):
        """Apply random bit-flip errors."""
        for qubit in range(self.n_data_qubits):
            if np.random.random() < self.error_rate:
                self._apply_x(qubit)

    def _measure_syndrome(self):
        """
        Measure parity check syndromes.

        S1 = Z_0 Z_1 (parity of qubits 0,1)
        S2 = Z_1 Z_2 (parity of qubits 1,2)
        """
        # Compute syndrome from state
        s1 = 0.0  # ⟨Z_0 Z_1⟩
        s2 = 0.0  # ⟨Z_1 Z_2⟩

        for i in range(8):
            prob = np.abs(self.logical_state[i]) ** 2

            b0 = (i >> 0) & 1
            b1 = (i >> 1) & 1
            b2 = (i >> 2) & 1

            s1 += prob * (1 - 2 * (b0 ^ b1))
            s2 += prob * (1 - 2 * (b1 ^ b2))

        # Threshold to binary
        syndrome_1 = 0 if s1 > 0 else 1
        syndrome_2 = 0 if s2 > 0 else 1

        return (syndrome_1, syndrome_2)

    def _compute_logical_fidelity(self):
        """Compute fidelity with correct logical state."""
        if self.initial_logical == 0:
            # Should be |000⟩
            return np.abs(self.logical_state[0]) ** 2
        else:
            # Should be |111⟩
            return np.abs(self.logical_state[7]) ** 2

    def _count_errors(self):
        """Count number of bit flips from logical state."""
        target = 0 if self.initial_logical == 0 else 7

        weighted_errors = 0
        for i in range(8):
            prob = np.abs(self.logical_state[i]) ** 2
            errors = bin(i ^ target).count('1')
            weighted_errors += prob * errors

        return weighted_errors

    def _get_observation(self):
        """Get observation for agent."""
        # Recent syndromes
        recent = self.syndrome_history[-5:] if self.syndrome_history else []
        syndrome_features = []
        for s in recent:
            syndrome_features.extend([s[0], s[1]])

        # Pad if needed
        while len(syndrome_features) < 10:
            syndrome_features.append(0)

        fidelity = self._compute_logical_fidelity()
        progress = self.cycle_count / self.max_cycles

        return np.array(syndrome_features + [fidelity, progress])

# Example
qec = QECController(error_rate=0.1, max_cycles=15)
obs = qec.reset(logical_value=0)

print("QEC simulation (3-qubit bit-flip code)")
print(f"Error rate: {qec.error_rate}")

# Simple strategy: correct based on syndrome
def syndrome_decoder(syndrome):
    """Standard syndrome decoding."""
    s1, s2 = syndrome
    if s1 == 0 and s2 == 0:
        return 0  # No error
    elif s1 == 1 and s2 == 0:
        return 1  # Error on qubit 0
    elif s1 == 1 and s2 == 1:
        return 2  # Error on qubit 1
    else:  # s1 == 0 and s2 == 1
        return 3  # Error on qubit 2

print("\nRunning QEC cycles...")
for cycle in range(15):
    # Get current syndrome
    syndrome = qec._measure_syndrome()
    action = syndrome_decoder(syndrome)

    obs, reward, done, info = qec.step(action)

    print(f"Cycle {cycle+1}: syndrome={info['syndrome']}, action={action}, "
          f"fidelity={info['fidelity']:.4f}")

    if done:
        break

print(f"\nFinal fidelity: {info['fidelity']:.4f}")
```

## 6. Benchmarking and Comparison

Systematic evaluation is essential for QRL applications.

<Formula>
Performance Metrics:
- **Fidelity**: $F = |\langle\psi_\text{target}|\psi\rangle|^2$
- **Approximation Ratio**: $r = \text{solution} / \text{optimal}$
- **Sample Efficiency**: Episodes to reach threshold
- **Circuit Depth**: Total gates used
</Formula>

### Implementation

```python
class QRLBenchmarkSuite:
    """
    Benchmark suite for evaluating QRL agents.
    """

    def __init__(self):
        self.results = {}

    def benchmark_state_prep(self, agent, n_episodes=50):
        """Benchmark state preparation."""
        from typing import List

        env = StatePreparationEnv(n_qubits=2, target_type='bell', max_steps=20)

        metrics = {
            'final_fidelities': [],
            'steps_to_success': [],
            'episode_rewards': []
        }

        for ep in range(n_episodes):
            obs = env.reset()
            total_reward = 0
            success_step = None

            for step in range(20):
                action = agent.select_action(obs)
                obs, reward, done, info = env.step(action)
                total_reward += reward

                if success_step is None and info['fidelity'] > 0.99:
                    success_step = step + 1

                if done:
                    break

            metrics['final_fidelities'].append(info['fidelity'])
            metrics['steps_to_success'].append(success_step if success_step else 20)
            metrics['episode_rewards'].append(total_reward)

        return {
            'mean_fidelity': np.mean(metrics['final_fidelities']),
            'std_fidelity': np.std(metrics['final_fidelities']),
            'success_rate': np.mean(np.array(metrics['final_fidelities']) > 0.99),
            'mean_steps': np.mean(metrics['steps_to_success']),
            'mean_reward': np.mean(metrics['episode_rewards'])
        }

    def benchmark_circuit_opt(self, agent, n_episodes=30):
        """Benchmark circuit optimization."""
        env = CircuitOptimizer(n_qubits=1, max_depth=15)

        metrics = {
            'final_fidelities': [],
            'circuit_depths': [],
            'efficiency': []  # fidelity / depth
        }

        for ep in range(n_episodes):
            obs = env.reset()

            for step in range(15):
                action = agent.select_action(obs)
                obs, reward, done, info = env.step(action)

                if done:
                    break

            circuit, fidelity = env.get_best_circuit()
            metrics['final_fidelities'].append(fidelity)
            metrics['circuit_depths'].append(len(circuit))
            metrics['efficiency'].append(fidelity / max(len(circuit), 1))

        return {
            'mean_fidelity': np.mean(metrics['final_fidelities']),
            'mean_depth': np.mean(metrics['circuit_depths']),
            'mean_efficiency': np.mean(metrics['efficiency'])
        }

    def benchmark_maxcut(self, agent, n_episodes=30):
        """Benchmark MaxCut optimization."""
        metrics = {
            'approximation_ratios': [],
            'expected_cuts': []
        }

        for ep in range(n_episodes):
            env = MaxCutQRL(n_nodes=4, max_steps=15)
            obs = env.reset()

            for step in range(15):
                action = agent.select_action(obs)
                obs, reward, done, info = env.step(action)

                if done:
                    break

            metrics['approximation_ratios'].append(info['approximation_ratio'])
            metrics['expected_cuts'].append(info['expected_cut'])

        return {
            'mean_ratio': np.mean(metrics['approximation_ratios']),
            'std_ratio': np.std(metrics['approximation_ratios']),
            'best_ratio': np.max(metrics['approximation_ratios'])
        }

    def run_full_benchmark(self, agent, agent_name='Agent'):
        """Run complete benchmark suite."""
        print(f"\n{'='*60}")
        print(f"Benchmarking: {agent_name}")
        print('='*60)

        print("\n1. State Preparation (Bell State)")
        state_results = self.benchmark_state_prep(agent)
        print(f"   Mean Fidelity: {state_results['mean_fidelity']:.4f} ± {state_results['std_fidelity']:.4f}")
        print(f"   Success Rate: {state_results['success_rate']*100:.1f}%")
        print(f"   Mean Steps: {state_results['mean_steps']:.1f}")

        print("\n2. Circuit Optimization")
        circuit_results = self.benchmark_circuit_opt(agent)
        print(f"   Mean Fidelity: {circuit_results['mean_fidelity']:.4f}")
        print(f"   Mean Depth: {circuit_results['mean_depth']:.1f}")
        print(f"   Efficiency: {circuit_results['mean_efficiency']:.4f}")

        print("\n3. MaxCut Optimization")
        maxcut_results = self.benchmark_maxcut(agent)
        print(f"   Mean Approximation Ratio: {maxcut_results['mean_ratio']:.4f}")
        print(f"   Best Ratio: {maxcut_results['best_ratio']:.4f}")

        self.results[agent_name] = {
            'state_prep': state_results,
            'circuit_opt': circuit_results,
            'maxcut': maxcut_results
        }

        return self.results[agent_name]

# Example with random agent
class RandomAgent:
```

## Summary

Quantum reinforcement learning has diverse applications:

1. **Quantum control**: Learning optimal pulses and gate sequences for state preparation
2. **Circuit optimization**: Finding efficient compilations to native gate sets
3. **Adaptive VQE**: Discovering problem-specific ansätze for variational algorithms
4. **Combinatorial optimization**: Solving MaxCut and other NP-hard problems
5. **Error correction**: Adaptive syndrome decoding and correction strategies
6. **Benchmarking**: Systematic evaluation across application domains

The key to successful QRL applications is careful reward design, appropriate observation spaces, and thorough benchmarking.
