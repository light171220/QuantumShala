# Quantum Fisher Information

The Quantum Fisher Information (QFI) extends classical Fisher information to quantum states, providing fundamental limits on parameter estimation precision and defining the natural geometry of quantum state space.

## Learning Objectives

- Define quantum Fisher information for pure and mixed states
- Compute QFI using different methods
- Understand the quantum Cramér-Rao bound
- Connect QFI to quantum metrology
- Implement QFI computation for variational circuits
- Apply QFI to quantum parameter estimation

## 1. QFI for Pure States

For pure quantum states, the QFI takes a particularly elegant form.

<Definition term="Quantum Fisher Information (Pure States)">
For a parameterized pure state $|\psi(\theta)\rangle$, the QFI is:
$$F_Q(\theta) = 4\left(\langle\partial_\theta\psi|\partial_\theta\psi\rangle - |\langle\psi|\partial_\theta\psi\rangle|^2\right)$$

This equals four times the variance of the generator of the parameter transformation.
</Definition>

### Implementation

```python
import numpy as np

class PureStateQFI:
    """
    Compute Quantum Fisher Information for pure states.
    """

    def __init__(self, circuit_fn, n_qubits):
        """
        Args:
            circuit_fn: Function that returns |ψ(θ)⟩ given parameters θ
            n_qubits: Number of qubits
        """
        self.circuit_fn = circuit_fn
        self.n_qubits = n_qubits
        self.dim = 2 ** n_qubits

    def compute_state_derivative(self, params, param_idx, eps=1e-5):
        """
        Compute ∂|ψ⟩/∂θ_i using finite differences.
        """
        params_plus = params.copy()
        params_plus[param_idx] += eps

        params_minus = params.copy()
        params_minus[param_idx] -= eps

        psi_plus = self.circuit_fn(params_plus)
        psi_minus = self.circuit_fn(params_minus)

        return (psi_plus - psi_minus) / (2 * eps)

    def qfi_single_param(self, params, param_idx):
        """
        Compute QFI for a single parameter.

        F_Q = 4(⟨∂ψ|∂ψ⟩ - |⟨ψ|∂ψ⟩|²)
        """
        psi = self.circuit_fn(params)
        dpsi = self.compute_state_derivative(params, param_idx)

        # ⟨∂ψ|∂ψ⟩
        dpsi_dpsi = np.vdot(dpsi, dpsi)

        # ⟨ψ|∂ψ⟩
        psi_dpsi = np.vdot(psi, dpsi)

        qfi = 4 * (np.real(dpsi_dpsi) - np.abs(psi_dpsi)**2)
        return qfi

    def qfi_matrix(self, params):
        """
        Compute full QFI matrix for all parameters.

        [F_Q]_ij = 4 Re(⟨∂_iψ|∂_jψ⟩ - ⟨∂_iψ|ψ⟩⟨ψ|∂_jψ⟩)
        """
        n_params = len(params)
        psi = self.circuit_fn(params)

        # Compute all derivatives
        dpsi = [self.compute_state_derivative(params, i) for i in range(n_params)]

        # Compute QFI matrix
        qfi = np.zeros((n_params, n_params))

        for i in range(n_params):
            for j in range(n_params):
                # ⟨∂_iψ|∂_jψ⟩
                dpsi_i_dpsi_j = np.vdot(dpsi[i], dpsi[j])

                # ⟨∂_iψ|ψ⟩⟨ψ|∂_jψ⟩
                dpsi_i_psi = np.vdot(dpsi[i], psi)
                psi_dpsi_j = np.vdot(psi, dpsi[j])

                qfi[i, j] = 4 * np.real(dpsi_i_dpsi_j - dpsi_i_psi * psi_dpsi_j)

        return qfi


# Example: Single qubit rotation
def single_qubit_circuit(params):
    """Create state |ψ(θ)⟩ = Ry(θ)|0⟩."""
    theta = params[0]
    psi = np.array([np.cos(theta/2), np.sin(theta/2)], dtype=complex)
    return psi

qfi_calculator = PureStateQFI(single_qubit_circuit, n_qubits=1)

# QFI is constant = 1 for Ry rotation
for theta in [0, np.pi/4, np.pi/2, np.pi]:
    qfi = qfi_calculator.qfi_single_param(np.array([theta]), 0)
    print(f"θ = {theta:.4f}: QFI = {qfi:.4f}")

print("\nExpected: QFI = 1 for single-qubit rotation")

# Two-parameter example
def two_param_circuit(params):
    """Create state Rz(φ)Ry(θ)|0⟩."""
    theta, phi = params

    # Ry(θ)|0⟩
    psi = np.array([np.cos(theta/2), np.sin(theta/2)], dtype=complex)

    # Rz(φ)
    psi[0] *= np.exp(-1j * phi/2)
    psi[1] *= np.exp(1j * phi/2)

    return psi

qfi_calc2 = PureStateQFI(two_param_circuit, n_qubits=1)
params = np.array([np.pi/4, np.pi/3])
qfi_matrix = qfi_calc2.qfi_matrix(params)
print(f"\nQFI Matrix for Rz(φ)Ry(θ)|0⟩:")
print(np.round(qfi_matrix, 4))
```

For pure states, the QFI equals 4 times the Fubini-Study metric. This connects quantum estimation theory to the natural geometry of quantum state space.
</Tip>

## 2. QFI for Mixed States

For mixed states (density matrices), QFI requires the symmetric logarithmic derivative.

<Formula>
Symmetric Logarithmic Derivative (SLD):
$$\frac{\partial \rho}{\partial \theta} = \frac{1}{2}\left(L_\theta \rho + \rho L_\theta\right)$$

Quantum Fisher Information:
$$F_Q(\theta) = \text{Tr}(\rho L_\theta^2)$$
</Formula>

### Implementation

```python
class MixedStateQFI:
    """
    Compute QFI for mixed quantum states using density matrices.
    """

    def __init__(self, rho_fn):
        """
        Args:
            rho_fn: Function that returns density matrix ρ(θ)
        """
        self.rho_fn = rho_fn

    def compute_rho_derivative(self, params, param_idx, eps=1e-5):
        """Compute ∂ρ/∂θ using finite differences."""
        params_plus = params.copy()
        params_plus[param_idx] += eps

        params_minus = params.copy()
        params_minus[param_idx] -= eps

        rho_plus = self.rho_fn(params_plus)
        rho_minus = self.rho_fn(params_minus)

        return (rho_plus - rho_minus) / (2 * eps)

    def compute_sld(self, rho, drho, reg=1e-10):
        """
        Compute Symmetric Logarithmic Derivative.

        Solves: drho = (L ρ + ρ L) / 2
        """
        dim = rho.shape[0]

        # Eigendecomposition of ρ
        eigenvalues, eigenvectors = np.linalg.eigh(rho)
        eigenvalues = np.maximum(eigenvalues, reg)

        # Compute SLD in eigenbasis
        L = np.zeros((dim, dim), dtype=complex)

        # drho in eigenbasis
        drho_eig = eigenvectors.conj().T @ drho @ eigenvectors

        for i in range(dim):
            for j in range(dim):
                denom = eigenvalues[i] + eigenvalues[j]
                if denom > 2 * reg:
                    L[i, j] = 2 * drho_eig[i, j] / denom

        # Transform back to original basis
        L = eigenvectors @ L @ eigenvectors.conj().T

        return L

    def qfi_single_param(self, params, param_idx):
        """
        Compute QFI for single parameter.

        F_Q = Tr(ρ L²)
        """
        rho = self.rho_fn(params)
        drho = self.compute_rho_derivative(params, param_idx)

        L = self.compute_sld(rho, drho)

        # F_Q = Tr(ρ L²)
        qfi = np.real(np.trace(rho @ L @ L))

        return qfi

    def qfi_matrix(self, params):
        """
        Compute full QFI matrix.

        [F_Q]_ij = Re(Tr(ρ L_i L_j))
        """
        n_params = len(params)
        rho = self.rho_fn(params)

        # Compute all SLDs
        slds = []
        for i in range(n_params):
            drho = self.compute_rho_derivative(params, i)
            L = self.compute_sld(rho, drho)
            slds.append(L)

        # Compute QFI matrix
        qfi = np.zeros((n_params, n_params))

        for i in range(n_params):
            for j in range(n_params):
                qfi[i, j] = np.real(np.trace(rho @ slds[i] @ slds[j]))

        # Symmetrize
        qfi = (qfi + qfi.T) / 2

        return qfi


# Example: Depolarizing channel on pure state
def depolarized_state(params):
    """
    Create depolarized state: ρ = (1-p)|ψ⟩⟨ψ| + p*I/2

    params: [theta, p] where theta parameterizes |ψ⟩ and p is depolarizing prob
    """
    theta, p = params

    # Pure state |ψ⟩ = Ry(θ)|0⟩
    psi = np.array([np.cos(theta/2), np.sin(theta/2)], dtype=complex)
    rho_pure = np.outer(psi, psi.conj())

    # Depolarizing channel
    rho = (1 - p) * rho_pure + p * np.eye(2) / 2

    return rho

# Compute QFI
mixed_qfi = MixedStateQFI(depolarized_state)

print("QFI for depolarized state ρ = (1-p)|ψ⟩⟨ψ| + p*I/2")
print("="*60)

for p in [0.0, 0.1, 0.3, 0.5]:
    params = np.array([np.pi/4, p])
    qfi = mixed_qfi.qfi_single_param(params, 0)  # QFI for θ
    print(f"p = {p:.1f}: QFI(θ) = {qfi:.4f}")

print("\nNote: QFI decreases with noise (higher p)")
```

<Warning>
Computing QFI for mixed states requires handling the SLD carefully when the density matrix has small eigenvalues. Add regularization to avoid numerical instability.
</Warning>

## 3. Quantum Cramér-Rao Bound

The QFI sets the ultimate precision limit for quantum parameter estimation.

<Definition term="Quantum Cramér-Rao Bound">
For any unbiased estimator of parameter θ from quantum measurements:
$$\text{Var}(\hat{\theta}) \geq \frac{1}{n \cdot F_Q(\theta)}$$

where n is the number of copies of the state. This bound is tighter than the classical CR bound and can sometimes be saturated.
</Definition>

### Implementation

```python
class QuantumCRB:
    """
    Explore the Quantum Cramér-Rao Bound through simulation.
    """

    def __init__(self, state_fn, qfi_fn):
        self.state_fn = state_fn
        self.qfi_fn = qfi_fn

    def theoretical_bound(self, params, n_copies):
        """Compute the quantum CR bound."""
        qfi = self.qfi_fn(params)
        return 1 / (n_copies * qfi)

    def estimate_from_measurements(self, params, n_copies, measurement_basis='optimal'):
        """
        Estimate parameter from measurement outcomes.

        Uses projective measurement and MLE.
        """
        # Get true state
        psi = self.state_fn(params)

        # Measurement outcomes
        probs = np.abs(psi) ** 2

        # Sample outcomes
        outcomes = np.random.choice(len(psi), size=n_copies, p=probs)

        # MLE estimation (simplified for single qubit)
        # For Ry(θ)|0⟩, measure in Z basis
        # P(|1⟩) = sin²(θ/2)
        p1_estimate = np.mean(outcomes == 1)

        # Invert to get θ estimate
        p1_estimate = np.clip(p1_estimate, 0.01, 0.99)
        theta_estimate = 2 * np.arcsin(np.sqrt(p1_estimate))

        return theta_estimate

    def run_estimation_experiment(self, true_params, n_copies, n_trials=1000):
        """
        Run multiple estimation trials to verify CR bound.
        """
        estimates = []

        for _ in range(n_trials):
            estimate = self.estimate_from_measurements(true_params, n_copies)
            estimates.append(estimate)

        estimates = np.array(estimates)

        empirical_var = np.var(estimates)
        qcrb = self.theoretical_bound(true_params, n_copies)

        return {
            'empirical_variance': empirical_var,
            'quantum_crb': qcrb,
            'efficiency': qcrb / empirical_var if empirical_var > 0 else 0,
            'mean_estimate': np.mean(estimates),
            'true_value': true_params[0]
        }


# Example: Rotation angle estimation
def ry_state(params):
    """Ry(θ)|0⟩ state."""
    theta = params[0]
    return np.array([np.cos(theta/2), np.sin(theta/2)], dtype=complex)

def ry_qfi(params):
    """QFI for Ry rotation is 1."""
    return 1.0

print("Quantum Cramér-Rao Bound Verification")
print("="*60)
print("Task: Estimate θ from Ry(θ)|0⟩ measured in Z basis")

true_params = np.array([np.pi/4])

for n_copies in [10, 50, 100, 500, 1000]:
    results = crb_exp.run_estimation_experiment(true_params, n_copies, n_trials=2000)

    print(f"\nn = {n_copies}:")
    print(f"  Quantum CRB: {results['quantum_crb']:.6f}")
    print(f"  Empirical Variance: {results['empirical_variance']:.6f}")
    print(f"  Efficiency: {results['efficiency']:.4f}")
```

## 4. QFI for Parameterized Quantum Circuits

For variational quantum circuits, the QFI matrix is essential for natural gradient methods.

<Formula>
QFI for VQC:
$$[F_Q]_{ij} = 4\text{Re}\left[\langle\partial_i\psi|\partial_j\psi\rangle - \langle\partial_i\psi|\psi\rangle\langle\psi|\partial_j\psi\rangle\right]$$

For parameter-shift compatible gates: $[F_Q]_{ij}$ can be computed via circuit evaluations.
</Formula>

### Implementation

```python
class VQC_QFI:
    """
    Compute QFI for variational quantum circuits.
    """

    def __init__(self, n_qubits, n_layers):
        self.n_qubits = n_qubits
        self.n_layers = n_layers
        self.dim = 2 ** n_qubits

    def circuit(self, params):
        """
        Hardware-efficient ansatz with RY and CNOT gates.
        """
        # Reshape params: (n_layers, n_qubits)
        params = params.reshape(self.n_layers, self.n_qubits)

        # Initialize |0...0⟩
        psi = np.zeros(self.dim, dtype=complex)
        psi[0] = 1.0

        for layer in range(self.n_layers):
            # RY rotations
            for qubit in range(self.n_qubits):
                psi = self._apply_ry(psi, params[layer, qubit], qubit)

            # Entangling layer
            for qubit in range(self.n_qubits - 1):
                psi = self._apply_cnot(psi, qubit, qubit + 1)

        return psi

    def _apply_ry(self, psi, theta, qubit):
        """Apply RY gate."""
        new_psi = np.zeros(self.dim, dtype=complex)
        c, s = np.cos(theta/2), np.sin(theta/2)

        for i in range(self.dim):
            bit = (i >> qubit) & 1
            i_flip = i ^ (1 << qubit)

            if bit == 0:
                new_psi[i] += c * psi[i]
                new_psi[i_flip] += s * psi[i]
            else:
                new_psi[i] += c * psi[i]
                new_psi[i_flip] -= s * psi[i]

        return new_psi

    def _apply_cnot(self, psi, control, target):
        """Apply CNOT gate."""
        new_psi = np.zeros(self.dim, dtype=complex)

        for i in range(self.dim):
            if (i >> control) & 1:
                new_psi[i ^ (1 << target)] = psi[i]
            else:
                new_psi[i] = psi[i]

        return new_psi

    def compute_qfi_matrix(self, params, eps=1e-5):
        """
        Compute QFI matrix using finite differences.
        """
        n_params = len(params)
        psi = self.circuit(params)

        # Compute state derivatives
        dpsi = []
        for i in range(n_params):
            params_plus = params.copy()
            params_plus[i] += eps

            params_minus = params.copy()
            params_minus[i] -= eps

            psi_plus = self.circuit(params_plus)
            psi_minus = self.circuit(params_minus)

            dpsi.append((psi_plus - psi_minus) / (2 * eps))

        # Compute QFI matrix
        qfi = np.zeros((n_params, n_params))

        for i in range(n_params):
            for j in range(n_params):
                term1 = np.vdot(dpsi[i], dpsi[j])
                term2 = np.vdot(dpsi[i], psi) * np.vdot(psi, dpsi[j])
                qfi[i, j] = 4 * np.real(term1 - term2)

        return qfi

    def compute_qfi_parameter_shift(self, params):
        """
        Compute QFI using parameter shift rule (more efficient).

        For gates with generator G where e^{-iθG}:
        F_Q = 4 Var(G) = 4(⟨G²⟩ - ⟨G⟩²)
        """
        n_params = len(params)
        qfi = np.zeros((n_params, n_params))
        shift = np.pi / 2

        psi = self.circuit(params)

        for i in range(n_params):
            for j in range(i, n_params):
                # Parameter shift for QFI
                # This is an approximation using fidelity differences

                # For diagonal: F_ii = 1 - |⟨ψ(θ)|ψ(θ+π)⟩|² at θ_i
                if i == j:
                    params_shift = params.copy()
                    params_shift[i] += np.pi
                    psi_shift = self.circuit(params_shift)
                    fidelity = np.abs(np.vdot(psi, psi_shift))**2
                    qfi[i, i] = 4 * (1 - fidelity)
                else:
                    # Off-diagonal elements require more complex estimation
                    # Using finite difference approximation here
                    eps = 1e-4

                    params_pp = params.copy()
                    params_pp[i] += eps
                    params_pp[j] += eps

                    params_pm = params.copy()
                    params_pm[i] += eps
                    params_pm[j] -= eps

                    params_mp = params.copy()
                    params_mp[i] -= eps
                    params_mp[j] += eps

                    params_mm = params.copy()
                    params_mm[i] -= eps
                    params_mm[j] -= eps

                    psi_pp = self.circuit(params_pp)
                    psi_pm = self.circuit(params_pm)
                    psi_mp = self.circuit(params_mp)
                    psi_mm = self.circuit(params_mm)

                    # Second derivative of fidelity
                    f_pp = np.abs(np.vdot(psi, psi_pp))**2
                    f_pm = np.abs(np.vdot(psi, psi_pm))**2
                    f_mp = np.abs(np.vdot(psi, psi_mp))**2
                    f_mm = np.abs(np.vdot(psi, psi_mm))**2

                    qfi[i, j] = -(f_pp - f_pm - f_mp + f_mm) / (4 * eps**2)
                    qfi[j, i] = qfi[i, j]

        return qfi


# Example
vqc = VQC_QFI(n_qubits=2, n_layers=2)
n_params = 2 * 2  # n_layers * n_qubits

# Random parameters
params = np.random.randn(n_params) * 0.1

print("VQC QFI Computation")
print("="*60)
print(f"Circuit: {vqc.n_qubits} qubits, {vqc.n_layers} layers")
print(f"Number of parameters: {n_params}")

# Compute QFI
qfi_matrix = vqc.compute_qfi_matrix(params)
print(f"\nQFI Matrix:")
print(np.round(qfi_matrix, 4))

# Check properties
eigenvalues = np.linalg.eigvalsh(qfi_matrix)
print(f"\nQFI eigenvalues: {np.round(eigenvalues, 4)}")
print(f"Positive semi-definite: {np.all(eigenvalues >= -1e-10)}")
```

<Tip>
The QFI matrix for VQCs often has a block structure reflecting the circuit architecture. Understanding this structure can lead to more efficient QFI approximations.
</Tip>

## 5. QFI and Quantum Metrology

QFI determines the ultimate precision limits in quantum sensing and metrology.

<Definition term="Heisenberg Limit">
For n quantum resources (qubits, photons), the best possible precision scales as:
$$\Delta\theta \geq \frac{1}{\sqrt{n \cdot F_Q}} \sim \frac{1}{n}$$

This is the Heisenberg limit, achieved when F_Q scales as n². Classical strategies achieve at best the standard quantum limit: $\Delta\theta \sim 1/\sqrt{n}$.
</Definition>

### Implementation

```python
class QuantumMetrology:
    """
    Explore quantum advantage in parameter estimation.
    """

    def __init__(self, n_qubits):
        self.n_qubits = n_qubits
        self.dim = 2 ** n_qubits

    def product_state(self, theta):
        """
        Product state: |ψ⟩ = Ry(θ)|0⟩^⊗n

        Achieves standard quantum limit.
        """
        single_qubit = np.array([np.cos(theta/2), np.sin(theta/2)])

        psi = single_qubit.copy()
        for _ in range(self.n_qubits - 1):
            psi = np.kron(psi, single_qubit)

        return psi.astype(complex)

    def ghz_phase_state(self, theta):
        """
        GHZ state with collective phase: (|0...0⟩ + e^{inθ}|1...1⟩)/√2

        Achieves Heisenberg limit for phase estimation.
        """
        psi = np.zeros(self.dim, dtype=complex)
        psi[0] = 1 / np.sqrt(2)
        psi[-1] = np.exp(1j * self.n_qubits * theta) / np.sqrt(2)
        return psi

    def compute_qfi(self, state_fn, theta, eps=1e-6):
        """Compute QFI numerically."""
        psi = state_fn(theta)

        psi_plus = state_fn(theta + eps)
        psi_minus = state_fn(theta - eps)
        dpsi = (psi_plus - psi_minus) / (2 * eps)

        term1 = np.vdot(dpsi, dpsi)
        term2 = np.abs(np.vdot(psi, dpsi))**2

        return 4 * np.real(term1 - term2)

    def compare_precision_limits(self, theta=0.1):
        """
        Compare precision limits for different states.
        """
        # Product state QFI
        qfi_product = self.compute_qfi(self.product_state, theta)

        # GHZ state QFI
        qfi_ghz = self.compute_qfi(self.ghz_phase_state, theta)

        return {
            'n_qubits': self.n_qubits,
            'product_qfi': qfi_product,
            'ghz_qfi': qfi_ghz,
            'product_precision': 1 / np.sqrt(qfi_product),
            'ghz_precision': 1 / np.sqrt(qfi_ghz),
            'quantum_advantage': qfi_ghz / qfi_product
        }


# Compare for different system sizes
print("Quantum Metrology: Product vs GHZ States")
print("="*60)
print(f"{'n':>3} {'QFI_prod':>12} {'QFI_GHZ':>12} {'Advantage':>12}")
print("-"*45)

for n in range(1, 6):
    metrology = QuantumMetrology(n)
    results = metrology.compare_precision_limits(theta=0.1)

    print(f"{n:>3} {results['product_qfi']:>12.4f} {results['ghz_qfi']:>12.4f} "
          f"{results['quantum_advantage']:>12.4f}")

print("\nNote: Product state QFI scales as n (SQL)")
print("      GHZ state QFI scales as n² (Heisenberg limit)")
```

<Warning>
While GHZ states achieve the Heisenberg limit, they are extremely fragile to noise. For realistic noisy systems, the advantage is often reduced or lost entirely.
</Warning>

## 6. Applications to Variational Algorithms

QFI guides optimization in variational quantum algorithms.

<Formula>
Natural Gradient for VQA:
$$\theta_{t+1} = \theta_t - \eta \cdot F_Q^{-1}(\theta_t) \nabla_\theta \langle H \rangle$$

This respects the geometry of quantum state space.
</Formula>

### Implementation

```python
class QFI_VQE:
    """
    Use QFI for natural gradient optimization in VQE.
    """

    def __init__(self, n_qubits, hamiltonian):
        self.n_qubits = n_qubits
        self.dim = 2 ** n_qubits
        self.hamiltonian = hamiltonian
        self.n_layers = 2
        self.n_params = self.n_layers * n_qubits

    def circuit(self, params):
        """Hardware-efficient ansatz."""
        params = params.reshape(self.n_layers, self.n_qubits)
        psi = np.zeros(self.dim, dtype=complex)
        psi[0] = 1.0

        for layer in range(self.n_layers):
            for qubit in range(self.n_qubits):
                psi = self._ry(psi, params[layer, qubit], qubit)
            for qubit in range(self.n_qubits - 1):
                psi = self._cnot(psi, qubit, qubit + 1)

        return psi

    def _ry(self, psi, theta, qubit):
        """Apply RY gate."""
        new_psi = np.zeros(self.dim, dtype=complex)
        c, s = np.cos(theta/2), np.sin(theta/2)

        for i in range(self.dim):
            bit = (i >> qubit) & 1
            i_flip = i ^ (1 << qubit)
            if bit == 0:
                new_psi[i] += c * psi[i]
                new_psi[i_flip] += s * psi[i]
            else:
                new_psi[i] += c * psi[i]
                new_psi[i_flip] -= s * psi[i]

        return new_psi

    def _cnot(self, psi, control, target):
        """Apply CNOT gate."""
        new_psi = np.zeros(self.dim, dtype=complex)
        for i in range(self.dim):
            if (i >> control) & 1:
                new_psi[i ^ (1 << target)] = psi[i]
            else:
                new_psi[i] = psi[i]
        return new_psi

    def energy(self, params):
        """Compute ⟨H⟩."""
        psi = self.circuit(params)
        return np.real(np.vdot(psi, self.hamiltonian @ psi))

    def gradient(self, params, eps=1e-5):
        """Compute energy gradient."""
        grad = np.zeros(self.n_params)
        for i in range(self.n_params):
            params_p = params.copy()
            params_p[i] += eps
            params_m = params.copy()
            params_m[i] -= eps
            grad[i] = (self.energy(params_p) - self.energy(params_m)) / (2 * eps)
        return grad

    def qfi_matrix(self, params, eps=1e-5):
        """Compute QFI matrix."""
        psi = self.circuit(params)
        dpsi = []

        for i in range(self.n_params):
            params_p = params.copy()
            params_p[i] += eps
            params_m = params.copy()
            params_m[i] -= eps
            dpsi.append((self.circuit(params_p) - self.circuit(params_m)) / (2 * eps))

        qfi = np.zeros((self.n_params, self.n_params))
        for i in range(self.n_params):
            for j in range(self.n_params):
                term1 = np.vdot(dpsi[i], dpsi[j])
                term2 = np.vdot(dpsi[i], psi) * np.vdot(psi, dpsi[j])
                qfi[i, j] = 4 * np.real(term1 - term2)

        return qfi

    def natural_gradient_step(self, params, lr=0.1, damping=1e-3):
        """Perform natural gradient update."""
        grad = self.gradient(params)
        qfi = self.qfi_matrix(params)

        # Regularize QFI
        qfi_reg = qfi + damping * np.eye(self.n_params)

        # Natural gradient
        nat_grad = np.linalg.solve(qfi_reg, grad)

        return params - lr * nat_grad

    def standard_gradient_step(self, params, lr=0.01):
        """Perform standard gradient update."""
        grad = self.gradient(params)
        return params - lr * grad


# Example: 2-qubit VQE
# Simple Hamiltonian: H = Z0 + Z1 + 0.5 * X0X1
I = np.eye(2)
X = np.array([[0, 1], [1, 0]])
Z = np.array([[1, 0], [0, -1]])

H = np.kron(Z, I) + np.kron(I, Z) + 0.5 * np.kron(X, X)
exact_gs = np.min(np.linalg.eigvalsh(H))

print("VQE with Natural Gradient")
print("="*60)
print(f"Exact ground state energy: {exact_gs:.6f}")

std_energies = []

for step in range(n_steps):
    nat_energies.append(vqe.energy(params_nat))
    std_energies.append(vqe.energy(params_std))

    params_nat = vqe.natural_gradient_step(params_nat, lr=0.5)
```

## Summary

Quantum Fisher Information is fundamental to quantum machine learning:

1. **Pure state QFI**: 4 times the Fubini-Study metric, computed from state derivatives
2. **Mixed state QFI**: Requires symmetric logarithmic derivative computation
3. **Quantum Cramér-Rao bound**: Ultimate precision limit for quantum estimation
4. **VQC applications**: QFI matrix enables natural gradient optimization
5. **Quantum metrology**: QFI determines quantum advantage in sensing
6. **Variational algorithms**: QFI-based natural gradient improves VQE convergence

Understanding QFI is essential for the natural gradient methods we explore next.
