# The Nine-Qubit Shor Code

## Introduction

In 1995, Peter Shor proved that quantum error correction is possible by constructing the first code that can correct arbitrary single-qubit errors. The Shor code [[9,1,3]] combines the bit-flip and phase-flip codes through concatenation, achieving what neither could alone.

## The Concatenation Strategy

```
Concatenation: Layering Codes for Complete Protection
═════════════════════════════════════════════════════

Problem: Bit-flip code corrects X but fails for Z
         Phase-flip code corrects Z but fails for X

Solution: Use BOTH codes in nested layers!

Layer 1 (Outer): Phase-flip code protects against Z errors
    |0⟩ → |+++⟩ = |+⟩|+⟩|+⟩
    |1⟩ → |---⟩ = |-⟩|-⟩|-⟩

Layer 2 (Inner): Bit-flip code on EACH qubit protects against X
    |+⟩ → (|000⟩ + |111⟩)/√2
    |-⟩ → (|000⟩ - |111⟩)/√2

Combined: Each logical qubit uses 3 blocks of 3 qubits = 9 total

┌──────────────────────────────────────────────────────────────┐
│  Outer code (phase-flip):                                    │
│     Protects the RELATIVE SIGN between blocks               │
│                                                              │
│  Inner code (bit-flip):                                     │
│     Protects EACH block individually from bit flips         │
└──────────────────────────────────────────────────────────────┘
```

## Shor Code Logical States

```python
import numpy as np
from typing import Tuple, List, Optional

class ShorCode:
    """
    The 9-qubit Shor code [[9,1,3]].

    Encoding:
        |0⟩_L = (|000⟩+|111⟩)(|000⟩+|111⟩)(|000⟩+|111⟩) / 2√2
        |1⟩_L = (|000⟩-|111⟩)(|000⟩-|111⟩)(|000⟩-|111⟩) / 2√2

    Parameters:
        n = 9 physical qubits
        k = 1 logical qubit
        d = 3 code distance (corrects 1 arbitrary error)
    """

    def __init__(self):
        # Single-qubit basis
        self.ket_0 = np.array([1, 0], dtype=complex)
        self.ket_1 = np.array([0, 1], dtype=complex)

        # Pauli operators
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

        # Build logical basis states
        self.ket_0_L = self._build_logical_0()
        self.ket_1_L = self._build_logical_1()

    def _tensor(self, states: list) -> np.ndarray:
        """Tensor product of states."""
        result = states[0]
        for s in states[1:]:
            result = np.kron(result, s)
        return result

    def _tensor_ops(self, ops: list) -> np.ndarray:
        """Tensor product of operators."""
        result = ops[0]
        for op in ops[1:]:
            result = np.kron(result, op)
        return result

    def _build_logical_0(self) -> np.ndarray:
        """
        Build |0⟩_L = (|000⟩+|111⟩)⊗(|000⟩+|111⟩)⊗(|000⟩+|111⟩) / 2√2
        """
        ket_000 = self._tensor([self.ket_0, self.ket_0, self.ket_0])
        ket_111 = self._tensor([self.ket_1, self.ket_1, self.ket_1])

        # One block (bit-flip encoded |+⟩)
        block_plus = (ket_000 + ket_111) / np.sqrt(2)

        # Three blocks tensored together
        return self._tensor([block_plus, block_plus, block_plus])

    def _build_logical_1(self) -> np.ndarray:
        """
        Build |1⟩_L = (|000⟩-|111⟩)⊗(|000⟩-|111⟩)⊗(|000⟩-|111⟩) / 2√2
        """
        ket_000 = self._tensor([self.ket_0, self.ket_0, self.ket_0])
        ket_111 = self._tensor([self.ket_1, self.ket_1, self.ket_1])

        # One block (bit-flip encoded |-⟩)
        block_minus = (ket_000 - ket_111) / np.sqrt(2)

        # Three blocks tensored together
        return self._tensor([block_minus, block_minus, block_minus])

    def encode(self, alpha: complex, beta: complex) -> np.ndarray:
        """Encode logical qubit |ψ⟩ = α|0⟩ + β|1⟩."""
        norm = np.sqrt(abs(alpha)**2 + abs(beta)**2)
        alpha, beta = alpha/norm, beta/norm
        return alpha * self.ket_0_L + beta * self.ket_1_L

    def show_structure(self):
        """Display the code structure."""
        print("""
Shor Code Structure: 9 Qubits in 3 Blocks
══════════════════════════════════════════

|0⟩_L:                              |1⟩_L:

Block A (qubits 0,1,2):             Block A (qubits 0,1,2):
  (|000⟩ + |111⟩)/√2                  (|000⟩ - |111⟩)/√2
         ⊗                                  ⊗
Block B (qubits 3,4,5):             Block B (qubits 3,4,5):
  (|000⟩ + |111⟩)/√2                  (|000⟩ - |111⟩)/√2
         ⊗                                  ⊗
Block C (qubits 6,7,8):             Block C (qubits 6,7,8):
  (|000⟩ + |111⟩)/√2                  (|000⟩ - |111⟩)/√2

┌───────────────────────────────────────────────────────────┐
│  The + vs - signs within each block encode the logical   │
│  bit value. All three blocks have the SAME sign.         │
│                                                           │
│  |0⟩_L: all blocks have +    |1⟩_L: all blocks have -   │
└───────────────────────────────────────────────────────────┘
        """)


# Initialize and show structure
```

## The Eight Stabilizers

```python
class ShorStabilizers:
    """
    The Shor code has 8 stabilizer generators for 9 qubits.
    (n - k = 9 - 1 = 8 generators)
    """

    def __init__(self):
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

    def show_stabilizers(self):
        """Display all stabilizer generators."""
        print("""
Shor Code Stabilizer Generators
═══════════════════════════════

Six X-type stabilizers (detect Z errors within blocks):
┌──────────────────────────────────────────────────────────────┐
│ Block A: Z₀Z₁ (qubits 0,1)    Z₁Z₂ (qubits 1,2)             │
│ Block B: Z₃Z₄ (qubits 3,4)    Z₄Z₅ (qubits 4,5)             │
│ Block C: Z₆Z₇ (qubits 6,7)    Z₇Z₈ (qubits 7,8)             │
└──────────────────────────────────────────────────────────────┘
These are inherited from the bit-flip code in each block.

Two Z-type stabilizers (detect X errors that flip blocks):
┌──────────────────────────────────────────────────────────────┐
│ X₀X₁X₂X₃X₄X₅ (blocks A and B must have same sign)           │
│ X₃X₄X₅X₆X₇X₈ (blocks B and C must have same sign)           │
└──────────────────────────────────────────────────────────────┘
These are inherited from the phase-flip code across blocks.

Visual representation (qubits 0-8):
         Block A         Block B         Block C
        ┌───────┐       ┌───────┐       ┌───────┐
Qubits: │ 0 1 2 │       │ 3 4 5 │       │ 6 7 8 │
        └───────┘       └───────┘       └───────┘

Z-type:  Z₀Z₁           Z₃Z₄           Z₆Z₇
         Z₁Z₂           Z₄Z₅           Z₇Z₈

X-type:  ←─X₀X₁X₂X₃X₄X₅──→
                ←─X₃X₄X₅X₆X₇X₈──→
        """)

    def build_stabilizers(self):
        """Build all 8 stabilizer operators as matrices."""
        # Z-type stabilizers (for X error detection within blocks)
        # Block A
        Z01 = self._build_operator([(0, 'Z'), (1, 'Z')])
        Z12 = self._build_operator([(1, 'Z'), (2, 'Z')])
        # Block B
        Z34 = self._build_operator([(3, 'Z'), (4, 'Z')])
        Z45 = self._build_operator([(4, 'Z'), (5, 'Z')])
        # Block C
        Z67 = self._build_operator([(6, 'Z'), (7, 'Z')])
        Z78 = self._build_operator([(7, 'Z'), (8, 'Z')])

        # X-type stabilizers (for Z error detection across blocks)
        X_AB = self._build_operator([(i, 'X') for i in range(6)])
        X_BC = self._build_operator([(i, 'X') for i in range(3, 9)])

        return {
            'Z01': Z01, 'Z12': Z12,
            'Z34': Z34, 'Z45': Z45,
            'Z67': Z67, 'Z78': Z78,
            'X_AB': X_AB, 'X_BC': X_BC
        }

    def _build_operator(self, positions):
        """Build 9-qubit operator from position-Pauli pairs."""
        ops = [self.I] * 9
        for pos, pauli in positions:
            if pauli == 'X':
                ops[pos] = self.X
            elif pauli == 'Z':
                ops[pos] = self.Z
        result = ops[0]
        for op in ops[1:]:
            result = np.kron(result, op)
        return result


stabs = ShorStabilizers()
stabs.show_stabilizers()
```

## Error Correction in the Shor Code

```python
class ShorErrorCorrection:
    """
    Complete error correction for the Shor code.
    """

    def __init__(self):
        self.shor = ShorCode()
        self.stabs = ShorStabilizers()
        self.stabilizers = self.stabs.build_stabilizers()

    def apply_error(self, state: np.ndarray, qubit: int,
                    error_type: str) -> np.ndarray:
        """Apply X, Y, or Z error to specified qubit."""
        I = np.eye(2, dtype=complex)
        paulis = {
            'I': I,
            'X': np.array([[0, 1], [1, 0]], dtype=complex),
            'Y': np.array([[0, -1j], [1j, 0]], dtype=complex),
            'Z': np.array([[1, 0], [0, -1]], dtype=complex)
        }

        ops = [I] * 9
        ops[qubit] = paulis[error_type]

        error_op = ops[0]
        for op in ops[1:]:
            error_op = np.kron(error_op, op)

        return error_op @ state

    def measure_syndrome(self, state: np.ndarray) -> dict:
        """Measure all 8 syndrome bits."""
        syndrome = {}
        for name, stab in self.stabilizers.items():
            eigenvalue = np.real(np.vdot(state, stab @ state))
            syndrome[name] = 0 if eigenvalue > 0.5 else 1
        return syndrome

    def decode_syndrome(self, syndrome: dict) -> Tuple[Optional[int], str]:
        """Determine error location and type from syndrome."""
        # Z-type syndromes → X error detection (within blocks)
        z_syndromes = {
            'A': (syndrome['Z01'], syndrome['Z12']),
            'B': (syndrome['Z34'], syndrome['Z45']),
            'C': (syndrome['Z67'], syndrome['Z78'])
        }

        # X-type syndromes → Z error detection (across blocks)
        x_syndromes = (syndrome['X_AB'], syndrome['X_BC'])

        # Detect X errors (from Z-type stabilizers)
        x_error_qubit = None
        for block, (s1, s2) in z_syndromes.items():
            offset = {'A': 0, 'B': 3, 'C': 6}[block]
            if (s1, s2) == (1, 0):
                x_error_qubit = offset + 0
            elif (s1, s2) == (1, 1):
                x_error_qubit = offset + 1
            elif (s1, s2) == (0, 1):
                x_error_qubit = offset + 2

        # Detect Z errors (from X-type stabilizers)
        z_error_block = None
        if x_syndromes == (1, 0):
            z_error_block = 'A'  # Qubits 0,1,2
        elif x_syndromes == (1, 1):
            z_error_block = 'B'  # Qubits 3,4,5
        elif x_syndromes == (0, 1):
            z_error_block = 'C'  # Qubits 6,7,8

        # Determine error type and location
        if x_error_qubit is not None and z_error_block is not None:
            # Both X and Z detected → Y error
            # The qubit must be in the detected block
            block_offset = {'A': 0, 'B': 3, 'C': 6}[z_error_block]
            if block_offset <= x_error_qubit < block_offset + 3:
                return (x_error_qubit, 'Y')
            else:
                # Errors on different qubits - assume dominant
                return (x_error_qubit, 'X')
        elif x_error_qubit is not None:
            return (x_error_qubit, 'X')
        elif z_error_block is not None:
            # Z error - any qubit in block (all equivalent for logical)
            offset = {'A': 0, 'B': 3, 'C': 6}[z_error_block]
            return (offset, 'Z')  # Pick first qubit in block
        else:
            return (None, 'I')

    def correct(self, state: np.ndarray,
                qubit: Optional[int], error_type: str) -> np.ndarray:
        """Apply correction."""
        if qubit is None:
            return state
        # Apply same Pauli again (P² = I)
        return self.apply_error(state, qubit, error_type)

    def demonstrate_full_correction(self):
        """Demonstrate correction of X, Y, Z errors on each qubit."""
        print("Shor Code Error Correction Demonstration")
        print("=" * 60)

        # Encode |+⟩
        alpha, beta = 1/np.sqrt(2), 1/np.sqrt(2)
        encoded = self.shor.encode(alpha, beta)

        print("\nEncoded state: |+⟩_L = (|0⟩_L + |1⟩_L)/√2")

        # Test each error type on qubit 0
        for error_type in ['X', 'Y', 'Z']:
            print(f"\n{'─' * 60}")
            print(f"Testing {error_type} error on qubit 0:")

            # Apply error
            errored = self.apply_error(encoded, 0, error_type)

            # Get syndrome
            syndrome = self.measure_syndrome(errored)
            print(f"Syndrome: {syndrome}")

            # Decode
            qubit, detected_type = self.decode_syndrome(syndrome)
            print(f"Detected: {detected_type} error on qubit {qubit}")

            # Correct
            corrected = self.correct(errored, qubit, detected_type)

            # Check fidelity
            fidelity = abs(np.vdot(encoded, corrected))**2
            print(f"Fidelity after correction: {fidelity:.6f}")
            print(f"Result: {'SUCCESS ✓' if fidelity > 0.999 else 'FAILED ✗'}")


correction = ShorErrorCorrection()
correction.demonstrate_full_correction()
```

## How X, Y, and Z Errors Are Corrected

```
Error Correction in the Shor Code
═════════════════════════════════

X Error (bit flip):
┌─────────────────────────────────────────────────────────────────┐
│  X on qubit i flips |0⟩↔|1⟩ within its block                  │
│  Detected by Z-type stabilizers in that block                  │
│  Syndrome pattern same as 3-qubit bit-flip code                │
│  Example: X₁ gives Z₀Z₁=-1, Z₁Z₂=-1 → syndrome (1,1) in block A │
└─────────────────────────────────────────────────────────────────┘

Z Error (phase flip):
┌─────────────────────────────────────────────────────────────────┐
│  Z on qubit i flips the sign within its block: +→−             │
│  Detected by X-type stabilizers across blocks                   │
│  Syndrome pattern same as 3-qubit phase-flip code              │
│  Example: Z₄ affects block B → X_AB=-1, X_BC=-1 → syndrome (1,1)│
└─────────────────────────────────────────────────────────────────┘

Y Error (combined XZ):
┌─────────────────────────────────────────────────────────────────┐
│  Y = iXZ acts as both bit and phase flip                       │
│  Triggers BOTH X-type and Z-type syndromes                      │
│  Location found from intersection of detections                 │
│  Example: Y₄ triggers (1,1) for X AND block B for Z            │
└─────────────────────────────────────────────────────────────────┘

Syndrome Lookup Table (Block A example):
┌────────────┬────────────┬───────────────────────────────────────┐
│ Z-syndrome │ X-syndrome │ Error                                 │
├────────────┼────────────┼───────────────────────────────────────┤
│ (0,0)      │ (0,0)      │ No error                              │
│ (1,0)      │ (0,0)      │ X on qubit 0                          │
│ (1,1)      │ (0,0)      │ X on qubit 1                          │
│ (0,1)      │ (0,0)      │ X on qubit 2                          │
│ (0,0)      │ (1,0)      │ Z on block A (apply Z₀)               │
│ (1,0)      │ (1,0)      │ Y on qubit 0                          │
│ (1,1)      │ (1,0)      │ Y on qubit 1                          │
│ (0,1)      │ (1,0)      │ Y on qubit 2                          │
└────────────┴────────────┴───────────────────────────────────────┘
```

```python
└────────────────────────────────────────────────────────────────┘

Logical Operators:
┌────────────────────────────────────────────────────────────────┐
│ X_L = X₀X₁X₂X₃X₄X₅X₆X₇X₈  (X on all 9 qubits)                 │
│                                                                │
│ Z_L = Z₀Z₃Z₆  (Z on first qubit of each block)                 │
│     = Z₁Z₄Z₇  (equivalent)                                     │
│     = Z₂Z₅Z₈  (equivalent)                                     │
└────────────────────────────────────────────────────────────────┘

Error Correction Capability:
┌────────────────────────────────────────────────────────────────┐
│ ✓ Corrects any single X error                                  │
│ ✓ Corrects any single Y error                                  │
│ ✓ Corrects any single Z error                                  │
│ ✓ Corrects any superposition of single-qubit errors           │
│                                                                │
│ ✗ Cannot correct 2 or more errors                              │
│   (might misidentify and cause logical error)                  │
└────────────────────────────────────────────────────────────────┘

Overhead:
┌────────────────────────────────────────────────────────────────┐
│ Rate: k/n = 1/9 ≈ 11%                                          │
│ Encoding: 8 CNOTs + 3 Hadamards                                │
│ Syndrome: 8 measurements                                       │
│                                                                │
│ This is not optimal! The Steane [[7,1,3]] code does the same   │
│ with only 7 qubits.                                            │
└────────────────────────────────────────────────────────────────┘
        """)

    def verify_logical_operators(self):
        """Verify logical X and Z operators."""
        print("\nVerifying Logical Operators")
        print("─" * 50)

        shor = ShorCode()

        # Build logical X (X on all qubits)
        I = np.eye(2, dtype=complex)
        X = np.array([[0, 1], [1, 0]], dtype=complex)
        Z = np.array([[1, 0], [0, -1]], dtype=complex)

        X_L = X
        for _ in range(8):
            X_L = np.kron(X_L, X)

        # Build logical Z (Z on qubits 0, 3, 6)
        Z_L = Z
        for i in range(1, 9):
            if i in [3, 6]:
                Z_L = np.kron(Z_L, Z)
            else:
                Z_L = np.kron(Z_L, I)

        # Verify X_L|0⟩_L = |1⟩_L
        result = X_L @ shor.ket_0_L
        fidelity_to_1L = abs(np.vdot(shor.ket_1_L, result))**2
        print(f"X_L|0⟩_L = |1⟩_L? Fidelity: {fidelity_to_1L:.6f}")

        # Verify Z_L|0⟩_L = |0⟩_L
        result = Z_L @ shor.ket_0_L
        fidelity_to_0L = abs(np.vdot(shor.ket_0_L, result))**2
        print(f"Z_L|0⟩_L = |0⟩_L? Fidelity: {fidelity_to_0L:.6f}")

        # Verify Z_L|1⟩_L = -|1⟩_L
        result = Z_L @ shor.ket_1_L
        overlap = np.vdot(shor.ket_1_L, result)
        print(f"Z_L|1⟩_L = -|1⟩_L? Eigenvalue: {overlap:.6f}")
```

## Summary

The Shor code [[9,1,3]] demonstrates several crucial principles:

1. **Concatenation**: Combining codes gives protection against multiple error types
2. **Complete protection**: First code to correct arbitrary single-qubit errors
3. **Stabilizer structure**: 8 generators detect both X and Z errors
4. **Error discretization**: Continuous errors collapse to discrete Paulis upon syndrome measurement
5. **Overhead**: Uses 9 physical qubits per logical qubit (not optimal)

While historically important, the Shor code is superseded by more efficient codes like the Steane code [[7,1,3]] and CSS codes in general.
