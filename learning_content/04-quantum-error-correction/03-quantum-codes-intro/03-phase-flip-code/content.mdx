# The Three-Qubit Phase-Flip Code

## Introduction

The phase-flip code is the "dual" of the bit-flip code—it protects against Z errors instead of X errors. By understanding both codes, we lay the groundwork for the Shor code, which combines them to achieve full quantum error protection.

## Understanding Phase Errors

```
Phase Errors: A Uniquely Quantum Phenomenon
═══════════════════════════════════════════

Classical bits have no phase - they're just 0 or 1.
Quantum superpositions have RELATIVE PHASE between components.

    |ψ⟩ = α|0⟩ + β|1⟩
              ↑
         This can be e^(iφ)·|β|

Z Error Effect:
    Z|0⟩ = |0⟩       (no change)
    Z|1⟩ = -|1⟩      (phase flip)

    Z(α|0⟩ + β|1⟩) = α|0⟩ - β|1⟩

Example on special states:
┌─────────────────────────────────────────────────────────────┐
│  State     │  Z|ψ⟩                   │  Effect             │
├────────────┼─────────────────────────┼─────────────────────┤
│  |0⟩       │  |0⟩                    │  No visible change  │
│  |1⟩       │  -|1⟩ (global phase)    │  No visible change  │
│  |+⟩       │  (|0⟩-|1⟩)/√2 = |-⟩     │  FLIPPED!           │
│  |-⟩       │  (|0⟩+|1⟩)/√2 = |+⟩     │  FLIPPED!           │
│  |i⟩       │  |-i⟩                   │  FLIPPED!           │
└─────────────────────────────────────────────────────────────┘

Key insight: Z is a "bit flip" in the Hadamard basis!
    H|+⟩ = |0⟩,  H|-⟩ = |1⟩
    HZH = X

So: Z in the computational basis = X in the Hadamard basis
```

## The Hadamard Transform Duality

```python
import numpy as np

class HadamardDuality:
    """
    Demonstrates how Hadamard transforms relate X and Z errors.
    """

    def __init__(self):
        self.H = np.array([[1, 1], [1, -1]], dtype=complex) / np.sqrt(2)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)
        self.I = np.eye(2, dtype=complex)

    def show_conjugation(self):
        """Show that HXH = Z and HZH = X."""
        print("Hadamard Conjugation Relations")
        print("=" * 45)

        HXH = self.H @ self.X @ self.H
        HZH = self.H @ self.Z @ self.H

        print("\nH·X·H =")
        print(np.round(np.real(HXH), 4))
        print("This is Z! ✓")

        print("\nH·Z·H =")
        print(np.round(np.real(HZH), 4))
        print("This is X! ✓")

        print("""

        ╔═══════════════════════════════════════════════════╗
        ║  Key Duality:                                     ║
        ║                                                   ║
        ║    H·X·H = Z    (X becomes Z under Hadamard)     ║
        ║    H·Z·H = X    (Z becomes X under Hadamard)     ║
        ║                                                   ║
        ║  This means:                                      ║
        ║    Bit-flip code in H-basis = Phase-flip code!   ║
        ╚═══════════════════════════════════════════════════╝
        """)

    def show_basis_states(self):
        """Show how Z acts like X in the Hadamard basis."""
        print("\nBasis State Transformations")
        print("─" * 45)

        ket_0 = np.array([1, 0], dtype=complex)
        ket_1 = np.array([0, 1], dtype=complex)
        ket_plus = (ket_0 + ket_1) / np.sqrt(2)
        ket_minus = (ket_0 - ket_1) / np.sqrt(2)

        print("In computational basis {|0⟩, |1⟩}:")
        print(f"  X|0⟩ = |1⟩  (bit flip)")
        print(f"  X|1⟩ = |0⟩  (bit flip)")
        print(f"  Z|0⟩ = |0⟩  (no change)")
        print(f"  Z|1⟩ = -|1⟩ (phase)")

        print("\nIn Hadamard basis {|+⟩, |-⟩}:")
        print(f"  Z|+⟩ = |-⟩  (basis flip!)")
        print(f"  Z|-⟩ = |+⟩  (basis flip!)")
        print(f"  X|+⟩ = |+⟩  (no change)")
        print(f"  X|-⟩ = -|-⟩ (phase)")


duality = HadamardDuality()
```

## The Phase-Flip Code

```python
import numpy as np
from typing import Tuple, Optional

class PhaseFlipCode:
    """
    The 3-qubit phase-flip code [[3,1,1]] for Z errors.

    Encoding:
        |0⟩_L = |+++⟩
        |1⟩_L = |---⟩

    General state:
        |ψ⟩_L = α|0⟩_L + β|1⟩_L = α|+++⟩ + β|---⟩

    Correctable errors: Single Z (phase-flip) on any qubit
    """

    def __init__(self):
        # Basis states
        self.ket_0 = np.array([1, 0], dtype=complex)
        self.ket_1 = np.array([0, 1], dtype=complex)

        # Hadamard basis
        self.ket_plus = (self.ket_0 + self.ket_1) / np.sqrt(2)
        self.ket_minus = (self.ket_0 - self.ket_1) / np.sqrt(2)

        # Operators
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)
        self.H = np.array([[1, 1], [1, -1]], dtype=complex) / np.sqrt(2)

        # Logical basis states (3 qubits)
        self.ket_ppp = self._tensor([self.ket_plus, self.ket_plus, self.ket_plus])
        self.ket_mmm = self._tensor([self.ket_minus, self.ket_minus, self.ket_minus])

    def _tensor(self, states: list) -> np.ndarray:
        """Tensor product of states."""
        result = states[0]
        for s in states[1:]:
            result = np.kron(result, s)
        return result

    def _tensor_ops(self, ops: list) -> np.ndarray:
        """Tensor product of operators."""
        result = ops[0]
        for op in ops[1:]:
            result = np.kron(result, op)
        return result

    def encode(self, alpha: complex, beta: complex) -> np.ndarray:
        """
        Encode logical qubit |ψ⟩ = α|0⟩ + β|1⟩ into three physical qubits.

        |ψ⟩_L = α|+++⟩ + β|---⟩
        """
        norm = np.sqrt(abs(alpha)**2 + abs(beta)**2)
        alpha, beta = alpha/norm, beta/norm
        return alpha * self.ket_ppp + beta * self.ket_mmm

    def encode_circuit_description(self):
        """Show the encoding circuit for phase-flip code."""
        print("""
        Phase-Flip Code Encoding Circuit
        ═════════════════════════════════

        Method 1: Start in computational basis, apply Hadamards

        |ψ⟩ ───●───●───H───  →  α|+++⟩ + β|---⟩
               │   │
        |0⟩ ───⊕───│───H───
                   │
        |0⟩ ───────⊕───H───

        Step by step:
        1. Start: (α|0⟩ + β|1⟩) ⊗ |0⟩ ⊗ |0⟩
        2. CNOT₁₂: α|00⟩ + β|11⟩  (first two qubits)
        3. CNOT₁₃: α|000⟩ + β|111⟩
        4. H⊗H⊗H: α|+++⟩ + β|---⟩

        ┌──────────────────────────────────────────────────────┐
        │  Key insight: This is just the bit-flip encoding    │
        │  followed by Hadamard on every qubit!               │
        │                                                      │
        │  Phase-flip code = H⊗³ × (Bit-flip code) × H⊗³     │
        └──────────────────────────────────────────────────────┘
        """)

    def apply_error(self, state: np.ndarray, qubit: int) -> np.ndarray:
        """Apply Z error to specified qubit (0, 1, or 2)."""
        ops = [self.I, self.I, self.I]
        ops[qubit] = self.Z
        error_op = self._tensor_ops(ops)
        return error_op @ state

    def measure_syndrome(self, state: np.ndarray) -> Tuple[int, int]:
        """
        Measure error syndrome using stabilizer operators.

        Stabilizers (compare phases between qubits):
            X₁X₂ = X⊗X⊗I  (compares qubits 1 and 2 in X basis)
            X₂X₃ = I⊗X⊗X  (compares qubits 2 and 3 in X basis)

        Why X instead of Z?
            - For bit-flip code: Z detects X errors
            - For phase-flip code: X detects Z errors

        Syndrome interpretation:
            (0, 0) → No error
            (1, 0) → Error on qubit 1
            (1, 1) → Error on qubit 2
            (0, 1) → Error on qubit 3
        """
        X1X2 = self._tensor_ops([self.X, self.X, self.I])
        X2X3 = self._tensor_ops([self.I, self.X, self.X])

        s1 = int((1 - np.real(np.vdot(state, X1X2 @ state))) / 2)
        s2 = int((1 - np.real(np.vdot(state, X2X3 @ state))) / 2)

        return (s1, s2)

    def syndrome_to_error(self, syndrome: Tuple[int, int]) -> Optional[int]:
        """Map syndrome to error location."""
        syndrome_table = {
            (0, 0): None,
            (1, 0): 0,
            (1, 1): 1,
            (0, 1): 2
        }
        return syndrome_table.get(syndrome)

    def correct(self, state: np.ndarray, syndrome: Tuple[int, int]) -> np.ndarray:
        """Apply Z correction based on syndrome."""
        error_loc = self.syndrome_to_error(syndrome)
        if error_loc is None:
            return state
        return self.apply_error(state, error_loc)  # Z² = I

    def decode(self, state: np.ndarray) -> Tuple[complex, complex]:
        """Extract α and β from encoded state."""
        alpha = np.vdot(self.ket_ppp, state)
        beta = np.vdot(self.ket_mmm, state)
        return (alpha, beta)

    def demonstrate_error_correction(self):
        """Full demonstration of phase-flip error correction."""
        print("Phase-Flip Code Error Correction Demonstration")
        print("=" * 55)

        # Encode |+⟩ = (|0⟩ + |1⟩)/√2
        alpha, beta = 1/np.sqrt(2), 1/np.sqrt(2)
        encoded = self.encode(alpha, beta)

        print(f"\nOriginal state: |+⟩ = (|0⟩ + |1⟩)/√2")
        print(f"Encoded state: (|+++⟩ + |---⟩)/√2")

        for error_qubit in [None, 0, 1, 2]:
            print(f"\n{'─' * 55}")
            if error_qubit is None:
                state = encoded.copy()
                print(f"No error applied")
            else:
                state = self.apply_error(encoded, error_qubit)
                print(f"Z error on qubit {error_qubit}")

            syndrome = self.measure_syndrome(state)
            print(f"Syndrome: {syndrome}")

            error_loc = self.syndrome_to_error(syndrome)
            print(f"Detected error location: {error_loc}")

            corrected = self.correct(state, syndrome)
            alpha_out, beta_out = self.decode(corrected)

            fidelity = abs(alpha * np.conj(alpha_out) +
                          beta * np.conj(beta_out))**2

            print(f"Recovery fidelity: {fidelity:.6f}")
            print(f"Result: {'SUCCESS ✓' if fidelity > 0.999 else 'FAILED ✗'}")


# Run demonstration
code = PhaseFlipCode()
code.encode_circuit_description()
code.demonstrate_error_correction()
```

## Why the Phase-Flip Code Fails for X Errors

```python
class PhaseFlipLimitation:
    """
    Demonstrate that the phase-flip code cannot correct X errors.
    """

    def __init__(self):
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)

    def show_x_error_problem(self):
        """Show why X errors are invisible to phase-flip code."""
        print("The Phase-Flip Code's Weakness: Bit-Flip (X) Errors")
        print("=" * 55)

        print("""
        Consider the encoded state: |ψ⟩_L = (|+++⟩ + |---⟩)/√2

        Apply an X error on qubit 1: X₁|ψ⟩_L

        First, note that:
            X|+⟩ = |+⟩    (eigenstate!)
            X|-⟩ = -|-⟩   (eigenstate with -1 eigenvalue)

        So:
            X₁|+++⟩ = |+++⟩       (unchanged)
            X₁|---⟩ = -|---⟩      (picks up phase)

        Result: X₁(|+++⟩ + |---⟩)/√2 = (|+++⟩ - |---⟩)/√2

        ┌─────────────────────────────────────────────────────────┐
        │  This is a LOGICAL X ERROR!                            │
        │  (|+++⟩ + |---⟩)/√2 → (|+++⟩ - |---⟩)/√2              │
        │                                                        │
        │  The logical state flipped from |+⟩_L to |-⟩_L         │
        └─────────────────────────────────────────────────────────┘

        But what does the syndrome say?

            X₁X₂|++-⟩ eigenvalue = (+1)(+1) = +1
            (Same for all components: |+⟩ is X eigenstate)

        ╔═════════════════════════════════════════════════════════╗
        ║  SYNDROME IS (0,0) - NO ERROR DETECTED!                ║
        ║  The X error is INVISIBLE to the phase-flip code!     ║
        ╚═════════════════════════════════════════════════════════╝

        """)
```

## Summary

The three-qubit phase-flip code demonstrates several key principles:

1. **Duality**: The phase-flip code is the Hadamard-conjugate of the bit-flip code
2. **Complementary protection**: Each code protects against one error type but is vulnerable to the other
3. **Stabilizer structure**: X₁X₂ and X₂X₃ stabilizers detect Z errors
4. **Path forward**: Concatenating both codes yields the Shor code

The next lesson explores the Shor code, which achieves the seemingly impossible task of protecting against arbitrary single-qubit errors.
