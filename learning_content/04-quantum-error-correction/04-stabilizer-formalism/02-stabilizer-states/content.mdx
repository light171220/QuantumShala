# Stabilizer States

## Introduction

Stabilizer states are quantum states that can be uniquely defined by the Pauli operators that leave them unchanged. This seemingly abstract definition leads to remarkably efficient classical representations and simulation algorithms, forming the theoretical foundation of stabilizer codes.

## What is a Stabilizer State?

```python
import numpy as np
from typing import List, Set

class StabilizerStateBasics:
    """
    A stabilizer state |ψ⟩ is defined by a group S of Pauli operators
    such that S|ψ⟩ = |ψ⟩ for all S ∈ S.
    """

    def __init__(self):
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

    def show_concept(self):
        """Explain the stabilizer state concept."""
        print("""
Stabilizer States: States Defined by Their Stabilizers
═══════════════════════════════════════════════════════

Definition:
    |ψ⟩ is a stabilizer state if there exists a group S of Pauli operators
    such that S|ψ⟩ = |ψ⟩ (eigenvalue +1) for all S ∈ S.

The group S is called the STABILIZER GROUP of |ψ⟩.

Example: The state |0⟩
    Z|0⟩ = |0⟩    (Z stabilizes |0⟩)
    Stabilizer: S = ⟨Z⟩ = {I, Z}

Example: The state |+⟩ = (|0⟩ + |1⟩)/√2
    X|+⟩ = |+⟩    (X stabilizes |+⟩)
    Stabilizer: S = ⟨X⟩ = {I, X}

Example: The Bell state |Φ⁺⟩ = (|00⟩ + |11⟩)/√2
    X₁X₂|Φ⁺⟩ = |Φ⁺⟩
    Z₁Z₂|Φ⁺⟩ = |Φ⁺⟩
    Stabilizer: S = ⟨X₁X₂, Z₁Z₂⟩

Key Properties:
┌──────────────────────────────────────────────────────────────┐
│ 1. The stabilizer group is always abelian (all S commute)   │
│ 2. For n qubits, an n-qubit state needs n generators        │
│ 3. Stabilizers never include -I (would give eigenvalue -1)  │
│ 4. The state is the unique +1 eigenspace of all stabilizers │
└──────────────────────────────────────────────────────────────┘
        """)

    def verify_stabilizer(self, state: np.ndarray,
                         stabilizers: List[np.ndarray]) -> bool:
        """Verify that operators stabilize a state."""
        for S in stabilizers:
            result = S @ state
            if not np.allclose(result, state):
                return False
        return True


basics = StabilizerStateBasics()
basics.show_concept()
```

class CommonStabilizerStates:

        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

    def show_examples(self):
        """Display common stabilizer states."""
        print("""
Common Stabilizer States
════════════════════════

SINGLE-QUBIT STABILIZER STATES (6 total):

┌─────────────────────────────────────────────────────────────────┐
│ State      │ Stabilizer  │ Description                         │
├────────────┼─────────────┼─────────────────────────────────────┤
│ |0⟩        │ ⟨Z⟩         │ Z-basis eigenstate                  │
│ |1⟩        │ ⟨-Z⟩        │ Z-basis eigenstate (eigenvalue -1)  │
│ |+⟩        │ ⟨X⟩         │ X-basis eigenstate                  │
│ |-⟩        │ ⟨-X⟩        │ X-basis eigenstate                  │
│ |+i⟩       │ ⟨Y⟩         │ Y-basis eigenstate                  │
│ |-i⟩       │ ⟨-Y⟩        │ Y-basis eigenstate                  │
└─────────────────────────────────────────────────────────────────┘

Note: |1⟩ has stabilizer -Z, since Z|1⟩ = -|1⟩, so (-Z)|1⟩ = +|1⟩

TWO-QUBIT STABILIZER STATES:

Bell states:
    |Φ⁺⟩ = (|00⟩+|11⟩)/√2    Stabilizers: ⟨XX, ZZ⟩
    |Φ⁻⟩ = (|00⟩-|11⟩)/√2    Stabilizers: ⟨-XX, ZZ⟩
    |Ψ⁺⟩ = (|01⟩+|10⟩)/√2    Stabilizers: ⟨XX, -ZZ⟩
    |Ψ⁻⟩ = (|01⟩-|10⟩)/√2    Stabilizers: ⟨-XX, -ZZ⟩

Product states:
    |00⟩        Stabilizers: ⟨Z₁, Z₂⟩
    |++⟩        Stabilizers: ⟨X₁, X₂⟩

THREE-QUBIT STABILIZER STATES:

GHZ state: |GHZ⟩ = (|000⟩+|111⟩)/√2
    Stabilizers: ⟨X₁X₂X₃, Z₁Z₂, Z₂Z₃⟩
    (Note: Z₁Z₃ is also a stabilizer, but it's the product Z₁Z₂·Z₂Z₃)
        """)

    def verify_bell_stabilizers(self):
        """Verify Bell state stabilizers."""
        print("\nVerifying Bell State Stabilizers")
        print("─" * 45)

        # Build |Φ⁺⟩
        ket_00 = np.array([1, 0, 0, 0], dtype=complex)
        ket_11 = np.array([0, 0, 0, 1], dtype=complex)
        phi_plus = (ket_00 + ket_11) / np.sqrt(2)

        # Build stabilizers
        XX = np.kron(self.X, self.X)
        ZZ = np.kron(self.Z, self.Z)

        # Verify
        print(f"XX|Φ⁺⟩ = |Φ⁺⟩? {np.allclose(XX @ phi_plus, phi_plus)}")
        print(f"ZZ|Φ⁺⟩ = |Φ⁺⟩? {np.allclose(ZZ @ phi_plus, phi_plus)}")

        # Check eigenvalues
        xx_ev = np.real(np.vdot(phi_plus, XX @ phi_plus))
        zz_ev = np.real(np.vdot(phi_plus, ZZ @ phi_plus))
        print(f"Eigenvalues: ⟨XX⟩ = {xx_ev:.1f}, ⟨ZZ⟩ = {zz_ev:.1f}")


states = CommonStabilizerStates()
states.show_examples()
states.verify_bell_stabilizers()

## Stabilizer Group Properties

```
Properties of Stabilizer Groups
═══════════════════════════════

For an n-qubit stabilizer state:

1. GROUP STRUCTURE
   The stabilizer S is a subgroup of the n-qubit Pauli group Gₙ.
   - Contains identity I⊗n
   - Closed under multiplication
   - Contains inverse of each element (S² = I for Paulis)

2. ABELIAN
   All elements of S commute: [S₁, S₂] = 0 for all S₁, S₂ ∈ S.
   (Non-commuting stabilizers would give contradictory constraints)

3. SIZE AND GENERATORS
   |S| = 2ⁿ for an n-qubit state
   Need exactly n independent generators: S = ⟨g₁, g₂, ..., gₙ⟩

4. NO NEGATIVE IDENTITY
   -I ∉ S  (otherwise (-I)|ψ⟩ = -|ψ⟩ contradicts S|ψ⟩ = |ψ⟩)

5. UNIQUENESS
   Given n independent commuting Paulis (not containing -I),
   there is a UNIQUE state stabilized by all of them.

Example: 3-qubit GHZ state
    Generators: g₁ = X₁X₂X₃, g₂ = Z₁Z₂, g₃ = Z₂Z₃
    Full group: S = {I, g₁, g₂, g₃, g₁g₂, g₁g₃, g₂g₃, g₁g₂g₃}
              = {III, XXX, ZZI, IZZ, YYI, XIZ, IYY, XZZ, ...}
    Size: 2³ = 8 elements ✓
```

## The Tableau Representation

```python
class TableauRepresentation:
    """
    Efficient classical representation of stabilizer states.

    For n qubits, represent n generators using:
        - n × n binary matrix for X components
        - n × n binary matrix for Z components
        - n-bit phase vector
    """

    def __init__(self, n: int):
        self.n = n
        # Tableau: rows are generators, columns are qubits
        self.x_matrix = np.zeros((n, n), dtype=int)
        self.z_matrix = np.zeros((n, n), dtype=int)
        self.phases = np.zeros(n, dtype=int)  # 0 for +, 1 for -

    def set_generator(self, row: int, pauli_string: str, phase: int = 0):
        """Set a generator from a Pauli string."""
        for col, p in enumerate(pauli_string):
            if p == 'X':
                self.x_matrix[row, col] = 1
            elif p == 'Y':
                self.x_matrix[row, col] = 1
                self.z_matrix[row, col] = 1
            elif p == 'Z':
                self.z_matrix[row, col] = 1
        self.phases[row] = phase

    def get_generator(self, row: int) -> str:
        """Get generator as a Pauli string."""
        result = "-" if self.phases[row] else "+"
        for col in range(self.n):
            x, z = self.x_matrix[row, col], self.z_matrix[row, col]
            if x == 0 and z == 0:
                result += 'I'
            elif x == 1 and z == 0:
                result += 'X'
            elif x == 1 and z == 1:
                result += 'Y'
            else:
                result += 'Z'
        return result

    def display(self):
        """Display the tableau."""
        print(f"\nTableau for {self.n}-qubit stabilizer state:")
        print("─" * (self.n * 4 + 10))
        print(f"{'Gen':>4} │ Operator")
        print("─" * (self.n * 4 + 10))
        for i in range(self.n):
            print(f"g{i+1:>2}  │ {self.get_generator(i)}")

    @classmethod
    def from_state(cls, state_name: str):
        """Create tableau for known states."""
        if state_name == "GHZ":
            t = cls(3)
            t.set_generator(0, "XXX")
            t.set_generator(1, "ZZI")
            t.set_generator(2, "IZZ")
            return t
        elif state_name == "Bell":
            t = cls(2)
            t.set_generator(0, "XX")
            t.set_generator(1, "ZZ")
            return t
        elif state_name == "|000>":
            t = cls(3)
            t.set_generator(0, "ZII")
            t.set_generator(1, "IZI")
            t.set_generator(2, "IIZ")
            return t
        else:
            raise ValueError(f"Unknown state: {state_name}")


# Demonstrate tableau representation
print("Tableau Representation of Stabilizer States")
print("=" * 50)

for state in ["GHZ", "Bell", "|000>"]:
    t = TableauRepresentation.from_state(state)
    print(f"\n{state} state:")
    t.display()
```

## Generating States from Stabilizers

```python
class StabilizerStateGeneration:
    """
    Generate the actual quantum state from its stabilizers.
    """

    def __init__(self, n: int):
        self.n = n
        self.dim = 2**n
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

    def pauli_string_to_matrix(self, pauli_string: str) -> np.ndarray:
        """Convert Pauli string to matrix."""
        paulis = {'I': self.I, 'X': self.X,
                  'Y': 1j * self.X @ self.Z, 'Z': self.Z}

        result = paulis[pauli_string[0]]
        for p in pauli_string[1:]:
            result = np.kron(result, paulis[p])
        return result

    def projector_from_stabilizers(self,
                                   generators: List[str]) -> np.ndarray:
        """
        Build projector onto stabilizer state.

        P = (1/2ⁿ) ∏ᵢ (I + gᵢ)

        This projects onto the +1 eigenspace of all generators.
        """
        projector = np.eye(self.dim, dtype=complex)

        for gen in generators:
            # Handle phase
            if gen[0] == '-':
                phase = -1
                gen = gen[1:]
            elif gen[0] == '+':
                phase = 1
                gen = gen[1:]
            else:
                phase = 1

            g_matrix = phase * self.pauli_string_to_matrix(gen)
            # (I + g)/2 projects onto +1 eigenspace of g
            projector = projector @ (np.eye(self.dim) + g_matrix) / 2

        return projector

    def generate_state(self, generators: List[str]) -> np.ndarray:
        """Generate the stabilizer state from generators."""
        projector = self.projector_from_stabilizers(generators)

        # The projector projects onto a 1-dimensional space
        # Any column of the projector (normalized) is the state
        # Find a non-zero column
        for col in range(self.dim):
            state = projector[:, col]
            if np.linalg.norm(state) > 1e-10:
                return state / np.linalg.norm(state)

        return None

    def demonstrate(self):
        """Show state generation for various examples."""
        print("\nGenerating States from Stabilizers")
        print("=" * 50)

        examples = [
            ("GHZ", ["XXX", "ZZI", "IZZ"]),
            ("Bell", ["XX", "ZZ"]),
            ("|000⟩", ["ZII", "IZI", "IIZ"]),
            ("|+++⟩", ["XII", "IXI", "IIX"]),
        ]

        for name, gens in examples:
            n = len(gens[0])
            generator = StabilizerStateGeneration(n)
            state = generator.generate_state(gens)

            print(f"\n{name} state from stabilizers {gens}:")
            # Show non-zero amplitudes
            for i, amp in enumerate(state):
                if abs(amp) > 1e-10:
                    binary = format(i, f'0{n}b')
                    print(f"  |{binary}⟩: {amp:.4f}")


gen = StabilizerStateGeneration(3)
gen.demonstrate()
```

## Graph States

```
Graph States: A Special Class of Stabilizer States
═══════════════════════════════════════════════════

Every graph G = (V, E) defines a stabilizer state:

1. Start with all qubits in |+⟩
2. Apply CZ between connected vertices

Stabilizers of graph state on graph G:
    For each vertex v: Kᵥ = Xᵥ ∏_{u∈N(v)} Zᵤ

    where N(v) = neighbors of v in G

Example: Linear graph (path graph P₃)

    1 ─── 2 ─── 3

    K₁ = X₁Z₂        (vertex 1, neighbor is 2)
    K₂ = Z₁X₂Z₃      (vertex 2, neighbors are 1,3)
    K₃ = Z₂X₃        (vertex 3, neighbor is 2)

Example: Star graph S₃

       2
       │
    1 ─ 3 ─ 4

    K₁ = X₁Z₃
    K₂ = X₂Z₃
    K₃ = Z₁Z₂X₃Z₄
    K₄ = X₄Z₃

Graph states are important because:
┌──────────────────────────────────────────────────────────────────┐
│ • Every stabilizer state is locally equivalent to a graph state  │
│ • Efficient for measurement-based quantum computation (MBQC)     │
│ • Natural structure for many quantum codes                       │
│ • Generated by simple circuit: H^⊗n followed by CZ gates        │
└──────────────────────────────────────────────────────────────────┘
```

## Counting Stabilizer States

```python
class StabilizerStateCounting:
    """
    How many stabilizer states exist?
    """

    def show_counting(self):
        """Display stabilizer state counting."""
        print("""
Counting Stabilizer States
══════════════════════════

For n qubits:

Number of n-qubit stabilizer states:
    N_stab(n) = 2^n × ∏_{k=1}^{n} (2^k + 1)

First few values:
    n = 1:  N = 2 × 3 = 6
    n = 2:  N = 4 × 3 × 5 = 60
    n = 3:  N = 8 × 3 × 5 × 9 = 1,080
    n = 4:  N = 16 × 3 × 5 × 9 × 17 = 36,720

Compare to total number of pure states:
    Pure states form continuous manifold (infinite)
    But with polynomial precision, roughly 2^(2n) distinct states

Stabilizer states are a TINY fraction of all states!

Yet they are important because:
┌───────────────────────────────────────────────────────────────┐
│ 1. Include all Pauli eigenstates                             │
│ 2. Include all Clifford circuit outputs from |0⟩^n           │
│ 3. Efficiently classically simulable (Gottesman-Knill)       │
│ 4. Form the basis for all stabilizer error-correcting codes  │
│ 5. Approximate other states well for many purposes           │
└───────────────────────────────────────────────────────────────┘
        """)

        # Compute exact counts
        print("\nExact stabilizer state counts:")
        for n in range(1, 6):
            count = 2**n
            for k in range(1, n+1):
                count *= (2**k + 1)
            print(f"  n = {n}: {count:,} stabilizer states")


counting = StabilizerStateCounting()
counting.show_counting()
```

## Summary

Stabilizer states bridge quantum mechanics and classical computation:

1. **Definition**: States |ψ⟩ where S|ψ⟩ = |ψ⟩ for stabilizer group S
2. **Structure**: Abelian group with n generators for n qubits
3. **Tableau**: Efficient O(n²) classical representation
4. **Graph states**: Important subclass with simple structure
5. **Efficiency**: Can be simulated classically (Gottesman-Knill theorem)

Understanding stabilizer states is essential for both quantum error correction and quantum computation theory.
