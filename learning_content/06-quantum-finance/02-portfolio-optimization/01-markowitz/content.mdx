## 1. Introduction to Modern Portfolio Theory

In 1952, Harry Markowitz published "Portfolio Selection," which revolutionized investment theory by formalizing how investors should construct portfolios.

<Definition term="Modern Portfolio Theory">
Modern Portfolio Theory (MPT) is a mathematical framework for assembling a portfolio of assets such that the expected return is maximized for a given level of risk.
</Definition>

### Key Insight

The fundamental insight of MPT is that the risk of a portfolio depends not just on individual asset risks, but on how assets move together (correlations). By combining assets that don't move in lockstep, investors can reduce portfolio risk without sacrificing expected return.

<Tip>
Markowitz received the Nobel Prize in Economics in 1990 for this work. MPT remains the foundation of modern investment management.
</Tip>

## 2. Assumptions of MPT

MPT relies on several simplifying assumptions:

1. **Investors are rational and risk-averse**: They prefer higher returns and lower risk
2. **Markets are efficient**: All relevant information is reflected in prices
3. **Returns are normally distributed**: Can be described by mean and variance
4. **Investors have the same time horizon**: Single-period optimization
5. **No transaction costs or taxes**: Frictionless trading
6. **Investors can borrow and lend at the risk-free rate**

<Warning>
These assumptions are idealizations. Real markets have frictions, non-normal returns, and heterogeneous investors. Despite this, MPT provides valuable insights and a starting point for practical portfolio construction.
</Warning>

## 3. The Mathematical Framework

### Portfolio Return

For a portfolio with $n$ assets:

<Formula>
$$R_p = \sum_{i=1}^n w_i R_i = \mathbf{w}^T \mathbf{R}$$
</Formula>

The expected portfolio return is:

<Formula>
$$\mu_p = \mathbb{E}[R_p] = \sum_{i=1}^n w_i \mu_i = \mathbf{w}^T \boldsymbol{\mu}$$
</Formula>

### Portfolio Risk

The portfolio variance is:

<Formula>
$$\sigma_p^2 = \sum_{i=1}^n \sum_{j=1}^n w_i w_j \sigma_{ij} = \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w}$$
</Formula>

where $\boldsymbol{\Sigma}$ is the covariance matrix.

```python
import numpy as np

def portfolio_metrics(weights, expected_returns, cov_matrix):
    """
    Calculate portfolio expected return and volatility.

    Args:
        weights: Portfolio weights (n,)
        expected_returns: Expected returns (n,)
        cov_matrix: Covariance matrix (n, n)

    Returns:
        Tuple of (expected_return, volatility)
    """
    weights = np.array(weights)
    expected_returns = np.array(expected_returns)
    cov_matrix = np.array(cov_matrix)

    # Expected return
    portfolio_return = weights @ expected_returns

    # Portfolio variance and volatility
    portfolio_variance = weights @ cov_matrix @ weights
    portfolio_volatility = np.sqrt(portfolio_variance)

    return portfolio_return, portfolio_volatility

# Example with 3 assets
expected_returns = np.array([0.12, 0.08, 0.05])  # 12%, 8%, 5%
cov_matrix = np.array([
    [0.04, 0.01, 0.005],   # 20% volatility
    [0.01, 0.0225, 0.003], # 15% volatility
    [0.005, 0.003, 0.01]   # 10% volatility
])

weights = np.array([0.4, 0.4, 0.2])
ret, vol = portfolio_metrics(weights, expected_returns, cov_matrix)
print(f"Portfolio Return: {ret:.2%}")
print(f"Portfolio Volatility: {vol:.2%}")
```

## 4. The Optimization Problem

The Markowitz optimization problem can be formulated in several equivalent ways:

### Formulation 1: Minimize Risk for Target Return

<Formula>
$$\begin{align}
\min_{\mathbf{w}} \quad & \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w} \\
\text{s.t.} \quad & \mathbf{w}^T \boldsymbol{\mu} = \mu_{target} \\
& \mathbf{w}^T \mathbf{1} = 1
\end{align}$$
</Formula>

### Formulation 2: Maximize Return for Target Risk

<Formula>
$$\begin{align}
\max_{\mathbf{w}} \quad & \mathbf{w}^T \boldsymbol{\mu} \\
\text{s.t.} \quad & \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w} \leq \sigma_{target}^2 \\
& \mathbf{w}^T \mathbf{1} = 1
\end{align}$$
</Formula>

### Formulation 3: Risk-Return Tradeoff

<Formula>
$$\begin{align}
\max_{\mathbf{w}} \quad & \mathbf{w}^T \boldsymbol{\mu} - \frac{\gamma}{2} \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w} \\
\text{s.t.} \quad & \mathbf{w}^T \mathbf{1} = 1
\end{align}$$
</Formula>

where $\gamma$ is the risk aversion parameter.

```python
import numpy as np
from scipy.optimize import minimize

def optimize_portfolio(expected_returns, cov_matrix, target_return=None,
                       risk_aversion=None, bounds=None):
    """
    Optimize portfolio using Markowitz mean-variance optimization.

    Args:
        expected_returns: Expected returns for each asset
        cov_matrix: Covariance matrix
        target_return: Target portfolio return (optional)
        risk_aversion: Risk aversion parameter (optional)
        bounds: Tuple of (min, max) weights per asset

    Returns:
        Optimal weights
    """
    n = len(expected_returns)
    mu = np.array(expected_returns)
    sigma = np.array(cov_matrix)

    # Initial weights
    w0 = np.ones(n) / n

    # Constraints: weights sum to 1
    constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]

    if target_return is not None:
        # Minimize variance for target return
        constraints.append({
            'type': 'eq',
            'fun': lambda w: w @ mu - target_return
        })
        objective = lambda w: w @ sigma @ w

    elif risk_aversion is not None:
        # Maximize utility: return - (gamma/2) * variance
        objective = lambda w: -(w @ mu - (risk_aversion/2) * w @ sigma @ w)

    else:
        # Default: maximize Sharpe ratio (assuming rf=0)
        def objective(w):
            ret = w @ mu
            vol = np.sqrt(w @ sigma @ w)
            return -ret / vol if vol > 0 else 0

    # Bounds
    if bounds is None:
        bounds = [(0, 1) for _ in range(n)]  # Long only

    result = minimize(objective, w0, method='SLSQP',
                     bounds=bounds, constraints=constraints)

    return result.x

# Optimize for maximum Sharpe ratio
optimal_weights = optimize_portfolio(expected_returns, cov_matrix)
print("Optimal weights:", optimal_weights.round(3))

ret, vol = portfolio_metrics(optimal_weights, expected_returns, cov_matrix)
print(f"Return: {ret:.2%}, Volatility: {vol:.2%}")
print(f"Sharpe Ratio: {ret/vol:.3f}")
```

## 5. The Efficient Frontier

The efficient frontier is the set of all optimal portfolios.

<Definition term="Efficient Frontier">
The efficient frontier is the curve in risk-return space representing portfolios that offer the maximum expected return for each level of risk (or minimum risk for each level of return).
</Definition>

```python
import numpy as np
import matplotlib.pyplot as plt

def generate_efficient_frontier(expected_returns, cov_matrix, n_points=100):
    """
    Generate the efficient frontier.
    """
    mu = np.array(expected_returns)
    sigma = np.array(cov_matrix)

    # Range of target returns
    min_ret = mu.min()
    max_ret = mu.max()
    target_returns = np.linspace(min_ret, max_ret, n_points)

    frontier_volatilities = []
    frontier_returns = []
    frontier_weights = []

    for target in target_returns:
        try:
            weights = optimize_portfolio(mu, sigma, target_return=target)
            ret, vol = portfolio_metrics(weights, mu, sigma)
            frontier_returns.append(ret)
            frontier_volatilities.append(vol)
            frontier_weights.append(weights)
        except:
            continue

    return frontier_volatilities, frontier_returns, frontier_weights

# Generate efficient frontier
vols, rets, weights = generate_efficient_frontier(expected_returns, cov_matrix)

# Plot
plt.figure(figsize=(10, 6))
plt.plot(vols, rets, 'b-', linewidth=2, label='Efficient Frontier')

# Plot individual assets
asset_vols = np.sqrt(np.diag(cov_matrix))
plt.scatter(asset_vols, expected_returns, s=100, c='red', marker='o',
            label='Individual Assets')

for i, (vol, ret) in enumerate(zip(asset_vols, expected_returns)):
    plt.annotate(f'Asset {i+1}', (vol, ret), textcoords="offset points",
                 xytext=(5, 5))

plt.xlabel('Volatility (Risk)')
plt.ylabel('Expected Return')
plt.title('Markowitz Efficient Frontier')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

## 6. The Minimum Variance Portfolio

A special point on the efficient frontier is the minimum variance portfolio (MVP).

<Definition term="Minimum Variance Portfolio">
The minimum variance portfolio is the portfolio with the lowest possible volatility among all possible portfolios.
</Definition>

The MVP has an analytical solution:

<Formula>
$$\mathbf{w}_{MVP} = \frac{\boldsymbol{\Sigma}^{-1} \mathbf{1}}{\mathbf{1}^T \boldsymbol{\Sigma}^{-1} \mathbf{1}}$$
</Formula>

```python
def minimum_variance_portfolio(cov_matrix):
    """
    Calculate the minimum variance portfolio weights.
    """
    sigma = np.array(cov_matrix)
    n = len(sigma)
    ones = np.ones(n)

    # Analytical solution
    sigma_inv = np.linalg.inv(sigma)
    weights = sigma_inv @ ones / (ones @ sigma_inv @ ones)

    return weights

mvp_weights = minimum_variance_portfolio(cov_matrix)
print("MVP Weights:", mvp_weights.round(3))

ret, vol = portfolio_metrics(mvp_weights, expected_returns, cov_matrix)
print(f"MVP Return: {ret:.2%}")
print(f"MVP Volatility: {vol:.2%}")
```

## 7. The Tangent Portfolio

When a risk-free asset is available, the optimal risky portfolio is the tangent portfolio.

<Definition term="Tangent Portfolio">
The tangent portfolio is the portfolio on the efficient frontier with the highest Sharpe ratio - it is tangent to the capital allocation line from the risk-free rate.
</Definition>

```python
def tangent_portfolio(expected_returns, cov_matrix, risk_free_rate=0.02):
    """
    Calculate the tangent (maximum Sharpe ratio) portfolio.
    """
    mu = np.array(expected_returns)
    sigma = np.array(cov_matrix)
    n = len(mu)

    # Excess returns
    excess_returns = mu - risk_free_rate

    # Analytical solution
    sigma_inv = np.linalg.inv(sigma)
    weights = sigma_inv @ excess_returns / (np.ones(n) @ sigma_inv @ excess_returns)

    return weights

tangent_weights = tangent_portfolio(expected_returns, cov_matrix, 0.02)
print("Tangent Portfolio Weights:", tangent_weights.round(3))

ret, vol = portfolio_metrics(tangent_weights, expected_returns, cov_matrix)
sharpe = (ret - 0.02) / vol
print(f"Return: {ret:.2%}, Volatility: {vol:.2%}")
print(f"Sharpe Ratio: {sharpe:.3f}")
```

## 8. Why Quantum Computing?

The Markowitz optimization is a convex quadratic program with analytical solutions for simple cases. However, practical portfolio optimization faces challenges:

| Challenge | Classical Issue | Quantum Potential |
|-----------|-----------------|-------------------|
| **Large universes** | O(n³) matrix inversion | Potential speedup |
| **Integer constraints** | NP-hard combinatorial | QAOA optimization |
| **Transaction costs** | Non-convex | Variational methods |
| **Cardinality constraints** | Exponential search | Quantum sampling |

<Tip>
While the basic Markowitz problem is efficiently solvable classically, adding realistic constraints often makes it NP-hard - exactly where quantum computing may provide advantages.
</Tip>
