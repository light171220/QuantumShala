# The Variational Principle

Variational quantum algorithms (VQAs) are the cornerstone of near-term quantum computing, combining quantum circuits with classical optimization to solve complex problems.

## Learning Objectives

After completing this lesson, you will be able to:

- Understand the variational principle in quantum mechanics
- Learn the structure of variational quantum algorithms
- Implement basic variational circuits
- Understand the quantum-classical optimization loop
- Analyze variational algorithm convergence

## 1. The Variational Principle in Quantum Mechanics

The variational principle states that for any trial wavefunction, the expectation value of the Hamiltonian provides an upper bound to the ground state energy.

<Formula>
Variational Principle:
$$E_0 \leq \langle\psi(\theta)|\hat{H}|\psi(\theta)\rangle = E(\theta)$$
where $E_0$ is the true ground state energy and $|\psi(\theta)\rangle$ is a parameterized trial state.
</Formula>

```python
import numpy as np

def variational_principle_demo():
    """
    Demonstrate the variational principle with a simple example.
    """
    print("Variational Principle Demonstration:")
    print("=" * 50)

    # Simple 2x2 Hamiltonian
    H = np.array([[1, 0.5],
                  [0.5, -1]])

    # True ground state energy
    eigenvalues, eigenvectors = np.linalg.eigh(H)
    E0_true = eigenvalues[0]
    ground_state = eigenvectors[:, 0]

    print(f"\nHamiltonian:\n{H}")
    print(f"\nTrue ground state energy: {E0_true:.6f}")

    # Trial wavefunction |ψ(θ)⟩ = cos(θ)|0⟩ + sin(θ)|1⟩
    def trial_state(theta):
        return np.array([np.cos(theta), np.sin(theta)])

    def energy(theta):
        psi = trial_state(theta)
        return np.real(psi.conj() @ H @ psi)

    # Evaluate for different angles
    print("\nVariational energies for different θ:")
    print("-" * 40)

    thetas = np.linspace(0, np.pi, 9)
    for theta in thetas:
        E = energy(theta)
        print(f"  θ = {theta:.4f} rad: E(θ) = {E:+.6f} {'← minimum' if abs(E - E0_true) < 0.01 else ''}")

    # Find optimal theta
    from scipy.optimize import minimize_scalar
    result = minimize_scalar(energy, bounds=(0, np.pi), method='bounded')

    print(f"\nOptimal θ: {result.x:.6f} rad")
    print(f"Variational energy: {result.fun:.6f}")
    print(f"True ground state: {E0_true:.6f}")
    print(f"Error: {abs(result.fun - E0_true):.2e}")

variational_principle_demo()
```

<Tip>
The variational principle guarantees we can only overestimate the ground state energy, never underestimate it. This makes it a powerful tool for finding ground states.
</Tip>

## 2. Structure of Variational Quantum Algorithms

VQAs consist of three main components: state preparation, measurement, and classical optimization.

<Definition term="Variational Quantum Algorithm (VQA)">
A hybrid quantum-classical algorithm that uses a parameterized quantum circuit (ansatz) to prepare trial states, measures expectation values on a quantum computer, and optimizes parameters using a classical computer.
</Definition>

```python
import numpy as np

class VariationalQuantumAlgorithm:
    """
    Basic framework for variational quantum algorithms.
    """

    def __init__(self, n_qubits, hamiltonian):
        self.n_qubits = n_qubits
        self.dim = 2**n_qubits
        self.hamiltonian = hamiltonian

    def ansatz(self, params):
        """
        Create parameterized quantum state.
        Simple rotation ansatz for demonstration.
        """
        # Start with |0...0⟩
        state = np.zeros(self.dim, dtype=complex)
        state[0] = 1.0

        # Apply parameterized rotations (simplified)
        n_params = len(params)
        for i, theta in enumerate(params):
            # Rotation matrix (simplified single-qubit rotation)
            qubit = i % self.n_qubits
            state = self._apply_ry(state, qubit, theta)

        return state

    def _apply_ry(self, state, qubit, theta):
        """Apply RY rotation to a specific qubit."""
        ry = np.array([[np.cos(theta/2), -np.sin(theta/2)],
                       [np.sin(theta/2), np.cos(theta/2)]])

        new_state = np.zeros_like(state)
        for i in range(self.dim):
            bit = (i >> (self.n_qubits - 1 - qubit)) & 1
            partner = i ^ (1 << (self.n_qubits - 1 - qubit))

            if bit == 0:
                new_state[i] += ry[0, 0] * state[i] + ry[0, 1] * state[partner]
            else:
                new_state[i] += ry[1, 0] * state[partner] + ry[1, 1] * state[i]

        return new_state

    def expectation_value(self, params):
        """Compute ⟨ψ(θ)|H|ψ(θ)⟩."""
        state = self.ansatz(params)
        return np.real(state.conj() @ self.hamiltonian @ state)

    def optimize(self, initial_params, method='COBYLA', maxiter=100):
        """Run classical optimization loop."""
        from scipy.optimize import minimize

        history = {'params': [], 'energies': []}

        def objective(params):
            energy = self.expectation_value(params)
            history['params'].append(params.copy())
            history['energies'].append(energy)
            return energy

        result = minimize(objective, initial_params, method=method,
                         options={'maxiter': maxiter})

        return result, history

# Demonstration
print("Variational Quantum Algorithm Framework:")
print("=" * 50)

# Create simple Hamiltonian (2-qubit Heisenberg)
I = np.eye(2)
X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])

H = (np.kron(X, X) + np.kron(Y, Y) + np.kron(Z, Z)) / 4

vqa = VariationalQuantumAlgorithm(n_qubits=2, hamiltonian=H)

# Initial parameters
np.random.seed(42)
initial_params = np.random.uniform(0, 2*np.pi, 4)

print(f"\nHamiltonian: Heisenberg XXX (2 qubits)")
print(f"Initial parameters: {initial_params}")
print(f"Initial energy: {vqa.expectation_value(initial_params):.6f}")

# Optimize
result, history = vqa.optimize(initial_params, maxiter=50)

print(f"\nOptimized energy: {result.fun:.6f}")
print(f"Iterations: {len(history['energies'])}")

# True ground state
eigenvalues = np.linalg.eigvalsh(H)
print(f"True ground state: {eigenvalues[0]:.6f}")
```

## 3. The Quantum-Classical Optimization Loop

Understanding the interplay between quantum and classical components is essential.

```python
import numpy as np

def optimization_loop_analysis():
    """
    Analyze the quantum-classical optimization loop.
    """
    print("Quantum-Classical Optimization Loop:")
    print("=" * 50)

    # Simulated cost landscape
    def cost_function(params):
        x, y = params
        # Multiple local minima landscape
        return (np.sin(x) * np.cos(y) + 0.5 * (x**2 + y**2) / 10
                - np.exp(-((x-1)**2 + (y-1)**2) / 2))

    # Compare different optimizers
    optimizers = {
        'COBYLA': {'method': 'COBYLA'},
        'Nelder-Mead': {'method': 'Nelder-Mead'},
        'Powell': {'method': 'Powell'},
        'L-BFGS-B': {'method': 'L-BFGS-B'}
    }

    print("\nComparing classical optimizers:")
    print("-" * 40)

    from scipy.optimize import minimize

    np.random.seed(42)
    initial = np.random.uniform(-2, 2, 2)

    for name, opts in optimizers.items():
        result = minimize(cost_function, initial, **opts,
                         options={'maxiter': 100})
        print(f"  {name:15s}: f = {result.fun:+.6f}, "
              f"iterations = {result.nfev:3d}")

    # Demonstrate shot noise effect
    print("\n\nEffect of shot noise on optimization:")
    print("-" * 40)

    def noisy_cost(params, n_shots):
        true_cost = cost_function(params)
        # Add shot noise (scales as 1/sqrt(n_shots))
        noise = np.random.normal(0, 1.0 / np.sqrt(n_shots))
        return true_cost + noise

    shot_counts = [100, 1000, 10000, 100000]
    print(f"\nTrue cost at initial point: {cost_function(initial):.6f}")
    print("\nNoisy estimates:")

    for n_shots in shot_counts:
        estimates = [noisy_cost(initial, n_shots) for _ in range(10)]
        mean_est = np.mean(estimates)
        std_est = np.std(estimates)
        print(f"  {n_shots:6d} shots: {mean_est:.6f} ± {std_est:.6f}")

optimization_loop_analysis()
```

<Warning>
Shot noise from finite measurements can significantly impact optimization convergence. Using too few shots leads to noisy gradients, while too many shots wastes quantum resources.
</Warning>

## 4. Implementing a Basic VQE

The Variational Quantum Eigensolver (VQE) is the prototypical variational algorithm.

```python
import numpy as np
from scipy.optimize import minimize

class SimpleVQE:
    """
    Simple VQE implementation for educational purposes.
    """

    def __init__(self, hamiltonian, n_qubits, n_layers=2):
        self.H = hamiltonian
        self.n_qubits = n_qubits
        self.n_layers = n_layers
        self.dim = 2**n_qubits

        # Count parameters: 2 rotations per qubit per layer + entangling
        self.n_params = n_qubits * 2 * n_layers

    def prepare_state(self, params):
        """
        Prepare variational state using hardware-efficient ansatz.
        """
        state = np.zeros(self.dim, dtype=complex)
        state[0] = 1.0  # |00...0⟩

        param_idx = 0
        for layer in range(self.n_layers):
            # Single-qubit rotations
            for q in range(self.n_qubits):
                state = self._ry(state, q, params[param_idx])
                param_idx += 1
                state = self._rz(state, q, params[param_idx])
                param_idx += 1

            # Entangling layer (CZ gates)
            for q in range(self.n_qubits - 1):
                state = self._cz(state, q, q + 1)

        return state

    def _ry(self, state, qubit, theta):
        """Apply RY gate."""
        c, s = np.cos(theta/2), np.sin(theta/2)
        new_state = np.zeros_like(state)

        for i in range(self.dim):
            bit = (i >> (self.n_qubits - 1 - qubit)) & 1
            partner = i ^ (1 << (self.n_qubits - 1 - qubit))

            if bit == 0:
                new_state[i] += c * state[i] - s * state[partner]
                new_state[partner] += s * state[i] + c * state[partner]

        return new_state

    def _rz(self, state, qubit, theta):
        """Apply RZ gate."""
        new_state = state.copy()
        for i in range(self.dim):
            bit = (i >> (self.n_qubits - 1 - qubit)) & 1
            phase = np.exp(-1j * theta / 2) if bit == 0 else np.exp(1j * theta / 2)
            new_state[i] *= phase
        return new_state

    def _cz(self, state, q1, q2):
        """Apply CZ gate."""
        new_state = state.copy()
        for i in range(self.dim):
            b1 = (i >> (self.n_qubits - 1 - q1)) & 1
            b2 = (i >> (self.n_qubits - 1 - q2)) & 1
            if b1 == 1 and b2 == 1:
                new_state[i] *= -1
        return new_state

    def energy(self, params):
        """Compute expectation value ⟨H⟩."""
        state = self.prepare_state(params)
        return np.real(state.conj() @ self.H @ state)

    def run(self, initial_params=None, maxiter=200):
        """Run VQE optimization."""
        if initial_params is None:
            initial_params = np.random.uniform(0, 2*np.pi, self.n_params)

        self.history = []

        def callback_energy(params):
            e = self.energy(params)
            self.history.append(e)
            return e

        result = minimize(callback_energy, initial_params,
                         method='COBYLA', options={'maxiter': maxiter})

        return result

# Demonstration
print("Simple VQE Implementation:")
print("=" * 50)

# H2 molecule Hamiltonian (simplified)
H_h2 = np.array([
    [-1.0524, 0, 0, 0.1809],
    [0, -0.3913, 0.1809, 0],
    [0, 0.1809, -0.3913, 0],
    [0.1809, 0, 0, -1.0524]
])

vqe = SimpleVQE(H_h2, n_qubits=2, n_layers=2)

print(f"\nNumber of parameters: {vqe.n_params}")
print(f"Number of layers: {vqe.n_layers}")

np.random.seed(42)
result = vqe.run(maxiter=100)

print(f"\nVQE Result:")
print(f"  Final energy: {result.fun:.6f}")
print(f"  Iterations: {len(vqe.history)}")

# Compare to exact
exact_energy = np.min(np.linalg.eigvalsh(H_h2))
print(f"  Exact ground state: {exact_energy:.6f}")
print(f"  Error: {abs(result.fun - exact_energy):.6f}")
```

## 5. Convergence Analysis

Understanding when and why VQAs converge (or fail to converge).

```python
import numpy as np

def convergence_analysis():
    """
    Analyze VQA convergence behavior.
    """
    print("VQA Convergence Analysis:")
    print("=" * 50)

    # Simple cost landscape
    def cost(params):
        return np.sum(np.sin(params)**2) + 0.1 * np.sum(params**2)

    def gradient(params):
        return np.sin(2 * params) + 0.2 * params

    # Gradient descent with different learning rates
    n_params = 5
    n_steps = 100

    learning_rates = [0.01, 0.1, 0.5, 1.0]

    print("\nGradient descent convergence:")
    print("-" * 40)

    np.random.seed(42)
    initial = np.random.uniform(-np.pi, np.pi, n_params)

    for lr in learning_rates:
        params = initial.copy()
        costs = [cost(params)]

        for _ in range(n_steps):
            params = params - lr * gradient(params)
            costs.append(cost(params))

        final_cost = costs[-1]
        converged = final_cost < 0.1

        print(f"  lr = {lr:.2f}: final cost = {final_cost:.6f} "
              f"{'(converged)' if converged else '(not converged)'}")

    # Effect of initialization
    print("\n\nEffect of initialization:")
    print("-" * 40)

    n_trials = 20
    results = []

    for trial in range(n_trials):
        np.random.seed(trial)
        params = np.random.uniform(-np.pi, np.pi, n_params)

        for _ in range(100):
            params = params - 0.1 * gradient(params)

        results.append(cost(params))

    print(f"  Best result: {min(results):.6f}")
    print(f"  Worst result: {max(results):.6f}")
    print(f"  Mean result: {np.mean(results):.6f}")
    print(f"  Std deviation: {np.std(results):.6f}")

    # Landscape ruggedness
    print("\n\nCost landscape statistics:")
    print("-" * 40)

    n_samples = 1000
    random_costs = []
    for _ in range(n_samples):
        random_params = np.random.uniform(-np.pi, np.pi, n_params)
        random_costs.append(cost(random_params))

    print(f"  Random sampling:")
    print(f"    Min: {min(random_costs):.6f}")
    print(f"    Max: {max(random_costs):.6f}")
    print(f"    Mean: {np.mean(random_costs):.6f}")

convergence_analysis()
```

<Tip>
Multiple random initializations often help find better minima. Consider using techniques like basin-hopping or genetic algorithms for challenging optimization landscapes.
</Tip>

## Summary

The variational principle provides the theoretical foundation for VQAs:

1. **Variational Principle**: Trial states give upper bounds to ground state energy
2. **VQA Structure**: Parameterized circuits + classical optimization
3. **Optimization Loop**: Quantum measurement feeds classical optimizer
4. **VQE**: Flagship algorithm for quantum chemistry
5. **Convergence**: Depends on ansatz, optimizer, and initialization

Understanding these fundamentals is essential for designing effective variational algorithms.
