# Quantum Support Vector Machines

Quantum Support Vector Machines (QSVMs) combine the power of quantum kernels with the classical SVM optimization framework, potentially offering advantages for certain classification tasks.

## Learning Objectives

After completing this lesson, you will be able to:

- Understand SVM optimization with quantum kernels
- Implement quantum kernel SVM classifiers
- Apply quantum SVM to real datasets
- Compare quantum and classical SVM performance
- Handle multi-class classification with quantum kernels

## 1. SVM with Quantum Kernels

The classical SVM framework seamlessly integrates with quantum kernels.

<Definition term="Quantum SVM">
A support vector machine that uses a quantum kernel K(x,y) = |⟨φ(x)|φ(y)⟩|² computed on a quantum computer, where φ(x) is a quantum feature map encoding classical data into quantum states.
</Definition>

```python

```

<Warning>
The quantum advantage in QSVM comes from the kernel, not the SVM optimization. The SVM training is still performed classically after computing the quantum kernel matrix.
</Warning>

## 2. Implementing Quantum SVM

A complete implementation of QSVM with simulated quantum kernels.

```python
import numpy as np
from scipy.optimize import minimize

class QuantumSVM:
    """
    Quantum Support Vector Machine classifier.
    """

    def __init__(self, quantum_kernel, C=1.0):
        """
        Initialize QSVM.

        Args:
            quantum_kernel: Function K(x, y) -> kernel value
            C: Regularization parameter
        """
        self.kernel = quantum_kernel
        self.C = C
        self.alpha = None
        self.support_vectors = None
        self.support_labels = None
        self.b = None

    def _compute_kernel_matrix(self, X):
        """Compute Gram matrix."""
        n = len(X)
        K = np.zeros((n, n))
        for i in range(n):
            for j in range(n):
                K[i, j] = self.kernel(X[i], X[j])
        return K

    def fit(self, X, y):
        """Train the QSVM."""
        n = len(X)
        y = np.array(y, dtype=float)

        # Compute kernel matrix
        K = self._compute_kernel_matrix(X)

        # SVM dual objective
        def objective(alpha):
            return 0.5 * np.sum(alpha.reshape(-1,1) * alpha * y.reshape(-1,1) * y * K) - np.sum(alpha)

        def gradient(alpha):
            return np.sum(alpha * y * y.reshape(-1,1) * K, axis=1) - 1

        # Constraints
        constraints = [
            {'type': 'eq', 'fun': lambda a: np.dot(a, y)},  # Σαy = 0
        ]
        bounds = [(0, self.C) for _ in range(n)]

        # Optimize
        alpha0 = np.zeros(n)
        result = minimize(objective, alpha0, jac=gradient,
                         bounds=bounds, constraints=constraints,
                         method='SLSQP')

        self.alpha = result.x

        # Find support vectors
        sv_threshold = 1e-5
        sv_idx = self.alpha > sv_threshold

        self.support_vectors = X[sv_idx]
        self.support_labels = y[sv_idx]
        self.support_alpha = self.alpha[sv_idx]

        # Compute bias
        margin_sv = (self.alpha > sv_threshold) & (self.alpha < self.C - sv_threshold)
        if np.any(margin_sv):
            idx = np.where(margin_sv)[0][0]
            self.b = y[idx] - np.sum(
                self.support_alpha * self.support_labels *
                np.array([self.kernel(sv, X[idx]) for sv in self.support_vectors])
            )
        else:
            self.b = 0

        return self

    def predict(self, X):
        """Predict class labels."""
        predictions = []
        for x in X:
            score = self.b
            for alpha, y_sv, sv in zip(self.support_alpha,
                                        self.support_labels,
                                        self.support_vectors):
                score += alpha * y_sv * self.kernel(sv, x)
            predictions.append(np.sign(score))
        return np.array(predictions)

    def score(self, X, y):
        """Compute accuracy."""
        predictions = self.predict(X)
        return np.mean(predictions == y)

# Demonstration
print("Quantum SVM Implementation:")
print("=" * 50)

# Simulated quantum kernel (ZZ feature map style)
def quantum_kernel(x, y, gamma=1.0):
    """Simulated quantum kernel."""
    # ZZ-style interactions
    dot = np.dot(x, y)
    diff_sq = np.sum((x - y)**2)
    return np.exp(-gamma * diff_sq) * (0.5 + 0.5 * np.cos(dot))

# Generate sample data

# Two moons-like dataset
```

## 3. Quantum Kernel Design for Classification

Choosing appropriate quantum kernels for classification tasks.

```python
import numpy as np

def kernel_design_for_svm():
    """
    Explore kernel design choices for QSVM.
    """
    print("Quantum Kernel Design for Classification:")
    print("=" * 50)

    # Different quantum-inspired kernels
    class QuantumKernels:
        @staticmethod
        def amplitude_encoding(x, y):
            """Kernel from amplitude encoding."""
            x_norm = x / (np.linalg.norm(x) + 1e-10)
            y_norm = y / (np.linalg.norm(y) + 1e-10)
            overlap = np.dot(x_norm, y_norm)
            return overlap**2

        @staticmethod
        def angle_encoding(x, y, reps=1):
            """Kernel from angle/rotation encoding."""
            # Approximate IQP-style kernel
            phase = 0
            for r in range(reps):
                phase += np.sum(x * y)
                phase += np.sum(np.outer(x, y))
            return (1 + np.cos(phase)) / 2

        @staticmethod
        def zz_feature_map(x, y, reps=2):
            """Kernel from ZZ feature map."""
            inner = np.dot(x, y)
            cross = np.sum(np.outer(x, x) * np.outer(y, y))

            # Simulate multiple repetitions
            phase = reps * (inner + 0.5 * cross)
            return (1 + np.cos(phase)) / 2

        @staticmethod
        def iqp_kernel(x, y):
            """Instantaneous Quantum Polynomial kernel."""
            # Higher-order correlations
            linear = np.dot(x, y)
            quadratic = np.sum(x**2) * np.sum(y**2)
            cubic = np.sum(x**3) * np.sum(y**3)

            phase = linear + 0.1 * quadratic + 0.01 * cubic
            return np.exp(1j * phase).real ** 2

    print("\n1. Kernel Comparison:")
    print("-" * 40)

    np.random.seed(42)
    x = np.array([0.5, 1.0])
    y = np.array([0.7, 0.9])

    kernels = [
        ("Amplitude", QuantumKernels.amplitude_encoding),
        ("Angle", QuantumKernels.angle_encoding),
        ("ZZ (r=2)", lambda a,b: QuantumKernels.zz_feature_map(a,b,2)),
        ("IQP", QuantumKernels.iqp_kernel),
    ]

    print(f"\n  Test vectors: x={x}, y={y}")
    print(f"\n  {'Kernel':<15} {'K(x,y)':<10} {'K(x,x)':<10}")
    print("  " + "-" * 35)

    for name, kernel in kernels:
        kxy = kernel(x, y)
        kxx = kernel(x, x)
        print(f"  {name:<15} {kxy:<10.4f} {kxx:<10.4f}")

    # Test separability
    print("\n\n2. XOR Problem (Classically Non-Separable):")
    print("-" * 40)

    X_xor = np.array([[0, 0], [0, 1], [1, 0], [1, 1]], dtype=float)
    y_xor = np.array([1, -1, -1, 1])

    print("\n  Kernel matrix structure for XOR:")

    for name, kernel in kernels[:3]:
        K = np.array([[kernel(X_xor[i], X_xor[j])
                       for j in range(4)] for i in range(4)])
        # Check if kernel matrix allows separation
        eigenvalues = np.linalg.eigvalsh(K)
        print(f"\n  {name}:")
        print(f"    Eigenvalues: {eigenvalues.round(3)}")

kernel_design_for_svm()
```

<Tip>
For classification, choose quantum kernels that create separation between classes. ZZ feature maps with multiple repetitions often work well for non-linear decision boundaries.
</Tip>

## 4. Hyperparameter Tuning

Optimizing QSVM performance through hyperparameter selection.

```python
import numpy as np

def qsvm_hyperparameter_tuning():
    """
    Demonstrate hyperparameter tuning for QSVM.
    """
    print("QSVM Hyperparameter Tuning:")
    print("=" * 50)

    def parameterized_quantum_kernel(x, y, gamma=1.0, reps=1):
        """Quantum kernel with tunable parameters."""
        diff_sq = np.sum((x - y)**2)
        dot = np.dot(x, y)
        phase = reps * gamma * (dot + 0.5 * np.sum(np.outer(x, y)))
        return np.exp(-0.5 * diff_sq) * (1 + np.cos(phase)) / 2

    def cross_validation_score(X, y, kernel_fn, C, n_folds=5):
        """Simple k-fold cross-validation."""
        n = len(X)
        indices = np.arange(n)
        np.random.shuffle(indices)
        fold_size = n // n_folds

        scores = []
        for fold in range(n_folds):
            # Split data
            val_idx = indices[fold * fold_size:(fold + 1) * fold_size]
            train_idx = np.concatenate([
                indices[:fold * fold_size],
                indices[(fold + 1) * fold_size:]
            ])

            X_train, y_train = X[train_idx], y[train_idx]
            X_val, y_val = X[val_idx], y[val_idx]

            # Train simple model (using kernel regression for speed)
            K_train = np.array([[kernel_fn(X_train[i], X_train[j])
                                for j in range(len(X_train))]
                               for i in range(len(X_train))])

            # Ridge regression with kernel
            alpha = np.linalg.solve(K_train + C * np.eye(len(K_train)), y_train)

            # Predict
            preds = []
            for x in X_val:
                k_vec = np.array([kernel_fn(x, X_train[i])
                                 for i in range(len(X_train))])
                preds.append(np.sign(np.dot(k_vec, alpha)))

            scores.append(np.mean(preds == y_val))

        return np.mean(scores)

    # Generate dataset
    np.random.seed(42)
    n_samples = 80

    # Spiral dataset
    t = np.linspace(0, 2*np.pi, n_samples // 2)
    r = t / (2 * np.pi)
    X_pos = np.column_stack([r * np.cos(t), r * np.sin(t)]) + 0.05 * np.random.randn(n_samples//2, 2)
    X_neg = np.column_stack([r * np.cos(t + np.pi), r * np.sin(t + np.pi)]) + 0.05 * np.random.randn(n_samples//2, 2)

    X = np.vstack([X_pos, X_neg])
    y = np.array([1] * (n_samples//2) + [-1] * (n_samples//2))

    print("\n1. Grid Search Over Parameters:")
    print("-" * 40)

    gammas = [0.5, 1.0, 2.0, 5.0]
    reps_list = [1, 2, 3]
    C_values = [0.1, 1.0, 10.0]

    best_score = 0
    best_params = {}

    print(f"\n  {'gamma':<8} {'reps':<8} {'C':<8} {'CV Score':<10}")
    print("  " + "-" * 34)

    for gamma in gammas:
        for reps in reps_list:
            kernel = lambda x, y, g=gamma, r=reps: parameterized_quantum_kernel(x, y, g, r)
            for C in C_values:
                score = cross_validation_score(X, y, kernel, C, n_folds=3)
                if score > best_score:
                    best_score = score
                    best_params = {'gamma': gamma, 'reps': reps, 'C': C}

    print(f"\n  Best parameters:")
    print(f"    gamma = {best_params['gamma']}")
    print(f"    reps = {best_params['reps']}")
    print(f"    C = {best_params['C']}")
    print(f"    CV Score = {best_score:.2%}")

    print("\n\n2. Parameter Effects:")
    print("-" * 40)

    effects = [
        ("gamma (↑)", "Sharper kernel", "May overfit"),
        ("reps (↑)", "More expressible", "Harder to train"),
        ("C (↑)", "Stricter margin", "Less regularization"),
    ]

    for param, effect, risk in effects:
        print(f"  {param:<12}: {effect:<20} Risk: {risk}")

qsvm_hyperparameter_tuning()
```

## 5. Multi-Class Classification

Extending QSVM to multi-class problems.

```python
import numpy as np

def multiclass_qsvm():
    """
    Multi-class classification with Quantum SVM.
    """
    print("Multi-Class Quantum SVM:")
    print("=" * 50)

    print("\n1. Strategies for Multi-Class:")
    print("-" * 40)

    strategies = [
        ("One-vs-Rest (OvR)", "Train K binary classifiers, each separating one class from rest"),
        ("One-vs-One (OvO)", "Train K(K-1)/2 classifiers, one for each pair of classes"),
        ("Direct Multi-class", "Modify SVM formulation to handle multiple classes directly"),
    ]

    for strategy, desc in strategies:
        print(f"\n  {strategy}:")
        print(f"    {desc}")

    # Implement One-vs-Rest
    class MultiClassQSVM:
        def __init__(self, kernel, C=1.0):
            self.kernel = kernel
            self.C = C
            self.classifiers = {}
            self.classes = None

        def fit(self, X, y):
            self.classes = np.unique(y)
            n = len(X)

            for cls in self.classes:
                # Binary labels: current class vs rest
                y_binary = np.where(y == cls, 1, -1).astype(float)

                # Compute kernel matrix
                K = np.array([[self.kernel(X[i], X[j])
                              for j in range(n)] for i in range(n)])

                # Simple kernel ridge classifier
                alpha = np.linalg.solve(K + self.C * np.eye(n), y_binary)

                self.classifiers[cls] = {'X': X.copy(), 'alpha': alpha}

            return self

        def predict(self, X):
            scores = {}
            for cls, model in self.classifiers.items():
                clf_scores = []
                for x in X:
                    k_vec = np.array([self.kernel(x, xi)
                                     for xi in model['X']])
                    clf_scores.append(np.dot(k_vec, model['alpha']))
                scores[cls] = np.array(clf_scores)

            # Predict class with highest score
            predictions = []
            for i in range(len(X)):
                best_cls = max(self.classes, key=lambda c: scores[c][i])
                predictions.append(best_cls)

            return np.array(predictions)

    # Test on 3-class problem
    print("\n\n2. Example: 3-Class Classification")
    print("-" * 40)

    np.random.seed(42)

    # Generate 3 clusters
    n_per_class = 30
    centers = np.array([[0, 0], [2, 0], [1, 1.7]])

    X = np.vstack([
        centers[i] + 0.3 * np.random.randn(n_per_class, 2)
        for i in range(3)
    ])
    y = np.repeat([0, 1, 2], n_per_class)

    # Quantum kernel
    def qkernel(x, y):
        diff = np.sum((x - y)**2)
        dot = np.dot(x, y)
        return np.exp(-diff) * (1 + np.cos(dot)) / 2

    # Train and evaluate
    clf = MultiClassQSVM(qkernel, C=0.1)
    clf.fit(X, y)

    predictions = clf.predict(X)
    accuracy = np.mean(predictions == y)

    print(f"\n  Classes: {list(clf.classes)}")
    print(f"  Training samples: {len(X)}")
    print(f"  Accuracy: {accuracy:.2%}")

    # Confusion matrix
    print("\n  Confusion Matrix:")
    for true_cls in clf.classes:
        row = []
        for pred_cls in clf.classes:
            count = np.sum((y == true_cls) & (predictions == pred_cls))
            row.append(count)
        print(f"    True {true_cls}: {row}")

multiclass_qsvm()
```

<Warning>
One-vs-One requires O(K²) classifiers and kernel evaluations, which can become expensive on quantum hardware when K is large.
</Warning>

## 6. Performance Comparison

Comparing QSVM with classical SVM.

```python
import numpy as np

def qsvm_vs_classical():
    """
    Compare Quantum SVM with Classical SVM.
    """
    print("QSVM vs Classical SVM:")
    print("=" * 50)

    # Define kernels
    def classical_rbf(x, y, gamma=1.0):
        return np.exp(-gamma * np.sum((x - y)**2))

    def quantum_zz(x, y, gamma=1.0, reps=2):
        diff = np.sum((x - y)**2)
        dot = np.dot(x, y)
        phase = reps * (dot + 0.3 * np.sum(np.outer(x, y)))
        return np.exp(-0.5 * gamma * diff) * (1 + np.cos(phase)) / 2

    def kernel_classifier(X_train, y_train, X_test, kernel, reg=0.1):
        """Simple kernel ridge classifier."""
        n = len(X_train)
        K = np.array([[kernel(X_train[i], X_train[j])
                      for j in range(n)] for i in range(n)])
        alpha = np.linalg.solve(K + reg * np.eye(n), y_train)

        predictions = []
        for x in X_test:
            k_vec = np.array([kernel(x, X_train[i]) for i in range(n)])
            predictions.append(np.sign(np.dot(k_vec, alpha)))
        return np.array(predictions)

    # Datasets
    datasets = []

    # Linear separable
    np.random.seed(42)
    n = 50
    X1 = np.vstack([np.random.randn(n//2, 2) + [2, 0],
                    np.random.randn(n//2, 2) + [-2, 0]])
    y1 = np.array([1]*(n//2) + [-1]*(n//2))
    datasets.append(("Linear", X1, y1))

    # Circles
    t = np.linspace(0, 2*np.pi, n//2)
    X2 = np.vstack([0.5 * np.column_stack([np.cos(t), np.sin(t)]) + 0.1*np.random.randn(n//2, 2),
                    1.5 * np.column_stack([np.cos(t), np.sin(t)]) + 0.1*np.random.randn(n//2, 2)])
    y2 = np.array([1]*(n//2) + [-1]*(n//2))
    datasets.append(("Circles", X2, y2))

    # XOR-like
    X3 = np.random.randn(n, 2)
    y3 = np.sign(X3[:, 0] * X3[:, 1])
    datasets.append(("XOR-like", X3, y3))

    print("\n  Comparison Results:")
    print("-" * 50)
    print(f"  {'Dataset':<12} {'Classical RBF':<15} {'Quantum ZZ':<15}")
    print("  " + "-" * 42)

    for name, X, y in datasets:
        # Split data
        split = int(0.7 * len(X))
        X_train, X_test = X[:split], X[split:]
        y_train, y_test = y[:split], y[split:]

        # Evaluate
        pred_classical = kernel_classifier(X_train, y_train, X_test, classical_rbf)
        pred_quantum = kernel_classifier(X_train, y_train, X_test, quantum_zz)

        acc_classical = np.mean(pred_classical == y_test)
        acc_quantum = np.mean(pred_quantum == y_test)

        print(f"  {name:<12} {acc_classical:<15.2%} {acc_quantum:<15.2%}")

    print("\n\n  Key Observations:")
    print("-" * 40)
    observations = [
        "Similar performance on simple problems",
        "Quantum kernels may help on specific structures",
        "Real advantage depends on data and feature map",
        "Hardware noise affects quantum kernel accuracy"
    ]
    for obs in observations:
        print(f"  • {obs}")

qsvm_vs_classical()
```

## Summary

Quantum SVMs combine quantum kernels with classical SVM optimization:

1. **Framework**: Quantum kernel replaces classical kernel in SVM
2. **Implementation**: Compute kernel matrix on quantum device, optimize classically
3. **Kernel Design**: Choose feature maps suited to classification task
4. **Hyperparameters**: Tune gamma, reps, C via cross-validation
5. **Multi-Class**: Use One-vs-Rest or One-vs-One strategies
6. **Performance**: Advantage depends on data structure and feature map

QSVM is often the simplest entry point for quantum machine learning classification.
