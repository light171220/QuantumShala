## 1. RSA Overview

RSA (Rivest–Shamir–Adleman) is the most widely used public-key cryptosystem:

<Definition term="Public-Key Cryptography">
A cryptographic system using a pair of keys:
- **Public key**: Can be shared with anyone (used for encryption)
- **Private key**: Must be kept secret (used for decryption)
</Definition>

```python
def rsa_overview():
    """Overview of RSA key usage."""
    print("RSA Cryptosystem Overview")
    print("=" * 60)
    print("""
    ┌─────────────────────────────────────────────────────────┐

    │  1. Choose large primes p, q                            │
    │  2. Compute N = p × q (public modulus)                  │
    │  3. Compute φ(N) = (p-1)(q-1)                          │

    Private Key: (d)     ← Keep secret!
    
    ┌─────────────────────────────────────────────────────────┐

    │  Ciphertext C = M^e mod N                               │
    │  (Anyone can encrypt using public key)                  │
    └─────────────────────────────────────────────────────────┘
    
    ┌─────────────────────────────────────────────────────────┐
    │                    DECRYPTION                            │
    │  Message M = C^d mod N                                  │
    │  (Only private key holder can decrypt)                  │
    └─────────────────────────────────────────────────────────┘
    """)
```

## 2. Mathematical Foundation

RSA relies on **Euler's Theorem**:

<Formula>
$$a^{\phi(N)} \equiv 1 \pmod{N}$$
</Formula>

where φ(N) is Euler's totient function.

For N = p × q (semiprime):
<Formula>
$$\phi(N) = (p-1)(q-1)$$
</Formula>

```python
import math

def euler_totient(n):
    """Compute Euler's totient function φ(n)."""
    result = n
    p = 2
    while p * p <= n:
        if n % p == 0:
            while n % p == 0:
                n //= p
            result -= result // p
        p += 1
    if n > 1:
        result -= result // n
    return result

def euler_totient_semiprime(p, q):
    """For N = p*q, φ(N) = (p-1)(q-1)."""
    return (p - 1) * (q - 1)

# Demonstrate Euler's theorem
print("Euler's Theorem Demonstration")
print("=" * 50)

p, q = 11, 13
N = p * q
phi_N = euler_totient_semiprime(p, q)

print(f"p = {p}, q = {q}")
print(f"N = p × q = {N}")
print(f"φ(N) = (p-1)(q-1) = {phi_N}")
print()

# Verify: a^φ(N) ≡ 1 (mod N) for any a coprime to N
for a in [2, 3, 5, 7]:
    result = pow(a, phi_N, N)
    print(f"{a}^{phi_N} mod {N} = {result}")
```

## 3. Complete RSA Implementation

```python
import random

def is_prime(n, k=10):
    """Miller-Rabin primality test."""
    if n < 2:
        return False
    if n == 2 or n == 3:
        return True
    if n % 2 == 0:
        return False
    
    r, d = 0, n - 1
    while d % 2 == 0:
        r += 1
        d //= 2
    
    for _ in range(k):
        a = random.randrange(2, n - 1)
        x = pow(a, d, n)
        if x == 1 or x == n - 1:
            continue
        for _ in range(r - 1):
            x = pow(x, 2, n)
            if x == n - 1:
                break
        else:
            return False
    return True

def generate_prime(bits):
    """Generate a random prime of specified bit length."""
    while True:
        p = random.getrandbits(bits) | (1 << (bits - 1)) | 1
        if is_prime(p):
            return p

def gcd(a, b):
    """Greatest common divisor."""
    while b:
        a, b = b, a % b
    return a

def rsa_keygen(bits=512):
    """
    Generate RSA key pair.
    
    Returns:
        public_key: (N, e)
        private_key: (N, d)
        secrets: (p, q, phi_N) - for educational purposes
    """
    # Step 1: Generate two large primes
    p = generate_prime(bits // 2)
    q = generate_prime(bits // 2)
    
    # Step 2: Compute modulus
    N = p * q
    
    # Step 3: Compute Euler's totient
    phi_N = (p - 1) * (q - 1)
    
    # Step 4: Choose public exponent e
    # Common choice: 65537 = 2^16 + 1 (Fermat prime)
    e = 65537
    assert gcd(e, phi_N) == 1, "e must be coprime to φ(N)"
    
    # Step 5: Compute private exponent d
    d = pow(e, -1, phi_N)
    
    return (N, e), (N, d), (p, q, phi_N)

def rsa_encrypt(message, public_key):
    """Encrypt message using public key."""
    N, e = public_key
    return pow(message, e, N)

def rsa_decrypt(ciphertext, private_key):
    """Decrypt ciphertext using private key."""
    N, d = private_key
    return pow(ciphertext, d, N)

# Demonstrate complete RSA
print("Complete RSA Demonstration")
print("=" * 60)

# Generate keys (small for demonstration)
pub, priv, secrets = rsa_keygen(bits=64)
N, e = pub
_, d = priv
p, q, phi_N = secrets

print(f"\nKey Generation:")
print(f"  p = {p}")
print(f"  q = {q}")
print(f"  N = {N}")
print(f"  φ(N) = {phi_N}")
print(f"  e = {e}")
print(f"  d = {d}")

# Encrypt and decrypt
message = 12345
ciphertext = rsa_encrypt(message, pub)
decrypted = rsa_decrypt(ciphertext, priv)

print(f"\nEncryption/Decryption:")
print(f"  Original:  {message}")
print(f"  Encrypted: {ciphertext}")
print(f"  Decrypted: {decrypted}")
print(f"  Success:   {message == decrypted}")
```

## 4. Why RSA Works

The key insight is that decryption reverses encryption:

<Formula>
$$(M^e)^d = M^{ed} = M^{1 + k\phi(N)} = M \cdot (M^{\phi(N)})^k = M \cdot 1^k = M \pmod{N}$$
</Formula>

```python
def why_rsa_works():
    """Demonstrate the mathematical chain in RSA."""
    p, q = 61, 53
    N = p * q
    phi_N = (p - 1) * (q - 1)
    e = 17
    d = pow(e, -1, phi_N)
    
    print("Why RSA Decryption Works")
    print("=" * 50)
    print(f"p={p}, q={q}, N={N}, φ(N)={phi_N}")
    print(f"e={e}, d={d}")
    print(f"e × d = {e * d}")
    print(f"e × d mod φ(N) = {(e * d) % phi_N}")
    print()
    
    M = 42
    print(f"Message M = {M}")
    print()
    
    # Step by step
    C = pow(M, e, N)
    print(f"1. Encrypt: C = M^e mod N = {M}^{e} mod {N} = {C}")
    
    M_recovered = pow(C, d, N)
    print(f"2. Decrypt: M = C^d mod N = {C}^{d} mod {N} = {M_recovered}")
    
    # The math behind it
    ed = e * d
    k = ed // phi_N
    print(f"\n3. Mathematical explanation:")
    print(f"   e × d = {ed} = 1 + {k} × {phi_N}")
    print(f"   C^d = M^(ed) = M^(1 + k×φ(N))")
    print(f"       = M × (M^φ(N))^k")
    print(f"       = M × 1^k  (by Euler's theorem)")
    print(f"       = M = {M}")

why_rsa_works()
```

## 5. How Factoring Breaks RSA

If an attacker can factor N, they can compute the private key:

```python
def break_rsa_with_factors(public_key, p, q):
    """
    Demonstrate how factoring breaks RSA.
    Given N = p × q, recover the private key.
    """
    N, e = public_key
    
    print("Breaking RSA with Factorization")
    print("=" * 50)
    print(f"Public key: (N={N}, e={e})")
    print(f"\n🔓 Attacker factors N and finds:")
    print(f"   p = {p}")
    print(f"   q = {q}")
    
    # Compute φ(N) - this is the secret!
    phi_N = (p - 1) * (q - 1)
    print(f"\n📐 Attacker computes φ(N) = (p-1)(q-1) = {phi_N}")
    
    # Compute private key
    d = pow(e, -1, phi_N)
    print(f"\n🔑 Attacker computes d = e⁻¹ mod φ(N) = {d}")
    
    # Verify
    print(f"\n✓ Verification: e × d mod φ(N) = {(e * d) % phi_N}")
    
    return d

# Example
p, q = 61, 53
N = p * q
e = 17
public_key = (N, e)

# Attacker breaks RSA
d_recovered = break_rsa_with_factors(public_key, p, q)

# Attacker can now decrypt any message
message = 1234
ciphertext = pow(message, e, N)
decrypted = pow(ciphertext, d_recovered, N)

print(f"\n🎯 Attacker decrypts intercepted message:")
print(f"   Ciphertext: {ciphertext}")
print(f"   Decrypted:  {decrypted}")
print(f"   Original:   {message}")
```

## Summary

In this lesson, you learned:

- **RSA Components**: Public key (N, e), Private key (d), Secrets (p, q, φ(N))
- **Key Generation**: Choose primes p, q → N = pq → φ(N) = (p-1)(q-1) → d = e⁻¹ mod φ(N)
- **Encryption**: C = M^e mod N
- **Decryption**: M = C^d mod N
- **Breaking RSA**: Factor N → find p, q → compute φ(N) → compute d
- **Quantum Threat**: Shor's algorithm makes factoring easy → RSA is broken

## Key Formula

<Formula>
$$\text{RSA Security: } \text{Hard to find } d = e^{-1} \mod \phi(N) \text{ without knowing } p, q$$
</Formula>
