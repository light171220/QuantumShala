## 1. What Are Quantum Gates?

In classical computing, logic gates (AND, OR, NOT) transform bits. In quantum computing, **quantum gates** transform qubits through matrix multiplication.

<Definition term="Quantum Gate">
A quantum gate is a unitary operation that transforms quantum states. Mathematically, it's represented by a unitary matrix U where U†U = UU† = I.
</Definition>

<Definition term="Unitary Matrix">
A matrix U is unitary if its conjugate transpose equals its inverse: U† = U⁻¹. This ensures quantum operations are reversible and preserve probability (normalization).
</Definition>

<Formula>
$$|\psi_{out}\rangle = U|\psi_{in}\rangle$$
</Formula>

```python
import numpy as np

def is_unitary(matrix, tolerance=1e-10):
    """Check if a matrix is unitary: U†U = I"""
    n = matrix.shape[0]
    identity = np.eye(n)
    product = matrix.conj().T @ matrix
    return np.allclose(product, identity, atol=tolerance)

# Example: Check some matrices
X = np.array([[0, 1], [1, 0]])  # Pauli X
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)  # Hadamard
random = np.array([[1, 2], [3, 4]])  # Not unitary

print("Unitarity Check:")
print(f"  Pauli X: {is_unitary(X)}")
print(f"  Hadamard: {is_unitary(H)}")
print(f"  Random matrix: {is_unitary(random)}")

# Verify U†U = I
print(f"\nX†X = \n{X.conj().T @ X}")
print(f"\nH†H = \n{np.round(H.conj().T @ H, 10)}")
```

## 2. The Identity Gate

The simplest gate does nothing - it's the identity matrix.

<Formula>
$$I = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$$
$$I|\psi\rangle = |\psi\rangle$$
</Formula>

```python
import numpy as np

I = np.eye(2)
psi = np.array([0.6, 0.8])  # Some state

result = I @ psi
print(f"I|ψ⟩ = {result}")
print(f"Same as |ψ⟩: {np.allclose(result, psi)}")
```

The identity gate represents "do nothing" or "wait" in a quantum circuit.

## 3. The Pauli Gates: X, Y, Z

The **Pauli gates** are three fundamental single-qubit gates.

### Pauli X Gate (Quantum NOT)

<Definition term="Pauli X Gate">
The X gate flips |0⟩ to |1⟩ and vice versa. It's the quantum analog of the classical NOT gate.
</Definition>

<Formula>
$$X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$$
$$X|0\rangle = |1\rangle, \quad X|1\rangle = |0\rangle$$
</Formula>

### Pauli Y Gate

<Formula>
$$Y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}$$
$$Y|0\rangle = i|1\rangle, \quad Y|1\rangle = -i|0\rangle$$
</Formula>

### Pauli Z Gate (Phase Flip)

<Definition term="Pauli Z Gate">
The Z gate leaves |0⟩ unchanged but flips the sign of |1⟩. It's a "phase flip" gate.
</Definition>

<Formula>
$$Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$
$$Z|0\rangle = |0\rangle, \quad Z|1\rangle = -|1\rangle$$
</Formula>

```python
import numpy as np

# Define Pauli gates
X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])

# Basis states
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

print("Pauli Gate Actions:")
print("=" * 40)

print("\nPauli X (bit flip):")
print(f"  X|0⟩ = {X @ ket_0}")
print(f"  X|1⟩ = {X @ ket_1}")

print("\nPauli Y:")
print(f"  Y|0⟩ = {Y @ ket_0}")
print(f"  Y|1⟩ = {Y @ ket_1}")

print("\nPauli Z (phase flip):")
print(f"  Z|0⟩ = {Z @ ket_0}")
print(f"  Z|1⟩ = {Z @ ket_1}")

# Key property: X² = Y² = Z² = I
print("\nPauli gates are their own inverses:")
print(f"  X² = I: {np.allclose(X @ X, np.eye(2))}")
print(f"  Y² = I: {np.allclose(Y @ Y, np.eye(2))}")
print(f"  Z² = I: {np.allclose(Z @ Z, np.eye(2))}")
```

<Tip>
The Pauli gates correspond to 180° rotations around the X, Y, and Z axes of the Bloch sphere. X flips horizontally, Z flips the phase, and Y does both.
</Tip>

## 4. The Hadamard Gate

The **Hadamard gate** is arguably the most important single-qubit gate - it creates superposition.

<Definition term="Hadamard Gate">
The Hadamard gate transforms computational basis states into equal superpositions. It's the gateway to quantum parallelism.
</Definition>

<Formula>
$$H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$$
$$H|0\rangle = |+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)$$
$$H|1\rangle = |-\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle)$$
</Formula>

```python
import numpy as np

H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

print("Hadamard Gate:")
print("=" * 40)
print(f"H|0⟩ = {np.round(H @ ket_0, 4)} = |+⟩")
print(f"H|1⟩ = {np.round(H @ ket_1, 4)} = |-⟩")

# Key property: H² = I
print(f"\nH² = I: {np.allclose(H @ H, np.eye(2))}")

# Applying H twice returns to original
print(f"\nH(H|0⟩) = {np.round(H @ H @ ket_0, 4)} = |0⟩")

# Hadamard on superposition
plus = H @ ket_0
print(f"\nH|+⟩ = H(H|0⟩) = {np.round(H @ plus, 4)} = |0⟩")
```

<Warning>
The Hadamard gate is its own inverse: H² = I. Applying H twice returns the state to its original form. This is essential for many quantum algorithms!
</Warning>

## 5. Phase Gates: S and T

Phase gates modify the relative phase between |0⟩ and |1⟩.

### S Gate (√Z)

<Formula>
$$S = \begin{pmatrix} 1 & 0 \\ 0 & i \end{pmatrix}$$
$$S|0\rangle = |0\rangle, \quad S|1\rangle = i|1\rangle$$
</Formula>

### T Gate (π/8 gate)

<Formula>
$$T = \begin{pmatrix} 1 & 0 \\ 0 & e^{i\pi/4} \end{pmatrix}$$
$$T|0\rangle = |0\rangle, \quad T|1\rangle = e^{i\pi/4}|1\rangle$$
</Formula>

```python
import numpy as np

# Phase gates
S = np.array([[1, 0], [0, 1j]])
T = np.array([[1, 0], [0, np.exp(1j * np.pi / 4)]])

ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])
plus = np.array([1, 1]) / np.sqrt(2)

print("Phase Gates:")
print("=" * 40)

print("\nS Gate (90° phase):")
print(f"  S|0⟩ = {S @ ket_0}")
print(f"  S|1⟩ = {S @ ket_1}")
print(f"  S|+⟩ = {np.round(S @ plus, 3)}")

print("\nT Gate (45° phase):")
print(f"  T|0⟩ = {T @ ket_0}")
print(f"  T|1⟩ = {np.round(T @ ket_1, 3)}")

# Relationship: S = T², Z = S²
print("\nRelationships:")
print(f"  S = T²: {np.allclose(S, T @ T)}")
print(f"  Z = S²: {np.allclose(np.array([[1,0],[0,-1]]), S @ S)}")
```

## 6. Rotation Gates: Rx, Ry, Rz

General rotation gates rotate the qubit state around an axis by an arbitrary angle.

<Formula>
$$R_x(\theta) = \begin{pmatrix} \cos(\theta/2) & -i\sin(\theta/2) \\ -i\sin(\theta/2) & \cos(\theta/2) \end{pmatrix}$$

$$R_y(\theta) = \begin{pmatrix} \cos(\theta/2) & -\sin(\theta/2) \\ \sin(\theta/2) & \cos(\theta/2) \end{pmatrix}$$

$$R_z(\theta) = \begin{pmatrix} e^{-i\theta/2} & 0 \\ 0 & e^{i\theta/2} \end{pmatrix}$$
</Formula>

```python
import numpy as np

def Rx(theta):
    """Rotation around X axis"""
    c, s = np.cos(theta/2), np.sin(theta/2)
    return np.array([[c, -1j*s], [-1j*s, c]])

def Ry(theta):
    """Rotation around Y axis"""
    c, s = np.cos(theta/2), np.sin(theta/2)
    return np.array([[c, -s], [s, c]])

def Rz(theta):
    """Rotation around Z axis"""
    return np.array([[np.exp(-1j*theta/2), 0], [0, np.exp(1j*theta/2)]])

# Show that Pauli gates are π rotations
print("Pauli Gates as π Rotations:")
print("=" * 40)

X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])

# Up to global phase
print(f"Rx(π) ~ X: {np.allclose(np.abs(Rx(np.pi)), np.abs(X))}")
print(f"Ry(π) ~ Y: {np.allclose(np.abs(Ry(np.pi)), np.abs(Y))}")
print(f"Rz(π) ~ Z: {np.allclose(np.abs(Rz(np.pi)), np.abs(Z))}")

# Example: rotate |0⟩ around Y by π/2 to get |+⟩
ket_0 = np.array([1, 0])
result = Ry(np.pi/2) @ ket_0
print(f"\nRy(π/2)|0⟩ = {np.round(result, 3)}")
print(f"This is |+⟩ = (|0⟩+|1⟩)/√2: {np.allclose(np.abs(result), [1/np.sqrt(2), 1/np.sqrt(2)])}")
```

## 7. The Bloch Sphere

The **Bloch sphere** is a geometric representation of single-qubit states.

<Definition term="Bloch Sphere">
The Bloch sphere is a unit sphere where every point on or inside the sphere represents a valid qubit state. Pure states lie on the surface; mixed states lie inside.
</Definition>

```python
import numpy as np

def state_to_bloch(state):
    """
    Convert a qubit state to Bloch sphere coordinates.
    |ψ⟩ = cos(θ/2)|0⟩ + e^(iφ)sin(θ/2)|1⟩
    
    Returns: (x, y, z) on Bloch sphere
    """
    # Normalize
    state = np.array(state) / np.linalg.norm(state)
    
    # Extract amplitudes
    alpha, beta = state[0], state[1]
    
    # Calculate Bloch coordinates
    # x = 2*Re(α*β̄)
    # y = 2*Im(α*β̄)
    # z = |α|² - |β|²
    x = 2 * np.real(alpha * np.conj(beta))
    y = 2 * np.imag(alpha * np.conj(beta))
    z = np.abs(alpha)**2 - np.abs(beta)**2
    
    return x, y, z

# Test states
states = {
    '|0⟩': np.array([1, 0]),
    '|1⟩': np.array([0, 1]),
    '|+⟩': np.array([1, 1]) / np.sqrt(2),
    '|-⟩': np.array([1, -1]) / np.sqrt(2),
    '|+i⟩': np.array([1, 1j]) / np.sqrt(2),
    '|-i⟩': np.array([1, -1j]) / np.sqrt(2),
}

print("Bloch Sphere Coordinates:")
print("=" * 50)
print(f"{'State':<8} {'x':>8} {'y':>8} {'z':>8}")
print("-" * 50)

for name, state in states.items():
    x, y, z = state_to_bloch(state)
    print(f"{name:<8} {x:>8.3f} {y:>8.3f} {z:>8.3f}")

print("\n|0⟩ is at North pole (+z)")
print("|1⟩ is at South pole (-z)")
print("|+⟩, |-⟩ are on equator along x-axis")
print("|+i⟩, |-i⟩ are on equator along y-axis")
```

## 8. Gate Composition

Gates can be composed by matrix multiplication. The order matters!

<Formula>
$$|\psi_{final}\rangle = U_n \cdots U_2 U_1 |\psi_{initial}\rangle$$
</Formula>

**Note**: Gates are applied right-to-left in the matrix product.

```python
import numpy as np

# Gates
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
X = np.array([[0, 1], [1, 0]])
Z = np.array([[1, 0], [0, -1]])

ket_0 = np.array([1, 0])

# Apply H then X
result_HX = X @ H @ ket_0  # X applied after H
print("Apply H then X to |0⟩:")
print(f"  X(H|0⟩) = {np.round(result_HX, 3)}")

# Apply X then H
result_XH = H @ X @ ket_0  # H applied after X
print("\nApply X then H to |0⟩:")
print(f"  H(X|0⟩) = {np.round(result_XH, 3)}")

print("\nNote: XH ≠ HX (gates don't commute in general)")

# Some gates do commute
print(f"\nXZ = ZX? {np.allclose(X @ Z, Z @ X)}")  # No
print(f"HH = I? {np.allclose(H @ H, np.eye(2))}")  # Yes
```

<Warning>
Matrix multiplication is not commutative: AB ≠ BA in general. When reading a quantum circuit left-to-right, the matrices multiply right-to-left!
</Warning>

## 9. Universal Single-Qubit Gates

Any single-qubit unitary can be decomposed into rotations:

<Formula>
$$U = e^{i\alpha} R_z(\beta) R_y(\gamma) R_z(\delta)$$
</Formula>

This means {Rz, Ry} form a universal set for single-qubit operations.

```python
import numpy as np

def decompose_unitary(U):
    """
    Decompose arbitrary single-qubit unitary into Rz-Ry-Rz.
    U = e^(iα) Rz(β) Ry(γ) Rz(δ)
    """
    # This is a simplified version; full decomposition is more complex
    # Just demonstrate the concept
    
    det = np.linalg.det(U)
    alpha = np.angle(det) / 2
    
    # Remove global phase
    U_normalized = U * np.exp(-1j * alpha)
    
    return alpha, U_normalized

# Example: decompose Hadamard
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
alpha, H_norm = decompose_unitary(H)

print("Any single-qubit gate can be decomposed into rotations!")
print(f"Hadamard global phase: {alpha:.4f}")
print(f"H = e^(iα) Rz(β) Ry(π/2) Rz(γ) for appropriate angles")
```
