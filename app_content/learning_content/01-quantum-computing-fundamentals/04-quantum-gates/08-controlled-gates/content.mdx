## 1. Controlled-U Gates

A controlled-U gate applies unitary U to target when control is |1⟩:

<Formula>
$$CU = |0\rangle\langle 0| \otimes I + |1\rangle\langle 1| \otimes U$$
</Formula>

```python
import numpy as np

def controlled_U(U):
    """Create controlled-U gate (4x4 matrix)."""
    I = np.eye(2)
    proj_0 = np.array([[1, 0], [0, 0]])  # |0⟩⟨0|
    proj_1 = np.array([[0, 0], [0, 1]])  # |1⟩⟨1|
    return np.kron(proj_0, I) + np.kron(proj_1, U)

# Example: Controlled-Z
Z = np.array([[1, 0], [0, -1]])
CZ = controlled_U(Z)
print("CZ =")
print(CZ)
```

## 2. The Toffoli Gate (CCNOT)

The Toffoli gate has two controls and one target:

<Formula>
$$\text{CCNOT}|c_1, c_2, t\rangle = |c_1, c_2, t \oplus (c_1 \wedge c_2)\rangle$$
</Formula>

```python
import numpy as np

# Toffoli: flip target only if BOTH controls are |1⟩
Toffoli = np.eye(8)
Toffoli[6, 6] = 0
Toffoli[7, 7] = 0
Toffoli[6, 7] = 1
Toffoli[7, 6] = 1

print("Toffoli gate action:")
print("  |110⟩ → |111⟩")
print("  |111⟩ → |110⟩")
print("  All others unchanged")
```

## 3. Applications

Toffoli is universal for classical reversible computation and useful for:
- AND gate implementation
- Quantum arithmetic
- Quantum error correction ancilla preparation

## Summary

- CU applies U when control is |1⟩
- Toffoli = CCNOT, flips target when both controls are |1⟩
- Toffoli is universal for reversible classical computation
- Multi-controlled gates built from basic gates
