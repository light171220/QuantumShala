## 1. Introduction to Pauli Gates

The **Pauli gates** (X, Y, Z) along with the identity (I) form a complete basis for 2×2 Hermitian matrices. They are fundamental to quantum computing, quantum error correction, and quantum information theory.

<Definition term="Pauli Matrices">
The Pauli matrices are a set of three 2×2 complex Hermitian unitary matrices. Together with the identity matrix, they form a basis for the vector space of 2×2 Hermitian matrices.
</Definition>

<Formula>
$$I = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}, \quad
X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}, \quad
Y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}, \quad
Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$
</Formula>

```python
import numpy as np

# Define Pauli matrices
I = np.eye(2, dtype=complex)
X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

# Store in dictionary for convenience
pauli = {'I': I, 'X': X, 'Y': Y, 'Z': Z}

print("Pauli Matrices:")
for name, matrix in pauli.items():
    print(f"\n{name} =")
    print(matrix)
```

## 2. The Pauli X Gate: Bit Flip

The **Pauli X gate** is the quantum analog of the classical NOT gate. It flips the computational basis states.

<Definition term="Pauli X Gate">
The X gate, also called the NOT gate or bit-flip gate, exchanges |0⟩ and |1⟩. It performs a 180° rotation around the X-axis of the Bloch sphere.
</Definition>

<Formula>
$$X|0\rangle = |1\rangle, \quad X|1\rangle = |0\rangle$$
$$X = |0\rangle\langle 1| + |1\rangle\langle 0|$$
</Formula>

### Action on Basis States

```python
import numpy as np

X = np.array([[0, 1], [1, 0]])

# Basis states
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

print("X Gate Action on Basis States:")
print(f"X|0⟩ = {X @ ket_0} = |1⟩")
print(f"X|1⟩ = {X @ ket_1} = |0⟩")
```

### Action on Superposition States

```python
import numpy as np

X = np.array([[0, 1], [1, 0]])

# Superposition states
plus = np.array([1, 1]) / np.sqrt(2)   # |+⟩ = (|0⟩+|1⟩)/√2
minus = np.array([1, -1]) / np.sqrt(2)  # |-⟩ = (|0⟩-|1⟩)/√2

print("X Gate on Superposition States:")
print(f"X|+⟩ = {np.round(X @ plus, 4)} = |+⟩")
print(f"X|-⟩ = {np.round(X @ minus, 4)} = -|-⟩")

# General state
alpha, beta = 0.6, 0.8
psi = np.array([alpha, beta])
print(f"\nX({alpha}|0⟩ + {beta}|1⟩) = {X @ psi}")
print(f"This is {beta}|0⟩ + {alpha}|1⟩")
```

<Tip>
The X gate is an eigenstate operator for |+⟩ and |-⟩ states:
- X|+⟩ = +|+⟩ (eigenvalue +1)
- X|-⟩ = -|-⟩ (eigenvalue -1)

This makes |±⟩ the "X basis" or "Hadamard basis."
</Tip>

### Bloch Sphere Interpretation

The X gate performs a **180° rotation around the X-axis**:
- |0⟩ (north pole) → |1⟩ (south pole)
- |+⟩ (positive X) → |+⟩ (unchanged, on rotation axis)
- |+i⟩ (positive Y) → |-i⟩ (negative Y)

## 3. The Pauli Z Gate: Phase Flip

The **Pauli Z gate** leaves |0⟩ unchanged but introduces a phase flip to |1⟩.

<Definition term="Pauli Z Gate">
The Z gate, or phase-flip gate, multiplies the |1⟩ component by -1. It performs a 180° rotation around the Z-axis of the Bloch sphere.
</Definition>

<Formula>
$$Z|0\rangle = |0\rangle, \quad Z|1\rangle = -|1\rangle$$
$$Z = |0\rangle\langle 0| - |1\rangle\langle 1|$$
</Formula>

```python
import numpy as np

Z = np.array([[1, 0], [0, -1]])

# Basis states
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

print("Z Gate Action on Basis States:")
print(f"Z|0⟩ = {Z @ ket_0} = |0⟩")
print(f"Z|1⟩ = {Z @ ket_1} = -|1⟩")

# Superposition states
plus = np.array([1, 1]) / np.sqrt(2)
minus = np.array([1, -1]) / np.sqrt(2)

print("\nZ Gate on Superposition States:")
print(f"Z|+⟩ = {np.round(Z @ plus, 4)} = |-⟩")
print(f"Z|-⟩ = {np.round(Z @ minus, 4)} = |+⟩")
```

<Warning>
The Z gate has no classical analog! It changes the relative phase between basis states, which is a purely quantum phenomenon. This phase is crucial for interference effects in quantum algorithms.
</Warning>

### Eigenvalue Properties

```python
import numpy as np

Z = np.array([[1, 0], [0, -1]])

# Compute eigenvalues and eigenvectors
eigenvalues, eigenvectors = np.linalg.eig(Z)

print("Z Gate Eigendecomposition:")
print(f"Eigenvalues: {eigenvalues}")
print(f"\nEigenvector for λ=+1: {eigenvectors[:,0]} = |0⟩")
print(f"Eigenvector for λ=-1: {eigenvectors[:,1]} = |1⟩")

# Verify
print(f"\nZ|0⟩ = (+1)|0⟩: {np.allclose(Z @ eigenvectors[:,0], eigenvalues[0] * eigenvectors[:,0])}")
print(f"Z|1⟩ = (-1)|1⟩: {np.allclose(Z @ eigenvectors[:,1], eigenvalues[1] * eigenvectors[:,1])}")
```

## 4. The Pauli Y Gate: Combined Bit and Phase Flip

The **Pauli Y gate** combines the effects of X and Z gates (with an additional phase).

<Definition term="Pauli Y Gate">
The Y gate performs both a bit flip and a phase flip. It's equivalent to iXZ (up to global phase) and represents a 180° rotation around the Y-axis.
</Definition>

<Formula>
$$Y|0\rangle = i|1\rangle, \quad Y|1\rangle = -i|0\rangle$$
$$Y = i \cdot XZ = -i \cdot ZX$$
</Formula>

```python
import numpy as np

X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

# Basis states
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

print("Y Gate Action:")
print(f"Y|0⟩ = {Y @ ket_0} = i|1⟩")
print(f"Y|1⟩ = {Y @ ket_1} = -i|0⟩")

# Relationship with X and Z
print("\nRelationship: Y = iXZ")
print(f"iXZ = \n{1j * X @ Z}")
print(f"Y = \n{Y}")
print(f"Y == iXZ: {np.allclose(Y, 1j * X @ Z)}")
```

### Y Basis States

```python
import numpy as np

# Y eigenstates: |+i⟩ and |-i⟩
plus_i = np.array([1, 1j]) / np.sqrt(2)   # Y eigenvalue +1
minus_i = np.array([1, -1j]) / np.sqrt(2)  # Y eigenvalue -1

Y = np.array([[0, -1j], [1j, 0]])

print("Y Gate Eigenstates:")
print(f"|+i⟩ = (|0⟩ + i|1⟩)/√2 = {np.round(plus_i, 3)}")
print(f"|-i⟩ = (|0⟩ - i|1⟩)/√2 = {np.round(minus_i, 3)}")

print(f"\nY|+i⟩ = {np.round(Y @ plus_i, 3)} = +|+i⟩")
print(f"Y|-i⟩ = {np.round(Y @ minus_i, 3)} = -|-i⟩")
```

## 5. Fundamental Properties of Pauli Matrices

### Hermitian Property

All Pauli matrices are **Hermitian** (equal to their own conjugate transpose):

<Formula>
$$X^\dagger = X, \quad Y^\dagger = Y, \quad Z^\dagger = Z$$
</Formula>

```python
import numpy as np

X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

print("Hermitian Property (P† = P):")
print(f"X† = X: {np.allclose(X.conj().T, X)}")
print(f"Y† = Y: {np.allclose(Y.conj().T, Y)}")
print(f"Z† = Z: {np.allclose(Z.conj().T, Z)}")
```

### Unitary Property

All Pauli matrices are **unitary** (their inverse equals their conjugate transpose):

<Formula>
$$X^\dagger X = XX^\dagger = I$$
</Formula>

### Involutory Property

Each Pauli matrix is its own inverse (squaring gives identity):

<Formula>
$$X^2 = Y^2 = Z^2 = I$$
</Formula>

```python
import numpy as np

I = np.eye(2)
X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

print("Involutory Property (P² = I):")
print(f"X² = I: {np.allclose(X @ X, I)}")
print(f"Y² = I: {np.allclose(Y @ Y, I)}")
print(f"Z² = I: {np.allclose(Z @ Z, I)}")
```

<Tip>
Since P² = I, each Pauli gate is its own inverse! This means applying the same gate twice returns to the original state. This is incredibly useful for implementing "undo" operations.
</Tip>

### Trace Property

All Pauli matrices (except I) are **traceless**:

<Formula>
$$\text{Tr}(X) = \text{Tr}(Y) = \text{Tr}(Z) = 0, \quad \text{Tr}(I) = 2$$
</Formula>

```python
import numpy as np

I = np.eye(2)
X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])

print("Trace Property:")
print(f"Tr(I) = {np.trace(I)}")
print(f"Tr(X) = {np.trace(X)}")
print(f"Tr(Y) = {np.real(np.trace(Y))}")
print(f"Tr(Z) = {np.trace(Z)}")
```

### Determinant Property

<Formula>
$$\det(X) = \det(Y) = \det(Z) = -1, \quad \det(I) = 1$$
</Formula>

## 6. Pauli Algebra: Products and Commutators

### Multiplication Rules

<Formula>
$$XY = iZ, \quad YZ = iX, \quad ZX = iY$$
$$YX = -iZ, \quad ZY = -iX, \quad XZ = -iY$$
</Formula>

```python
import numpy as np

I = np.eye(2, dtype=complex)
X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

print("Pauli Multiplication Table:")
print("=" * 40)

products = [
    ('XY', X @ Y, '= iZ'),
    ('YZ', Y @ Z, '= iX'),
    ('ZX', Z @ X, '= iY'),
    ('YX', Y @ X, '= -iZ'),
    ('ZY', Z @ Y, '= -iX'),
    ('XZ', X @ Z, '= -iY'),
]

for name, result, expected in products:
    print(f"{name} {expected}")

# Verify
print("\nVerification:")
print(f"XY = iZ: {np.allclose(X @ Y, 1j * Z)}")
print(f"YZ = iX: {np.allclose(Y @ Z, 1j * X)}")
print(f"ZX = iY: {np.allclose(Z @ X, 1j * Y)}")
```

### Commutation Relations

The **commutator** [A, B] = AB - BA measures how much two operators fail to commute:

<Formula>
$$[X, Y] = 2iZ, \quad [Y, Z] = 2iX, \quad [Z, X] = 2iY$$
</Formula>

```python
import numpy as np

X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

def commutator(A, B):
    """Calculate [A, B] = AB - BA"""
    return A @ B - B @ A

print("Commutation Relations:")
print("=" * 40)

print(f"\n[X, Y] = XY - YX =")
comm_XY = commutator(X, Y)
print(comm_XY)
print(f"= 2iZ: {np.allclose(comm_XY, 2j * Z)}")

print(f"\n[Y, Z] = YZ - ZY =")
comm_YZ = commutator(Y, Z)
print(comm_YZ)
print(f"= 2iX: {np.allclose(comm_YZ, 2j * X)}")

print(f"\n[Z, X] = ZX - XZ =")
comm_ZX = commutator(Z, X)
print(comm_ZX)
print(f"= 2iY: {np.allclose(comm_ZX, 2j * Y)}")
```

### Anticommutation Relations

The **anticommutator** {A, B} = AB + BA:

<Formula>
$$\{X, Y\} = \{Y, Z\} = \{Z, X\} = 0$$
$$\{X, X\} = \{Y, Y\} = \{Z, Z\} = 2I$$
</Formula>

```python
import numpy as np

I = np.eye(2, dtype=complex)
X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

def anticommutator(A, B):
    """Calculate {A, B} = AB + BA"""
    return A @ B + B @ A

print("Anticommutation Relations:")
print("=" * 40)

print(f"\n{{X, Y}} = 0: {np.allclose(anticommutator(X, Y), 0)}")
print(f"{{Y, Z}} = 0: {np.allclose(anticommutator(Y, Z), 0)}")
print(f"{{Z, X}} = 0: {np.allclose(anticommutator(Z, X), 0)}")

print(f"\n{{X, X}} = 2I: {np.allclose(anticommutator(X, X), 2*I)}")
print(f"{{Y, Y}} = 2I: {np.allclose(anticommutator(Y, Y), 2*I)}")
print(f"{{Z, Z}} = 2I: {np.allclose(anticommutator(Z, Z), 2*I)}")
```

<Warning>
The non-commuting nature of Pauli operators ([X,Y] ≠ 0) is fundamental to quantum mechanics. It's related to the uncertainty principle - you cannot simultaneously know the precise values of non-commuting observables.
</Warning>

## 7. Pauli Decomposition

Any 2×2 matrix can be written as a linear combination of Pauli matrices:

<Formula>
$$M = \frac{1}{2}\left[\text{Tr}(M)I + \text{Tr}(XM)X + \text{Tr}(YM)Y + \text{Tr}(ZM)Z\right]$$
</Formula>

```python
import numpy as np

I = np.eye(2, dtype=complex)
X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

def pauli_decompose(M):
    """
    Decompose a 2x2 matrix into Pauli basis.
    M = c_I * I + c_X * X + c_Y * Y + c_Z * Z
    """
    c_I = np.trace(M) / 2
    c_X = np.trace(X @ M) / 2
    c_Y = np.trace(Y @ M) / 2
    c_Z = np.trace(Z @ M) / 2
    
    return {'I': c_I, 'X': c_X, 'Y': c_Y, 'Z': c_Z}

def reconstruct_from_pauli(coeffs):
    """Reconstruct matrix from Pauli coefficients."""
    return coeffs['I']*I + coeffs['X']*X + coeffs['Y']*Y + coeffs['Z']*Z

# Example: Hadamard gate
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

coeffs = pauli_decompose(H)
print("Hadamard Gate Pauli Decomposition:")
print(f"H = {coeffs['I']:.4f}·I + {coeffs['X']:.4f}·X + {coeffs['Y']:.4f}·Y + {coeffs['Z']:.4f}·Z")

# Verify reconstruction
H_reconstructed = reconstruct_from_pauli(coeffs)
print(f"\nReconstruction matches: {np.allclose(H, H_reconstructed)}")
```

## 8. Applications in Quantum Computing

### Quantum Error Correction

Pauli operators describe the three types of single-qubit errors:

- **Bit-flip error**: X gate applied erroneously
- **Phase-flip error**: Z gate applied erroneously  
- **Bit-phase flip error**: Y gate applied erroneously

```python
import numpy as np

X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

# Original state
psi = np.array([0.6, 0.8])

print("Error Types on |ψ⟩ = 0.6|0⟩ + 0.8|1⟩:")
print("=" * 50)

print(f"\nOriginal: {psi}")
print(f"Bit-flip (X error): {X @ psi}")
print(f"Phase-flip (Z error): {Z @ psi}")
print(f"Bit-phase flip (Y error): {Y @ psi}")
```

### Expectation Values

Measuring a qubit in different bases corresponds to measuring Pauli expectation values:

```python
import numpy as np

X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

def expectation(state, observable):
    """Calculate ⟨ψ|O|ψ⟩"""
    return np.real(state.conj() @ observable @ state)

# Example: |+⟩ state
plus = np.array([1, 1]) / np.sqrt(2)

print("Expectation Values for |+⟩ state:")
print(f"⟨X⟩ = {expectation(plus, X):.4f}")
print(f"⟨Y⟩ = {expectation(plus, Y):.4f}")
print(f"⟨Z⟩ = {expectation(plus, Z):.4f}")

print("\n|+⟩ points along +X axis of Bloch sphere")
print("So ⟨X⟩ = 1, ⟨Y⟩ = 0, ⟨Z⟩ = 0")
```

### Quantum Circuits

```python
import numpy as np

# Build a simple circuit: H → X → Z
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
X = np.array([[0, 1], [1, 0]])
Z = np.array([[1, 0], [0, -1]])

# Initial state |0⟩
ket_0 = np.array([1, 0])

# Apply gates (right to left in matrix product)
# Circuit: |0⟩ → H → X → Z → |ψ⟩
state_after_H = H @ ket_0
state_after_X = X @ state_after_H
state_final = Z @ state_after_X

print("Quantum Circuit: |0⟩ → H → X → Z")
print("=" * 40)
print(f"After H: {np.round(state_after_H, 4)}")
print(f"After X: {np.round(state_after_X, 4)}")
print(f"After Z: {np.round(state_final, 4)}")

# Combined unitary
U_total = Z @ X @ H
print(f"\nCombined unitary ZXH:")
print(np.round(U_total, 4))
```
