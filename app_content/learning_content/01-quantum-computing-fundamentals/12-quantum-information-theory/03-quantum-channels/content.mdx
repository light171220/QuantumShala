# Quantum Channels

## Introduction

Quantum channels are the mathematical framework describing any physical process on a quantum system. This includes unitary evolution, measurements, noise, and communication.

## What Is a Quantum Channel?

```
Quantum Channel Definition
══════════════════════════

A quantum channel ε: ρ → ε(ρ) must satisfy:

1. LINEARITY
   ε(aρ + bσ) = aε(ρ) + bε(σ)

2. COMPLETELY POSITIVE (CP)
   ε ⊗ I maps positive operators to positive operators
   for any ancilla system

3. TRACE PRESERVING (TP)
   Tr(ε(ρ)) = Tr(ρ) = 1


WHY "COMPLETELY" POSITIVE?

    If ρAB is entangled with ancilla B,
    applying ε only to A must give valid ρ'AB

    Just positivity isn't enough!

    ┌─────────────────────┐
    │    ε    │    I     │
    │   ───   │   ───    │
    │    A    │    B     │
    └─────────────────────┘
    Both outputs must be valid density matrices
```

## Kraus Representation

```python
import numpy as np

def apply_channel(rho, kraus_operators):
    """
    Apply quantum channel via Kraus operators.

    ε(ρ) = Σₖ Eₖ ρ Eₖ†

    where Σₖ Eₖ†Eₖ = I (completeness)
    """
    result = np.zeros_like(rho, dtype=complex)
    for E in kraus_operators:
        result += E @ rho @ E.conj().T
    return result

# Example: Depolarizing channel
def depolarizing_channel(p):
    """
    ε(ρ) = (1-p)ρ + (p/3)(XρX + YρY + ZρZ)

    With probability p, applies random Pauli.
    """
    I = np.eye(2)
    X = np.array([[0, 1], [1, 0]])
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]])

    # Kraus operators
    E0 = np.sqrt(1 - 3*p/4) * I
    E1 = np.sqrt(p/4) * X
    E2 = np.sqrt(p/4) * Y
    E3 = np.sqrt(p/4) * Z

    return [E0, E1, E2, E3]

print("Depolarizing Channel")
print("=" * 40)
print("|0⟩ after depolarizing (p=0.1):")
rho = np.array([[1, 0], [0, 0]])
kraus = depolarizing_channel(0.1)
result = apply_channel(rho, kraus)
print(result)
```

## Common Quantum Channels

```
Important Quantum Channels
══════════════════════════

1. DEPOLARIZING CHANNEL
   ε(ρ) = (1-p)ρ + pI/d
   "Completely random with probability p"

   Kraus: √(1-p)I, √(p/3)X, √(p/3)Y, √(p/3)Z

2. DEPHASING (PHASE DAMPING)
   ε(ρ) = (1-p)ρ + pZρZ
   "Loses phase coherence"

   Kraus: √(1-p)I, √p Z

   Effect: |0⟩⟨1| → (1-2p)|0⟩⟨1|

3. AMPLITUDE DAMPING
   "Energy relaxation to ground state"

   Kraus: E₀ = |0⟩⟨0| + √(1-γ)|1⟩⟨1|
          E₁ = √γ |0⟩⟨1|

   Models: T₁ decay in qubits

4. BIT FLIP
   ε(ρ) = (1-p)ρ + pXρX
   "Classical bit error"

5. PHASE FLIP
   ε(ρ) = (1-p)ρ + pZρZ
   "Sign error on |1⟩"
```

## Channel Representations

```
Three Equivalent Representations
════════════════════════════════

1. KRAUS OPERATORS
   ε(ρ) = Σₖ Eₖ ρ Eₖ†
   with Σₖ Eₖ†Eₖ = I

2. CHOI MATRIX
   Φε = (I ⊗ ε)(|Φ⁺⟩⟨Φ⁺|)
   where |Φ⁺⟩ = Σᵢ|i⟩|i⟩/√d

   Choi-Jamiołkowski isomorphism:
   Channel ↔ Bipartite state

3. STINESPRING DILATION
   ε(ρ) = TrE[U(ρ ⊗ |0⟩⟨0|)U†]

   Any channel = unitary on larger system
                 + trace out environment

   ┌───────────────────┐
   │     U             │
   │  ρ ──┼── ε(ρ)    │
   │  |0⟩ ─┼── (trace) │
   └───────────────────┘
```

## Degradable and Antidegradable Channels

```
Channel Classification
══════════════════════

DEGRADABLE CHANNEL:
    Environment gets less info than receiver

    ε(ρ) can be degraded to εE(ρ)
    by a channel D: εE = D ∘ ε

    Examples: Erasure channel, pure loss

    Property: Q = Ic (coherent information)

ANTIDEGRADABLE CHANNEL:
    Receiver gets less than environment

    ε = D ∘ εE for some D

    Examples: High-noise depolarizing

    Property: Q = 0 (no quantum capacity)


NEITHER:
    Many channels fall in between
    Capacity hard to compute

    Example: Amplitude damping for some γ
```

## Quantum vs Classical Channels

```
Comparison: Quantum and Classical
═════════════════════════════════

CLASSICAL CHANNEL:
    p(y|x): Conditional probabilities
    Transition matrix
    Always has diagonal Choi matrix

QUANTUM CHANNEL:
    ε: Density matrices → Density matrices
    Non-commutative, coherent effects
    Generally non-diagonal Choi matrix


CLASSICAL CAPACITY OF QUANTUM CHANNEL:
    C = max I(X;B)
        p(x),ρₓ

    Holevo quantity: χ = S(Σpₓρₓ) - ΣpₓS(ρₓ)

    C ≥ χ (Holevo bound)


SUPERADDITIVITY:
    C(ε ⊗ ε) can exceed 2C(ε)!
    Entangled inputs help
    (Hastings, 2009)
```

## Summary

Quantum channels are fundamental to quantum information:

1. **CPTP maps**: Complete positivity + trace preservation
2. **Kraus operators**: Practical representation ε(ρ) = ΣEₖρEₖ†
3. **Common channels**: Depolarizing, dephasing, amplitude damping
4. **Stinespring**: Every channel from unitary + partial trace
5. **Capacity**: Classical and quantum capacity differ

Channels model all quantum operations, from gates to noise to communication.
