# Advanced Qiskit Circuits

## Introduction

Beyond basic gates, Qiskit offers powerful tools for building complex circuits including parameterized gates, custom unitaries, and a rich circuit library.

## Parameterized Circuits

```python
from qiskit import QuantumCircuit
from qiskit.circuit import Parameter
import numpy as np

# Create parameters
theta = Parameter('θ')
phi = Parameter('φ')

# Build parameterized circuit
qc = QuantumCircuit(2)
qc.ry(theta, 0)
qc.rz(phi, 0)
qc.cx(0, 1)

print("Parameterized Circuit:")
print(qc.draw())

# Bind parameters
bound_circuit = qc.assign_parameters({
    theta: np.pi/4,
    phi: np.pi/2
})

print("\nBound Circuit:")
print(bound_circuit.draw())

# Multiple bindings
param_values = [
    {theta: 0, phi: 0},
    {theta: np.pi/2, phi: np.pi},
    {theta: np.pi, phi: np.pi/2}
]

circuits = [qc.assign_parameters(p) for p in param_values]
```

## Custom Gates

```python
from qiskit import QuantumCircuit
from qiskit.circuit.library import UnitaryGate
import numpy as np

# Define custom unitary matrix
sqrt_x = np.array([
    [0.5+0.5j, 0.5-0.5j],
    [0.5-0.5j, 0.5+0.5j]
])

# Create gate from unitary
sqrt_x_gate = UnitaryGate(sqrt_x, label='√X')

# Use in circuit
qc = QuantumCircuit(1)
qc.append(sqrt_x_gate, [0])
qc.append(sqrt_x_gate, [0])  # Two √X = X
print(qc.draw())

# Create gate from circuit
def create_toffoli_decomposition():
    """Decompose Toffoli into 1 and 2-qubit gates."""
    qc = QuantumCircuit(3, name='Toffoli')
    # Simplified decomposition
    qc.h(2)
    qc.cx(1, 2)
    qc.tdg(2)
    qc.cx(0, 2)
    qc.t(2)
    qc.cx(1, 2)
    qc.tdg(2)
    qc.cx(0, 2)
    qc.t(1)
    qc.t(2)
    qc.h(2)
    qc.cx(0, 1)
    qc.t(0)
    qc.tdg(1)
    qc.cx(0, 1)
    return qc.to_gate()

toffoli = create_toffoli_decomposition()
```

## Circuit Library

```python
from qiskit.circuit.library import (
    QFT,
    RealAmplitudes,
    EfficientSU2,
    TwoLocal,
    HGate, XGate, CXGate,
    MCXGate
)

# Quantum Fourier Transform
qft = QFT(num_qubits=4)
print("QFT Circuit:")
print(qft.decompose().draw())

# Variational ansatz
ansatz = RealAmplitudes(num_qubits=3, reps=2)
print("\nRealAmplitudes Ansatz:")
print(ansatz.draw())

# Hardware-efficient ansatz
efficient = EfficientSU2(num_qubits=4, reps=1)

# Two-local circuit
twolocal = TwoLocal(
    num_qubits=3,
    rotation_blocks=['ry', 'rz'],
    entanglement_blocks='cx',
    entanglement='linear',
    reps=2
)

print("\nCircuit Library Highlights:")
print("- QFT: Quantum Fourier Transform")
print("- RealAmplitudes: VQE/QAOA ansatz")
print("- EfficientSU2: Hardware-efficient")
print("- TwoLocal: Customizable variational")
```

## Circuit Composition

```python
from qiskit import QuantumCircuit

# Create sub-circuits
def bell_pair():
    qc = QuantumCircuit(2, name='Bell')
    qc.h(0)
    qc.cx(0, 1)
    return qc

def measure_circuit():
    qc = QuantumCircuit(2, name='Meas')
    qc.measure_all()
    return qc

# Compose circuits
bell = bell_pair()
meas = measure_circuit()

# Method 1: compose()
full = bell.compose(meas)

# Method 2: append as gate
main = QuantumCircuit(4)
main.append(bell.to_gate(), [0, 1])
main.append(bell.to_gate(), [2, 3])

# Method 3: tensor product
two_bells = bell.tensor(bell)

print("Composition Methods:")
print(f"  compose(): Sequential composition")
print(f"  append(): Add as sub-circuit")
print(f"  tensor(): Parallel composition")
```

## Summary

Advanced Qiskit features for complex circuits:

1. **Parameters**: Create variational circuits with Parameter
2. **Custom gates**: Build from unitaries or sub-circuits
3. **Library**: Pre-built QFT, ansatze, and more
4. **Composition**: Combine circuits sequentially or parallel
5. **Control flow**: Conditional operations on measurements

These tools enable sophisticated quantum algorithm implementation.
