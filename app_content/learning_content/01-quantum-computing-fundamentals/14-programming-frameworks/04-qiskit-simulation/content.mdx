# Qiskit Simulation

## Introduction

Qiskit provides powerful simulation tools to test quantum circuits before running on real hardware. Understanding these simulators is essential for debugging and algorithm development.

## Sampler Primitive

```python
from qiskit import QuantumCircuit
from qiskit.primitives import Sampler

# Create circuit
qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

# Run with Sampler
sampler = Sampler()
job = sampler.run(qc, shots=1000)
result = job.result()

# Get quasi-probability distribution
quasi_dist = result.quasi_dists[0]
print("Sampler Results:")
for outcome, prob in quasi_dist.items():
    print(f"  {bin(outcome)[2:].zfill(2)}: {prob:.3f}")
```

## Estimator Primitive

```python
from qiskit import QuantumCircuit
from qiskit.primitives import Estimator
from qiskit.quantum_info import SparsePauliOp

# Create circuit (no measurement)
qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)

# Define observable
observable = SparsePauliOp(['ZZ', 'XX'], coeffs=[1.0, 0.5])

# Estimate expectation value
estimator = Estimator()
job = estimator.run(qc, observable)
result = job.result()

print(f"Expectation value: {result.values[0]:.4f}")
print(f"Standard error: {result.metadata[0].get('variance', 'N/A')}")
```

## Statevector Simulation

```python
from qiskit import QuantumCircuit
from qiskit.quantum_info import Statevector

# Create circuit
qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)

# Get exact statevector
sv = Statevector(qc)

print("Statevector:")
print(sv.data)
print(f"\nProbabilities: {sv.probabilities()}")

# Visualize
# sv.draw('latex')  # In Jupyter
print(f"\nState: {sv.draw('text')}")
```

## Noise Simulation

```python
from qiskit_aer.noise import NoiseModel, depolarizing_error

# Create noise model
noise_model = NoiseModel()

# Add depolarizing error
error_1q = depolarizing_error(0.01, 1)  # 1% single-qubit
error_2q = depolarizing_error(0.02, 2)  # 2% two-qubit

noise_model.add_all_qubit_quantum_error(error_1q, ['h', 'x', 'ry'])
noise_model.add_all_qubit_quantum_error(error_2q, ['cx', 'cz'])

print("Noise Model:")
print(noise_model)
```

## Simulator Comparison

```
Simulation Methods
══════════════════

┌─────────────────────────────────────────────────────────────┐
│  Method          │ Max Qubits │ Speed    │ Use Case        │
├─────────────────────────────────────────────────────────────┤
│  Statevector     │ ~30        │ Medium   │ Exact analysis  │
│  Sampler         │ ~30        │ Fast     │ Measurement     │
│  Density matrix  │ ~15        │ Slow     │ Noise, mixed    │
│  MPS tensor      │ ~100       │ Medium   │ Low entanglement│
│  Clifford        │ ~1000s     │ Very fast│ Clifford only   │
└─────────────────────────────────────────────────────────────┘

Memory scaling:
    Statevector: O(2^n)
    Density matrix: O(4^n)
```

## Summary

Qiskit simulation capabilities:

1. **Sampler**: Measurement statistics from circuits
2. **Estimator**: Expectation values of observables
3. **Statevector**: Exact quantum state access
4. **Noise models**: Realistic error simulation
5. **Choose wisely**: Method depends on circuit and goals
