# Introduction to PennyLane

## Introduction

PennyLane is a cross-platform library for quantum machine learning, differentiable programming, and optimization. It seamlessly integrates quantum computing with popular ML frameworks.

## PennyLane Philosophy

```
PennyLane Core Concepts
═══════════════════════

1. DIFFERENTIABLE QUANTUM COMPUTING
   Gradients of quantum circuits
   Automatic differentiation
   Optimization built-in

2. HARDWARE AGNOSTIC
   Plugins for many backends
   Same code, different hardware

3. ML INTEGRATION
   Works with PyTorch, TensorFlow, JAX
   Hybrid quantum-classical models

4. QUANTUM NODES (QNodes)
   Quantum circuits as functions
   Inputs → quantum circuit → outputs
```

## Basic Circuit

```python
import pennylane as qml
import numpy as np

# Create device
dev = qml.device('default.qubit', wires=2)

# Define quantum function
@qml.qnode(dev)
def circuit(params):
    qml.RY(params[0], wires=0)
    qml.CNOT(wires=[0, 1])
    qml.RZ(params[1], wires=1)
    return qml.expval(qml.PauliZ(0) @ qml.PauliZ(1))

# Run circuit
params = np.array([0.5, 0.3])
result = circuit(params)
print(f"Expectation value: {result:.4f}")
```

## Automatic Differentiation

```python
import pennylane as qml
from pennylane import numpy as np

dev = qml.device('default.qubit', wires=1)

@qml.qnode(dev)
def circuit(x):
    qml.RX(x, wires=0)
    return qml.expval(qml.PauliZ(0))

# Compute gradient
x = np.array(0.5, requires_grad=True)
grad_fn = qml.grad(circuit)
gradient = grad_fn(x)

print(f"f(x) = cos(x)")
print(f"f(0.5) = {circuit(x):.4f}")
print(f"f'(0.5) = {gradient:.4f}")  # Should be -sin(0.5)
```

## Optimization

```python
import pennylane as qml
from pennylane import numpy as np

dev = qml.device('default.qubit', wires=2)

@qml.qnode(dev)
def cost_circuit(params):
    qml.RY(params[0], wires=0)
    qml.RY(params[1], wires=1)
    qml.CNOT(wires=[0, 1])
    return qml.expval(qml.PauliZ(0) @ qml.PauliZ(1))

# Optimize parameters
params = np.array([0.1, 0.2], requires_grad=True)
opt = qml.GradientDescentOptimizer(stepsize=0.4)

print("Optimization:")
for i in range(10):
    params = opt.step(cost_circuit, params)
    if i % 3 == 0:
        print(f"  Step {i}: cost = {cost_circuit(params):.4f}")
```

## Summary

PennyLane excels at quantum machine learning:

1. **QNodes**: Quantum circuits as differentiable functions
2. **Autodiff**: Built-in gradient computation
3. **Optimization**: Various quantum-aware optimizers
4. **ML integration**: PyTorch, TensorFlow, JAX compatible
5. **Hardware agnostic**: Many backend plugins available
