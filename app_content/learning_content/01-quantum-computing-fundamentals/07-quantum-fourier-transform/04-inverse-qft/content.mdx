## 1. Inverse QFT Definition

<Formula>
$$\text{QFT}^{-1}|k\rangle = \frac{1}{\sqrt{N}} \sum_{j=0}^{N-1} e^{-2\pi i jk/N} |j\rangle$$
</Formula>

```python
import numpy as np

def qft_inverse_matrix(n):
    """Create inverse QFT matrix."""
    N = 2**n
    omega = np.exp(-2j * np.pi / N)  # Note: negative sign
    F_inv = np.array([[omega**(j*k) for k in range(N)] for j in range(N)])
    return F_inv / np.sqrt(N)

QFT = qft_inverse_matrix(2)
print("Inverse QFT (2-qubit):")
print(np.round(QFT, 3))
```

## 2. QFT⁻¹ = QFT†

Since QFT is unitary, its inverse is its conjugate transpose:

```python
import numpy as np

def qft_matrix(n):
    N = 2**n
    omega = np.exp(2j * np.pi / N)
    return np.array([[omega**(j*k) for k in range(N)] for j in range(N)]) / np.sqrt(N)

QFT = qft_matrix(2)
QFT_inv = QFT.conj().T

print("QFT⁻¹ = QFT†:", np.allclose(QFT @ QFT_inv, np.eye(4)))
```

## 3. Inverse QFT Circuit

Run QFT circuit in reverse with conjugated phases:

```python
print("Inverse QFT circuit:")
print("1. Reverse the order of gates")
print("2. Replace R_k with R_k† (negate phases)")
print("3. Reverse bit order first (SWAPs)")
```

## Summary

- QFT⁻¹|k⟩ = (1/√N) Σⱼ e^(-2πijk/N)|j⟩
- QFT⁻¹ = QFT† (conjugate transpose)
- Circuit: reverse QFT with negated phases
- Used in QPE, Shor's algorithm
