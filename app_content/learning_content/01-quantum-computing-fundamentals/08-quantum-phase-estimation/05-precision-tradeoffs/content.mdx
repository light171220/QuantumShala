## Learning Objectives

## 1. Precision vs Qubits

With t counting qubits, QPE can distinguish phases differing by at least 1/2^t.

<Formula>
$$\Delta\phi_{min} = \frac{1}{2^t}$$
</Formula>

```python
import numpy as np

print("QPE Precision vs Counting Qubits:")
print(f"{'Qubits t':<10} {'Precision 1/2^t':<18} {'Decimal':<15}")
print("-" * 45)

for t in range(4, 17, 2):
    precision = 1 / (2**t)
    print(f"{t:<10} {f'1/{2**t}':<18} {precision:<15.10f}")
```

## 2. Success Probability Tradeoff

To achieve precision n bits with probability ≥ 1 - ε:

<Formula>
$$t = n + \lceil \log_2(2 + \frac{1}{2\varepsilon}) \rceil$$
</Formula>

```python
import numpy as np

def qubits_needed(precision_bits, epsilon):
    """Calculate qubits for given precision and confidence."""
    extra = int(np.ceil(np.log2(2 + 1/(2*epsilon))))
    return precision_bits + extra

print("Extra qubits needed for confidence:")
print(f"{'Confidence':<15} {'ε':<10} {'Extra qubits':<15}")
print("-" * 40)

for conf, eps in [(90, 0.1), (99, 0.01), (99.9, 0.001), (99.99, 0.0001)]:
    extra = int(np.ceil(np.log2(2 + 1/(2*eps/100))))
    print(f"{conf}%           {eps:<10} {extra:<15}")
```

## 3. Resource Estimation Example

For Shor's algorithm factoring an n-bit number:
- Need ~2n counting qubits for phase estimation
- Plus ~n qubits for modular arithmetic
- Total: ~3n qubits

```python
def shor_resources(n_bits):
    """Estimate resources for Shor's algorithm."""
    counting = 2 * n_bits
    arithmetic = n_bits
    ancilla = n_bits  # Approximate
    
    return {
        'counting': counting,
        'arithmetic': arithmetic,
        'ancilla': ancilla,
        'total': counting + arithmetic + ancilla
    }

print("Shor's Algorithm Resource Estimates:")
for n in [16, 64, 256, 1024, 2048]:
    r = shor_resources(n)
    print(f"  {n}-bit number: ~{r['total']} qubits")
```

## Summary

- **t qubits** → precision 1/2^t
- **Higher confidence** → logarithmically more qubits
- **Shor's**: ~3n qubits for n-bit factoring
- Trade-off is favorable: linear qubits for exponential precision
