## 1. Circuit Overview

The QPE circuit has two registers:

- **Counting register**: n qubits, initialized to |0⟩^⊗n
- **Eigenstate register**: holds |ψ⟩ (eigenvector of U)

```
Counting:   |0⟩ ─H─────●─────────────────────────┐
            |0⟩ ─H─────┼──────●──────────────────┤
            |0⟩ ─H─────┼──────┼───────●──────────┤ QFT⁻¹ ──▶ |φ̃⟩
            ...        │      │       │          │
            |0⟩ ─H─────┼──────┼───────┼────●─────┘
                       │      │       │    │
Eigenstate: |ψ⟩ ───────U────U²──────U⁴───U^(2^(n-1))───▶ |ψ⟩
```

## 2. Step-by-Step Algorithm

```python
import numpy as np

def qpe_simulation(U, psi, n_counting_qubits):
    """
    Simulate Quantum Phase Estimation.
    
    Args:
        U: Unitary operator (matrix)
        psi: Eigenvector of U
        n_counting_qubits: Number of precision qubits
    
    Returns:
        estimated_phase, probabilities
    """
    N = 2**n_counting_qubits
    
    # Step 1: Initialize counting register to uniform superposition
    # After H^⊗n, state is (1/√N) Σⱼ|j⟩
    
    # Step 2: Apply controlled-U^(2^k) operations
    # This encodes phase: |j⟩|ψ⟩ → e^(2πi·j·φ)|j⟩|ψ⟩
    
    # Find the eigenvalue phase
    eigenvalue = (U @ psi)[np.argmax(np.abs(psi))] / psi[np.argmax(np.abs(psi))]
    true_phase = np.angle(eigenvalue) / (2 * np.pi)
    if true_phase < 0:
        true_phase += 1
    
    # Simulate the state after controlled-U operations
    counting_state = np.zeros(N, dtype=complex)
    for j in range(N):
        counting_state[j] = np.exp(2j * np.pi * j * true_phase) / np.sqrt(N)
    
    # Step 3: Apply inverse QFT
    omega = np.exp(-2j * np.pi / N)
    qft_inv = np.array([[omega**(j*k) for k in range(N)] 
                        for j in range(N)]) / np.sqrt(N)
    
    final_state = qft_inv @ counting_state
    probabilities = np.abs(final_state)**2
    
    # Step 4: Measure to get phase estimate
    measured = np.argmax(probabilities)
    estimated_phase = measured / N
    
    return estimated_phase, probabilities, true_phase

# Example: Phase estimation for T gate
T = np.array([[1, 0], [0, np.exp(1j * np.pi / 4)]])
ket_1 = np.array([0, 1])

print("QPE for T gate on |1⟩")
print("=" * 40)
print("T|1⟩ = e^(iπ/4)|1⟩, so φ = 1/8 = 0.125")
print()

for n in [3, 4, 5, 6]:
    est, probs, true = qpe_simulation(T, ket_1, n)
    error = abs(est - true)
    print(f"n={n} qubits: estimated={est:.4f}, true={true:.4f}, error={error:.4f}")
```

## 3. Why It Works

The key insight is how phases accumulate:

<Formula>
$$\text{controlled-}U^{2^k}|1\rangle_k|\psi\rangle = |1\rangle_k U^{2^k}|\psi\rangle = e^{2\pi i \cdot 2^k \cdot \phi}|1\rangle_k|\psi\rangle$$
</Formula>

After all controlled operations on superposition:

<Formula>
$$\frac{1}{\sqrt{N}}\sum_{j=0}^{N-1} e^{2\pi i j \phi}|j\rangle|\psi\rangle$$
</Formula>

This is exactly the state that inverse QFT converts to |N·φ⟩!

```python
import numpy as np

def trace_qpe_evolution(phi, n):
    """
    Trace the state evolution through QPE.
    """
    N = 2**n
    
    print(f"QPE Evolution for φ = {phi}")
    print("=" * 50)
    
    # Initial state
    print(f"\n1. Initial: |0⟩^⊗{n} ⊗ |ψ⟩")
    
    # After Hadamards
    print(f"\n2. After H^⊗{n}: (1/√{N}) Σⱼ|j⟩ ⊗ |ψ⟩")
    
    # After controlled-U operations
    print(f"\n3. After controlled-U^(2^k):")
    print(f"   (1/√{N}) Σⱼ e^(2πi·j·φ)|j⟩ ⊗ |ψ⟩")
    print(f"   = (1/√{N}) Σⱼ ω^(j·{N}φ)|j⟩ ⊗ |ψ⟩")
    
    # After inverse QFT
    expected_outcome = round(N * phi) % N
    print(f"\n4. After QFT⁻¹:")
    print(f"   Peak at |{expected_outcome}⟩, which gives φ̃ = {expected_outcome}/{N} = {expected_outcome/N}")

trace_qpe_evolution(0.125, 4)  # φ = 1/8
```

## 4. Complete Implementation

```python
import numpy as np

def full_qpe(U, psi, n_qubits, return_all=False):
    """
    Full QPE implementation with detailed output.
    """
    N = 2**n_qubits
    d = len(psi)  # Dimension of eigenstate space
    
    # Get true phase from U|ψ⟩ = e^(2πiφ)|ψ⟩
    result = U @ psi
    for i in range(d):
        if np.abs(psi[i]) > 1e-10:
            eigenvalue = result[i] / psi[i]
            break
    true_phase = np.angle(eigenvalue) / (2 * np.pi)
    if true_phase < 0:
        true_phase += 1
    
    # Simulate counting register state after controlled-U operations
    # |ψ_counting⟩ = (1/√N) Σⱼ e^(2πi·j·φ)|j⟩
    counting = np.array([np.exp(2j * np.pi * j * true_phase) 
                         for j in range(N)]) / np.sqrt(N)
    
    # Apply inverse QFT
    omega = np.exp(-2j * np.pi / N)
    qft_inv = np.array([[omega**(j*k) for k in range(N)] 
                        for j in range(N)]) / np.sqrt(N)
    
    final = qft_inv @ counting
    probs = np.abs(final)**2
    
    # Measure
    estimated_phase = np.argmax(probs) / N
    
    if return_all:
        return {
            'estimated': estimated_phase,
            'true': true_phase,
            'error': abs(estimated_phase - true_phase),
            'probabilities': probs,
            'success_prob': np.max(probs)
        }
    return estimated_phase

# Comprehensive test
gates = [
    ("Z", np.array([[1,0],[0,-1]]), np.array([0,1]), 0.5),
    ("S", np.array([[1,0],[0,1j]]), np.array([0,1]), 0.25),
    ("T", np.array([[1,0],[0,np.exp(1j*np.pi/4)]]), np.array([0,1]), 0.125),
]

print("QPE Results Summary")
print("=" * 60)
for name, U, psi, expected in gates:
    result = full_qpe(U, psi, 5, return_all=True)
    print(f"\n{name} gate:")
    print(f"  Expected φ = {expected}")
    print(f"  Estimated φ = {result['estimated']}")
    print(f"  Error = {result['error']:.6f}")
    print(f"  Success probability = {result['success_prob']:.4f}")
```

## Summary

- QPE uses counting register + eigenstate register
- Controlled-U^(2^k) operations encode phase in superposition
- Inverse QFT extracts phase as computational basis state
- n qubits give n bits of precision (when phase is exact multiple of 1/2^n)

## Key Circuit Steps

1. **Prepare**: |0⟩^⊗n|ψ⟩
2. **Hadamard**: Apply H to all counting qubits
3. **Controlled-U**: Apply controlled-U^(2^k) for k = 0, 1, ..., n-1
4. **Inverse QFT**: Apply QFT⁻¹ to counting register
5. **Measure**: Read off phase estimate
