## 1. Shor's Algorithm

QPE is the quantum core of Shor's factoring algorithm:

1. Reduce factoring to period finding
2. Use QPE to find period r of f(x) = a^x mod N
3. Extract factors from r using classical post-processing

```python
import numpy as np
from math import gcd

def classical_shor_step(a, N, r):
    """
    Given period r of a^x mod N, attempt to find factors.
    """
    if r % 2 == 1:
        return None  # Need even period
    
    x = pow(a, r//2, N)
    if x == N - 1:
        return None  # Trivial case
    
    p = gcd(x - 1, N)
    q = gcd(x + 1, N)
    
    if p > 1 and p < N:
        return p, N // p
    if q > 1 and q < N:
        return q, N // q
    
    return None

# Example: factoring N=15
N = 15
a = 7
r = 4  # Period of 7^x mod 15 (found by QPE)

factors = classical_shor_step(a, N, r)
print(f"Factoring {N}: {factors}")
```

## 2. Quantum Chemistry

QPE finds ground state energies of molecular Hamiltonians:

- Prepare trial state |ψ⟩
- Apply QPE with U = e^(-iHt)
- Measure energy eigenvalue

```python
# Simplified: energy estimation
def energy_from_phase(phi, t):
    """Convert QPE phase to energy."""
    # U = e^(-iHt) has eigenvalue e^(-iEt)
    # Phase φ = Et/(2π)
    return 2 * np.pi * phi / t

# Example
phi_measured = 0.3  # QPE output
t = 1.0  # Evolution time
E = energy_from_phase(phi_measured, t)
print(f"Estimated energy: {E:.4f} Hartree")
```
