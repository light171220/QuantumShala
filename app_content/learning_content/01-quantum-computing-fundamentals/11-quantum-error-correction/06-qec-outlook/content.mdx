# QEC Outlook and Future

## Introduction

Quantum error correction is transitioning from theory to experiment. This lesson surveys where we are today and what lies ahead on the path to fault-tolerant quantum computing.

## Error Mitigation (Near-Term)

```
Error Mitigation vs Error Correction
════════════════════════════════════

ERROR MITIGATION:
    For NISQ era (no full QEC)
    Classical post-processing to reduce errors
    Works for expectation values

TECHNIQUES:

1. ZERO-NOISE EXTRAPOLATION
    - Run at multiple noise levels
    - Extrapolate to zero noise
    - No qubit overhead

2. PROBABILISTIC ERROR CANCELLATION
    - Characterize noise
    - Invert noise probabilistically
    - Sampling overhead

3. MEASUREMENT ERROR MITIGATION
    - Calibrate readout errors
    - Apply inverse matrix
    - Simple and effective

4. DYNAMICAL DECOUPLING
    - Insert identity sequences
    - Average out slow noise
    - Works during idle times


LIMITATIONS:
    ┌─────────────────────────────────────────────────┐
    │  Feature         │  Mitigation │  Correction  │
    ├─────────────────────────────────────────────────┤
    │  Qubit overhead  │  None       │  Large       │
    │  Scales with     │  Sampling   │  Polynomially│
    │  Circuit depth   │  Limited    │  Unlimited   │
    │  Full state      │  No         │  Yes         │
    │  Timeline        │  Now        │  Future      │
    └─────────────────────────────────────────────────┘
```

## The Road Ahead

```
Roadmap to Fault-Tolerant QC
════════════════════════════

PHASE 1: NOW (2024-2026)
    - Demonstrate logical advantage
    - Surface code d=7-11
    - ~1000-10000 qubits
    - Error mitigation + limited QEC

PHASE 2: NEAR-TERM (2027-2030)
    - Multiple logical qubits
    - Logical two-qubit gates
    - d=15-25 codes
    - ~100k qubits
    - First useful applications?

PHASE 3: FAULT-TOLERANT (2030+)
    - Millions of qubits
    - Full fault tolerance
    - Arbitrary computation
    - Quantum advantage at scale


REMAINING CHALLENGES:

    1. QUBIT COUNT
       Need 10⁶+ qubits
       Currently ~10³

    2. ERROR RATES
       Need consistent <0.1%
       Currently ~0.1-1%

    3. REAL-TIME DECODING
       Must keep up with QEC cycles
       ~1μs decision time

    4. WIRING/CONTROL
       Million-qubit control systems
       Cryogenic electronics

    5. MAGIC STATES
       For non-Clifford gates
       Significant overhead
```

## Summary

The outlook for quantum error correction:

1. **Current status**: Proof-of-concept demonstrated, below threshold achieved
2. **Surface code**: Leading candidate due to high threshold and 2D layout
3. **Error mitigation**: Near-term bridge while QEC matures
4. **Timeline**: Fault-tolerant QC likely 5-15+ years away
5. **Challenges**: Qubit count, error rates, and engineering scale-up

QEC research is accelerating, bringing fault-tolerant quantum computing closer to reality.
