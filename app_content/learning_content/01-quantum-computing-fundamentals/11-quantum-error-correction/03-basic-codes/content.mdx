# Basic Quantum Codes

## Introduction

Starting with simple codes helps build intuition for quantum error correction. We'll explore codes that protect against specific error types before combining them.

## 3-Qubit Bit Flip Code

```
The 3-Qubit Bit Flip Code
═════════════════════════

ENCODING:
    |0⟩ → |0_L⟩ = |000⟩
    |1⟩ → |1_L⟩ = |111⟩

    General: α|0⟩ + β|1⟩ → α|000⟩ + β|111⟩

ENCODING CIRCUIT:
    |ψ⟩ ────●────●────
            │    │
    |0⟩ ────X────┼────
                 │
    |0⟩ ─────────X────

    Uses two CNOT gates to spread information


SYNDROME TABLE:
┌───────────────────────────────────────────────────┐
│  Error    │  State        │  Z₁Z₂  │  Z₂Z₃  │     │
├───────────────────────────────────────────────────┤
│  None     │  α|000⟩+β|111⟩│   +1   │   +1   │     │
│  X on q1  │  α|100⟩+β|011⟩│   -1   │   +1   │     │
│  X on q2  │  α|010⟩+β|101⟩│   -1   │   -1   │     │
│  X on q3  │  α|001⟩+β|110⟩│   +1   │   -1   │     │
└───────────────────────────────────────────────────┘

    Z₁Z₂ means "measure if qubits 1,2 have same parity"


CORRECTION:
    Syndrome (0,0): No error
    Syndrome (1,0): Flip qubit 1
    Syndrome (1,1): Flip qubit 2
    Syndrome (0,1): Flip qubit 3


LIMITATIONS:
    ✓ Corrects single X errors
    ✗ Cannot correct Z errors
    ✗ Two X errors causes logical error
```

## 3-Qubit Phase Flip Code

```python
def phase_flip_code():
    """
    The 3-qubit phase flip code.
    """
    print("""
3-Qubit Phase Flip Code
═══════════════════════

IDEA: Work in Hadamard basis

ENCODING:
    |0⟩ → |0_L⟩ = |+++⟩
    |1⟩ → |1_L⟩ = |---⟩

    where |+⟩ = (|0⟩+|1⟩)/√2, |-⟩ = (|0⟩-|1⟩)/√2

ENCODING CIRCUIT:
    |ψ⟩ ────●────●────[H]────
            │    │
    |0⟩ ────X────┼────[H]────
                 │
    |0⟩ ─────────X────[H]────

    Same as bit flip, plus Hadamards


EFFECT OF Z ERROR:
    Z|+⟩ = |-⟩    Z|-⟩ = |+⟩

    Z error acts like "bit flip" in +/- basis!


SYNDROME MEASUREMENT:
    Measure X₁X₂ and X₂X₃ (in +/- basis)

    ┌───────────────────────────────────────────────┐
    │  Error    │  State        │  X₁X₂  │  X₂X₃  │
    ├───────────────────────────────────────────────┤
    │  None     │  α|+++⟩+β|---⟩│   +1   │   +1   │
    │  Z on q1  │  α|-++⟩+β|+--⟩│   -1   │   +1   │
    │  Z on q2  │  α|+-+⟩+β|-+-⟩│   -1   │   -1   │
    │  Z on q3  │  α|++-⟩+β|--+⟩│   +1   │   -1   │
    └───────────────────────────────────────────────┘


LIMITATIONS:
    ✓ Corrects single Z errors
    ✗ Cannot correct X errors
    ✗ Dual of bit flip code
    """)

phase_flip_code()
```

## Shor's 9-Qubit Code

```
Shor's 9-Qubit Code
═══════════════════

IDEA: Combine bit flip and phase flip protection

ENCODING (concatenation):

    First, phase flip encode:
    |0⟩ → |+++⟩
    |1⟩ → |---⟩

    Then, bit flip encode each qubit:
    |+⟩ → |+++⟩ (3 qubits)
    |-⟩ → |---⟩ (3 qubits)


FINAL ENCODING (9 qubits):
    |0_L⟩ = (|000⟩+|111⟩)(|000⟩+|111⟩)(|000⟩+|111⟩) / 2√2
    |1_L⟩ = (|000⟩-|111⟩)(|000⟩-|111⟩)(|000⟩-|111⟩) / 2√2


STRUCTURE:
    ┌─────────┐ ┌─────────┐ ┌─────────┐
    │Block 1  │ │Block 2  │ │Block 3  │
    │ q1 q2 q3│ │ q4 q5 q6│ │ q7 q8 q9│
    └─────────┘ └─────────┘ └─────────┘

    Within block: Bit flip protected
    Between blocks: Phase flip protected


SYNDROME MEASUREMENTS:

    Bit flip detection (within blocks):
    Z₁Z₂, Z₂Z₃, Z₄Z₅, Z₅Z₆, Z₇Z₈, Z₈Z₉

    Phase flip detection (between blocks):
    X₁X₂X₃X₄X₅X₆, X₄X₅X₆X₇X₈X₉


CAN CORRECT:
    ✓ Any single X error
    ✓ Any single Z error
    ✓ Any single Y error (Y = iXZ)
    = Any single-qubit error!


ENCODING CIRCUIT:
    Start: |ψ⟩ on qubit 1

    1. CNOT 1→4, CNOT 1→7  (spread)
    2. H on 1, 4, 7        (phase encode)
    3. CNOT 1→2, CNOT 1→3  (block 1 bit encode)
    4. CNOT 4→5, CNOT 4→6  (block 2 bit encode)
    5. CNOT 7→8, CNOT 7→9  (block 3 bit encode)
```

## Code Comparison

```
Comparing Basic Codes
═════════════════════

┌──────────────────────────────────────────────────────────────┐
│  Code           │ Qubits │ Corrects    │ Encoding Depth     │
├──────────────────────────────────────────────────────────────┤
│  Bit flip       │   3    │ 1 X error   │ 2 CNOTs            │
│  Phase flip     │   3    │ 1 Z error   │ 2 CNOTs + 3 H      │
│  Shor           │   9    │ Any 1 error │ 8 CNOTs + 3 H      │
└──────────────────────────────────────────────────────────────┘


CODE NOTATION:

    [[n, k, d]] code

    n = physical qubits
    k = logical qubits encoded
    d = code distance (detects d-1, corrects ⌊(d-1)/2⌋)

    Bit flip code: [[3, 1, 3]]
    Phase flip code: [[3, 1, 3]]
    Shor code: [[9, 1, 3]]


THE OVERHEAD:
    To protect 1 logical qubit from 1 error:
    - Minimum: 5 physical qubits ([[5,1,3]] code exists)
    - Shor uses 9 (not optimal, but conceptually clear)


NEXT STEPS:
    More efficient codes exist:
    - Steane code: [[7, 1, 3]]
    - Perfect code: [[5, 1, 3]]
    - Surface code: [[O(d²), 1, d]]
```

## Summary

Basic quantum codes demonstrate key QEC concepts:

1. **Bit flip code**: Protects against X errors using redundancy
2. **Phase flip code**: Protects against Z errors in Hadamard basis
3. **Shor code**: Concatenates both for complete protection
4. **Syndromes**: Reveal error location without disturbing encoded information

These codes pave the way for more sophisticated and efficient codes.
