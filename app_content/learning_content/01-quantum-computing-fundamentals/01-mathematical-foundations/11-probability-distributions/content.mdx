## 1. Introduction to Probability Distributions

A probability distribution describes how likely different outcomes are in a random experiment.

<Definition term="Probability Distribution">
A probability distribution assigns probabilities to all possible outcomes of a random variable. For discrete variables, this is a probability mass function (PMF). For continuous variables, it's a probability density function (PDF).
</Definition>

In quantum computing, distributions appear in:
- **Measurement outcomes**: The Born rule gives a discrete distribution
- **Shot noise**: Statistical uncertainty in finite measurements
- **Error models**: Describing gate and decoherence errors
- **Quantum sampling**: Loading classical distributions into quantum states

## 2. Bernoulli Distribution

The simplest distribution - a single yes/no outcome.

<Definition term="Bernoulli Distribution">
A Bernoulli random variable $X \sim \text{Bernoulli}(p)$ takes value 1 with probability $p$ and value 0 with probability $1-p$:
$$P(X=k) = p^k(1-p)^{1-k} \text{ for } k \in \{0, 1\}$$
</Definition>

Properties:
- $E[X] = p$
- $\text{Var}(X) = p(1-p)$

<Example>
A single qubit measurement is a Bernoulli trial:

$$|\psi\rangle = \sqrt{1-p}|0\rangle + \sqrt{p}|1\rangle$$

Measuring gives $X \sim \text{Bernoulli}(p)$ where $X=1$ means measuring $|1\rangle$.

```python
import numpy as np
from scipy import stats

# Qubit with P(1) = 0.3
p = 0.3
bernoulli = stats.bernoulli(p)

print(f"Bernoulli(p={p}):")
print(f"  P(X=0) = {bernoulli.pmf(0)}")
print(f"  P(X=1) = {bernoulli.pmf(1)}")
print(f"  E[X] = {bernoulli.mean()}")
print(f"  Var(X) = {bernoulli.var()}")

# Simulate 10 measurements
samples = bernoulli.rvs(size=10)
print(f"  10 samples: {samples}")
```
</Example>

## 3. Binomial Distribution

Multiple independent Bernoulli trials.

<Definition term="Binomial Distribution">
If we perform $n$ independent Bernoulli trials each with success probability $p$, the number of successes $X \sim \text{Binomial}(n, p)$:
$$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$$
</Definition>

Properties:
- $E[X] = np$
- $\text{Var}(X) = np(1-p)$
- $\sigma_X = \sqrt{np(1-p)}$

<Example>
Measuring a qubit 1000 times (1000 "shots"):

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# Parameters
p = 0.3  # P(measuring 1)
n = 1000  # number of shots

binomial = stats.binom(n, p)

print(f"Binomial(n={n}, p={p}):")
print(f"  E[X] = {binomial.mean()}")
print(f"  Std(X) = {binomial.std():.2f}")
print(f"  P(X=300) = {binomial.pmf(300):.4f}")
print(f"  P(290 ≤ X ≤ 310) = {binomial.cdf(310) - binomial.cdf(289):.4f}")

# 95% of measurements fall in this range
lower, upper = binomial.ppf([0.025, 0.975])
print(f"  95% interval: [{lower:.0f}, {upper:.0f}]")
```

This tells us: with 1000 shots, we expect about 300 ± 14.5 ones (one standard deviation).
</Example>

<Tip>
The binomial distribution is key to understanding "shot noise" in quantum computing. More shots = narrower distribution = more precise probability estimates.
</Tip>

## 4. Discrete Uniform Distribution

All outcomes equally likely.

<Definition term="Discrete Uniform Distribution">
$X \sim \text{Uniform}\{1, 2, \ldots, n\}$ has:
$$P(X=k) = \frac{1}{n} \text{ for } k = 1, 2, \ldots, n$$
</Definition>

<Example>
An n-qubit system in equal superposition:
$$|+\rangle^{\otimes n} = \frac{1}{\sqrt{2^n}} \sum_{x=0}^{2^n-1} |x\rangle$$

Measurement outcome is uniform over $\{0, 1, \ldots, 2^n-1\}$.

```python
import numpy as np

def uniform_quantum_state(n_qubits):
    """Create equal superposition state."""
    dim = 2**n_qubits
    state = np.ones(dim) / np.sqrt(dim)
    probs = np.abs(state)**2
    return probs

n = 3
probs = uniform_quantum_state(n)
print(f"{n}-qubit uniform superposition:")
for i, p in enumerate(probs):
    print(f"  P({format(i, f'0{n}b')}) = {p:.4f}")
```
</Example>

## 5. Gaussian (Normal) Distribution

The most important continuous distribution.

<Definition term="Gaussian Distribution">
$X \sim \mathcal{N}(\mu, \sigma^2)$ has probability density:
$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$
</Definition>

Properties:
- Mean: $E[X] = \mu$
- Variance: $\text{Var}(X) = \sigma^2$
- 68-95-99.7 rule: ~68% within 1σ, ~95% within 2σ, ~99.7% within 3σ

<Example>
By the Central Limit Theorem, for large $n$, the binomial distribution approximates a Gaussian:

$$\text{Binomial}(n, p) \approx \mathcal{N}(np, np(1-p))$$

```python
import numpy as np
from scipy import stats

# Compare binomial to Gaussian approximation
n, p = 1000, 0.3
mu = n * p
sigma = np.sqrt(n * p * (1-p))

binomial = stats.binom(n, p)
gaussian = stats.norm(mu, sigma)

# Compare CDFs
x = 310
print(f"P(X ≤ {x}):")
print(f"  Binomial: {binomial.cdf(x):.6f}")
print(f"  Gaussian: {gaussian.cdf(x):.6f}")

# Compare probability of specific range
print(f"\nP(295 ≤ X ≤ 305):")
print(f"  Binomial: {binomial.cdf(305) - binomial.cdf(294):.6f}")
print(f"  Gaussian: {gaussian.cdf(305) - gaussian.cdf(295):.6f}")
```
</Example>

## 6. Exponential Distribution

Models waiting times and decay processes.

<Definition term="Exponential Distribution">
$X \sim \text{Exp}(\lambda)$ has PDF:
$$f(x) = \lambda e^{-\lambda x} \text{ for } x \geq 0$$
</Definition>

Properties:
- $E[X] = 1/\lambda$
- $\text{Var}(X) = 1/\lambda^2$
- Memoryless property: $P(X > s+t | X > s) = P(X > t)$

<Example>
Qubit decoherence follows exponential decay:

$$|\langle\psi(t)|\psi(0)\rangle|^2 = e^{-t/T_2}$$

The "waiting time" until decoherence is exponentially distributed.

```python
import numpy as np
from scipy import stats

# T2 decoherence time (microseconds)
T2 = 100  # μs
rate = 1/T2

exponential = stats.expon(scale=T2)

print(f"Decoherence with T2 = {T2} μs:")
print(f"  Expected lifetime: {exponential.mean():.1f} μs")
print(f"  P(survive > 50 μs) = {1 - exponential.cdf(50):.4f}")
print(f"  P(survive > 100 μs) = {1 - exponential.cdf(100):.4f}")
print(f"  P(survive > 200 μs) = {1 - exponential.cdf(200):.4f}")
```
</Example>

## 7. Shot Noise and Statistical Uncertainty

When we measure quantum probabilities with finite shots, we get statistical uncertainty.

<Definition term="Shot Noise">
Shot noise is the statistical uncertainty in measured frequencies due to finite sample size. For $n$ measurements of a probability $p$:
$$\sigma_{\hat{p}} = \sqrt{\frac{p(1-p)}{n}}$$
</Definition>

<Example>
How many shots do we need to estimate a probability to 1% precision?

```python
import numpy as np

def shots_for_precision(target_error, p=0.5):
    """Calculate shots needed for given precision."""
    # Standard error: sqrt(p(1-p)/n) = target_error
    # n = p(1-p) / target_error^2
    return int(np.ceil(p * (1-p) / target_error**2))

print("Shots needed for different precisions:")
for error in [0.1, 0.05, 0.01, 0.001]:
    n = shots_for_precision(error)
    print(f"  ±{error*100:.1f}% precision: {n:,} shots")

# Verify with simulation
np.random.seed(42)
p_true = 0.3
for n_shots in [100, 1000, 10000]:
    estimates = [np.mean(np.random.binomial(1, p_true, n_shots)) 
                 for _ in range(1000)]
    std_estimate = np.std(estimates)
    theoretical = np.sqrt(p_true * (1-p_true) / n_shots)
    print(f"\n{n_shots} shots: std(p̂) = {std_estimate:.4f} (theory: {theoretical:.4f})")
```
</Example>

<Warning>
Shot noise scales as $1/\sqrt{n}$. To halve your error, you need 4x more shots. To get 10x better precision, you need 100x more shots!
</Warning>

## 8. Loading Distributions into Quantum States

Quantum algorithms can encode probability distributions in amplitude.

<Definition term="Amplitude Encoding">
A probability distribution $\{p_0, p_1, \ldots, p_{N-1}\}$ can be encoded in a quantum state:
$$|\psi\rangle = \sum_{i=0}^{N-1} \sqrt{p_i} |i\rangle$$
Measuring this state samples from the distribution.
</Definition>

<Example>
Encoding a simple distribution:

```python
import numpy as np

def encode_distribution(probabilities):
    """Create quantum state encoding a probability distribution."""
    # Amplitudes are square roots of probabilities
    amplitudes = np.sqrt(probabilities)
    
    # Normalize (should already be normalized if probs sum to 1)
    amplitudes = amplitudes / np.linalg.norm(amplitudes)
    
    return amplitudes

# Example: encode [0.1, 0.2, 0.3, 0.4]
probs = np.array([0.1, 0.2, 0.3, 0.4])
state = encode_distribution(probs)

print("Target distribution:", probs)
print("Quantum state amplitudes:", state)
print("Measurement probabilities:", np.abs(state)**2)

# Verify
print(f"Sum of probabilities: {np.sum(np.abs(state)**2):.6f}")
```
</Example>

## 9. Quantum Advantage in Sampling

Some distributions are hard to sample classically but easy quantumly.

<Tip>
Quantum sampling is a key area where quantum computers may achieve advantage. Examples include:
- **Boson sampling**: Sampling from permanents
- **IQP circuits**: Instantaneous quantum polynomial circuits
- **Random circuit sampling**: Used in quantum supremacy demonstrations
</Tip>

```python
import numpy as np

def classical_sampling_cost(n_qubits):
    """Estimate classical cost to sample from random quantum circuit."""
    # State vector has 2^n amplitudes
    dim = 2**n_qubits
    # Each amplitude is a complex number (16 bytes)
    memory_bytes = dim * 16
    return memory_bytes

print("Classical memory needed to simulate quantum sampling:")
for n in [10, 20, 30, 40, 50]:
    mem = classical_sampling_cost(n)
    if mem < 1e9:
        print(f"  {n} qubits: {mem/1e6:.1f} MB")
    elif mem < 1e12:
        print(f"  {n} qubits: {mem/1e9:.1f} GB")
    elif mem < 1e15:
        print(f"  {n} qubits: {mem/1e12:.1f} TB")
    else:
        print(f"  {n} qubits: {mem/1e15:.1f} PB")
```

## 10. Applications in Quantum Algorithms

Distributions appear throughout quantum computing:

### Grover's Algorithm
Success probability after $k$ iterations:
$$P(\text{success}) = \sin^2((2k+1)\theta)$$

### Quantum Phase Estimation  
The estimated phase has a peaked distribution around the true value.

### VQE and QAOA
Measurement outcomes follow distributions determined by the variational parameters.

```python
import numpy as np

def grover_success_probability(n_items, n_solutions, n_iterations):
    """Calculate Grover's algorithm success probability."""
    N = n_items
    M = n_solutions
    
    theta = np.arcsin(np.sqrt(M/N))
    prob = np.sin((2*n_iterations + 1) * theta)**2
    
    return prob

# Example: searching for 1 item among 1024
N = 1024
M = 1
optimal_iterations = int(np.round(np.pi/4 * np.sqrt(N/M)))

print(f"Grover search: {M} solution among {N} items")
print(f"Optimal iterations: {optimal_iterations}")

for k in range(0, optimal_iterations + 5):
    p = grover_success_probability(N, M, k)
    bar = "█" * int(p * 50)
    print(f"  k={k:2d}: P(success) = {p:.4f} {bar}")
```
