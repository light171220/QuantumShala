## 1. Operator Sandwich: ⟨φ|A|ψ⟩

The expression ⟨φ|A|ψ⟩ is ubiquitous in quantum mechanics.

<Definition term="Operator Sandwich">
For operator A and states |ψ⟩, |φ⟩, the expression ⟨φ|A|ψ⟩ is a complex number called the "matrix element" of A between states φ and ψ.
</Definition>

<Formula>
Read right to left: Apply A to |ψ⟩, then take inner product with |φ⟩:
$$\langle\phi|A|\psi\rangle = \langle\phi|(A|\psi\rangle) = (A|\psi\rangle)^\dagger|\phi\rangle$$
</Formula>

<Example>
Calculate ⟨0|X|+⟩ where X is the Pauli X gate:

$$\langle 0|X|+\rangle = \langle 0|X\left(\frac{|0\rangle + |1\rangle}{\sqrt{2}}\right) = \frac{1}{\sqrt{2}}\langle 0|(|1\rangle + |0\rangle) = \frac{1}{\sqrt{2}}(0 + 1) = \frac{1}{\sqrt{2}}$$

```python
import numpy as np

ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])
ket_plus = (ket_0 + ket_1) / np.sqrt(2)

X = np.array([[0, 1], [1, 0]])

# ⟨0|X|+⟩
result = ket_0.conj().T @ X @ ket_plus
print(f"⟨0|X|+⟩ = {result[0,0]:.4f}")
print(f"Expected: 1/√2 = {1/np.sqrt(2):.4f}")

# More examples
def sandwich(bra_state, operator, ket_state):
    """Compute ⟨bra|operator|ket⟩"""
    return (bra_state.conj().T @ operator @ ket_state)[0, 0]

Z = np.array([[1, 0], [0, -1]])
Y = np.array([[0, -1j], [1j, 0]])

print(f"\n⟨0|Z|0⟩ = {sandwich(ket_0, Z, ket_0)}")
print(f"⟨1|Z|1⟩ = {sandwich(ket_1, Z, ket_1)}")
print(f"⟨+|Z|+⟩ = {sandwich(ket_plus, Z, ket_plus)}")
```
</Example>

Special case - **expectation value** when φ = ψ:
$$\langle A \rangle_\psi = \langle\psi|A|\psi\rangle$$

## 2. Trace of an Operator

The trace is a fundamental operation in quantum mechanics.

<Definition term="Trace">
The trace of an operator A is the sum of its diagonal elements, which equals the sum of expectation values over any orthonormal basis:
$$\text{Tr}(A) = \sum_i A_{ii} = \sum_i \langle e_i|A|e_i\rangle$$
</Definition>

<Example>
```python
import numpy as np

def trace_via_basis(A, basis):
    """Calculate trace using Tr(A) = Σᵢ ⟨eᵢ|A|eᵢ⟩"""
    tr = 0
    for basis_state in basis:
        tr += (basis_state.conj().T @ A @ basis_state)[0, 0]
    return tr

# Pauli matrices
X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]], dtype=complex)
I = np.eye(2, dtype=complex)

# Computational basis
comp_basis = [np.array([[1], [0]]), np.array([[0], [1]])]

# Hadamard basis
had_basis = [np.array([[1], [1]])/np.sqrt(2), 
             np.array([[1], [-1]])/np.sqrt(2)]

print("Traces of Pauli matrices:")
for name, M in [('I', I), ('X', X), ('Y', Y), ('Z', Z)]:
    tr_numpy = np.trace(M)
    tr_comp = trace_via_basis(M, comp_basis)
    tr_had = trace_via_basis(M, had_basis)
    print(f"  Tr({name}) = {tr_numpy} (comp: {tr_comp:.1f}, had: {tr_had:.1f})")
```
</Example>

**Key trace properties:**
- $\text{Tr}(A + B) = \text{Tr}(A) + \text{Tr}(B)$
- $\text{Tr}(cA) = c \cdot \text{Tr}(A)$
- $\text{Tr}(AB) = \text{Tr}(BA)$ (cyclic property)
- $\text{Tr}(|\psi\rangle\langle\phi|) = \langle\phi|\psi\rangle$

<Tip>
The cyclic property is extremely useful: Tr(ABC) = Tr(BCA) = Tr(CAB). Note this doesn't mean ABC = BCA!
</Tip>

## 3. Trace of Outer Products

A particularly useful identity:

<Formula>
$$\text{Tr}(|\psi\rangle\langle\phi|) = \langle\phi|\psi\rangle$$
</Formula>

<Example>
```python
import numpy as np

ket_0 = np.array([[1], [0]])
ket_plus = np.array([[1], [1]]) / np.sqrt(2)

# Outer product |0⟩⟨+|
outer = ket_0 @ ket_plus.conj().T
print("|0⟩⟨+| =")
print(outer)

# Trace of outer product
tr_outer = np.trace(outer)
print(f"\nTr(|0⟩⟨+|) = {tr_outer:.4f}")

# Compare to inner product ⟨+|0⟩
inner = (ket_plus.conj().T @ ket_0)[0, 0]
print(f"⟨+|0⟩ = {inner:.4f}")

print(f"\nThey match: {np.isclose(tr_outer, inner)}")
```
</Example>

## 4. Commutators and Anticommutators

Commutators measure how much operators fail to commute.

<Definition term="Commutator">
The commutator of operators A and B is:
$$[A, B] = AB - BA$$
If [A, B] = 0, we say A and B commute.
</Definition>

<Definition term="Anticommutator">
The anticommutator is:
$$\{A, B\} = AB + BA$$
</Definition>

<Example>
The Pauli matrices have famous commutation relations:

$$[X, Y] = 2iZ, \quad [Y, Z] = 2iX, \quad [Z, X] = 2iY$$

```python
import numpy as np

X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]], dtype=complex)

def commutator(A, B):
    return A @ B - B @ A

def anticommutator(A, B):
    return A @ B + B @ A

print("Commutators [A, B]:")
print(f"[X, Y] =\n{commutator(X, Y)}")
print(f"2iZ =\n{2j * Z}")
print(f"Match: {np.allclose(commutator(X, Y), 2j*Z)}")

print(f"\n[Y, Z] =\n{commutator(Y, Z)}")
print(f"2iX =\n{2j * X}")
print(f"Match: {np.allclose(commutator(Y, Z), 2j*X)}")

print("\nAnticommutators {A, B}:")
print(f"{{X, Y}} =\n{anticommutator(X, Y)}")
print("(should be zero)")
```
</Example>

**Physical significance:**
- **Commuting observables** can be measured simultaneously with certainty
- **Non-commuting observables** satisfy uncertainty relations: $\Delta A \cdot \Delta B \geq \frac{1}{2}|\langle[A,B]\rangle|$

## 5. Resolution of Identity

The completeness relation is a powerful computational tool.

<Definition term="Resolution of Identity">
For an orthonormal basis $\{|e_i\rangle\}$:
$$I = \sum_i |e_i\rangle\langle e_i|$$
This can be inserted anywhere in calculations without changing the result.
</Definition>

<Example>
Inserting identity to compute ⟨φ|A|ψ⟩ in a specific basis:

$$\langle\phi|A|\psi\rangle = \langle\phi|IAI|\psi\rangle = \sum_{i,j} \langle\phi|e_i\rangle\langle e_i|A|e_j\rangle\langle e_j|\psi\rangle$$

```python
import numpy as np

def sandwich_via_basis(phi, A, psi, basis):
    """Compute ⟨φ|A|ψ⟩ by inserting resolutions of identity."""
    result = 0
    for ei in basis:
        for ej in basis:
            # ⟨φ|eᵢ⟩
            phi_ei = (phi.conj().T @ ei)[0, 0]
            # ⟨eᵢ|A|eⱼ⟩
            ei_A_ej = (ei.conj().T @ A @ ej)[0, 0]
            # ⟨eⱼ|ψ⟩
            ej_psi = (ej.conj().T @ psi)[0, 0]
            
            result += phi_ei * ei_A_ej * ej_psi
    return result

# Test
ket_plus = np.array([[1], [1]]) / np.sqrt(2)
ket_minus = np.array([[1], [-1]]) / np.sqrt(2)
X = np.array([[0, 1], [1, 0]], dtype=complex)

comp_basis = [np.array([[1], [0]]), np.array([[0], [1]])]

# Direct calculation
direct = (ket_plus.conj().T @ X @ ket_minus)[0, 0]
print(f"⟨+|X|−⟩ direct: {direct:.4f}")

# Via resolution of identity
via_basis = sandwich_via_basis(ket_plus, X, ket_minus, comp_basis)
print(f"⟨+|X|−⟩ via basis: {via_basis:.4f}")
```
</Example>

## 6. Tensor Products in Dirac Notation

Multi-qubit systems use tensor product notation.

<Formula>
$$|a\rangle \otimes |b\rangle = |a\rangle|b\rangle = |ab\rangle$$
$$\langle a| \otimes \langle b| = \langle a|\langle b| = \langle ab|$$
</Formula>

**Key rules:**
- $(|a\rangle \otimes |b\rangle)(|c\rangle \otimes |d\rangle) = |a\rangle|c\rangle \otimes |b\rangle|d\rangle$
- $\langle ab|cd\rangle = \langle a|c\rangle \cdot \langle b|d\rangle$
- $(A \otimes B)(|a\rangle \otimes |b\rangle) = (A|a\rangle) \otimes (B|b\rangle)$

<Example>
```python
import numpy as np

ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])

# |00⟩ = |0⟩ ⊗ |0⟩
ket_00 = np.kron(ket_0, ket_0)
ket_01 = np.kron(ket_0, ket_1)
ket_10 = np.kron(ket_1, ket_0)
ket_11 = np.kron(ket_1, ket_1)

print("Two-qubit basis states:")
print(f"|00⟩ = {ket_00.flatten()}")
print(f"|01⟩ = {ket_01.flatten()}")
print(f"|10⟩ = {ket_10.flatten()}")
print(f"|11⟩ = {ket_11.flatten()}")

# Inner products
print("\nInner products ⟨ab|cd⟩:")
print(f"⟨00|00⟩ = {(ket_00.conj().T @ ket_00)[0,0]}")
print(f"⟨00|01⟩ = {(ket_00.conj().T @ ket_01)[0,0]}")
print(f"⟨01|10⟩ = {(ket_01.conj().T @ ket_10)[0,0]}")

# Operator tensor products
X = np.array([[0, 1], [1, 0]])
I = np.eye(2)

# X ⊗ I (X on first qubit)
X_I = np.kron(X, I)
print("\nX ⊗ I =")
print(X_I)

# Apply to |00⟩
result = X_I @ ket_00
print(f"\n(X⊗I)|00⟩ = {result.flatten()} = |10⟩")
```
</Example>

## 7. Partial Trace

The partial trace "traces out" one subsystem.

<Definition term="Partial Trace">
For a bipartite operator $\rho_{AB}$, the partial trace over B is:
$$\rho_A = \text{Tr}_B(\rho_{AB}) = \sum_i (I_A \otimes \langle e_i|_B)\rho_{AB}(I_A \otimes |e_i\rangle_B)$$
</Definition>

<Example>
```python
import numpy as np

def partial_trace_B(rho_AB, dim_A, dim_B):
    """
    Trace out system B from rho_AB.
    Returns reduced density matrix rho_A.
    """
    rho_AB = rho_AB.reshape(dim_A, dim_B, dim_A, dim_B)
    rho_A = np.trace(rho_AB, axis1=1, axis2=3)
    return rho_A

# Bell state |Φ+⟩ = (|00⟩ + |11⟩)/√2
ket_00 = np.array([[1], [0], [0], [0]])
ket_11 = np.array([[0], [0], [0], [1]])
bell = (ket_00 + ket_11) / np.sqrt(2)

# Density matrix of Bell state
rho_AB = bell @ bell.conj().T
print("Bell state density matrix ρ_AB:")
print(rho_AB)

# Partial trace over B
rho_A = partial_trace_B(rho_AB, 2, 2)
print("\nReduced density matrix ρ_A = Tr_B(ρ_AB):")
print(rho_A)
print("\nThis is the maximally mixed state I/2!")
```
</Example>

## 8. Useful Identities

Here are essential identities for quantum calculations:

| Identity | Formula |
|----------|---------|
| Trace of outer product | $\text{Tr}(\|ψ⟩⟨φ\|) = ⟨φ\|ψ⟩$ |
| Cyclic trace | $\text{Tr}(ABC) = \text{Tr}(CAB)$ |
| Completeness | $\sum_i \|e_i⟩⟨e_i\| = I$ |
| Product of outer products | $\|a⟩⟨b\| \cdot \|c⟩⟨d\| = ⟨b\|c⟩\|a⟩⟨d\|$ |
| Tensor inner product | $⟨ab\|cd⟩ = ⟨a\|c⟩⟨b\|d⟩$ |

```python
import numpy as np

# Verify: |a⟩⟨b| · |c⟩⟨d| = ⟨b|c⟩|a⟩⟨d|
a = np.array([[1], [0]])
b = np.array([[0], [1]])
c = np.array([[1], [1]]) / np.sqrt(2)
d = np.array([[1], [-1]]) / np.sqrt(2)

# Left side
left = (a @ b.conj().T) @ (c @ d.conj().T)

# Right side
bc = (b.conj().T @ c)[0, 0]  # ⟨b|c⟩
right = bc * (a @ d.conj().T)

print("Verify |a⟩⟨b| · |c⟩⟨d| = ⟨b|c⟩|a⟩⟨d|")
print(f"Left side:\n{left}")
print(f"Right side:\n{right}")
print(f"Match: {np.allclose(left, right)}")
```
