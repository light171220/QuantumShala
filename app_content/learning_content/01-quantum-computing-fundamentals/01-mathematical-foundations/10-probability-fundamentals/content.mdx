## 1. Why Probability in Quantum Computing?

Quantum mechanics is fundamentally probabilistic. Unlike classical physics where outcomes are deterministic, quantum systems give probabilistic measurement results.

<Definition term="Quantum Measurement">
When we measure a quantum system, we get a random outcome according to probabilities determined by the quantum state. The Born rule tells us: for state $|\psi\rangle = \sum_i \alpha_i |i\rangle$, the probability of measuring outcome $i$ is $P(i) = |\alpha_i|^2$.
</Definition>

Understanding probability is essential for:
- Predicting measurement outcomes
- Designing quantum algorithms
- Analyzing quantum error rates
- Understanding quantum advantage

## 2. Sample Spaces and Events

<Definition term="Sample Space">
The sample space $\Omega$ is the set of all possible outcomes of an experiment. Each element $\omega \in \Omega$ is called an outcome or sample point.
</Definition>

<Example>
For a single qubit measurement in the computational basis:
- Sample space: $\Omega = \{0, 1\}$
- Outcomes: measuring $|0\rangle$ or $|1\rangle$

For two qubits:
- Sample space: $\Omega = \{00, 01, 10, 11\}$

For n qubits:
- Sample space: $\Omega = \{0, 1\}^n$ with $|\Omega| = 2^n$ outcomes
</Example>

<Definition term="Event">
An event $E$ is a subset of the sample space: $E \subseteq \Omega$. Events can contain multiple outcomes.
</Definition>

```python
# Sample spaces for qubit systems
def qubit_sample_space(n_qubits):
    """Generate sample space for n-qubit measurement."""
    outcomes = []
    for i in range(2**n_qubits):
        binary = format(i, f'0{n_qubits}b')
        outcomes.append(binary)
    return outcomes

print("1 qubit:", qubit_sample_space(1))
print("2 qubits:", qubit_sample_space(2))
print("3 qubits:", qubit_sample_space(3))
```

## 3. Probability Axioms (Kolmogorov Axioms)

Probability is a function $P: \mathcal{F} \to [0,1]$ satisfying:

<Definition term="Kolmogorov Axioms">
1. **Non-negativity**: $P(E) \geq 0$ for all events $E$
2. **Normalization**: $P(\Omega) = 1$
3. **Countable additivity**: For mutually exclusive events $E_1, E_2, \ldots$:
   $$P\left(\bigcup_{i=1}^{\infty} E_i\right) = \sum_{i=1}^{\infty} P(E_i)$$
</Definition>

Consequences:
- $P(\emptyset) = 0$ (impossible event)
- $P(E^c) = 1 - P(E)$ (complement rule)
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ (inclusion-exclusion)

<Example>
For a qubit in state $|\psi\rangle = \frac{1}{\sqrt{2}}|0\rangle + \frac{1}{\sqrt{2}}|1\rangle$:

- $P(0) = |1/\sqrt{2}|^2 = 1/2$
- $P(1) = |1/\sqrt{2}|^2 = 1/2$
- $P(0) + P(1) = 1$ ✓ (normalization)
- $P(\{0,1\}) = P(\Omega) = 1$ ✓

```python
import numpy as np

# Define |+⟩ state
alpha = 1/np.sqrt(2)
beta = 1/np.sqrt(2)

P_0 = np.abs(alpha)**2
P_1 = np.abs(beta)**2

print(f"P(0) = {P_0}")
print(f"P(1) = {P_1}")
print(f"P(0) + P(1) = {P_0 + P_1}")  # Should be 1
```
</Example>

## 4. Conditional Probability

<Definition term="Conditional Probability">
The probability of event $A$ given that event $B$ has occurred:
$$P(A|B) = \frac{P(A \cap B)}{P(B)} \quad \text{(assuming } P(B) > 0\text{)}$$
</Definition>

In quantum computing, conditional probability appears when:
- Post-selecting on measurement outcomes
- Analyzing multi-qubit correlations
- Understanding entanglement

<Example>
Consider two qubits in state $|\psi\rangle = \frac{1}{2}(|00\rangle + |01\rangle + |10\rangle + |11\rangle)$.

If we measure the first qubit and get $0$, what's the probability the second qubit is $1$?

- $P(\text{first} = 0) = P(00) + P(01) = 1/4 + 1/4 = 1/2$
- $P(\text{first} = 0 \text{ AND second} = 1) = P(01) = 1/4$
- $P(\text{second} = 1 | \text{first} = 0) = \frac{1/4}{1/2} = \frac{1}{2}$

```python
# Two-qubit equal superposition
probs = {'00': 0.25, '01': 0.25, '10': 0.25, '11': 0.25}

# P(first = 0)
P_first_0 = probs['00'] + probs['01']
print(f"P(first = 0) = {P_first_0}")

# P(first = 0 AND second = 1)
P_00_and_second_1 = probs['01']
print(f"P(first=0, second=1) = {P_00_and_second_1}")

# Conditional probability
P_second_1_given_first_0 = P_00_and_second_1 / P_first_0
print(f"P(second=1 | first=0) = {P_second_1_given_first_0}")
```
</Example>

## 5. Independence

<Definition term="Independent Events">
Events $A$ and $B$ are independent if:
$$P(A \cap B) = P(A) \cdot P(B)$$
Equivalently: $P(A|B) = P(A)$ and $P(B|A) = P(B)$.
</Definition>

<Tip>
For separable quantum states, measurement outcomes on different qubits are independent. For entangled states, they are correlated (not independent)!
</Tip>

<Example>
**Separable state** $|+\rangle \otimes |+\rangle$: qubits are independent
- $P(00) = P(\text{first}=0) \cdot P(\text{second}=0) = 0.5 \times 0.5 = 0.25$ ✓

**Entangled state** $(|00\rangle + |11\rangle)/\sqrt{2}$: qubits are correlated
- $P(01) = 0$ but $P(\text{first}=0) \cdot P(\text{second}=1) = 0.5 \times 0.5 = 0.25$
- These are NOT equal, so qubits are not independent!

```python
import numpy as np

# Bell state probabilities
print("Bell state (|00⟩ + |11⟩)/√2:")
bell_probs = {'00': 0.5, '01': 0, '10': 0, '11': 0.5}

P_first_0 = bell_probs['00'] + bell_probs['01']  # 0.5
P_second_1 = bell_probs['01'] + bell_probs['11']  # 0.5

print(f"P(first=0) = {P_first_0}")
print(f"P(second=1) = {P_second_1}")
print(f"P(first=0) × P(second=1) = {P_first_0 * P_second_1}")
print(f"P(01) = {bell_probs['01']}")
print(f"Independent? {np.isclose(bell_probs['01'], P_first_0 * P_second_1)}")
```
</Example>

## 6. Bayes' Theorem

<Definition term="Bayes' Theorem">
$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

This relates the conditional probability $P(A|B)$ to the "reverse" probability $P(B|A)$.
</Definition>

Bayes' theorem is crucial in:
- Quantum state tomography (inferring state from measurements)
- Bayesian quantum inference
- Quantum error correction (inferring errors from syndromes)

<Example>
A quantum error correction code can detect errors. Suppose:
- Error probability: $P(\text{error}) = 0.01$
- Detection probability given error: $P(\text{detect}|\text{error}) = 0.99$
- False alarm rate: $P(\text{detect}|\text{no error}) = 0.001$

If we detect an error, what's the probability an actual error occurred?

Using Bayes' theorem:
$$P(\text{error}|\text{detect}) = \frac{P(\text{detect}|\text{error}) \cdot P(\text{error})}{P(\text{detect})}$$

```python
# Given probabilities
P_error = 0.01
P_detect_given_error = 0.99
P_detect_given_no_error = 0.001

# Total probability of detection
P_detect = (P_detect_given_error * P_error + 
            P_detect_given_no_error * (1 - P_error))

# Bayes' theorem
P_error_given_detect = (P_detect_given_error * P_error) / P_detect

print(f"P(detect) = {P_detect:.6f}")
print(f"P(error | detect) = {P_error_given_detect:.4f}")
```
</Example>

## 7. Random Variables and Expected Values

<Definition term="Random Variable">
A random variable $X$ is a function from the sample space to real numbers: $X: \Omega \to \mathbb{R}$. It assigns a numerical value to each outcome.
</Definition>

<Definition term="Expected Value">
The expected value (mean) of a discrete random variable:
$$E[X] = \sum_x x \cdot P(X = x)$$
</Definition>

<Example>
Measuring a qubit in state $|\psi\rangle = \sqrt{0.3}|0\rangle + \sqrt{0.7}|1\rangle$:

Let $X$ = measurement outcome (0 or 1).
- $P(X=0) = 0.3$
- $P(X=1) = 0.7$

Expected value: $E[X] = 0 \cdot 0.3 + 1 \cdot 0.7 = 0.7$

```python
import numpy as np

# State amplitudes
alpha = np.sqrt(0.3)
beta = np.sqrt(0.7)

P_0 = np.abs(alpha)**2
P_1 = np.abs(beta)**2

expected_value = 0 * P_0 + 1 * P_1
print(f"P(0) = {P_0}")
print(f"P(1) = {P_1}")
print(f"E[X] = {expected_value}")
```
</Example>

## 8. Variance and Standard Deviation

<Definition term="Variance">
The variance measures the spread of a random variable:
$$\text{Var}(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2$$
</Definition>

<Definition term="Standard Deviation">
$$\sigma_X = \sqrt{\text{Var}(X)}$$
</Definition>

<Example>
For the previous qubit state with $E[X] = 0.7$:

$E[X^2] = 0^2 \cdot 0.3 + 1^2 \cdot 0.7 = 0.7$

$\text{Var}(X) = 0.7 - 0.7^2 = 0.7 - 0.49 = 0.21$

$\sigma_X = \sqrt{0.21} \approx 0.458$

```python
import numpy as np

P_0, P_1 = 0.3, 0.7
E_X = 0.7
E_X2 = 0**2 * P_0 + 1**2 * P_1

variance = E_X2 - E_X**2
std_dev = np.sqrt(variance)

print(f"E[X²] = {E_X2}")
print(f"Var(X) = {variance}")
print(f"σ_X = {std_dev:.4f}")
```
</Example>

## 9. The Law of Large Numbers

<Definition term="Law of Large Numbers">
As the number of independent measurements $n$ increases, the sample mean converges to the expected value:
$$\frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{n \to \infty} E[X]$$
</Definition>

This is why we can estimate quantum probabilities by repeated measurement!

```python
import numpy as np

# Simulate measuring a qubit many times
np.random.seed(42)

P_1 = 0.7  # True probability of measuring 1
n_measurements = [10, 100, 1000, 10000]

for n in n_measurements:
    measurements = np.random.choice([0, 1], size=n, p=[0.3, 0.7])
    sample_mean = np.mean(measurements)
    print(f"n={n:5d}: Sample mean = {sample_mean:.4f} (true = {P_1})")
```

## 10. Classical vs Quantum Randomness

There's a fundamental difference between classical and quantum randomness:

| Classical Randomness | Quantum Randomness |
|---------------------|-------------------|
| Arises from ignorance | Fundamental to nature |
| Deterministic in principle | Intrinsically random |
| Can be derandomized | Cannot be predicted even in principle |
| Hidden variables possible | Bell's theorem excludes local hidden variables |

<Warning>
Quantum randomness is not due to our lack of knowledge - it's a fundamental feature of nature! Before measurement, the qubit genuinely has no definite value for the measured property.
</Warning>
