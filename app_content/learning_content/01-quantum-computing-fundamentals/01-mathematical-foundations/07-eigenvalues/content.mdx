## 1. The Eigenvalue Problem

Some vectors are special: when transformed by a matrix, they only get scaled, not rotated. These are eigenvectors.

<Definition term="Eigenvector and Eigenvalue">
For a square matrix A, a non-zero vector v is an eigenvector with eigenvalue λ if:
$$A\mathbf{v} = \lambda\mathbf{v}$$
The matrix A transforms v by simply scaling it by λ.
</Definition>

<Tip>
"Eigen" comes from German meaning "own" or "characteristic." Eigenvectors are the "characteristic vectors" of a matrix - they reveal the matrix's fundamental behavior.
</Tip>

## 2. Why Eigenvalues Matter in Quantum Computing

In quantum mechanics:
- **Observables** (measurable quantities) are represented by Hermitian matrices
- **Eigenvalues** are the possible measurement outcomes
- **Eigenvectors** are the states with definite values for that observable

<Example>
The Pauli-Z matrix represents measuring spin along the z-axis:

$$Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$

Check: $Z|0\rangle = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}\begin{pmatrix} 1 \\ 0 \end{pmatrix} = \begin{pmatrix} 1 \\ 0 \end{pmatrix} = 1 \cdot |0\rangle$

So |0⟩ is an eigenvector of Z with eigenvalue +1 (spin up).

Similarly, |1⟩ is an eigenvector with eigenvalue -1 (spin down).
</Example>

## 3. Finding Eigenvalues: The Characteristic Equation

<Definition term="Characteristic Equation">
The eigenvalues of A are solutions to:
$$\det(A - \lambda I) = 0$$
This is called the characteristic equation. For an n×n matrix, it's a polynomial of degree n.
</Definition>

### Derivation

Starting from $A\mathbf{v} = \lambda\mathbf{v}$:
$$A\mathbf{v} - \lambda\mathbf{v} = \mathbf{0}$$
$$(A - \lambda I)\mathbf{v} = \mathbf{0}$$

For non-zero v to exist, $(A - \lambda I)$ must be singular:
$$\det(A - \lambda I) = 0$$

<Example>
Find eigenvalues of $A = \begin{pmatrix} 4 & 1 \\ 2 & 3 \end{pmatrix}$

$$\det(A - \lambda I) = \det\begin{pmatrix} 4-\lambda & 1 \\ 2 & 3-\lambda \end{pmatrix}$$
$$= (4-\lambda)(3-\lambda) - (1)(2)$$
$$= \lambda^2 - 7\lambda + 12 - 2$$
$$= \lambda^2 - 7\lambda + 10$$
$$= (\lambda - 5)(\lambda - 2)$$

So $\lambda_1 = 5$ and $\lambda_2 = 2$.

```python
import numpy as np

A = np.array([[4, 1], [2, 3]])
eigenvalues, eigenvectors = np.linalg.eig(A)

print(f"Eigenvalues: {eigenvalues}")
```
</Example>

## 4. Finding Eigenvectors

Once you have an eigenvalue λ, find eigenvectors by solving:
$$(A - \lambda I)\mathbf{v} = \mathbf{0}$$

<Example>
For $A = \begin{pmatrix} 4 & 1 \\ 2 & 3 \end{pmatrix}$ with $\lambda_1 = 5$:

$$(A - 5I)\mathbf{v} = \begin{pmatrix} -1 & 1 \\ 2 & -2 \end{pmatrix}\begin{pmatrix} v_1 \\ v_2 \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \end{pmatrix}$$

This gives $-v_1 + v_2 = 0$, so $v_1 = v_2$.

Eigenvector: $\mathbf{v}_1 = \begin{pmatrix} 1 \\ 1 \end{pmatrix}$ (or any scalar multiple)

```python
import numpy as np

A = np.array([[4, 1], [2, 3]])
eigenvalues, eigenvectors = np.linalg.eig(A)

print("Eigenvalues:", eigenvalues)
print("\nEigenvectors (as columns):")
print(eigenvectors)

# Verify
for i in range(len(eigenvalues)):
    lam = eigenvalues[i]
    v = eigenvectors[:, i]
    Av = A @ v
    lam_v = lam * v
    print(f"\nλ={lam:.1f}: A@v = {Av}, λ*v = {lam_v}")
```
</Example>

## 5. Properties of Eigenvalues and Eigenvectors

### For Any Matrix
- Sum of eigenvalues = trace of matrix: $\sum \lambda_i = \text{tr}(A)$
- Product of eigenvalues = determinant: $\prod \lambda_i = \det(A)$

### For Hermitian Matrices (Quantum Observables)
- All eigenvalues are **real** (essential for measurement outcomes!)
- Eigenvectors for different eigenvalues are **orthogonal**
- There exists an orthonormal basis of eigenvectors

<Warning>
Non-Hermitian matrices can have complex eigenvalues. Since measurement outcomes must be real numbers, quantum observables must be Hermitian!
</Warning>

```python
import numpy as np

# Hermitian matrix (observable)
H = np.array([[1, 1-1j], [1+1j, 2]])

eigenvalues, eigenvectors = np.linalg.eig(H)

print("Hermitian matrix H:")
print(H)
print(f"\nEigenvalues: {eigenvalues}")
print(f"All real? {np.allclose(eigenvalues.imag, 0)}")

# Check orthogonality
v1, v2 = eigenvectors[:, 0], eigenvectors[:, 1]
inner = np.vdot(v1, v2)  # Conjugate inner product
print(f"\nInner product of eigenvectors: {inner:.6f}")
print(f"Orthogonal? {np.abs(inner) < 1e-10}")
```

## 6. Spectral Decomposition

<Definition term="Spectral Decomposition">
A Hermitian matrix A can be written as:
$$A = \sum_i \lambda_i |\psi_i\rangle\langle\psi_i|$$
where λᵢ are eigenvalues and |ψᵢ⟩ are orthonormal eigenvectors.
</Definition>

This is profound: any observable is a weighted sum of projection operators onto its eigenstates!

<Example>
Spectral decomposition of Pauli-Z:

$$Z = (+1)|0\rangle\langle0| + (-1)|1\rangle\langle1|$$
$$= \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix} - \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}$$
$$= \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$

```python
import numpy as np

# Eigenstates of Z
ket_0 = np.array([[1], [0]])
ket_1 = np.array([[0], [1]])

# Projectors
P0 = ket_0 @ ket_0.conj().T
P1 = ket_1 @ ket_1.conj().T

# Spectral decomposition
Z_reconstructed = 1*P0 + (-1)*P1

print("|0⟩⟨0| =")
print(P0)
print("\n|1⟩⟨1| =")
print(P1)
print("\nZ = (+1)|0⟩⟨0| + (-1)|1⟩⟨1| =")
print(Z_reconstructed)
```
</Example>

## 7. Diagonalization

<Definition term="Diagonalization">
A matrix A is diagonalizable if there exists an invertible matrix P such that:
$$P^{-1}AP = D$$
where D is diagonal with eigenvalues on the diagonal, and columns of P are eigenvectors.
</Definition>

For unitary matrices (quantum gates), we can use $P^\dagger$ instead of $P^{-1}$:
$$U = PDP^\dagger$$

```python
import numpy as np

A = np.array([[4, 1], [2, 3]])
eigenvalues, P = np.linalg.eig(A)

# Diagonal matrix
D = np.diag(eigenvalues)

# Verify diagonalization
P_inv = np.linalg.inv(P)
reconstructed = P @ D @ P_inv

print("A =")
print(A)
print("\nEigenvector matrix P =")
print(P)
print("\nDiagonal matrix D =")
print(D)
print("\nP @ D @ P⁻¹ =")
print(reconstructed)
print(f"\nEquals A? {np.allclose(A, reconstructed)}")
```

## 8. Eigenvalues of Quantum Gates

### Pauli Gates

| Gate | Matrix | Eigenvalues | Eigenvectors |
|------|--------|-------------|--------------|
| I | [[1,0],[0,1]] | 1, 1 | Any vector |
| X | [[0,1],[1,0]] | +1, -1 | \|+⟩, \|-⟩ |
| Y | [[0,-i],[i,0]] | +1, -1 | \|+i⟩, \|-i⟩ |
| Z | [[1,0],[0,-1]] | +1, -1 | \|0⟩, \|1⟩ |

### Hadamard Gate

$$H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$$

Eigenvalues: $+1$ and $-1$

```python
import numpy as np

# Define gates
X = np.array([[0, 1], [1, 0]], dtype=complex)
Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)
H = np.array([[1, 1], [1, -1]], dtype=complex) / np.sqrt(2)

gates = {'X': X, 'Y': Y, 'Z': Z, 'H': H}

for name, gate in gates.items():
    evals, evecs = np.linalg.eig(gate)
    print(f"{name} gate:")
    print(f"  Eigenvalues: {evals}")
    print()
```

## 9. Measurement in Eigenbasis

When measuring observable A on state |ψ⟩:

1. **Possible outcomes**: Eigenvalues λᵢ of A
2. **Probability of λᵢ**: $|\langle\psi_i|\psi\rangle|^2$ where |ψᵢ⟩ is the eigenvector
3. **Post-measurement state**: Collapses to |ψᵢ⟩

<Example>
Measure Z on state $|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)$:

- Outcome +1 (state |0⟩): probability $|\langle0|+\rangle|^2 = |1/\sqrt{2}|^2 = 1/2$
- Outcome -1 (state |1⟩): probability $|\langle1|+\rangle|^2 = |1/\sqrt{2}|^2 = 1/2$

```python
import numpy as np

# Z eigenstates
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

# State to measure
ket_plus = np.array([1, 1]) / np.sqrt(2)

# Probabilities
p0 = np.abs(np.vdot(ket_0, ket_plus))**2
p1 = np.abs(np.vdot(ket_1, ket_plus))**2

print(f"Measuring Z on |+⟩:")
print(f"  P(+1, collapse to |0⟩) = {p0:.4f}")
print(f"  P(-1, collapse to |1⟩) = {p1:.4f}")
print(f"  Total probability: {p0 + p1:.4f}")
```
</Example>

## 10. Computing Functions of Matrices

Using spectral decomposition, we can compute functions of matrices:

<Formula>
$$f(A) = \sum_i f(\lambda_i) |\psi_i\rangle\langle\psi_i|$$
</Formula>

This is how we compute matrix exponentials for quantum time evolution!

<Example>
$$e^{i\theta Z} = e^{i\theta}|0\rangle\langle0| + e^{-i\theta}|1\rangle\langle1| = \begin{pmatrix} e^{i\theta} & 0 \\ 0 & e^{-i\theta} \end{pmatrix}$$

```python
import numpy as np
from scipy.linalg import expm

Z = np.array([[1, 0], [0, -1]], dtype=complex)
theta = np.pi / 4

# Using spectral decomposition
exp_Z_spectral = np.array([
    [np.exp(1j*theta), 0],
    [0, np.exp(-1j*theta)]
])

# Using scipy
exp_Z_scipy = expm(1j * theta * Z)

print("e^(iθZ) via spectral decomposition:")
print(exp_Z_spectral)
print("\ne^(iθZ) via scipy.linalg.expm:")
print(exp_Z_scipy)
print(f"\nMatch: {np.allclose(exp_Z_spectral, exp_Z_scipy)}")
```
</Example>

## Summary

Eigenvalues and eigenvectors are central to quantum mechanics:

1. **Definition**: $A\mathbf{v} = \lambda\mathbf{v}$ - eigenvectors are scaled, not rotated
2. **Characteristic Equation**: $\det(A - \lambda I) = 0$ finds eigenvalues
3. **Hermitian Properties**: Real eigenvalues, orthogonal eigenvectors
4. **Spectral Decomposition**: $A = \sum_i \lambda_i |\psi_i\rangle\langle\psi_i|$
5. **Quantum Measurement**: Eigenvalues = outcomes, eigenvectors = post-measurement states
6. **Matrix Functions**: Computed via spectral decomposition

Every quantum measurement is fundamentally about finding the eigenbasis of the observable being measured!
