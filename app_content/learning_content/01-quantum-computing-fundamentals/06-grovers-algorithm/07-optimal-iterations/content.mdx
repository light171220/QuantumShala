## 1. The Geometry of Grover's Algorithm

Recall that Grover's algorithm operates in a 2D subspace spanned by two orthogonal states:

<Definition term="Solution State |w>">
The superposition of all marked (solution) states: |w> = (1/sqrt(k)) * sum of all marked |x>
</Definition>

<Definition term="Non-solution State |s'>">
The uniform superposition of all unmarked states, orthogonal to |w>.
</Definition>

The initial uniform superposition |s> can be written as:

<Formula>
$$|s\rangle = \sin(\theta)|w\rangle + \cos(\theta)|s'\rangle$$
</Formula>

where the angle theta satisfies:

<Formula>
$$\sin(\theta) = \sqrt{\frac{k}{N}}$$
</Formula>

For a single solution (k=1):

<Formula>
$$\sin(\theta) = \frac{1}{\sqrt{N}} \quad \Rightarrow \quad \theta \approx \frac{1}{\sqrt{N}} \text{ for large N}$$
</Formula>

```python
import numpy as np

def initial_angle(N, k=1):
    """Calculate the initial angle theta."""
    return np.arcsin(np.sqrt(k/N))

print("Initial angle theta for various N (k=1 solution):")
print(f"{'N':>8} {'theta (rad)':>12} {'theta (deg)':>12} {'sin(theta)':>12}")
print("-" * 48)

for N in [4, 16, 64, 256, 1024]:
    theta = initial_angle(N)
    print(f"{N:>8} {theta:>12.4f} {np.degrees(theta):>12.2f} {np.sin(theta):>12.4f}")
```

## 2. Rotation Per Iteration

Each Grover iteration G = D * O rotates the state by exactly 2*theta towards |w>:

<Formula>
$$G|s\rangle = \sin(3\theta)|w\rangle + \cos(3\theta)|s'\rangle$$
</Formula>

After r iterations:

<Formula>
$$G^r|s\rangle = \sin((2r+1)\theta)|w\rangle + \cos((2r+1)\theta)|s'\rangle$$
</Formula>

```python
import numpy as np

def state_after_iterations(N, r, k=1):
    """
    Calculate amplitude of solution state after r iterations.
    Returns (amplitude_w, amplitude_s_prime)
    """
    theta = np.arcsin(np.sqrt(k/N))
    angle = (2*r + 1) * theta
    return np.sin(angle), np.cos(angle)

# Track rotation
N = 64
theta = np.arcsin(1/np.sqrt(N))

print(f"N={N}, theta = {theta:.4f} rad = {np.degrees(theta):.2f} deg")
print(f"\nState evolution:")
print(f"{'r':>3} {'Angle':>10} {'sin (|w>)':>12} {'Probability':>12}")
print("-" * 40)

for r in range(8):
    angle = (2*r + 1) * theta
    amp_w, amp_s = state_after_iterations(N, r)
    prob = amp_w**2
    print(f"{r:>3} {np.degrees(angle):>10.2f}° {amp_w:>12.4f} {prob:>12.4f}")
```

## 3. Deriving the Optimal Iteration Count

We want to maximize the probability of measuring |w>, which means maximizing sin^2((2r+1)*theta).

This is maximized when:

<Formula>
$$(2r_{opt}+1)\theta = \frac{\pi}{2}$$
</Formula>

Solving for r_opt:

<Formula>
$$r_{opt} = \frac{\pi/2 - \theta}{2\theta} = \frac{\pi}{4\theta} - \frac{1}{2}$$
</Formula>

For large N where theta is approximately 1/sqrt(N):

<Formula>
$$r_{opt} \approx \frac{\pi}{4} \cdot \frac{1}{\theta} \approx \frac{\pi}{4}\sqrt{N}$$
</Formula>

```python
import numpy as np

def optimal_iterations_exact(N, k=1):
    """Calculate exact optimal iterations using the angle formula."""
    theta = np.arcsin(np.sqrt(k/N))
    r_exact = (np.pi/2 - theta) / (2 * theta)
    return r_exact

def optimal_iterations_approx(N, k=1):
    """Approximate formula: (pi/4) * sqrt(N/k)"""
    return (np.pi/4) * np.sqrt(N/k)

print("Comparison of exact vs approximate optimal iterations:")
print(f"{'N':>8} {'Exact':>12} {'Approx':>12} {'Rounded':>10} {'Error %':>10}")
print("-" * 55)

for N in [4, 16, 64, 256, 1024, 4096]:
    exact = optimal_iterations_exact(N)
    approx = optimal_iterations_approx(N)
    rounded = round(exact)
    error = 100 * abs(exact - approx) / exact
    print(f"{N:>8} {exact:>12.4f} {approx:>12.4f} {rounded:>10} {error:>10.2f}%")
```

<Tip>
The approximation theta approximately equals 1/sqrt(N) improves as N increases. For practical purposes, round to the nearest integer.
</Tip>

## 5. Over-rotation: Too Many Iterations

A critical insight: running more than the optimal number of iterations **decreases** the success probability!

<Definition term="Over-rotation">
When the state rotates past the optimal angle (90 degrees from |s'>), the amplitude of |w> starts decreasing, reducing the probability of finding the solution.
</Definition>

```python
import numpy as np

def demonstrate_overrotation(N):
    """Show what happens when we over-rotate."""
    theta = np.arcsin(1/np.sqrt(N))
    r_opt = int(round(np.pi / (4*theta) - 0.5))

    print(f"N = {N}, Optimal iterations = {r_opt}")
    print(f"\nDemonstrating over-rotation:")
    print(f"{'r':>4} {'Angle':>10} {'Prob':>10} {'Status':>15}")
    print("-" * 45)

    for r in range(r_opt + 6):
        angle = (2*r + 1) * theta
        prob = np.sin(angle)**2

        if r < r_opt:
            status = "approaching"
        elif r == r_opt:
            status = "OPTIMAL"
        else:
            status = "OVER-ROTATED"

        print(f"{r:>4} {np.degrees(angle):>10.1f}° {prob:>10.4f} {status:>15}")

demonstrate_overrotation(256)
```

## 7. Oscillation Period

The success probability oscillates with a period related to theta:

<Formula>
$$T = \frac{\pi}{\theta} \text{ iterations (approximately)}$$
</Formula>

This means after T iterations, you return close to the initial probability!

```python
import numpy as np

def analyze_oscillation(N):
    """Analyze the oscillation period of Grover's algorithm."""
    theta = np.arcsin(1/np.sqrt(N))

    # Period of oscillation
    T = np.pi / theta
    r_opt = np.pi / (4*theta) - 0.5

    print(f"Oscillation analysis for N={N}:")
    print(f"  theta = {theta:.6f} rad = {np.degrees(theta):.2f} deg")
    print(f"  Full period T = {T:.2f} iterations")
    print(f"  Optimal r = {r_opt:.2f}")
    print(f"  At 2×optimal ({2*r_opt:.0f}): prob ≈ {np.sin((2*int(2*r_opt)+1)*theta)**2:.4f}")
    print(f"  At full period: prob ≈ initial prob")

    return T

for N in [16, 64, 256, 1024]:
    analyze_oscillation(N)
    print()
```

## 8. Practical Implementation

Here's a complete implementation with optimal iteration calculation:

```python
import numpy as np

def grovers_search(n_qubits, marked_items, verbose=True):
    """
    Complete Grover's algorithm with optimal iterations.

    Args:
        n_qubits: Number of qubits (N = 2^n_qubits)
        marked_items: List of marked item indices
        verbose: Print progress

    Returns:
        (final_state, optimal_iterations, success_probability)
    """
    N = 2**n_qubits
    k = len(marked_items)

    # Calculate optimal iterations
    theta = np.arcsin(np.sqrt(k/N))
    r_opt = int(round(np.pi / (4*theta) - 0.5))

    # Initial uniform superposition
    state = np.ones(N) / np.sqrt(N)

    # Oracle: flip phase of marked items
    oracle = np.eye(N)
    for item in marked_items:
        oracle[item, item] = -1

    # Diffusion operator
    s = np.ones(N) / np.sqrt(N)
    diffusion = 2 * np.outer(s, s) - np.eye(N)

    # Apply r_opt iterations
    for i in range(r_opt):
        state = diffusion @ (oracle @ state)

    # Calculate success probability (sum of probabilities of marked items)
    success_prob = sum(np.abs(state[item])**2 for item in marked_items)

    if verbose:
        print(f"Grover's Search Results:")
        print(f"  N = {N} (n = {n_qubits} qubits)")
        print(f"  k = {k} marked items")
        print(f"  Optimal iterations: {r_opt}")
        print(f"  Success probability: {success_prob:.6f}")
        print(f"  Expected angle: {np.degrees((2*r_opt+1)*theta):.1f}°")

    return state, r_opt, success_prob

# Test with single solution
state, r, prob = grovers_search(4, [7])
print()

# Test with multiple solutions
state, r, prob = grovers_search(4, [3, 7, 11])
```

## Summary

In this lesson, you learned:

- The initial state makes angle theta with |s'>, where sin(theta) = sqrt(k/N)
- Each Grover iteration rotates by 2*theta toward |w>
- Optimal iterations: r_opt = round((pi/2 - theta) / (2*theta)) approximately equals (pi/4)*sqrt(N/k)
- Over-rotation decreases success probability
- Success probability oscillates with period T = pi/theta

## Key Formulas

| Quantity | Formula |
|----------|---------|
| Initial angle | sin(theta) = sqrt(k/N) |
| After r iterations | angle = (2r+1)*theta |
| Optimal iterations | r_opt approximately equals (pi/4)*sqrt(N/k) |
| Success probability | P = sin^2((2r+1)*theta) |
| Oscillation period | T = pi/theta iterations |
