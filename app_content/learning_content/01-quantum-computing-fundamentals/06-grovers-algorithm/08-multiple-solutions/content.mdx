## Multiple Marked Items

When k items are marked (k > 1), the optimal iteration count changes:

<Formula>
$$k_{optimal} \approx \frac{\pi}{4}\sqrt{\frac{N}{k}}$$
</Formula>

```python
import numpy as np

def optimal_iterations_k(N, k):
    """Optimal iterations when k solutions exist."""
    return int(np.round(np.pi/4 * np.sqrt(N/k)))

N = 256
for k in [1, 2, 4, 8, 16]:
    iters = optimal_iterations_k(N, k)
    print(f"k={k:>2} solutions: {iters} iterations")
```

## Unknown Number of Solutions

When k is unknown:
1. Use quantum counting to estimate k
2. Or use a randomized approach with repeated trials

## Summary

- More solutions → fewer iterations needed
- Optimal: (π/4)√(N/k) for k solutions
- Quantum counting can find k if unknown
