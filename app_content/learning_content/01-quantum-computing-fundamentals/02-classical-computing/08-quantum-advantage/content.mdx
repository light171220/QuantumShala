## 1. The Quantum Promise

We've seen classical computing limitations. Now let's preview how quantum mechanics offers solutions!

<Definition term="Quantum Advantage">
The ability of quantum computers to solve certain problems faster than any classical computer, either provably or empirically.
</Definition>

**Key quantum resources:**
- **Superposition**: Explore many possibilities simultaneously
- **Entanglement**: Create correlations impossible classically
- **Interference**: Amplify correct answers, cancel wrong ones

## 2. Superposition: Quantum Parallelism

Classical bits are 0 OR 1. Qubits can be 0 AND 1 simultaneously!

<Formula>
$$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$$

where $|\alpha|^2 + |\beta|^2 = 1$
</Formula>

<Example>
```python
import numpy as np

def classical_search(database: list, target: int) -> int:
    """Classical: check one item at a time - O(N)"""
    steps = 0
    for i, item in enumerate(database):
        steps += 1
        if item == target:
            return i, steps
    return -1, steps

def quantum_advantage_preview(n_items: int):
    """Preview quantum search advantage."""
    classical_steps = n_items  # Worst case O(N)
    quantum_steps = int(np.sqrt(n_items) * np.pi / 4)  # Grover: O(√N)
    
    print(f"Searching {n_items:,} items:")
    print(f"  Classical (linear): {classical_steps:,} steps")
    print(f"  Quantum (Grover):   {quantum_steps:,} steps")
    print(f"  Speedup: {classical_steps / quantum_steps:.1f}x")

print("=== Quantum Search Advantage ===")
for n in [100, 1000, 1000000, 1000000000]:
    quantum_advantage_preview(n)
    print()
```
</Example>

<Warning>
Superposition doesn't mean checking all items at once! You can't just read out all answers. The magic is in using interference to amplify the right answer.
</Warning>

## 3. Entanglement: Quantum Correlations

Entangled qubits have correlated measurements that can't be explained classically.

<Definition term="Entanglement">
A quantum state where the measurement outcomes of multiple qubits are correlated, even when measured far apart.
</Definition>

**Bell State:**
$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

<Example>
```python
import numpy as np

def simulate_bell_state(n_measurements: int = 1000):
    """
    Simulate measurements of an entangled Bell state.
    Shows perfect correlation between qubits.
    """
    # Bell state: |00⟩ + |11⟩ (unnormalized for simplicity)
    # Measure: always get 00 or 11, never 01 or 10
    
    outcomes = {'00': 0, '01': 0, '10': 0, '11': 0}
    
    for _ in range(n_measurements):
        # 50% chance each of |00⟩ or |11⟩
        if np.random.random() < 0.5:
            outcomes['00'] += 1
        else:
            outcomes['11'] += 1
    
    print(f"Bell State Measurements ({n_measurements} trials):")
    for outcome, count in outcomes.items():
        prob = count / n_measurements
        print(f"  |{outcome}⟩: {count} ({prob*100:.1f}%)")
    
    print("\nKey insight: Qubits are ALWAYS correlated!")
    print("This correlation persists even if qubits are light-years apart!")

simulate_bell_state(10000)
```
</Example>

## 4. Key Quantum Algorithms Preview

### Shor's Algorithm: Breaking Cryptography

<Example>
```python
import math

def shor_speedup_analysis():
    """Compare classical vs quantum factoring."""
    print("=== Shor's Algorithm: Factoring ===\"")
    print(f"{'Bits':>6} | {'RSA Key':>10} | {'Classical':>15} | {'Quantum':>12} | {'Speedup':>12}")
    print("-" * 65)
    
    for bits in [512, 1024, 2048, 4096]:
        # Classical: Number field sieve O(exp(n^(1/3) * (log n)^(2/3)))
        classical_ops = math.exp(1.9 * (bits ** (1/3)) * (math.log(bits) ** (2/3)))
        
        # Quantum: Shor's algorithm O(n^3)
        quantum_ops = bits ** 3
        
        speedup = classical_ops / quantum_ops
        
        print(f"{bits:>6} | RSA-{bits:<4} | {classical_ops:>15.2e} | {quantum_ops:>12,} | {speedup:>12.2e}")
    
    print("\nImplication: RSA encryption is BROKEN by quantum computers!")

shor_speedup_analysis()
```
</Example>

### Grover's Algorithm: Faster Search

<Example>
```python
import math

def grover_analysis():
    """Analyze Grover's search speedup."""
    print("=== Grover's Algorithm: Unstructured Search ===")
    print(f"{'Items':>12} | {'Classical':>12} | {'Quantum':>12} | {'Speedup':>10}")
    print("-" * 55)
    
    for n in [100, 10000, 1000000, 10**9, 10**12]:
        classical = n  # O(N)
        quantum = int(math.pi / 4 * math.sqrt(n))  # O(√N)
        speedup = classical / quantum
        
        print(f"{n:>12,} | {classical:>12,} | {quantum:>12,} | {speedup:>10.1f}x")
    
    print("\nApplications:")
    print("  • Database search")
    print("  • Cryptographic key search (doubles effective key length)")
    print("  • Optimization problems")

grover_analysis()
```
</Example>

### Quantum Simulation: Nature's Computer

<Example>
```python
def quantum_simulation_advantage():
    """Show advantage for simulating quantum systems."""
    print("=== Quantum Simulation ===\")
    print("Simulating molecules/materials:")
    print()
    print(f"{'Electrons':>10} | {'Classical Memory':>18} | {'Quantum Qubits':>15}")
    print("-" * 50)
    
    molecules = [
        ("H₂", 2),
        ("H₂O", 10),
        ("Caffeine", 102),
        ("Hemoglobin", 2952),
        ("Protein folding", 10000),
    ]
    
    for name, electrons in molecules:
        # Classical: need 2^electrons states
        classical_states = 2 ** min(electrons, 60)  # Cap at 60 for display
        
        # Memory in bytes (16 per complex number)
        memory_bytes = classical_states * 16
        
        # Format memory
        if electrons > 60:
            memory_str = ">>> Universe"
        elif memory_bytes < 1e9:
            memory_str = f"{memory_bytes/1e6:.1f} MB"
        elif memory_bytes < 1e12:
            memory_str = f"{memory_bytes/1e9:.1f} GB"
        elif memory_bytes < 1e15:
            memory_str = f"{memory_bytes/1e12:.1f} TB"
        else:
            memory_str = f"{memory_bytes/1e15:.1f} PB"
        
        # Quantum: roughly 2 qubits per electron
        qubits_needed = electrons * 2
        
        print(f"{name:>10} ({electrons:>4}) | {memory_str:>18} | {qubits_needed:>15}")
    
    print("\nQuantum computers simulate quantum systems NATIVELY!")

quantum_simulation_advantage()
```
</Example>

| Search | O(N) | O(√N) | Database search |

| Machine Learning | Varies | Potential speedup | Classification |

## 6. Where Quantum Computers DON'T Help (Much)

<Warning>
Quantum computers are NOT magic! They don't speed up everything.
</Warning>

**Limited or no quantum advantage:**
- Sorting (O(n log n) already optimal)
- Most NP-complete problems (only quadratic speedup)
- Simple arithmetic
- Word processing, web browsing
- Most everyday computing tasks
