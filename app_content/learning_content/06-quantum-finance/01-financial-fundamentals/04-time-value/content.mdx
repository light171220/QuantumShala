## 1. Why Does Money Have Time Value?

A dollar today is worth more than a dollar tomorrow. This fundamental principle underlies all of finance.

<Definition term="Time Value of Money">
The time value of money (TVM) is the concept that money available now is worth more than the same amount in the future due to its potential earning capacity.
</Definition>

### Three Reasons for Time Value

1. **Opportunity Cost**: Money today can be invested to earn returns
2. **Inflation**: Purchasing power typically decreases over time
3. **Risk**: Future payments carry uncertainty

```python
import numpy as np

# Illustration: $1000 today vs $1000 in 5 years
principal = 1000
rate = 0.05  # 5% annual return
years = 5

# Future value of $1000 invested today
future_value = principal * (1 + rate) ** years
print(f"$1000 today grows to ${future_value:.2f} in {years} years")
print(f"Therefore, $1000 today is equivalent to ${future_value:.2f} in {years} years")
print(f"Or: ${future_value:.2f} in {years} years is worth $1000 today")
```

## 2. Future Value

Future value calculates what an investment today will be worth at a future date.

<Definition term="Future Value">
The future value (FV) of an investment is the value of a current asset at a specified date in the future, based on an assumed rate of growth.
</Definition>

### Simple Interest

With simple interest, you earn interest only on the principal:

<Formula>
$$FV = PV \times (1 + r \times t)$$
</Formula>

### Compound Interest

With compound interest, you earn interest on interest:

<Formula>
$$FV = PV \times (1 + r)^t$$
</Formula>

where:
- $PV$ = Present Value (initial investment)
- $r$ = Interest rate per period
- $t$ = Number of periods

```python
def future_value(pv, rate, periods, compounding='annual'):
    """
    Calculate future value with different compounding frequencies.

    Args:
        pv: Present value
        rate: Annual interest rate
        periods: Number of years
        compounding: 'annual', 'semi-annual', 'quarterly', 'monthly', 'continuous'
    """
    freq = {
        'annual': 1,
        'semi-annual': 2,
        'quarterly': 4,
        'monthly': 12,
        'daily': 365
    }

    if compounding == 'continuous':
        return pv * np.exp(rate * periods)
    else:
        n = freq[compounding]
        return pv * (1 + rate/n) ** (n * periods)

# Compare compounding frequencies
pv = 1000
rate = 0.10  # 10%
years = 10

print(f"Initial investment: ${pv}")
print(f"Rate: {rate:.0%}, Period: {years} years\n")

for comp in ['annual', 'semi-annual', 'quarterly', 'monthly', 'continuous']:
    fv = future_value(pv, rate, years, comp)
    print(f"{comp.capitalize():15} FV: ${fv:,.2f}")
```

<Tip>
The more frequently interest compounds, the higher the future value. Continuous compounding gives the maximum possible growth.
</Tip>

## 3. Present Value

Present value calculates the current worth of a future cash flow.

<Definition term="Present Value">
Present value (PV) is the current value of a future sum of money, discounted at the appropriate rate of return.
</Definition>

<Formula>
$$PV = \frac{FV}{(1 + r)^t}$$
</Formula>

```python
def present_value(fv, rate, periods):
    """Calculate present value of a future cash flow."""
    return fv / (1 + rate) ** periods

# Example: What is $10,000 in 5 years worth today?
fv = 10000
rate = 0.06  # 6% discount rate
years = 5

pv = present_value(fv, rate, years)
print(f"${fv:,} in {years} years")
print(f"Discount rate: {rate:.0%}")
print(f"Present value: ${pv:,.2f}")
```

### The Discount Factor

<Definition term="Discount Factor">
The discount factor is the multiplier used to calculate present value:
$$DF = \frac{1}{(1 + r)^t}$$
</Definition>

```python
def discount_factor(rate, period):
    """Calculate discount factor for a given rate and period."""
    return 1 / (1 + rate) ** period

# Discount factors for different periods
rate = 0.05
print("Discount Factors at 5%:")
print("-" * 30)
for t in range(1, 11):
    df = discount_factor(rate, t)
    print(f"Year {t:2}: {df:.4f}")
```

## 4. Annuities

An annuity is a series of equal payments at regular intervals.

<Definition term="Annuity">
An annuity is a series of equal cash flows occurring at regular intervals over a specified period.
</Definition>

### Present Value of an Annuity

<Formula>
$$PV_{annuity} = C \times \frac{1 - (1 + r)^{-n}}{r}$$
</Formula>

where:
- $C$ = Cash flow per period
- $r$ = Discount rate per period
- $n$ = Number of periods

```python
def pv_annuity(payment, rate, periods):
    """Calculate present value of an ordinary annuity."""
    if rate == 0:
        return payment * periods
    return payment * (1 - (1 + rate) ** -periods) / rate

# Example: Mortgage payment analysis
loan_amount = 300000
annual_rate = 0.06
years = 30

monthly_rate = annual_rate / 12
n_payments = years * 12

# Monthly payment that has PV equal to loan amount
# Rearranging: payment = PV * r / (1 - (1+r)^-n)
monthly_payment = loan_amount * monthly_rate / (1 - (1 + monthly_rate) ** -n_payments)

print(f"Loan Amount: ${loan_amount:,}")
print(f"Interest Rate: {annual_rate:.1%}")
print(f"Term: {years} years")
print(f"Monthly Payment: ${monthly_payment:,.2f}")
print(f"Total Paid: ${monthly_payment * n_payments:,.2f}")
print(f"Total Interest: ${monthly_payment * n_payments - loan_amount:,.2f}")
```

### Future Value of an Annuity

<Formula>
$$FV_{annuity} = C \times \frac{(1 + r)^n - 1}{r}$$
</Formula>

```python
def fv_annuity(payment, rate, periods):
    """Calculate future value of an ordinary annuity."""
    if rate == 0:
        return payment * periods
    return payment * ((1 + rate) ** periods - 1) / rate

# Example: Retirement savings
monthly_savings = 500
annual_return = 0.07
years = 30

monthly_return = annual_return / 12
n_months = years * 12

future_savings = fv_annuity(monthly_savings, monthly_return, n_months)
total_contributed = monthly_savings * n_months

print(f"Monthly Savings: ${monthly_savings}")
print(f"Annual Return: {annual_return:.1%}")
print(f"Years: {years}")
print(f"Total Contributed: ${total_contributed:,}")
print(f"Future Value: ${future_savings:,.2f}")
print(f"Investment Gains: ${future_savings - total_contributed:,.2f}")
```

## 5. Perpetuities

A perpetuity is an annuity that continues forever.

<Definition term="Perpetuity">
A perpetuity is an infinite series of equal periodic payments. Examples include preferred stock dividends and certain bonds.
</Definition>

<Formula>
$$PV_{perpetuity} = \frac{C}{r}$$
</Formula>

### Growing Perpetuity

<Formula>
$$PV_{growing\,perpetuity} = \frac{C}{r - g}$$
</Formula>

where $g$ is the constant growth rate (must be less than $r$).

```python
def pv_perpetuity(payment, rate, growth=0):
    """Calculate present value of a (growing) perpetuity."""
    if growth >= rate:
        raise ValueError("Growth rate must be less than discount rate")
    return payment / (rate - growth)

# Example: Valuing a stock with constant dividends
dividend = 2.00  # Annual dividend per share
required_return = 0.08  # 8%

pv = pv_perpetuity(dividend, required_return)
print(f"Dividend: ${dividend}")
print(f"Required Return: {required_return:.0%}")
print(f"Stock Value: ${pv:.2f}")

# With dividend growth
growth_rate = 0.03  # 3% annual growth
pv_growing = pv_perpetuity(dividend, required_return, growth_rate)
print(f"\nWith {growth_rate:.0%} growth:")
print(f"Stock Value: ${pv_growing:.2f}")
```

<Warning>
The Gordon Growth Model (growing perpetuity) only works when the growth rate is less than the discount rate. Otherwise, the present value would be infinite!
</Warning>

## 6. Continuous Compounding

In the limit of infinite compounding frequency, we get continuous compounding.

<Definition term="Continuous Compounding">
Continuous compounding assumes interest is calculated and added infinitely many times per period:
$$FV = PV \times e^{rt}$$
$$PV = FV \times e^{-rt}$$
</Definition>

```python
import numpy as np

def continuous_fv(pv, rate, time):
    """Future value with continuous compounding."""
    return pv * np.exp(rate * time)

def continuous_pv(fv, rate, time):
    """Present value with continuous discounting."""
    return fv * np.exp(-rate * time)

# Example
pv = 1000
rate = 0.08
years = 10

fv = continuous_fv(pv, rate, years)
print(f"Continuous compounding:")
print(f"${pv} at {rate:.0%} for {years} years = ${fv:.2f}")

# Verify: what PV gives this FV?
pv_check = continuous_pv(fv, rate, years)
print(f"Present value check: ${pv_check:.2f}")
```

### Why Continuous Compounding Matters

1. **Mathematical convenience**: Exponential functions have nice derivatives
2. **Option pricing**: Black-Scholes uses continuous compounding
3. **Quantum finance**: Many quantum algorithms assume continuous time

<Tip>
The continuously compounded rate $r_c$ relates to the discrete rate $r_d$ by:
$e^{r_c} = 1 + r_d$, so $r_c = \ln(1 + r_d)$
</Tip>

## 7. Net Present Value (NPV)

NPV is the fundamental tool for investment decisions.

<Definition term="Net Present Value">
Net Present Value is the sum of present values of all cash flows (including the initial investment):
$$NPV = \sum_{t=0}^{n} \frac{CF_t}{(1 + r)^t}$$
</Definition>

```python
def npv(cash_flows, discount_rate):
    """
    Calculate net present value of a series of cash flows.

    Args:
        cash_flows: List of cash flows, starting with initial investment (negative)
        discount_rate: Discount rate per period
    """
    return sum(cf / (1 + discount_rate)**t for t, cf in enumerate(cash_flows))

# Example: Investment project
initial_investment = -100000
annual_cash_flows = [25000, 30000, 35000, 40000, 30000]
discount_rate = 0.10

all_cash_flows = [initial_investment] + annual_cash_flows
project_npv = npv(all_cash_flows, discount_rate)

print("Project Cash Flows:")
print(f"Initial Investment: ${initial_investment:,}")
for i, cf in enumerate(annual_cash_flows, 1):
    print(f"Year {i}: ${cf:,}")
print(f"\nDiscount Rate: {discount_rate:.0%}")
print(f"NPV: ${project_npv:,.2f}")
print(f"Decision: {'Accept' if project_npv > 0 else 'Reject'}")
```

### Internal Rate of Return (IRR)

<Definition term="Internal Rate of Return">
The IRR is the discount rate that makes NPV equal to zero:
$$0 = \sum_{t=0}^{n} \frac{CF_t}{(1 + IRR)^t}$$
</Definition>

```python
from scipy.optimize import brentq

def irr(cash_flows):
    """Calculate internal rate of return."""
    def npv_at_rate(r):
        return sum(cf / (1 + r)**t for t, cf in enumerate(cash_flows))

    # Find rate where NPV = 0
    try:
        return brentq(npv_at_rate, -0.99, 10.0)
    except ValueError:
        return None

# Example
cash_flows = [-100000, 25000, 30000, 35000, 40000, 30000]
project_irr = irr(cash_flows)

print(f"Project IRR: {project_irr:.2%}")
print(f"If required return < {project_irr:.2%}, accept the project")
```

## 8. Bond Valuation

Bonds are valued as the present value of their future cash flows.

<Formula>
$$P = \sum_{t=1}^{n} \frac{C}{(1+y)^t} + \frac{F}{(1+y)^n}$$
</Formula>

where:
- $P$ = Bond price
- $C$ = Coupon payment
- $F$ = Face value
- $y$ = Yield to maturity
- $n$ = Number of periods

```python
def bond_price(face_value, coupon_rate, yield_rate, years, frequency=2):
    """
    Calculate bond price.

    Args:
        face_value: Par value of the bond
        coupon_rate: Annual coupon rate
        yield_rate: Yield to maturity (annual)
        years: Years to maturity
        frequency: Coupon payments per year (2 = semi-annual)
    """
    # Adjust for payment frequency
    coupon = (face_value * coupon_rate) / frequency
    y = yield_rate / frequency
    n = years * frequency

    # PV of coupons
    if y != 0:
        pv_coupons = coupon * (1 - (1 + y)**-n) / y
    else:
        pv_coupons = coupon * n

    # PV of face value
    pv_face = face_value / (1 + y)**n

    return pv_coupons + pv_face

# Example: 10-year Treasury bond
face_value = 1000
coupon_rate = 0.05  # 5% coupon
years = 10

# Price at different yields
print("Bond Price at Different Yields:")
print("-" * 40)
for yield_rate in [0.03, 0.04, 0.05, 0.06, 0.07]:
    price = bond_price(face_value, coupon_rate, yield_rate, years)
    status = "Premium" if price > face_value else "Discount" if price < face_value else "Par"
    print(f"Yield: {yield_rate:.0%}  Price: ${price:,.2f}  ({status})")
```

<Tip>
When yield < coupon rate, the bond trades at a **premium** (above par).
When yield > coupon rate, the bond trades at a **discount** (below par).
When yield = coupon rate, the bond trades at **par**.
</Tip>
