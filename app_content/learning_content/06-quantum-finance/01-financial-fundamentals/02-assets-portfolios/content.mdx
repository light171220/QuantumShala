## 1. What Are Financial Assets?

A financial asset is any asset that derives value from a contractual claim. Unlike physical assets (real estate, machinery), financial assets represent ownership or lending relationships.

<Definition term="Financial Asset">
A financial asset is a non-physical asset whose value is derived from a contractual claim, such as bank deposits, stocks, bonds, or derivatives.
</Definition>

### Asset Classification

| Category | Examples | Characteristics |
|----------|----------|-----------------|
| **Equity** | Stocks, ETFs | Ownership, dividends, voting rights |
| **Fixed Income** | Bonds, T-bills | Lending, interest payments, maturity |
| **Cash Equivalents** | Money market, CDs | High liquidity, low risk |
| **Derivatives** | Options, futures | Derived value, leverage |
| **Alternatives** | Commodities, crypto | Non-traditional, diversification |

## 2. Asset Returns

The return on an asset measures the gain or loss relative to the initial investment.

### Simple Return

<Definition term="Simple Return">
The simple return $R_t$ over period $t$ is the percentage change in price:
$$R_t = \frac{P_t - P_{t-1}}{P_{t-1}} = \frac{P_t}{P_{t-1}} - 1$$
</Definition>

```python
import numpy as np

def simple_return(prices):
    """Calculate simple returns from a price series."""
    prices = np.array(prices)
    returns = (prices[1:] - prices[:-1]) / prices[:-1]
    return returns

# Example: Calculate returns for a stock
prices = [100, 102, 99, 105, 103]
returns = simple_return(prices)
print("Daily returns:", [f"{r:.2%}" for r in returns])
# Output: ['2.00%', '-2.94%', '6.06%', '-1.90%']
```

### Log Return (Continuously Compounded)

<Definition term="Log Return">
The log return (or continuously compounded return) is:
$$r_t = \ln\left(\frac{P_t}{P_{t-1}}\right) = \ln(P_t) - \ln(P_{t-1})$$
</Definition>

Log returns have useful mathematical properties:
- **Additive over time**: $r_{t_1 \to t_3} = r_{t_1 \to t_2} + r_{t_2 \to t_3}$
- **Symmetric**: Equal positive and negative returns have equal magnitude
- **Better for modeling**: More suitable for normal distribution assumptions

```python
def log_return(prices):
    """Calculate log returns from a price series."""
    prices = np.array(prices)
    returns = np.log(prices[1:] / prices[:-1])
    return returns

prices = [100, 102, 99, 105, 103]
log_returns = log_return(prices)
print("Log returns:", [f"{r:.4f}" for r in log_returns])

# Verify additivity
total_simple = (prices[-1] / prices[0]) - 1
total_log = np.sum(log_returns)
print(f"Total simple return: {total_simple:.2%}")
print(f"Sum of log returns: {total_log:.4f}")
print(f"Exp of log sum: {np.exp(total_log) - 1:.2%}")  # Should match simple
```

<Tip>
For short time periods (daily returns), simple and log returns are very similar. For longer periods or volatile assets, log returns are preferred because they're additive across time.
</Tip>

## 3. Vector Representation of Assets

In portfolio theory, we represent assets as vectors of returns over multiple time periods.

<Formula>
$$\mathbf{r}_i = \begin{pmatrix} r_{i,1} \\ r_{i,2} \\ \vdots \\ r_{i,T} \end{pmatrix}$$
</Formula>

where $r_{i,t}$ is the return of asset $i$ at time $t$.

```python
import numpy as np

# Returns for 3 assets over 5 periods
asset_returns = np.array([
    [0.02, -0.01, 0.03, 0.01, 0.02],   # Asset 1 (Tech stock)
    [0.01, 0.01, -0.01, 0.02, 0.01],   # Asset 2 (Bond)
    [0.03, -0.02, 0.04, -0.01, 0.03]   # Asset 3 (Commodity)
])

# Each row is an asset, each column is a time period
print("Asset return matrix shape:", asset_returns.shape)  # (3, 5)

# Expected (mean) return for each asset
expected_returns = np.mean(asset_returns, axis=1)
print("Expected returns:", expected_returns)
```

### Expected Return

<Definition term="Expected Return">
The expected return $\mu_i$ of asset $i$ is the mean of its historical or projected returns:
$$\mu_i = \mathbb{E}[r_i] = \frac{1}{T}\sum_{t=1}^{T} r_{i,t}$$
</Definition>

## 4. What Is a Portfolio?

A portfolio is a collection of assets held by an investor. Mathematically, it's defined by the **weights** allocated to each asset.

<Definition term="Portfolio Weights">
Portfolio weights $w_i$ represent the fraction of total capital invested in each asset. The weights must sum to 1:
$$\sum_{i=1}^{n} w_i = 1$$
</Definition>

```python
# Define a portfolio with 3 assets
weights = np.array([0.5, 0.3, 0.2])  # 50% asset 1, 30% asset 2, 20% asset 3

# Verify weights sum to 1
print(f"Sum of weights: {weights.sum()}")  # 1.0

# Portfolio with short selling
weights_short = np.array([0.6, 0.5, -0.1])  # Short 10% of asset 3
print(f"Sum with short: {weights_short.sum()}")  # 1.0
```

<Warning>
A negative weight indicates a **short position** - borrowing and selling an asset you don't own, hoping to buy it back cheaper. Short selling adds complexity and risk.
</Warning>

### Long-Only Constraints

Many institutional investors can only hold long positions:

<Formula>
$$w_i \geq 0 \quad \forall i$$
$$\sum_{i=1}^{n} w_i = 1$$
</Formula>

## 5. Portfolio Return

The portfolio return is the weighted average of individual asset returns.

<Definition term="Portfolio Return">
For a portfolio with weights $\mathbf{w} = (w_1, \ldots, w_n)$ and asset returns $\mathbf{r} = (r_1, \ldots, r_n)$:
$$R_p = \sum_{i=1}^{n} w_i r_i = \mathbf{w}^T \mathbf{r}$$
</Definition>

```python
import numpy as np

# Asset returns for a single period
returns = np.array([0.08, 0.03, 0.12])  # 8%, 3%, 12%

# Portfolio weights
weights = np.array([0.5, 0.3, 0.2])  # 50%, 30%, 20%

# Portfolio return (dot product)
portfolio_return = np.dot(weights, returns)
print(f"Portfolio return: {portfolio_return:.2%}")  # 6.30%

# Breakdown
for i, (w, r) in enumerate(zip(weights, returns)):
    contribution = w * r
    print(f"Asset {i+1}: {w:.0%} × {r:.0%} = {contribution:.2%}")
```

### Expected Portfolio Return

<Formula>
$$\mathbb{E}[R_p] = \sum_{i=1}^{n} w_i \mu_i = \mathbf{w}^T \boldsymbol{\mu}$$
</Formula>

where $\boldsymbol{\mu}$ is the vector of expected returns.

```python
# Expected returns for each asset
expected_returns = np.array([0.10, 0.04, 0.15])  # 10%, 4%, 15%

# Expected portfolio return
expected_portfolio = np.dot(weights, expected_returns)
print(f"Expected portfolio return: {expected_portfolio:.2%}")
```

## 6. Covariance and Correlation

Assets don't move independently - their returns are correlated. Understanding these relationships is crucial for portfolio construction.

### Covariance

<Definition term="Covariance">
The covariance between assets $i$ and $j$ measures how their returns move together:
$$\sigma_{ij} = \text{Cov}(r_i, r_j) = \mathbb{E}[(r_i - \mu_i)(r_j - \mu_j)]$$
</Definition>

- $\sigma_{ij} > 0$: Assets tend to move together
- $\sigma_{ij} < 0$: Assets tend to move in opposite directions
- $\sigma_{ij} = 0$: No linear relationship

### Correlation

<Definition term="Correlation">
The correlation coefficient normalizes covariance to a range of [-1, 1]:
$$\rho_{ij} = \frac{\sigma_{ij}}{\sigma_i \sigma_j}$$
</Definition>

```python
import numpy as np

# Historical returns for 3 assets (252 trading days)
np.random.seed(42)
n_days = 252

# Simulate correlated returns
# Tech and growth stocks are positively correlated
# Bonds have low/negative correlation with stocks
returns_data = np.random.multivariate_normal(
    mean=[0.0004, 0.0002, 0.0006],  # Daily expected returns
    cov=[
        [0.0004, 0.0001, 0.0002],   # Variance/covariance matrix
        [0.0001, 0.0001, -0.00005],
        [0.0002, -0.00005, 0.0006]
    ],
    size=n_days
).T

# Calculate correlation matrix
correlation_matrix = np.corrcoef(returns_data)
print("Correlation Matrix:")
print(np.round(correlation_matrix, 3))
```

## 7. The Covariance Matrix

For portfolio analysis, we organize all covariances into a matrix.

<Definition term="Covariance Matrix">
The covariance matrix $\boldsymbol{\Sigma}$ is an $n \times n$ symmetric matrix where:
$$\Sigma_{ij} = \text{Cov}(r_i, r_j)$$
The diagonal elements $\Sigma_{ii} = \sigma_i^2$ are variances.
</Definition>

<Formula>
$$\boldsymbol{\Sigma} = \begin{pmatrix}
\sigma_1^2 & \sigma_{12} & \cdots & \sigma_{1n} \\
\sigma_{21} & \sigma_2^2 & \cdots & \sigma_{2n} \\
\vdots & \vdots & \ddots & \vdots \\
\sigma_{n1} & \sigma_{n2} & \cdots & \sigma_n^2
\end{pmatrix}$$
</Formula>

```python
import numpy as np

# Calculate covariance matrix from returns
def calculate_covariance_matrix(returns):
    """
    Calculate the sample covariance matrix.

    Args:
        returns: Array of shape (n_assets, n_periods)

    Returns:
        Covariance matrix of shape (n_assets, n_assets)
    """
    return np.cov(returns)

# Example with 3 assets
returns = np.array([
    [0.02, -0.01, 0.03, 0.01, 0.02, -0.01, 0.02],
    [0.01, 0.01, -0.01, 0.02, 0.01, 0.00, 0.01],
    [0.03, -0.02, 0.04, -0.01, 0.03, -0.02, 0.02]
])

cov_matrix = calculate_covariance_matrix(returns)
print("Covariance Matrix:")
print(np.round(cov_matrix, 6))

# Extract standard deviations (volatilities)
volatilities = np.sqrt(np.diag(cov_matrix))
print(f"\nVolatilities: {volatilities}")
```

<Tip>
The covariance matrix must be **positive semi-definite** - all eigenvalues are non-negative. This ensures portfolio variance is always non-negative.
</Tip>

## 8. Portfolio Variance

The portfolio's risk (variance) depends on individual asset variances AND their covariances.

<Definition term="Portfolio Variance">
For a portfolio with weights $\mathbf{w}$ and covariance matrix $\boldsymbol{\Sigma}$:
$$\sigma_p^2 = \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w} = \sum_{i=1}^{n} \sum_{j=1}^{n} w_i w_j \sigma_{ij}$$
</Definition>

```python
import numpy as np

# Portfolio weights
weights = np.array([0.5, 0.3, 0.2])

# Covariance matrix
cov_matrix = np.array([
    [0.04, 0.01, 0.02],    # Asset 1: 20% volatility
    [0.01, 0.01, -0.005],  # Asset 2: 10% volatility
    [0.02, -0.005, 0.09]   # Asset 3: 30% volatility
])

# Portfolio variance
portfolio_variance = weights @ cov_matrix @ weights
portfolio_volatility = np.sqrt(portfolio_variance)

print(f"Portfolio Variance: {portfolio_variance:.4f}")
print(f"Portfolio Volatility: {portfolio_volatility:.2%}")

# Compare to weighted average volatility (ignoring correlations)
individual_vols = np.sqrt(np.diag(cov_matrix))
weighted_avg_vol = np.dot(weights, individual_vols)
print(f"\nWeighted Average Volatility: {weighted_avg_vol:.2%}")
print(f"Diversification Benefit: {weighted_avg_vol - portfolio_volatility:.2%}")
```

<Warning>
Portfolio volatility is generally LESS than the weighted average of individual volatilities. This reduction is the **diversification benefit** - one of the core principles of modern portfolio theory.
</Warning>

## 9. The Power of Diversification

Diversification reduces portfolio risk without sacrificing expected return.

```python
import numpy as np
import matplotlib.pyplot as plt

def portfolio_volatility(weights, cov_matrix):
    """Calculate portfolio volatility."""
    return np.sqrt(weights @ cov_matrix @ weights)

# Two assets with 20% volatility each
vol = 0.20
correlations = [-0.5, 0, 0.5, 1.0]

# Plot portfolio volatility for different correlations
weights_range = np.linspace(0, 1, 100)

for corr in correlations:
    cov_matrix = np.array([
        [vol**2, corr * vol * vol],
        [corr * vol * vol, vol**2]
    ])

    vols = []
    for w1 in weights_range:
        w = np.array([w1, 1 - w1])
        vols.append(portfolio_volatility(w, cov_matrix))

    plt.plot(weights_range, vols, label=f'ρ = {corr}')

plt.xlabel('Weight in Asset 1')
plt.ylabel('Portfolio Volatility')
plt.title('Diversification Effect')
plt.legend()
plt.grid(True)
plt.show()
```

<Tip>
The lower the correlation between assets, the greater the diversification benefit. With perfectly negative correlation ($\rho = -1$), you can theoretically eliminate all risk!
</Tip>
