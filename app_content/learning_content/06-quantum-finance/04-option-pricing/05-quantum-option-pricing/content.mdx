## 1. Quantum Advantage for Option Pricing

Quantum computing offers potential speedup for Monte Carlo-based pricing.

<Definition term="Quantum Amplitude Estimation (QAE)">
A quantum algorithm that estimates the amplitude (and thus probability) of a quantum state. For option pricing, it achieves quadratic speedup: O(1/ε) vs classical O(1/ε²) for accuracy ε.
</Definition>

```python
import numpy as np
from scipy.stats import norm

class QuantumOptionPricing:
    """
    Conceptual framework for quantum option pricing.

    Demonstrates the principles of quantum speedup for
    Monte Carlo-based option pricing.
    """

    def __init__(self, S, K, T, r, sigma):
        self.S = S
        self.K = K
        self.T = T
        self.r = r
        self.sigma = sigma

    def classical_mc_complexity(self, target_accuracy):
        """
        Classical Monte Carlo complexity.

        Error = σ/√n, so n = σ²/ε² for accuracy ε
        """
        # Estimate payoff standard deviation
        sigma_payoff = self.S * self.sigma * np.sqrt(self.T)
        n_samples = (sigma_payoff / target_accuracy) ** 2
        return int(n_samples)

    def quantum_qae_complexity(self, target_accuracy):
        """
        Quantum amplitude estimation complexity.

        Error = O(1/M) where M is Grover iterations
        """
        # Quantum achieves ε accuracy with O(1/ε) queries
        n_queries = 1 / target_accuracy
        return int(n_queries)

    def theoretical_speedup(self, target_accuracy):
        """
        Calculate theoretical quantum speedup.
        """
        classical = self.classical_mc_complexity(target_accuracy)
        quantum = self.quantum_qae_complexity(target_accuracy)
        return classical / quantum

# Example
qop = QuantumOptionPricing(S=100, K=100, T=1.0, r=0.05, sigma=0.20)

print("Quantum vs Classical Complexity:")
print("="*50)
for accuracy in [0.1, 0.01, 0.001]:
    classical = qop.classical_mc_complexity(accuracy)
    quantum = qop.quantum_qae_complexity(accuracy)
    speedup = qop.theoretical_speedup(accuracy)

    print(f"\nAccuracy = {accuracy}:")
    print(f"  Classical samples: {classical:,}")
    print(f"  Quantum queries: {quantum:,}")
    print(f"  Speedup: {speedup:.0f}x")
```

<Formula>
Quantum Speedup:
- Classical MC: Error ∝ 1/√N, need N = O(1/ε²) samples
- Quantum QAE: Error ∝ 1/M, need M = O(1/ε) queries
- Speedup: Quadratic, O(√N) fewer operations
</Formula>

## 2. Quantum Circuit for Option Pricing

The quantum approach involves loading distributions and computing payoffs.

```python
class QuantumCircuitSimulation:
    """
    Simulate quantum option pricing circuit concepts.

    A real implementation would use Qiskit or similar.
    """

    def __init__(self, n_qubits=4):
        """
        Args:
            n_qubits: Number of qubits for price discretization
        """
        self.n_qubits = n_qubits
        self.n_states = 2 ** n_qubits

    def discretize_lognormal(self, S0, r, sigma, T, n_points):
        """
        Discretize lognormal distribution for quantum loading.

        Returns price levels and their probabilities.
        """
        # Price range (cover 99.9% of distribution)
        mu = np.log(S0) + (r - 0.5*sigma**2)*T
        std = sigma * np.sqrt(T)

        log_prices = np.linspace(mu - 4*std, mu + 4*std, n_points)
        prices = np.exp(log_prices)

        # Lognormal probabilities
        pdf = norm.pdf(log_prices, mu, std)
        probs = pdf / pdf.sum()

        return prices, probs

    def load_distribution_circuit(self, probs):
        """
        Conceptual: Create circuit to load probability distribution.

        In Qiskit, this would use amplitude loading techniques.
        """
        # Normalize probabilities
        probs = np.array(probs)
        probs = probs / probs.sum()

        # Calculate amplitudes (sqrt of probabilities)
        amplitudes = np.sqrt(probs)

        # In reality, we'd decompose this into gates
        # Here we just verify it's a valid quantum state
        assert np.isclose(np.sum(amplitudes**2), 1.0)

        return amplitudes

    def payoff_operator(self, prices, K, option_type='call'):
        """
        Create payoff operator for option.

        Returns payoff for each discretized price.
        """
        if option_type == 'call':
            payoffs = np.maximum(prices - K, 0)
        else:
            payoffs = np.maximum(K - prices, 0)

        return payoffs

    def quantum_expectation(self, amplitudes, payoffs, r, T):
        """
        Simulate quantum amplitude estimation of expected payoff.

        In reality, this would use QAE circuit.
        """
        probs = amplitudes ** 2
        expected_payoff = np.sum(probs * payoffs)
        discounted = np.exp(-r * T) * expected_payoff

        return discounted

# Example
qcs = QuantumCircuitSimulation(n_qubits=8)

S, K, T, r, sigma = 100, 100, 1.0, 0.05, 0.20
n_points = 2**8  # 256 price levels

# Discretize distribution
prices, probs = qcs.discretize_lognormal(S, r, sigma, T, n_points)

# Load as quantum state
amplitudes = qcs.load_distribution_circuit(probs)

# Create payoff
payoffs = qcs.payoff_operator(prices, K, 'call')

# Compute quantum expectation
quantum_price = qcs.quantum_expectation(amplitudes, payoffs, r, T)

# Compare to Black-Scholes
d1 = (np.log(S/K) + (r + sigma**2/2)*T) / (sigma*np.sqrt(T))
d2 = d1 - sigma*np.sqrt(T)
bs_price = S * norm.cdf(d1) - K * np.exp(-r*T) * norm.cdf(d2)

print("Quantum Circuit Simulation:")
print(f"  Discretization: {n_points} price levels")
print(f"  Quantum Price: ${quantum_price:.4f}")
print(f"  Black-Scholes: ${bs_price:.4f}")
print(f"  Error: ${abs(quantum_price - bs_price):.4f}")
```

## 3. Amplitude Estimation Algorithm

The core quantum algorithm for option pricing.

```python
class AmplitudeEstimation:
    """
    Simulate amplitude estimation for option pricing.
    """

    def __init__(self, n_eval_qubits=4):
        """
        Args:
            n_eval_qubits: Qubits for phase estimation (controls precision)
        """
        self.n_eval = n_eval_qubits
        self.n_grover = 2 ** n_eval_qubits

    def grover_iterate(self, amplitude, n_iterations):
        """
        Simulate effect of Grover iterations.

        Amplitude after k iterations:
        a_k = sin((2k+1) * arcsin(a_0))
        """
        theta = np.arcsin(amplitude)
        return np.sin((2 * n_iterations + 1) * theta)

    def estimate_amplitude(self, true_amplitude, n_shots=1000):
        """
        Simulate amplitude estimation measurement.

        Returns estimated amplitude with quantum uncertainty.
        """
        # Phase estimation gives us theta where a = sin(theta)
        theta = np.arcsin(true_amplitude)

        # Discretize to n_eval bits
        theta_discrete = round(theta * self.n_grover / np.pi)
        theta_estimate = theta_discrete * np.pi / self.n_grover

        # Add measurement noise
        noise = np.random.normal(0, np.pi / self.n_grover / 2)
        theta_measured = theta_estimate + noise

        amplitude_estimate = np.sin(theta_measured)

        return amplitude_estimate

    def option_price_qae(self, S, K, T, r, sigma, option_type='call'):
        """
        Option pricing using simulated QAE.
        """
        # True Black-Scholes probability of ITM
        d1 = (np.log(S/K) + (r + sigma**2/2)*T) / (sigma*np.sqrt(T))
        d2 = d1 - sigma*np.sqrt(T)

        if option_type == 'call':
            true_prob_itm = norm.cdf(d2)  # Risk-neutral probability
        else:
            true_prob_itm = norm.cdf(-d2)

        # Estimate using QAE
        estimated_prob = self.estimate_amplitude(np.sqrt(true_prob_itm)) ** 2

        # Calculate price (simplified - actual implementation more complex)
        if option_type == 'call':
            bs_price = S * norm.cdf(d1) - K * np.exp(-r*T) * norm.cdf(d2)
        else:
            bs_price = K * np.exp(-r*T) * norm.cdf(-d2) - S * norm.cdf(-d1)

        return bs_price, estimated_prob, true_prob_itm

# Example
ae = AmplitudeEstimation(n_eval_qubits=8)

price, est_prob, true_prob = ae.option_price_qae(100, 100, 1.0, 0.05, 0.20)

print("Amplitude Estimation for Option Pricing:")
print(f"  True ITM probability: {true_prob:.4f}")
print(f"  Estimated probability: {est_prob:.4f}")
print(f"  Black-Scholes price: ${price:.4f}")
print(f"  Precision: {2**8} = 256 levels")
```

<Warning>
Current NISQ (Noisy Intermediate-Scale Quantum) devices cannot realize practical quantum advantage for option pricing. The required circuit depths exceed current capabilities, and noise degrades the quadratic speedup.
</Warning>

## 5. Current Limitations and Future Potential

```python
class QuantumReadiness:
    """
    Assess quantum computing readiness for option pricing.
    """

    @staticmethod
    def current_hardware_limits():
        """
        Current NISQ device limitations.
        """
        return {
            'max_qubits': '100-1000 noisy qubits',
            'coherence_time': '~100 microseconds',
            'gate_error': '0.1-1% per gate',
            'max_circuit_depth': '~100 gates before noise dominates',
            'practical_qubits': '10-20 for meaningful computation'
        }

    @staticmethod
    def option_pricing_requirements(n_assets, accuracy):
        """
        Estimate quantum resources for option pricing.
        """
        # Qubits for price discretization
        price_qubits = int(np.ceil(np.log2(1 / accuracy)))

        # Qubits for amplitude estimation precision
        precision_qubits = int(np.ceil(np.log2(1 / accuracy)))

        # Ancilla and working qubits
        ancilla = 2 * price_qubits

        # Total for multi-asset
        total_qubits = n_assets * price_qubits + precision_qubits + ancilla

        # Circuit depth estimate
        grover_iterations = int(np.pi / (4 * accuracy))
        gates_per_grover = 50 * n_assets  # Rough estimate
        circuit_depth = grover_iterations * gates_per_grover

        return {
            'total_qubits': total_qubits,
            'circuit_depth': circuit_depth,
            'grover_iterations': grover_iterations
        }

    @staticmethod
    def timeline_estimate():
        """
        Estimated timeline for quantum advantage.
        """
        return {
            '2024-2026': {
                'status': 'Research and development',
```

## 6. Hybrid Classical-Quantum Approaches

Near-term hybrid approaches combining classical and quantum.

```python
class HybridOptionPricing:
    """
    Hybrid classical-quantum option pricing.
    """

    def __init__(self, S, K, T, r, sigma):
        self.S = S
        self.K = K
        self.T = T
        self.r = r
        self.sigma = sigma

    def importance_sampling_quantum(self, n_classical, n_quantum):
        """
        Use quantum for importance sampling, classical for pricing.

        Quantum identifies high-payoff regions.
        Classical samples more densely there.
        """
        # Classical: estimate payoff distribution
        Z = np.random.standard_normal(n_classical)
        S_T = self.S * np.exp((self.r - 0.5*self.sigma**2)*self.T +
                              self.sigma*np.sqrt(self.T)*Z)
        payoffs = np.maximum(S_T - self.K, 0)

        # "Quantum" identifies high-payoff region
        high_payoff_mask = payoffs > np.percentile(payoffs, 80)

        # Importance sample from high-payoff region
        Z_important = Z[high_payoff_mask]

        # Resample with importance weights
        weights = np.ones_like(payoffs)
        weights[high_payoff_mask] = 2.0  # Higher weight for important region
        weights = weights / weights.sum()

        price = np.exp(-self.r * self.T) * np.average(payoffs, weights=weights)

        return price

    def variational_quantum_pricing(self, n_layers=2):
        """
        Variational quantum eigensolver approach (conceptual).

        Use parameterized quantum circuit optimized classically.
        """
        # This would involve:
        # 1. Parameterized quantum circuit
        # 2. Classical optimizer (COBYLA, SPSA)
        # 3. Iterative optimization of parameters

        # Simulated result
        d1 = (np.log(self.S/self.K) + (self.r + self.sigma**2/2)*self.T) / \
             (self.sigma*np.sqrt(self.T))
        d2 = d1 - self.sigma*np.sqrt(self.T)

        bs_price = self.S * norm.cdf(d1) - self.K * np.exp(-self.r*self.T) * norm.cdf(d2)

        # Add simulated VQE noise
        noise = np.random.normal(0, 0.01 * bs_price)

        return bs_price + noise
```
