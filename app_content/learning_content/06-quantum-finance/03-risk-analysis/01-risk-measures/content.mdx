## 1. What is Financial Risk?

Financial risk refers to the possibility that an investment's actual return will differ from the expected return.

<Definition term="Financial Risk">
The quantifiable uncertainty about the future value of an investment or portfolio. Risk measures attempt to summarize this uncertainty in a single number or set of statistics.
</Definition>

### Types of Financial Risk

| Risk Type | Description | Example |
|-----------|-------------|---------|
| Market Risk | Price movements due to market factors | Stock price decline |
| Credit Risk | Counterparty default | Bond issuer bankruptcy |
| Liquidity Risk | Inability to trade at fair prices | Wide bid-ask spread |
| Operational Risk | System or process failures | Trading system outage |

## 2. Volatility and Standard Deviation

The most common risk measure is volatility, measured by standard deviation.

<Formula>
$$\sigma = \sqrt{\frac{1}{n-1} \sum_{i=1}^{n} (r_i - \bar{r})^2}$$
where $r_i$ are returns and $\bar{r}$ is the mean return.
</Formula>

```python
import numpy as np
import pandas as pd

def calculate_volatility(returns, annualization_factor=252):
    """
    Calculate annualized volatility from returns.

    Args:
        returns: Array or Series of returns
        annualization_factor: Trading days per year

    Returns:
        Annualized volatility
    """
    daily_std = np.std(returns, ddof=1)
    annual_vol = daily_std * np.sqrt(annualization_factor)
    return annual_vol

def calculate_rolling_volatility(returns, window=21, annualization_factor=252):
    """
    Calculate rolling volatility.

    Args:
        returns: Return series
        window: Rolling window size (e.g., 21 days = 1 month)

    Returns:
        Rolling volatility series
    """
    returns = pd.Series(returns)
    rolling_std = returns.rolling(window=window).std()
    return rolling_std * np.sqrt(annualization_factor)

# Example
np.random.seed(42)
n_days = 252
daily_returns = np.random.normal(0.0004, 0.015, n_days)  # ~10% annual return, ~24% vol

vol = calculate_volatility(daily_returns)
print(f"Annualized volatility: {vol:.2%}")

# Rolling volatility
rolling_vol = calculate_rolling_volatility(daily_returns, window=21)
print(f"Rolling vol range: {rolling_vol.min():.2%} - {rolling_vol.max():.2%}")
```

### Limitations of Volatility

<Warning>
Volatility treats upside and downside movements equally. An investment that often gains 5% is considered as "risky" as one that often loses 5%.
</Warning>

```python
def calculate_upside_downside_volatility(returns):
    """
    Separate volatility into upside and downside components.
    """
    returns = np.array(returns)
    upside = returns[returns > 0]
    downside = returns[returns < 0]

    upside_vol = np.std(upside, ddof=1) * np.sqrt(252) if len(upside) > 1 else 0
    downside_vol = np.std(downside, ddof=1) * np.sqrt(252) if len(downside) > 1 else 0

    return upside_vol, downside_vol

up_vol, down_vol = calculate_upside_downside_volatility(daily_returns)
print(f"Upside volatility: {up_vol:.2%}")
print(f"Downside volatility: {down_vol:.2%}")
```

## 3. Downside Risk Measures

Downside risk measures focus only on losses.

### Semi-Deviation

<Formula>
$$\sigma_{down} = \sqrt{\frac{1}{n-1} \sum_{i: r_i < \mu} (r_i - \mu)^2}$$
</Formula>

```python
def semi_deviation(returns, threshold=0):
    """
    Calculate semi-deviation (downside deviation).

    Args:
        returns: Return array
        threshold: Return threshold (default 0 = losses)

    Returns:
        Annualized semi-deviation
    """
    returns = np.array(returns)
    downside_returns = returns[returns < threshold]

    if len(downside_returns) < 2:
        return 0

    semi_var = np.sum((downside_returns - threshold)**2) / (len(returns) - 1)
    return np.sqrt(semi_var) * np.sqrt(252)

semi_dev = semi_deviation(daily_returns)
print(f"Semi-deviation: {semi_dev:.2%}")
print(f"Ratio to full vol: {semi_dev/vol:.2f}")
```

### Sortino Ratio

Uses downside deviation instead of standard deviation:

<Formula>
$$\text{Sortino} = \frac{R_p - R_f}{\sigma_{down}}$$
</Formula>

```python
def sortino_ratio(returns, risk_free_rate=0.02, target_return=0):
    """
    Calculate Sortino ratio.

    Args:
        returns: Daily returns
        risk_free_rate: Annual risk-free rate
        target_return: Target return threshold

    Returns:
        Annualized Sortino ratio
    """
    annual_return = np.mean(returns) * 252
    downside_std = semi_deviation(returns, threshold=target_return/252)

    if downside_std == 0:
        return np.inf

    return (annual_return - risk_free_rate) / downside_std

sortino = sortino_ratio(daily_returns)
sharpe = (np.mean(daily_returns) * 252 - 0.02) / vol
print(f"Sortino ratio: {sortino:.3f}")
print(f"Sharpe ratio: {sharpe:.3f}")
```

## 4. Maximum Drawdown

Drawdown measures peak-to-trough decline.

<Definition term="Maximum Drawdown">
The largest peak-to-trough decline in portfolio value before a new peak is reached. It measures the worst-case loss an investor could have experienced.
</Definition>

```python
def calculate_drawdown(returns):
    """
    Calculate drawdown series and maximum drawdown.

    Args:
        returns: Array of returns

    Returns:
        drawdown: Drawdown series
        max_dd: Maximum drawdown
        max_dd_duration: Duration in periods
    """
    # Cumulative returns
    cum_returns = np.cumprod(1 + np.array(returns))

    # Running maximum
    running_max = np.maximum.accumulate(cum_returns)

    # Drawdown series
    drawdown = (cum_returns - running_max) / running_max

    # Maximum drawdown
    max_dd = np.min(drawdown)

    # Duration (time from peak to trough)
    trough_idx = np.argmin(drawdown)
    peak_idx = np.argmax(cum_returns[:trough_idx+1]) if trough_idx > 0 else 0
    max_dd_duration = trough_idx - peak_idx

    return drawdown, max_dd, max_dd_duration

dd_series, max_dd, duration = calculate_drawdown(daily_returns)
print(f"Maximum drawdown: {max_dd:.2%}")
print(f"Drawdown duration: {duration} days")
```

### Calmar Ratio

Returns relative to maximum drawdown:

<Formula>
$$\text{Calmar} = \frac{\text{Annualized Return}}{|\text{Max Drawdown}|}$$
</Formula>

```python
def calmar_ratio(returns):
    """
    Calculate Calmar ratio (return / max drawdown).
    """
    annual_return = np.mean(returns) * 252
    _, max_dd, _ = calculate_drawdown(returns)

    if max_dd == 0:
        return np.inf

    return annual_return / abs(max_dd)

calmar = calmar_ratio(daily_returns)
print(f"Calmar ratio: {calmar:.3f}")
```

## 5. Coherent Risk Measures

Modern risk theory defines properties that "good" risk measures should satisfy.

<Definition term="Coherent Risk Measure">
A risk measure ρ is coherent if it satisfies:
1. **Monotonicity**: If X ≤ Y always, then ρ(X) ≥ ρ(Y)
2. **Sub-additivity**: ρ(X+Y) ≤ ρ(X) + ρ(Y)
3. **Positive homogeneity**: ρ(λX) = λρ(X) for λ > 0
4. **Translation invariance**: ρ(X+c) = ρ(X) - c for constant c
</Definition>

### Properties Explained

| Property | Meaning | Implication |
|----------|---------|-------------|
| Monotonicity | Worse outcomes = higher risk | Intuitive ordering |
| Sub-additivity | Diversification reduces risk | Portfolio benefits |
| Positive homogeneity | Double position = double risk | Scale consistency |
| Translation invariance | Adding cash reduces risk | Risk offset |

<Warning>
Standard deviation is NOT a coherent risk measure because it doesn't satisfy monotonicity (it treats gains and losses symmetrically).
</Warning>

## 6. Comparing Risk Measures

```python
def comprehensive_risk_analysis(returns, risk_free_rate=0.02):
    """
    Calculate multiple risk measures for comparison.
    """
    returns = np.array(returns)

    # Basic statistics
    mean_return = np.mean(returns) * 252
    volatility = np.std(returns, ddof=1) * np.sqrt(252)

    # Downside measures
    semi_dev = semi_deviation(returns)
    _, max_dd, duration = calculate_drawdown(returns)

    # Ratios
    sharpe = (mean_return - risk_free_rate) / volatility if volatility > 0 else 0
    sortino = sortino_ratio(returns, risk_free_rate)
    calmar = calmar_ratio(returns)

    return {
        'Annual Return': mean_return,
        'Volatility': volatility,
        'Semi-deviation': semi_dev,
        'Max Drawdown': max_dd,
        'DD Duration': duration,
        'Sharpe Ratio': sharpe,
        'Sortino Ratio': sortino,
        'Calmar Ratio': calmar
    }

# Compare two return series
np.random.seed(42)

# Strategy A: Consistent but lower returns
returns_a = np.random.normal(0.0003, 0.010, 252)

# Strategy B: Higher returns but with occasional large losses
returns_b = np.random.normal(0.0005, 0.015, 252)
# Add some large losses
returns_b[50] = -0.08
returns_b[150] = -0.06

print("Strategy A:")
for k, v in comprehensive_risk_analysis(returns_a).items():
    if isinstance(v, float):
        print(f"  {k}: {v:.4f}")
    else:
        print(f"  {k}: {v}")

print("\nStrategy B:")
for k, v in comprehensive_risk_analysis(returns_b).items():
    if isinstance(v, float):
        print(f"  {k}: {v:.4f}")
    else:
        print(f"  {k}: {v}")
```

<Tip>
No single risk measure captures all aspects of risk. Use multiple measures to get a complete picture of an investment's risk profile.
</Tip>
