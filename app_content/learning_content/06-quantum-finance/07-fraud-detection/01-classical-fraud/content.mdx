# Classical Fraud Detection Methods

Fraud detection is a critical application in financial systems. This lesson covers traditional approaches that form the foundation for understanding how quantum computing can enhance fraud detection.

## The Fraud Detection Problem

<Definition term="Financial Fraud">
Intentional deception to secure unfair or unlawful gain, including credit card fraud, identity theft, money laundering, and insurance fraud.
</Definition>

Financial fraud costs billions annually. Detection systems must balance catching fraud (high recall) while minimizing false positives that frustrate legitimate customers.

### Key Challenges

## Rule-Based Detection

<Tip>
Rule-based systems are interpretable and fast, making them suitable for initial screening. However, they struggle with novel fraud patterns.
</Tip>

### Implementing Rule-Based Detection

```python
from typing import Callable


@dataclass
class Rule:
    """A fraud detection rule."""
    name: str
    condition: Callable[[Transaction], bool]
    risk_score: float
    description: str


class RuleBasedFraudDetector:
    """
    Rule-based fraud detection system.

    Uses expert-defined rules to flag suspicious transactions.
    """

    def __init__(self, threshold: float = 50.0):
        self.rules: List[Rule] = []
        self.threshold = threshold
        self._setup_default_rules()

    def _setup_default_rules(self):
        """Setup common fraud detection rules."""

        # Rule 1: High amount transactions
        self.add_rule(Rule(
            name="high_amount",
            condition=lambda t: t.amount > 5000,
            risk_score=25.0,
            description="Transaction amount exceeds $5,000"
        ))

        # Rule 2: International card-not-present
        self.add_rule(Rule(
            name="intl_cnp",
            condition=lambda t: t.is_international and not t.card_present,
            risk_score=30.0,
            description="International transaction without card present"
        ))

        # Rule 3: Unusual hours
        self.add_rule(Rule(
            name="odd_hours",
            condition=lambda t: t.hour_of_day < 6 or t.hour_of_day > 23,
            risk_score=15.0,
            description="Transaction during unusual hours (midnight-6am)"
        ))

        # Rule 4: High-risk merchant category
        high_risk_categories = {'gambling', 'cryptocurrency', 'wire_transfer'}
        self.add_rule(Rule(
            name="high_risk_merchant",
            condition=lambda t: t.merchant_category in high_risk_categories,
            risk_score=20.0,
            description="High-risk merchant category"
        ))

        # Rule 5: Distance anomaly
        self.add_rule(Rule(
            name="distance_anomaly",
            condition=lambda t: t.distance_from_home > 500,
            risk_score=20.0,
            description="Transaction far from customer's usual location"
        ))

    def add_rule(self, rule: Rule):
        """Add a detection rule."""
        self.rules.append(rule)

    def evaluate(self, transaction: Transaction) -> Dict[str, Any]:
        """
        Evaluate a transaction against all rules.

        Returns risk assessment with triggered rules.
        """
        triggered_rules = []
        total_score = 0.0

        for rule in self.rules:
            try:
                if rule.condition(transaction):
                    triggered_rules.append({
                        'name': rule.name,
                        'score': rule.risk_score,
                        'description': rule.description
                    })
                    total_score += rule.risk_score
            except Exception as e:
                warnings.warn(f"Rule {rule.name} failed: {e}")

        is_fraud = total_score >= self.threshold

        return {
            'transaction_id': transaction.transaction_id,
            'risk_score': total_score,
            'is_fraud': is_fraud,
            'triggered_rules': triggered_rules,
            'rule_count': len(triggered_rules)
        }

    def batch_evaluate(
        self,
        transactions: List[Transaction]
    ) -> List[Dict[str, Any]]:
        """Evaluate multiple transactions."""
        return [self.evaluate(t) for t in transactions]


# Example usage
detector = RuleBasedFraudDetector(threshold=40.0)

# Test transaction
test_transaction = Transaction(
    transaction_id="TXN001",
    amount=7500,
    merchant_category="electronics",
    location="New York",
    timestamp="2024-01-15T02:30:00",
    customer_id="CUST001",
    card_present=False,
    is_international=True,
    hour_of_day=2,
    distance_from_home=50
)

result = detector.evaluate(test_transaction)
print(f"Risk Score: {result['risk_score']}")
print(f"Flagged as Fraud: {result['is_fraud']}")
print(f"Triggered Rules: {result['triggered_rules']}")
```

```python
from collections import Counter


class SimpleFraudClassifier:
    """
    Random Forest-style fraud classifier.

    Demonstrates key concepts without external dependencies.
    """

    def __init__(
        self,
        n_trees: int = 10,
        max_depth: int = 5,
        min_samples_split: int = 10,
        random_state: int = 42
    ):
        self.n_trees = n_trees
        self.max_depth = max_depth
        self.min_samples_split = min_samples_split
        self.random_state = random_state
        self.trees = []
        self.feature_names = []

    def fit(self, X: np.ndarray, y: np.ndarray, feature_names: List[str] = None):
        """Train the ensemble."""
        np.random.seed(self.random_state)
        self.feature_names = feature_names or [f"f{i}" for i in range(X.shape[1])]

        for i in range(self.n_trees):
            # Bootstrap sample
            indices = np.random.choice(len(X), size=len(X), replace=True)
            X_bootstrap = X[indices]
            y_bootstrap = y[indices]

            # Build tree
            tree = self._build_tree(X_bootstrap, y_bootstrap, depth=0)
            self.trees.append(tree)

        return self

    def _build_tree(
        self,
        X: np.ndarray,
        y: np.ndarray,
        depth: int
    ) -> Dict:
        """Recursively build a decision tree."""
        n_samples = len(y)
        n_classes = len(np.unique(y))

        # Stopping conditions
        if (depth >= self.max_depth or
            n_samples < self.min_samples_split or
            n_classes == 1):
            return {'leaf': True, 'prediction': np.mean(y)}

        # Find best split
        best_gain = -np.inf
        best_split = None

        # Random feature subset (sqrt of total)
        n_features = X.shape[1]
        feature_subset = np.random.choice(
            n_features,
            size=int(np.sqrt(n_features)) + 1,
            replace=False
        )

        for feature_idx in feature_subset:
            thresholds = np.unique(X[:, feature_idx])

            for threshold in thresholds[::max(1, len(thresholds)//10)]:
                left_mask = X[:, feature_idx] <= threshold
                right_mask = ~left_mask

                if left_mask.sum() < 2 or right_mask.sum() < 2:
                    continue

                gain = self._information_gain(y, left_mask, right_mask)

                if gain > best_gain:
                    best_gain = gain
                    best_split = {
                        'feature': feature_idx,
                        'threshold': threshold,
                        'left_mask': left_mask,
                        'right_mask': right_mask
                    }

        if best_split is None:
            return {'leaf': True, 'prediction': np.mean(y)}

        # Recurse
        left_tree = self._build_tree(
            X[best_split['left_mask']],
            y[best_split['left_mask']],
            depth + 1
        )
        right_tree = self._build_tree(
            X[best_split['right_mask']],
            y[best_split['right_mask']],
            depth + 1
        )

        return {
            'leaf': False,
            'feature': best_split['feature'],
            'threshold': best_split['threshold'],
            'left': left_tree,
            'right': right_tree
        }

    def _information_gain(
        self,
        y: np.ndarray,
        left_mask: np.ndarray,
        right_mask: np.ndarray
    ) -> float:
        """Calculate information gain for a split."""
        def gini(y_subset):
            if len(y_subset) == 0:
                return 0
            p = np.mean(y_subset)
            return 1 - p**2 - (1-p)**2

        n = len(y)
        n_left = left_mask.sum()
        n_right = right_mask.sum()

        parent_gini = gini(y)
        child_gini = (
            (n_left / n) * gini(y[left_mask]) +
            (n_right / n) * gini(y[right_mask])
        )

        return parent_gini - child_gini

    def predict_proba(self, X: np.ndarray) -> np.ndarray:
        """Predict fraud probability."""
        predictions = np.zeros((len(X), len(self.trees)))

        for i, tree in enumerate(self.trees):
            predictions[:, i] = self._predict_tree(tree, X)

        return predictions.mean(axis=1)

    def _predict_tree(self, tree: Dict, X: np.ndarray) -> np.ndarray:
        """Get predictions from a single tree."""
        predictions = np.zeros(len(X))

        for i, x in enumerate(X):
            node = tree
            while not node['leaf']:
                if x[node['feature']] <= node['threshold']:
                    node = node['left']
                else:
                    node = node['right']
            predictions[i] = node['prediction']

        return predictions

    def predict(self, X: np.ndarray, threshold: float = 0.5) -> np.ndarray:
        """Predict fraud labels."""
        proba = self.predict_proba(X)
        return (proba >= threshold).astype(int)

    def feature_importance(self) -> Dict[str, float]:
        """Calculate feature importance scores."""
        importance = np.zeros(len(self.feature_names))

        for tree in self.trees:
            self._accumulate_importance(tree, importance, 1.0)

        importance = importance / importance.sum()
        return dict(zip(self.feature_names, importance))

    def _accumulate_importance(
        self,
        node: Dict,
        importance: np.ndarray,
        weight: float
    ):
        """Recursively accumulate feature importance."""
        if node['leaf']:
            return

        importance[node['feature']] += weight
        self._accumulate_importance(node['left'], importance, weight * 0.5)
        self._accumulate_importance(node['right'], importance, weight * 0.5)
```

## Handling Class Imbalance

<Warning>
Standard machine learning algorithms assume balanced classes. With fraud rates below 1%, models will learn to predict everything as legitimate unless we address the imbalance.
</Warning>

### Resampling Techniques

```python
class ImbalanceHandler:
    """
    Techniques for handling class imbalance in fraud detection.
    """

    @staticmethod
    def random_oversample(
        X: np.ndarray,
        y: np.ndarray,
        target_ratio: float = 1.0,
        random_state: int = 42
    ) -> Tuple[np.ndarray, np.ndarray]:
        """
        Oversample minority class by random duplication.

        Args:
            target_ratio: Desired ratio of minority to majority class
        """
        np.random.seed(random_state)

        minority_mask = y == 1
        majority_mask = y == 0

        X_minority = X[minority_mask]
        y_minority = y[minority_mask]
        X_majority = X[majority_mask]
        y_majority = y[majority_mask]

        n_majority = len(y_majority)
        n_minority = len(y_minority)
        target_minority = int(n_majority * target_ratio)

        if target_minority <= n_minority:
            return X, y

        # Oversample minority
        oversample_indices = np.random.choice(
            n_minority,
            size=target_minority - n_minority,
            replace=True
        )

        X_oversampled = np.vstack([
            X_minority,
            X_minority[oversample_indices]
        ])
        y_oversampled = np.hstack([
            y_minority,
            y_minority[oversample_indices]
        ])

        # Combine
        X_balanced = np.vstack([X_majority, X_oversampled])
        y_balanced = np.hstack([y_majority, y_oversampled])

        # Shuffle
        shuffle_idx = np.random.permutation(len(y_balanced))

        return X_balanced[shuffle_idx], y_balanced[shuffle_idx]

    @staticmethod
    def random_undersample(
        X: np.ndarray,
        y: np.ndarray,
        target_ratio: float = 1.0,
        random_state: int = 42
    ) -> Tuple[np.ndarray, np.ndarray]:
        """
        Undersample majority class by random removal.
        """
        np.random.seed(random_state)

        minority_mask = y == 1
        majority_mask = y == 0

        X_minority = X[minority_mask]
        y_minority = y[minority_mask]
        X_majority = X[majority_mask]
        y_majority = y[majority_mask]

        n_minority = len(y_minority)
        target_majority = int(n_minority / target_ratio)

        # Undersample majority
        undersample_indices = np.random.choice(
            len(y_majority),
            size=min(target_majority, len(y_majority)),
            replace=False
        )

        X_undersampled = X_majority[undersample_indices]
        y_undersampled = y_majority[undersample_indices]

        # Combine
        X_balanced = np.vstack([X_undersampled, X_minority])
        y_balanced = np.hstack([y_undersampled, y_minority])

        # Shuffle
        shuffle_idx = np.random.permutation(len(y_balanced))

        return X_balanced[shuffle_idx], y_balanced[shuffle_idx]

    @staticmethod
    def smote(
        X: np.ndarray,
        y: np.ndarray,
        k_neighbors: int = 5,
        target_ratio: float = 1.0,
        random_state: int = 42
    ) -> Tuple[np.ndarray, np.ndarray]:
        """
        Synthetic Minority Oversampling Technique (SMOTE).

        Creates synthetic samples by interpolating between neighbors.
        """
        np.random.seed(random_state)

        minority_mask = y == 1
        X_minority = X[minority_mask]
        n_minority = len(X_minority)
        n_majority = (y == 0).sum()

        target_minority = int(n_majority * target_ratio)
        n_synthetic = target_minority - n_minority

        if n_synthetic <= 0:
            return X, y

        # Find k nearest neighbors for each minority sample
        synthetic_samples = []

        for _ in range(n_synthetic):
            # Pick random minority sample
            idx = np.random.randint(n_minority)
            sample = X_minority[idx]

            # Find distances to other minority samples
            distances = np.sqrt(((X_minority - sample) ** 2).sum(axis=1))
            distances[idx] = np.inf  # Exclude self

            # Get k nearest neighbors
            neighbor_indices = np.argsort(distances)[:k_neighbors]
            neighbor_idx = np.random.choice(neighbor_indices)
            neighbor = X_minority[neighbor_idx]

            # Interpolate
            alpha = np.random.random()
            synthetic = sample + alpha * (neighbor - sample)
            synthetic_samples.append(synthetic)

        X_synthetic = np.array(synthetic_samples)
        y_synthetic = np.ones(n_synthetic)

        X_balanced = np.vstack([X, X_synthetic])
        y_balanced = np.hstack([y, y_synthetic])

        # Shuffle
        shuffle_idx = np.random.permutation(len(y_balanced))

        return X_balanced[shuffle_idx], y_balanced[shuffle_idx]

n_legitimate = n_samples - n_fraud
```

## Evaluation Metrics

<Formula>
Precision = TP / (TP + FP) — Of flagged transactions, how many were actually fraud?
</Formula>

<Formula>
Recall = TP / (TP + FN) — Of all fraud, how much did we catch?
</Formula>

### Fraud Detection Metrics

```python
class FraudMetrics:
    """
    Evaluation metrics for fraud detection.
    """

    @staticmethod
    def confusion_matrix(
        y_true: np.ndarray,
        y_pred: np.ndarray
    ) -> Dict[str, int]:
        """Calculate confusion matrix elements."""
        tp = ((y_true == 1) & (y_pred == 1)).sum()
        tn = ((y_true == 0) & (y_pred == 0)).sum()
        fp = ((y_true == 0) & (y_pred == 1)).sum()
        fn = ((y_true == 1) & (y_pred == 0)).sum()

        return {'TP': tp, 'TN': tn, 'FP': fp, 'FN': fn}

    @staticmethod
    def precision(y_true: np.ndarray, y_pred: np.ndarray) -> float:
        """Calculate precision."""
        cm = FraudMetrics.confusion_matrix(y_true, y_pred)
        tp, fp = cm['TP'], cm['FP']
        return tp / (tp + fp) if (tp + fp) > 0 else 0.0

    @staticmethod
    def recall(y_true: np.ndarray, y_pred: np.ndarray) -> float:
        """Calculate recall (sensitivity)."""
        cm = FraudMetrics.confusion_matrix(y_true, y_pred)
        tp, fn = cm['TP'], cm['FN']
        return tp / (tp + fn) if (tp + fn) > 0 else 0.0

    @staticmethod
    def f1_score(y_true: np.ndarray, y_pred: np.ndarray) -> float:
        """Calculate F1 score."""
        p = FraudMetrics.precision(y_true, y_pred)
        r = FraudMetrics.recall(y_true, y_pred)
        return 2 * p * r / (p + r) if (p + r) > 0 else 0.0

    @staticmethod
    def auc_roc(
        y_true: np.ndarray,
        y_scores: np.ndarray
    ) -> float:
        """Calculate AUC-ROC."""
        # Sort by scores descending
        sorted_indices = np.argsort(y_scores)[::-1]
        y_true_sorted = y_true[sorted_indices]

        n_pos = y_true.sum()
        n_neg = len(y_true) - n_pos

        if n_pos == 0 or n_neg == 0:
            return 0.5

        # Calculate AUC using trapezoidal rule
        tpr_prev, fpr_prev = 0.0, 0.0
        auc = 0.0
        tp, fp = 0, 0

        for i, label in enumerate(y_true_sorted):
            if label == 1:
                tp += 1
            else:
                fp += 1

            tpr = tp / n_pos
            fpr = fp / n_neg

            # Trapezoidal area
            auc += (fpr - fpr_prev) * (tpr + tpr_prev) / 2
            tpr_prev, fpr_prev = tpr, fpr

        return auc

    @staticmethod
    def precision_at_k(
        y_true: np.ndarray,
        y_scores: np.ndarray,
        k: int
    ) -> float:
        """
        Precision at top k predictions.

        Useful when you can only investigate k alerts.
        """
        # Get top k predictions
        top_k_indices = np.argsort(y_scores)[-k:]
        top_k_true = y_true[top_k_indices]

        return top_k_true.sum() / k

    @staticmethod
    def cost_based_evaluation(
        y_true: np.ndarray,
        y_pred: np.ndarray,
        fraud_cost: float = 500.0,
        investigation_cost: float = 10.0
    ) -> Dict[str, float]:
        """
        Evaluate model based on business costs.
        """
        cm = FraudMetrics.confusion_matrix(y_true, y_pred)

        # Missed fraud cost
        missed_fraud_cost = cm['FN'] * fraud_cost

        # Investigation cost (all positives)
        investigation_total = (cm['TP'] + cm['FP']) * investigation_cost

        # Savings from caught fraud
        saved_fraud = cm['TP'] * fraud_cost

        total_cost = missed_fraud_cost + investigation_total
        net_savings = saved_fraud - investigation_total

        return {
            'missed_fraud_cost': missed_fraud_cost,
            'investigation_cost': investigation_total,
            'saved_fraud_value': saved_fraud,
            'total_cost': total_cost,
            'net_savings': net_savings,
            'roi': net_savings / investigation_total if investigation_total > 0 else 0
        }
```

## Key Takeaways

1. **Class Imbalance**: Fraud is rare (<1%), making standard metrics like accuracy misleading
2. **Rule-Based Systems**: Fast and interpretable but struggle with novel patterns
3. **Cost Asymmetry**: Missing fraud is much costlier than false positives
4. **Resampling**: SMOTE and other techniques help balance training data
5. **Appropriate Metrics**: Use precision, recall, F1, and AUC rather than accuracy
6. **Concept Drift**: Fraudsters adapt, requiring continuous model updates

Classical methods provide a solid foundation, but quantum computing offers potential advantages for detecting complex fraud patterns through enhanced feature spaces and optimization.
