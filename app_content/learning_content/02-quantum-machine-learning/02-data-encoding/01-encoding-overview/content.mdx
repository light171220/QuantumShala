## 1. The Encoding Problem

Quantum machine learning operates on quantum states, but most real-world data is classical.

<Definition term="Quantum Data Encoding">
The process of mapping classical data x ∈ ℝⁿ to a quantum state |ψ(x)⟩ through a unitary transformation U(x), such that the quantum state contains information about the classical data.
</Definition>

```python
import numpy as np

def encoding_taxonomy():
    """Overview of quantum encoding strategies."""

    encodings = {
        "Basis Encoding": {
            "data_type": "Binary/Integer",
            "qubits": "O(log N) for N values",
            "circuit_depth": "O(log N)",
            "expressivity": "Low",
            "use_case": "Database indexing, Grover's search"
        },
        "Amplitude Encoding": {
            "data_type": "Real/Complex vectors",
            "qubits": "O(log N) for N-dim vector",
            "circuit_depth": "O(N) general, O(poly log N) structured",
            "expressivity": "High",
            "use_case": "Dense data, quantum linear algebra"
        },
        "Angle Encoding": {
            "data_type": "Continuous features",
            "qubits": "O(N) for N features",
            "circuit_depth": "O(N × layers)",
            "expressivity": "Medium-High",
            "use_case": "Variational classifiers, quantum kernels"
        },
        "Kernel Encoding": {
            "data_type": "Continuous features",
            "qubits": "O(N) for N features",

encoding_taxonomy()
```

## 2. Encoding as Feature Maps

Data encoding can be viewed as a feature map from classical to quantum space.

<Formula>
$$\phi: \mathbb{R}^n \rightarrow \mathcal{H}^{2^m}$$
$$x \mapsto |\psi(x)\rangle = U(x)|0\rangle^{\otimes m}$$
</Formula>

```python
import numpy as np

def compare_feature_spaces():
    """Compare classical and quantum feature spaces."""

    print("Feature Space Dimensions:")
    print("=" * 60)
    print()
    print(f"{'Input Dim':<12} {'Classical Poly(d=2)':<20} {'Quantum (m qubits)':<20}")
    print("-" * 60)

    for n in [2, 4, 8, 16]:
        # Classical polynomial features
        classical_poly = int((n * (n + 1)) / 2 + n + 1)  # 1 + n + n(n+1)/2

        # Quantum Hilbert space with n qubits
        quantum_dim = 2 ** n

        print(f"{n:<12} {classical_poly:<20} {quantum_dim:<20,}")

compare_feature_spaces()
```

<Tip>
The exponentially large quantum feature space is a potential source of advantage. With n qubits, you have access to 2ⁿ dimensions - but exploiting this advantage requires careful encoding design.
</Tip>

## 3. The Encoding-Expressivity Trade-off

Different encodings trade off between resources and expressivity.

```python
import numpy as np

def encoding_tradeoffs():
    """Analyze encoding trade-offs."""

    print("Encoding Trade-offs Matrix:")
    print("=" * 70)
    print()
    print(f"{'Encoding':<20} {'Qubits':<12} {'Depth':<12} {'Express.':<12} {'Practical':<12}")
    print("-" * 70)

    encodings = [
        ("Basis", "log N", "log N", "Low", "Easy"),
        ("Amplitude", "log N", "N", "High", "Hard"),
        ("Angle (single)", "N", "N", "Low", "Easy"),
        ("Angle (multi)", "N", "N×L", "High", "Medium"),
        ("Kernel", "N", "N×D", "High", "Medium"),
    ]

    for enc in encodings:
        print(f"{enc[0]:<20} {enc[1]:<12} {enc[2]:<12} {enc[3]:<12} {enc[4]:<12}")

encoding_tradeoffs()
```

<Warning>
There's no universally best encoding. The optimal choice depends on your data structure, available qubits, acceptable circuit depth, and the learning task at hand.
</Warning>

## 4. Encoding Circuit Structures

Common patterns for building encoding circuits.

```python
import numpy as np

def encoding_patterns():
    """Common encoding circuit patterns."""

    patterns = {
        "Rotation Layer": {
            "structure": "Ry(f(x)) or Rz(f(x)) on each qubit",
            "gates": "N rotation gates",
            "entanglement": "None",
            "code": "for i in range(n): circuit.ry(x[i], i)"
        },
        "Entangling Layer": {
            "structure": "CNOT or CZ between adjacent qubits",
            "gates": "N-1 two-qubit gates",
            "entanglement": "Creates correlations",
            "code": "for i in range(n-1): circuit.cx(i, i+1)"
        },
        "Data Re-uploading": {
            "structure": "Multiple encoding layers interleaved with variational",
            "gates": "L × N rotations",
            "entanglement": "Via variational layers",
            "code": "for l in range(L): encode(x); variational(θ[l])"
        },
        "IQP (Instantaneous Quantum Polynomial)": {
            "structure": "H - Rz(x) - ZZ(x) - H",
            "gates": "N + N² interactions",
            "entanglement": "High (all-to-all possible)",
            "code": "H_all(); Rz(x); ZZ(x); H_all()"
        }
    }

    print("Encoding Circuit Patterns:")
    print("=" * 70)

    for name, details in patterns.items():
        print(f"\n{name}:")
        for key, value in details.items():
            print(f"  {key}: {value}")

encoding_patterns()
```
