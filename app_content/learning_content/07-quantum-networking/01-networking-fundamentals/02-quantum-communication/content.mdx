# Quantum Communication Principles

Quantum communication leverages the unique properties of quantum mechanics to transmit information with capabilities impossible in classical systems.

## Quantum vs Classical Information

```
Fundamental Differences:
═══════════════════════════════════════════════════════════════════

Classical Bit:
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│    Classical bit: Either 0 OR 1 (definite state)                │
│                                                                  │
│         ┌───┐         ┌───┐                                     │
│         │ 0 │   OR    │ 1 │                                     │
│         └───┘         └───┘                                     │
│                                                                  │
│    • Can be copied perfectly                                    │
│    • Can be measured without disturbance                        │
│    • Can be broadcast to many receivers                         │
│    • State is objective and deterministic                       │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

Quantum Bit (Qubit):
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│    |ψ⟩ = α|0⟩ + β|1⟩    where |α|² + |β|² = 1                  │
│                                                                  │
│                   |0⟩                                           │
│                    ↑                                            │
│                    │  θ                                         │
│                    │ ╱                                          │
│                    │╱                                           │
│           ────────●──────── |1⟩                                 │
│                  ╱│                                             │
│                 ╱ │                                             │
│                φ  │                                             │
│                                                                  │
│    • Cannot be copied (no-cloning theorem)                      │
│    • Measurement disturbs the state                             │
│    • Cannot be broadcast without destroying                     │
│    • Carries quantum coherence and entanglement                 │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

```python
import numpy as np
from dataclasses import dataclass
from typing import Tuple, Optional, List

@dataclass
class Qubit:
    """
    Representation of a quantum bit.
    """
    alpha: complex  # Amplitude for |0⟩
    beta: complex   # Amplitude for |1⟩

    def __post_init__(self):
        # Ensure normalization
        norm = np.sqrt(abs(self.alpha)**2 + abs(self.beta)**2)
        if not np.isclose(norm, 1.0):
            self.alpha /= norm
            self.beta /= norm

    @property
    def state_vector(self) -> np.ndarray:
        """Return state as column vector."""
        return np.array([[self.alpha], [self.beta]], dtype=complex)

    @classmethod
    def from_bloch(cls, theta: float, phi: float) -> 'Qubit':
        """
        Create qubit from Bloch sphere coordinates.

        Args:
            theta: Polar angle (0 to π)
            phi: Azimuthal angle (0 to 2π)
        """
        alpha = np.cos(theta / 2)
        beta = np.exp(1j * phi) * np.sin(theta / 2)
        return cls(alpha, beta)

    def measure(self, basis: str = "computational") -> Tuple[int, 'Qubit']:
        """
        Measure the qubit, returning outcome and collapsed state.

        Args:
            basis: "computational" (Z), "hadamard" (X), or "circular" (Y)
        """
        if basis == "computational":
            prob_0 = abs(self.alpha)**2
            outcome = 0 if np.random.random() < prob_0 else 1
            collapsed = Qubit(1, 0) if outcome == 0 else Qubit(0, 1)
        elif basis == "hadamard":
            # Transform to X basis, then measure
            plus = (self.alpha + self.beta) / np.sqrt(2)
            minus = (self.alpha - self.beta) / np.sqrt(2)
            prob_plus = abs(plus)**2
            outcome = 0 if np.random.random() < prob_plus else 1
            collapsed = Qubit(1/np.sqrt(2), 1/np.sqrt(2)) if outcome == 0 \
                       else Qubit(1/np.sqrt(2), -1/np.sqrt(2))
        else:
            raise ValueError(f"Unknown basis: {basis}")

        return outcome, collapsed

    def fidelity(self, other: 'Qubit') -> float:
        """Calculate fidelity with another qubit state."""
        inner_product = np.conj(self.alpha) * other.alpha + \
                       np.conj(self.beta) * other.beta
        return abs(inner_product)**2

class NoCloning:
    """
    Demonstration of the no-cloning theorem.
    """

    @staticmethod
    def attempt_clone(qubit: Qubit) -> Tuple[Qubit, str]:
        """
        Attempt to clone a qubit (impossible in general!).

        This demonstrates why cloning fails.
        """
        # A hypothetical cloning operation would need:
        # U|ψ⟩|0⟩ = |ψ⟩|ψ⟩ for all |ψ⟩

        # But this violates linearity of quantum mechanics:
        # If |ψ⟩ = α|0⟩ + β|1⟩, then cloning would require:
        # U(α|0⟩ + β|1⟩)|0⟩ = (α|0⟩ + β|1⟩)(α|0⟩ + β|1⟩)
        #                   = α²|00⟩ + αβ|01⟩ + αβ|10⟩ + β²|11⟩

        # But linearity gives:
        # αU|0⟩|0⟩ + βU|1⟩|0⟩ = α|00⟩ + β|11⟩

        # These are DIFFERENT unless α=0 or β=0

        explanation = """
NO-CLONING THEOREM:
═══════════════════════════════════════════════════════════════

It is impossible to create an identical copy of an arbitrary
unknown quantum state.

Proof sketch:
1. Assume cloning unitary U exists: U|ψ⟩|0⟩ = |ψ⟩|ψ⟩

2. For |ψ⟩ = α|0⟩ + β|1⟩:
   U(α|0⟩ + β|1⟩)|0⟩ should give (α|0⟩ + β|1⟩)⊗(α|0⟩ + β|1⟩)
                                = α²|00⟩ + αβ|01⟩ + αβ|10⟩ + β²|11⟩

3. But by linearity:
   αU|0⟩|0⟩ + βU|1⟩|0⟩ = α|00⟩ + β|11⟩

4. These expressions are different! Contradiction.

Implications:
• Eavesdropping disturbs quantum states → detectable
• Quantum information cannot be broadcast
• Quantum error correction requires entanglement
"""
        # Return original qubit unchanged
        return qubit, explanation

# Demonstrate no-cloning
q = Qubit.from_bloch(np.pi/4, np.pi/3)
print(f"Original qubit: α={q.alpha:.3f}, β={q.beta:.3f}")

_, explanation = NoCloning.attempt_clone(q)
print(explanation)
```

## Quantum Channels

```
Types of Quantum Channels:
═══════════════════════════════════════════════════════════════════

Ideal Quantum Channel:
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│    |ψ⟩ ─────────────────────────────────────────────▶ |ψ⟩       │
│    Input                                               Output   │
│                                                                  │
│    Perfect transmission: ρ → ρ (identity channel)              │
│    Fidelity = 1.0                                               │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

Depolarizing Channel:
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│    With probability (1-p): state unchanged                      │
│    With probability p/3:   X error (bit flip)                   │
│    With probability p/3:   Y error                              │
│    With probability p/3:   Z error (phase flip)                 │
│                                                                  │
│    ε(ρ) = (1-p)ρ + (p/3)(XρX + YρY + ZρZ)                       │
│                                                                  │
│    At p = 1: completely mixed state I/2 (no info)              │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

Amplitude Damping Channel (Photon Loss):
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│    Models energy dissipation (T₁ decay)                         │
│                                                                  │
│    |1⟩ → √(1-γ)|1⟩ + √γ|0⟩    (excited state decays)           │
│    |0⟩ → |0⟩                   (ground state stable)            │
│                                                                  │
│    γ = 1 - exp(-t/T₁)                                           │
│                                                                  │
│    Critical for photon transmission in fiber!                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

Dephasing Channel:
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│    Models loss of phase coherence (T₂ decay)                    │
│                                                                  │
│    ε(ρ) = (1-p)ρ + p·ZρZ                                        │
│                                                                  │
│    Off-diagonal elements of ρ decay:                            │
│    ρ₀₁ → (1-2p)ρ₀₁                                              │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

```python
class QuantumChannel:
    """
    Quantum channel models for communication.
    """

    def __init__(self, channel_type: str, parameter: float):
        """
        Initialize quantum channel.

        Args:
            channel_type: "depolarizing", "amplitude_damping", "dephasing"
            parameter: Channel error parameter (interpretation varies)
        """
        self.channel_type = channel_type
        self.parameter = parameter

        # Pauli matrices
        self.I = np.array([[1, 0], [0, 1]], dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

    def apply(self, rho: np.ndarray) -> np.ndarray:
        """
        Apply channel to density matrix.

        Args:
            rho: 2x2 density matrix
        """
        if self.channel_type == "depolarizing":
            return self._depolarizing(rho)
        elif self.channel_type == "amplitude_damping":
            return self._amplitude_damping(rho)
        elif self.channel_type == "dephasing":
            return self._dephasing(rho)
        else:
            raise ValueError(f"Unknown channel: {self.channel_type}")

    def _depolarizing(self, rho: np.ndarray) -> np.ndarray:
        """Depolarizing channel."""
        p = self.parameter
        return (1 - p) * rho + (p / 3) * (
            self.X @ rho @ self.X +
            self.Y @ rho @ self.Y +
            self.Z @ rho @ self.Z
        )

    def _amplitude_damping(self, rho: np.ndarray) -> np.ndarray:
        """Amplitude damping channel (photon loss)."""
        gamma = self.parameter
        K0 = np.array([[1, 0], [0, np.sqrt(1 - gamma)]], dtype=complex)
        K1 = np.array([[0, np.sqrt(gamma)], [0, 0]], dtype=complex)
        return K0 @ rho @ K0.conj().T + K1 @ rho @ K1.conj().T

    def _dephasing(self, rho: np.ndarray) -> np.ndarray:
        """Dephasing channel."""
        p = self.parameter
        return (1 - p) * rho + p * self.Z @ rho @ self.Z

    def channel_fidelity(self, input_state: np.ndarray) -> float:
        """
        Calculate fidelity between input and output states.
        """
        output = self.apply(input_state)

        # Fidelity for pure input state
        if np.allclose(input_state @ input_state, input_state):
            return np.real(np.trace(input_state @ output))

        # General fidelity (simplified)
        return np.real(np.trace(np.sqrt(
            np.sqrt(input_state) @ output @ np.sqrt(input_state)
        )))**2

class FiberChannel(QuantumChannel):
    """
    Optical fiber quantum channel model.
    """

    def __init__(self, length_km: float,
                 attenuation_db_per_km: float = 0.2):
        """
        Initialize fiber channel.

        Args:
            length_km: Fiber length in kilometers
            attenuation_db_per_km: Loss coefficient (0.2 for SMF at 1550nm)
        """
        self.length_km = length_km
        self.attenuation = attenuation_db_per_km

        # Calculate transmission probability
        total_loss_db = attenuation_db_per_km * length_km
        self.transmission_prob = 10 ** (-total_loss_db / 10)

        # Model as amplitude damping with γ = 1 - η
        super().__init__("amplitude_damping", 1 - self.transmission_prob)

    def photon_survives(self) -> bool:
        """Simulate whether a photon survives transmission."""
        return np.random.random() < self.transmission_prob

    def expected_key_rate(self, source_rate_hz: float) -> float:
        """
        Estimate QKD key rate through this fiber.

        Args:
            source_rate_hz: Photon source rate
        """
        # Simplified model: key rate ∝ η × source_rate
        # Real systems have additional factors (detector efficiency, etc.)
        return source_rate_hz * self.transmission_prob * 0.5  # 50% basis match

    print(f"  Expected key rate (1MHz source): {fiber.expected_key_rate(1e6):.0f} Hz")
```

## Quantum Teleportation

```
Quantum Teleportation Protocol:
═══════════════════════════════════════════════════════════════════

Alice wants to send |ψ⟩ = α|0⟩ + β|1⟩ to Bob without sending the qubit.

Prerequisites:
• Alice and Bob share entangled pair: |Φ⁺⟩ = (|00⟩ + |11⟩)/√2
• Classical communication channel

Protocol Steps:
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  Alice                                          Bob             │
│    │                                              │              │
│    │  1. Alice has |ψ⟩ and half of |Φ⁺⟩           │              │
│    │                                              │              │
│    ├────────────── EPR pair shared ──────────────┤              │
│    │     A                                B       │              │
│    │                                              │              │
│    │  2. Alice performs Bell measurement          │              │
│    │     on |ψ⟩ and her half of EPR               │              │
│    │                                              │              │
│    │     Gets result: 00, 01, 10, or 11          │              │
│    │                                              │              │
│    │  3. Alice sends 2 classical bits ──────────▶│              │
│    │                                              │              │
│    │  4. Bob applies correction:                  │              │
│    │     00 → I (do nothing)                      │              │
│    │     01 → X (bit flip)                        │              │
│    │     10 → Z (phase flip)                      │              │
│    │     11 → XZ (both)                           │              │
│    │                                              │              │
│    │  5. Bob now has |ψ⟩!                         │              │
│    │                                              │              │
└─────────────────────────────────────────────────────────────────┘

Key Points:
• No faster-than-light communication (classical bits required!)
• Original state |ψ⟩ is destroyed at Alice (no cloning)
• Entanglement is consumed (one use per teleportation)
```

```python
class QuantumTeleportation:
    """
    Quantum teleportation protocol implementation.
    """

    def __init__(self):
        # Bell states
        self.bell_states = {
            "Φ+": np.array([1, 0, 0, 1]) / np.sqrt(2),
            "Φ-": np.array([1, 0, 0, -1]) / np.sqrt(2),
            "Ψ+": np.array([0, 1, 1, 0]) / np.sqrt(2),
            "Ψ-": np.array([0, 1, -1, 0]) / np.sqrt(2),
        }

        # Pauli corrections
        self.corrections = {
            (0, 0): np.eye(2),                    # I
            (0, 1): np.array([[0, 1], [1, 0]]),   # X
            (1, 0): np.array([[1, 0], [0, -1]]),  # Z
            (1, 1): np.array([[0, -1], [1, 0]]),  # XZ
        }

    def prepare_epr_pair(self) -> np.ndarray:
        """Create Bell state |Φ⁺⟩ = (|00⟩ + |11⟩)/√2."""
        return self.bell_states["Φ+"].copy()

    def bell_measurement(self, state_to_send: np.ndarray,
                        alice_epr: np.ndarray) -> Tuple[int, int]:
        """
        Perform Bell measurement on Alice's qubits.

        Returns:
            Tuple of classical bits (2 bits)
        """
        # In real implementation, this would be a projective measurement
        # onto Bell basis. Here we simulate the probabilistic outcome.

        # Combine state_to_send with alice's half of EPR
        # Full 3-qubit state: |ψ⟩_1 ⊗ |Φ⁺⟩_23

        # Bell measurement on qubits 1 and 2 gives one of four outcomes
        # with equal probability for arbitrary |ψ⟩
        outcome = np.random.randint(0, 4)
        bit1 = outcome // 2
        bit2 = outcome % 2

        return (bit1, bit2)

    def apply_correction(self, bob_qubit: np.ndarray,
                        classical_bits: Tuple[int, int]) -> np.ndarray:
        """
        Bob applies Pauli correction based on Alice's measurement.
        """
        correction = self.corrections[classical_bits]
        return correction @ bob_qubit

    def teleport(self, state: Qubit, verbose: bool = True) -> Qubit:
        """
        Teleport a qubit state.

        Args:
            state: Qubit to teleport
            verbose: Print protocol steps

        Returns:
            Teleported qubit at Bob's location
        """
        if verbose:
            print("QUANTUM TELEPORTATION PROTOCOL")
            print("=" * 50)
            print(f"\nOriginal state: α={state.alpha:.3f}, β={state.beta:.3f}")

        # Step 1: Prepare EPR pair
        epr = self.prepare_epr_pair()
        if verbose:
            print("\n1. EPR pair |Φ⁺⟩ shared between Alice and Bob")

        # Step 2: Bell measurement at Alice
        classical_bits = self.bell_measurement(state.state_vector, epr)
        if verbose:
            print(f"\n2. Alice's Bell measurement result: {classical_bits}")
            print("   (Alice sends these 2 classical bits to Bob)")

        # Step 3: Bob's initial state (after Alice's measurement)
        # Due to entanglement, Bob's state is related to |ψ⟩
        # but needs correction based on measurement outcome
        bob_state = state.state_vector.flatten()

        # Step 4: Apply correction
        corrected = self.apply_correction(bob_state, classical_bits)
        if verbose:
            correction_name = ["I", "X", "Z", "XZ"][classical_bits[0]*2 + classical_bits[1]]
            print(f"\n3. Bob applies correction: {correction_name}")

        result = Qubit(corrected[0], corrected[1])
        if verbose:
            print(f"\n4. Bob's final state: α={result.alpha:.3f}, β={result.beta:.3f}")
            print(f"\nFidelity with original: {state.fidelity(result):.4f}")

        return result

# Demonstrate teleportation
teleporter = QuantumTeleportation()
original = Qubit.from_bloch(np.pi/3, np.pi/4)  # Random state
teleported = teleporter.teleport(original)
```

## Summary

Quantum communication fundamentals include:

1. **Quantum vs Classical**: Qubits carry superposition; cannot be copied
2. **No-cloning theorem**: Fundamental limit enabling security
3. **Quantum channels**: Model noise and loss in transmission
4. **Teleportation**: Transfer quantum states using entanglement + classical bits

These principles underpin all quantum networking protocols including QKD.
