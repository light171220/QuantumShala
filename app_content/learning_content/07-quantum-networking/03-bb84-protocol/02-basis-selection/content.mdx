# Basis Selection and Encoding

The security of BB84 fundamentally depends on the proper selection and use of conjugate measurement bases. This lesson explores the mathematical properties of conjugate bases, practical encoding schemes, and the critical requirements for random basis selection.

## Conjugate Bases Theory

Conjugate (or mutually unbiased) bases are the mathematical heart of BB84 security.

### Mutually Unbiased Bases

```
┌─────────────────────────────────────────────────────────────────┐
│              Mutually Unbiased Bases (MUBs)                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Definition: Two orthonormal bases {|a_i⟩} and {|b_j⟩} are   │
│   mutually unbiased if:                                         │
│                                                                 │
│           |⟨a_i|b_j⟩|² = 1/d   for all i,j                    │
│                                                                 │
│   where d is the dimension of the Hilbert space                 │
│                                                                 │
│   For qubits (d=2):  |⟨a_i|b_j⟩|² = 1/2                       │
│                                                                 │
│   BB84 BASES:                                                   │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                         │  │
│   │   Z-basis: {|0⟩, |1⟩}                                   │  │
│   │   X-basis: {|+⟩, |-⟩}                                   │  │
│   │                                                         │  │
│   │   Overlaps:                                             │  │
│   │   |⟨0|+⟩|² = |⟨0|-⟩|² = |⟨1|+⟩|² = |⟨1|-⟩|² = 1/2      │  │
│   │                                                         │  │
│   │   Y-basis (for six-state): {|R⟩, |L⟩}                  │  │
│   │   |R⟩ = (|0⟩ + i|1⟩)/√2                                │  │
│   │   |L⟩ = (|0⟩ - i|1⟩)/√2                                │  │
│   │                                                         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   PHYSICAL MEANING:                                             │
│   A state prepared in one basis appears completely random       │
│   when measured in a conjugate basis.                           │
│                                                                 │
│   This is the quantum property that enables secure key          │
│   distribution - eavesdropping causes disturbance.              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Mathematical Framework

```python
import numpy as np
from typing import List, Tuple, Dict
from dataclasses import dataclass

class MutuallyUnbiasedBases:
    """
    Mathematical framework for mutually unbiased bases.
    """

    def __init__(self, dimension: int = 2):
        self.d = dimension

    @staticmethod
    def pauli_matrices() -> Dict[str, np.ndarray]:
        """Return Pauli matrices."""
        return {
            'I': np.array([[1, 0], [0, 1]], dtype=complex),
            'X': np.array([[0, 1], [1, 0]], dtype=complex),
            'Y': np.array([[0, -1j], [1j, 0]], dtype=complex),
            'Z': np.array([[1, 0], [0, -1]], dtype=complex)
        }

    @staticmethod
    def compute_bases() -> Dict[str, List[np.ndarray]]:
        """
        Compute the three MUBs for a qubit.

        These correspond to eigenstates of Pauli X, Y, Z.
        """
        # Z-basis (computational)
        z_basis = [
            np.array([1, 0], dtype=complex),  # |0⟩
            np.array([0, 1], dtype=complex)   # |1⟩
        ]

        # X-basis (Hadamard)
        x_basis = [
            np.array([1, 1], dtype=complex) / np.sqrt(2),   # |+⟩
            np.array([1, -1], dtype=complex) / np.sqrt(2)   # |-⟩
        ]

        # Y-basis (circular)
        y_basis = [
            np.array([1, 1j], dtype=complex) / np.sqrt(2),  # |R⟩
            np.array([1, -1j], dtype=complex) / np.sqrt(2)  # |L⟩
        ]

        return {'Z': z_basis, 'X': x_basis, 'Y': y_basis}

    def verify_mub(self, basis1: List[np.ndarray],
                   basis2: List[np.ndarray]) -> bool:
        """Verify two bases are mutually unbiased."""
        expected_overlap = 1 / self.d

        for v1 in basis1:
            for v2 in basis2:
                overlap = abs(np.vdot(v1, v2)) ** 2
                if not np.isclose(overlap, expected_overlap):
                    return False
        return True

    def hadamard_transform(self, state: np.ndarray) -> np.ndarray:
        """
        Apply Hadamard gate - transforms between Z and X bases.

        H|0⟩ = |+⟩
        H|1⟩ = |-⟩
        H|+⟩ = |0⟩
        H|-⟩ = |1⟩
        """
        H = np.array([[1, 1], [1, -1]], dtype=complex) / np.sqrt(2)
        return H @ state

    def measurement_probabilities(self, state: np.ndarray,
                                  basis: str) -> Dict[str, float]:
        """
        Calculate measurement probabilities in given basis.
        """
        bases = self.compute_bases()
        basis_states = bases[basis]

        probs = {}
        labels = {
            'Z': ['|0⟩', '|1⟩'],
            'X': ['|+⟩', '|-⟩'],
            'Y': ['|R⟩', '|L⟩']
        }

        for i, basis_state in enumerate(basis_states):
            prob = abs(np.vdot(basis_state, state)) ** 2
            probs[labels[basis][i]] = prob

        return probs

# Demonstrate MUB properties
mub = MutuallyUnbiasedBases()
bases = mub.compute_bases()

print("Verifying mutual unbiasedness:")
for b1_name, b1 in bases.items():
    for b2_name, b2 in bases.items():
        if b1_name != b2_name:
            is_mub = mub.verify_mub(b1, b2)
            print(f"  {b1_name} and {b2_name}: {'MUB ✓' if is_mub else 'Not MUB ✗'}")

# Show measurement in wrong basis
print("\n|0⟩ measured in different bases:")
state_0 = bases['Z'][0]
for basis_name in ['Z', 'X', 'Y']:
    probs = mub.measurement_probabilities(state_0, basis_name)
    print(f"  {basis_name}: {probs}")
```

### Phase Encoding

```
┌─────────────────────────────────────────────────────────────────┐
│                    Phase Encoding                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Preferred for fiber-based QKD (stable over long distances)   │
│                                                                 │
│   PRINCIPLE:                                                    │
│   Use relative phase between two time bins or paths            │
│                                                                 │
│   TIME-BIN ENCODING:                                            │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                         │  │
│   │   |early⟩ ──┬── Δt ──┬── |late⟩                        │  │
│   │             │        │                                  │  │
│   │   Qubit state = α|early⟩ + e^(iφ)|late⟩                │  │
│   │                                                         │  │
│   │   Z-basis: φ = 0 (|0⟩) or φ = π (|1⟩)                  │  │
│   │   X-basis: φ = π/2 (|+⟩) or φ = 3π/2 (|-⟩)            │  │
│   │                                                         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   INTERFEROMETRIC IMPLEMENTATION:                               │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                         │  │
│   │   Alice                     Bob                         │  │
│   │   ┌─────┐                  ┌─────┐                     │  │
│   │   │     │──── fiber ────── │     │                     │  │
│   │   │ MZI │                  │ MZI │                     │  │
│   │   │     │                  │     │                     │  │
│   │   └──┬──┘                  └──┬──┘                     │  │

│   │    φ_A                      φ_B                        │  │
│   │                                                         │  │
│   │   Detection depends on φ_A - φ_B                       │  │
│   │   Same basis when (φ_A - φ_B) = 0 or π                │  │
│   │                                                         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   ADVANTAGES:                                                   │
│   • Stable in fiber (immune to polarization drift)             │
│   • Long-distance performance                                  │
│                                                                 │
│   CHALLENGES:                                                   │
│   • Phase stability required                                   │
│   • More complex setup                                         │
```

### Encoding Implementation

```python
from enum import Enum, auto

class EncodingScheme(Enum):
    POLARIZATION = auto()
    PHASE = auto()
    TIME_BIN = auto()

@dataclass
class PhysicalState:
    """Physical realization of a BB84 state."""
    bit: int
    basis_index: int  # 0 = Z/rectilinear, 1 = X/diagonal
    encoding: EncodingScheme

    def polarization_angle(self) -> float:
        """Get polarization angle in degrees."""
        if self.encoding != EncodingScheme.POLARIZATION:
            raise ValueError("Not polarization encoded")

        angles = {
            (0, 0): 0,    # |0⟩ → H → 0°
            (1, 0): 90,   # |1⟩ → V → 90°
            (0, 1): 45,   # |+⟩ → D → +45°
            (1, 1): -45   # |-⟩ → A → -45°
        }
        return angles[(self.bit, self.basis_index)]

    def phase_value(self) -> float:
        """Get phase value in radians."""
        phases = {
            (0, 0): 0,           # |0⟩ → 0
            (1, 0): np.pi,       # |1⟩ → π
            (0, 1): np.pi/2,     # |+⟩ → π/2
            (1, 1): 3*np.pi/2    # |-⟩ → 3π/2
        }
        return phases[(self.bit, self.basis_index)]

class EncodingComparison:
    """Compare different encoding schemes."""

    @staticmethod
    def compare() -> Dict:
        return {
            'polarization': {
                'stability': 'Poor in fiber, good in free-space',
                'distance': 'Limited by polarization mode dispersion',
                'complexity': 'Simple optical components',
                'applications': 'Free-space, satellite, short fiber'
            },
            'phase': {
                'stability': 'Excellent in fiber',
                'distance': 'Long-distance capable',
                'complexity': 'Interferometer required',
                'applications': 'Fiber networks, metro/backbone'
            },
            'time_bin': {
                'stability': 'Good, uses single path',
                'distance': 'Very long distance',
                'complexity': 'Requires precise timing',
                'applications': 'Long-haul fiber, research'
            }
        }
```

## Random Basis Selection

Proper randomness is critical for BB84 security.

### Randomness Requirements

```
┌─────────────────────────────────────────────────────────────────┐
│              Randomness Requirements for BB84                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   WHAT MUST BE RANDOM:                                          │
│   1. Alice's bit values (key material)                         │
│   2. Alice's basis choices                                     │
│   3. Bob's basis choices                                       │
│   4. Privacy amplification seed                                │
│                                                                 │
│   SECURITY IMPLICATIONS:                                        │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                         │  │
│   │ If Eve knows basis choices in advance:                  │  │
│   │                                                         │  │
│   │   Eve intercepts → measures in correct basis →          │  │
│   │   learns bit → resends identical state →                │  │
│   │   ZERO QBER while learning entire key!                  │  │
│   │                                                         │  │
│   │ Therefore: Basis choices MUST be truly random and      │  │
│   │            unknown to Eve until after measurement       │  │
│   │                                                         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   SOURCES OF RANDOMNESS:                                        │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                         │  │
│   │ ✓ QRNG (Quantum Random Number Generator)               │  │
│   │   - Vacuum fluctuations                                 │  │
│   │   - Photon path at beam splitter                       │  │
│   │   - Certifiably random by physics                      │  │
│   │                                                         │  │
│   │ ✗ PRNG (Pseudorandom Number Generator)                 │  │
│   │   - Deterministic algorithm                             │  │
│   │   - Could be predicted with enough information         │  │
│   │   - May weaken security                                 │  │
│   │                                                         │  │
│   │ ? TRNG (True Random Number Generator - classical)      │  │
│   │   - Thermal noise, radioactive decay                   │  │
│   │   - Usually acceptable but not certifiable             │  │
│   │                                                         │  │
```

### Basis Selection Implementation

```python
class BasisSelector:
    """
    Secure basis selection for BB84.
    """

    def __init__(self, use_qrng: bool = True):
        self.use_qrng = use_qrng
        self.qrng = QuantumRandomGenerator() if use_qrng else None

    def select_bases(self, n: int) -> np.ndarray:
        """
        Select n random bases for BB84.

        Returns array of 0 (Z-basis) or 1 (X-basis).
        """
        if self.use_qrng:
            return self.qrng.generate_bits(n)
        else:
            # Fallback to numpy (NOT recommended for production)
            return np.random.randint(0, 2, n)

    def select_bits_and_bases(self, n: int) -> Tuple[np.ndarray, np.ndarray]:
        """Select both bits and bases."""
        bits = self.select_bases(n)
        bases = self.select_bases(n)
        return bits, bases

class QuantumRandomGenerator:
    """
    Simulated quantum random number generator.

    In real systems, this would interface with hardware QRNG.
    """

    def __init__(self, entropy_source: str = 'vacuum'):
        self.entropy_source = entropy_source
        self.bits_generated = 0

    def generate_bits(self, n: int) -> np.ndarray:
        """Generate n random bits."""
        # Simulate quantum randomness
        # Real QRNG would use actual quantum measurements

        if self.entropy_source == 'vacuum':
            # Simulate homodyne measurement of vacuum
            samples = np.random.normal(0, 1, n)
            bits = (samples > 0).astype(int)

        elif self.entropy_source == 'beamsplitter':
            # Simulate single-photon at 50:50 beamsplitter
            bits = np.random.randint(0, 2, n)

        else:
            bits = np.random.randint(0, 2, n)

        self.bits_generated += n
        return bits

    def test_randomness(self, sample_size: int = 10000) -> Dict:
        """
        Test randomness quality.
        """
        bits = self.generate_bits(sample_size)

        # Frequency test
        ones = np.sum(bits)
        zeros = sample_size - ones
        frequency_bias = abs(ones - zeros) / sample_size

        # Runs test (consecutive same values)
        runs = 1 + np.sum(bits[:-1] != bits[1:])
        expected_runs = (sample_size + 1) / 2
        runs_deviation = abs(runs - expected_runs) / expected_runs

        # Autocorrelation
        autocorr = np.corrcoef(bits[:-1], bits[1:])[0, 1]

        return {
            'sample_size': sample_size,
            'ones_fraction': ones / sample_size,
            'frequency_bias': frequency_bias,
            'runs': runs,
            'runs_deviation': runs_deviation,
            'autocorrelation': autocorr,
            'passes_basic_tests': (
                frequency_bias < 0.02 and
                runs_deviation < 0.1 and
                abs(autocorr) < 0.02
            )
        }

# Test QRNG quality
qrng = QuantumRandomGenerator()
results = qrng.test_randomness(100000)
print("QRNG Quality Test:")
for key, value in results.items():
    print(f"  {key}: {value}")
```

## Basis Reconciliation

After measurements, Alice and Bob must reconcile their basis choices.

```python
class BasisReconciliation:
    """
    Basis reconciliation phase of BB84.
    """

    def __init__(self):
        self.alice_bases = None
        self.bob_bases = None
        self.matching_indices = None

    def reconcile(self, alice_bases: np.ndarray,
                  bob_bases: np.ndarray) -> Dict:
        """
        Perform basis reconciliation.

        In practice, this happens over authenticated classical channel.
        """
        self.alice_bases = alice_bases
        self.bob_bases = bob_bases

        # Find matching basis positions
        self.matching_indices = np.where(alice_bases == bob_bases)[0]

        # Statistics
        n_total = len(alice_bases)
        n_matching = len(self.matching_indices)

        return {
            'total_transmitted': n_total,
            'matching_bases': n_matching,
            'sifting_efficiency': n_matching / n_total,
            'matching_indices': self.matching_indices,
            'discarded': n_total - n_matching
        }

    def sift_key(self, alice_bits: np.ndarray,
                 bob_results: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
        """Extract sifted key from matching-basis measurements."""
        if self.matching_indices is None:
            raise ValueError("Must reconcile bases first")

        sifted_alice = alice_bits[self.matching_indices]
        sifted_bob = bob_results[self.matching_indices]

        return sifted_alice, sifted_bob

    @staticmethod
    def simulate_classical_communication() -> str:
        """Describe classical communication for reconciliation."""
        return """
        CLASSICAL CHANNEL COMMUNICATION:

        1. Alice → Bob: "My bases were: Z, X, Z, X, Z, Z, X, ..."
           (Can be public - doesn't reveal bit values)

        2. Bob compares with his bases

        3. Bob → Alice: "Keep positions: 1, 3, 4, 7, 8, ..."
           (Positions where bases matched)

        4. Both discard non-matching positions

        SECURITY NOTE:
        - This communication MUST be authenticated
        - If Eve can modify messages, she can mount
          man-in-the-middle attack
        - Pre-shared secret or public key auth required
        """
```

## Summary

Proper basis selection and encoding are fundamental to BB84 security:

- **Conjugate bases**: Z and X bases are mutually unbiased, providing security
- **Physical encoding**: Polarization (free-space) or phase (fiber) encoding
- **Random selection**: True quantum randomness essential for security
- **Reconciliation**: Classical communication reveals bases but not bits

Understanding these concepts is crucial for implementing secure BB84 systems.
