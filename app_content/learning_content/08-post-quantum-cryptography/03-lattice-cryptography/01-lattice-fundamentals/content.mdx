## 1. What is a Lattice?

<Definition term="Lattice">
A lattice $\mathcal{L}$ is a discrete additive subgroup of $\mathbb{R}^n$. Equivalently, it is the set of all integer linear combinations of a set of linearly independent vectors $\mathbf{b}_1, \mathbf{b}_2, \ldots, \mathbf{b}_n \in \mathbb{R}^n$:

$$\mathcal{L}(\mathbf{B}) = \left\{ \sum_{i=1}^{n} z_i \mathbf{b}_i : z_i \in \mathbb{Z} \right\}$$
</Definition>

The matrix $\mathbf{B} = [\mathbf{b}_1 | \mathbf{b}_2 | \cdots | \mathbf{b}_n]$ whose columns are the basis vectors is called a **basis** for the lattice.

### Visual Intuition


### Mathematical Properties

<Formula>
$$\text{Key Properties:}$$
$$1. \text{ Discrete: Points are isolated (minimum distance } > 0\text{)}$$
$$2. \text{ Infinite: Extends infinitely in all directions}$$
$$3. \text{ Periodic: } \mathbf{v} \in \mathcal{L} \Rightarrow \mathbf{v} + \mathbf{b}_i \in \mathcal{L}$$
$$4. \text{ Group: Closed under addition and subtraction}$$
</Formula>

```python
import numpy as np
import matplotlib.pyplot as plt
from typing import List, Tuple

class Lattice:
    """
    A class representing a mathematical lattice.
    
    A lattice is defined by its basis matrix B, where each column
    is a basis vector. The lattice consists of all integer linear
    combinations of these basis vectors.
    """
    
    def __init__(self, basis: np.ndarray):
        """
        Initialize lattice with basis matrix.
        
        Args:
            basis: n×n matrix where columns are basis vectors
        """
        self.basis = np.array(basis, dtype=float)
        self.dimension = self.basis.shape[0]
        self.rank = self.basis.shape[1]
        
        # Validate: must be full rank for a proper lattice
        if np.linalg.matrix_rank(self.basis) != self.rank:
            raise ValueError("Basis vectors must be linearly independent")
    
    def generate_point(self, coefficients: np.ndarray) -> np.ndarray:
        """
        Generate a lattice point from integer coefficients.
        
        Args:
            coefficients: Integer vector z
        
        Returns:
            Lattice point B·z
        """
        return self.basis @ coefficients
    
    def enumerate_points(self, bound: int) -> List[np.ndarray]:
        """
        Enumerate all lattice points with coefficients in [-bound, bound].
        
        This is useful for visualization and analysis of small lattices.
        """
        points = []
        ranges = [range(-bound, bound + 1) for _ in range(self.rank)]
        
        # Generate all coefficient combinations
        from itertools import product
        for coeffs in product(*ranges):
            point = self.generate_point(np.array(coeffs))
            points.append(point)
        
        return points
    
    def determinant(self) -> float:
        """
        Compute the lattice determinant (volume of fundamental domain).
        
        For full-rank lattices: det(L) = |det(B)|
        """
        if self.dimension == self.rank:
            return abs(np.linalg.det(self.basis))
        else:
            # For non-square bases: sqrt(det(B^T · B))
            return np.sqrt(np.linalg.det(self.basis.T @ self.basis))
    
    def shortest_basis_vector(self) -> Tuple[np.ndarray, float]:
        """
        Find the shortest vector among the basis vectors.
        
        Note: This is NOT necessarily the shortest vector in the lattice!
        """
        norms = [np.linalg.norm(self.basis[:, i]) for i in range(self.rank)]
        min_idx = np.argmin(norms)
        return self.basis[:, min_idx], norms[min_idx]


def demonstrate_lattice_basics():
    """

    """

        [2, 1],
        [0, 2]
    ])
    
    lattice = Lattice(B)
    
    print("LATTICE FUNDAMENTALS DEMONSTRATION")
    print("=" * 50)
    print(f"\nBasis Matrix B:")
    print(B)
    print(f"\nDimension: {lattice.dimension}")
    print(f"Rank: {lattice.rank}")
    print(f"Determinant: {lattice.determinant():.4f}")
    
    # Generate some lattice points
    print("\nSample Lattice Points:")
    for coeffs in [[1, 0], [0, 1], [1, 1], [2, -1], [-1, 3]]:
        z = np.array(coeffs)
        point = lattice.generate_point(z)
        print(f"  {coeffs} → {point}")

    print(f"  v2 = B·[3,-1] = {v2}")
```

## 2. Lattice Bases and Their Properties

A crucial property of lattices: **the same lattice can have infinitely many different bases**.

<Definition term="Unimodular Matrix">
A square integer matrix $\mathbf{U}$ is **unimodular** if $\det(\mathbf{U}) = \pm 1$. Unimodular matrices are exactly the invertible matrices over the integers.
</Definition>

<Formula>
$$\text{If } \mathbf{B} \text{ is a basis for } \mathcal{L}, \text{ then } \mathbf{B}' = \mathbf{B} \cdot \mathbf{U}$$
$$\text{is also a basis for } \mathcal{L} \text{ for any unimodular } \mathbf{U}$$
</Formula>

```python

    print(f"\nOriginal Basis B1 (standard basis):")
    print(B1)
    print(f"det(B1) = {np.linalg.det(B1)}")
    
    print(f"\nUnimodular Matrix U:")
```

### Good vs. Bad Bases

The quality of a basis dramatically affects the difficulty of lattice problems.

<Definition term="Orthogonality Defect">
The orthogonality defect of a basis $\mathbf{B}$ measures how far it is from being orthogonal:

$$\delta(\mathbf{B}) = \frac{\prod_{i=1}^{n} \|\mathbf{b}_i\|}{\det(\mathcal{L})}$$

For an orthogonal basis, $\delta = 1$. Larger values indicate worse bases.
</Definition>

```python
def analyze_basis_quality(B: np.ndarray) -> dict:
    """
    Analyze the quality of a lattice basis.
    
    Returns multiple quality metrics.
    """
    n = B.shape[1]
    
    # Compute norms of basis vectors
    norms = [np.linalg.norm(B[:, i]) for i in range(n)]
    
    # Lattice determinant
    det_L = abs(np.linalg.det(B))
    
    # Orthogonality defect
    product_norms = np.prod(norms)
    orth_defect = product_norms / det_L
    
    # Hadamard ratio (inverse of defect, closer to 1 is better)
    hadamard = det_L / product_norms
    
    # Condition number of basis matrix
    cond = np.linalg.cond(B)
    
    # Gram-Schmidt orthogonalized norms (for more detailed analysis)
    Q, R = np.linalg.qr(B)
    gs_norms = [abs(R[i, i]) for i in range(n)]
    
    return {
        'basis_norms': norms,
        'determinant': det_L,
        'orthogonality_defect': orth_defect,
        'hadamard_ratio': hadamard,
        'condition_number': cond,
        'gram_schmidt_norms': gs_norms,
        'quality': 'good' if hadamard > 0.5 else ('fair' if hadamard > 0.1 else 'poor')
    }


def compare_basis_quality():
```

## 3. The Fundamental Domain

<Definition term="Fundamental Domain">
The **fundamental domain** (or fundamental parallelepiped) of a lattice $\mathcal{L}$ with basis $\mathbf{B}$ is:

$$\mathcal{P}(\mathbf{B}) = \left\{ \sum_{i=1}^{n} x_i \mathbf{b}_i : 0 \leq x_i < 1 \right\}$$

This is the region that, when translated by all lattice vectors, tiles $\mathbb{R}^n$ without overlap.
</Definition>

<Formula>
$$\text{Volume}(\mathcal{P}) = |\det(\mathbf{B})| = \det(\mathcal{L})$$
</Formula>

The determinant is a lattice invariant—it's the same regardless of which basis you use.


## 4. Important Lattice Quantities

### Successive Minima

<Definition term="Successive Minima">
The **i-th successive minimum** $\lambda_i(\mathcal{L})$ is the smallest radius $r$ such that $\mathcal{L}$ contains $i$ linearly independent vectors of length at most $r$:

$$\lambda_i(\mathcal{L}) = \inf\{r : \dim(\text{span}(\mathcal{L} \cap \bar{B}_r)) \geq i\}$$

where $\bar{B}_r$ is the closed ball of radius $r$ centered at origin.
</Definition>

The first successive minimum $\lambda_1$ is the length of the **shortest nonzero vector** in the lattice.

<Formula>
$$\text{Minkowski's First Theorem:}$$
$$\lambda_1(\mathcal{L}) \leq \sqrt{n} \cdot \det(\mathcal{L})^{1/n}$$
</Formula>

```python
def compute_successive_minima_approx(lattice: Lattice, search_bound: int = 10) -> List[float]:
    """
    Approximate successive minima by enumeration (for small lattices only).
    
    Warning: Exponential complexity! Only for demonstration.
    """
    # Generate many lattice points
    points = lattice.enumerate_points(search_bound)
    
    # Compute norms and sort
    point_norms = []
    for p in points:
        norm = np.linalg.norm(p)
        if norm > 1e-10:  # Exclude origin
            point_norms.append((norm, p))
    
    point_norms.sort(key=lambda x: x[0])
    
    # Find successive minima
    successive_minima = []
    found_vectors = []
    
    for norm, vec in point_norms:
        # Check if vec is linearly independent from found vectors
        if len(found_vectors) == 0:
            found_vectors.append(vec)
            successive_minima.append(norm)
        else:
            # Check linear independence
            test_matrix = np.column_stack(found_vectors + [vec])
            if np.linalg.matrix_rank(test_matrix) > len(found_vectors):
                found_vectors.append(vec)
                successive_minima.append(norm)
        
        if len(successive_minima) >= lattice.rank:
            break
    
    return successive_minima


def analyze_lattice_minima():
    """
    Analyze successive minima and compare to theoretical bounds.
    """
    # Create a lattice
    B = np.array([
        [5, 2],
        [1, 4]
    ])
    
    lattice = Lattice(B)
    n = lattice.dimension
    det = lattice.determinant()
    
    print("SUCCESSIVE MINIMA ANALYSIS")
    print("=" * 50)
    
    print(f"\nLattice basis:")
    print(B)

    print(f"Determinant: {det:.4f}")

    # Compute actual successive minima

    print(f"\nComputed successive minima:")
    for i, lam in enumerate(minima, 1):
        print(f"  λ_{i} ≈ {lam:.4f}")
    
    print(f"\nVerification: λ₁ = {minima[0]:.4f} ≤ {minkowski_bound:.4f} ✓")
    
    # Hermite's constant relationship

    gamma_2 = 2 / np.sqrt(3)  # Hermite constant for n=2
```

### The Gaussian Heuristic

<Definition term="Gaussian Heuristic">
For a "random" lattice in dimension $n$ with determinant $\det(\mathcal{L})$, the expected length of the shortest vector is approximately:

$$\lambda_1(\mathcal{L}) \approx \sqrt{\frac{n}{2\pi e}} \cdot \det(\mathcal{L})^{1/n}$$
</Definition>

```python
def gaussian_heuristic(n: int, det: float) -> float:
    """
    Compute the Gaussian heuristic estimate for λ₁.
    """
    return np.sqrt(n / (2 * np.pi * np.e)) * (det ** (1/n))


def compare_heuristics():
```

## 6. Implementing a Complete Lattice Library

```python
class CryptographicLattice(Lattice):
    """
    Extended lattice class with cryptographic operations.
    """
    
    def nearest_plane_decode(self, target: np.ndarray) -> np.ndarray:
        """
        Decode a target vector to the nearest lattice point using
        Babai's nearest plane algorithm.
        
        This is NOT optimal but is efficient: O(n³)
        
        Args:
            target: Vector to decode
        
        Returns:
            Closest lattice point (approximately)
        """
        # Gram-Schmidt orthogonalization
        B = self.basis.copy()
        n = self.rank
        
        # Compute Gram-Schmidt basis
        B_star = np.zeros_like(B)
        mu = np.zeros((n, n))
        
        for i in range(n):
            B_star[:, i] = B[:, i]
            for j in range(i):
                mu[i, j] = np.dot(B[:, i], B_star[:, j]) / np.dot(B_star[:, j], B_star[:, j])
                B_star[:, i] -= mu[i, j] * B_star[:, j]
        
        # Nearest plane algorithm
        b = target.copy()
        coeffs = np.zeros(n)
        
        for i in range(n - 1, -1, -1):
            coeffs[i] = round(np.dot(b, B_star[:, i]) / np.dot(B_star[:, i], B_star[:, i]))
            b -= coeffs[i] * B[:, i]
        
        return self.generate_point(coeffs.astype(int))
    
    def reduce_LLL(self, delta: float = 0.75) -> 'CryptographicLattice':
        """
        Apply LLL basis reduction algorithm.
        
        Returns a new lattice with reduced (shorter, more orthogonal) basis.
        
        Args:
            delta: Lovász condition parameter (0.25 < delta < 1)
        
        Returns:
            New lattice with LLL-reduced basis
        """
        B = self.basis.copy()
        n = self.rank
        
        # Gram-Schmidt
        def gram_schmidt(B):
            B_star = np.zeros_like(B, dtype=float)
            mu = np.zeros((n, n))
            
            for i in range(n):
                B_star[:, i] = B[:, i]
                for j in range(i):
                    mu[i, j] = np.dot(B[:, i], B_star[:, j]) / np.dot(B_star[:, j], B_star[:, j])
                    B_star[:, i] -= mu[i, j] * B_star[:, j]
            
            return B_star, mu
        
        k = 1
        while k < n:
            B_star, mu = gram_schmidt(B)
            
            # Size reduction
            for j in range(k - 1, -1, -1):
                if abs(mu[k, j]) > 0.5:
                    B[:, k] -= round(mu[k, j]) * B[:, j]
                    B_star, mu = gram_schmidt(B)
            
            # Lovász condition
            lhs = np.dot(B_star[:, k], B_star[:, k])
            rhs = (delta - mu[k, k-1]**2) * np.dot(B_star[:, k-1], B_star[:, k-1])
            
            if lhs >= rhs:
                k += 1
            else:
                # Swap b_k and b_{k-1}
                B[:, [k, k-1]] = B[:, [k-1, k]]
                k = max(k - 1, 1)
        
        return CryptographicLattice(B)
```

## Summary

In this lesson, you learned the mathematical foundations of lattices:

1. **Lattices** are discrete additive subgroups of $\mathbb{R}^n$ generated by basis vectors
2. **Multiple bases** generate the same lattice, connected by unimodular transformations
3. **Basis quality** dramatically affects the difficulty of lattice problems
4. **The fundamental domain** has volume equal to the lattice determinant
5. **Successive minima** $\lambda_i$ measure the shortest independent vectors
6. **The Gaussian heuristic** predicts $\lambda_1$ for random lattices
7. **LLL reduction** finds shorter, more orthogonal bases efficiently
8. **Hard problems** (SVP, CVP, LWE) provide cryptographic security

These foundations will be essential for understanding LWE and lattice-based cryptographic constructions in the following lessons.
