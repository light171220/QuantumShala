## 1. The Case for Stateless Signatures

<Definition term="Stateless Signature Scheme">
A signature scheme where the signing algorithm requires no persistent state between signatures. Each signature is independent—signing the same message twice produces different (both valid) signatures.
</Definition>

```python

    │ ✗ Multi-device signing very difficult                              │

    │ Best for: HSMs, controlled environments, firmware signing          │
    └────────────────────────────────────────────────────────────────────┘
    
    ┌────────────────────────────────────────────────────────────────────┐
```

## 3. The Hypertree Structure

<Definition term="Hypertree">
A tree of trees: $d$ layers of XMSS-style Merkle trees stacked on top of each other. Each tree at layer $i$ authenticates the roots of $2^{h'}$ trees at layer $i-1$, where $h' = h/d$ is the height per layer.
</Definition>

```python
class Hypertree:
    """
    SPHINCS+ Hypertree structure.
    
    Instead of one massive tree of height h (requiring 2^h OTS keys),
    use d layers of smaller trees, each of height h' = h/d.
    
    Layer d-1 (top):    One tree, root = public key
    Layer d-2:          2^h' trees (roots signed by layer above)
    ...
    Layer 0 (bottom):   2^((d-1)*h') trees (sign FORS roots)
    """
    
    def __init__(self, h: int, d: int, n: int = 32):
        """
        Args:
            h: Total tree height
            d: Number of layers
            n: Hash output bytes
        """
        self.h = h
        self.d = d
        self.hp = h // d  # Height per layer
        self.n = n
        
        # Virtual tree has 2^h leaves
        self.num_leaves = 2 ** h
        
    def _hash(self, data: bytes) -> bytes:
        return hashlib.sha256(data).digest()[:self.n]
    
    def leaf_to_tree_indices(self, leaf_idx: int) -> List[Tuple[int, int]]:
        """
        Convert global leaf index to (layer, tree_index, leaf_in_tree) for each layer.
        
        Returns list of (tree_index_in_layer, leaf_index_in_tree) from bottom to top.
        """
        indices = []
        idx = leaf_idx
        
        for layer in range(self.d):
            # At this layer, which tree and which leaf?
            leaves_per_tree = 2 ** self.hp
            leaf_in_tree = idx % leaves_per_tree
            tree_idx = idx // leaves_per_tree
            
            indices.append((tree_idx, leaf_in_tree))
            idx = tree_idx  # Move up: tree index becomes leaf index for next layer
        
        return indices
    
    def describe_structure(self):
        """Describe the hypertree structure."""
        print(f"\nHYPERTREE STRUCTURE")
        print("=" * 60)
        print(f"Total height: h = {self.h}")
        print(f"Number of layers: d = {self.d}")
        print(f"Height per layer: h' = {self.hp}")
        print(f"Virtual leaves: 2^{self.h} = {2**self.h if self.h <= 30 else f'2^{self.h}'}")
        
        print(f"\nLayer breakdown:")
        for layer in range(self.d - 1, -1, -1):
            trees_at_layer = 2 ** (layer * self.hp)
            leaves_per_tree = 2 ** self.hp
            
            if layer == self.d - 1:
                role = "(root = public key)"
            elif layer == 0:
                role = "(signs FORS)"
            else:
                role = ""
            
            print(f"  Layer {layer}: {trees_at_layer if trees_at_layer < 1000 else f'2^{layer*self.hp}'} "
                  f"trees × {leaves_per_tree} leaves {role}")
```

## 4. FORS: Few-Time Signature for Messages

<Definition term="FORS (Forest of Random Subsets)">
A few-time signature scheme used in SPHINCS+ to sign the message digest. FORS uses $k$ binary trees of height $a$, selecting one leaf from each tree based on the message hash.

Compared to WOTS+:
- FORS: $O(1)$ hash operations per signature
- WOTS+: $O(w \cdot l)$ hash operations per signature
</Definition>

```python
class FORS:
    """
    FORS: Forest of Random Subsets.
    
    Structure:
    - k trees, each of height a (2^a leaves per tree)
    - Message hash split into k chunks of a bits each
    - Each chunk selects one leaf from corresponding tree
    - Signature reveals k leaves + k authentication paths
    
    Security: Can sign up to ~2^(k*a/2) messages before birthday collision
    """
    
    def __init__(self, k: int, a: int, n: int = 32):
        """
        Args:
            k: Number of trees
            a: Height of each tree (bits per tree)
            n: Hash output bytes
        """
        self.k = k
        self.a = a
        self.n = n
        self.leaves_per_tree = 2 ** a
        self.total_leaves = k * self.leaves_per_tree
    
    def _hash(self, data: bytes) -> bytes:
        return hashlib.sha256(data).digest()[:self.n]
    
    def _prf(self, key: bytes, addr: bytes) -> bytes:
        """PRF for leaf generation."""
        return hashlib.sha256(key + addr).digest()[:self.n]
    
    def _generate_tree(self, sk_seed: bytes, tree_idx: int) -> Tuple[List[bytes], bytes]:
        """
        Generate one FORS tree.
        
        Returns:
            leaves: All 2^a leaf values
            root: Tree root
        """
        # Generate leaves
        leaves = []
        for i in range(self.leaves_per_tree):
            addr = tree_idx.to_bytes(4, 'big') + i.to_bytes(4, 'big')
            leaf = self._prf(sk_seed, addr)
            leaves.append(self._hash(leaf))  # Public leaf = hash of secret
        
        # Build tree
        current_level = leaves
        for _ in range(self.a):
            next_level = []
            for j in range(0, len(current_level), 2):
                parent = self._hash(current_level[j] + current_level[j + 1])
                next_level.append(parent)
            current_level = next_level
        
        return leaves, current_level[0]
    
    def _get_auth_path(self, leaves: List[bytes], leaf_idx: int) -> List[bytes]:
        """Get authentication path for a leaf."""
        path = []
        current_level = leaves
        idx = leaf_idx
        
        for _ in range(self.a):
            sibling_idx = idx ^ 1
            path.append(current_level[sibling_idx])
            
            # Move up
            next_level = []
            for j in range(0, len(current_level), 2):
                parent = self._hash(current_level[j] + current_level[j + 1])
                next_level.append(parent)
            current_level = next_level
            idx //= 2
        
        return path
    
    def keygen(self, seed: bytes = None) -> Tuple[bytes, bytes]:
        """
        Generate FORS keys.
        
        Returns:
            sk_seed: Secret seed
            pk: Public key (hash of all tree roots)
        """
        if seed is None:
            seed = secrets.token_bytes(self.n)
        
        # Compute all tree roots
        roots = []
        for tree_idx in range(self.k):
            _, root = self._generate_tree(seed, tree_idx)
            roots.append(root)
        
        # PK = hash of concatenated roots
        pk = self._hash(b''.join(roots))
        
        return seed, pk
    
    def message_to_indices(self, message_hash: bytes) -> List[int]:
        """
        Convert message hash to k tree indices.
        
        Each index is 'a' bits, selecting a leaf in its tree.
        """
        # Convert hash to bits
        bits = ''.join(format(b, '08b') for b in message_hash)
        
        indices = []
        for i in range(self.k):
            start = i * self.a
            end = start + self.a
            if end <= len(bits):
                idx = int(bits[start:end], 2)
            else:
                idx = 0
            indices.append(idx)
        
        return indices
    
    def sign(self, sk_seed: bytes, message: bytes) -> dict:
        """
        Sign a message with FORS.
        
        Returns signature containing:
        - k secret leaf values
        - k authentication paths
        """
        msg_hash = self._hash(message)
        indices = self.message_to_indices(msg_hash)
        
        signature = {
            'indices': indices,
            'leaves': [],
            'auth_paths': []
        }
        
        for tree_idx in range(self.k):
            leaves, _ = self._generate_tree(sk_seed, tree_idx)
            leaf_idx = indices[tree_idx]
            
            # Get secret leaf value
            addr = tree_idx.to_bytes(4, 'big') + leaf_idx.to_bytes(4, 'big')
            secret_leaf = self._prf(sk_seed, addr)
            signature['leaves'].append(secret_leaf)
            
            # Get auth path
            path = self._get_auth_path(leaves, leaf_idx)
            signature['auth_paths'].append(path)
        
        return signature
    
    def verify(self, pk: bytes, message: bytes, signature: dict) -> bool:
        """Verify a FORS signature."""
        msg_hash = self._hash(message)
        indices = self.message_to_indices(msg_hash)
        
        # Recompute each tree root
        roots = []
        for tree_idx in range(self.k):
            leaf_idx = indices[tree_idx]
            secret_leaf = signature['leaves'][tree_idx]
            auth_path = signature['auth_paths'][tree_idx]
            
            # Compute leaf from secret
            leaf = self._hash(secret_leaf)
            
            # Climb tree
            current = leaf
            idx = leaf_idx
            for sibling in auth_path:
                if idx % 2 == 0:
                    current = self._hash(current + sibling)
                else:
                    current = self._hash(sibling + current)
                idx //= 2
            
            roots.append(current)
        
        # Check PK
        computed_pk = self._hash(b''.join(roots))
        return computed_pk == pk
```

## 5. Complete SPHINCS+ Signing

```python
class SPHINCSPlus:
    """
    Simplified SPHINCS+ implementation.
    
    Full signature process:
    1. Hash message to get random leaf index + FORS message
    2. Sign FORS message → FORS signature
    3. Use WOTS+ to sign FORS public key at each hypertree layer
    4. Include all authentication paths
    """
    
    def __init__(self, variant: str = 'SLH-DSA-SHA2-128s'):
        params = SPHINCSParameters.PARAMS.get(variant, SPHINCSParameters.PARAMS['SLH-DSA-SHA2-128s'])
        
        self.n = params['n']
        self.h = params['h']
        self.d = params['d']
        self.hp = params['hp']
        self.a = params['a']
        self.k = params['k']
        self.w = params['w']
        
        self.variant = variant
    
    def _hash(self, data: bytes) -> bytes:
        return hashlib.sha256(data).digest()[:self.n]
    
    def _prf(self, key: bytes, addr: bytes) -> bytes:
        return hashlib.sha256(key + addr).digest()[:self.n]
    
    def keygen(self) -> Tuple[bytes, bytes]:
        """
        Generate SPHINCS+ key pair.
        
        SK = (SK.seed, SK.prf, PK.seed, PK.root)
        PK = (PK.seed, PK.root)
        """
        sk_seed = secrets.token_bytes(self.n)
        sk_prf = secrets.token_bytes(self.n)
        pk_seed = secrets.token_bytes(self.n)
        
        # Compute root of top hypertree layer
        # (Simplified: just hash the seeds)
        pk_root = self._hash(sk_seed + pk_seed)
        
        sk = sk_seed + sk_prf + pk_seed + pk_root
        pk = pk_seed + pk_root
        
        return sk, pk
    
    def _randomized_message_hash(self, sk_prf: bytes, pk: bytes, 
                                  message: bytes) -> Tuple[bytes, int]:
        """
        Compute randomized message hash.
        
        Returns:
            digest: Hash for FORS
            idx: Leaf index for hypertree
        """
        # R = PRF(SK.prf, OptRand || M)
        opt_rand = secrets.token_bytes(self.n)
        R = self._prf(sk_prf, opt_rand + message)
        
        # digest = H(R || PK || M)
        digest = self._hash(R + pk + message)
        
        # Extract index from digest (h bits)
        idx_bytes = self._hash(digest + b'\x00')
        idx = int.from_bytes(idx_bytes[:8], 'big') % (2 ** self.h)
        
        return digest, idx, R
    
    def sign(self, sk: bytes, message: bytes) -> dict:
        """
        Sign a message.
        
        This is a simplified version showing the structure.
        Real implementation computes all WOTS+ and tree nodes.
        """
        # Parse secret key
        sk_seed = sk[:self.n]
        sk_prf = sk[self.n:2*self.n]
        pk_seed = sk[2*self.n:3*self.n]
        pk_root = sk[3*self.n:]
        pk = pk_seed + pk_root
        
        # Get randomized hash and index
        digest, idx, R = self._randomized_message_hash(sk_prf, pk, message)
        
        # FORS signature on digest
        fors = FORS(self.k, self.a, self.n)
        fors_seed = self._prf(sk_seed, idx.to_bytes(8, 'big') + b'fors')
        fors_sig = fors.sign(fors_seed, digest)
        fors_sk, fors_pk = fors.keygen(fors_seed)
        
        # Hypertree signatures (simplified)
        # In reality: WOTS+ signatures + auth paths for each layer
        ht_sig = {
            'leaf_idx': idx,
            'wots_sigs': [b'wots_placeholder'] * self.d,
            'auth_paths': [b'path_placeholder'] * self.d
        }
        
        return {
            'R': R,
            'fors_sig': fors_sig,
            'fors_pk': fors_pk,
            'ht_sig': ht_sig
        }
    
    def verify(self, pk: bytes, message: bytes, signature: dict) -> bool:
        """
        Verify a signature (simplified).
        """
        pk_seed = pk[:self.n]
        pk_root = pk[self.n:]
        
        R = signature['R']
        
        # Recompute digest
        digest = self._hash(R + pk + message)
        
        # Verify FORS
        fors = FORS(self.k, self.a, self.n)
        if not fors.verify(signature['fors_pk'], digest, signature['fors_sig']):
            return False
        
        # Verify hypertree (simplified - just return True for demo)
        # Real implementation: verify WOTS+ chain and auth paths
        
        return True
    
    def estimate_signature_size(self) -> int:
        """Estimate signature size in bytes."""
        # R randomizer
        r_size = self.n
        
        # FORS signature
        fors_leaves = self.k * self.n  # k secret leaves
        fors_paths = self.k * self.a * self.n  # k paths of a nodes
        fors_size = fors_leaves + fors_paths
        
        # WOTS+ signatures (d layers)
        log_w = int(math.log2(self.w))
        l1 = math.ceil(8 * self.n / log_w)
        l2 = math.floor(math.log2(l1 * (self.w - 1)) / log_w) + 1
        wots_elements = l1 + l2
        wots_sig_size = wots_elements * self.n
        wots_total = self.d * wots_sig_size
        
        # Authentication paths (d layers × hp nodes per layer)
        auth_total = self.d * self.hp * self.n
        
        return r_size + fors_size + wots_total + auth_total

        print(f"   {variant}: ~{size/1024:.1f} KB (actual: {params['sig_bytes']/1024:.1f} KB)")

demonstrate_sphincs()
```

## Summary

In this lesson, you learned about SPHINCS+ (SLH-DSA):

1. **Stateless Advantage**: No state management, safe backup/restore
2. **Hypertree Structure**: d layers of smaller trees, virtual 2^h leaves
3. **FORS**: Few-time signature with k trees for message signing
4. **WOTS+ Integration**: Each hypertree layer uses WOTS+ signatures
5. **Parameter Tradeoffs**: Fast (f) vs Small (s) variants
6. **Minimal Assumptions**: Security from hash functions only
7. **NIST Standard**: FIPS 205 (SLH-DSA)
8. **Size Cost**: 8-50 KB signatures (tradeoff for statelessness)

SPHINCS+ is the most conservative post-quantum signature choice, ideal when state management is impractical.
