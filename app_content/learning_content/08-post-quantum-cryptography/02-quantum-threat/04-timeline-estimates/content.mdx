## 1. Defining "Cryptographically Relevant"

Not all quantum computers pose a cryptographic threat. We need to define the target.

<Definition term="Cryptographically Relevant Quantum Computer (CRQC)">
A quantum computer capable of running Shor's algorithm at scale to break real-world cryptographic systems like RSA-2048 or ECC P-256 within a practical timeframe.
</Definition>

### The "Mosca Theorem"

<Definition term="Mosca Theorem">
If x + y > z, then worry. Where: x = shelf life of data, y = time to migrate to PQC, z = time until CRQC exists.
</Definition>

<Formula>
$$\text{If } \underbrace{x}_{\text{data lifetime}} + \underbrace{y}_{\text{migration time}} > \underbrace{z}_{\text{time to CRQC}} \text{, then start NOW}$$
</Formula>

```python
def mosca_analysis(data_lifetime_years: int, 
                   migration_years: int,
                   crqc_estimate_years: int) -> dict:
    """
    Apply Mosca's theorem to determine migration urgency.
    """
    x = data_lifetime_years
    y = migration_years
    z = crqc_estimate_years
    
    total_needed = x + y
    
    if total_needed > z:
        urgency = "CRITICAL - Start migration immediately"
        years_late = total_needed - z
    elif total_needed > z - 5:
        urgency = "HIGH - Start planning now"
        years_late = 0
    else:
        urgency = "MODERATE - Plan migration within 5 years"
        years_late = 0
    
    return {
        'data_lifetime': f"{x} years",
        'migration_time': f"{y} years",
        'crqc_estimate': f"{z} years from now",
        'total_protection_needed': f"{total_needed} years",
        'urgency': urgency,
        'years_late_if_optimistic': years_late
    }

# Example scenarios
scenarios = [
    ("Top Secret data", 25, 10, 15),
    ("Medical records", 20, 5, 15),
    ("Financial records", 10, 3, 15),
    ("Session tokens", 1, 1, 15)
]

print("Mosca Analysis for Different Data Types")
print("=" * 60)

for name, lifetime, migration, crqc in scenarios:
    result = mosca_analysis(lifetime, migration, crqc)
    print(f"\n{name}:")
    for key, value in result.items():
        print(f"  {key}: {value}")
```

1. **CRQC requirements** are substantial: 4,000+ logical qubits, low error rates
