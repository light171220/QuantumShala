## 2. Goppa Codes: The Secret Structure

<Definition term="Binary Goppa Code">
Let $g(x) \in \mathbb{F}_{2^m}[x]$ be a polynomial of degree $t$ with no roots in $\mathbb{F}_{2^m}$, and let $L = (\alpha_1, \ldots, \alpha_n)$ be distinct elements of $\mathbb{F}_{2^m}$. The binary Goppa code $\Gamma(L, g)$ consists of all binary vectors $(c_1, \ldots, c_n)$ such that:

$$\sum_{i=1}^{n} \frac{c_i}{x - \alpha_i} \equiv 0 \pmod{g(x)}$$
</Definition>

```python
class SimplifiedGoppaCode:
    """
    Simplified Goppa code implementation.
    
    Real Goppa codes use polynomial arithmetic over F_{2^m}.
    This implementation demonstrates the concepts with simplified operations.
    """
    
    def __init__(self, n: int, t: int, seed: int = None):
        """
        Initialize Goppa code.
        
        Args:
            n: Code length
            t: Error correction capability
            seed: Random seed
        """
        if seed is not None:
            np.random.seed(seed)
        
        self.n = n
        self.t = t
        
        # Approximate parameters
        self.m = int(np.ceil(np.log2(n)))  # Extension degree
        self.r = self.m * t  # Redundancy (parity bits)
        self.k = n - self.r  # Dimension
        
        if self.k <= 0:
            raise ValueError(f"Invalid parameters: n={n}, t={t} gives k={self.k}")
        
        # Generate secret parity-check matrix with algebraic structure
        self.H_secret = self._generate_goppa_parity_check()
        
        # Compute generator matrix
        self.G_secret = self._parity_to_generator(self.H_secret)
    
    def _generate_goppa_parity_check(self) -> np.ndarray:
        """
        Generate parity-check matrix.
        
        For real Goppa codes, H has special structure from the Goppa polynomial.
        Here we use a random matrix as placeholder.
        """
        # In practice: H is constructed from g(x) and support L
        # H[i,j] = α_j^i / g(α_j) in F_{2^m}
        
        # Simplified: random full-rank matrix
        while True:
            H = np.random.randint(0, 2, size=(self.r, self.n))
            if np.linalg.matrix_rank(H.astype(float)) == self.r:
                return H
    
    def _parity_to_generator(self, H: np.ndarray) -> np.ndarray:
        """Convert parity-check to systematic generator matrix."""
        r, n = H.shape
        k = n - r
        
        # Put H in form [A | I_r] using Gaussian elimination
        H_work = H.copy()
        
        # Work on last r columns to make them identity
        for col in range(n - r, n):
            rel_col = col - (n - r)
            
            # Find pivot in this column
            pivot_row = None
            for row in range(rel_col, r):
                if H_work[row, col] == 1:
                    pivot_row = row
                    break
            
            if pivot_row is None:
                # Need to swap with earlier column
                for swap_col in range(col):
                    if H_work[rel_col, swap_col] == 1:
                        H_work[:, [col, swap_col]] = H_work[:, [swap_col, col]]
                        break
                pivot_row = rel_col
            
            # Swap rows
            H_work[[rel_col, pivot_row]] = H_work[[pivot_row, rel_col]]
            
            # Eliminate
            for row in range(r):
                if row != rel_col and H_work[row, col] == 1:
                    H_work[row] = (H_work[row] + H_work[rel_col]) % 2
        
        # H is now [A | I_r], so G = [I_k | A^T]
        A = H_work[:, :k]
        G = np.hstack([np.eye(k, dtype=int), A.T]) % 2
        
        return G
    
    def encode(self, message: np.ndarray) -> np.ndarray:
        """Encode message to codeword."""
        return (message @ self.G_secret) % 2
    
    def decode(self, received: np.ndarray) -> Tuple[Optional[np.ndarray], np.ndarray]:
        """
        Decode received word using Patterson's algorithm (simplified).
        
        Real implementation uses polynomial factorization over F_{2^m}.
        """
        # Compute syndrome
        syndrome = (self.H_secret @ received) % 2
        
        if np.all(syndrome == 0):
            # No errors
            return received[:self.k], np.zeros(self.n, dtype=int)
        
        # Simplified: syndrome lookup for small codes
        # Real Patterson algorithm is O(n^2)
        
        for weight in range(1, self.t + 1):
            for error in self._generate_errors(weight):
                if np.array_equal((self.H_secret @ error) % 2, syndrome):
                    codeword = (received + error) % 2
                    return codeword[:self.k], error
        
        return None, received  # Decoding failure
    
    def _generate_errors(self, weight: int):
        """Generate all error patterns of given weight."""
        from itertools import combinations
        for positions in combinations(range(self.n), weight):
            error = np.zeros(self.n, dtype=int)
            for p in positions:
                error[p] = 1
            yield error
    
    def get_parameters(self) -> dict:
        """Return code parameters."""
        return {
            'n': self.n,
            'k': self.k,
            't': self.t,
            'm': self.m,
            'r': self.r,
            'rate': self.k / self.n
        }
```

## 3. McEliece Encryption Scheme

```python
class McElieceCryptosystem:
    """
    The McEliece Public-Key Encryption Scheme.
    
    Key Generation:
    1. Generate random Goppa code with generator G and decoder
    2. Generate random k×k invertible matrix S (scrambling)
    3. Generate random n×n permutation matrix P
    4. Public key: G' = S·G·P (disguised generator)
    5. Secret key: (S, G, P, decoder)
    
    Encryption: c = m·G' + e (add t random errors)
    Decryption: Use P^{-1}, decoder, S^{-1}
    """
    
    def __init__(self, n: int, t: int, seed: int = None):
        """
        Initialize McEliece cryptosystem.
        
        Args:
            n: Code length (public key size related)
            t: Error correction capability (security parameter)
            seed: Random seed for reproducibility
        """
        if seed is not None:
            np.random.seed(seed)
        
        # Generate Goppa code
        self.goppa = SimplifiedGoppaCode(n, t, seed)
        self.n = n
        self.t = t
        self.k = self.goppa.k
        
        # Generate scrambling matrix S (k × k invertible)
        self.S = self._random_invertible_matrix(self.k)
        self.S_inv = self._invert_binary_matrix(self.S)
        
        # Generate permutation matrix P (n × n)
        self.perm = np.random.permutation(n)
        self.P = np.eye(n, dtype=int)[self.perm]
        self.P_inv = self.P.T  # Permutation inverse is transpose
        
        # Compute public key: G' = S · G · P
        self.G_public = (self.S @ self.goppa.G_secret @ self.P) % 2
    
    def _random_invertible_matrix(self, n: int) -> np.ndarray:
        """Generate random invertible binary matrix."""
        while True:
            M = np.random.randint(0, 2, size=(n, n))
            if self._is_invertible(M):
                return M
    
    def _is_invertible(self, M: np.ndarray) -> bool:
        """Check if binary matrix is invertible."""
        try:
            self._invert_binary_matrix(M)
            return True
        except:
            return False
    
    def _invert_binary_matrix(self, M: np.ndarray) -> np.ndarray:
        """Invert binary matrix using Gaussian elimination."""
        n = M.shape[0]
        aug = np.hstack([M.copy(), np.eye(n, dtype=int)]) % 2
        
        for col in range(n):
            # Find pivot
            pivot = None
            for row in range(col, n):
                if aug[row, col] == 1:
                    pivot = row
                    break
            
            if pivot is None:
                raise ValueError("Matrix is singular")
            
            aug[[col, pivot]] = aug[[pivot, col]]
            
            for row in range(n):
                if row != col and aug[row, col] == 1:
                    aug[row] = (aug[row] + aug[col]) % 2
        
        return aug[:, n:]
    
    def get_public_key(self) -> np.ndarray:
        """Return public key (disguised generator matrix)."""
        return self.G_public
    
    def get_public_key_size(self) -> int:
        """Return public key size in bits."""
        return self.k * self.n
    
    def encrypt(self, message: np.ndarray, error: np.ndarray = None) -> np.ndarray:
        """
        Encrypt a message.
        
        Args:
            message: k-bit message
            error: n-bit error vector (optional, random if not provided)
        
        Returns:
            n-bit ciphertext
        """
        if len(message) != self.k:
            raise ValueError(f"Message must be {self.k} bits")
        
        # Generate random error of weight t if not provided
        if error is None:
            error = np.zeros(self.n, dtype=int)
            error_positions = np.random.choice(self.n, self.t, replace=False)
            error[error_positions] = 1
        
        if np.sum(error) > self.t:
            raise ValueError(f"Error weight must be at most {self.t}")
        
        # Compute ciphertext: c = m·G' + e
        codeword = (message @ self.G_public) % 2
        ciphertext = (codeword + error) % 2
        
        return ciphertext
    
    def decrypt(self, ciphertext: np.ndarray) -> Optional[np.ndarray]:
        """
        Decrypt a ciphertext.
        
        Steps:
        1. Apply P^{-1}: c' = c·P^{-1}
        2. Decode c' using Goppa decoder: get m'·G + e' = c' → m'
        3. Apply S^{-1}: m = m'·S^{-1}
        """
        # Step 1: Remove permutation
        c_prime = (ciphertext @ self.P_inv) % 2
        
        # Step 2: Decode using Goppa code
        m_prime, error = self.goppa.decode(c_prime)
        
        if m_prime is None:
            return None  # Decoding failure
        
        # Step 3: Unscramble
        message = (m_prime @ self.S_inv) % 2
        
        return message
    
    def get_parameters(self) -> dict:
        """Return system parameters."""
        return {
            'n': self.n,
            'k': self.k,
            't': self.t,
            'public_key_bits': self.k * self.n,
            'public_key_bytes': (self.k * self.n) // 8,
            'ciphertext_bits': self.n,
            'security_estimate': self._estimate_security()
        }
    
    def _estimate_security(self) -> int:
        """Estimate security level in bits (simplified)."""
        # Work factor for ISD is approximately:
        # W ≈ (n choose t) · (k/n)^t
        # Simplified estimate
        import math
        try:
            log_work = (self.t * math.log2(self.n / self.t) + 
                       self.t * math.log2(self.n / self.k))
            return int(log_work)
        except:
            return 0
```

## Summary

In this lesson, you learned about the McEliece cryptosystem:

1. **Historical Significance**: Oldest unbroken public-key scheme (1978)
2. **Goppa Codes**: Algebraic codes with efficient secret decoding
3. **Key Disguise**: G' = S·G·P looks random but enables decryption
4. **Encryption**: c = m·G' + e (add controlled errors)
5. **Decryption**: Unpermute, decode, unscramble
6. **Security**: Based on syndrome decoding and code distinguishing
7. **ISD Attacks**: Best known, still exponential complexity
8. **Classic McEliece**: NIST standardized, 261KB-1.3MB public keys

McEliece offers the strongest post-quantum security confidence at the cost of large public keys.
