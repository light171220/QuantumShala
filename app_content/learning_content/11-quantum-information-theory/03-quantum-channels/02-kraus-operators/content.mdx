## 1. The Operator-Sum Representation

### From Stinespring to Kraus

<Definition title="Kraus Representation">
Every CPTP map $\mathcal{E}$ can be written as:
$$\mathcal{E}(\rho) = \sum_{k=1}^{r} K_k \rho K_k^\dagger$$
where $\{K_k\}$ are Kraus operators satisfying $\sum_k K_k^\dagger K_k = I$.
</Definition>

### Derivation from Stinespring

Starting from the Stinespring dilation $\mathcal{E}(\rho) = \text{Tr}_E[V\rho V^\dagger]$:

<Formula title="Kraus from Isometry">
If $V = \sum_k |k\rangle_E \otimes K_k$, then:
$$\mathcal{E}(\rho) = \text{Tr}_E\left[\sum_{j,k} |j\rangle\langle k|_E \otimes K_j \rho K_k^\dagger\right] = \sum_k K_k \rho K_k^\dagger$$
</Formula>

The environment basis $\{|k\rangle_E\}$ determines the Kraus operators.

## 2. Properties of Kraus Operators

### Completeness Relation

<Formula title="Trace Preservation">
$$\sum_k K_k^\dagger K_k = I \quad \Leftrightarrow \quad \mathcal{E} \text{ is trace-preserving}$$
</Formula>

For trace-non-increasing maps (quantum operations):
$$\sum_k K_k^\dagger K_k \leq I$$

### Number of Kraus Operators

<Tip>
The minimum number of Kraus operators equals the rank of the Choi matrix:
$$r = \text{rank}(J(\mathcal{E}))$$
For a channel from $d$-dimensional to $d'$-dimensional system: $r \leq dd'$.
</Tip>

### Non-Uniqueness

<Warning>
Kraus decomposition is NOT unique! Different sets of Kraus operators can represent the same channel.
</Warning>

<Formula title="Unitary Freedom">
If $\{K_k\}$ and $\{L_j\}$ represent the same channel, then:
$$L_j = \sum_k U_{jk} K_k$$
for some unitary matrix $U$.
</Formula>

## 3. Computing Kraus Operators

### From Choi Matrix

```python
import numpy as np

def kraus_from_choi(choi, dim_in, dim_out):
    """Extract Kraus operators from Choi matrix via eigendecomposition."""
    # Eigendecomposition of Choi matrix
    eigvals, eigvecs = np.linalg.eigh(choi)

    kraus_ops = []
    for i, (val, vec) in enumerate(zip(eigvals, eigvecs.T)):
        if val > 1e-10:  # Non-zero eigenvalue
            # Reshape eigenvector to Kraus operator
            # Choi vec shape: (dim_out * dim_in,)
            # Kraus shape: (dim_out, dim_in)
            K = np.sqrt(val * dim_in) * vec.reshape(dim_out, dim_in, order='F')
            kraus_ops.append(K)

    return kraus_ops

def verify_kraus(kraus_ops, dim):
    """Verify completeness relation."""
    total = sum(K.conj().T @ K for K in kraus_ops)
    return np.allclose(total, np.eye(dim))

# Example: Depolarizing channel
def depolarizing_choi(p, dim=2):
    """Choi matrix for depolarizing channel."""
    # Maximally entangled state
    phi = np.zeros(dim**2)
    for i in range(dim):
        phi[i*dim + i] = 1/np.sqrt(dim)
    phi_proj = np.outer(phi, phi.conj())

    return (1-p) * phi_proj + p * np.eye(dim**2) / dim**2

choi = depolarizing_choi(0.2, 2)
kraus = kraus_from_choi(choi, 2, 2)
print(f"Number of Kraus operators: {len(kraus)}")
print(f"Completeness check: {verify_kraus(kraus, 2)}")
```

## 4. Standard Quantum Channels

### Pauli Channel

<Definition title="Pauli Channel">
$$\mathcal{E}(\rho) = (1-p_x-p_y-p_z)\rho + p_x X\rho X + p_y Y\rho Y + p_z Z\rho Z$$
Kraus operators: $K_0 = \sqrt{1-p_x-p_y-p_z}I$, $K_1 = \sqrt{p_x}X$, $K_2 = \sqrt{p_y}Y$, $K_3 = \sqrt{p_z}Z$
</Definition>

### Depolarizing Channel

<Example title="Depolarizing Kraus Operators">
$$\mathcal{D}_p(\rho) = (1-p)\rho + \frac{p}{3}(X\rho X + Y\rho Y + Z\rho Z)$$
Kraus operators:
- $K_0 = \sqrt{1-p}I$
- $K_1 = \sqrt{p/3}X$
- $K_2 = \sqrt{p/3}Y$
- $K_3 = \sqrt{p/3}Z$
</Example>

### Amplitude Damping

<Definition title="Amplitude Damping Channel">
Models decay from $|1\rangle$ to $|0\rangle$ with probability $\gamma$:
$$K_0 = \begin{pmatrix} 1 & 0 \\ 0 & \sqrt{1-\gamma} \end{pmatrix}, \quad K_1 = \begin{pmatrix} 0 & \sqrt{\gamma} \\ 0 & 0 \end{pmatrix}$$
</Definition>

```python
import numpy as np

def amplitude_damping_kraus(gamma):
    """Kraus operators for amplitude damping."""
    K0 = np.array([[1, 0], [0, np.sqrt(1-gamma)]])
    K1 = np.array([[0, np.sqrt(gamma)], [0, 0]])
    return [K0, K1]

def apply_channel(rho, kraus_ops):
    """Apply channel with given Kraus operators."""
    return sum(K @ rho @ K.conj().T for K in kraus_ops)

# Apply to |1><1|
gamma = 0.3
kraus = amplitude_damping_kraus(gamma)
rho_1 = np.array([[0, 0], [0, 1]])
output = apply_channel(rho_1, kraus)
print(f"Input |1><1|:\n{rho_1}")
print(f"Output (gamma={gamma}):\n{output}")
print(f"Population in |0>: {output[0,0]:.3f}")
print(f"Population in |1>: {output[1,1]:.3f}")
```

### Phase Damping

<Definition title="Phase Damping Channel">
Models loss of coherence without energy decay:
$$K_0 = \begin{pmatrix} 1 & 0 \\ 0 & \sqrt{1-\lambda} \end{pmatrix}, \quad K_1 = \begin{pmatrix} 0 & 0 \\ 0 & \sqrt{\lambda} \end{pmatrix}$$
</Definition>

### Generalized Amplitude Damping

Models thermal relaxation to finite temperature:

$$K_0 = \sqrt{p}\begin{pmatrix} 1 & 0 \\ 0 & \sqrt{1-\gamma} \end{pmatrix}, \quad K_1 = \sqrt{p}\begin{pmatrix} 0 & \sqrt{\gamma} \\ 0 & 0 \end{pmatrix}$$
$$K_2 = \sqrt{1-p}\begin{pmatrix} \sqrt{1-\gamma} & 0 \\ 0 & 1 \end{pmatrix}, \quad K_3 = \sqrt{1-p}\begin{pmatrix} 0 & 0 \\ \sqrt{\gamma} & 0 \end{pmatrix}$$

where $p = 1/(1 + e^{-\beta\omega})$ is thermal population.

## 5. Channel Composition

### Sequential Channels

<Formula title="Kraus Composition">
For $\mathcal{E}_2 \circ \mathcal{E}_1$ with Kraus operators $\{K_k\}$ and $\{L_j\}$:
$$(\mathcal{E}_2 \circ \mathcal{E}_1)(\rho) = \sum_{j,k} L_j K_k \rho K_k^\dagger L_j^\dagger$$
The composite Kraus operators are $\{L_j K_k\}_{j,k}$.
</Formula>

```python
import numpy as np

def compose_channels(kraus1, kraus2):
    """Compose two channels given their Kraus operators."""
    return [L @ K for L in kraus2 for K in kraus1]

# Compose two amplitude damping channels
gamma1, gamma2 = 0.2, 0.3
kraus1 = amplitude_damping_kraus(gamma1)
kraus2 = amplitude_damping_kraus(gamma2)
composed = compose_channels(kraus1, kraus2)

print(f"Number of composed Kraus ops: {len(composed)}")

# Verify: should be equivalent to AD with gamma_eff
# For AD: gamma_eff = gamma1 + gamma2 - gamma1*gamma2
gamma_eff = gamma1 + gamma2 - gamma1 * gamma2
print(f"Effective gamma: {gamma_eff:.4f}")
```

## 6. Physical Interpretation

### Measurement Interpretation

Each Kraus operator can be viewed as a measurement outcome:

<Definition title="Selective Operations">
$$\rho \to \frac{K_k \rho K_k^\dagger}{\text{Tr}[K_k \rho K_k^\dagger]}$$
with probability $p_k = \text{Tr}[K_k \rho K_k^\dagger]$.
</Definition>

The full channel is the average over all outcomes:
$$\mathcal{E}(\rho) = \sum_k p_k \cdot \frac{K_k \rho K_k^\dagger}{p_k} = \sum_k K_k \rho K_k^\dagger$$

### Environment Model

<Tip>
Kraus operators encode how the environment "branches" the quantum state. Each $K_k$ corresponds to a different environment state after interaction.
</Tip>

## 7. Error Analysis with Kraus Operators

### Fidelity Decay

```python
import numpy as np

def average_fidelity(kraus_ops, dim):
    """Compute average gate fidelity for a channel."""
    # Average fidelity = (d*F_e + 1)/(d+1) where F_e is entanglement fidelity
    # Entanglement fidelity F_e = sum_k |Tr[K_k]|^2 / d^2
    F_e = sum(np.abs(np.trace(K))**2 for K in kraus_ops) / dim**2
    return (dim * F_e + 1) / (dim + 1)

def diamond_norm_bound(kraus_ops, dim):
    """Lower bound on diamond distance from identity."""
    # Diamond norm >= 1 - average fidelity (approximate)
    F_avg = average_fidelity(kraus_ops, dim)
    return 1 - F_avg

# Analyze depolarizing channel
for p in [0.01, 0.05, 0.1, 0.2]:
    kraus = [
        np.sqrt(1-p) * np.eye(2),
        np.sqrt(p/3) * np.array([[0,1],[1,0]]),  # X
        np.sqrt(p/3) * np.array([[0,-1j],[1j,0]]),  # Y
        np.sqrt(p/3) * np.array([[1,0],[0,-1]])  # Z
    ]
    F = average_fidelity(kraus, 2)
    print(f"Depolarizing p={p}: Average fidelity = {F:.4f}")
```

## Summary

The Kraus representation provides a powerful tool for quantum channel analysis:

| Channel | Kraus Operators | Physical Process |
|---------|-----------------|------------------|
| Depolarizing | $\sqrt{1-p}I$, $\sqrt{p/3}X,Y,Z$ | Random Pauli errors |
| Amplitude Damping | $K_0, K_1$ with $\gamma$ | Energy decay (T1) |
| Phase Damping | Diagonal $K_0, K_1$ | Dephasing (T2) |
| Unitary | Single $U$ | Coherent evolution |

Key properties:
- Non-unique up to unitary mixing
- Number of operators = Choi rank
- Completeness: $\sum_k K_k^\dagger K_k = I$
- Composition: products of Kraus operators
