## 1. Linear Maps on Density Matrices

### Quantum Operations

A quantum channel describes any physical process that transforms quantum states:
$$\mathcal{E}: \mathcal{B}(\mathcal{H}_A) \to \mathcal{B}(\mathcal{H}_B)$$

<Definition title="Quantum Channel">
A map is a valid quantum channel if it takes every valid density matrix to a valid density matrix, even when acting on part of a larger system.
</Definition>

### Basic Requirements

For $\mathcal{E}$ to be physical:
1. **Linearity**: $\mathcal{E}(\alpha\rho + \beta\sigma) = \alpha\mathcal{E}(\rho) + \beta\mathcal{E}(\sigma)$
2. **Trace preservation**: $\text{Tr}[\mathcal{E}(\rho)] = \text{Tr}[\rho]$
3. **Complete positivity**: Preserves positivity even with ancilla

## 2. Positivity vs Complete Positivity

### Positive Maps

<Definition title="Positive Map">
A map $\mathcal{E}$ is positive if $\rho \geq 0 \Rightarrow \mathcal{E}(\rho) \geq 0$.
</Definition>

### The Transpose Map

The transpose is positive but NOT completely positive:
$$T(\rho) = \rho^T$$

<Warning>
Positive maps can fail when extended to composite systems! This is why we need complete positivity.
</Warning>

### Complete Positivity

<Definition title="Completely Positive (CP)">
A map $\mathcal{E}$ is completely positive if $(\mathcal{E} \otimes \mathcal{I}_n)(\rho_{AR}) \geq 0$ for all $n$ and all positive $\rho_{AR}$ on extended systems.
</Definition>

<Formula title="CP Condition">
$$\mathcal{E} \text{ is CP} \iff (\mathcal{E} \otimes \mathcal{I}_d)(\Phi_d^+) \geq 0$$
where $|\Phi_d^+\rangle = \frac{1}{\sqrt{d}}\sum_{i=0}^{d-1}|ii\rangle$ is maximally entangled.
</Formula>

## 3. The Choi-Jamiolkowski Isomorphism

### Choi Matrix

<Definition title="Choi Matrix">
The Choi matrix of $\mathcal{E}$ is:
$$J(\mathcal{E}) = (\mathcal{E} \otimes \mathcal{I})(|\Phi^+\rangle\langle\Phi^+|) = \frac{1}{d}\sum_{i,j} \mathcal{E}(|i\rangle\langle j|) \otimes |i\rangle\langle j|$$
</Definition>

### Fundamental Theorem

<Formula title="Choi-Jamiolkowski Theorem">
$$\mathcal{E} \text{ is CP} \iff J(\mathcal{E}) \geq 0$$
$$\mathcal{E} \text{ is TP} \iff \text{Tr}_B[J(\mathcal{E})] = \frac{I_A}{d}$$
</Formula>

### Computing with Choi Matrices

```python
import numpy as np

def compute_choi_matrix(channel_func, dim):
    """Compute Choi matrix for a quantum channel."""
    # Maximally entangled state |Phi+>
    phi_plus = np.zeros((dim**2, dim**2), dtype=complex)
    for i in range(dim):
        for j in range(dim):
            # |ij><kl| contributions
            ket_ij = np.zeros(dim**2, dtype=complex)
            ket_ij[i*dim + j] = 1
            for k in range(dim):
                for l in range(dim):
                    ket_kl = np.zeros(dim**2, dtype=complex)
                    ket_kl[k*dim + l] = 1
                    phi_plus += np.outer(ket_ij, ket_kl.conj()) / dim

    # Apply channel to first subsystem
    choi = np.zeros((dim**2, dim**2), dtype=complex)
    for i in range(dim):
        for j in range(dim):
            # Input basis element |i><j|
            basis_ij = np.zeros((dim, dim), dtype=complex)
            basis_ij[i, j] = 1
            # Apply channel
            output = channel_func(basis_ij)
            # Tensor with |i><j| on ancilla
            for k in range(dim):
                for l in range(dim):
                    choi[i*dim + k, j*dim + l] = output[k, l] / dim

    return choi

def is_cptp(choi_matrix, dim):
    """Check if Choi matrix represents CPTP map."""
    # Check CP: Choi matrix positive semidefinite
    eigvals = np.linalg.eigvalsh(choi_matrix)
    is_cp = np.all(eigvals >= -1e-10)

    # Check TP: partial trace = I/d
    partial_trace = np.zeros((dim, dim), dtype=complex)
    for i in range(dim):
        for j in range(dim):
            for k in range(dim):
                partial_trace[i, j] += choi_matrix[i*dim + k, j*dim + k]

    is_tp = np.allclose(partial_trace, np.eye(dim) / dim)

    return is_cp, is_tp

# Example: Depolarizing channel
def depolarizing_channel(rho, p=0.1):
    """Depolarizing channel: rho -> (1-p)rho + p*I/d"""
    d = rho.shape[0]
    return (1 - p) * rho + p * np.eye(d) / d

# Compute and verify
dim = 2
choi = compute_choi_matrix(lambda rho: depolarizing_channel(rho, 0.1), dim)
cp, tp = is_cptp(choi, dim)
print(f"Depolarizing channel - CP: {cp}, TP: {tp}")
```

## 4. Examples of CPTP Maps

### The Depolarizing Channel

$$\mathcal{D}_p(\rho) = (1-p)\rho + p\frac{I}{d}$$

<Example title="Depolarizing Channel Choi Matrix">
For qubits with depolarizing parameter $p$:
$$J(\mathcal{D}_p) = (1-p)|\Phi^+\rangle\langle\Phi^+| + \frac{p}{4}I_4$$
Eigenvalues: $(1 - 3p/4)$ with multiplicity 1, and $p/4$ with multiplicity 3.
CP requires $p \leq 4/3$, physical range is $p \in [0, 1]$.
</Example>

### The Amplitude Damping Channel

Models energy decay (e.g., photon loss, T1 relaxation):

$$K_0 = \begin{pmatrix} 1 & 0 \\ 0 & \sqrt{1-\gamma} \end{pmatrix}, \quad K_1 = \begin{pmatrix} 0 & \sqrt{\gamma} \\ 0 & 0 \end{pmatrix}$$

### The Phase Damping Channel

Models dephasing (T2 relaxation):

$$K_0 = \begin{pmatrix} 1 & 0 \\ 0 & \sqrt{1-\lambda} \end{pmatrix}, \quad K_1 = \begin{pmatrix} 0 & 0 \\ 0 & \sqrt{\lambda} \end{pmatrix}$$

## 5. Properties of CPTP Maps

### Composition

<Formula title="Channel Composition">
If $\mathcal{E}_1$ and $\mathcal{E}_2$ are CPTP, then $\mathcal{E}_2 \circ \mathcal{E}_1$ is CPTP.
$$J(\mathcal{E}_2 \circ \mathcal{E}_1) = \text{Tr}_B[(J(\mathcal{E}_2) \otimes I_A)(I_C \otimes J(\mathcal{E}_1)^{T_B})]$$
</Formula>

### Convex Combinations

<Tip>
The set of CPTP maps is convex: if $\mathcal{E}_1, \mathcal{E}_2$ are CPTP and $\lambda \in [0,1]$, then $\lambda\mathcal{E}_1 + (1-\lambda)\mathcal{E}_2$ is CPTP.
</Tip>

### Unitary Channels

Every unitary evolution is CPTP:
$$\mathcal{U}(\rho) = U\rho U^\dagger$$

The Choi matrix is:
$$J(\mathcal{U}) = |U\rangle\rangle\langle\langle U|$$
where $|U\rangle\rangle = (U \otimes I)|\Phi^+\rangle$ is the Choi vector.

## 6. Detecting Non-CP Maps

### The Partial Transpose Test

<Formula title="Peres-Horodecki Criterion">
The transpose map $T$ is positive but not CP. This provides entanglement detection:
$$\rho^{T_B} \not\geq 0 \Rightarrow \rho \text{ is entangled}$$
</Formula>

```python
import numpy as np

def partial_transpose(rho, dim_A, dim_B, system='B'):
    """Compute partial transpose of bipartite state."""
    rho_reshaped = rho.reshape(dim_A, dim_B, dim_A, dim_B)
    if system == 'B':
        rho_pt = rho_reshaped.transpose(0, 3, 2, 1)
    else:  # system == 'A'
        rho_pt = rho_reshaped.transpose(2, 1, 0, 3)
    return rho_pt.reshape(dim_A * dim_B, dim_A * dim_B)

def check_ppt(rho, dim_A, dim_B):
    """Check if state has positive partial transpose."""
    rho_pt = partial_transpose(rho, dim_A, dim_B)
    eigvals = np.linalg.eigvalsh(rho_pt)
    return np.all(eigvals >= -1e-10), min(eigvals)

# Bell state - should have negative partial transpose
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
rho_bell = np.outer(bell, bell.conj())
is_ppt, min_eig = check_ppt(rho_bell, 2, 2)
print(f"Bell state PPT: {is_ppt}, min eigenvalue: {min_eig:.4f}")

# Product state - should have positive partial transpose
product = np.array([1, 0, 0, 0])  # |00>
rho_product = np.outer(product, product.conj())
is_ppt, min_eig = check_ppt(rho_product, 2, 2)
print(f"Product state PPT: {is_ppt}, min eigenvalue: {min_eig:.4f}")
```

## 7. Stinespring Dilation

### The Dilation Theorem

<Definition title="Stinespring Dilation">
Every CPTP map $\mathcal{E}$ can be written as:
$$\mathcal{E}(\rho) = \text{Tr}_E[V\rho V^\dagger]$$
where $V: \mathcal{H}_A \to \mathcal{H}_B \otimes \mathcal{H}_E$ is an isometry ($V^\dagger V = I_A$).
</Definition>

This shows every quantum channel arises from:
1. Coupling to environment in pure state
2. Unitary evolution on system + environment
3. Tracing out environment

<Tip>
Stinespring dilation provides the physical interpretation: noise comes from entanglement with unmeasured degrees of freedom.
</Tip>

## Summary

CPTP maps are the mathematical framework for quantum channels:

| Property | Mathematical Condition | Physical Meaning |
|----------|----------------------|------------------|
| Linearity | $\mathcal{E}(\sum p_i\rho_i) = \sum p_i\mathcal{E}(\rho_i)$ | Preserves mixtures |
| CP | $J(\mathcal{E}) \geq 0$ | Works with entanglement |
| TP | $\text{Tr}_B[J(\mathcal{E})] = I/d$ | Preserves probability |

The Choi-Jamiolkowski isomorphism connects channels to states, enabling powerful analysis techniques.
