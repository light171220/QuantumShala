## 1. Channel Simulation Problem

### The Basic Question

<Definition title="Channel Simulation">
Given channels $\mathcal{N}$ and $\mathcal{M}$ plus shared resources (entanglement, communication), can we simulate $\mathcal{N}$ using $\mathcal{M}$?
</Definition>

### Resources for Simulation

Key resources that can assist channel simulation:
1. **Shared entanglement**: Pre-shared Bell pairs
2. **Classical communication**: Forward or backward
3. **Free quantum communication**: Another channel
4. **Local operations**: LOCC protocols

## 2. Teleportation-Based Simulation

### The Basic Protocol

<Formula title="Teleportation Simulation">
Any quantum channel $\mathcal{N}$ can be simulated using:
1. Shared entanglement (1 ebit per qubit)
2. Classical communication (2 bits per qubit)
3. Application of $\mathcal{N}$ locally
</Formula>

```python
import numpy as np

def teleportation_simulation(rho, channel_func):
    """Simulate channel via teleportation.

    Protocol:
    1. Alice and Bob share |Phi+>
    2. Alice does Bell measurement on rho and her half
    3. Alice sends 2 classical bits to Bob
    4. Bob applies correction, then channel N
    """
    # In simulation, this reduces to just applying the channel
    # The teleportation is "consumed" in the protocol
    return channel_func(rho)

def standard_teleportation_cost(dim):
    """Resources for teleporting d-dimensional state."""
    return {
        'entanglement_ebits': np.log2(dim),
        'classical_bits': 2 * np.log2(dim)
    }

print("Teleportation resources for simulation:")
for d in [2, 4, 8]:
    cost = standard_teleportation_cost(d)
    print(f"dim={d}: {cost['entanglement_ebits']:.1f} ebits, {cost['classical_bits']:.1f} cbits")
```

## 3. Reverse Shannon Theorem

### Classical Reverse Shannon

<Definition title="Classical Reverse Shannon">
Simulating a noisy channel $W$ using a noiseless channel requires rate:
$$R = I(X:Y)_{\pi,W}$$
bits of noiseless communication per channel use, where $\pi$ is the optimal input distribution.
</Definition>

### Quantum Reverse Shannon

<Formula title="Quantum Reverse Shannon Theorem">
Simulating quantum channel $\mathcal{N}$ using:
- Noiseless quantum communication at rate $Q$
- Unlimited shared entanglement

requires $Q = \frac{1}{2}I(A:B)_\rho$ where $\rho$ is the Choi state.
</Formula>

<Tip>
The quantum reverse Shannon theorem shows that channel simulation and channel coding are dual problems.
</Tip>

## 4. Entanglement Cost of Channels

### Definition

<Definition title="Entanglement Cost">
The **entanglement cost** $E_C(\mathcal{N})$ is the minimum rate of entanglement (ebits) needed to simulate $\mathcal{N}$ using LOCC and shared entanglement.
</Definition>

### Bounds

<Formula title="Entanglement Cost Bounds">
For channel $\mathcal{N}$ with Choi state $\rho_{\mathcal{N}}$:
$$E_C(\mathcal{N}) \leq E_F(\rho_{\mathcal{N}})$$
where $E_F$ is the entanglement of formation.
</Formula>

### Exact Values

<Example title="Entanglement Cost Examples">
1. **Noiseless channel**: $E_C(\mathcal{I}) = \log d$ ebits
2. **EB channels**: $E_C(\mathcal{N}_{EB}) = 0$ (no entanglement needed)
3. **Erasure channel**: $E_C(\mathcal{E}_p) = (1-p)\log d$
</Example>

```python
import numpy as np

def estimate_entanglement_cost(choi_state):
    """Estimate entanglement cost from Choi state.

    Upper bound: entanglement of formation of Choi state.
    """
    # Compute entropy of reduced state
    dim = int(np.sqrt(choi_state.shape[0]))
    choi_reshaped = choi_state.reshape(dim, dim, dim, dim)

    # Trace out second system
    rho_A = np.trace(choi_reshaped, axis1=1, axis2=3)

    eigvals = np.linalg.eigvalsh(rho_A)
    eigvals = eigvals[eigvals > 1e-15]
    S_A = -np.sum(eigvals * np.log2(eigvals))

    return S_A  # Upper bound

def depolarizing_choi(p, d=2):
    phi = np.zeros(d**2)
    for i in range(d):
        phi[i*d + i] = 1/np.sqrt(d)
    phi_proj = np.outer(phi, phi.conj())
    return (1-p) * phi_proj + p * np.eye(d**2) / d**2

print("Entanglement cost upper bounds:")
for p in [0, 0.2, 0.5, 0.8, 1.0]:
    choi = depolarizing_choi(p)
    E_upper = estimate_entanglement_cost(choi)
    print(f"Depolarizing p={p}: E_C <= {E_upper:.4f} ebits")
```

## 5. Channel Simulation Protocols

### Protocol 1: Teleportation + Local Channel

For any channel $\mathcal{N}$:
1. Share maximally entangled state
2. Teleport input to Bob
3. Bob applies $\mathcal{N}$ locally
4. Total cost: $\log d$ ebits + $2\log d$ cbits

### Protocol 2: Port-Based Teleportation

<Definition title="Port-Based Teleportation">
Uses $N$ ports (copies of maximally entangled state). Sender measures, announces port number. Achieves:
- Perfect fidelity as $N \to \infty$
- No Pauli correction needed
</Definition>

### Protocol 3: Quantum Combs

<Tip>
Quantum combs (higher-order quantum maps) provide a framework for channel simulation where the "channel" itself can be an input/output.
</Tip>

## 6. Resource Theory of Channels

### Channels as Resources

<Definition title="Channel Resource Theory">
A resource theory of quantum channels consists of:
1. **Free channels**: e.g., LOCC channels, EB channels
2. **Resource channels**: e.g., noiseless channels, coherent channels
3. **Conversion protocols**: How to transform one channel to another
</Definition>

### Channel Ordering

<Formula title="Degradation Ordering">
$\mathcal{N} \succeq \mathcal{M}$ (N dominates M) if:
$$\mathcal{M} = \mathcal{D} \circ \mathcal{N}$$
for some free operation $\mathcal{D}$.
</Formula>

### Channel Monotones

<Definition title="Channel Monotones">
A function $f$ on channels is a monotone if:
$$\mathcal{N} \succeq \mathcal{M} \Rightarrow f(\mathcal{N}) \geq f(\mathcal{M})$$
</Definition>

Examples of monotones:
- Quantum capacity: $Q(\mathcal{N})$
- Coherent information: $\max_\rho I_c(\rho, \mathcal{N})$
- Diamond norm distance from identity

```python
import numpy as np

def channel_fidelity(choi1, choi2):
    """Compute fidelity between two channels via Choi matrices."""
    # F = Tr[sqrt(sqrt(rho) sigma sqrt(rho))]^2
    sqrt_choi1 = np.linalg.matrix_power(choi1 + 1e-10*np.eye(len(choi1)), 1)
    # Simplified: use trace distance bound
    diff = choi1 - choi2
    eigvals = np.linalg.eigvalsh(diff)
    trace_dist = 0.5 * np.sum(np.abs(eigvals))

    fidelity = 1 - trace_dist
    return max(0, fidelity)

def diamond_norm_from_identity(choi, dim):
    """Estimate diamond norm distance from identity channel."""
    # Identity Choi matrix
    phi = np.zeros(dim**2)
    for i in range(dim):
        phi[i*dim + i] = 1/np.sqrt(dim)
    choi_id = np.outer(phi, phi.conj())

    # Upper bound via trace distance of Choi
    diff = choi - choi_id
    eigvals = np.linalg.eigvalsh(diff)
    return np.sum(np.abs(eigvals))

print("Channel distance from identity:")
for p in [0, 0.1, 0.3, 0.5]:
    choi = depolarizing_choi(p)
    dist = diamond_norm_from_identity(choi, 2)
    print(f"Depolarizing p={p}: ||N - I||_diamond <= {dist:.4f}")
```

## 7. Applications

### Error Correction as Simulation

<Example title="Error Correction Perspective">
Quantum error correction can be viewed as channel simulation:
- Given: Noisy channel $\mathcal{N}$
- Goal: Simulate noiseless channel $\mathcal{I}$
- Resources: Encoding/decoding operations

Achievable when $Q(\mathcal{N}) > 0$.
</Example>

### Quantum Repeaters

<Definition title="Quantum Repeaters">
Extend quantum communication over long distances by:
1. Dividing into segments
2. Creating entanglement in each segment
3. Entanglement swapping to create end-to-end entanglement
4. Using shared entanglement to simulate noiseless channel
</Definition>

### Channel Discrimination

<Tip>
Channel simulation relates to channel discrimination: if two channels can simulate each other with vanishing error, they are operationally equivalent.
</Tip>
