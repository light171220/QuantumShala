## 1. Definition and Motivation

### The Entanglement-Breaking Property

<Definition title="Entanglement-Breaking Channel">
A channel $\mathcal{N}_{A \to B}$ is **entanglement-breaking (EB)** if for any state $\rho_{RA}$:
$$(\mathcal{I}_R \otimes \mathcal{N}_A)(\rho_{RA}) \text{ is separable}$$
</Definition>

In other words, applying $\mathcal{N}$ to one part of any entangled state destroys the entanglement.

### Physical Interpretation

<Tip>
Entanglement-breaking channels represent processes where all quantum correlations are lost - the channel effectively performs a measurement and prepares a new state based on the result.
</Tip>

## 2. The Measure-and-Prepare Characterization

### Holevo Form

<Formula title="Holevo Form Theorem">
$\mathcal{N}$ is entanglement-breaking if and only if:
$$\mathcal{N}(\rho) = \sum_k \text{Tr}[M_k \rho] \sigma_k$$
where $\{M_k\}$ is a POVM and $\{\sigma_k\}$ are density matrices.
</Formula>

This is the "measure-and-prepare" or "quantum-classical-quantum" form.

### Proof Outline

**Direction 1**: Measure-and-prepare $\Rightarrow$ EB
- Measuring $A$ destroys entanglement with $R$
- Preparing $\sigma_k$ creates no new entanglement

**Direction 2**: EB $\Rightarrow$ Measure-and-prepare
- Apply to half of maximally entangled state
- Output must be separable
- Extract measurement operators from separable decomposition

```python
import numpy as np

def measure_and_prepare_channel(rho, povm, prepared_states):
    """Apply measure-and-prepare channel.

    Args:
        rho: Input density matrix
        povm: List of POVM elements M_k
        prepared_states: List of states sigma_k
    """
    output = np.zeros_like(prepared_states[0], dtype=complex)

    for M_k, sigma_k in zip(povm, prepared_states):
        prob_k = np.real(np.trace(M_k @ rho))
        output += prob_k * sigma_k

    return output

# Example: Completely dephasing channel (measure in Z, prepare in Z)
def z_dephasing():
    """Complete Z-dephasing as measure-and-prepare."""
    # POVM: Z measurement
    M0 = np.array([[1, 0], [0, 0]])  # |0><0|
    M1 = np.array([[0, 0], [0, 1]])  # |1><1|
    povm = [M0, M1]

    # Prepared states: same as measurement outcomes
    sigma0 = M0
    sigma1 = M1
    states = [sigma0, sigma1]

    return povm, states

povm, states = z_dephasing()

# Test on superposition state
rho_plus = np.array([[0.5, 0.5], [0.5, 0.5]])
output = measure_and_prepare_channel(rho_plus, povm, states)
print(f"Input |+><+|:\n{rho_plus}")
print(f"Output (dephased):\n{output}")
```

## 3. Characterizations of EB Channels

### Choi Matrix Criterion

<Formula title="EB via Choi Matrix">
$\mathcal{N}$ is entanglement-breaking if and only if:
$$J(\mathcal{N}) \text{ is a separable state}$$
</Formula>

### Kraus Operator Criterion

<Definition title="EB Kraus Form">
$\mathcal{N}$ is EB if and only if it has a Kraus representation with **rank-1** Kraus operators:
$$K_k = |\psi_k\rangle\langle\phi_k|$$
</Definition>

```python
import numpy as np

def check_eb_kraus(kraus_ops):
    """Check if Kraus operators are all rank-1."""
    for K in kraus_ops:
        rank = np.linalg.matrix_rank(K, tol=1e-10)
        if rank > 1:
            return False
    return True

def random_eb_channel(dim_in, dim_out, num_ops):
    """Generate random entanglement-breaking channel."""
    kraus_ops = []
    for _ in range(num_ops):
        # Random rank-1: |psi><phi|
        psi = np.random.randn(dim_out) + 1j * np.random.randn(dim_out)
        psi /= np.linalg.norm(psi)
        phi = np.random.randn(dim_in) + 1j * np.random.randn(dim_in)
        phi /= np.linalg.norm(phi)
        K = np.outer(psi, phi.conj())
        kraus_ops.append(K)

    # Normalize for trace preservation
    total = sum(K.conj().T @ K for K in kraus_ops)
    scale = np.sqrt(np.trace(total) / dim_in)
    kraus_ops = [K / scale for K in kraus_ops]

    return kraus_ops

# Generate and verify
kraus = random_eb_channel(2, 2, 4)
print(f"All rank-1: {check_eb_kraus(kraus)}")
```

## 4. Properties of EB Channels

### Zero Quantum Capacity

<Formula title="EB Channels Have Q = 0">
For any entanglement-breaking channel $\mathcal{N}$:
$$Q(\mathcal{N}) = 0$$
</Formula>

<Warning>
EB channels cannot transmit any quantum information reliably! All coherence is lost.
</Warning>

### Anti-Degradability

<Tip>
All entanglement-breaking channels are anti-degradable. The environment (which performs the "measurement") has complete information.
</Tip>

### Classical Capacity Preserved

EB channels can still have positive classical capacity $C > 0$.

The channel $\mathcal{N}$ behaves like a classical channel for information transmission.

## 5. Examples of EB Channels

### Complete Dephasing

<Example title="Complete Dephasing">
$$\mathcal{D}(\rho) = \sum_k |k\rangle\langle k| \rho |k\rangle\langle k| = \sum_k \rho_{kk} |k\rangle\langle k|$$
Measures in computational basis and reprepares in same basis.
</Example>

### Depolarizing Channel (High Noise)

<Example title="Depolarizing becomes EB">
The depolarizing channel $\mathcal{D}_p(\rho) = (1-p)\rho + p\frac{I}{d}$ is EB when:
$$p \geq \frac{d}{d+1}$$
For qubits: $p \geq 2/3$.
</Example>

### Amplitude Damping (Complete Decay)

<Example title="Complete Amplitude Damping">
Amplitude damping with $\gamma = 1$:
$$\mathcal{A}_1(\rho) = |0\rangle\langle 0|$$
Always outputs ground state - clearly EB.
</Example>

```python
import numpy as np

def is_depolarizing_eb(p, d=2):
    """Check if depolarizing channel is EB."""
    threshold = d / (d + 1)
    return p >= threshold

def depolarizing_choi(p, d=2):
    """Compute Choi matrix for depolarizing channel."""
    # Maximally entangled state
    phi = np.zeros(d**2)
    for i in range(d):
        phi[i*d + i] = 1/np.sqrt(d)
    phi_proj = np.outer(phi, phi.conj())

    # Depolarizing Choi
    return (1-p) * phi_proj + p * np.eye(d**2) / d**2

def check_separable_ppt(choi, dim):
    """Check PPT criterion (necessary for separability)."""
    # Partial transpose
    choi_reshaped = choi.reshape(dim, dim, dim, dim)
    choi_pt = choi_reshaped.transpose(0, 3, 2, 1).reshape(dim**2, dim**2)

    eigvals = np.linalg.eigvalsh(choi_pt)
    return np.all(eigvals >= -1e-10)

print("Depolarizing Channel EB Analysis:")
print("-" * 40)
for p in [0, 0.3, 0.5, 0.667, 0.8, 1.0]:
    is_eb = is_depolarizing_eb(p)
    choi = depolarizing_choi(p)
    is_ppt = check_separable_ppt(choi, 2)
    print(f"p={p:.3f}: EB={is_eb}, Choi PPT={is_ppt}")
```

## 6. Composition Properties

### EB After Any Channel

<Formula title="EB Composition">
For any channel $\mathcal{M}$:
- $\mathcal{N}_{EB} \circ \mathcal{M}$ is EB
- $\mathcal{M} \circ \mathcal{N}_{EB}$ is EB
</Formula>

Once entanglement is broken, it cannot be restored by further processing.

### Tensor Products

If $\mathcal{N}_1$ is EB, then $\mathcal{N}_1 \otimes \mathcal{N}_2$ is NOT necessarily EB (unless $\mathcal{N}_2$ is also EB).

## 7. Detection and Certification

### Entanglement Witness for Channels

<Definition title="Entanglement Certification">
Channel $\mathcal{N}$ is NOT EB if there exists input $\rho_{RA}$ such that:
$$(\mathcal{I} \otimes \mathcal{N})(\rho_{RA}) \text{ is entangled (e.g., NPT)}$$
</Definition>

```python
import numpy as np

def certify_not_eb(channel_func, dim):
    """Certify channel is not EB using Bell state."""
    # Use maximally entangled state
    phi = np.zeros(dim**2, dtype=complex)
    for i in range(dim):
        phi[i*dim + i] = 1/np.sqrt(dim)
    rho_AB = np.outer(phi, phi.conj())

    # Apply channel to B
    output = np.zeros((dim**2, dim**2), dtype=complex)
    for i in range(dim):
        for j in range(dim):
            for k in range(dim):
                for l in range(dim):
                    # Element (i*dim+k, j*dim+l)
                    basis_kl = np.zeros((dim, dim), dtype=complex)
                    basis_kl[k, l] = 1
                    channel_out = channel_func(basis_kl)
                    # Tensor with |i><j| on A
                    output[i*dim:i*dim+dim, j*dim:j*dim+dim] += rho_AB[i*dim+k, j*dim+l] * channel_out

    # Check PPT
    output_reshaped = output.reshape(dim, dim, dim, dim)
    output_pt = output_reshaped.transpose(0, 3, 2, 1).reshape(dim**2, dim**2)
    eigvals = np.linalg.eigvalsh(output_pt)

    min_eig = min(eigvals)
    is_npt = min_eig < -1e-10

    return {
        'is_entangled': is_npt,
        'min_pt_eigenvalue': min_eig,
        'is_eb': not is_npt
    }

# Test various channels
def depolarizing(rho, p):
    d = rho.shape[0]
    return (1-p) * rho + p * np.eye(d) / d

print("EB Certification:")
for p in [0.3, 0.5, 0.667, 0.8]:
    result = certify_not_eb(lambda r: depolarizing(r, p), 2)
    print(f"Depolarizing p={p}: EB={result['is_eb']}")
```

## Summary

| Property | Characterization |
|----------|------------------|
| Definition | $(\mathcal{I} \otimes \mathcal{N})(\rho)$ always separable |
| Holevo form | $\mathcal{N}(\rho) = \sum_k \text{Tr}[M_k\rho]\sigma_k$ |
| Kraus form | All Kraus operators are rank-1 |
| Choi criterion | Choi matrix is separable |
| Quantum capacity | $Q = 0$ |
| Degradability | Always anti-degradable |

Key examples:
- Complete dephasing: always EB
- Depolarizing: EB for $p \geq d/(d+1)$
- Amplitude damping: EB for $\gamma = 1$
