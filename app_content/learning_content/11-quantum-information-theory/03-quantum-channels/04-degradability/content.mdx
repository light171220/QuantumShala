## 1. Complementary Channels

### Stinespring Dilation Revisited

<Definition title="Complementary Channel">
For channel $\mathcal{N}_{A \to B}$ with Stinespring isometry $V: A \to B \otimes E$:
- **Main channel**: $\mathcal{N}(\rho) = \text{Tr}_E[V\rho V^\dagger]$
- **Complementary channel**: $\mathcal{N}^c(\rho) = \text{Tr}_B[V\rho V^\dagger]$
</Definition>

The complementary channel describes what the environment "learns" about the input.

### Kraus Representation

<Formula title="Complementary from Kraus">
If $\mathcal{N}(\rho) = \sum_k K_k \rho K_k^\dagger$, then:
$$\mathcal{N}^c(\rho) = \sum_{j,k} \text{Tr}[K_j^\dagger K_k \rho] |j\rangle\langle k|_E$$
</Formula>

```python
import numpy as np

def complementary_channel_matrix(kraus_ops, rho):
    """Compute complementary channel output."""
    n_kraus = len(kraus_ops)
    rho_E = np.zeros((n_kraus, n_kraus), dtype=complex)

    for j, Kj in enumerate(kraus_ops):
        for k, Kk in enumerate(kraus_ops):
            rho_E[j, k] = np.trace(Kj.conj().T @ Kk @ rho)

    return rho_E

def amplitude_damping_kraus(gamma):
    K0 = np.array([[1, 0], [0, np.sqrt(1-gamma)]])
    K1 = np.array([[0, np.sqrt(gamma)], [0, 0]])
    return [K0, K1]

# Test: Pure state input
gamma = 0.3
kraus = amplitude_damping_kraus(gamma)
psi = np.array([0.6, 0.8])
rho = np.outer(psi, psi.conj())

rho_E = complementary_channel_matrix(kraus, rho)
print(f"Environment state:\n{rho_E}")
print(f"Trace: {np.trace(rho_E):.4f}")
```

## 2. Degradable Channels

### Definition

<Definition title="Degradable Channel">
A channel $\mathcal{N}$ is **degradable** if there exists a degrading map $\mathcal{D}$ such that:
$$\mathcal{N}^c = \mathcal{D} \circ \mathcal{N}$$
The receiver can simulate what the environment sees.
</Definition>

### Physical Interpretation

<Tip>
In a degradable channel, Bob (receiver) has at least as much information as Eve (environment). This is the "good" case for quantum communication.
</Tip>

### Key Theorem

<Formula title="Degradable Channel Capacity">
For degradable $\mathcal{N}$:
$$Q(\mathcal{N}) = \max_\rho I_c(\rho, \mathcal{N}) = \max_\rho [S(\mathcal{N}(\rho)) - S(\mathcal{N}^c(\rho))]$$
No regularization needed! Single-letter formula.
</Formula>

Also: $Q(\mathcal{N}) = P(\mathcal{N})$ (quantum capacity equals private capacity).

### Proof Sketch

For degradable channels:
1. Data processing: $I_c(\rho, \mathcal{N}^{\otimes n}) \leq n \cdot \max_\sigma I_c(\sigma, \mathcal{N})$
2. Additivity of coherent information for degradable channels
3. Therefore regularization is unnecessary

## 3. Examples of Degradable Channels

### Erasure Channel

<Example title="Erasure Channel is Degradable">
The erasure channel $\mathcal{E}_p$ is degradable for all $p$:
- Bob receives either the state or erasure flag
- Degrading map: if Bob gets state, output to Eve; if erasure, output erasure
</Example>

### Amplitude Damping (Low Noise)

<Example title="Amplitude Damping Degradability">
The amplitude damping channel $\mathcal{A}_\gamma$ is:
- **Degradable** for $\gamma \leq 1/2$
- **Anti-degradable** for $\gamma \geq 1/2$

At $\gamma = 1/2$: channel equals its complement (self-complementary).
</Example>

```python
import numpy as np

def von_neumann_entropy(rho):
    eigvals = np.linalg.eigvalsh(rho)
    eigvals = eigvals[eigvals > 1e-15]
    return -np.sum(eigvals * np.log2(eigvals)) if len(eigvals) > 0 else 0

def amplitude_damping(rho, gamma):
    K0 = np.array([[1, 0], [0, np.sqrt(1-gamma)]])
    K1 = np.array([[0, np.sqrt(gamma)], [0, 0]])
    return K0 @ rho @ K0.conj().T + K1 @ rho @ K1.conj().T

def check_degradability_ad(gamma):
    """Check degradability of amplitude damping numerically."""
    # For AD: degradable iff gamma <= 0.5

    # Test: can we express N^c as D o N?
    # If gamma <= 0.5: environment gets "less" than receiver
    # If gamma > 0.5: environment gets "more" than receiver

    # Simple criterion: compare coherent information signs
    psi = np.array([1/np.sqrt(2), 1/np.sqrt(2)])
    rho = np.outer(psi, psi.conj())

    rho_B = amplitude_damping(rho, gamma)
    # For complementary: use Kraus structure
    kraus = amplitude_damping_kraus(gamma)
    rho_E = complementary_channel_matrix(kraus, rho)

    S_B = von_neumann_entropy(rho_B)
    S_E = von_neumann_entropy(rho_E)
    I_c = S_B - S_E

    return {
        'gamma': gamma,
        'S_B': S_B,
        'S_E': S_E,
        'I_c': I_c,
        'degradable': gamma <= 0.5,
        'anti_degradable': gamma >= 0.5
    }

print("Amplitude Damping Degradability:")
print("-" * 50)
for gamma in [0.1, 0.3, 0.5, 0.7, 0.9]:
    result = check_degradability_ad(gamma)
    status = "Degradable" if result['degradable'] else "Anti-degradable"
    print(f"gamma={gamma}: I_c={result['I_c']:.4f}, {status}")
```

### Generalized Dephasing

<Example title="Dephasing is Degradable">
Pure dephasing channels (diagonal Kraus operators) are degradable.
$$\mathcal{N}(\rho) = \sum_k p_k Z^k \rho Z^k$$
where $Z^k = \text{diag}(\omega^{0k}, \omega^{1k}, \ldots)$ with $\omega = e^{2\pi i/d}$.
</Example>

## 4. Anti-Degradable Channels

### Definition

<Definition title="Anti-Degradable Channel">
A channel $\mathcal{N}$ is **anti-degradable** if:
$$\mathcal{N} = \mathcal{D} \circ \mathcal{N}^c$$
The environment can simulate what the receiver sees.
</Definition>

### Zero Quantum Capacity

<Formula title="Anti-Degradable Implies Q = 0">
For anti-degradable $\mathcal{N}$:
$$Q(\mathcal{N}) = 0$$
</Formula>

<Warning>
Anti-degradable channels cannot transmit any quantum information reliably! The environment has too much information.
</Warning>

### Proof

For anti-degradable channel:
1. $I_c(\rho, \mathcal{N}) = S(B) - S(E)$
2. Since $\mathcal{N} = \mathcal{D} \circ \mathcal{N}^c$: $S(B) \leq S(E)$ (data processing)
3. Therefore $I_c \leq 0$ for all inputs
4. Hence $Q = 0$

### Examples

<Example title="Anti-Degradable Examples">
1. **Amplitude damping** with $\gamma > 1/2$
2. **Entanglement-breaking channels** (all are anti-degradable)
3. **Measure-and-prepare channels**
</Example>

## 5. Characterizing Degradability

### Sufficient Conditions

```python
import numpy as np
from scipy.linalg import sqrtm

def test_degradability(channel_func, complement_func, dim, num_tests=100):
    """Numerically test if channel might be degradable."""
    # Check if S(E) <= S(B) for random inputs (necessary condition)
    all_pass = True

    for _ in range(num_tests):
        # Random pure state
        psi = np.random.randn(dim) + 1j * np.random.randn(dim)
        psi /= np.linalg.norm(psi)
        rho = np.outer(psi, psi.conj())

        rho_B = channel_func(rho)
        rho_E = complement_func(rho)

        S_B = von_neumann_entropy(rho_B)
        S_E = von_neumann_entropy(rho_E)

        if S_E > S_B + 1e-10:
            all_pass = False
            break

    return all_pass

def test_anti_degradability(channel_func, complement_func, dim, num_tests=100):
    """Numerically test if channel might be anti-degradable."""
    # Check if S(B) <= S(E) for random inputs
    all_pass = True

    for _ in range(num_tests):
        psi = np.random.randn(dim) + 1j * np.random.randn(dim)
        psi /= np.linalg.norm(psi)
        rho = np.outer(psi, psi.conj())

        rho_B = channel_func(rho)
        rho_E = complement_func(rho)

        S_B = von_neumann_entropy(rho_B)
        S_E = von_neumann_entropy(rho_E)

        if S_B > S_E + 1e-10:
            all_pass = False
            break

    return all_pass
```

### The Degradability Hierarchy

| Channel Type | Condition | Quantum Capacity |
|--------------|-----------|-----------------|
| Degradable | $\mathcal{N}^c = \mathcal{D} \circ \mathcal{N}$ | $Q = \max I_c$ (single-letter) |
| Neither | General | Requires regularization |
| Anti-degradable | $\mathcal{N} = \mathcal{D} \circ \mathcal{N}^c$ | $Q = 0$ |

## 6. Approximate Degradability

### $\epsilon$-Degradable Channels

<Definition title="Approximately Degradable">
$\mathcal{N}$ is $\epsilon$-degradable if there exists $\mathcal{D}$ with:
$$\|\mathcal{N}^c - \mathcal{D} \circ \mathcal{N}\|_\diamond \leq \epsilon$$
</Definition>

<Tip>
For $\epsilon$-degradable channels, quantum capacity is approximately single-letter with error $O(\epsilon)$.
</Tip>

## Summary

| Property | Definition | Consequence |
|----------|------------|-------------|
| Degradable | $\mathcal{N}^c = \mathcal{D} \circ \mathcal{N}$ | Single-letter $Q$, $Q = P$ |
| Anti-degradable | $\mathcal{N} = \mathcal{D} \circ \mathcal{N}^c$ | $Q = 0$ |
| Self-complementary | $\mathcal{N} = \mathcal{N}^c$ | $Q = 0$, boundary case |

Key examples:
- Erasure channel: always degradable
- Amplitude damping: degradable for $\gamma \leq 1/2$
- Depolarizing: neither (for intermediate noise)
- Entanglement-breaking: always anti-degradable
