## 1. Definition

<Definition term="Quantum Conditional Entropy">
For a bipartite state $\rho_{AB}$, the conditional entropy of $A$ given $B$ is:

$$S(A|B) = S(\rho_{AB}) - S(\rho_B) = S(A,B) - S(B)$$

where $\rho_B = \text{Tr}_A(\rho_{AB})$ is the reduced state.
</Definition>

Unlike classical conditional entropy, **quantum conditional entropy can be negative!**

```python
import numpy as np

def conditional_entropy(rho_AB, dims):
    """Calculate S(A|B) = S(AB) - S(B)."""
    dim_A, dim_B = dims

    # Von Neumann entropy
    def vn_entropy(rho):
        eig = np.linalg.eigvalsh(rho)
        eig = eig[eig > 1e-15]
        return -np.sum(eig * np.log2(eig))

    # Partial trace to get rho_B
    rho_reshaped = rho_AB.reshape(dim_A, dim_B, dim_A, dim_B)
    rho_B = np.trace(rho_reshaped, axis1=0, axis2=2)

    S_AB = vn_entropy(rho_AB)
    S_B = vn_entropy(rho_B)

    return S_AB - S_B

# Product state: S(A|B) = S(A)
psi_prod = np.array([1, 0, 0, 0])  # |00⟩
rho_prod = np.outer(psi_prod, psi_prod.conj())
print(f"Product |00⟩: S(A|B) = {conditional_entropy(rho_prod, (2,2)):.4f}")

# Bell state: S(A|B) = -1
psi_bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
rho_bell = np.outer(psi_bell, psi_bell.conj())
print(f"Bell state: S(A|B) = {conditional_entropy(rho_bell, (2,2)):.4f}")
```

Output:
```
Product |00⟩: S(A|B) = 0.0000
Bell state: S(A|B) = -1.0000
```

## 2. Negative Conditional Entropy

<Warning>
Classical conditional entropy is always non-negative: $H(X|Y) \geq 0$. Quantum conditional entropy can be negative: $S(A|B)$ can be as low as $-S(A)$.
</Warning>

The negativity of $S(A|B)$ indicates **quantum correlations** (entanglement):

<Formula>
$$-S(A) \leq S(A|B) \leq S(A)$$
</Formula>

For pure bipartite states:
$$S(A|B) = S(A,B) - S(B) = 0 - S(B) = -S(B) = -S(A)$$

## 3. Operational Interpretation: State Merging

<Definition term="Quantum State Merging">
Given many copies of $\rho_{AB}$ with $A$ held by Alice and $B$ by Bob, state merging transfers Alice's part to Bob using:
- $S(A|B)$ qubits of quantum communication (if positive)
- Or generates $|S(A|B)|$ EPR pairs (if negative)
</Definition>

**Negative conditional entropy means Alice can transfer her state AND generate entanglement!**

```python
def analyze_state_merging(rho_AB, dims, name):
    """Analyze state merging requirements."""
    S_A_given_B = conditional_entropy(rho_AB, dims)

    print(f"{name}:")
    if S_A_given_B > 0:
        print(f"  S(A|B) = {S_A_given_B:.4f} qubits")
        print(f"  State merging costs {S_A_given_B:.4f} qubits of communication")
    else:
        print(f"  S(A|B) = {S_A_given_B:.4f} qubits")
        print(f"  State merging generates {-S_A_given_B:.4f} EPR pairs")
    print()

# Examples
analyze_state_merging(rho_prod, (2,2), "Product state")
analyze_state_merging(rho_bell, (2,2), "Bell state")

# Partial entanglement
theta = np.pi / 6
psi_partial = np.array([np.cos(theta), 0, 0, np.sin(theta)])
rho_partial = np.outer(psi_partial, psi_partial.conj())
analyze_state_merging(rho_partial, (2,2), f"Partial (θ=π/6)")
```

## 4. Quantum Mutual Information

The quantum mutual information is:

<Formula>
$$I(A:B) = S(A) + S(B) - S(A,B) = S(A) - S(A|B) = S(B) - S(B|A)$$
</Formula>

For classical states, $0 \leq I(X:Y) \leq \min\{H(X), H(Y)\}$.

For quantum states, $0 \leq I(A:B) \leq 2\min\{S(A), S(B)\}$ due to possible negativity of conditional entropy!

<Tip>
Quantum mutual information can be up to **twice** the classical maximum! This extra capacity enables superdense coding.
</Tip>

## 5. Properties

### Chain Rule

<Formula>
$$S(A,B) = S(A) + S(B|A) = S(B) + S(A|B)$$
</Formula>

### Strong Subadditivity (restated)

<Formula>
$$S(A|BC) \leq S(A|B)$$
</Formula>

Conditioning on more systems reduces conditional entropy.

### Araki-Lieb (restated)

For pure $\rho_{ABC}$:
$$S(A|B) = -S(A|C)$$
