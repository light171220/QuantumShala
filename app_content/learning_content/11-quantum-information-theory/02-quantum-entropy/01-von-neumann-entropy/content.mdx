## 1. From Classical to Quantum Entropy

Shannon entropy measures uncertainty in a classical probability distribution. For quantum systems, we need a measure that captures:

1. Classical uncertainty about which state was prepared
2. Intrinsically quantum uncertainty from superposition
3. Correlations including entanglement

<Definition term="Von Neumann Entropy">
The von Neumann entropy of a quantum state with density matrix $\rho$ is:

$$S(\rho) = -\text{Tr}(\rho \log_2 \rho)$$

Equivalently, in terms of eigenvalues $\lambda_i$ of $\rho$:

$$S(\rho) = -\sum_i \lambda_i \log_2 \lambda_i = H(\{\lambda_i\})$$
</Definition>

The von Neumann entropy is the Shannon entropy of the eigenvalue spectrum!

<Tip>
The key insight: von Neumann entropy quantifies quantum uncertainty. For a pure state, there's no uncertainty ($S = 0$). For a maximally mixed state, uncertainty is maximal.
</Tip>

## 2. Computing Von Neumann Entropy

The practical way to compute $S(\rho)$:

1. Find the eigenvalues $\lambda_1, \ldots, \lambda_d$ of $\rho$
2. Compute $S = -\sum_i \lambda_i \log_2 \lambda_i$

```python
import numpy as np
from scipy import linalg

def von_neumann_entropy(rho):
    """
    Calculate von Neumann entropy of a density matrix.

    Args:
        rho: Density matrix (numpy array)

    Returns:
        Entropy in bits
    """
    # Get eigenvalues
    eigenvalues = np.linalg.eigvalsh(rho)

    # Filter positive eigenvalues (handle numerical errors)
    eigenvalues = eigenvalues[eigenvalues > 1e-15]

    # Compute entropy
    entropy = -np.sum(eigenvalues * np.log2(eigenvalues))

    return entropy

# Example: Pure state |0⟩
rho_pure = np.array([[1, 0], [0, 0]])
print(f"Pure state |0⟩: S = {von_neumann_entropy(rho_pure):.4f} bits")

# Example: Maximally mixed state (I/2)
rho_mixed = np.array([[0.5, 0], [0, 0.5]])
print(f"Maximally mixed (I/2): S = {von_neumann_entropy(rho_mixed):.4f} bits")

# Example: |+⟩ state
rho_plus = np.array([[0.5, 0.5], [0.5, 0.5]])
print(f"|+⟩ state: S = {von_neumann_entropy(rho_plus):.4f} bits")

# Example: Partially mixed state
rho_partial = np.array([[0.8, 0], [0, 0.2]])
print(f"Partially mixed: S = {von_neumann_entropy(rho_partial):.4f} bits")
```

Output:
```
Pure state |0⟩: S = 0.0000 bits
Maximally mixed (I/2): S = 1.0000 bits
|+⟩ state: S = 0.0000 bits
Partially mixed: S = 0.7219 bits
```

<Warning>
Note that $|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)$ is a **pure state** with zero entropy, even though it's a superposition. The entropy measures our uncertainty about the quantum state, not the outcomes of measurements.
</Warning>

## 3. Properties of Von Neumann Entropy

### Property 1: Non-negativity

<Formula>
$$S(\rho) \geq 0$$
</Formula>

with equality if and only if $\rho$ is a pure state ($\rho = |\psi\rangle\langle\psi|$).

### Property 2: Maximum Value

<Formula>
$$S(\rho) \leq \log_2 d$$
</Formula>

where $d$ is the dimension of the Hilbert space. Equality holds for the maximally mixed state $\rho = I/d$.

### Property 3: Invariance under Unitaries

<Formula>
$$S(U\rho U^\dagger) = S(\rho)$$
</Formula>

Unitary evolution preserves entropy. Entropy only changes through measurements or interactions with the environment.

### Property 4: Concavity

<Formula>
$$S\left(\sum_i p_i \rho_i\right) \geq \sum_i p_i S(\rho_i)$$
</Formula>

Mixing states increases entropy (or keeps it the same).

```python
import numpy as np

def verify_entropy_properties():
    """Verify key properties of von Neumann entropy."""

    # Property 1: Pure state has zero entropy
    psi = np.array([1, 0])
    rho_pure = np.outer(psi, psi.conj())
    print(f"Pure state entropy: {von_neumann_entropy(rho_pure):.6f} (expected 0)")

    # Property 2: Maximum entropy for maximally mixed state
    d = 4  # Dimension
    rho_max = np.eye(d) / d
    print(f"Max mixed state entropy (d={d}): {von_neumann_entropy(rho_max):.6f} (expected {np.log2(d):.6f})")

    # Property 3: Unitary invariance
    rho = np.array([[0.7, 0.1j], [-0.1j, 0.3]])
    U = np.array([[1, 1], [1, -1]]) / np.sqrt(2)  # Hadamard
    rho_transformed = U @ rho @ U.conj().T
    print(f"Before unitary: S = {von_neumann_entropy(rho):.6f}")
    print(f"After unitary: S = {von_neumann_entropy(rho_transformed):.6f}")

    # Property 4: Concavity
    rho1 = np.array([[1, 0], [0, 0]])  # |0⟩
    rho2 = np.array([[0, 0], [0, 1]])  # |1⟩
    p = 0.3
    rho_mix = p * rho1 + (1-p) * rho2

    S_mix = von_neumann_entropy(rho_mix)
    S_weighted = p * von_neumann_entropy(rho1) + (1-p) * von_neumann_entropy(rho2)

    print(f"\nConcavity check:")
    print(f"  S(mixture) = {S_mix:.6f}")
    print(f"  Weighted sum of S = {S_weighted:.6f}")
    print(f"  Concavity satisfied: {S_mix >= S_weighted - 1e-10}")

verify_entropy_properties()
```

## 4. Entropy and Purity

<Definition term="Purity">
The purity of a quantum state is:

$$\gamma(\rho) = \text{Tr}(\rho^2)$$

with $1/d \leq \gamma \leq 1$, where equality with 1 occurs for pure states.
</Definition>

Entropy and purity are related but distinct measures:
- Pure state: $\gamma = 1$, $S = 0$
- Maximally mixed: $\gamma = 1/d$, $S = \log_2 d$

<Formula>
$$S(\rho) = -\text{Tr}(\rho \log \rho) \quad \text{vs} \quad \gamma(\rho) = \text{Tr}(\rho^2)$$
</Formula>

```python
def purity(rho):
    """Calculate purity of a density matrix."""
    return np.real(np.trace(rho @ rho))

def analyze_state(name, rho):
    """Analyze entropy and purity of a state."""
    S = von_neumann_entropy(rho)
    gamma = purity(rho)
    print(f"{name}:")
    print(f"  Entropy: {S:.4f} bits")
    print(f"  Purity: {gamma:.4f}")
    print(f"  Is pure: {np.isclose(gamma, 1)}")
    print()

# Various states
analyze_state("Pure |0⟩", np.array([[1, 0], [0, 0]]))
analyze_state("Maximally mixed", np.array([[0.5, 0], [0, 0.5]]))
analyze_state("Partially mixed", np.array([[0.9, 0], [0, 0.1]]))
analyze_state("Pure |+⟩", np.array([[0.5, 0.5], [0.5, 0.5]]))
```

## 5. Bipartite Systems and Entanglement

For a bipartite system $AB$ with state $\rho_{AB}$:

<Definition term="Reduced Density Matrix">
The reduced density matrix of subsystem $A$ is:

$$\rho_A = \text{Tr}_B(\rho_{AB})$$

obtained by tracing out system $B$.
</Definition>

### Araki-Lieb Inequality

<Formula>
$$|S(\rho_A) - S(\rho_B)| \leq S(\rho_{AB}) \leq S(\rho_A) + S(\rho_B)$$
</Formula>

### Pure Bipartite States

For a pure state $|\psi_{AB}\rangle$:

$$S(\rho_{AB}) = 0 \quad \text{but} \quad S(\rho_A) = S(\rho_B) \geq 0$$

The entropy of subsystems measures entanglement!

<Definition term="Entropy of Entanglement">
For a pure bipartite state $|\psi_{AB}\rangle$:

$$E(|\psi\rangle) = S(\rho_A) = S(\rho_B)$$

This is the entanglement entropy.
</Definition>

```python
def partial_trace_B(rho_AB, dim_A, dim_B):
    """Compute partial trace over B: Tr_B(rho_AB)."""
    rho_AB = rho_AB.reshape(dim_A, dim_B, dim_A, dim_B)
    return np.trace(rho_AB, axis1=1, axis2=3)

def analyze_bipartite(name, psi_AB, dim_A, dim_B):
    """Analyze entanglement in a pure bipartite state."""
    # Density matrix
    rho_AB = np.outer(psi_AB, psi_AB.conj())

    # Reduced density matrices
    rho_A = partial_trace_B(rho_AB, dim_A, dim_B)
    rho_B = partial_trace_B(rho_AB.reshape(dim_B, dim_A, dim_B, dim_A).transpose(1,0,3,2).reshape(dim_A*dim_B, dim_A*dim_B), dim_B, dim_A)

    # Actually, let's use a simpler approach
    rho_AB_2d = rho_AB.reshape(dim_A, dim_B, dim_A, dim_B)
    rho_A = np.einsum('ijik->jk', rho_AB_2d)

    S_AB = von_neumann_entropy(rho_AB)
    S_A = von_neumann_entropy(rho_A)

    print(f"{name}:")
    print(f"  S(AB) = {S_AB:.4f} bits")
    print(f"  S(A) = {S_A:.4f} bits (entanglement entropy)")
    print()

# Product state |00⟩
psi_00 = np.array([1, 0, 0, 0])
analyze_bipartite("Product |00⟩", psi_00, 2, 2)

# Bell state |Φ+⟩ = (|00⟩ + |11⟩)/√2
psi_bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
analyze_bipartite("Bell state |Φ+⟩", psi_bell, 2, 2)

# Partially entangled state
theta = np.pi / 6
psi_partial = np.array([np.cos(theta), 0, 0, np.sin(theta)])
analyze_bipartite(f"Partial (θ=π/6)", psi_partial, 2, 2)
```

## 6. Thermodynamic Connection

Von Neumann entropy connects to thermodynamic entropy via Boltzmann's constant:

<Formula>
$$S_{\text{thermo}} = k_B \ln 2 \cdot S(\rho)$$
</Formula>

For a thermal state at temperature $T$:

<Formula>
$$\rho_{\text{thermal}} = \frac{e^{-H/k_B T}}{Z}, \quad Z = \text{Tr}(e^{-H/k_B T})$$
</Formula>

The von Neumann entropy then equals the thermodynamic entropy (in natural units).

<Tip>
This connection explains why quantum computers must operate at low temperatures: reducing thermal noise means reducing the entropy of environmental states that could disturb the computation.
</Tip>

## Summary

In this lesson, you learned about von Neumann entropy:

1. **Definition**: $S(\rho) = -\text{Tr}(\rho \log_2 \rho) = H(\text{eigenvalues})$
2. **Range**: $0 \leq S(\rho) \leq \log_2 d$
3. **Pure states**: $S = 0$ (no uncertainty)
4. **Maximally mixed**: $S = \log_2 d$ (maximum uncertainty)
5. **Entanglement**: For pure bipartite states, $S(\rho_A)$ measures entanglement
6. **Properties**: Non-negative, concave, unitarily invariant

Von Neumann entropy is the foundation for all quantum information measures.
