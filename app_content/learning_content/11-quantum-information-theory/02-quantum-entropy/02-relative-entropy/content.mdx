## 1. Definition and Motivation

<Definition term="Quantum Relative Entropy">
The quantum relative entropy between states $\rho$ and $\sigma$ is:

$$D(\rho \| \sigma) = \text{Tr}(\rho \log_2 \rho) - \text{Tr}(\rho \log_2 \sigma)$$

or equivalently:
$$D(\rho \| \sigma) = \text{Tr}(\rho (\log_2 \rho - \log_2 \sigma))$$
</Definition>

<Warning>
$D(\rho \| \sigma)$ is only finite when $\text{supp}(\rho) \subseteq \text{supp}(\sigma)$. If $\rho$ has support outside the support of $\sigma$, then $D(\rho \| \sigma) = +\infty$.
</Warning>

The relative entropy measures how "far" $\rho$ is from $\sigma$ in an information-theoretic sense.

```python
import numpy as np
from scipy import linalg

def quantum_relative_entropy(rho, sigma, tol=1e-12):
    """
    Calculate quantum relative entropy D(ρ || σ).

    Returns infinity if supp(ρ) is not contained in supp(σ).
    """
    # Eigendecomposition
    eig_rho, U_rho = np.linalg.eigh(rho)
    eig_sigma, U_sigma = np.linalg.eigh(sigma)

    # Check support condition
    for i, lam in enumerate(eig_rho):
        if lam > tol:
            # Check if this eigenvector has support in sigma
            vec = U_rho[:, i]
            sigma_component = vec.conj() @ sigma @ vec
            if sigma_component < tol:
                return np.inf

    # Compute D(ρ || σ) = Tr(ρ log ρ) - Tr(ρ log σ)
    # First term: Tr(ρ log ρ) = -S(ρ)
    eig_rho_pos = eig_rho[eig_rho > tol]
    term1 = np.sum(eig_rho_pos * np.log2(eig_rho_pos))

    # Second term: Tr(ρ log σ)
    log_sigma = U_sigma @ np.diag(np.where(eig_sigma > tol,
                                            np.log2(np.maximum(eig_sigma, tol)),
                                            -1e10)) @ U_sigma.conj().T
    term2 = np.real(np.trace(rho @ log_sigma))

    return term1 - term2

# Examples
rho = np.array([[0.8, 0.1], [0.1, 0.2]])
sigma = np.array([[0.5, 0], [0, 0.5]])  # Maximally mixed

D = quantum_relative_entropy(rho, sigma)
print(f"D(ρ || I/2) = {D:.4f} bits")
```

## 2. Key Properties

### Non-negativity (Klein's Inequality)

<Formula>
$$D(\rho \| \sigma) \geq 0$$
</Formula>

with equality if and only if $\rho = \sigma$.

### Monotonicity under Quantum Channels

<Definition term="Monotonicity">
For any quantum channel $\mathcal{E}$ (CPTP map):

$$D(\mathcal{E}(\rho) \| \mathcal{E}(\sigma)) \leq D(\rho \| \sigma)$$
</Definition>

This is the **data processing inequality** for quantum systems.

### Joint Convexity

<Formula>
$$D\left(\sum_i p_i \rho_i \| \sum_i p_i \sigma_i\right) \leq \sum_i p_i D(\rho_i \| \sigma_i)$$
</Formula>

### Relationship to Von Neumann Entropy

When $\sigma = I/d$ (maximally mixed state):

<Formula>
$$D(\rho \| I/d) = \log_2 d - S(\rho)$$
</Formula>

## 3. Operational Interpretation: Hypothesis Testing

The relative entropy has a beautiful operational interpretation in quantum hypothesis testing.

<Definition term="Quantum Hypothesis Testing">
Given many copies of either state $\rho$ (null hypothesis) or $\sigma$ (alternative), the optimal type-II error (missing $\sigma$) decreases as:

$$\beta_n^* \approx 2^{-n D(\rho \| \sigma)}$$

when the type-I error (false alarm) is bounded.
</Definition>

This is the **quantum Stein's lemma**.

```python
import numpy as np

def hypothesis_testing_simulation(rho, sigma, n_copies, n_trials=1000):
    """
    Simulate quantum hypothesis testing.

    For simplicity, we use classical simulation of measurement outcomes.
    """
    # This is a simplified classical simulation
    # True quantum hypothesis testing would use POVM optimization

    # Eigenvalues give "classical" distribution
    eig_rho = np.linalg.eigvalsh(rho)
    eig_sigma = np.linalg.eigvalsh(sigma)

    # Likelihood ratio test threshold
    D = quantum_relative_entropy(rho, sigma)

    errors = {'type1': 0, 'type2': 0}

    for _ in range(n_trials):
        # Generate samples from rho (null hypothesis true)
        samples_rho = np.random.choice(len(eig_rho), size=n_copies, p=eig_rho)

        # Likelihood ratio
        log_ratio = sum(np.log2(eig_rho[s] / max(eig_sigma[s], 1e-10))
                       for s in samples_rho)

        # Decision: accept null if ratio is high
        if log_ratio < n_copies * D / 2:  # Threshold
            errors['type1'] += 1

    print(f"n = {n_copies}: Type-I error ≈ {errors['type1']/n_trials:.4f}")
    print(f"Expected Type-II error: 2^(-n*D) = {2**(-n_copies * D):.6f}")
```

## 4. Pinsker's Inequality

<Definition term="Pinsker's Inequality">
$$D(\rho \| \sigma) \geq \frac{1}{2 \ln 2} \|\rho - \sigma\|_1^2$$

where $\|A\|_1 = \text{Tr}(\sqrt{A^\dagger A})$ is the trace norm.
</Definition>

This connects relative entropy to the trace distance, the natural metric for quantum state distinguishability.

## 5. Thermodynamic Interpretation

In quantum thermodynamics, relative entropy quantifies:

<Formula>
$$D(\rho \| \rho_{\text{thermal}}) = \beta (F(\rho) - F_{\text{eq}})$$
</Formula>

where $F$ is the free energy, $\beta = 1/k_B T$, and $\rho_{\text{thermal}} = e^{-\beta H}/Z$.

The relative entropy measures the **free energy difference** from thermal equilibrium.

<Tip>
The monotonicity of relative entropy under channels reflects the second law of thermodynamics: free energy can only decrease in contact with a thermal bath.
</Tip>

## 6. Conditional Relative Entropy

For bipartite systems:

<Definition term="Conditional Relative Entropy">
$$D(\rho_{AB} \| \sigma_{AB}) \geq D(\rho_A \| \sigma_A)$$

with equality if $\rho_{AB} = \rho_A \otimes \rho_B$ and $\sigma_{AB} = \sigma_A \otimes \sigma_B$.
</Definition>

Relative entropy is the parent quantity from which many other information measures derive.
