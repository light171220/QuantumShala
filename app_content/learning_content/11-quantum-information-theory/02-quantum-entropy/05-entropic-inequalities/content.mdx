## 1. The Entropy Cone: Foundations

The set of achievable entropy vectors forms a convex cone, which encodes fundamental constraints on quantum correlations.

<Definition term="Quantum Entropy Cone">
For n parties, the quantum entropy cone is:
$$\Gamma_n^* = \{(S_A)_{A \subseteq [n]} : \exists \rho_{1...n} \text{ such that } S_A = S(\rho_A) \text{ for all } A \subseteq [n]\}$$
This is the closure of all achievable entropy vectors from quantum states.
</Definition>

### The Structure of Entropy Vectors

For a multipartite system, the entropy vector has $2^n - 1$ components (excluding the empty set):

<Formula>
For $n=3$ parties (A, B, C), the entropy vector is:
$$\vec{S} = (S_A, S_B, S_C, S_{AB}, S_{BC}, S_{AC}, S_{ABC})$$
This lives in $\mathbb{R}^7$.
</Formula>

```python
import numpy as np
from scipy.linalg import logm
from itertools import combinations
from typing import List, Tuple, Dict

class EntropyCone:
    """Tools for analyzing entropy cones and entropic inequalities."""

    def __init__(self, n_parties: int):
        """Initialize entropy cone analyzer for n parties."""
        self.n = n_parties
        self.parties = list(range(n_parties))
        self.subsets = self._generate_subsets()
        self.dim = len(self.subsets) - 1  # Exclude empty set

    def _generate_subsets(self) -> List[Tuple]:
        """Generate all non-empty subsets of parties."""
        subsets = []
        for r in range(1, self.n + 1):
            for combo in combinations(self.parties, r):
                subsets.append(combo)
        return subsets

    def von_neumann_entropy(self, rho: np.ndarray) -> float:
        """Compute von Neumann entropy S(rho) = -Tr(rho log rho)."""
        eigenvalues = np.linalg.eigvalsh(rho)
        eigenvalues = eigenvalues[eigenvalues > 1e-15]
        if len(eigenvalues) == 0:
            return 0.0
        return -np.sum(eigenvalues * np.log2(eigenvalues))

    def partial_trace(self, rho: np.ndarray, dims: List[int],
                      keep: List[int]) -> np.ndarray:
        """
        Compute partial trace over specified subsystems.

        Args:
            rho: Full density matrix
            dims: List of dimensions for each subsystem
            keep: Indices of subsystems to keep
        """
        n = len(dims)
        total_dim = np.prod(dims)

        # Reshape to tensor
        rho_tensor = rho.reshape(dims + dims)

        # Determine which indices to trace over
        trace_over = [i for i in range(n) if i not in keep]

        for idx in sorted(trace_over, reverse=True):
            # Contract axis idx with axis idx + n
            rho_tensor = np.trace(rho_tensor, axis1=idx, axis2=idx + n)
            # Update n since we removed two axes
            n -= 1

        # Reshape back to matrix
        kept_dims = [dims[i] for i in keep]
        new_dim = np.prod(kept_dims) if kept_dims else 1
        return rho_tensor.reshape(new_dim, new_dim)

    def compute_entropy_vector(self, rho: np.ndarray,
                                dims: List[int]) -> np.ndarray:
        """
        Compute full entropy vector for a multipartite state.

        Args:
            rho: Full density matrix
            dims: Dimension of each subsystem
        """
        entropy_vector = []

        for subset in self.subsets:
            rho_reduced = self.partial_trace(rho, dims, list(subset))
            S = self.von_neumann_entropy(rho_reduced)
            entropy_vector.append(S)

        return np.array(entropy_vector)

    def subset_to_index(self, subset: Tuple) -> int:
        """Convert subset to index in entropy vector."""
        return self.subsets.index(subset)

# Example: Three-party entropy vector
np.random.seed(42)
cone = EntropyCone(3)

# Create a random three-qubit pure state
dim = 8
psi = np.random.randn(dim) + 1j * np.random.randn(dim)
psi /= np.linalg.norm(psi)
rho = np.outer(psi, psi.conj())

dims = [2, 2, 2]  # Three qubits
entropy_vec = cone.compute_entropy_vector(rho, dims)

print("Three-Party Entropy Vector")
print("=" * 50)
for subset, S in zip(cone.subsets, entropy_vec):
    subset_str = ''.join(['ABC'[i] for i in subset])
    print(f"S({subset_str}) = {S:.4f}")
```

## 2. Two-Party Entropy Cone: Complete Characterization

<Definition term="Two-Party Quantum Entropy Cone">
The two-party quantum entropy cone $\Gamma_2^*$ is completely characterized by:
1. **Non-negativity**: $S_A \geq 0$, $S_B \geq 0$, $S_{AB} \geq 0$
2. **Subadditivity**: $S_{AB} \leq S_A + S_B$
3. **Araki-Lieb inequality**: $|S_A - S_B| \leq S_{AB}$
</Definition>

<Formula>
The Araki-Lieb inequality:
$$S(\rho_{AB}) \geq |S(\rho_A) - S(\rho_B)|$$
with equality for pure states where $S_{AB} = 0$ and $S_A = S_B$.
</Formula>

```python
def check_two_party_cone(S_A: float, S_B: float, S_AB: float,
                          tol: float = 1e-10) -> Dict[str, bool]:
    """
    Check if entropy vector satisfies all 2-party cone constraints.

    Returns dictionary of constraint satisfaction.
    """
    constraints = {
        'non_negativity_A': S_A >= -tol,
        'non_negativity_B': S_B >= -tol,
        'non_negativity_AB': S_AB >= -tol,
        'subadditivity': S_AB <= S_A + S_B + tol,
        'araki_lieb': abs(S_A - S_B) <= S_AB + tol
    }
    constraints['all_satisfied'] = all(constraints.values())
    return constraints

def visualize_two_party_cone():

    for _ in range(n_samples):
        # Random bipartite state on 2x2 system
        dim = 4

        rho /= np.trace(rho)
```

## 3. Beyond Strong Subadditivity

For three or more parties, additional inequalities beyond SSA become relevant.

<Definition term="Strong Subadditivity (SSA)">
For any tripartite quantum state $\rho_{ABC}$:
$$S(ABC) + S(B) \leq S(AB) + S(BC)$$
Equivalently, $S(A|BC) \leq S(A|B)$: conditioning reduces entropy.
</Definition>

### Weak Monotonicity

<Formula>
Weak monotonicity is equivalent to SSA:
$$S(AB) + S(BC) \geq S(B) + S(ABC)$$
This is SSA rewritten with subsystems relabeled.
</Formula>

### Constrained Inequalities for Pure States

For pure global states, additional inequalities hold:

<Formula>
**Pure State Constraints**: If $\rho_{ABC}$ is pure ($S_{ABC} = 0$):
1. $S_A = S_{BC}$, $S_B = S_{AC}$, $S_C = S_{AB}$
2. $S_{AB} + S_{BC} + S_{AC} \leq S_A + S_B + S_C + S_{ABC}$ (becomes equality)
</Formula>

```python
class ThreePartyInequalities:
    """Analyze three-party entropic inequalities."""

    def __init__(self):
        self.cone = EntropyCone(3)

    def check_ssa(self, S: Dict[str, float], tol: float = 1e-10) -> Dict:
        """Check strong subadditivity for three parties."""
        # S(ABC) + S(B) <= S(AB) + S(BC)
        ssa_value = S['AB'] + S['BC'] - S['ABC'] - S['B']

        # Equivalent form: S(A|BC) <= S(A|B)
        S_A_given_BC = S['ABC'] - S['BC']
        S_A_given_B = S['AB'] - S['B']

        return {
            'ssa_satisfied': ssa_value >= -tol,
            'ssa_value': ssa_value,
            'S(A|BC)': S_A_given_BC,
            'S(A|B)': S_A_given_B,
            'conditioning_reduces': S_A_given_BC <= S_A_given_B + tol
        }

    def check_weak_monotonicity(self, S: Dict[str, float],
                                 tol: float = 1e-10) -> Dict:
        """Check weak monotonicity: S(AB) + S(BC) >= S(B) + S(ABC)."""
        wm_value = S['AB'] + S['BC'] - S['B'] - S['ABC']
        return {
            'wm_satisfied': wm_value >= -tol,
            'wm_value': wm_value
        }

    def check_all_ssa_forms(self, S: Dict[str, float]) -> Dict:
        """Check all six forms of SSA (for all permutations)."""
        forms = []

        # SSA: S(XY) + S(YZ) >= S(Y) + S(XYZ) for all orderings
        orderings = [
            ('A', 'B', 'C'), ('A', 'C', 'B'), ('B', 'A', 'C'),
            ('B', 'C', 'A'), ('C', 'A', 'B'), ('C', 'B', 'A')
        ]

        for X, Y, Z in orderings:
            XY = ''.join(sorted([X, Y]))
            YZ = ''.join(sorted([Y, Z]))
            XYZ = 'ABC'

            value = S[XY] + S[YZ] - S[Y] - S[XYZ]
            forms.append({
                'form': f"S({XY}) + S({YZ}) >= S({Y}) + S({XYZ})",
                'value': value,
                'satisfied': value >= -1e-10
            })

        return {'forms': forms, 'all_satisfied': all(f['satisfied'] for f in forms)}
```

## 4. Polymatroid Inequalities

<Definition term="Polymatroid">
A polymatroid is a function $h: 2^N \to \mathbb{R}$ satisfying:
1. $h(\emptyset) = 0$
2. $h(A) \leq h(B)$ for $A \subseteq B$ (monotonicity)
3. $h(A) + h(B) \geq h(A \cup B) + h(A \cap B)$ (submodularity)

Classical entropies form a polymatroid. Quantum entropies violate monotonicity.
</Definition>

<Formula>
**Ingleton Inequality** (classical only):
$$I(A;B) + I(A;B|C) + I(C;D|B) + I(A;B|D) + I(A;D|BC) \leq I(A;D)$$
Simplified form for four parties A, B, C, D.
</Formula>

```python
def check_ingleton_inequality(S: Dict[str, float], tol: float = 1e-10) -> Dict:
    """
    Check the Ingleton inequality for 4-party systems.

    Ingleton: 2*I(C;D) <= I(A;B) + I(A;C,D) + I(B;C,D) + I(C;D|A) + I(C;D|B)

    Simplified:
    S_AC + S_AD + S_BC + S_BD >= S_A + S_B + S_CD + S_ABCD
    """
    # Standard Ingleton form
    lhs = S['AC'] + S['AD'] + S['BC'] + S['BD']
    rhs = S['A'] + S['B'] + S['CD'] + S['ABCD']

    margin = lhs - rhs

    return {
        'ingleton_satisfied': margin >= -tol,
        'margin': margin,
        'lhs': lhs,
        'rhs': rhs
    }

def create_quantum_ingleton_violator():
    """
    Create a 4-party quantum state that violates Ingleton inequality.

    The key is to use entangled states strategically.
    """
    # Four qubit state: |GHZ>_AC tensor |GHZ>_BD
    # This creates specific correlation structure

    dim = 16  # 2^4

    # Bell state |Phi+> = (|00> + |11>)/sqrt(2)
    phi_plus = np.array([1, 0, 0, 1]) / np.sqrt(2)

    # Create |Phi+>_AC tensor |Phi+>_BD
    # Reorder indices: A=0, B=1, C=2, D=3
    # |Phi+>_AC means entanglement between parties 0 and 2

    psi = np.zeros(dim, dtype=complex)

    # |Phi+>_AC: |00>_AC + |11>_AC (positions 0,2 in binary)
    # |Phi+>_BD: |00>_BD + |11>_BD (positions 1,3 in binary)

    # Combined: coefficients for |0000>, |0101>, |1010>, |1111>
    # Binary: 0000=0, 0101=5, 1010=10, 1111=15

    psi[5] = 0.5   # |0101>
    psi[10] = 0.5  # |1010>
```

## 5. Quantum vs Classical Inequality Violations

<Definition term="Quantum Advantage in Entropy">
Quantum entropy differs from classical entropy in key ways:
1. **Negative conditional entropy**: $S(A|B) < 0$ possible for entangled states
2. **Superadditivity violations**: Certain classical inequalities fail
3. **Non-Shannon inequalities**: Additional constraints exist for classical systems
</Definition>

```python
class QuantumVsClassical:
    """Compare quantum and classical entropy regions."""

    def __init__(self):
        pass

    def classical_conditional_entropy(self, p_joint: np.ndarray) -> float:
        """H(X|Y) for classical joint distribution."""
        p_y = np.sum(p_joint, axis=0)
        h_xy = -np.sum(p_joint[p_joint > 1e-15] * np.log2(p_joint[p_joint > 1e-15]))
        h_y = -np.sum(p_y[p_y > 1e-15] * np.log2(p_y[p_y > 1e-15]))
        return h_xy - h_y

    def find_negative_conditional_entropy(self):
        """Find quantum states with S(A|B) < 0."""
        # Bell state: maximally entangled
        psi_bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
        rho_bell = np.outer(psi_bell, psi_bell.conj())

        # Partial trace to get rho_B
        rho_B = np.trace(rho_bell.reshape(2, 2, 2, 2), axis1=0, axis2=2)

        def entropy(rho):
            eig = np.linalg.eigvalsh(rho)
            eig = eig[eig > 1e-15]
            return -np.sum(eig * np.log2(eig)) if len(eig) > 0 else 0

        S_AB = entropy(rho_bell)
        S_B = entropy(rho_B)
        S_A_given_B = S_AB - S_B

        return {
            'S_AB': S_AB,
            'S_B': S_B,
            'S(A|B)': S_A_given_B,
            'negative': S_A_given_B < 0
        }

    def demonstrate_mutual_info_superadditivity(self):
        """
        Quantum mutual information can exhibit superadditivity-like effects
        in certain correlation measures.
        """
        # For two uses of a channel, quantum correlations can exceed classical

        # Create bipartite state with varying entanglement
        results = []
        for theta in np.linspace(0, np.pi/2, 10):
            # Parameterized state: cos(theta)|00> + sin(theta)|11>
            psi = np.array([np.cos(theta), 0, 0, np.sin(theta)])
            rho = np.outer(psi, psi.conj())

            rho_A = np.trace(rho.reshape(2, 2, 2, 2), axis1=1, axis2=3)
            rho_B = np.trace(rho.reshape(2, 2, 2, 2), axis1=0, axis2=2)

            def entropy(r):
                eig = np.linalg.eigvalsh(r)
                eig = eig[eig > 1e-15]
                return -np.sum(eig * np.log2(eig)) if len(eig) > 0 else 0

            S_A = entropy(rho_A)
            S_B = entropy(rho_B)
            S_AB = entropy(rho)

            I_AB = S_A + S_B - S_AB  # Quantum mutual information

            results.append({
                'theta': theta,
                'S_A': S_A,
                'I(A:B)': I_AB
            })

        return results
```

## 6. Connection to Quantum Error Correction

<Definition term="Entropic QEC Bounds">
Entropy constrains quantum error correction capabilities:
$$S(R)_{\mathcal{E}(\rho)} - S(RE)_{\mathcal{E}(\rho)} \leq \frac{1}{2} I(R;E)_\rho$$
where $\mathcal{E}$ is the encoding channel, R is reference, E is environment.
</Definition>

<Formula>
**Singleton Bound** (classical):
For an $[n, k, d]$ code: $k \leq n - 2(d-1)$

**Quantum Singleton Bound**:
For an $[[n, k, d]]$ quantum code: $k \leq n - 2(d-1)$
Entropy arguments prove this bound.
</Formula>

```python
class EntropicQECBounds:
    """Entropic bounds on quantum error correction."""

    def __init__(self):
        pass

    def singleton_bound(self, n: int, d: int) -> int:
        """
        Maximum k (logical qubits) for [[n, k, d]] code.
        Quantum Singleton bound: k <= n - 2(d-1) = n - 2d + 2
        """
        return max(0, n - 2 * (d - 1))

    def coherent_information_bound(self, S_output: float,
                                    S_env: float) -> float:
        """
        Coherent information I_c = S(B) - S(E) bounds capacity.
        """
        return S_output - S_env

    def demonstrate_singleton_bound(self):
        """Show how entropy proves the Singleton bound."""
        print("Quantum Singleton Bound via Entropy")
        print("-" * 50)

        # The argument uses strong subadditivity
        print("\nProof sketch using SSA:")
        print("1. Consider n-qubit code encoding k logical qubits")
        print("2. Split physical qubits into A (erased) and B (kept)")
        print("3. Let R be reference system entangled with encoded info")
        print("4. For distance d, any d-1 erasures must be correctable")
        print("5. SSA: S(R) + S(B) >= S(RB)")
        print("6. If |A| = d-1 erasures are correctable: S(R) = S(RB)")
        print("7. This requires S(B) >= S(R) = k")
        print("8. For any bipartition: min(|A|, |B|) >= d-1")
        print("9. With |A| + |B| = n: k <= |B| = n - |A| <= n - (d-1)")
        print("10. Both bipartitions give: k <= n - 2(d-1)")

        print("\nExamples:")
        for n in [5, 7, 9]:
            for d in [2, 3]:
                k_max = self.singleton_bound(n, d)
                print(f"  [[{n}, k, {d}]]: k <= {k_max}")

        print("\n  [[5, 1, 3]] achieves the bound (5-qubit code)")
        print("  [[7, 1, 3]] achieves the bound (Steane code)")

    def no_cloning_from_entropy(self):
```

## 7. Computational Approaches to Entropy Cones

<Definition term="Linear Programming Approach">
Determining if an entropy vector is achievable can be formulated as:
$$\text{Is } \vec{h} \in \Gamma_n^* \text{?}$$
For classical entropy, this is equivalent to checking finitely many linear inequalities (Shannon inequalities). For quantum entropy, the problem is more complex.
</Definition>

```python
from scipy.optimize import linprog

class EntropyConeSolver:
    """Computational tools for entropy cone problems."""

    def __init__(self, n_parties: int):
        self.n = n_parties
        self.cone = EntropyCone(n_parties)
        self.constraints = self._build_shannon_constraints()

    def _build_shannon_constraints(self) -> Dict:
        """Build Shannon entropy inequality constraints."""
        # For classical case: monotonicity + submodularity
        n = self.n
        dim = 2**n - 1  # Number of entropy components

        constraints = {'A_ub': [], 'b_ub': []}

        # Submodularity: H(A) + H(B) >= H(A union B) + H(A intersect B)
        # Implemented as: H(A union B) + H(A cap B) - H(A) - H(B) <= 0

        for subset_A in self.cone.subsets:
            for subset_B in self.cone.subsets:
                union_AB = tuple(sorted(set(subset_A) | set(subset_B)))
                intersect_AB = tuple(sorted(set(subset_A) & set(subset_B)))

                if intersect_AB and union_AB in self.cone.subsets:
                    # Create constraint row
                    row = np.zeros(dim)
                    row[self.cone.subset_to_index(union_AB)] = 1
                    row[self.cone.subset_to_index(intersect_AB)] = 1
                    row[self.cone.subset_to_index(subset_A)] = -1
                    row[self.cone.subset_to_index(subset_B)] = -1

                    # Avoid duplicates
                    if not any(np.allclose(row, r) for r in constraints['A_ub']):
                        constraints['A_ub'].append(row)
                        constraints['b_ub'].append(0)

        # Non-negativity: -H(A) <= 0 for all A
        for i in range(dim):
            row = np.zeros(dim)
            row[i] = -1
            constraints['A_ub'].append(row)
            constraints['b_ub'].append(0)

        constraints['A_ub'] = np.array(constraints['A_ub'])
        constraints['b_ub'] = np.array(constraints['b_ub'])

        return constraints

    def check_in_classical_cone(self, entropy_vec: np.ndarray) -> Dict:
        """Check if entropy vector satisfies Shannon inequalities."""
        violations = []

        for i, (row, bound) in enumerate(zip(self.constraints['A_ub'],
                                              self.constraints['b_ub'])):
            value = np.dot(row, entropy_vec)
            if value > bound + 1e-10:
                violations.append({
                    'constraint': i,
                    'value': value,
                    'bound': bound,
                    'violation': value - bound
                })

        return {
            'in_cone': len(violations) == 0,
            'violations': violations
        }

    def find_extremal_ray(self, objective: np.ndarray) -> Dict:
        """
        Find an extremal ray of the entropy cone.
        Maximizes objective subject to Shannon constraints.
        """
        # Add normalization constraint
        A_eq = np.ones((1, len(objective)))
        b_eq = np.array([1.0])

        result = linprog(
            c=-objective,  # Maximize by minimizing negative
            A_ub=self.constraints['A_ub'],
            b_ub=self.constraints['b_ub'],
            A_eq=A_eq,
            b_eq=b_eq,
            bounds=[(0, None) for _ in objective]
        )

        return {
            'success': result.success,
            'optimal_vec': result.x if result.success else None,
            'optimal_value': -result.fun if result.success else None
        }

print("\nComputational Entropy Cone Analysis")
print("=" * 60)

solver = EntropyConeSolver(3)
print(f"\nBuilt {len(solver.constraints['A_ub'])} Shannon constraints for 3 parties")

# Test with GHZ state entropy vector
ghz = create_ghz_state()
S_ghz = compute_entropy_dict(ghz)
vec_ghz = np.array([S_ghz['A'], S_ghz['B'], S_ghz['C'],
                    S_ghz['AB'], S_ghz['AC'], S_ghz['BC'], S_ghz['ABC']])

result = solver.check_in_classical_cone(vec_ghz)
print(f"\nGHZ entropy vector in classical cone: {result['in_cone']}")
if not result['in_cone']:
    print(f"  {len(result['violations'])} violations found")
    for v in result['violations'][:3]:
        print(f"    Constraint {v['constraint']}: {v['violation']:.4f} excess")
```

## 8. Advanced Topics: Non-Shannon Inequalities

<Definition term="Non-Shannon Inequality">
A non-Shannon inequality is a valid entropic constraint that cannot be derived from the basic Shannon inequalities (submodularity, monotonicity, non-negativity). The Zhang-Yeung inequality was the first discovered for 4 variables.
</Definition>

<Formula>
**Zhang-Yeung Inequality** (4 variables):
$$2I(C;D) \leq I(A;B) + I(A;C,D) + 3I(C;D|A) + I(C;D|B)$$

In terms of entropies:
$$2H(CD) + H(A) + H(B) + H(AC) + H(BC) + H(AD) + H(BD) \leq 2H(C) + 2H(D) + H(AB) + H(ACD) + H(BCD) + 2H(ABCD)$$
</Formula>

```python
def zhang_yeung_inequality(S: Dict[str, float], tol: float = 1e-10) -> Dict:
    """
    Check the Zhang-Yeung inequality for 4 parties.

    This is the first known non-Shannon inequality.
    """
    # Simplified form using mutual information
    # 2I(C;D) <= I(A;B) + I(A;CD) + 3I(C;D|A) + I(C;D|B)

    # I(X;Y) = H(X) + H(Y) - H(XY)
    I_CD = S['C'] + S['D'] - S['CD']
    I_AB = S['A'] + S['B'] - S['AB']
    I_A_CD = S['A'] + S['CD'] - S['ACD']

    # I(C;D|A) = H(AC) + H(AD) - H(A) - H(ACD)
    I_CD_given_A = S['AC'] + S['AD'] - S['A'] - S['ACD']
    I_CD_given_B = S['BC'] + S['BD'] - S['B'] - S['BCD']

    lhs = 2 * I_CD
    rhs = I_AB + I_A_CD + 3 * I_CD_given_A + I_CD_given_B

    return {
        'zhang_yeung_satisfied': lhs <= rhs + tol,
        'lhs': lhs,
        'rhs': rhs,
        'margin': rhs - lhs
    }

print("\nNon-Shannon Inequalities")
print("=" * 60)

# Create 4-party state to test
np.random.seed(123)
dim = 16
A = np.random.randn(dim, dim) + 1j * np.random.randn(dim, dim)
rho_random = A @ A.conj().T
rho_random /= np.trace(rho_random)

S_random = compute_4party_entropy_dict(rho_random)
zy_result = zhang_yeung_inequality(S_random)

print("\nZhang-Yeung inequality for random 4-qubit state:")
print(f"  LHS = 2*I(C;D) = {zy_result['lhs']:.4f}")
print(f"  RHS = I(A;B) + I(A;CD) + 3*I(C;D|A) + I(C;D|B) = {zy_result['rhs']:.4f}")
print(f"  Satisfied: {zy_result['zhang_yeung_satisfied']}")
print(f"  Margin: {zy_result['margin']:.4f}")
```

## Summary

In this lesson, you learned about the complete landscape of entropic inequalities:

1. **Entropy Cones** encode all constraints on achievable entropy vectors
2. **Two-party cone** is fully characterized by subadditivity and Araki-Lieb
3. **Strong subadditivity** is the fundamental three-party inequality
4. **Polymatroid inequalities** characterize classical entropy (not quantum)
5. **Quantum advantages** include negative conditional entropy and SSA violations
6. **Error correction bounds** follow from entropic arguments
7. **Computational methods** help explore entropy regions
8. **Non-Shannon inequalities** provide additional constraints beyond basic axioms

<Warning>
The quantum entropy cone for 4+ parties is NOT fully characterized. There may exist undiscovered entropic inequalities. This is an active area of research connecting quantum information, algebraic geometry, and optimization theory.
</Warning>
