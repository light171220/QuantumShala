## 1. Introduction to E91

In 1991, Artur Ekert proposed a revolutionary approach to QKD based on quantum entanglement and Bell's theorem. Instead of Alice preparing and sending states (like BB84), E91 uses shared entangled pairs.

<Definition term="E91 Protocol">
An entanglement-based QKD protocol where Alice and Bob share Bell pairs and detect eavesdropping by testing Bell inequalities. Security is guaranteed by the fundamental non-local correlations of entanglement.
</Definition>

The key insight: if the Bell inequality is violated, the particles are genuinely entangled and no eavesdropper could have disturbed them without detection.

## 2. The E91 Protocol

### Setup

A source produces Bell pairs in the singlet state and sends one qubit to Alice and one to Bob:

<Formula>
$$|\Psi^-\rangle = \frac{1}{\sqrt{2}}(|01\rangle - |10\rangle)$$
</Formula>

<Tip>
The singlet state has perfect anti-correlation: if Alice measures $|0\rangle$, Bob always measures $|1\rangle$ in the same basis, and vice versa.
</Tip>

### Measurement Bases

Alice and Bob each choose from three measurement angles:

<Formula>
$$\text{Alice: } a_1 = 0°, \quad a_2 = 45°, \quad a_3 = 90°$$
$$\text{Bob: } b_1 = 45°, \quad b_2 = 90°, \quad b_3 = 135°$$
</Formula>

```python
import numpy as np

def define_e91_bases():
    """
    Define the measurement bases for E91 protocol.
    """
    # Convert angles to measurement operators
    def measurement_operator(theta_deg):
        theta = np.radians(theta_deg)
        return np.array([
            [np.cos(theta/2), np.sin(theta/2)],
            [np.sin(theta/2), -np.cos(theta/2)]
        ])

    # Alice's three bases
    alice_angles = [0, 45, 90]
    alice_bases = {f'a{i+1}': measurement_operator(a)
                   for i, a in enumerate(alice_angles)}

    # Bob's three bases
    bob_angles = [45, 90, 135]
    bob_bases = {f'b{i+1}': measurement_operator(b)
                 for i, b in enumerate(bob_angles)}

    print("E91 Measurement Bases")
    print("=" * 45)
    print("Alice's angles: 0°, 45°, 90° (a1, a2, a3)")
    print("Bob's angles: 45°, 90°, 135° (b1, b2, b3)")
    print()
    print("Matching bases for key generation:")
    print("  a3 (90°) = b2 (90°) - same angle")
    print()
    print("Bell test combinations:")
    print("  a1-b1, a1-b3, a2-b1, a2-b3")

    return alice_bases, bob_bases

alice_bases, bob_bases = define_e91_bases()
```

### Protocol Steps

**Step 1: Entanglement Distribution**
A source sends Bell pairs - one qubit to Alice, one to Bob.

**Step 2: Random Measurement**
Each party randomly chooses one of their three bases and measures.

```python
import numpy as np

def e91_measurement(singlet, alice_angle, bob_angle):
    """
    Simulate E91 measurement on a singlet state.

    Args:
        singlet: the |Psi-⟩ state
        alice_angle: Alice's measurement angle in degrees
        bob_angle: Bob's measurement angle in degrees

    Returns:
        alice_result, bob_result: measurement outcomes (+1 or -1)
    """
    # Convert to radians
    a = np.radians(alice_angle)
    b = np.radians(bob_angle)

    # For singlet state, correlation is:
    # E(a,b) = -cos(a - b)

    # Measurement outcomes
    theta_diff = a - b

    # Quantum correlation
    correlation = -np.cos(theta_diff)

    # Generate correlated outcomes
    # P(same) = (1 - correlation) / 2
    # P(different) = (1 + correlation) / 2
    p_same = (1 - correlation) / 2

    alice_result = np.random.choice([+1, -1])

    if np.random.random() < p_same:
        bob_result = alice_result
    else:
        bob_result = -alice_result

    return alice_result, bob_result

# Test correlation
print("E91 Measurement Correlations:")
print("=" * 50)

test_pairs = [(0, 90), (45, 45), (0, 45), (45, 90)]
singlet = np.array([0, 1, -1, 0]) / np.sqrt(2)

for a_angle, b_angle in test_pairs:
    # Run many trials
    results = [e91_measurement(singlet, a_angle, b_angle)
               for _ in range(10000)]
    correlation = np.mean([a * b for a, b in results])
    expected = -np.cos(np.radians(a_angle - b_angle))
    print(f"Angles ({a_angle}°, {b_angle}°): "
          f"Measured E = {correlation:.3f}, "
          f"Expected = {expected:.3f}")
```

**Step 3: Basis Announcement**
Alice and Bob publicly announce which bases they used (not the results).

**Step 4: Key Generation vs Bell Test**
- **Same basis** (a3 = b2 = 90°): Use for key generation
- **Different bases** (a1-b1, a1-b3, a2-b1, a2-b3): Use for Bell test

```python
import numpy as np

def e91_protocol_round(num_pairs):
    """
    Run E91 protocol and separate key bits from Bell test bits.
    """
    alice_angles = [0, 45, 90]  # a1, a2, a3
    bob_angles = [45, 90, 135]  # b1, b2, b3

    singlet = np.array([0, 1, -1, 0]) / np.sqrt(2)

    # Storage
    key_bits_alice = []
    key_bits_bob = []
    bell_test_data = []

    for _ in range(num_pairs):
        # Random basis choice
        a_idx = np.random.choice([0, 1, 2])
        b_idx = np.random.choice([0, 1, 2])

        a_angle = alice_angles[a_idx]
        b_angle = bob_angles[b_idx]

        # Measure
        a_result, b_result = e91_measurement(singlet, a_angle, b_angle)

        # Categorize
        if a_angle == 90 and b_angle == 90:
            # Same basis - key generation
            # Convert +1/-1 to 0/1
            key_bits_alice.append(0 if a_result == 1 else 1)
            key_bits_bob.append(0 if b_result == 1 else 1)
        else:
            # Different bases - Bell test
            bell_test_data.append({
                'a_angle': a_angle,
                'b_angle': b_angle,
                'a_result': a_result,
                'b_result': b_result
            })

    return key_bits_alice, key_bits_bob, bell_test_data

# Run protocol
np.random.seed(42)
alice_key, bob_key, bell_data = e91_protocol_round(10000)

print("E91 Protocol Results:")
print("=" * 50)
print(f"Total pairs: 10000")
print(f"Key bits generated: {len(alice_key)}")
print(f"Bell test pairs: {len(bell_data)}")
print(f"Key match rate: {np.mean(np.array(alice_key) != np.array(bob_key)):.2%} error")
print("(For singlet state, Alice and Bob's bits are ANTI-correlated)")
```

## 3. The CHSH Bell Test

The security of E91 comes from testing the CHSH inequality:

<Formula>
$$S = |E(a_1, b_1) - E(a_1, b_3) + E(a_2, b_1) + E(a_2, b_3)|$$
$$\text{Classical limit: } S \leq 2$$
$$\text{Quantum maximum: } S = 2\sqrt{2} \approx 2.83$$
</Formula>

```python
import numpy as np

def calculate_chsh(bell_test_data):
    """
    Calculate the CHSH value from Bell test data.
    """
    # Separate data by angle combinations
    combinations = {
        (0, 45): [],    # a1, b1
        (0, 135): [],   # a1, b3
        (45, 45): [],   # a2, b1
        (45, 135): [],  # a2, b3
    }

    for d in bell_test_data:
        key = (d['a_angle'], d['b_angle'])
        if key in combinations:
            combinations[key].append(d['a_result'] * d['b_result'])

    # Calculate expectation values
    E = {}
    print("CHSH Correlation Values:")
    print("-" * 45)
    for angles, products in combinations.items():
        if len(products) > 0:
            E[angles] = np.mean(products)
            expected = -np.cos(np.radians(angles[0] - angles[1]))
            print(f"E({angles[0]}°, {angles[1]}°) = {E[angles]:.3f} "
                  f"(expected: {expected:.3f})")

    # CHSH value
    if len(E) == 4:
        S = abs(E[(0, 45)] - E[(0, 135)] + E[(45, 45)] + E[(45, 135)])
        return S, E
    return None, E

# Calculate CHSH from our data
S, correlations = calculate_chsh(bell_data)

print()
print("=" * 45)
print(f"CHSH Value S = {S:.3f}")
print(f"Classical bound: 2.000")
print(f"Quantum bound: {2*np.sqrt(2):.3f}")
print(f"Bell violation: {'YES' if S > 2 else 'NO'}")
```

## 4. Security Through Bell Violation

If an eavesdropper (Eve) intercepts the particles, she breaks the entanglement. This reduces the Bell violation:

```python
import numpy as np

def e91_with_eavesdropper(num_pairs, eve_intercept_prob):
    """
    Simulate E91 with an eavesdropper who breaks entanglement.
    """
    alice_angles = [0, 45, 90]
    bob_angles = [45, 90, 135]

    bell_test_data = []

    for _ in range(num_pairs):
        a_idx = np.random.choice([0, 1, 2])
        b_idx = np.random.choice([0, 1, 2])

        a_angle = alice_angles[a_idx]
        b_angle = bob_angles[b_idx]

        if np.random.random() < eve_intercept_prob:
            # Eve intercepts - breaks entanglement
            # Results become uncorrelated
            a_result = np.random.choice([+1, -1])
            b_result = np.random.choice([+1, -1])
        else:
            # Normal quantum correlation
            a = np.radians(a_angle)
            b = np.radians(b_angle)
            correlation = -np.cos(a - b)
            p_same = (1 - correlation) / 2

            a_result = np.random.choice([+1, -1])
            bob_result = a_result if np.random.random() < p_same else -a_result
            b_result = bob_result

        bell_test_data.append({
            'a_angle': a_angle,
            'b_angle': b_angle,
            'a_result': a_result,
            'b_result': b_result
        })

    return bell_test_data

# Test various intercept rates
print("E91 Security Analysis: CHSH vs Eavesdropping")
print("=" * 55)
print()

for eve_prob in [0, 0.1, 0.2, 0.5, 1.0]:
    np.random.seed(42)
    data = e91_with_eavesdropper(20000, eve_prob)
    S, _ = calculate_chsh(data)
    secure = "SECURE" if S and S > 2 else "INSECURE"
    s_val = f"{S:.3f}" if S else "N/A"
    print(f"Eve intercepts {eve_prob:.0%}: S = {s_val} [{secure}]")
```

<Definition term="Device-Independent Security">
E91's security can be proven without trusting the measurement devices. If the Bell inequality is violated, the key is secure regardless of how the devices work internally. This is because Bell violation proves genuine quantum correlations exist.
</Definition>

## 5. Complete E91 Implementation

```python
import numpy as np

class E91Protocol:
    """
    Complete implementation of the E91 QKD protocol.
    """

    def __init__(self, num_pairs=10000):
        self.num_pairs = num_pairs
        self.alice_angles = [0, 45, 90]  # degrees
        self.bob_angles = [45, 90, 135]  # degrees
        self.classical_bound = 2.0
        self.quantum_bound = 2 * np.sqrt(2)

    def run_protocol(self, eve_present=False, eve_strength=0.5):
        """
        Execute the complete E91 protocol.
        """
        print("E91 Entanglement-Based QKD Protocol")
        print("=" * 55)

        # Generate and measure Bell pairs
        key_alice = []
        key_bob = []
        bell_data = []

        for _ in range(self.num_pairs):
            # Random basis selection
            a_idx = np.random.choice([0, 1, 2])
            b_idx = np.random.choice([0, 1, 2])

            a_angle = self.alice_angles[a_idx]
            b_angle = self.bob_angles[b_idx]

            # Simulate measurement
            if eve_present and np.random.random() < eve_strength:
                # Eve breaks correlation
                a_result = np.random.choice([+1, -1])
                b_result = np.random.choice([+1, -1])
            else:
                # Quantum correlation
                a_result, b_result = self._quantum_measurement(a_angle, b_angle)

            # Categorize
            if a_angle == 90 and b_angle == 90:
                # Key generation (same basis)
                key_alice.append(0 if a_result == 1 else 1)
                key_bob.append(0 if b_result == 1 else 1)
            else:
                # Bell test
                bell_data.append((a_angle, b_angle, a_result, b_result))

        print(f"Bell pairs measured: {self.num_pairs}")
        print(f"Key bits (same basis): {len(key_alice)}")
        print(f"Bell test data: {len(bell_data)}")

        # Calculate CHSH
        S = self._calculate_chsh(bell_data)
        print(f"\nCHSH Value: S = {S:.4f}")
        print(f"Classical bound: {self.classical_bound:.4f}")
        print(f"Quantum bound: {self.quantum_bound:.4f}")

        # Security check
        if S > self.classical_bound:
            print("\n*** Bell inequality VIOLATED - Key is SECURE ***")
            secure = True
        else:
            print("\n*** Bell inequality NOT violated - ABORT ***")
            secure = False

        # For singlet state, flip Bob's bits (anti-correlation)
        key_bob_corrected = [1 - b for b in key_bob]

        # Check key agreement
        errors = sum(a != b for a, b in zip(key_alice, key_bob_corrected))
        qber = errors / len(key_alice) if key_alice else 0

        print(f"\nKey error rate (after correction): {qber:.2%}")

        return {
            'secure': secure,
            'chsh_value': S,
            'key_length': len(key_alice),
            'qber': qber,
            'key_alice': key_alice,
            'key_bob': key_bob_corrected
        }

    def _quantum_measurement(self, a_angle, b_angle):
        """Simulate quantum correlated measurement."""
        a = np.radians(a_angle)
        b = np.radians(b_angle)
        correlation = -np.cos(a - b)
        p_same = (1 - correlation) / 2

        a_result = np.random.choice([+1, -1])
        b_result = a_result if np.random.random() < p_same else -a_result

        return a_result, b_result

    def _calculate_chsh(self, bell_data):
        """Calculate CHSH value from Bell test data."""
        combos = {(0, 45): [], (0, 135): [], (45, 45): [], (45, 135): []}

        for a_ang, b_ang, a_res, b_res in bell_data:
            if (a_ang, b_ang) in combos:
                combos[(a_ang, b_ang)].append(a_res * b_res)

        E = {}
        for key, vals in combos.items():
            E[key] = np.mean(vals) if vals else 0

        S = abs(E[(0, 45)] - E[(0, 135)] + E[(45, 45)] + E[(45, 135)])
        return S


# Run without eavesdropper
print("\n" + "=" * 60)
print("RUN 1: No Eavesdropper")
print("=" * 60 + "\n")
e91 = E91Protocol(num_pairs=20000)
result1 = e91.run_protocol(eve_present=False)

# Run with eavesdropper
print("\n" + "=" * 60)
print("RUN 2: With Eavesdropper (50% interception)")
print("=" * 60 + "\n")
result2 = e91.run_protocol(eve_present=True, eve_strength=0.5)
```

## 6. E91 vs BB84 Comparison

| Aspect | BB84 | E91 |
|--------|------|-----|
| **Resource** | Single photons | Entangled pairs |
| **Security basis** | No-cloning, disturbance | Bell inequality |
| **Device independence** | Limited | Can be device-independent |
| **Source** | Alice prepares states | Third-party source |
| **Detection** | QBER threshold | CHSH violation |
| **Complexity** | Simpler | More complex |

```python
def compare_protocols():
    """
    Compare BB84 and E91 QKD protocols.
    """
    print("BB84 vs E91 Protocol Comparison")
    print("=" * 60)
    print()
    print("BB84 (Bennett-Brassard 1984):")
    print("  - Alice prepares random states in Z or X basis")
    print("  - Bob measures in random basis")
    print("  - Security from measurement disturbance")
    print("  - Eavesdropper detected by QBER > 11%")
    print()
    print("E91 (Ekert 1991):")
    print("  - Source produces entangled pairs")
    print("  - Both parties measure in one of 3 bases")
    print("  - Security from Bell inequality violation")
    print("  - Eavesdropper detected by CHSH ≤ 2")
    print()
    print("Key Differences:")
    print("  1. E91 uses entanglement as resource")
    print("  2. E91 can achieve device-independent security")
    print("  3. BB84 is simpler to implement")
    print("  4. E91 source can be untrusted")

compare_protocols()
```
