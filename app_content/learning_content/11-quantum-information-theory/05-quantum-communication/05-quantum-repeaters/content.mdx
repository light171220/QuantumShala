## 1. The Distance Problem

Quantum communication through optical fibers faces exponential loss. For every ~20 km, about half the photons are lost:

<Formula>
$$P_{transmission} = e^{-\alpha L} = 10^{-\frac{\alpha_{dB} L}{10}}$$
</Formula>

Where $\alpha_{dB} \approx 0.2$ dB/km for telecom fiber.

```python
import numpy as np

def transmission_probability(distance_km, loss_db_per_km=0.2):
    """
    Calculate photon transmission probability through fiber.

    Args:
        distance_km: Fiber length in kilometers
        loss_db_per_km: Loss coefficient (0.2 dB/km for telecom)

    Returns:
        Probability of photon arriving
    """
    loss_db = loss_db_per_km * distance_km
    return 10 ** (-loss_db / 10)

# Analyze distance scaling
print("Direct Transmission Probability")
print("=" * 45)
distances = [10, 50, 100, 200, 500, 1000]

for d in distances:
    p = transmission_probability(d)
    print(f"{d:4} km: P = {p:.2e} ({p*100:.6f}%)")

print("\nConclusion: Direct transmission beyond ~500 km is impractical")
```

<Warning>
Unlike classical signals, we cannot simply amplify quantum states. The no-cloning theorem prevents us from making copies to boost the signal. We need a fundamentally different approach: quantum repeaters.
</Warning>

## 2. Quantum Repeaters Overview

A quantum repeater divides the total distance into shorter segments and uses entanglement swapping to connect them:

<Definition term="Quantum Repeater">
A device that extends the range of quantum communication by creating entanglement over short segments and then connecting them through entanglement swapping, without needing to transmit quantum information directly over the full distance.
</Definition>

```python
import numpy as np

def repeater_chain_analysis(total_distance, num_segments):
    """
    Analyze a quantum repeater chain.
    """
    segment_length = total_distance / num_segments

    # Direct transmission probability
    p_direct = transmission_probability(total_distance)

    # Per-segment probability
    p_segment = transmission_probability(segment_length)

    # Need entanglement in each segment (simplified model)
    # Actual probability is more complex due to swapping
    p_repeater = p_segment ** num_segments

    # With heralded entanglement (can retry)
    # Expected attempts per segment: 1/p_segment
    expected_time_direct = 1 / p_direct if p_direct > 0 else float('inf')
    expected_time_repeater = num_segments / p_segment

    print(f"Repeater Chain Analysis: {total_distance} km")
    print("=" * 55)
    print(f"Number of segments: {num_segments}")
    print(f"Segment length: {segment_length:.1f} km")
    print(f"Direct transmission P: {p_direct:.2e}")
    print(f"Per-segment P: {p_segment:.4f}")
    print(f"")
    print(f"Expected attempts (direct): {expected_time_direct:.2e}")
    print(f"Expected attempts (repeater): {expected_time_repeater:.1f}")

    return p_direct, p_segment

# Compare scenarios

print()
```

## 3. Entanglement Swapping

The core operation of quantum repeaters is **entanglement swapping**: connecting two independent entangled pairs into one longer-range entangled pair.

<Formula>
$$|\Phi^+\rangle_{AB} \otimes |\Phi^+\rangle_{CD} \xrightarrow{\text{BSM on BC}} |\Phi^+\rangle_{AD}$$
</Formula>

```python
import numpy as np

def entanglement_swapping():
    """
    Demonstrate entanglement swapping.

    Bell measurement on B,C
    Final: A-D entangled (distant parties!)
    """
```

## 4. Multi-Segment Chains

Real quantum repeaters chain multiple swapping operations:

```python
import numpy as np

class QuantumRepeaterChain:
    """
    Simulate a quantum repeater chain.
    """

    def __init__(self, total_distance, num_repeaters):
        self.total_distance = total_distance
        self.num_repeaters = num_repeaters
        self.num_segments = num_repeaters + 1
        self.segment_length = total_distance / self.num_segments

    def simulate_entanglement_generation(self, success_prob=None):
        """
        Simulate entanglement generation in each segment.
        """
        if success_prob is None:
            success_prob = transmission_probability(self.segment_length)

        # Each segment attempts until success
        attempts_per_segment = []
        for i in range(self.num_segments):
            attempts = 1
            while np.random.random() > success_prob:
                attempts += 1
            attempts_per_segment.append(attempts)

        return attempts_per_segment

    def simulate_swapping(self, swap_success_prob=0.5):
        """
        Simulate entanglement swapping at each repeater.
        """
        # Binary tree of swappings
        successful_swaps = 0
        failed_swaps = 0

        for i in range(self.num_repeaters):
            if np.random.random() < swap_success_prob:
                successful_swaps += 1
            else:
                failed_swaps += 1

        return successful_swaps, failed_swaps

    def run_protocol(self, num_trials=1000):
        """
        Run the full repeater protocol.
        """
        results = {
            'total_attempts': [],
            'success': []
        }

        p_segment = transmission_probability(self.segment_length)
        p_swap = 0.5  # Typical BSM success

        for _ in range(num_trials):
            # Generate entanglement in all segments
            attempts = self.simulate_entanglement_generation(p_segment)
            total_attempts = sum(attempts)

            # Perform swapping
            swaps_ok, swaps_fail = self.simulate_swapping(p_swap)

            success = (swaps_fail == 0)

            results['total_attempts'].append(total_attempts)
            results['success'].append(success)

        return results


# Compare different chain lengths
print("Quantum Repeater Chain Performance")
print("=" * 55)

for num_rep in [0, 1, 3, 7, 15]:
    if num_rep == 0:
        # Direct transmission
        p = transmission_probability(1000)
        print(f"Direct (no repeaters): P = {p:.2e}")
    else:
        chain = QuantumRepeaterChain(1000, num_rep)
        results = chain.run_protocol(1000)

        success_rate = np.mean(results['success'])
        avg_attempts = np.mean(results['total_attempts'])

        print(f"{num_rep} repeaters ({chain.num_segments} segments, "
              f"{chain.segment_length:.0f} km each):")
        print(f"  Success rate: {success_rate:.2%}")
        print(f"  Avg attempts: {avg_attempts:.1f}")
```

## 5. Entanglement Purification

Real quantum channels introduce errors. **Entanglement purification** improves fidelity by sacrificing some entangled pairs:

<Definition term="Entanglement Purification">
A protocol that takes multiple copies of low-fidelity entangled pairs and produces fewer copies with higher fidelity. It's essential for long-distance quantum communication where noise accumulates.
</Definition>

```python
import numpy as np

def purification_round(fidelity_in, num_pairs=2):
    """
    Simulate one round of entanglement purification.

    Uses the BBPSSW protocol (simplified):
    - Take 2 pairs with fidelity F
    - CNOT operations + measurement
    - Keep one pair with improved fidelity (probability dependent)

    Args:
        fidelity_in: Input pair fidelity
        num_pairs: Number of pairs consumed (usually 2)

    Returns:
        fidelity_out: Output fidelity if successful
        p_success: Probability of success
    """
    F = fidelity_in

    # BBPSSW purification formula
    # F_out = (F^2 + (1-F)^2/9) / (F^2 + 2*F*(1-F)/3 + 5*(1-F)^2/9)

    numerator = F**2 + (1-F)**2 / 9
    denominator = F**2 + 2*F*(1-F)/3 + 5*(1-F)**2 / 9

    F_out = numerator / denominator
    p_success = denominator

    return F_out, p_success

def purification_analysis():
    """
    Analyze purification effectiveness.
    """
    print("Entanglement Purification Analysis")
    print("=" * 55)

    # Different initial fidelities
    initial_fidelities = [0.7, 0.8, 0.9, 0.95]

    for F_init in initial_fidelities:
        print(f"\nInitial fidelity: {F_init:.2f}")
        F = F_init
        round_num = 0

        while F < 0.99 and round_num < 5:
            F_new, p_success = purification_round(F)
            print(f"  Round {round_num+1}: F = {F:.4f} → {F_new:.4f} "
                  f"(P_success = {p_success:.3f})")
            F = F_new
            round_num += 1

purification_analysis()
```

## 6. Complete Repeater Architecture

```python
import numpy as np

class FullQuantumRepeater:
    """
    Full quantum repeater with generation, purification, and swapping.
    """

    def __init__(self, distance, num_segments, target_fidelity=0.95):
        self.distance = distance
        self.num_segments = num_segments
        self.segment_length = distance / num_segments
        self.target_fidelity = target_fidelity

    def channel_fidelity(self, length):
        """
        Model fidelity degradation with distance.
        """
        # Simplified: F = 0.5 + 0.5 * exp(-L/L0)
        L0 = 50  # characteristic length in km
        return 0.5 + 0.5 * np.exp(-length / L0)

    def generate_segment_entanglement(self):
        """
        Generate entanglement for one segment.
        """
        p_success = transmission_probability(self.segment_length)
        fidelity = self.channel_fidelity(self.segment_length)

        attempts = 1
        while np.random.random() > p_success:
            attempts += 1

        return fidelity, attempts

    def purify_to_target(self, initial_fidelity):
        """
        Purify until reaching target fidelity.
        """
        F = initial_fidelity
        pairs_used = 1
        rounds = 0

        while F < self.target_fidelity and rounds < 10:
            F_new, p_success = purification_round(F)
            if np.random.random() < p_success:
                F = F_new
                rounds += 1
            pairs_used += 1

        return F, pairs_used, rounds

    def run_full_protocol(self):
        """
        Execute the complete repeater protocol.
        """
        results = {
            'segment_fidelities': [],
            'segment_attempts': [],
            'purification_rounds': [],
            'final_fidelity': 0
        }

        # Generate and purify each segment
        for i in range(self.num_segments):
            F_init, attempts = self.generate_segment_entanglement()
            F_purified, pairs, rounds = self.purify_to_target(F_init)

            results['segment_fidelities'].append(F_purified)
            results['segment_attempts'].append(attempts * pairs)
            results['purification_rounds'].append(rounds)

        # Perform swapping (simplified: fidelity multiplication)
        # Real: more complex due to imperfect BSM
        final_F = np.prod(results['segment_fidelities']) ** (1/self.num_segments)
        results['final_fidelity'] = final_F

        return results


# Demonstrate
print("Full Quantum Repeater Simulation")
print("=" * 60)

for n_seg in [2, 4, 8]:
    repeater = FullQuantumRepeater(1000, n_seg, target_fidelity=0.95)
    result = repeater.run_full_protocol()

    total_attempts = sum(result['segment_attempts'])
    total_purif = sum(result['purification_rounds'])

    print(f"\n{n_seg} segments ({1000/n_seg:.0f} km each):")
    print(f"  Total generation attempts: {total_attempts}")
    print(f"  Total purification rounds: {total_purif}")
    print(f"  Final fidelity: {result['final_fidelity']:.4f}")
```

## 7. Quantum Memory Requirements

Quantum repeaters require **quantum memories** to store entanglement while waiting for other segments:

```python
def memory_requirements(num_segments, generation_rate_hz, memory_coherence_s):
    """
    Analyze quantum memory requirements for repeaters.
    """
    print("Quantum Memory Requirements")
    print("=" * 50)

    # Time to generate entanglement in all segments
    # (assuming parallel generation)
    p_segment = 0.1  # 10% success per attempt
    avg_attempts = 1 / p_segment
    time_per_segment = avg_attempts / generation_rate_hz

    # Total time including swapping
    total_time = time_per_segment * np.log2(num_segments)  # Parallel swapping

    print(f"Segments: {num_segments}")
    print(f"Generation rate: {generation_rate_hz} Hz")
    print(f"Memory coherence: {memory_coherence_s} s")
    print(f"")
    print(f"Time per segment: {time_per_segment:.3f} s")
    print(f"Total protocol time: {total_time:.3f} s")
    print(f"")

    if total_time < memory_coherence_s:
        print("FEASIBLE: Memory coherence exceeds protocol time")
    else:
        print("NOT FEASIBLE: Need longer coherence time")

    return total_time

memory_requirements(8, 1000, 1.0)
print()
memory_requirements(8, 10000, 1.0)
```
