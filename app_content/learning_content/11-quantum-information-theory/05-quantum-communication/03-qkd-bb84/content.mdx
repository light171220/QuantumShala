## 1. Introduction to Quantum Key Distribution

In 1984, Charles Bennett and Gilles Brassard proposed the first quantum key distribution protocol, BB84. It uses quantum mechanics to generate a shared secret key between two parties (Alice and Bob) with security guaranteed by the laws of physics.

<Definition term="Quantum Key Distribution (QKD)">
A method for securely sharing cryptographic keys using quantum mechanical properties. Unlike classical key exchange (which relies on computational hardness), QKD security is based on fundamental physics: the no-cloning theorem and the disturbance caused by measurement.
</Definition>

<Warning>
QKD does not transmit secret messages directly. It generates a shared random key that can then be used with classical encryption (like one-time pad) for provably secure communication.
</Warning>

## 2. The BB84 Protocol

### The Two Bases

BB84 uses two measurement bases:

<Formula>
$$\text{Z-basis (rectilinear):} \quad |0\rangle, |1\rangle$$
$$\text{X-basis (diagonal):} \quad |+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle), \quad |-\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle)$$
</Formula>

| Bit Value | Z-basis State | X-basis State |
|-----------|---------------|---------------|
| 0 | $\|0\rangle$ | $\|+\rangle$ |
| 1 | $\|1\rangle$ | $\|-\rangle$ |

```python
import numpy as np

def bb84_states():
    """
    Define the four BB84 states.
    """
    states = {
        ('0', 'Z'): np.array([1, 0]),           # |0⟩
        ('1', 'Z'): np.array([0, 1]),           # |1⟩
        ('0', 'X'): np.array([1, 1]) / np.sqrt(2),   # |+⟩
        ('1', 'X'): np.array([1, -1]) / np.sqrt(2),  # |-⟩
    }

    print("BB84 States:")
    print("=" * 40)
    for (bit, basis), state in states.items():
        print(f"Bit {bit}, Basis {basis}: {np.round(state, 3)}")

    return states

states = bb84_states()
```

### Protocol Steps

**Step 1: Alice's Preparation**
Alice generates random bits and random bases, then prepares qubits:

```python
import numpy as np

def alice_prepare(num_bits):
    """
    Alice prepares random bits in random bases.

    Returns:
        bits: Alice's random bits
        bases: Alice's random bases ('Z' or 'X')
        states: The quantum states sent
    """
    bits = np.random.randint(0, 2, num_bits)
    bases = np.random.choice(['Z', 'X'], num_bits)

    # Prepare states
    state_dict = {
        (0, 'Z'): np.array([1, 0]),
        (1, 'Z'): np.array([0, 1]),
        (0, 'X'): np.array([1, 1]) / np.sqrt(2),
        (1, 'X'): np.array([1, -1]) / np.sqrt(2),
    }

    states = [state_dict[(bits[i], bases[i])] for i in range(num_bits)]

    return bits, bases, states

# Example
np.random.seed(42)
alice_bits, alice_bases, alice_states = alice_prepare(10)
print("Alice's preparation:")
print(f"Bits:  {alice_bits}")
print(f"Bases: {alice_bases}")
```

**Step 2: Bob's Measurement**
Bob chooses random bases and measures each qubit:

```python
import numpy as np

def bob_measure(states, num_bits):
    """
    Bob measures received states in random bases.

    Returns:
        bases: Bob's measurement bases
        results: Measurement outcomes
    """
    bases = np.random.choice(['Z', 'X'], num_bits)
    results = []

    for i, state in enumerate(states):
        if bases[i] == 'Z':
            # Measure in Z-basis
            prob_0 = np.abs(state[0])**2
            result = 0 if np.random.random() < prob_0 else 1
        else:
            # Measure in X-basis
            # Project onto |+⟩ and |-⟩
            plus = np.array([1, 1]) / np.sqrt(2)
            minus = np.array([1, -1]) / np.sqrt(2)
            prob_plus = np.abs(np.vdot(plus, state))**2
            result = 0 if np.random.random() < prob_plus else 1

        results.append(result)

    return bases, np.array(results)

# Bob measures
bob_bases, bob_results = bob_measure(alice_states, 10)
print("\nBob's measurement:")
print(f"Bases:   {bob_bases}")
print(f"Results: {bob_results}")
```

**Step 3: Basis Reconciliation (Sifting)**
Alice and Bob publicly compare bases and keep only the bits where they used the same basis:

```python
import numpy as np

def sift_key(alice_bits, alice_bases, bob_bases, bob_results):
    """
    Sift the key by keeping only matching-basis bits.

    Returns:
        alice_key: Alice's sifted key
        bob_key: Bob's sifted key
        matching_indices: Which positions matched
    """
    matching = alice_bases == bob_bases
    alice_key = alice_bits[matching]
    bob_key = bob_results[matching]

    return alice_key, bob_key, np.where(matching)[0]

alice_key, bob_key, matches = sift_key(alice_bits, alice_bases, bob_bases, bob_results)
print("\nAfter sifting:")
print(f"Matching indices: {matches}")
print(f"Alice's key: {alice_key}")
print(f"Bob's key:   {bob_key}")
print(f"Keys match: {np.all(alice_key == bob_key)}")
```

## 3. Why BB84 is Secure

The security of BB84 comes from two quantum properties:

### 1. No-Cloning Theorem

An eavesdropper (Eve) cannot copy the quantum states to measure them in both bases.

<Definition term="No-Cloning Theorem">
It is impossible to create an identical copy of an arbitrary unknown quantum state. This prevents Eve from keeping a copy of each qubit to measure later after learning the bases.
</Definition>

### 2. Measurement Disturbance

If Eve measures in the wrong basis, she disturbs the state:

```python
import numpy as np

def demonstrate_measurement_disturbance():
    """
    Show how measuring in the wrong basis disturbs the state.
    """
    print("Measurement Disturbance in BB84")
    print("=" * 50)

    # Alice sends |+⟩ (bit 0 in X-basis)
    plus = np.array([1, 1]) / np.sqrt(2)
    print(f"\nAlice sends |+⟩ (X-basis, bit 0)")

    # If Eve measures in Z-basis (wrong)
    print("\nEve intercepts and measures in Z-basis:")
    prob_0 = np.abs(plus[0])**2
    prob_1 = np.abs(plus[1])**2
    print(f"  P(Eve gets 0) = {prob_0:.1%}")
    print(f"  P(Eve gets 1) = {prob_1:.1%}")

    # Eve must resend something - say she got |0⟩
    print("\nEve got |0⟩ and sends |0⟩ to Bob")
    eve_sends = np.array([1, 0])

    # Bob measures in X-basis (correct basis)
    plus_proj = np.array([1, 1]) / np.sqrt(2)
    minus_proj = np.array([1, -1]) / np.sqrt(2)

    prob_plus = np.abs(np.vdot(plus_proj, eve_sends))**2
    prob_minus = np.abs(np.vdot(minus_proj, eve_sends))**2

    print(f"\nBob measures in X-basis:")
    print(f"  P(Bob gets 0, i.e., |+⟩) = {prob_plus:.1%}")
    print(f"  P(Bob gets 1, i.e., |-⟩) = {prob_minus:.1%}")

    print(f"\n*** Error rate when Eve intercepts: 25% ***")
    print("(50% chance Eve uses wrong basis × 50% Bob gets wrong bit)")

demonstrate_measurement_disturbance()
```

## 4. Detecting Eavesdropping

After key sifting, Alice and Bob sacrifice some bits to check for errors:

```python
import numpy as np

def check_for_eavesdropping(alice_key, bob_key, sample_fraction=0.5):
    """
    Check for eavesdropping by comparing a sample of the key.

    Args:
        alice_key: Alice's sifted key
        bob_key: Bob's sifted key
        sample_fraction: Fraction of bits to reveal for checking

    Returns:
        qber: Quantum Bit Error Rate
        remaining_alice_key: Unused bits for final key
        remaining_bob_key: Unused bits for final key
    """
    n = len(alice_key)
    sample_size = int(n * sample_fraction)

    # Randomly select bits to check
    check_indices = np.random.choice(n, sample_size, replace=False)
    keep_indices = np.setdiff1d(np.arange(n), check_indices)

    # Calculate QBER
    errors = np.sum(alice_key[check_indices] != bob_key[check_indices])
    qber = errors / sample_size if sample_size > 0 else 0

    # Keep remaining bits
    remaining_alice = alice_key[keep_indices]
    remaining_bob = bob_key[keep_indices]

    return qber, remaining_alice, remaining_bob

# Simulate with no eavesdropper
np.random.seed(42)
alice_bits, alice_bases, alice_states = alice_prepare(100)
bob_bases, bob_results = bob_measure(alice_states, 100)
alice_key, bob_key, _ = sift_key(alice_bits, alice_bases, bob_bases, bob_results)

qber, final_alice, final_bob = check_for_eavesdropping(alice_key, bob_key)
print(f"QBER (no eavesdropper): {qber:.2%}")
print(f"Final key length: {len(final_alice)} bits")
```

<Definition term="Quantum Bit Error Rate (QBER)">
The fraction of bits that differ between Alice's and Bob's sifted keys. In a perfect channel with no eavesdropper, QBER = 0. An intercept-resend attack introduces QBER = 25%.
</Definition>

## 5. Complete BB84 Implementation

```python
import numpy as np

class BB84Protocol:
    """
    Complete implementation of the BB84 QKD protocol.
    """

    def __init__(self, num_qubits=1000, eve_present=False):
        self.num_qubits = num_qubits
        self.eve_present = eve_present
        self.threshold_qber = 0.11  # Theoretical maximum for security

    def run_protocol(self):
        """
        Execute the complete BB84 protocol.
        """
        print("BB84 Quantum Key Distribution")
        print("=" * 50)

        # Step 1: Alice prepares
        alice_bits = np.random.randint(0, 2, self.num_qubits)
        alice_bases = np.random.choice(['Z', 'X'], self.num_qubits)
        states = self._prepare_states(alice_bits, alice_bases)

        print(f"Alice prepared {self.num_qubits} qubits")

        # Step 2: Eve intercepts (if present)
        if self.eve_present:
            states, eve_bits = self._eve_intercept(states)
            print("*** EVE IS EAVESDROPPING ***")

        # Step 3: Bob measures
        bob_bases = np.random.choice(['Z', 'X'], self.num_qubits)
        bob_results = self._measure_states(states, bob_bases)

        print(f"Bob measured all qubits")

        # Step 4: Basis reconciliation
        matching = alice_bases == bob_bases
        alice_sifted = alice_bits[matching]
        bob_sifted = bob_results[matching]

        print(f"After sifting: {len(alice_sifted)} bits remain")

        # Step 5: Error checking
        n = len(alice_sifted)
        check_size = n // 2
        check_idx = np.random.choice(n, check_size, replace=False)

        errors = np.sum(alice_sifted[check_idx] != bob_sifted[check_idx])
        qber = errors / check_size

        print(f"QBER: {qber:.2%}")

        # Step 6: Security decision
        if qber > self.threshold_qber:
            print("*** ABORT: QBER too high, possible eavesdropping! ***")
            return None, None, qber

        # Step 7: Extract final key
        keep_idx = np.setdiff1d(np.arange(n), check_idx)
        final_key = alice_sifted[keep_idx]

        print(f"Final key length: {len(final_key)} bits")
        print(f"Protocol {'FAILED' if qber > 0.11 else 'SUCCEEDED'}")

        return final_key, qber, len(final_key)

    def _prepare_states(self, bits, bases):
        states = []
        for bit, basis in zip(bits, bases):
            if basis == 'Z':
                state = np.array([1, 0]) if bit == 0 else np.array([0, 1])
            else:
                state = np.array([1, 1])/np.sqrt(2) if bit == 0 else np.array([1, -1])/np.sqrt(2)
            states.append(state)
        return states

    def _measure_states(self, states, bases):
        results = []
        for state, basis in zip(states, bases):
            if basis == 'Z':
                prob_0 = np.abs(state[0])**2
            else:
                plus = np.array([1, 1]) / np.sqrt(2)
                prob_0 = np.abs(np.vdot(plus, state))**2

            result = 0 if np.random.random() < prob_0 else 1
            results.append(result)

        return np.array(results)

    def _eve_intercept(self, states):
        """
        Eve performs intercept-resend attack.
        """
        eve_bases = np.random.choice(['Z', 'X'], len(states))
        eve_bits = []
        new_states = []

        for state, basis in zip(states, eve_bases):
            # Eve measures
            if basis == 'Z':
                prob_0 = np.abs(state[0])**2
            else:
                plus = np.array([1, 1]) / np.sqrt(2)
                prob_0 = np.abs(np.vdot(plus, state))**2

            eve_bit = 0 if np.random.random() < prob_0 else 1
            eve_bits.append(eve_bit)

            # Eve resends
            if basis == 'Z':
                new_state = np.array([1, 0]) if eve_bit == 0 else np.array([0, 1])
            else:
                new_state = np.array([1, 1])/np.sqrt(2) if eve_bit == 0 else np.array([1, -1])/np.sqrt(2)

            new_states.append(new_state)

        return new_states, eve_bits


# Run without eavesdropper
print("\n=== RUN 1: No Eavesdropper ===\n")
bb84_safe = BB84Protocol(num_qubits=1000, eve_present=False)
key1, qber1, length1 = bb84_safe.run_protocol()

# Run with eavesdropper
print("\n=== RUN 2: With Eavesdropper ===\n")
bb84_attack = BB84Protocol(num_qubits=1000, eve_present=True)
key2, qber2, length2 = bb84_attack.run_protocol()
```

## 6. Security Analysis

The intercept-resend attack introduces a characteristic QBER:

<Formula>
$$\text{QBER}_{\text{intercept-resend}} = \frac{1}{4} = 25\%$$
</Formula>

This comes from:
- 50% chance Eve uses the wrong basis
- 50% chance Bob gets the wrong result when Eve used wrong basis
- Total: 0.5 × 0.5 = 0.25 = 25%

```python
def security_analysis():
    """
    Analyze BB84 security bounds.
    """
    print("BB84 Security Analysis")
    print("=" * 50)

    print("\nIntercept-Resend Attack:")
    print("  P(Eve wrong basis) = 50%")
    print("  P(Error | Eve wrong basis) = 50%")
    print("  Expected QBER = 25%")

    print("\nSecurity Threshold:")
    print("  Maximum tolerable QBER ≈ 11%")
    print("  Above this, secure key extraction is impossible")

    print("\nQBER Interpretation:")
    qbers = [0, 0.05, 0.11, 0.20, 0.25]
    for q in qbers:
        if q == 0:
            status = "No eavesdropping detected"
        elif q <= 0.11:
            status = "Secure key possible (with privacy amplification)"
        else:
            status = "ABORT - Too much eavesdropping"
        print(f"  QBER = {q:.0%}: {status}")

security_analysis()
```

## 7. Privacy Amplification

If QBER < 11%, Alice and Bob can extract a shorter but secure key using privacy amplification:

```python
import numpy as np

def privacy_amplification(key, qber, target_security=1e-10):
    """
    Reduce key length to amplify privacy.

    The amount of information Eve has is bounded by QBER.
    We hash the key to a shorter length to eliminate Eve's information.
    """
    n = len(key)

    # Estimate Eve's information (simplified)
    if qber == 0:
        eve_info = 0
    else:
        # Binary entropy
        h = -qber * np.log2(qber + 1e-10) - (1-qber) * np.log2(1-qber + 1e-10)
        eve_info = n * h

    # Secure key length
    secure_length = int(n - eve_info - np.log2(1/target_security))
    secure_length = max(0, secure_length)

    print(f"Privacy Amplification:")
    print(f"  Original key: {n} bits")
    print(f"  QBER: {qber:.2%}")
    print(f"  Eve's max info: ~{eve_info:.0f} bits")
    print(f"  Secure key length: {secure_length} bits")

    # In practice, use a hash function
    # Here we just truncate for demonstration
    secure_key = key[:secure_length]

    return secure_key

# Example
np.random.seed(42)
test_key = np.random.randint(0, 2, 500)
secure_key = privacy_amplification(test_key, qber=0.05)
```
