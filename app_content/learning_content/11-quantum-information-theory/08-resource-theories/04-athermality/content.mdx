## 1. Thermodynamics as Resource Theory

The resource theory of thermodynamics, also called the resource theory of athermality, provides a rigorous framework for understanding the second law and quantum thermodynamic phenomena.

<Definition term="Resource Theory of Athermality">
The resource theory of thermodynamics is defined by:
- **Free states**: Thermal (Gibbs) states $\gamma = \frac{e^{-\beta H}}{Z}$ at inverse temperature $\beta = 1/k_B T$
- **Free operations**: Thermal operations (energy-conserving interactions with a heat bath)
- **Resource**: Athermality - any deviation from thermal equilibrium
</Definition>

The partition function $Z = \text{Tr}(e^{-\beta H})$ normalizes the thermal state.

### Physical Motivation

Consider a system coupled to a large heat bath at temperature $T$:

1. The bath is "free" - we can access unlimited thermal resources
2. Energy-conserving interactions are "free" - no external work required
3. Any out-of-equilibrium state requires "work" to prepare

<Tip>
The resource-theoretic approach makes the second law of thermodynamics a mathematical theorem: thermal operations cannot create athermality from thermal states.
</Tip>

## 2. Thermal Operations

<Definition term="Thermal Operations">
A thermal operation is a quantum channel $\mathcal{E}$ that can be written as:

$$\mathcal{E}(\rho) = \text{Tr}_B\left[U(\rho \otimes \gamma_B)U^\dagger\right]$$

where:
- $\gamma_B$ is a thermal state of the bath
- $U$ is a global unitary satisfying $[U, H_S + H_B] = 0$ (energy conservation)
</Definition>

```python
import numpy as np
from typing import Tuple

def thermal_state(H: np.ndarray, beta: float) -> np.ndarray:
    """
    Compute thermal (Gibbs) state.

    Args:
        H: Hamiltonian matrix
        beta: Inverse temperature (1/kT)

    Returns:
        Thermal density matrix
    """
    exp_H = np.linalg.matrix_power(
        np.eye(H.shape[0]) + (-beta) * H + 0.5 * (beta**2) * H @ H,
        1
    )  # Approximate for small beta*H

    # More accurate: use eigendecomposition
    eigenvalues, eigenvectors = np.linalg.eigh(H)
    exp_eigenvalues = np.exp(-beta * eigenvalues)
    Z = np.sum(exp_eigenvalues)

    gamma = eigenvectors @ np.diag(exp_eigenvalues / Z) @ eigenvectors.conj().T
    return gamma

def partition_function(H: np.ndarray, beta: float) -> float:
    """Calculate partition function Z = Tr(e^{-βH})."""
    eigenvalues = np.linalg.eigvalsh(H)
    return np.sum(np.exp(-beta * eigenvalues))

# Example: Two-level system (qubit)
# H = ε|1⟩⟨1| (ground state |0⟩ has energy 0)
epsilon = 1.0  # Energy gap
H_qubit = np.array([[0, 0], [0, epsilon]])

for T in [0.5, 1.0, 2.0, 10.0]:
    beta = 1.0 / T  # k_B = 1
    gamma = thermal_state(H_qubit, beta)
    print(f"T = {T}: p(ground) = {gamma[0,0]:.4f}, p(excited) = {gamma[1,1]:.4f}")
```

### Properties of Thermal Operations

1. **Gibbs-preserving**: $\mathcal{E}(\gamma) = \gamma$
2. **Time-translation covariant**: $\mathcal{E}(e^{-iHt}\rho e^{iHt}) = e^{-iHt}\mathcal{E}(\rho)e^{iHt}$
3. **Completely positive trace-preserving (CPTP)**

## 3. Free Energy as Monotone

The key thermodynamic potential in this resource theory is the **non-equilibrium free energy**:

<Definition term="Non-equilibrium Free Energy">
$$F(\rho) = \langle H \rangle_\rho - T \cdot S(\rho) = \text{Tr}(\rho H) + \frac{1}{\beta}S(\rho)$$

where $S(\rho) = -\text{Tr}(\rho \ln \rho)$ is the von Neumann entropy.
</Definition>

The free energy difference from thermal equilibrium:

<Formula>
$$\Delta F(\rho) = F(\rho) - F(\gamma) = \frac{1}{\beta}D(\rho \| \gamma)$$
</Formula>

where $D(\rho \| \gamma)$ is the quantum relative entropy.

```python
def von_neumann_entropy(rho: np.ndarray) -> float:
    """Calculate von Neumann entropy S(ρ) = -Tr(ρ ln ρ)."""
    eigenvalues = np.linalg.eigvalsh(rho)
    eigenvalues = eigenvalues[eigenvalues > 1e-15]
    return -np.sum(eigenvalues * np.log(eigenvalues))

def free_energy(rho: np.ndarray, H: np.ndarray, beta: float) -> float:
    """Calculate non-equilibrium free energy F(ρ)."""
    energy = np.real(np.trace(rho @ H))
    entropy = von_neumann_entropy(rho)
    return energy - entropy / beta

def relative_entropy(rho: np.ndarray, sigma: np.ndarray) -> float:
    """Calculate quantum relative entropy D(ρ||σ)."""
    # Regularize to avoid log(0)
    eps = 1e-15
    rho_reg = rho + eps * np.eye(rho.shape[0])
    sigma_reg = sigma + eps * np.eye(sigma.shape[0])

    rho_reg = rho_reg / np.trace(rho_reg)
    sigma_reg = sigma_reg / np.trace(sigma_reg)

    log_rho = np.linalg.eigvalsh(rho_reg)
    log_sigma = np.linalg.eigvalsh(sigma_reg)

    # Using eigendecomposition for proper log
    eig_rho, V_rho = np.linalg.eigh(rho_reg)
    log_rho_matrix = V_rho @ np.diag(np.log(eig_rho)) @ V_rho.conj().T

    eig_sigma, V_sigma = np.linalg.eigh(sigma_reg)
    log_sigma_matrix = V_sigma @ np.diag(np.log(eig_sigma)) @ V_sigma.conj().T

    return np.real(np.trace(rho_reg @ (log_rho_matrix - log_sigma_matrix)))

def athermality(rho: np.ndarray, H: np.ndarray, beta: float) -> float:
    """
    Calculate athermality: β⁻¹ D(ρ || γ).
    This is the extractable work under ideal conditions.
    """
    gamma = thermal_state(H, beta)
    return relative_entropy(rho, gamma) / beta

# Example calculations
print("Free Energy Analysis")
print("=" * 50)

# Pure excited state |1⟩
excited = np.array([[0, 0], [0, 1]])

# Pure ground state |0⟩
ground = np.array([[1, 0], [0, 0]])

# Superposition |+⟩
plus = np.array([[0.5, 0.5], [0.5, 0.5]])

T = 1.0
beta = 1.0 / T
gamma = thermal_state(H_qubit, beta)

for name, state in [("Ground |0⟩", ground),
                    ("Excited |1⟩", excited),
                    ("Superposition |+⟩", plus),
                    ("Thermal γ", gamma)]:
    F = free_energy(state, H_qubit, beta)
    A = athermality(state, H_qubit, beta)
    print(f"{name}: F = {F:.4f}, Athermality = {A:.4f}")
```

### The Second Law

<Definition term="Second Law (Resource-Theoretic)">
Under thermal operations, the free energy cannot increase:

$$F(\mathcal{E}(\rho)) \leq F(\rho)$$

Equivalently, athermality (relative entropy to thermal state) is a monotone.
</Definition>

This is the modern, rigorous statement of the second law of thermodynamics.

## 4. Work Extraction

The ultimate utility of athermality is **work extraction**:

<Definition term="Extractable Work">
The maximum work extractable from state $\rho$ using thermal operations is bounded by:

$$W_{max} \leq \Delta F(\rho) = F(\rho) - F(\gamma)$$
</Definition>

```python
def max_extractable_work(rho: np.ndarray, H: np.ndarray, beta: float) -> float:
    """Calculate maximum extractable work from state ρ."""
    gamma = thermal_state(H, beta)
    F_rho = free_energy(rho, H, beta)
    F_gamma = free_energy(gamma, H, beta)
    return max(0, F_rho - F_gamma)

def work_extraction_protocol(rho: np.ndarray, H: np.ndarray,
                             beta: float, efficiency: float = 0.9) -> dict:
    """
    Simulate a work extraction protocol.

    Args:
        rho: Initial state
        H: System Hamiltonian
        beta: Inverse temperature
        efficiency: Protocol efficiency (< 1 for irreversibility)

    Returns:
        Dictionary with protocol results
    """
    W_max = max_extractable_work(rho, H, beta)
    W_extracted = efficiency * W_max

    # After work extraction, state approaches thermal
    gamma = thermal_state(H, beta)
    rho_final = efficiency * gamma + (1 - efficiency) * rho

    return {
        'initial_state': rho,
        'final_state': rho_final,
        'max_work': W_max,
        'extracted_work': W_extracted,
        'efficiency': efficiency,
        'remaining_athermality': athermality(rho_final, H, beta)
    }

# Example: Work from excited state
print("\nWork Extraction from Excited State:")
result = work_extraction_protocol(excited, H_qubit, beta)
print(f"  Maximum work: {result['max_work']:.4f}")
print(f"  Extracted work (90% efficiency): {result['extracted_work']:.4f}")
print(f"  Remaining athermality: {result['remaining_athermality']:.4f}")
```

### Single-Shot Work Extraction

In the single-shot regime (one copy), work extraction is probabilistic:

<Formula>
$$W_{det} = F_0(\rho) - F_0(\gamma) = -\frac{1}{\beta}\ln\left(\frac{1}{d}\text{Tr}(\Pi_\rho)\right)$$
</Formula>

where $F_0$ is the single-shot free energy and $\Pi_\rho$ projects onto the support of $\rho$.

## 5. Thermo-Majorization

For state transformations under thermal operations, the relevant criterion is **thermo-majorization**:

<Definition term="Thermo-Majorization">
State $\rho$ can be transformed to $\sigma$ under thermal operations iff:

$$p^\downarrow_\rho \succ_\beta p^\downarrow_\sigma$$

where $p^\downarrow_\rho$ are the populations in the energy eigenbasis sorted appropriately for temperature $\beta$.
</Definition>

```python
def energy_populations(rho: np.ndarray, H: np.ndarray) -> np.ndarray:
    """Get populations in energy eigenbasis."""
    eigenvalues_H, eigenvectors_H = np.linalg.eigh(H)
    populations = np.array([
        np.real(eigenvectors_H[:, i].conj() @ rho @ eigenvectors_H[:, i])
        for i in range(len(eigenvalues_H))
    ])
    return populations, eigenvalues_H

def thermo_majorization_curve(rho: np.ndarray, H: np.ndarray, beta: float) -> Tuple[np.ndarray, np.ndarray]:
    """
    Compute the thermo-majorization curve.

    The curve is constructed by sorting states by their
    "effective temperatures" relative to the thermal state.
    """
    pops, energies = energy_populations(rho, H)
    gamma = thermal_state(H, beta)
    gamma_pops, _ = energy_populations(gamma, H)

    # Sort by p_i / gamma_i (Gibbs-rescaled populations)
    ratios = pops / (gamma_pops + 1e-15)
    sorted_indices = np.argsort(ratios)[::-1]

    # Cumulative sums for the curve
    x = np.concatenate([[0], np.cumsum(gamma_pops[sorted_indices])])
    y = np.concatenate([[0], np.cumsum(pops[sorted_indices])])

    return x, y

def can_transform_thermal(rho: np.ndarray, sigma: np.ndarray,
                          H: np.ndarray, beta: float) -> bool:
    """Check if ρ → σ is possible under thermal operations."""
    x_rho, y_rho = thermo_majorization_curve(rho, H, beta)
    x_sigma, y_sigma = thermo_majorization_curve(sigma, H, beta)

    # Interpolate for comparison
    from scipy.interpolate import interp1d

    # Check if ρ's curve lies above σ's curve
    x_common = np.linspace(0, 1, 100)

    f_rho = interp1d(x_rho, y_rho, bounds_error=False, fill_value=(0, 1))
    f_sigma = interp1d(x_sigma, y_sigma, bounds_error=False, fill_value=(0, 1))

    return np.all(f_rho(x_common) >= f_sigma(x_common) - 1e-10)

# Example: Check thermal transformations
print("\nThermo-Majorization Analysis:")
print(f"  Excited → Thermal: {can_transform_thermal(excited, gamma, H_qubit, beta)}")
print(f"  Thermal → Excited: {can_transform_thermal(gamma, excited, H_qubit, beta)}")
print(f"  Ground → Thermal: {can_transform_thermal(ground, gamma, H_qubit, beta)}")
```

## 6. Coherence in Thermodynamics

Quantum coherence in the energy basis plays a special role:

<Definition term="Thermodynamic Coherence">
Coherence in the energy eigenbasis represents a purely quantum contribution to thermodynamic resources. It can enable:
1. Enhanced work extraction (with coherent operations)
2. Quantum advantages in heat engines
3. Violation of classical fluctuation relations
</Definition>

```python
def energy_coherence(rho: np.ndarray, H: np.ndarray) -> float:
    """Calculate coherence in the energy eigenbasis."""
    _, eigenvectors = np.linalg.eigh(H)

    # Transform to energy basis
    rho_energy = eigenvectors.conj().T @ rho @ eigenvectors

    # l1-norm of coherence
    total = np.sum(np.abs(rho_energy))
    diagonal = np.sum(np.abs(np.diag(rho_energy)))
    return total - diagonal

def decompose_free_energy(rho: np.ndarray, H: np.ndarray, beta: float) -> dict:
    """
    Decompose free energy into classical and quantum contributions.

    F(ρ) = F_cl(ρ_diag) + T·C_rel(ρ)

    where ρ_diag is dephased in energy basis.
    """
    _, eigenvectors = np.linalg.eigh(H)

    # Dephase in energy basis
    rho_energy = eigenvectors.conj().T @ rho @ eigenvectors
    rho_dephased = np.diag(np.diag(rho_energy))
    rho_dephased_original = eigenvectors @ rho_dephased @ eigenvectors.conj().T

    F_total = free_energy(rho, H, beta)
    F_classical = free_energy(rho_dephased_original, H, beta)
    F_quantum = F_total - F_classical

    return {
        'total_free_energy': F_total,
        'classical_contribution': F_classical,
        'quantum_contribution': F_quantum,
        'energy_coherence': energy_coherence(rho, H)
    }

# Analyze states with and without coherence
print("\nFree Energy Decomposition:")

# State with coherence: |+⟩ = (|0⟩ + |1⟩)/√2
plus_state = np.array([[0.5, 0.5], [0.5, 0.5]])

# State without coherence: same populations, diagonal
diagonal_state = np.array([[0.5, 0], [0, 0.5]])

for name, state in [("Coherent |+⟩", plus_state),
                    ("Diagonal mixture", diagonal_state)]:
    decomp = decompose_free_energy(state, H_qubit, beta)
    print(f"\n{name}:")
    print(f"  Total F: {decomp['total_free_energy']:.4f}")
    print(f"  Classical: {decomp['classical_contribution']:.4f}")
    print(f"  Quantum: {decomp['quantum_contribution']:.4f}")
    print(f"  Energy coherence: {decomp['energy_coherence']:.4f}")
```

## 7. Quantum Heat Engines

Resource theory provides rigorous bounds on quantum heat engine performance:

<Definition term="Carnot Efficiency">
The maximum efficiency of a heat engine operating between temperatures $T_H$ and $T_C$:

$$\eta_{Carnot} = 1 - \frac{T_C}{T_H}$$
</Definition>

```python
def quantum_heat_engine(T_hot: float, T_cold: float,
                        H: np.ndarray, protocol_efficiency: float = 0.9) -> dict:
    """
    Simulate a quantum Otto cycle heat engine.

    Returns:
        Dictionary with engine performance metrics
    """
    beta_hot = 1.0 / T_hot
    beta_cold = 1.0 / T_cold

    # Thermal states at different temperatures
    gamma_hot = thermal_state(H, beta_hot)
    gamma_cold = thermal_state(H, beta_cold)

    # Work in compression/expansion strokes
    # Simplified model: work = change in energy expectation

    # Heat absorbed from hot reservoir
    E_hot = np.real(np.trace(gamma_hot @ H))
    E_cold = np.real(np.trace(gamma_cold @ H))

    Q_hot = E_hot - E_cold  # Heat absorbed (simplified)
    Q_cold = Q_hot - Q_hot * (1 - T_cold/T_hot)  # Heat rejected

    W_out = Q_hot - Q_cold
    W_actual = W_out * protocol_efficiency

    efficiency = W_actual / Q_hot if Q_hot > 0 else 0
    carnot_efficiency = 1 - T_cold / T_hot

    return {
        'T_hot': T_hot,
        'T_cold': T_cold,
        'Q_hot': Q_hot,
        'Q_cold': Q_cold,
        'W_ideal': W_out,
        'W_actual': W_actual,
        'efficiency': efficiency,
        'carnot_efficiency': carnot_efficiency,
        'efficiency_ratio': efficiency / carnot_efficiency if carnot_efficiency > 0 else 0
    }

# Example heat engine
print("\nQuantum Heat Engine:")
result = quantum_heat_engine(T_hot=2.0, T_cold=0.5, H=H_qubit)
for key, value in result.items():
    if isinstance(value, float):
        print(f"  {key}: {value:.4f}")
    else:
        print(f"  {key}: {value}")
```
