## 1. The Idea of Dynamical Decoupling

<Definition term="Dynamical Decoupling">
Dynamical decoupling (DD) is a technique to suppress unwanted system-environment interactions by applying carefully timed control pulses that effectively average out the noise.
</Definition>

**Key insight:** If the noise varies slowly compared to the pulse interval, we can "refocus" the accumulated phase errors.

<Formula>
**Basic principle:** A $\pi$-pulse (X or Y rotation by 180 degrees) inverts the sign of accumulated phase:
$$U_\pi \cdot e^{-i\phi\sigma_z} \cdot U_\pi^\dagger = e^{+i\phi\sigma_z}$$
Two consecutive periods with opposite signs cancel: $e^{-i\phi}e^{+i\phi} = 1$
</Formula>

## 2. The Hahn Echo

<Definition term="Hahn Echo">
The simplest dynamical decoupling sequence: apply a $\pi$-pulse at time $t/2$ to refocus dephasing at time $t$.
</Definition>

<Formula>
**Sequence:** Free evolution ($t/2$) - $\pi_X$ pulse - Free evolution ($t/2$)
$$U_{echo}(t) = e^{-iH_{env}t/2} \cdot X \cdot e^{-iH_{env}t/2}$$
</Formula>

<Example>
```python
import numpy as np

def simulate_free_evolution(rho0: np.ndarray, noise_realizations: np.ndarray,
                            times: np.ndarray) -> list:
    """
    Simulate free evolution under quasi-static noise.
    Each realization has a different random phase.
    """
    results = []

    for t in times:
        rho_avg = np.zeros((2, 2), dtype=complex)

        for delta in noise_realizations:
            # Phase accumulated: phi = delta * t
            phase = delta * t
            rho = rho0.copy()

            # Apply dephasing: rho_01 -> rho_01 * exp(-i*phi)
            rho[0, 1] *= np.exp(-1j * phase)
            rho[1, 0] *= np.exp(1j * phase)

            rho_avg += rho

        rho_avg /= len(noise_realizations)
        results.append(rho_avg)

    return results

def simulate_hahn_echo(rho0: np.ndarray, noise_realizations: np.ndarray,
                       total_time: float) -> np.ndarray:
    """
    Simulate Hahn echo sequence.
    """
    X = np.array([[0, 1], [1, 0]], dtype=complex)

    rho_avg = np.zeros((2, 2), dtype=complex)

    for delta in noise_realizations:
        rho = rho0.copy()

        # First half: evolve for t/2
        phase1 = delta * total_time / 2
        rho[0, 1] *= np.exp(-1j * phase1)
        rho[1, 0] *= np.exp(1j * phase1)

        # Pi pulse (X gate)
        rho = X @ rho @ X.conj().T

        # Second half: evolve for t/2
        phase2 = delta * total_time / 2
        rho[0, 1] *= np.exp(-1j * phase2)
        rho[1, 0] *= np.exp(1j * phase2)

        rho_avg += rho

    rho_avg /= len(noise_realizations)
    return rho_avg

n_realizations = 1000
```

## 3. CPMG Sequence

<Definition term="CPMG">
Carr-Purcell-Meiboom-Gill sequence: Multiple evenly-spaced $\pi$-pulses provide better protection against time-varying noise.
$$\text{CPMG}_n: \quad (\tau - \pi_Y - \tau)^n$$
</Definition>

<Formula>
**Pulse timing:**
- Total time: $T$
- Number of pulses: $n$
- Pulse spacing: $\tau = T/(2n)$
- Pulses at times: $\tau, 3\tau, 5\tau, ..., (2n-1)\tau$
</Formula>

<Example>
```python
import numpy as np

def cpmg_sequence(rho0: np.ndarray, noise_func, total_time: float,
                  n_pulses: int, dt: float = 0.001) -> np.ndarray:
    """
    Simulate CPMG sequence with time-dependent noise.

    noise_func(t): returns instantaneous noise value at time t
    """
    Y = np.array([[0, -1j], [1j, 0]], dtype=complex)

    tau = total_time / (2 * n_pulses)
    pulse_times = [(2*k + 1) * tau for k in range(n_pulses)]

    rho = rho0.copy()
    accumulated_phase = 0
    current_time = 0

    # Time evolution
    times = np.arange(0, total_time, dt)
    pulse_idx = 0
    sign = 1  # Phase accumulation sign

    for t in times:
        # Check for pulse
        if pulse_idx < n_pulses and t >= pulse_times[pulse_idx]:
            # Apply Y pulse (flips sign of Z component)
            rho = Y @ rho @ Y.conj().T
            sign *= -1
            pulse_idx += 1

        # Accumulate phase
        noise = noise_func(t)
        accumulated_phase += sign * noise * dt

    # Apply final accumulated phase
    rho[0, 1] *= np.exp(-1j * accumulated_phase)
    rho[1, 0] *= np.exp(1j * accumulated_phase)

    return rho

def simulate_cpmg_ensemble(rho0: np.ndarray, noise_spectrum: str,
                           total_time: float, n_pulses: int,
                           n_realizations: int = 100) -> float:
    """
    Simulate CPMG with ensemble averaging.
    """
    coherences = []

    for _ in range(n_realizations):
        if noise_spectrum == 'white':
            # Ornstein-Uhlenbeck process (fast fluctuating)
            noise_func = lambda t: np.random.normal(0, 1)
        elif noise_spectrum == '1/f':
            # 1/f noise (slowly varying)
            base_freq = 0.1
            phase = np.random.uniform(0, 2*np.pi)
            amplitude = np.random.exponential(1)
            noise_func = lambda t, a=amplitude, p=phase: a * np.sin(base_freq * t + p)
        else:  # quasi-static
            delta = np.random.normal(0, 1)
            noise_func = lambda t, d=delta: d

        rho_final = cpmg_sequence(rho0, noise_func, total_time, n_pulses)
        coherences.append(np.abs(rho_final[0, 1]))

    return np.mean(coherences)
```

## 4. XY Sequences

<Definition term="XY Sequences">
XY sequences alternate between X and Y pulses to suppress pulse errors while maintaining decoherence protection.
</Definition>

<Formula>
**XY-4:**
$$\tau - X - \tau - Y - \tau - X - \tau - Y$$

**XY-8:**
$$XY4 - \overline{XY4}$$
where $\overline{XY4}$ has reversed pulse order.
</Formula>

<Example>
```python
import numpy as np

def apply_pulse(rho: np.ndarray, axis: str) -> np.ndarray:
    """Apply pi pulse around specified axis."""
    if axis == 'X':
        U = np.array([[0, 1], [1, 0]], dtype=complex)
    elif axis == 'Y':
        U = np.array([[0, -1j], [1j, 0]], dtype=complex)
    elif axis == 'Z':
        U = np.array([[1, 0], [0, -1]], dtype=complex)
    else:
        raise ValueError(f"Unknown axis: {axis}")

    return U @ rho @ U.conj().T

def simulate_dd_sequence(rho0: np.ndarray, sequence: list,
                          noise_realizations: np.ndarray,
                          total_time: float) -> np.ndarray:
    """
    Simulate arbitrary DD sequence.

    sequence: list of pulse axes, e.g., ['X', 'Y', 'X', 'Y'] for XY4
    """
    n_pulses = len(sequence)
    tau = total_time / (2 * n_pulses) if n_pulses > 0 else total_time

    rho_avg = np.zeros((2, 2), dtype=complex)

    for delta in noise_realizations:
        rho = rho0.copy()
        sign = 1

        for i, axis in enumerate(sequence):
            # Evolve for tau
            phase = sign * delta * tau
            rho[0, 1] *= np.exp(-1j * phase)
            rho[1, 0] *= np.exp(1j * phase)

            # Apply pulse
            rho = apply_pulse(rho, axis)

            # Update sign based on pulse
            if axis in ['X', 'Y']:
                sign *= -1

            # Evolve for tau (after pulse)
            phase = sign * delta * tau
            rho[0, 1] *= np.exp(-1j * phase)
            rho[1, 0] *= np.exp(1j * phase)

        rho_avg += rho

    rho_avg /= len(noise_realizations)
    return rho_avg

def simulate_with_pulse_errors(rho0: np.ndarray, sequence: list,
                                noise_realizations: np.ndarray,
                                total_time: float,
                                pulse_error: float) -> np.ndarray:
    """
    Simulate DD with imperfect pulses.
    pulse_error: fractional rotation error
    """
    n_pulses = len(sequence)
    tau = total_time / (2 * n_pulses) if n_pulses > 0 else total_time

    rho_avg = np.zeros((2, 2), dtype=complex)

    for delta in noise_realizations:
        rho = rho0.copy()

        accumulated_phase = 0
        sign = 1

        for i, axis in enumerate(sequence):
            # Evolve for tau
            accumulated_phase += sign * delta * tau

            # Imperfect pulse (rotation by pi + error)
            angle = np.pi * (1 + np.random.uniform(-pulse_error, pulse_error))

            if axis == 'X':
                c, s = np.cos(angle/2), -1j * np.sin(angle/2)
                U = np.array([[c, s], [s, c]], dtype=complex)
            elif axis == 'Y':
                c, s = np.cos(angle/2), np.sin(angle/2)
                U = np.array([[c, -s], [s, c]], dtype=complex)

            rho = U @ rho @ U.conj().T
            sign *= -1

            # Evolve for tau
            accumulated_phase += sign * delta * tau

        # Apply accumulated phase
        rho[0, 1] *= np.exp(-1j * accumulated_phase)
        rho[1, 0] *= np.exp(1j * accumulated_phase)

        rho_avg += rho

    rho_avg /= len(noise_realizations)
    return rho_avg

# Compare sequences
print("DD Sequence Comparison")
print("=" * 50)

rho0 = np.array([[0.5, 0.5], [0.5, 0.5]], dtype=complex)
np.random.seed(42)
noise = np.random.normal(0, 1, 500)

sequences = {
    'Free': [],
    'Hahn (X)': ['X'],
    'CPMG-4 (Y)': ['Y', 'Y', 'Y', 'Y'],
    'XY-4': ['X', 'Y', 'X', 'Y'],
    'XY-8': ['X', 'Y', 'X', 'Y', 'Y', 'X', 'Y', 'X'],
}

print("\nWithout pulse errors:")
print(f"{'Sequence':<15} {'Coherence':<12}")

for name, seq in sequences.items():
    rho = simulate_dd_sequence(rho0, seq, noise, total_time)
    coh = np.abs(rho[0, 1])
    print(f"{name:<15} {coh:<12.4f}")

print("\nWith 5% pulse errors:")
print(f"{'Sequence':<15} {'Coherence':<12}")

for name, seq in sequences.items():
    if len(seq) > 0:
        np.random.seed(42)
        rho = simulate_with_pulse_errors(rho0, seq, noise, total_time, 0.05)
        coh = np.abs(rho[0, 1])
    else:
        rho = simulate_dd_sequence(rho0, seq, noise, total_time)
        coh = np.abs(rho[0, 1])
    print(f"{name:<15} {coh:<12.4f}")
```

## 5. Filter Function Formalism

<Definition term="Filter Function">
The filter function $F(\omega, t)$ describes how a DD sequence suppresses noise at different frequencies:
$$\langle e^{i\phi(t)} \rangle = e^{-\chi(t)}$$
$$\chi(t) = \frac{1}{\pi}\int_0^\infty S(\omega) \frac{F(\omega, t)}{\omega^2} d\omega$$
where $S(\omega)$ is the noise power spectrum.
</Definition>

<Formula>
**Filter function for $n$ evenly-spaced pulses:**
$$F(\omega, t) = 8\sin^4\left(\frac{\omega t}{4n}\right) \frac{\sin^2(n\omega t/2n)}{\cos^2(\omega t/2n)}$$
</Formula>

<Example>
```python
import numpy as np

def filter_function_cpmg(omega: np.ndarray, t: float, n: int) -> np.ndarray:
    """
    Calculate CPMG filter function.
    """
    if n == 0:
        # Free evolution
        return 2 * (1 - np.cos(omega * t))

    tau = t / (2 * n)

    # Avoid division by zero
    omega = np.where(np.abs(omega) < 1e-10, 1e-10, omega)

    # Filter function
    F = 8 * np.sin(omega * tau / 2)**4

    # Geometric series term
    numerator = np.sin(n * omega * tau)**2
    denominator = np.cos(omega * tau)**2 + 1e-10

    F *= numerator / denominator

    return F

def calculate_coherence_decay(noise_spectrum: callable, filter_func: np.ndarray,
                               omega: np.ndarray) -> float:
    """
    Calculate coherence decay from noise spectrum and filter function.
    chi = (1/pi) * integral S(omega) * F(omega) / omega^2 d_omega
    """
    S = noise_spectrum(omega)

    # Avoid division by zero
    omega_safe = np.where(np.abs(omega) < 1e-10, 1e-10, omega)

    integrand = S * filter_func / omega_safe**2
    chi = np.trapz(integrand, omega) / np.pi

    return np.exp(-chi)

            first_zero = omega[zero_crossings[0]]

        else:
            print(f"  n={n}: no clear minimum")

    return np.ones_like(omega)
```

## 6. Optimized Pulse Sequences

<Definition term="Optimized DD">
Advanced sequences like Uhrig DD (UDD) and Concatenated DD (CDD) are optimized for specific noise spectra or robust operation.
</Definition>

<Formula>
**Uhrig DD:** Pulse times are placed at:
$$t_j = T \sin^2\left(\frac{\pi j}{2n+2}\right)$$
This provides optimal suppression for low-frequency noise up to order $n$.
</Formula>

<Example>
```python
import numpy as np

def uhrig_pulse_times(total_time: float, n_pulses: int) -> np.ndarray:
    """
    Calculate pulse times for Uhrig DD.
    """
    j = np.arange(1, n_pulses + 1)
    return total_time * np.sin(np.pi * j / (2 * n_pulses + 2))**2

def cpmg_pulse_times(total_time: float, n_pulses: int) -> np.ndarray:
    """
    Calculate pulse times for CPMG.
    """
    return total_time * (2 * np.arange(1, n_pulses + 1) - 1) / (2 * n_pulses)

def simulate_general_dd(rho0: np.ndarray, pulse_times: np.ndarray,
                        noise_realizations: np.ndarray,
                        total_time: float) -> np.ndarray:
    """
    Simulate DD with arbitrary pulse timing.
    """
    rho_avg = np.zeros((2, 2), dtype=complex)

    for delta in noise_realizations:
        rho = rho0.copy()

        # Track accumulated phase
        accumulated_phase = 0
        sign = 1
        prev_time = 0

        for t_pulse in pulse_times:
            # Evolve from prev_time to t_pulse
            dt = t_pulse - prev_time
            accumulated_phase += sign * delta * dt

            # Apply pi pulse (flip sign)
            sign *= -1
            prev_time = t_pulse

        # Final evolution from last pulse to total_time
        dt = total_time - prev_time
        accumulated_phase += sign * delta * dt

        # Apply accumulated phase
        rho[0, 1] *= np.exp(-1j * accumulated_phase)
        rho[1, 0] *= np.exp(1j * accumulated_phase)

        rho_avg += rho

    rho_avg /= len(noise_realizations)
    return rho_avg
```

## 7. Practical Considerations

<Tip>
Real implementations must account for:
- Finite pulse duration
- Pulse imperfections
- Heating from pulses
- Hardware constraints
</Tip>

<Example>
```python
import numpy as np

def simulate_finite_pulses(rho0: np.ndarray, pulse_times: np.ndarray,
                           noise_realizations: np.ndarray,
                           total_time: float, pulse_duration: float) -> np.ndarray:
    """
    Simulate DD with finite pulse duration.
    """
    rho_avg = np.zeros((2, 2), dtype=complex)

    for delta in noise_realizations:
        rho = rho0.copy()

        accumulated_phase = 0
        sign = 1
        prev_time = 0

        for t_pulse in pulse_times:
            # Free evolution before pulse
            dt = t_pulse - pulse_duration/2 - prev_time
            if dt > 0:
                accumulated_phase += sign * delta * dt

            # During pulse: partial dephasing (simplified model)
            accumulated_phase += sign * delta * pulse_duration * 0.5

            # After pulse
            sign *= -1
            prev_time = t_pulse + pulse_duration/2

        # Final free evolution
        dt = total_time - prev_time
        if dt > 0:
            accumulated_phase += sign * delta * dt

        rho[0, 1] *= np.exp(-1j * accumulated_phase)
        rho[1, 0] *= np.exp(1j * accumulated_phase)

        rho_avg += rho

    rho_avg /= len(noise_realizations)
    return rho_avg

        baseline = coh

    degradation = (baseline - coh) / baseline * 100

    print(f"{duration:<12.3f} {coh:<12.4f} {degradation:<12.1f}%")
```

## Summary

In this lesson, you learned about dynamical decoupling:

1. **Basic principle**: Pi-pulses invert accumulated phase errors
2. **Hahn echo**: Single pulse refocuses quasi-static noise
3. **CPMG**: Multiple pulses for time-varying noise
4. **XY sequences**: Alternate pulse axes for robustness
5. **Filter function**: Describes frequency-dependent suppression
6. **Optimized sequences**: UDD, CDD for specific noise spectra

<Warning>
DD extends coherence but cannot eliminate it completely. The benefit depends on matching the sequence to the noise spectrum.
