## 1. Beyond the Markovian Approximation

<Definition term="Markovian Dynamics">
Markovian quantum dynamics assumes the environment has no memory - the future evolution depends only on the present state, not on the history. This requires the environment correlation time to be much shorter than the system timescale.
</Definition>

<Definition term="Non-Markovian Dynamics">
Non-Markovian dynamics includes memory effects where information can flow back from the environment to the system. The evolution depends on the system's history.
</Definition>

**When non-Markovianity matters:**
- Structured environments (photonic crystals, coupled cavities)
- Strong system-environment coupling
- Low-temperature environments
- Short-time dynamics
- Environments with long correlation times

## 2. Signatures of Non-Markovianity

<Formula>
**Divisibility condition for Markovian dynamics:**
A dynamical map $\Lambda(t)$ is Markovian if it can be written as:
$$\Lambda(t_2) = \Lambda(t_2, t_1) \circ \Lambda(t_1)$$
where $\Lambda(t_2, t_1)$ is completely positive for all $t_2 > t_1$.
</Formula>

**Key signatures:**
1. **Non-monotonic decoherence**: Coherence can temporarily increase
2. **Distinguishability increase**: State pairs can become more distinguishable
3. **Information backflow**: Quantum information returns from environment
4. **Negative decay rates**: Effective Lindblad rates become negative

<Example>
```python
import numpy as np
from scipy.linalg import expm

def trace_distance(rho1: np.ndarray, rho2: np.ndarray) -> float:
    """
    Calculate trace distance D(rho1, rho2) = (1/2) ||rho1 - rho2||_1
    """
    diff = rho1 - rho2
    eigenvalues = np.linalg.eigvalsh(diff)
    return 0.5 * np.sum(np.abs(eigenvalues))

def detect_non_markovianity_blp(rho1_list: list, rho2_list: list,
                                 times: np.ndarray) -> dict:
    """
    Detect non-Markovianity using BLP measure.
    Non-Markovian if trace distance increases at any time.
    """
    distances = [trace_distance(r1, r2) for r1, r2 in zip(rho1_list, rho2_list)]

    # Calculate time derivative
    dt = times[1] - times[0]
    derivatives = np.gradient(distances, dt)

    # Non-Markovian when derivative is positive
    is_non_markovian = np.any(derivatives > 1e-10)

    # BLP measure: integral of positive derivatives
    positive_deriv = np.maximum(derivatives, 0)
    blp_measure = np.trapz(positive_deriv, times)

    return {
        'is_non_markovian': is_non_markovian,
        'blp_measure': blp_measure,
        'distances': distances,
        'derivatives': derivatives
    }

# Example: Markovian amplitude damping
def markovian_evolution(rho0: np.ndarray, gamma: float, t: float) -> np.ndarray:
    """Analytical solution for Markovian amplitude damping."""
    rho = np.zeros((2, 2), dtype=complex)
    rho[0, 0] = 1 - rho0[1, 1] * np.exp(-gamma * t)
    rho[1, 1] = rho0[1, 1] * np.exp(-gamma * t)
    rho[0, 1] = rho0[0, 1] * np.exp(-gamma * t / 2)
    rho[1, 0] = rho0[1, 0] * np.exp(-gamma * t / 2)
    return rho

# Example: Non-Markovian damped oscillation
def non_markovian_evolution(rho0: np.ndarray, gamma: float,
                             omega_c: float, t: float) -> np.ndarray:
    """
    Simplified non-Markovian model with oscillatory decay.
    Models strong coupling to a structured environment.
    """
    # Damped oscillatory decay factor
    decay = np.exp(-gamma * t / 2) * np.cos(omega_c * t)
    decay = np.maximum(decay, 0)  # Keep physical

    rho = np.zeros((2, 2), dtype=complex)
    p1_0 = np.real(rho0[1, 1])
    rho[1, 1] = p1_0 * decay**2
    rho[0, 0] = 1 - rho[1, 1]
    rho[0, 1] = rho0[0, 1] * decay
    rho[1, 0] = rho0[1, 0] * decay
    return rho

# Compare Markovian vs Non-Markovian
print("Non-Markovianity Detection")
print("=" * 50)

times = np.linspace(0, 10, 200)
gamma = 0.3

# Initial states for BLP measure
rho1_0 = np.array([[0.5, 0.5], [0.5, 0.5]], dtype=complex)  # |+⟩
rho2_0 = np.array([[0.5, -0.5], [-0.5, 0.5]], dtype=complex)  # |-⟩

# Markovian case
rho1_markov = [markovian_evolution(rho1_0, gamma, t) for t in times]
rho2_markov = [markovian_evolution(rho2_0, gamma, t) for t in times]
result_markov = detect_non_markovianity_blp(rho1_markov, rho2_markov, times)

print(f"\nMarkovian amplitude damping:")
print(f"  Non-Markovian: {result_markov['is_non_markovian']}")
print(f"  BLP measure: {result_markov['blp_measure']:.6f}")

# Non-Markovian case
omega_c = 2.0  # Environment frequency
rho1_nm = [non_markovian_evolution(rho1_0, gamma, omega_c, t) for t in times]
rho2_nm = [non_markovian_evolution(rho2_0, gamma, omega_c, t) for t in times]
result_nm = detect_non_markovianity_blp(rho1_nm, rho2_nm, times)

print(f"\nNon-Markovian (structured environment):")
print(f"  Non-Markovian: {result_nm['is_non_markovian']}")
print(f"  BLP measure: {result_nm['blp_measure']:.6f}")
```

## 3. Measures of Non-Markovianity

<Definition term="BLP Measure">
The Breuer-Laine-Piilo measure quantifies non-Markovianity as the total increase in trace distance between optimal state pairs:
$$\mathcal{N}_{BLP} = \max_{\rho_{1,2}} \int_{\sigma > 0} \sigma(t) dt$$
where $\sigma(t) = \frac{d}{dt}D(\rho_1(t), \rho_2(t))$ and $D$ is the trace distance.
</Definition>

<Definition term="RHP Measure">
The Rivas-Huelga-Plenio measure uses the Choi matrix positivity. Non-Markovianity occurs when the intermediate map becomes non-completely-positive.
</Definition>

<Example>
```python
import numpy as np

def coherence_l1(rho: np.ndarray) -> float:
    """Calculate l1-norm coherence."""
    dim = rho.shape[0]
    coh = 0
    for i in range(dim):
        for j in range(dim):
            if i != j:
                coh += np.abs(rho[i, j])
    return coh

def purity(rho: np.ndarray) -> float:
    """Calculate purity Tr(rho^2)."""
    return np.real(np.trace(rho @ rho))

def detect_coherence_revival(rho_list: list, times: np.ndarray) -> dict:
    """
    Detect non-Markovianity via coherence revival.
    """
    coherences = [coherence_l1(rho) for rho in rho_list]

    dt = times[1] - times[0]
    derivatives = np.gradient(coherences, dt)

    # Find revival events (positive derivative after initial decay)
    revival_times = []
    revival_amounts = []

    for i in range(1, len(derivatives)):
        if derivatives[i] > 1e-10 and coherences[i-1] < coherences[0] - 0.01:
            revival_times.append(times[i])
            revival_amounts.append(derivatives[i])

    return {
        'coherences': coherences,
        'has_revival': len(revival_times) > 0,
        'revival_times': revival_times,
        'total_revival': np.sum(np.maximum(derivatives, 0) * dt)
    }

def detect_purity_revival(rho_list: list, times: np.ndarray) -> dict:
    """
    Detect non-Markovianity via purity increase.
    For Markovian dynamics, purity is non-increasing.
    """
    purities = [purity(rho) for rho in rho_list]

    dt = times[1] - times[0]
    derivatives = np.gradient(purities, dt)

    has_increase = np.any(derivatives > 1e-10)

    return {
        'purities': purities,
        'has_increase': has_increase,
        'max_increase_rate': np.max(derivatives)
    }

# Test different measures
print("Multiple Non-Markovianity Measures")
print("=" * 50)

times = np.linspace(0, 15, 300)
gamma = 0.2
omega_c = 1.5

# Initial superposition
rho0 = np.array([[0.5, 0.5], [0.5, 0.5]], dtype=complex)

# Markovian evolution
rho_markov = [markovian_evolution(rho0, gamma, t) for t in times]

# Non-Markovian evolution
rho_nm = [non_markovian_evolution(rho0, gamma, omega_c, t) for t in times]

# Coherence analysis
coh_markov = detect_coherence_revival(rho_markov, times)
coh_nm = detect_coherence_revival(rho_nm, times)

print("\nCoherence-based detection:")
print(f"  Markovian - revival: {coh_markov['has_revival']}")
print(f"  Non-Markovian - revival: {coh_nm['has_revival']}")

# Purity analysis
pur_markov = detect_purity_revival(rho_markov, times)
pur_nm = detect_purity_revival(rho_nm, times)

print("\nPurity-based detection:")
print(f"  Markovian - increase: {pur_markov['has_increase']}")
print(f"  Non-Markovian - increase: {pur_nm['has_increase']}")
print(f"  Max increase rate: {pur_nm['max_increase_rate']:.6f}")
```
</Example>

## 4. Time-Convolutionless Master Equation

<Definition term="TCL Master Equation">
The time-convolutionless (TCL) master equation is an exact reformulation that avoids memory integrals:
$$\frac{d\rho(t)}{dt} = \mathcal{K}(t)\rho(t)$$
where the time-local generator $\mathcal{K}(t)$ can have negative rates for non-Markovian dynamics.
</Definition>

<Formula>
For a qubit with time-dependent dephasing:
$$\dot{\rho} = -i[H, \rho] + \gamma(t)\left(\sigma_z\rho\sigma_z - \rho\right)$$
where $\gamma(t)$ can become negative during non-Markovian periods.
</Formula>

<Example>
```python
import numpy as np
from scipy.integrate import solve_ivp

def time_dependent_lindblad(t, rho_vec, H, sigma_z, gamma_func):
    """
    Time-local master equation with time-dependent rate.
    """
    rho = rho_vec.reshape((2, 2))
    gamma_t = gamma_func(t)

    # Coherent part
    drho = -1j * (H @ rho - rho @ H)

    # Dephasing with time-dependent rate
    drho += gamma_t * (sigma_z @ rho @ sigma_z - rho)

    return drho.flatten()

def simulate_tcl_dynamics(rho0: np.ndarray, H: np.ndarray,
                          gamma_func, times: np.ndarray) -> list:
    """Simulate TCL master equation."""
    sigma_z = np.array([[1, 0], [0, -1]], dtype=complex)

    result = solve_ivp(
        time_dependent_lindblad,
        (times[0], times[-1]),
        rho0.flatten(),
        t_eval=times,
        args=(H, sigma_z, gamma_func),
        method='RK45'
    )

    rho_list = [result.y[:, i].reshape((2, 2)) for i in range(len(times))]
    return rho_list

# Example: Oscillating dephasing rate (non-Markovian)
def markovian_rate(t):
    return 0.5  # Constant positive rate

def non_markovian_rate(t):
    """Rate that oscillates and becomes negative."""
    return 0.5 * (1 - 1.5 * np.cos(2 * t)) * np.exp(-0.1 * t)

# Simulate both cases
H = np.zeros((2, 2), dtype=complex)
rho0 = np.array([[0.5, 0.5], [0.5, 0.5]], dtype=complex)
times = np.linspace(0, 15, 300)

print("Time-Convolutionless Dynamics")
print("=" * 50)

rho_markov = simulate_tcl_dynamics(rho0, H, markovian_rate, times)
rho_nm = simulate_tcl_dynamics(rho0, H, non_markovian_rate, times)

# Analyze coherence
coh_markov = [np.abs(rho[0, 1]) for rho in rho_markov]
coh_nm = [np.abs(rho[0, 1]) for rho in rho_nm]

print("\nCoherence at various times:")
for t in [0, 2, 5, 10, 15]:
    idx = np.argmin(np.abs(times - t))
    print(f"  t={t}: Markov={coh_markov[idx]:.4f}, Non-Markov={coh_nm[idx]:.4f}")

# Check for rate sign
print("\nDephasing rate γ(t):")
for t in [0, 1, 2, 3, 4]:
    rate_m = markovian_rate(t)
    rate_nm = non_markovian_rate(t)
    print(f"  t={t}: Markov={rate_m:.4f}, Non-Markov={rate_nm:.4f}")
```
</Example>

## 5. Memory Kernel Approach

<Definition term="Nakajima-Zwanzig Equation">
The exact integro-differential equation with memory kernel:
$$\frac{d\rho_S(t)}{dt} = -i[H_S, \rho_S(t)] + \int_0^t \mathcal{K}(t-s)\rho_S(s)ds$$
The memory kernel $\mathcal{K}(t-s)$ encodes the environment correlation function.
</Definition>

<Example>
```python
import numpy as np
from scipy.integrate import odeint

def memory_kernel_exponential(tau: float, gamma: float, omega: float) -> float:
    """
    Exponentially decaying memory kernel with oscillations.
    K(tau) = gamma * exp(-gamma*tau/2) * cos(omega*tau)
    """
    return gamma * np.exp(-gamma * tau / 2) * np.cos(omega * tau)

def nakajima_zwanzig_rhs(rho_flat, t, H, L, gamma, omega, dt, history):
    """
    RHS of Nakajima-Zwanzig equation with discrete memory integral.
    """
    dim = int(np.sqrt(len(rho_flat)))
    rho = rho_flat.reshape((dim, dim))

    # Coherent part
    drho = -1j * (H @ rho - rho @ H)

    # Memory integral (discretized)
    n_history = len(history)
    if n_history > 0:
        memory_integral = np.zeros((dim, dim), dtype=complex)
        for i, rho_past in enumerate(history):
            tau = (n_history - i) * dt
            K_tau = memory_kernel_exponential(tau, gamma, omega)
            # Simplified dissipator action
            LdL = L.conj().T @ L
            dissip = L @ rho_past @ L.conj().T - 0.5 * (LdL @ rho_past + rho_past @ LdL)
            memory_integral += K_tau * dissip * dt

        drho += memory_integral

    return drho.flatten()

def simulate_with_memory(rho0: np.ndarray, H: np.ndarray, L: np.ndarray,
                         gamma: float, omega: float,
                         times: np.ndarray) -> list:
    """
    Simulate non-Markovian dynamics with memory kernel.
    Uses simple forward integration with history tracking.
    """
    dt = times[1] - times[0]
    rho = rho0.copy()
    history = []
    results = [rho.copy()]

    for i, t in enumerate(times[1:]):
        # Calculate RHS with history
        drho_flat = nakajima_zwanzig_rhs(
            rho.flatten(), t, H, L, gamma, omega, dt, history
        )
        drho = drho_flat.reshape(rho.shape)

        # Update state
        rho = rho + drho * dt

        # Ensure Hermiticity and normalization
        rho = (rho + rho.conj().T) / 2
        rho = rho / np.trace(rho)

        # Update history (keep limited length for efficiency)
        history.append(rho.copy())
        if len(history) > 100:
            history.pop(0)

        results.append(rho.copy())

    return results

# Simulate non-Markovian dynamics
print("Memory Kernel Simulation")
print("=" * 50)

H = np.zeros((2, 2), dtype=complex)
sigma_m = np.array([[0, 0], [1, 0]], dtype=complex)
rho0 = np.array([[0, 0], [0, 1]], dtype=complex)  # Excited state

times = np.linspace(0, 10, 200)

# Compare different memory parameters
print("\nExcited state population P(|1⟩):")
for omega in [0.0, 1.0, 3.0]:
    gamma = 0.5
    rho_list = simulate_with_memory(rho0, H, sigma_m, gamma, omega, times)
    p_excited = [np.real(rho[1, 1]) for rho in rho_list]

    print(f"\n  omega = {omega}:")
    for t in [0, 2, 5, 10]:
        idx = np.argmin(np.abs(times - t))
        print(f"    t={t}: P(|1⟩) = {p_excited[idx]:.4f}")
```
</Example>

## 6. Hierarchical Equations of Motion

<Definition term="HEOM">
The Hierarchical Equations of Motion method is a numerically exact approach for non-Markovian dynamics with Gaussian environments characterized by a sum of exponentials.
</Definition>

<Formula>
For a system coupled to a bosonic bath:
$$\frac{\partial\rho_{\mathbf{n}}}{\partial t} = -i[H_S, \rho_{\mathbf{n}}] - \sum_k n_k \gamma_k \rho_{\mathbf{n}} + \text{coupling terms}$$
where $\rho_{\mathbf{n}}$ are auxiliary density matrices indexed by $\mathbf{n} = (n_1, n_2, ...)$.
</Formula>

<Example>
```python
import numpy as np
from itertools import product

def simple_heom_simulation(rho0: np.ndarray, H: np.ndarray,
                            Q: np.ndarray, gamma: float, lambda_c: float,
                            temp: float, n_levels: int,
                            times: np.ndarray) -> list:
    """
    Simplified HEOM for single-exponential bath correlation.

    Parameters:
    - Q: System-bath coupling operator
    - gamma: Bath correlation decay rate
    - lambda_c: Reorganization energy
    - temp: Temperature (k_B = 1)
    - n_levels: Hierarchy truncation level
    """
    dim = H.shape[0]
    dt = times[1] - times[0]

    # Bath parameters
    nu = gamma  # Single Matsubara frequency approximation
    c = lambda_c * gamma  # Coupling strength

    # Initialize hierarchy
    # rho_n for n = 0, 1, 2, ..., n_levels
    hierarchy = [np.zeros((dim, dim), dtype=complex) for _ in range(n_levels + 1)]
    hierarchy[0] = rho0.copy()

    results = [hierarchy[0].copy()]

    for t_idx, t in enumerate(times[1:]):
        new_hierarchy = []

        for n in range(n_levels + 1):
            rho_n = hierarchy[n]

            # System Hamiltonian
            drho = -1j * (H @ rho_n - rho_n @ H)

            # Hierarchy decay
            drho -= n * nu * rho_n

            # Coupling to level n-1 (if exists)
            if n > 0:
                drho -= 1j * c * (Q @ hierarchy[n-1] - hierarchy[n-1] @ Q)

            # Coupling to level n+1 (if exists)
            if n < n_levels:
                drho -= 1j * (Q @ hierarchy[n+1] - hierarchy[n+1] @ Q)

            new_rho = rho_n + drho * dt
            new_hierarchy.append(new_rho)

        hierarchy = new_hierarchy

        # Physical density matrix is rho_0
        rho_physical = hierarchy[0]
        rho_physical = (rho_physical + rho_physical.conj().T) / 2
        results.append(rho_physical)

    return results

# HEOM simulation example
print("Hierarchical Equations of Motion")
print("=" * 50)

# Two-level system
H = 0.5 * np.array([[1, 0], [0, -1]], dtype=complex)  # Energy splitting
Q = np.array([[0, 1], [1, 0]], dtype=complex)  # sigma_x coupling

rho0 = np.array([[1, 0], [0, 0]], dtype=complex)  # Ground state

times = np.linspace(0, 20, 400)

# Different bath parameters
print("\nPopulation dynamics for different bath parameters:")

for lambda_c, gamma in [(0.1, 0.5), (0.5, 0.5), (0.5, 2.0)]:
    rho_list = simple_heom_simulation(
        rho0, H, Q, gamma, lambda_c, temp=1.0,
        n_levels=5, times=times
    )

    p0 = [np.real(rho[0, 0]) for rho in rho_list]

    print(f"\nlambda={lambda_c}, gamma={gamma}:")
    for t in [0, 5, 10, 20]:
        idx = np.argmin(np.abs(times - t))
        print(f"  t={t}: P(|0⟩) = {p0[idx]:.4f}")
```
</Example>

## 7. Physical Examples

<Definition term="Structured Reservoir">
A structured reservoir has a non-flat spectral density, such as a photonic band gap material or coupled cavity array. The environment spectrum determines the memory effects.
</Definition>

<Example>
```python
import numpy as np

def spectral_density_lorentzian(omega: np.ndarray, omega_0: float,
                                 gamma: float, lambda_c: float) -> np.ndarray:
    """
    Lorentzian spectral density (damped cavity mode).
    J(omega) = lambda * gamma / ((omega - omega_0)^2 + gamma^2)
    """
    return lambda_c * gamma / ((omega - omega_0)**2 + gamma**2)

def spectral_density_ohmic(omega: np.ndarray, omega_c: float,
                           alpha: float) -> np.ndarray:
    """
    Ohmic spectral density with exponential cutoff.
    J(omega) = alpha * omega * exp(-omega/omega_c)
    """
    return alpha * omega * np.exp(-omega / omega_c)

def bath_correlation_function(t: np.ndarray, J_func, omega_range: tuple,
                               temp: float) -> np.ndarray:
    """
    Calculate bath correlation function C(t) from spectral density.
    C(t) = integral J(omega) * [coth(omega/2T) * cos(omega*t) - i*sin(omega*t)] domega
    """
    omegas = np.linspace(omega_range[0] + 0.01, omega_range[1], 500)
    domega = omegas[1] - omegas[0]

    C = np.zeros(len(t), dtype=complex)

    for i, tau in enumerate(t):
        # Bose-Einstein factor
        if temp > 0:
            n_omega = 1 / (np.exp(omegas / temp) - 1 + 1e-10)
            coth = 1 + 2 * n_omega
        else:
            coth = np.ones_like(omegas)

        J = J_func(omegas)
        integrand = J * (coth * np.cos(omegas * tau) - 1j * np.sin(omegas * tau))
        C[i] = np.trapz(integrand, omegas)

    return C

# Analyze different spectral densities
print("Spectral Density Analysis")
print("=" * 50)

omega_range = (0, 10)
t = np.linspace(0, 10, 200)

# Lorentzian (structured)
J_lorentz = lambda w: spectral_density_lorentzian(w, omega_0=2.0, gamma=0.5, lambda_c=1.0)
C_lorentz = bath_correlation_function(t, J_lorentz, omega_range, temp=1.0)

# Ohmic (unstructured)
J_ohmic = lambda w: spectral_density_ohmic(w, omega_c=5.0, alpha=0.1)
C_ohmic = bath_correlation_function(t, J_ohmic, omega_range, temp=1.0)

print("\nCorrelation function magnitude |C(t)|:")
for tau in [0, 1, 2, 5, 10]:
    idx = np.argmin(np.abs(t - tau))
    print(f"  t={tau}: Lorentzian={np.abs(C_lorentz[idx]):.4f}, "
          f"Ohmic={np.abs(C_ohmic[idx]):.4f}")

print("\nLorentzian has longer memory (structured environment)")
print("Ohmic decays faster (unstructured, more Markovian)")
```
</Example>
