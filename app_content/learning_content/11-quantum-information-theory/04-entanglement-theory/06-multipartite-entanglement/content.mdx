## 1. Beyond Bipartite Entanglement

When we move from two-party to many-party systems, the structure of entanglement becomes dramatically richer. Unlike bipartite entanglement, which has a single measure (for pure states), multipartite entanglement has infinitely many inequivalent forms.

<Definition term="Multipartite Entanglement">
A pure state of n parties is multipartite entangled if it cannot be written as a product state across any bipartition. A state has genuine n-partite entanglement if it is entangled across every possible bipartition.
</Definition>

<Formula>
$$|\psi\rangle_{ABC...} \text{ is genuinely entangled} \Leftrightarrow |\psi\rangle \neq |\phi\rangle_S \otimes |\chi\rangle_{\bar{S}} \text{ for all bipartitions } S|\bar{S}$$
</Formula>

## 2. The GHZ State

The Greenberger-Horne-Zeilinger (GHZ) state is the canonical example of maximally entangled multipartite state.

<Definition term="GHZ State">
The n-qubit GHZ state is:
$$|GHZ_n\rangle = \frac{1}{\sqrt{2}}(|00...0\rangle + |11...1\rangle)$$
It is a superposition of all qubits being 0 and all qubits being 1.
</Definition>

```python
import numpy as np

def ghz_state(n):
    """
    Create an n-qubit GHZ state.

    Args:
        n: number of qubits

    Returns:
        State vector of dimension 2^n
    """
    dim = 2**n
    state = np.zeros(dim, dtype=complex)

    # |00...0> has index 0
    state[0] = 1 / np.sqrt(2)

    # |11...1> has index 2^n - 1
    state[dim - 1] = 1 / np.sqrt(2)

    return state

def analyze_ghz(n):
    """
    Analyze properties of the n-qubit GHZ state.
    """
    state = ghz_state(n)
    dim = 2**n

    print(f"{n}-qubit GHZ State Analysis:")
    print("=" * 50)
    print(f"State dimension: {dim}")
    print(f"Non-zero amplitudes: |{'0'*n}> and |{'1'*n}> with amplitude 1/sqrt(2)")

    # Check normalization
    norm = np.linalg.norm(state)
    print(f"Normalization: {norm:.4f}")

    # Measurement statistics in computational basis
    probs = np.abs(state)**2
    print(f"\nMeasurement probabilities:")
    print(f"  P(all 0): {probs[0]:.4f}")
    print(f"  P(all 1): {probs[dim-1]:.4f}")

    return state

# Analyze for different n
for n in [2, 3, 4]:
    analyze_ghz(n)
    print()
```

### GHZ Properties

<Warning>
The GHZ state is maximally entangled but fragile: if any single qubit is lost or measured, the remaining state becomes completely separable. This is in stark contrast to the W state.
</Warning>

```python

    """

    # Trace out qubit 3 (last qubit)

def partial_transpose_2qubit(rho):

    rho_tensor = rho.reshape(2, 2, 2, 2)

    return rho_pt
```

## 3. The W State

The W state represents a fundamentally different type of multipartite entanglement.

<Definition term="W State">
The n-qubit W state is:
$$|W_n\rangle = \frac{1}{\sqrt{n}}(|100...0\rangle + |010...0\rangle + ... + |000...1\rangle)$$
It is a superposition of all states with exactly one qubit in |1>.
</Definition>

```python
def w_state(n):
    """
    Create an n-qubit W state.

    Args:
        n: number of qubits

    Returns:
        State vector of dimension 2^n
    """
    dim = 2**n
    state = np.zeros(dim, dtype=complex)

    # Add each single-excitation basis state
    for i in range(n):
        # State with qubit i in |1>, rest in |0>
        # Index is 2^(n-1-i) due to big-endian convention
        index = 2**(n - 1 - i)
        state[index] = 1 / np.sqrt(n)

    return state

def analyze_w_state(n):
    """
    Analyze properties of the n-qubit W state.
    """
    state = w_state(n)
    dim = 2**n

    print(f"{n}-qubit W State Analysis:")
    print("=" * 50)

    # Find non-zero amplitudes
    nonzero_indices = np.where(np.abs(state) > 1e-10)[0]
    print(f"Non-zero amplitudes at indices: {nonzero_indices}")

    # Each has amplitude 1/sqrt(n)
    amplitude = 1 / np.sqrt(n)
    print(f"Each amplitude: 1/sqrt({n}) = {amplitude:.4f}")

    return state

# Analyze W states
for n in [3, 4]:
    analyze_w_state(n)
    print()
```

### W State Robustness

<Tip>
The W state is robust: if one qubit is lost, the remaining qubits are still entangled! This makes W states valuable for quantum communication scenarios where qubit loss may occur.
</Tip>

## 4. SLOCC Classification

States are classified by what transformations can convert one to another.

<Definition term="SLOCC Equivalence">
Two states are SLOCC (Stochastic Local Operations and Classical Communication) equivalent if one can be converted to the other probabilistically using local operations. States in the same SLOCC class have the same entanglement type.
</Definition>

For three qubits, there are exactly six SLOCC classes:

<Formula>
$$\begin{aligned}
&\text{Separable: } |000\rangle \\
&\text{Biseparable: } |0\rangle \otimes |\Phi^+\rangle, \text{ etc.} \\
&\text{W class: } |W\rangle = \frac{1}{\sqrt{3}}(|100\rangle + |010\rangle + |001\rangle) \\
&\text{GHZ class: } |GHZ\rangle = \frac{1}{\sqrt{2}}(|000\rangle + |111\rangle)
\end{aligned}$$
</Formula>

```python
def classify_three_qubit_state(state):
    """
    Classify a three-qubit pure state into SLOCC classes.

    Uses the three-tangle and bipartite entanglement to classify.
    """
    # Compute reduced density matrices
    rho = np.outer(state, state.conj())
    rho_tensor = rho.reshape(2, 2, 2, 2, 2, 2)

    # Reduced states
    rho_A = np.trace(np.trace(rho_tensor, axis1=1, axis2=4), axis1=1, axis2=2)
    rho_B = np.trace(np.trace(rho_tensor, axis1=0, axis2=3), axis1=1, axis2=2)
    rho_C = np.trace(np.trace(rho_tensor, axis1=0, axis2=3), axis1=0, axis2=1)

    # Linear entropies of reduced states (proxy for entanglement)
    def linear_entropy(rho_reduced):
        return 1 - np.real(np.trace(rho_reduced @ rho_reduced))

    S_A = linear_entropy(rho_A)
    S_B = linear_entropy(rho_B)
    S_C = linear_entropy(rho_C)

    # Three-tangle (Coffman-Kundu-Wootters)
    # Simplified: check if state has GHZ-type or W-type structure
    tau = compute_three_tangle(state)

    print(f"Classification Analysis:")
    print(f"  Linear entropy rho_A: {S_A:.4f}")
    print(f"  Linear entropy rho_B: {S_B:.4f}")
    print(f"  Linear entropy rho_C: {S_C:.4f}")
    print(f"  Three-tangle tau: {tau:.4f}")

    if S_A < 0.01 and S_B < 0.01 and S_C < 0.01:
        return "Separable"
    elif tau > 0.01:
        return "GHZ class"
    elif S_A > 0.01 or S_B > 0.01 or S_C > 0.01:
        return "W class or biseparable"
    else:
        return "Unknown"

def compute_three_tangle(state):
    """
    Compute the three-tangle for a three-qubit state.
    Simplified implementation using concurrence.
    """
    # Reshape state to coefficient matrix
    psi = state.reshape(2, 2, 2)

    # Compute hyperdeterminant (related to three-tangle)
    # For computational simplicity, use approximation

    # The three-tangle distinguishes GHZ (tau > 0) from W (tau = 0)
    a = psi[0, 0, 0]
    d = psi[0, 1, 1]
    e = psi[1, 0, 1]
    f = psi[1, 1, 0]
    b = psi[0, 0, 1]
    c = psi[0, 1, 0]
    g = psi[1, 0, 0]
    h = psi[1, 1, 1]

    # Cayley hyperdeterminant formula
    term1 = (a*h)**2 + (b*g)**2 + (c*f)**2 + (d*e)**2
    term2 = -2 * (a*h*b*g + a*h*c*f + a*h*d*e + b*g*c*f + b*g*d*e + c*f*d*e)
    term3 = 4 * (a*c*f*h + b*d*e*g)

    tau = np.abs(term1 + term2 + term3)
    return tau

# Test classification
print("Three-Qubit SLOCC Classification:")
print("=" * 50)

print("\nGHZ state:")
classify_three_qubit_state(ghz_state(3))

print("\nW state:")
classify_three_qubit_state(w_state(3))

print("\nProduct state |000>:")
product = np.zeros(8)
product[0] = 1
classify_three_qubit_state(product)
```

## 5. Graph States and Cluster States

Graph states are a powerful class of multipartite entangled states with applications in measurement-based quantum computing.

<Definition term="Graph State">
Given a graph G = (V, E) with vertices V and edges E, the graph state |G> is constructed by:
1. Initialize all qubits in |+>
2. Apply CZ (controlled-Z) gates for each edge in E

$$|G\rangle = \prod_{(i,j) \in E} CZ_{ij} |+\rangle^{\otimes n}$$
</Definition>

```python
def create_graph_state(adjacency_matrix):
    """
    Create a graph state from an adjacency matrix.

    Args:
        adjacency_matrix: n x n symmetric matrix where A[i,j] = 1 if edge exists

    Returns:
        State vector of the graph state
    """
    n = len(adjacency_matrix)
    dim = 2**n

    # Start with |+>^n
    plus = np.array([1, 1]) / np.sqrt(2)
    state = plus.copy()
    for _ in range(n - 1):
        state = np.kron(state, plus)

    # Apply CZ for each edge
    for i in range(n):
        for j in range(i + 1, n):
            if adjacency_matrix[i, j] == 1:
                state = apply_cz(state, i, j, n)

    return state

def apply_cz(state, qubit1, qubit2, n):
    """
    Apply CZ gate between qubit1 and qubit2.
    """
    dim = 2**n
    new_state = state.copy()

    for idx in range(dim):
        # Convert index to binary string
        bits = format(idx, f'0{n}b')
        # CZ adds phase -1 if both qubits are |1>
        if bits[qubit1] == '1' and bits[qubit2] == '1':
            new_state[idx] *= -1

    return new_state

# Example: Linear cluster state (path graph)
print("Linear Cluster State (3 qubits):")
print("=" * 50)
# Adjacency matrix for path: 0-1-2
adj_linear = np.array([
    [0, 1, 0],
    [1, 0, 1],
    [0, 1, 0]
])
cluster_linear = create_graph_state(adj_linear)
print("Non-zero amplitudes:")
for idx, amp in enumerate(cluster_linear):
    if np.abs(amp) > 1e-10:
        bits = format(idx, '03b')
        print(f"  |{bits}>: {amp:.4f}")

print("\nStar Graph State (3 qubits, center at qubit 0):")
# Adjacency matrix for star: center 0, leaves 1,2
adj_star = np.array([
    [0, 1, 1],
    [1, 0, 0],
    [1, 0, 0]
])
cluster_star = create_graph_state(adj_star)
print("Non-zero amplitudes:")
for idx, amp in enumerate(cluster_star):
    if np.abs(amp) > 1e-10:
        bits = format(idx, '03b')
        print(f"  |{bits}>: {amp:.4f}")
```

## 6. Genuine Multipartite Entanglement

A state has genuine multipartite entanglement (GME) if it cannot be written as a mixture of biseparable states.

<Definition term="Genuine Multipartite Entanglement">
A state rho is genuinely n-partite entangled if it cannot be written as:
$$\rho = \sum_i p_i \rho^{(i)}_{S_i} \otimes \rho^{(i)}_{\bar{S}_i}$$
for any choice of bipartitions $S_i | \bar{S}_i$.
</Definition>

```python
def check_gme_witness(state, n):
    """
    Check for genuine multipartite entanglement using witness.

    For GHZ-diagonal states, we use the fidelity witness:
    W = alpha * I - |GHZ><GHZ|
    where alpha = 1/2 for optimal detection.
    """
    rho = np.outer(state, state.conj())
    ghz = ghz_state(n)

    # GHZ fidelity
    fidelity = np.abs(np.vdot(ghz, state))**2

    # For biseparable states, max GHZ fidelity is 1/2
    # If fidelity > 1/2, state is GME

    print(f"GME Analysis for {n}-qubit state:")
    print(f"  GHZ fidelity: {fidelity:.4f}")
    print(f"  Threshold for GME: 0.5")
    print(f"  Is GME: {fidelity > 0.5}")

    return fidelity > 0.5

def check_bipartite_entanglement(state, n):
    """
    Check entanglement across all bipartitions.
    """
    rho = np.outer(state, state.conj())

    results = {}
    for cut in range(1, n):
        # Bipartition: first 'cut' qubits vs rest
        dim_a = 2**cut
        dim_b = 2**(n - cut)

        # Compute partial transpose
        rho_tensor = rho.reshape(dim_a, dim_b, dim_a, dim_b)
        rho_pt = np.transpose(rho_tensor, (0, 3, 2, 1)).reshape(dim_a * dim_b, dim_a * dim_b)

        eigenvalues = np.linalg.eigvalsh(rho_pt)
        min_eig = np.min(eigenvalues)
        negativity = np.sum(np.abs(eigenvalues[eigenvalues < 0]))

        results[f"{cut}|{n-cut}"] = {
            'min_pt_eigenvalue': min_eig,
            'negativity': negativity,
            'is_entangled': negativity > 1e-10
        }

    return results

# Test GME detection
print("Genuine Multipartite Entanglement Analysis:")
print("=" * 50)

print("\nGHZ state:")
check_gme_witness(ghz_state(3), 3)

print("\nW state:")
check_gme_witness(w_state(3), 3)

print("\nBiseparable state |+> x |Bell>:")
plus = np.array([1, 1]) / np.sqrt(2)
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
bisep = np.kron(plus, bell)
check_gme_witness(bisep, 3)
```

## 7. Stabilizer Formalism

Graph states belong to the stabilizer formalism, which provides efficient classical description and manipulation.

<Definition term="Stabilizer">
A stabilizer state |psi> is an eigenstate of an abelian group of Pauli operators (the stabilizer group S). For an n-qubit stabilizer state, S has 2^n elements generated by n independent operators.
$$g|psi> = |psi> \text{ for all } g \in S$$
</Definition>

```python
# Pauli matrices
I = np.eye(2)
X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])

def pauli_string(operators, n):
    """
    Create n-qubit Pauli operator from list of single-qubit operators.
    operators: list like ['X', 'I', 'Z'] for X tensor I tensor Z
    """
    paulis = {'I': I, 'X': X, 'Y': Y, 'Z': Z}
    result = paulis[operators[0]]
    for op in operators[1:]:
        result = np.kron(result, paulis[op])
    return result

def check_stabilizer(state, generators):
    """
    Check if state is stabilized by given generators.
    """
    n = int(np.log2(len(state)))

    print(f"Stabilizer Check:")
    for gen in generators:
        op = pauli_string(gen, n)
        result = op @ state
        eigenvalue = np.vdot(state, result)
        is_stabilized = np.isclose(eigenvalue, 1)
        print(f"  {gen}: eigenvalue = {eigenvalue:.2f}, stabilized = {is_stabilized}")

    return True
```

## 8. Applications of Multipartite Entanglement

Multipartite entanglement enables powerful quantum protocols:

### Quantum Secret Sharing

<Formula>
$$|GHZ\rangle \xrightarrow{\text{measure}} \text{Secret shared among n parties}$$
</Formula>

Only if all parties cooperate can the secret be reconstructed.

### Measurement-Based Quantum Computing

Cluster states enable universal quantum computation through single-qubit measurements only:

```python
def mbqc_demo():
    """
    Demonstrate measurement-based quantum computing concept.
    """
    print("Measurement-Based Quantum Computing:")
    print("=" * 50)
    print("""
    1. Prepare cluster state (entangled resource)
    2. Perform adaptive single-qubit measurements
    3. Measurement outcomes + feed-forward = computation

    Key insight: Entanglement is consumed to perform gates.

    Example: Single qubit rotation Rz(theta)
    - Measure first qubit of 2-qubit cluster in basis:
      cos(theta)|0> + sin(theta)|1>, cos(theta)|0> - sin(theta)|1>
    - Second qubit undergoes Rz(theta) (up to Pauli corrections)
    """)

mbqc_demo()
```
