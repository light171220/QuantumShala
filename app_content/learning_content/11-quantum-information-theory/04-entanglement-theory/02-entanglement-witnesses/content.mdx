## 1. The Need for Entanglement Detection

Full quantum state tomography requires measuring an exponentially large number of observables. For practical entanglement verification, we need more efficient methods.

<Definition term="Entanglement Witness">
An entanglement witness W is a Hermitian operator such that:
1. Tr(W * rho_sep) >= 0 for all separable states rho_sep
2. There exists at least one entangled state rho_ent with Tr(W * rho_ent) < 0

A negative expectation value <W> certifies entanglement.
</Definition>

<Formula>
$$\text{Tr}(W\rho) < 0 \Rightarrow \rho \text{ is entangled}$$
</Formula>

<Tip>
Witnesses provide a sufficient but not necessary condition for entanglement. A non-negative witness value does not prove separability - you may need a different witness.
</Tip>

## 2. Geometric Picture

The set of all quantum states forms a convex set, with separable states forming a convex subset. Entanglement witnesses are hyperplanes separating entangled states from the separable set.

```python
import numpy as np
import matplotlib.pyplot as plt

def visualize_witness_geometry():
    """
    Visualize the geometric interpretation of entanglement witnesses
    in a simplified 2D representation.
    """
    fig, ax = plt.subplots(figsize=(10, 8))

    # Draw the set of all states (larger convex region)
    theta = np.linspace(0, 2*np.pi, 100)
    all_states_x = 2 * np.cos(theta)
    all_states_y = 1.5 * np.sin(theta)
    ax.fill(all_states_x, all_states_y, alpha=0.2, color='blue',
            label='All quantum states')

    # Draw separable states (smaller convex region inside)
    sep_x = 1.2 * np.cos(theta)
    sep_y = 0.9 * np.sin(theta)
    ax.fill(sep_x, sep_y, alpha=0.4, color='green',
            label='Separable states')

    # Draw a witness hyperplane (line in 2D)
    x_line = np.linspace(-2.5, 2.5, 100)
    y_line = 0.7 * x_line + 0.5  # Hyperplane: w.x = c
    ax.plot(x_line, y_line, 'r-', linewidth=2, label='Witness W')

    # Mark an entangled state detected by this witness
    ax.plot(1.5, 1.8, 'ro', markersize=12)
    ax.annotate('Entangled state\n(Tr(Wrho) < 0)', xy=(1.5, 1.8),

                arrowprops=dict(arrowstyle='->', color='red'))

    # Mark a separable state
    ax.plot(0, 0.3, 'go', markersize=10)

    ax.set_xlim(-3, 3)
    ax.set_ylim(-2, 3)
    ax.set_xlabel('State parameter 1')
    ax.set_ylabel('State parameter 2')
    ax.set_title('Geometric Picture of Entanglement Witnesses')
    ax.legend(loc='lower right')
    ax.grid(True, alpha=0.3)
    ax.set_aspect('equal')

    return fig

# Create visualization
fig = visualize_witness_geometry()
plt.show()
```

<Warning>
Not all entangled states can be detected by a single witness. Each witness only detects a subset of entangled states. This is why we need families of witnesses or other detection methods.
</Warning>

## 3. Constructing Witnesses

### Method 1: From a Known Entangled Pure State

The simplest witness construction uses a known entangled state:

<Formula>
$$W = \alpha I - |\psi\rangle\langle\psi|$$
</Formula>

where $\alpha = \max_{|\phi\rangle \in \text{product}} |\langle\phi|\psi\rangle|^2$ is the maximum overlap with product states.

```python
import numpy as np

def construct_witness_from_state(target_state):
    """
    Construct an entanglement witness that detects states
    similar to the target entangled state.

    W = alpha * I - |psi><psi|
    where alpha is the max overlap with product states.

    For a two-qubit state, the max overlap is related to
    the largest Schmidt coefficient squared.
    """
    # Normalize
    psi = target_state / np.linalg.norm(target_state)

    # Compute Schmidt decomposition to find alpha
    psi_matrix = psi.reshape(2, 2)
    U, S, Vh = np.linalg.svd(psi_matrix)

    # Alpha = largest Schmidt coefficient squared
    alpha = S[0]**2

    # Construct witness
    dim = len(psi)
    projector = np.outer(psi, psi.conj())
    W = alpha * np.eye(dim) - projector

    return W, alpha

# Construct witness for Bell state
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
W_bell, alpha = construct_witness_from_state(bell)

print("Witness for Bell state |Phi+>:")
print(f"alpha (max product overlap) = {alpha:.4f}")
print(f"Witness matrix W:\n{np.round(W_bell, 4)}")

# Test the witness
def test_witness(W, rho, name):
    """Compute Tr(W * rho) and report result."""
    expectation = np.real(np.trace(W @ rho))
    status = "ENTANGLED" if expectation < 0 else "Not detected"
    print(f"{name}: Tr(W*rho) = {expectation:.4f} -> {status}")
    return expectation

print("\nTesting witness on various states:")
print("=" * 50)

# Bell state (should be detected)
rho_bell = np.outer(bell, bell.conj())
test_witness(W_bell, rho_bell, "Bell |Phi+>")

# Different Bell state
psi_minus = np.array([0, 1, -1, 0]) / np.sqrt(2)
rho_psi_minus = np.outer(psi_minus, psi_minus.conj())
test_witness(W_bell, rho_psi_minus, "Bell |Psi->")

# Product state
product = np.array([1, 0, 0, 0])  # |00>
rho_product = np.outer(product, product.conj())
test_witness(W_bell, rho_product, "Product |00>")

# Maximally mixed state
rho_mixed = np.eye(4) / 4
test_witness(W_bell, rho_mixed, "Maximally mixed I/4")
```

### Method 2: From Partial Transpose

For states detected by the PPT criterion, we can construct witnesses from the partial transpose.

<Formula>
$$W = (|\phi\rangle\langle\phi|)^{T_B}$$
</Formula>

where $|\phi\rangle$ is an eigenvector of $\rho^{T_B}$ with negative eigenvalue.

```python
def construct_witness_from_ppt(rho):
    """
    Construct a witness from the partial transpose.

    If rho^TB has negative eigenvalues, the eigenvector
    corresponding to the most negative eigenvalue gives
    a witness.
    """
    # Compute partial transpose
    rho_pt = rho.reshape(2, 2, 2, 2)
    rho_pt = np.transpose(rho_pt, (0, 3, 2, 1))
    rho_pt = rho_pt.reshape(4, 4)

    # Find eigenvalues and eigenvectors
    eigenvalues, eigenvectors = np.linalg.eigh(rho_pt)

    # Find most negative eigenvalue
    min_idx = np.argmin(eigenvalues)
    min_eigenvalue = eigenvalues[min_idx]

    if min_eigenvalue >= 0:
        print("State is PPT - no witness from this method")
        return None, eigenvalues

    # Construct witness from eigenvector
    phi = eigenvectors[:, min_idx]
    projector = np.outer(phi, phi.conj())

    # Witness is the partial transpose of this projector
    W_tensor = projector.reshape(2, 2, 2, 2)
    W_tensor = np.transpose(W_tensor, (0, 3, 2, 1))
    W = W_tensor.reshape(4, 4)

    return W, eigenvalues

# Test on Bell state
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
rho_bell = np.outer(bell, bell.conj())

W_ppt, eigs = construct_witness_from_ppt(rho_bell)
print(f"PT eigenvalues: {np.round(eigs, 4)}")
print(f"\nWitness from PPT method:\n{np.round(W_ppt, 4)}")

# Verify it detects the Bell state
exp_val = np.real(np.trace(W_ppt @ rho_bell))
print(f"\nTr(W * rho_bell) = {exp_val:.4f}")
```

## 4. Optimal Witnesses

An optimal witness cannot be improved - it detects the maximal set of entangled states in its "direction."

<Definition term="Optimal Witness">
A witness W is optimal if for all positive operators P and epsilon > 0:
W - epsilon * P is not a valid witness (would give negative values on some separable states).
</Definition>

```python
def check_witness_validity(W, num_tests=10000):
    """
    Test if W is a valid witness by checking many product states.

    Returns the minimum expectation value found.
    """
    min_expectation = float('inf')

    for _ in range(num_tests):
        # Generate random product state |a>|b>
        a = np.random.randn(2) + 1j * np.random.randn(2)
        a = a / np.linalg.norm(a)
        b = np.random.randn(2) + 1j * np.random.randn(2)
        b = b / np.linalg.norm(b)

        product = np.kron(a, b)
        rho_product = np.outer(product, product.conj())

        exp_val = np.real(np.trace(W @ rho_product))
        min_expectation = min(min_expectation, exp_val)

    return min_expectation

def optimize_witness(W_initial, target_state, lr=0.01, iterations=100):
    """
    Attempt to optimize a witness to detect more states.

    Uses gradient descent to maximize detection while
    maintaining validity.
    """
    W = W_initial.copy()
    target_rho = np.outer(target_state, target_state.conj())

    for i in range(iterations):
        # Current detection value
        detection = np.real(np.trace(W @ target_rho))

        # Check validity
        min_sep = check_witness_validity(W, num_tests=1000)

        if min_sep < -1e-6:
            # Invalid - W gives negative on separable states
            # Add identity to make it valid again
            W = W + 0.01 * np.eye(4)
        else:
            # Try to improve detection
            # Gradient points toward the target state
            gradient = target_rho
            W = W - lr * gradient

    return W

# Example: Optimize witness for partially entangled state
partial_state = np.array([np.sqrt(0.8), 0, 0, np.sqrt(0.2)])
W_initial, _ = construct_witness_from_state(bell)

print("Initial witness performance:")
rho_partial = np.outer(partial_state, partial_state.conj())
exp_initial = np.real(np.trace(W_initial @ rho_partial))
print(f"Tr(W * rho_partial) = {exp_initial:.4f}")

# The initial witness may not optimally detect this state
```

## 5. Decomposable vs Non-Decomposable Witnesses

<Definition term="Decomposable Witness">
A witness W is decomposable if it can be written as:
$$W = P + Q^{T_B}$$
where P and Q are positive semidefinite operators. Decomposable witnesses can only detect NPT (negative partial transpose) entangled states.
</Definition>

<Warning>
To detect PPT entangled states (bound entanglement), we need non-decomposable witnesses. These are harder to construct and require specific techniques.
</Warning>

```python
def is_decomposable_witness(W):
    """
    Check if a witness might be decomposable.

    This is a heuristic check - full verification is complex.
    We check if W detects any state with positive partial transpose.
    """
    # Generate many random states
    detected_ppt = False

    for _ in range(1000):
        # Random density matrix
        G = np.random.randn(4, 4) + 1j * np.random.randn(4, 4)
        rho = G @ G.conj().T
        rho = rho / np.trace(rho)

        # Check if detected by witness
        exp_val = np.real(np.trace(W @ rho))

        if exp_val < -1e-6:
            # Check if this state is PPT
            rho_pt = rho.reshape(2, 2, 2, 2)
            rho_pt = np.transpose(rho_pt, (0, 3, 2, 1))
            rho_pt = rho_pt.reshape(4, 4)

            pt_eigenvalues = np.linalg.eigvalsh(rho_pt)
            if np.min(pt_eigenvalues) >= -1e-10:
                detected_ppt = True
                print("Found PPT state detected by witness!")
                break

    return not detected_ppt

# Check our Bell state witness
W_bell, _ = construct_witness_from_state(bell)
is_dec = is_decomposable_witness(W_bell)
print(f"Bell witness appears decomposable: {is_dec}")
```

## 6. Experimental Implementation

Witnesses are experimentally valuable because they require measuring only a few observables.

```python
def decompose_witness_into_paulis(W):
    """
    Decompose a two-qubit witness into Pauli basis.

    W = sum_{i,j} c_{ij} sigma_i x sigma_j

    This tells us which observables to measure experimentally.
    """
    # Pauli matrices
    paulis = {
        'I': np.eye(2),
        'X': np.array([[0, 1], [1, 0]]),
        'Y': np.array([[0, -1j], [1j, 0]]),
        'Z': np.array([[1, 0], [0, -1]])
    }

    labels = ['I', 'X', 'Y', 'Z']
    coefficients = {}

    for i, label_a in enumerate(labels):
        for j, label_b in enumerate(labels):
            # Two-qubit Pauli
            P = np.kron(paulis[label_a], paulis[label_b])

            # Coefficient is Tr(P W) / 4
            coeff = np.real(np.trace(P @ W)) / 4

            if np.abs(coeff) > 1e-10:
                coefficients[f"{label_a}{label_b}"] = coeff

    return coefficients

# Decompose Bell state witness
W_bell, _ = construct_witness_from_state(bell)
pauli_coeffs = decompose_witness_into_paulis(W_bell)

print("Pauli decomposition of Bell state witness:")
print("=" * 40)
for pauli, coeff in sorted(pauli_coeffs.items()):
    print(f"  {pauli}: {coeff:+.4f}")

# Calculate experimental requirements
print(f"\nExperimental requirements:")
print(f"  Number of measurement settings: {len(pauli_coeffs)}")
print(f"  (Compare to full tomography: 15 settings for 2 qubits)")
```

Output:
```
Pauli decomposition of Bell state witness:
========================================
  II: +0.2500
  XX: -0.2500
  YY: +0.2500
  ZZ: -0.2500

Experimental requirements:
  Number of measurement settings: 4
  (Compare to full tomography: 15 settings for 2 qubits)
```

<Tip>
The Bell state witness only requires 4 measurement settings (II, XX, YY, ZZ), compared to 15 for full tomography. This exponential reduction makes witness methods practical for larger systems.
</Tip>

## 7. Witness for Werner States

```python
def analyze_werner_with_witness(W, p_values):
    """
    Analyze Werner states using a witness.
    """
    bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
    rho_bell = np.outer(bell, bell.conj())

    results = []
    for p in p_values:
        rho_werner = p * rho_bell + (1 - p) * np.eye(4) / 4
        exp_val = np.real(np.trace(W @ rho_werner))
        results.append(exp_val)

    return np.array(results)

# Analyze
p_vals = np.linspace(0, 1, 100)
W_bell, _ = construct_witness_from_state(bell)
expectations = analyze_werner_with_witness(W_bell, p_vals)

# Find threshold
threshold_idx = np.argmin(np.abs(expectations))
threshold_p = p_vals[threshold_idx]

print(f"Werner state analysis with Bell witness:")
print(f"  Detection threshold: p = {threshold_p:.3f}")
print(f"  (True entanglement threshold: p = 1/3 = 0.333)")

# The witness detects states with p > 1/2
# This is not optimal - there's room for improvement
```

## Summary

In this lesson, you learned:

1. **Witness Definition**: Hermitian operators that detect entanglement via negative expectation values
2. **Geometric Picture**: Witnesses are hyperplanes separating entangled states from the separable set
3. **Construction Methods**: From target states or from partial transpose eigenvectors
4. **Optimality**: Optimal witnesses detect maximal sets of entangled states
5. **Decomposability**: Decomposable witnesses only detect NPT states; detecting bound entanglement requires non-decomposable witnesses
6. **Experimental Advantage**: Witnesses require fewer measurements than full tomography

Witnesses are essential tools for practical entanglement verification in quantum information experiments.
