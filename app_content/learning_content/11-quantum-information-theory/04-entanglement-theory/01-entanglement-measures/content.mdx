## 1. Why Quantify Entanglement?

Entanglement is the quintessential quantum resource. But how much entanglement does a state have? This seemingly simple question leads to rich mathematical structures and multiple valid answers depending on what operational task we consider.

<Definition term="Entanglement Measure">
An entanglement measure E is a function from quantum states to non-negative real numbers satisfying: (1) E(rho) = 0 for separable states, (2) E does not increase under LOCC (Local Operations and Classical Communication), and (3) E reduces to entropy for pure states.
</Definition>

Different measures capture different aspects of entanglement:
- **Entanglement entropy**: How mixed does a subsystem appear?
- **Negativity**: How negative are partial transpose eigenvalues?
- **Concurrence**: Two-qubit specific measure with operational meaning

## 2. Entanglement Entropy for Pure States

For a pure bipartite state, entanglement entropy is defined via the reduced density matrix.

<Formula>
$$E(\ket{\psi}_{AB}) = S(\rho_A) = S(\rho_B) = -\text{Tr}(\rho_A \log_2 \rho_A)$$
</Formula>

where $\rho_A = \text{Tr}_B(\ket{\psi}\bra{\psi})$ is the reduced density matrix.

<Definition term="Schmidt Decomposition">
Any pure bipartite state can be written as:
$$\ket{\psi}_{AB} = \sum_{i=1}^{r} \sqrt{\lambda_i} \ket{i}_A \ket{i}_B$$
where $\lambda_i > 0$, $\sum_i \lambda_i = 1$, and r is the Schmidt rank. The state is entangled if and only if r > 1.
</Definition>

The entanglement entropy in terms of Schmidt coefficients:

<Formula>
$$E(\ket{\psi}) = -\sum_{i=1}^{r} \lambda_i \log_2 \lambda_i = H(\{\lambda_i\})$$
</Formula>

```python
import numpy as np
from scipy.linalg import svd

def schmidt_decomposition(state_vector, dim_a, dim_b):
    """
    Compute Schmidt decomposition of a bipartite pure state.

    Args:
        state_vector: numpy array of shape (dim_a * dim_b,)
        dim_a: dimension of subsystem A
        dim_b: dimension of subsystem B

    Returns:
        schmidt_coefficients: array of sqrt(lambda_i)
        schmidt_vectors_a: basis vectors for A
        schmidt_vectors_b: basis vectors for B
    """
    # Reshape state into matrix form
    psi_matrix = state_vector.reshape(dim_a, dim_b)

    # SVD gives Schmidt decomposition
    U, S, Vh = svd(psi_matrix, full_matrices=False)

    # Filter out numerical zeros
    nonzero = S > 1e-10

    return S[nonzero], U[:, nonzero], Vh[nonzero, :]

def entanglement_entropy(state_vector, dim_a, dim_b):
    """
    Calculate von Neumann entanglement entropy.

    Returns entropy in bits (log base 2).
    """
    schmidt_coeffs, _, _ = schmidt_decomposition(state_vector, dim_a, dim_b)

    # Schmidt coefficients are sqrt(lambda), we need lambda
    lambdas = schmidt_coeffs**2

    # Compute entropy: -sum(lambda * log2(lambda))
    # Handle numerical issues with log(0)
    entropy = 0
    for lam in lambdas:
        if lam > 1e-15:
            entropy -= lam * np.log2(lam)

    return entropy

# Example: Bell state
bell_state = np.array([1, 0, 0, 1]) / np.sqrt(2)  # |Phi+>
E_bell = entanglement_entropy(bell_state, 2, 2)
print(f"Bell state entanglement entropy: {E_bell:.4f} ebits")

# Product state
product_state = np.array([1, 0, 0, 0])  # |00>
E_product = entanglement_entropy(product_state, 2, 2)
print(f"Product state entanglement entropy: {E_product:.4f} ebits")

# Partially entangled state: sqrt(0.8)|00> + sqrt(0.2)|11>
partial_state = np.array([np.sqrt(0.8), 0, 0, np.sqrt(0.2)])
E_partial = entanglement_entropy(partial_state, 2, 2)
print(f"Partial entanglement entropy: {E_partial:.4f} ebits")
```

Output:
```
Bell state entanglement entropy: 1.0000 ebits
Product state entanglement entropy: 0.0000 ebits
Partial entanglement entropy: 0.7219 ebits
```

<Tip>
The maximum entanglement entropy for a d x d system is log_2(d). For two qubits, max entropy is 1 ebit (entanglement bit). The Bell states achieve this maximum.
</Tip>

## 3. The Partial Transpose and PPT Criterion

For mixed states, entanglement detection becomes harder. The Partial Transpose criterion provides a powerful necessary condition.

<Definition term="Partial Transpose">
For a bipartite density matrix written in the computational basis as:
$$\rho = \sum_{ijkl} \rho_{ij,kl} \ket{i}\bra{k} \otimes \ket{j}\bra{l}$$
The partial transpose with respect to subsystem B is:
$$\rho^{T_B} = \sum_{ijkl} \rho_{ij,kl} \ket{i}\bra{k} \otimes \ket{l}\bra{j}$$
</Definition>

<Formula>
$$(\rho^{T_B})_{ij,kl} = \rho_{il,kj}$$
</Formula>

```python
def partial_transpose(rho, dim_a, dim_b, subsystem='B'):
    """
    Compute partial transpose of a bipartite density matrix.

    Args:
        rho: density matrix of shape (dim_a*dim_b, dim_a*dim_b)
        dim_a, dim_b: subsystem dimensions
        subsystem: 'A' or 'B' for which subsystem to transpose

    Returns:
        Partially transposed density matrix
    """
    # Reshape to 4-index tensor
    rho_tensor = rho.reshape(dim_a, dim_b, dim_a, dim_b)

    if subsystem == 'B':
        # Transpose indices 1 and 3 (B subsystem)
        rho_pt = np.transpose(rho_tensor, (0, 3, 2, 1))
    else:  # subsystem == 'A'
        # Transpose indices 0 and 2 (A subsystem)
        rho_pt = np.transpose(rho_tensor, (2, 1, 0, 3))

    return rho_pt.reshape(dim_a * dim_b, dim_a * dim_b)

# Test on Bell state
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
rho_bell = np.outer(bell, bell.conj())

rho_pt = partial_transpose(rho_bell, 2, 2)
eigenvalues = np.linalg.eigvalsh(rho_pt)

print("Bell state density matrix:")
print(np.round(rho_bell, 3))
print("\nPartial transpose:")
print(np.round(rho_pt, 3))
print("\nEigenvalues of partial transpose:", np.round(eigenvalues, 4))
```

Output:
```

[[0.5 0.  0.  0.5]
 [0.  0.  0.  0. ]
 [0.  0.  0.  0. ]
 [0.5 0.  0.  0.5]]

Partial transpose:
[[0.5 0.  0.  0. ]
 [0.  0.  0.5 0. ]
 [0.  0.5 0.  0. ]
 [0.  0.  0.  0.5]]

Eigenvalues of partial transpose: [-0.5  0.5  0.5  0.5]
```

<Warning>
Notice the negative eigenvalue! This indicates entanglement. Separable states always have positive semidefinite partial transposes (PPT). The converse is true only for 2x2 and 2x3 systems.
</Warning>

## 4. Negativity and Logarithmic Negativity

The negativity quantifies how much the partial transpose fails to be positive semidefinite.

<Formula>
$$\mathcal{N}(\rho) = \frac{\|\rho^{T_B}\|_1 - 1}{2} = \sum_{\lambda_i < 0} |\lambda_i|$$
</Formula>

where $\|\cdot\|_1$ is the trace norm (sum of absolute values of eigenvalues).

<Definition term="Logarithmic Negativity">
$$E_N(\rho) = \log_2(\|\rho^{T_B}\|_1) = \log_2(1 + 2\mathcal{N}(\rho))$$
This is an entanglement monotone that is additive under tensor products.
</Definition>

```python
def negativity(rho, dim_a, dim_b):
    """
    Calculate the negativity of a bipartite state.

    Returns:
        negativity: N(rho) = (||rho^TB||_1 - 1) / 2
        log_negativity: E_N = log2(||rho^TB||_1)
    """
    rho_pt = partial_transpose(rho, dim_a, dim_b)
    eigenvalues = np.linalg.eigvalsh(rho_pt)

    # Trace norm = sum of absolute values of eigenvalues
    trace_norm = np.sum(np.abs(eigenvalues))

    neg = (trace_norm - 1) / 2
    log_neg = np.log2(trace_norm) if trace_norm > 0 else 0

    return neg, log_neg

# Test various states
test_states = {
    'Bell |Phi+>': np.outer(
        np.array([1, 0, 0, 1]) / np.sqrt(2),
        np.array([1, 0, 0, 1]) / np.sqrt(2)
    ),
    'Product |00>': np.outer(
        np.array([1, 0, 0, 0]),
        np.array([1, 0, 0, 0])
    ),
    'Werner p=0.5': 0.5 * np.outer(
        np.array([1, 0, 0, 1]) / np.sqrt(2),
        np.array([1, 0, 0, 1]) / np.sqrt(2)
    ) + 0.5 * np.eye(4) / 4,
}

print("Negativity Analysis:")
print("=" * 50)
for name, rho in test_states.items():
    neg, log_neg = negativity(rho, 2, 2)
    print(f"{name}:")
    print(f"  Negativity: {neg:.4f}")
    print(f"  Log-negativity: {log_neg:.4f}")
```

## 5. Concurrence for Two-Qubit States

For two-qubit systems, concurrence provides a complete characterization of entanglement.

<Formula>
$$C(\rho) = \max(0, \lambda_1 - \lambda_2 - \lambda_3 - \lambda_4)$$
</Formula>

where $\lambda_i$ are the square roots of eigenvalues of $\rho \tilde{\rho}$ in decreasing order, and:

<Formula>
$$\tilde{\rho} = (\sigma_y \otimes \sigma_y) \rho^* (\sigma_y \otimes \sigma_y)$$
</Formula>

<Definition term="Entanglement of Formation">
For two-qubit states, the entanglement of formation is:
$$E_F(\rho) = h\left(\frac{1 + \sqrt{1 - C^2}}{2}\right)$$
where $h(x) = -x\log_2 x - (1-x)\log_2(1-x)$ is the binary entropy.
</Definition>

```python
def concurrence(rho):
    """
    Calculate concurrence for a two-qubit density matrix.

    Args:
        rho: 4x4 density matrix

    Returns:
        concurrence value in [0, 1]
    """
    # Pauli Y matrix
    sigma_y = np.array([[0, -1j], [1j, 0]])

    # sigma_y tensor sigma_y
    yy = np.kron(sigma_y, sigma_y)

    # Spin-flipped state: tilde_rho = (Y x Y) rho* (Y x Y)
    rho_tilde = yy @ rho.conj() @ yy

    # R = rho * tilde_rho
    R = rho @ rho_tilde

    # Get eigenvalues
    eigenvalues = np.linalg.eigvals(R)

    # Take square roots of eigenvalues (they should be real and non-negative)
    sqrt_eigs = np.sqrt(np.maximum(np.real(eigenvalues), 0))

    # Sort in decreasing order
    sqrt_eigs = np.sort(sqrt_eigs)[::-1]

    # Concurrence formula
    C = max(0, sqrt_eigs[0] - sqrt_eigs[1] - sqrt_eigs[2] - sqrt_eigs[3])

    return C

def entanglement_of_formation(rho):
    """
    Calculate entanglement of formation for two-qubit state.
    """
    C = concurrence(rho)

    if C == 0:
        return 0

    # Binary entropy function
    def h(x):
        if x <= 0 or x >= 1:
            return 0
        return -x * np.log2(x) - (1 - x) * np.log2(1 - x)

    x = (1 + np.sqrt(1 - C**2)) / 2
    return h(x)

# Test concurrence on various states
print("Concurrence and Entanglement of Formation:")
print("=" * 55)

# Pure Bell state
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
rho_bell = np.outer(bell, bell.conj())
C_bell = concurrence(rho_bell)
EoF_bell = entanglement_of_formation(rho_bell)
print(f"Bell state: C = {C_bell:.4f}, E_F = {EoF_bell:.4f}")

# Werner states with different mixing parameters
for p in [0.3, 0.5, 0.7, 0.9]:
    # Werner state: p * |Phi+><Phi+| + (1-p) * I/4
    rho_werner = p * rho_bell + (1 - p) * np.eye(4) / 4
    C_w = concurrence(rho_werner)
    EoF_w = entanglement_of_formation(rho_werner)
    ppt_threshold = 1/3  # Werner states are entangled iff p > 1/3
    print(f"Werner p={p}: C = {C_w:.4f}, E_F = {EoF_w:.4f}, "
          f"Entangled: {p > ppt_threshold}")
```

```

Werner p=0.3: C = 0.0000, E_F = 0.0000, Entangled: False
Werner p=0.5: C = 0.2500, E_F = 0.1208, Entangled: True
Werner p=0.7: C = 0.5500, E_F = 0.3969, Entangled: True
```

<Tip>
For Werner states, the critical threshold is p = 1/3. States with p below this are separable. Concurrence provides an exact formula, unlike negativity which can be zero even for some entangled states in higher dimensions.
</Tip>

## 6. Properties of Entanglement Measures

Good entanglement measures should satisfy several axioms:

| Property | Description | Entropy | Negativity | Concurrence |
|----------|-------------|---------|------------|-------------|
| Non-negative | E(rho) >= 0 | Yes | Yes | Yes |
| Zero for separable | E(rho_sep) = 0 | Yes | Yes | Yes |
| LOCC monotone | Non-increasing under LOCC | Yes | Yes | Yes |
| Convex | E(sum p_i rho_i) <= sum p_i E(rho_i) | Mixed only | No | Yes |
| Additive | E(rho^n) = n*E(rho) | Yes | Yes | No |

```python
def compare_entanglement_measures(rho, dim_a=2, dim_b=2, pure=False):
    """
    Compare different entanglement measures for a state.
    """
    measures = {}

    # For pure states, use entropy
    if pure:
        state = rho  # rho is actually the state vector
        rho_matrix = np.outer(state, state.conj())
        measures['Entropy (ebits)'] = entanglement_entropy(state, dim_a, dim_b)
    else:
        rho_matrix = rho

    # Negativity (works for pure and mixed)
    neg, log_neg = negativity(rho_matrix, dim_a, dim_b)
    measures['Negativity'] = neg
    measures['Log-Negativity'] = log_neg

    # Concurrence (only for 2x2)
    if dim_a == 2 and dim_b == 2:
        C = concurrence(rho_matrix)
        EoF = entanglement_of_formation(rho_matrix)
        measures['Concurrence'] = C
        measures['E of Formation'] = EoF

    return measures

# Compare for Bell state
print("Bell State |Phi+>:")
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
measures = compare_entanglement_measures(bell, pure=True)
for name, value in measures.items():
    print(f"  {name}: {value:.4f}")

# Compare for mixed Werner state
print("\nWerner State (p=0.7):")
rho_bell = np.outer(bell, bell.conj())
rho_werner = 0.7 * rho_bell + 0.3 * np.eye(4) / 4
measures = compare_entanglement_measures(rho_werner)
for name, value in measures.items():
    print(f"  {name}: {value:.4f}")
```

## 7. Higher-Dimensional Systems

For systems beyond qubits, entanglement quantification becomes more complex.

<Warning>
The PPT criterion is necessary but not sufficient for separability in dimensions higher than 2x2 or 2x3. There exist PPT entangled states (bound entangled states) that cannot be detected by negativity.
</Warning>

```python
def analyze_qutrit_state(state_vector):
    """
    Analyze entanglement in a 3x3 (qutrit-qutrit) system.
    """
    dim = 3

    # Schmidt decomposition
    coeffs, U, V = schmidt_decomposition(state_vector, dim, dim)
    schmidt_rank = len(coeffs)

    # Entanglement entropy
    E = entanglement_entropy(state_vector, dim, dim)

    # Negativity
    rho = np.outer(state_vector, state_vector.conj())
    neg, log_neg = negativity(rho, dim, dim)

    print(f"Qutrit System Analysis:")
    print(f"  Schmidt rank: {schmidt_rank}")
    print(f"  Schmidt coefficients: {np.round(coeffs, 4)}")
    print(f"  Entanglement entropy: {E:.4f} ebits (max: {np.log2(dim):.4f})")
    print(f"  Negativity: {neg:.4f}")
    print(f"  Log-negativity: {log_neg:.4f}")

    return E, neg

# Maximally entangled qutrit state: (|00> + |11> + |22>) / sqrt(3)
qutrit_bell = np.zeros(9)
qutrit_bell[0] = 1/np.sqrt(3)  # |00>
qutrit_bell[4] = 1/np.sqrt(3)  # |11>
qutrit_bell[8] = 1/np.sqrt(3)  # |22>

analyze_qutrit_state(qutrit_bell)
```

Output:
```
Qutrit System Analysis:
  Schmidt rank: 3
  Schmidt coefficients: [0.5774 0.5774 0.5774]
  Entanglement entropy: 1.5850 ebits (max: 1.5850)
  Negativity: 0.6667
  Log-negativity: 1.0000
```

## 8. Operational Interpretation

Entanglement measures have operational meanings:

<Definition term="Distillable Entanglement">
The rate at which Bell pairs can be extracted from many copies of a state using LOCC:
$$E_D(\rho) = \lim_{n \to \infty} \frac{1}{n} \max_{\text{LOCC}} |\text{Bell pairs from } \rho^{\otimes n}|$$
</Definition>

<Definition term="Entanglement Cost">
The rate at which Bell pairs are needed to create many copies of a state:
$$E_C(\rho) = \lim_{n \to \infty} \frac{1}{n} \min_{\text{LOCC}} |\text{Bell pairs for } \rho^{\otimes n}|$$
</Definition>

For pure states: $E_D = E_C = E$ (entanglement entropy)
For mixed states: $E_D \leq E_F \leq E_C$ in general
