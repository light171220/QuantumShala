## 1. The Black Hole Information Paradox

The black hole information paradox is perhaps the most important theoretical puzzle in fundamental physics, sitting at the intersection of quantum mechanics, general relativity, and information theory.

<Definition term="Black Hole Information Paradox">
The black hole information paradox arises from the apparent conflict between:
1. Quantum mechanics: Evolution is unitary, preserving information
2. General relativity: Black holes evaporate via Hawking radiation
3. Observation: Hawking radiation appears thermal (maximum entropy, no information)

If black holes destroy information, quantum mechanics breaks down.
</Definition>

### The Paradox in Detail

<Formula>
$$S_{radiation}(t) \stackrel{?}{=} S_{black hole}(t)$$
</Formula>

```python
import numpy as np
import matplotlib.pyplot as plt

class BlackHoleInformationParadox:
    """Illustrate the black hole information paradox."""

    @staticmethod
    def hawking_radiation_properties():
        """
        Key properties of Hawking radiation that create the paradox.
        """
        return {
            'thermal_spectrum': 'Radiation has Planck spectrum at T_H',
            'temperature': 'T_H = hbar * c^3 / (8 * pi * G * M * k_B)',
            'entropy_rate': 'dS/dt > 0 for thermal radiation',
            'problem': 'Thermal radiation carries no information about '
                      'what fell into the black hole',
            'consequence': 'Pure state -> Mixed state? Violates unitarity!'
        }

    @staticmethod
    def hawking_vs_unitary_entropy(t_page: float = 1.0):
        """
        Compare entropy evolution in Hawking's calculation vs unitary evolution.

        Args:
            t_page: Page time (when BH has lost half its entropy)

        Returns:
            Arrays of time and entropy values
        """
        t = np.linspace(0, 2 * t_page, 100)

        # Hawking's calculation: entropy always increases
        S_hawking = np.where(t < 2 * t_page,
                            t / t_page,  # Linear increase
                            2.0)  # Saturates at max

        # Unitary evolution (Page curve): entropy decreases after Page time
        S_unitary = np.where(t < t_page,
                            t / t_page,  # Increase before Page time
                            2 - t / t_page)  # Decrease after

        return t, S_hawking, S_unitary

# Illustrate the paradox
bh_paradox = BlackHoleInformationParadox()

print("Black Hole Information Paradox")
print("=" * 50)

props = bh_paradox.hawking_radiation_properties()
for key, value in props.items():
    print(f"{key}: {value}")

t, S_hawking, S_unitary = bh_paradox.hawking_vs_unitary_entropy()

print("\nEntropy evolution comparison:")
print("Time | Hawking | Unitary")
for i in range(0, len(t), 20):
    print(f"{t[i]:.2f} |  {S_hawking[i]:.2f}   |  {S_unitary[i]:.2f}")
```

## 2. The Page Curve

<Definition term="Page Curve">
The Page curve is the expected behavior of the entanglement entropy of Hawking radiation for a unitary evaporating black hole. It rises initially, peaks at the Page time (when the black hole has lost half its initial entropy), and then decreases back to zero.
</Definition>

<Formula>
$$S_{radiation}(t) = \begin{cases}
S_{coarse}(t) & t < t_{Page} \\
S_{BH}(t) & t > t_{Page}
\end{cases}$$
</Formula>

```python
class PageCurve:
    """Model the Page curve for black hole evaporation."""

    def __init__(self, S_0: float, t_evap: float):
        """
        Initialize Page curve model.

        Args:
            S_0: Initial black hole entropy (Bekenstein-Hawking)
            t_evap: Total evaporation time
        """
        self.S_0 = S_0
        self.t_evap = t_evap
        self.t_page = t_evap / 2  # Page time

    def black_hole_entropy(self, t: float) -> float:
        """
        Black hole entropy as function of time.

        S_BH(t) = S_0 * (1 - t/t_evap)^{2/3}

        (For Schwarzschild: S ~ M^2 and M ~ (t_evap - t)^{1/3})
        """
        if t >= self.t_evap:
            return 0
        return self.S_0 * (1 - t / self.t_evap) ** (2/3)

    def coarse_grained_radiation_entropy(self, t: float) -> float:
        """
        Coarse-grained entropy of radiation (thermal, no correlations).

        This is what Hawking calculated - always increasing.
        """
        if t <= 0:
            return 0
        if t >= self.t_evap:
            return self.S_0

        # Radiation entropy = initial BH entropy - current BH entropy
        return self.S_0 - self.black_hole_entropy(t)

    def page_curve(self, t: float) -> float:
        """
        The true (fine-grained) entanglement entropy of radiation.

        Before Page time: S = coarse-grained radiation entropy
        After Page time: S = black hole entropy
        """
        S_rad = self.coarse_grained_radiation_entropy(t)
        S_BH = self.black_hole_entropy(t)

        # Page curve is the minimum
        return min(S_rad, S_BH)

    def analyze(self, n_points: int = 100) -> dict:
        """
        Compute Page curve and related quantities.
        """
        times = np.linspace(0, self.t_evap, n_points)

        S_BH = [self.black_hole_entropy(t) for t in times]
        S_rad_coarse = [self.coarse_grained_radiation_entropy(t) for t in times]
        S_page = [self.page_curve(t) for t in times]

        # Find Page time (where curves cross)
        page_idx = np.argmin(np.abs(np.array(S_BH) - np.array(S_rad_coarse)))
        t_page_actual = times[page_idx]

        return {
            'times': times,
            'S_BH': np.array(S_BH),
            'S_rad_coarse': np.array(S_rad_coarse),
            'S_page': np.array(S_page),
            't_page': t_page_actual,
            'S_at_page': self.page_curve(t_page_actual)
        }

# Demonstrate Page curve
page = PageCurve(S_0=100, t_evap=10)
results = page.analyze()

print("\nPage Curve Analysis")
print("=" * 50)
print(f"Initial BH entropy S_0 = {page.S_0}")
print(f"Evaporation time = {page.t_evap}")
print(f"Page time t_page = {results['t_page']:.2f}")
print(f"Peak entropy = {results['S_at_page']:.2f}")

print("\nTime evolution:")
print("t    | S_BH  | S_rad(coarse) | S_page")
print("-" * 45)
for i in range(0, len(results['times']), 20):
    t = results['times'][i]
    print(f"{t:.1f}  | {results['S_BH'][i]:.1f}  |    {results['S_rad_coarse'][i]:.1f}       | {results['S_page'][i]:.1f}")
```

## 3. The Island Formula

The recent breakthrough in resolving the information paradox comes from the **island formula** for entanglement entropy.

<Definition term="Island Formula">
The fine-grained entropy of radiation R is computed by extremizing over possible "islands" I in the black hole interior:
$$S(R) = \min_{I} \text{ext}_I \left[ \frac{\text{Area}(\partial I)}{4G_N} + S_{bulk}(R \cup I) \right]$$
The island I is a region whose boundary contribution must be included.
</Definition>

<Formula>
$$S_{gen}(R) = \frac{A(\partial I)}{4G} + S_{matter}(R \cup I)$$
</Formula>

```python
class IslandFormula:
    """Implement the island formula for entanglement entropy."""

    def __init__(self, S_BH_0: float, c: float = 1.0):
        """
        Args:
            S_BH_0: Initial Bekenstein-Hawking entropy
            c: Central charge of matter CFT
        """
        self.S_BH_0 = S_BH_0
        self.c = c

    def no_island_entropy(self, t: float, t_evap: float) -> float:
        """
        Entropy without island contribution.

        This is the coarse-grained (Hawking) result.
        """
        if t <= 0:
            return 0
        if t >= t_evap:
            return self.S_BH_0

        # Linearly increasing thermal entropy
        return self.S_BH_0 * t / t_evap

    def island_entropy(self, t: float, t_evap: float) -> float:
        """
        Entropy with island contribution.

        S = S_BH(t) + S_matter(radiation in island)

        For large black hole, the matter entropy is negligible,
        so this approximately equals S_BH.
        """
        if t >= t_evap:
            return 0

        S_BH = self.S_BH_0 * (1 - t / t_evap)

        # Small correction from matter entropy
        S_matter = self.c * np.log(t + 1)  # Simplified

        return S_BH + S_matter

    def generalized_entropy(self, t: float, t_evap: float) -> dict:
        """
        Compute generalized entropy with and without island.

        The true entropy is the minimum of the two.
        """
        S_no_island = self.no_island_entropy(t, t_evap)
        S_with_island = self.island_entropy(t, t_evap)

        # Which surface is extremal?
        use_island = S_with_island < S_no_island

        return {
            'S_no_island': S_no_island,
            'S_with_island': S_with_island,
            'S_fine_grained': min(S_no_island, S_with_island),
            'island_dominates': use_island,
            'interpretation': 'Island connects radiation to BH interior' if use_island
                            else 'No island needed (early times)'
        }

    def compute_page_curve(self, t_evap: float, n_points: int = 100) -> dict:
        """
        Compute the full Page curve using the island formula.
        """
        times = np.linspace(0, t_evap, n_points)

        S_no_island = []
        S_with_island = []
        S_fine_grained = []

        for t in times:
            result = self.generalized_entropy(t, t_evap)
            S_no_island.append(result['S_no_island'])
            S_with_island.append(result['S_with_island'])
            S_fine_grained.append(result['S_fine_grained'])

        # Find Page time (transition point)
        crossover_idx = np.argmin(np.abs(
            np.array(S_no_island) - np.array(S_with_island)
        ))

        return {
            'times': times,
            'S_no_island': np.array(S_no_island),
            'S_with_island': np.array(S_with_island),
            'S_fine_grained': np.array(S_fine_grained),
            't_page': times[crossover_idx]
        }

# Demonstrate island formula
island = IslandFormula(S_BH_0=100, c=1.0)

print("\nIsland Formula Demonstration")
print("=" * 50)

page_result = island.compute_page_curve(t_evap=10)

print(f"Page time from island formula: t_page = {page_result['t_page']:.2f}")
print("\nEntropy comparison:")
print("t    | No Island | With Island | Fine-grained | Dominant")
print("-" * 60)

for i in range(0, len(page_result['times']), 20):
    t = page_result['times'][i]
    S_no = page_result['S_no_island'][i]
    S_with = page_result['S_with_island'][i]
    S_fine = page_result['S_fine_grained'][i]
    dominant = "Island" if S_with < S_no else "No Island"
    print(f"{t:.1f}  |   {S_no:.1f}    |    {S_with:.1f}     |    {S_fine:.1f}     | {dominant}")
```

<Tip>
The island formula provides a gravitational calculation that reproduces the Page curve. The key insight is that the entropy of radiation must include contributions from an "island" region inside the black hole, connected to the radiation through quantum entanglement encoded in the spacetime geometry.
</Tip>

## 4. Quantum Extremal Surfaces

<Definition term="Quantum Extremal Surface">
A quantum extremal surface (QES) is a surface that extremizes the generalized entropy:
$$S_{gen} = \frac{A}{4G_N} + S_{bulk}$$
The QES generalizes the Ryu-Takayanagi surface to include quantum corrections.
</Definition>

<Formula>
$$\delta S_{gen} = 0 \Rightarrow \frac{\delta A}{4G} + \delta S_{bulk} = 0$$
</Formula>

```python
class QuantumExtremalSurface:
    """Analyze quantum extremal surfaces."""

    def __init__(self, G_N: float = 1.0):
        self.G_N = G_N

    def classical_area_term(self, r: float) -> float:
        """
        Area term A/(4G) for a spherical surface at radius r.

        For 2D: "Area" = circumference = 2*pi*r
        For 3D sphere: Area = 4*pi*r^2
        """
        return (2 * np.pi * r) / (4 * self.G_N)

    def bulk_entropy(self, r: float, matter_density: float = 0.1) -> float:
        """
        Bulk matter entropy inside radius r.

        Simplified model: S_bulk ~ matter_density * r
        """
        return matter_density * r

    def generalized_entropy(self, r: float, matter_density: float = 0.1) -> float:
        """
        Generalized entropy: S_gen = A/(4G) + S_bulk
        """
        return self.classical_area_term(r) + self.bulk_entropy(r, matter_density)

    def find_qes(self, r_range: tuple, matter_density: float = 0.1) -> dict:
        """
        Find the quantum extremal surface by minimizing S_gen.

        Args:
            r_range: (r_min, r_max) to search
            matter_density: Matter entropy density

        Returns:
            Dictionary with QES location and entropy
        """
        r_values = np.linspace(r_range[0], r_range[1], 100)
        S_gen_values = [self.generalized_entropy(r, matter_density)
                       for r in r_values]

        # Find minimum
        min_idx = np.argmin(S_gen_values)
        r_qes = r_values[min_idx]
        S_qes = S_gen_values[min_idx]

        return {
            'r_qes': r_qes,
            'S_qes': S_qes,
            'A_term': self.classical_area_term(r_qes),
            'S_bulk_term': self.bulk_entropy(r_qes, matter_density),
            'r_values': r_values,
            'S_gen_values': np.array(S_gen_values)
        }

# Demonstrate QES
qes = QuantumExtremalSurface(G_N=1.0)

print("\nQuantum Extremal Surface Analysis")
print("=" * 50)

result = qes.find_qes(r_range=(0.1, 10), matter_density=0.1)

print(f"QES radius: r = {result['r_qes']:.2f}")
print(f"Generalized entropy at QES: S_gen = {result['S_qes']:.4f}")
print(f"  Area term A/(4G): {result['A_term']:.4f}")
print(f"  Bulk entropy: {result['S_bulk_term']:.4f}")

print("\nS_gen vs radius:")
for i in range(0, len(result['r_values']), 20):
    r = result['r_values'][i]
    S = result['S_gen_values'][i]
    marker = " <- QES" if abs(r - result['r_qes']) < 0.5 else ""
    print(f"  r = {r:.1f}: S_gen = {S:.4f}{marker}")
```

## 5. Information Recovery and Decoding

<Definition term="Hayden-Preskill Protocol">
The Hayden-Preskill thought experiment shows that information thrown into an old black hole (past its Page time) can be recovered from just a few Hawking quanta - but only if you also have access to the early radiation.
</Definition>

```python
class HaydenPreskill:
    """Simulate the Hayden-Preskill information recovery protocol."""

    def __init__(self, n_qubits: int = 10):
        """
        Args:
            n_qubits: Number of qubits representing the black hole
        """
        self.n = n_qubits
        self.page_time_qubits = n_qubits // 2

    def information_recovery_time(self, t_scramble: float = 1.0) -> dict:
        """
        Calculate when information can be recovered.

        Before Page time: Need almost all radiation
        After Page time: Need only O(log S) additional qubits
        """
        return {
            'before_page': {
                'qubits_needed': 'n - O(1)',
                'interpretation': 'Almost all radiation needed'
            },
            'after_page': {
                'qubits_needed': 'O(log n) after scrambling time',
                'scrambling_time': t_scramble * np.log(self.n),
                'interpretation': 'Only a few qubits needed, but must wait '
                                'for scrambling'
            }
        }

    def recovery_protocol(self) -> dict:
        """
        Describe the Hayden-Preskill recovery protocol.
        """
        return {
            'step_1': 'Alice throws her diary (k qubits) into an OLD black hole',
            'step_2': f'Bob has already collected early radiation ({self.page_time_qubits} qubits)',
            'step_3': 'Bob waits scrambling time t_* ~ log(S)',
            'step_4': 'Bob collects k + O(1) additional Hawking quanta',
            'step_5': 'Bob applies a (very complex) decoding operation',
            'result': "Bob recovers Alice's diary!",
            'caveat': 'Decoding operation has complexity ~ exp(S)'
        }

# Demonstrate Hayden-Preskill
hp = HaydenPreskill(n_qubits=100)

print("\nHayden-Preskill Protocol")
print("=" * 50)

recovery = hp.information_recovery_time()
print("\nInformation recovery requirements:")
print(f"Before Page time: {recovery['before_page']['interpretation']}")
print(f"After Page time: {recovery['after_page']['interpretation']}")
print(f"  Scrambling time: {recovery['after_page']['scrambling_time']:.2f}")

print("\nProtocol steps:")
protocol = hp.recovery_protocol()
for step, description in protocol.items():
    print(f"  {step}: {description}")
```

## 6. Firewalls and Complementarity

<Definition term="Firewall Paradox">
The firewall paradox (AMPS) argues that either:
1. The horizon has high-energy excitations (firewall) destroying infalling observers
2. Information is lost (violating unitarity)
3. Quantum mechanics is non-linear

The paradox arises from demanding that old black hole radiation is maximally entangled with BOTH the remaining black hole AND the interior modes.
</Definition>

```python
class FirewallParadox:
    """Analyze the firewall paradox and proposed resolutions."""

    @staticmethod
    def monogamy_conflict():
        """
        Explain the monogamy conflict at the heart of the paradox.
        """
        return {
            'setup': 'Consider three systems: B (early radiation), A (late radiation), C (interior)',
            'requirement_1': 'B and (A+C) must be maximally entangled (Page curve)',
            'requirement_2': 'A and C must be maximally entangled (smooth horizon)',
            'conflict': 'Monogamy of entanglement forbids both!',
            'options': [
                'Firewall at horizon (breaks requirement 2)',
                'Information loss (breaks requirement 1)',
                'Non-linear QM',
                'ER=EPR (change the rules)'
            ]
        }

    @staticmethod
    def proposed_resolutions():
        """
        Overview of proposed resolutions.
        """
        return {
            'firewall': {
                'proposal': 'High-energy excitations at horizon after Page time',
                'problem': 'Violates equivalence principle'
            },
            'complementarity': {
                'proposal': 'Interior and exterior are complementary descriptions',
                'problem': 'May lead to contradictions for infalling observers'
            },
            'er_epr': {
                'proposal': 'Entanglement creates wormholes connecting B to interior',
                'status': 'Active area of research'
            },
            'islands': {
                'proposal': 'Island formula includes interior in radiation entropy',
                'status': 'Partial resolution, ongoing research'
            }
        }

        print(f"{key}: {value}")
```

These developments suggest that spacetime itself may be an emergent phenomenon, arising from patterns of quantum entanglement. The geometry encodes quantum information, and operations on quantum systems can be understood as geometric transformations.
</Definition>

4. **Unitarity preserved**: Island formula resolution
