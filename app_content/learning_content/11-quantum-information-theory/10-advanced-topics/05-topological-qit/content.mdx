## 1. Topological Phases of Matter

<Definition term="Topological Phase">
A topological phase of matter is a quantum phase characterized by global entanglement patterns rather than local order parameters. The ground state degeneracy depends on the topology of the manifold, and local perturbations cannot distinguish between degenerate states.
</Definition>

Key properties of topological phases:
- **Ground state degeneracy** depends on topology (genus)
- **No local order parameter** - cannot be detected locally
- **Gap** protects against local perturbations
- **Anyonic excitations** with exotic exchange statistics

<Formula>
$$\text{Degeneracy} = d^g \text{ on genus-}g \text{ surface}$$
</Formula>

where d depends on the type of anyon.

## 2. Anyons: Beyond Bosons and Fermions

<Definition term="Anyon">
An anyon is a quasiparticle that exists only in two-dimensional systems with exchange statistics that are neither bosonic (phase factor 1) nor fermionic (phase factor -1). When two anyons are exchanged, the wave function acquires a phase e^{i*theta} or, for non-Abelian anyons, undergoes a unitary transformation.
</Definition>

### Abelian Anyons

<Formula>
$$\psi(r_1, r_2) \to e^{i\theta} \psi(r_2, r_1) \quad \text{under exchange}$$
</Formula>

### Non-Abelian Anyons

For non-Abelian anyons, exchange performs a unitary in the degenerate ground state space:

<Formula>
$$|\psi\rangle \to U_{ij} |\psi\rangle$$
</Formula>

```python
import numpy as np
from scipy.linalg import expm

class AnyonBraiding:
    """Simulate braiding of anyons."""

    @staticmethod
    def abelian_braid(theta: float) -> complex:
        """
        Braiding phase for Abelian anyons.

        Args:
            theta: Statistical angle (0 for bosons, pi for fermions)

        Returns:
            Phase factor e^{i*theta}
        """
        return np.exp(1j * theta)

    @staticmethod
    def fibonacci_f_matrix() -> np.ndarray:
        """
        F-matrix for Fibonacci anyons.

        F transforms between different fusion tree orderings.
        For Fibonacci anyons: F_{tau,tau}^{tau,tau}
        """
        phi = (1 + np.sqrt(5)) / 2  # Golden ratio
        F = np.array([
            [1/phi, 1/np.sqrt(phi)],
            [1/np.sqrt(phi), -1/phi]
        ], dtype=complex)
        return F

    @staticmethod
    def fibonacci_r_matrix() -> np.ndarray:
        """
        R-matrix for Fibonacci anyons.

        R gives the phase when braiding two tau anyons.
        """
        phi = (1 + np.sqrt(5)) / 2
        # R-matrix elements
        R = np.array([
            [np.exp(4j * np.pi / 5), 0],
            [0, np.exp(-3j * np.pi / 5)]
        ], dtype=complex)
        return R

    def fibonacci_braid_sigma1(self) -> np.ndarray:
        """
        Braiding matrix sigma_1 for 3 Fibonacci anyons.

        Exchanges first two anyons.
        """
        return self.fibonacci_r_matrix()

    def fibonacci_braid_sigma2(self) -> np.ndarray:
        """
        Braiding matrix sigma_2 for 3 Fibonacci anyons.

        Exchanges second and third anyons.
        Uses F-R-F^{-1} transformation.
        """
        F = self.fibonacci_f_matrix()
        R = self.fibonacci_r_matrix()
        F_inv = np.linalg.inv(F)
        return F @ R @ F_inv

# Demonstrate anyon braiding
braider = AnyonBraiding()

print("Anyon Braiding Statistics")
print("=" * 50)

# Abelian examples
print("\nAbelian anyons:")
print(f"  Boson (theta=0): {braider.abelian_braid(0):.4f}")
print(f"  Fermion (theta=pi): {braider.abelian_braid(np.pi):.4f}")
print(f"  Semion (theta=pi/2): {braider.abelian_braid(np.pi/2):.4f}")

# Non-Abelian: Fibonacci anyons
print("\nFibonacci anyons (non-Abelian):")
print(f"  R-matrix:")
print(f"    {braider.fibonacci_r_matrix()}")
print(f"\n  F-matrix:")
print(f"    {braider.fibonacci_f_matrix()}")

sigma1 = braider.fibonacci_braid_sigma1()
sigma2 = braider.fibonacci_braid_sigma2()
print(f"\n  sigma_1 * sigma_2 * sigma_1 = sigma_2 * sigma_1 * sigma_2?")
lhs = sigma1 @ sigma2 @ sigma1
rhs = sigma2 @ sigma1 @ sigma2
print(f"    Yang-Baxter satisfied: {np.allclose(lhs, rhs)}")
```

<Tip>
Non-Abelian anyons are the key to topological quantum computation. Because braiding operations are unitary transformations, they can implement quantum gates. The topological nature provides inherent protection against errors.
</Tip>

## 3. The Toric Code

<Definition term="Toric Code">
The toric code is a topological quantum error-correcting code defined on a 2D lattice on a torus. It has 4-fold degenerate ground states (encoding 2 logical qubits) protected by an energy gap, with anyonic excitations called e (electric) and m (magnetic) particles.
</Definition>

### Hamiltonian

<Formula>
$$H = -J_e \sum_v A_v - J_m \sum_p B_p$$
</Formula>

where:
- $A_v = \prod_{i \in v} X_i$ (vertex operators)
- $B_p = \prod_{i \in p} Z_i$ (plaquette operators)

```python
import numpy as np

class ToricCode:
    """Implement the toric code on a square lattice."""

    def __init__(self, L: int):
        """
        Initialize L x L toric code.

        Args:
            L: Linear size of the lattice
        """
        self.L = L
        self.n_qubits = 2 * L * L  # Qubits on edges

        # Pauli matrices
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

    def vertex_operator(self, v: tuple) -> list:
        """
        Get the qubits in vertex operator A_v.

        Args:
            v: Vertex coordinate (x, y)

        Returns:
            List of qubit indices around vertex v
        """
        x, y = v
        L = self.L

        # Qubits around vertex: up, down, left, right
        qubits = [
            2 * (x * L + y),                          # right edge
            2 * (x * L + (y - 1) % L),                # left edge
            2 * (x * L + y) + 1,                      # up edge
            2 * (((x - 1) % L) * L + y) + 1           # down edge
        ]
        return qubits

    def plaquette_operator(self, p: tuple) -> list:
        """
        Get the qubits in plaquette operator B_p.

        Args:
            p: Plaquette coordinate (x, y)

        Returns:
            List of qubit indices around plaquette p
        """
        x, y = p
        L = self.L

        # Qubits around plaquette
        qubits = [
            2 * (x * L + y),                          # bottom edge
            2 * (x * L + y) + 1,                      # left edge
            2 * (((x + 1) % L) * L + y) + 1,          # right edge
            2 * (x * L + (y + 1) % L)                 # top edge
        ]
        return qubits

    def create_ground_state_syndrome(self) -> dict:
        """
        Ground state has all A_v = +1 and B_p = +1.
        """
        return {
            'vertex_syndromes': {(x, y): +1 for x in range(self.L) for y in range(self.L)},
            'plaquette_syndromes': {(x, y): +1 for x in range(self.L) for y in range(self.L)},
            'is_ground_state': True
        }

    def apply_x_error(self, qubit: int, syndromes: dict) -> dict:
        """
        Apply X error to a qubit and update syndromes.

        X error creates/annihilates m anyons (plaquette violations).
        """
        L = self.L
        new_syndromes = {
            'vertex_syndromes': syndromes['vertex_syndromes'].copy(),
            'plaquette_syndromes': syndromes['plaquette_syndromes'].copy()
        }

        # Find which plaquettes this qubit belongs to
        edge_idx = qubit // 2
        is_vertical = qubit % 2

        x = edge_idx // L
        y = edge_idx % L

        if is_vertical:
            # Vertical edge: affects plaquettes (x, y) and (x, y-1)
            affected_plaquettes = [(x, y), (x, (y - 1) % L)]
        else:
            # Horizontal edge: affects plaquettes (x, y) and (x-1, y)
            affected_plaquettes = [(x, y), ((x - 1) % L, y)]

        for p in affected_plaquettes:
            new_syndromes['plaquette_syndromes'][p] *= -1

        new_syndromes['is_ground_state'] = all(
            s == 1 for s in new_syndromes['plaquette_syndromes'].values()
        ) and all(
            s == 1 for s in new_syndromes['vertex_syndromes'].values()
        )

        return new_syndromes

    def apply_z_error(self, qubit: int, syndromes: dict) -> dict:
        """
        Apply Z error to a qubit and update syndromes.

        Z error creates/annihilates e anyons (vertex violations).
        """
        L = self.L
        new_syndromes = {
            'vertex_syndromes': syndromes['vertex_syndromes'].copy(),
            'plaquette_syndromes': syndromes['plaquette_syndromes'].copy()
        }

        # Find which vertices this qubit belongs to
        edge_idx = qubit // 2
        is_vertical = qubit % 2

        x = edge_idx // L
        y = edge_idx % L

        if is_vertical:
            # Vertical edge: connects vertices (x, y) and (x+1, y)
            affected_vertices = [(x, y), ((x + 1) % L, y)]
        else:
            # Horizontal edge: connects vertices (x, y) and (x, y+1)
            affected_vertices = [(x, y), (x, (y + 1) % L)]

        for v in affected_vertices:
            new_syndromes['vertex_syndromes'][v] *= -1

        new_syndromes['is_ground_state'] = all(
            s == 1 for s in new_syndromes['plaquette_syndromes'].values()
        ) and all(
            s == 1 for s in new_syndromes['vertex_syndromes'].values()
        )

        return new_syndromes

    def logical_operators(self) -> dict:
        """
        Define the logical X and Z operators.

        On a torus, there are two independent non-contractible loops.
        """
        L = self.L

        # Logical Z_1: Z on all horizontal edges in a row
        Z_1 = [2 * (0 * L + y) for y in range(L)]

        # Logical Z_2: Z on all vertical edges in a column
        Z_2 = [2 * (x * L + 0) + 1 for x in range(L)]

        # Logical X_1: X on all vertical edges in a row
        X_1 = [2 * (0 * L + y) + 1 for y in range(L)]

        # Logical X_2: X on all horizontal edges in a column
        X_2 = [2 * (x * L + 0) for x in range(L)]

        return {
            'Z_1': Z_1,
            'Z_2': Z_2,
            'X_1': X_1,
            'X_2': X_2,
            'code_distance': L
        }

# Demonstrate toric code
tc = ToricCode(L=3)

print("Toric Code Properties")
print("=" * 50)
print(f"Lattice size: {tc.L} x {tc.L}")
print(f"Number of qubits: {tc.n_qubits}")
print(f"Encoded qubits: 2 (on torus)")

logical_ops = tc.logical_operators()
print(f"\nCode distance: {logical_ops['code_distance']}")
print(f"Logical Z_1 acts on qubits: {logical_ops['Z_1']}")
print(f"Logical X_1 acts on qubits: {logical_ops['X_1']}")

# Show error creates anyons
syndromes = tc.create_ground_state_syndrome()
print(f"\nGround state: all syndromes +1")

syndromes = tc.apply_x_error(0, syndromes)
violations = sum(1 for s in syndromes['plaquette_syndromes'].values() if s == -1)
print(f"After X error on qubit 0: {violations} plaquette violations (m anyons)")
```

## 4. Surface Codes

<Definition term="Surface Code">
The surface code is a variant of the toric code defined on a planar surface with boundaries. It encodes a single logical qubit with distance d on a d x d lattice, and is the leading candidate for fault-tolerant quantum computing due to its high threshold (~1%).
</Definition>

```python
class SurfaceCode:
    """Implement the surface code with boundaries."""

    def __init__(self, d: int):
        """
        Initialize d x d surface code.

        Args:
            d: Code distance
        """
        self.d = d
        # Qubits arranged in a d x d grid for data qubits
        # Plus (d-1) x (d-1) syndrome qubits
        self.n_data = d * d

    def threshold_estimate(self, physical_error_rate: float) -> dict:
        """
        Estimate logical error rate using threshold formula.

        Below threshold: logical error decreases exponentially with d
        Above threshold: logical error increases with d
        """
        threshold = 0.01  # ~1% for surface code

        if physical_error_rate < threshold:
            # Below threshold: exponential suppression
            ratio = physical_error_rate / threshold
            logical_error = physical_error_rate * (ratio ** ((self.d - 1) / 2))
        else:
            # Above threshold: errors accumulate
            logical_error = min(0.5, physical_error_rate * self.d)

        return {
            'physical_error_rate': physical_error_rate,
            'logical_error_rate': logical_error,
            'threshold': threshold,
            'below_threshold': physical_error_rate < threshold,
            'suppression_factor': physical_error_rate / logical_error if logical_error > 0 else np.inf
        }

# Demonstrate surface code scaling
print("\nSurface Code Error Suppression")
print("=" * 50)

for d in [3, 5, 9, 13]:
    sc = SurfaceCode(d)
    result = sc.threshold_estimate(physical_error_rate=0.001)
    print(f"d={d:2d}: p_phys=0.1%, p_logical~{result['logical_error_rate']:.2e}, "
          f"suppression={result['suppression_factor']:.1f}x")
```

## 5. Topological Quantum Computation

<Definition term="Topological Quantum Computation">
Topological quantum computation uses the braiding of non-Abelian anyons to implement quantum gates. Because braiding is a topological operation (depending only on the topology of world-lines), these gates are inherently protected against local perturbations.
</Definition>

### Universal Gate Sets from Braiding

<Formula>
$$\sigma_i \sigma_{i+1} \sigma_i = \sigma_{i+1} \sigma_i \sigma_{i+1} \quad \text{(Yang-Baxter equation)}$$
</Formula>

```python
class TopologicalQC:
    """Topological quantum computation with Fibonacci anyons."""

    def __init__(self):
        phi = (1 + np.sqrt(5)) / 2
        self.phi = phi

        # F-matrix
        self.F = np.array([
            [1/phi, 1/np.sqrt(phi)],
            [1/np.sqrt(phi), -1/phi]
        ], dtype=complex)

        # R-matrix
        self.R = np.array([
            [np.exp(4j * np.pi / 5), 0],
            [0, np.exp(-3j * np.pi / 5)]
        ], dtype=complex)

    def braid_sigma(self, i: int, n_anyons: int = 3) -> np.ndarray:
        """
        Compute braiding matrix sigma_i for n anyons.

        For n=3 (encoding 1 qubit): sigma_1 and sigma_2 generate gates.
        """
        if n_anyons == 3:
            if i == 1:
                return self.R
            elif i == 2:
                return self.F @ self.R @ np.linalg.inv(self.F)
        raise NotImplementedError("Only 3-anyon case implemented")

    def approximate_gate(self, target: np.ndarray, max_length: int = 10) -> dict:
        """
        Approximate a target gate using braiding sequences.

        Uses brute-force search over short sequences.

        Args:
            target: 2x2 target unitary
            max_length: Maximum braid word length

        Returns:
            Best approximation found
        """
        sigma1 = self.braid_sigma(1)
        sigma2 = self.braid_sigma(2)
        sigma1_inv = np.linalg.inv(sigma1)
        sigma2_inv = np.linalg.inv(sigma2)

        generators = {
            's1': sigma1,
            's2': sigma2,
            's1_inv': sigma1_inv,
            's2_inv': sigma2_inv
        }

        best_fidelity = 0
        best_sequence = []
        best_unitary = np.eye(2, dtype=complex)

        def gate_fidelity(U, V):
            """Average gate fidelity."""
            d = U.shape[0]
            inner = np.abs(np.trace(U.conj().T @ V)) ** 2
            return (inner + d) / (d * (d + 1))

        from itertools import product

        for length in range(1, max_length + 1):
            for seq in product(generators.keys(), repeat=length):
                U = np.eye(2, dtype=complex)
                for g in seq:
                    U = generators[g] @ U

                fidelity = gate_fidelity(U, target)
                if fidelity > best_fidelity:
                    best_fidelity = fidelity
                    best_sequence = list(seq)
                    best_unitary = U

            if best_fidelity > 0.999:
                break

        return {
            'target': target,
            'achieved': best_unitary,
            'fidelity': best_fidelity,
            'sequence': best_sequence,
            'length': len(best_sequence)
        }

# Demonstrate topological quantum computation
tqc = TopologicalQC()

print("\nTopological Quantum Computation with Fibonacci Anyons")
print("=" * 50)

# Approximate some standard gates
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)  # Hadamard
T = np.array([[1, 0], [0, np.exp(1j * np.pi / 4)]])  # T gate

print("\nApproximating Hadamard gate:")
result_H = tqc.approximate_gate(H, max_length=8)
print(f"  Fidelity: {result_H['fidelity']:.6f}")
print(f"  Braid sequence: {' -> '.join(result_H['sequence'][:5])}...")

print("\nApproximating T gate:")
result_T = tqc.approximate_gate(T, max_length=8)
print(f"  Fidelity: {result_T['fidelity']:.6f}")
print(f"  Braid sequence: {' -> '.join(result_T['sequence'][:5])}...")

print("\nNote: Fibonacci anyons provide universal quantum computation!")
print("Any gate can be approximated to arbitrary precision by longer braids.")
```

## 6. Topological Protection

<Definition term="Topological Protection">
Topological protection refers to the stability of encoded quantum information against local perturbations. In topological codes, errors must form closed loops (strings connecting the same type of anyon) to be logical errors, providing exponential suppression of errors with system size.
</Definition>

### Error Correction Threshold

<Formula>
$$p_{logical} \sim \left(\frac{p_{physical}}{p_{threshold}}\right)^{(d+1)/2}$$
</Formula>

```python
def topological_protection_analysis():
    """Analyze the protection offered by topological codes."""

    print("\nTopological Protection Analysis")
    print("=" * 50)

    # Threshold estimates for different codes
    codes = {
        'Toric code (perfect)': 0.11,
        'Surface code (MWPM)': 0.01,
        'Color code': 0.008,
        'Fibonacci anyons': 0.14  # Theoretical
    }

    print("\nError thresholds:")
    for code, threshold in codes.items():
        print(f"  {code}: {threshold*100:.1f}%")

    print("\nLogical error suppression (p_physical = 0.1%):")
    p_phys = 0.001
    for d in [3, 5, 7, 9, 11]:
        p_threshold = 0.01
        p_logical = p_phys * (p_phys / p_threshold) ** ((d - 1) / 2)
        print(f"  d={d:2d}: p_logical ~ {p_logical:.2e}")

    print("\nPhysical qubits required:")
    for d in [3, 5, 7, 9, 11]:
        n_physical = 2 * d * d - 1  # Surface code
        print(f"  d={d:2d}: {n_physical} physical qubits per logical qubit")

topological_protection_analysis()
```

<Warning>
While topological protection is powerful, it is not absolute. Errors that form non-trivial loops around the torus (or connect boundaries in the surface code) are logical errors. The protection comes from the low probability of such configurations occurring by chance.
</Warning>
