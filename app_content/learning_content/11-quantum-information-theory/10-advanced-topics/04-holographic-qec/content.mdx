## 1. The Holographic Principle

One of the most profound discoveries in theoretical physics is that quantum gravity in certain spacetimes can be completely described by a quantum field theory on the boundary.

<Definition term="Holographic Principle">
The holographic principle states that all information contained within a region of space can be encoded on its boundary. The maximum entropy scales with area, not volume: S_max ~ A/(4*l_P^2), where l_P is the Planck length.
</Definition>

<Definition term="AdS/CFT Correspondence">
The Anti-de Sitter/Conformal Field Theory correspondence states that quantum gravity in (d+1)-dimensional Anti-de Sitter (AdS) spacetime is exactly dual to a d-dimensional conformal field theory (CFT) living on its boundary.
</Definition>

<Formula>
$$\text{Bulk gravity in AdS}_{d+1} \Leftrightarrow \text{CFT}_d \text{ on boundary}$$
</Formula>

## 2. Holography as Quantum Error Correction

A remarkable insight is that the bulk-boundary mapping in AdS/CFT has the structure of a quantum error-correcting code.

<Definition term="Holographic Code">
A holographic code is a quantum error-correcting code where:
- Bulk (logical) information lives in lower-dimensional "interior" space
- Boundary (physical) qubits live on the boundary
- Bulk operators can be reconstructed from boundary subregions
- The code is protected against erasure of boundary regions
</Definition>

### Bulk Reconstruction

<Formula>
$$\phi_{bulk}(x) = \int_{boundary} K(x, y) \mathcal{O}(y) dy$$
</Formula>

where K is the bulk-boundary propagator (smearing function).

## 3. The Ryu-Takayanagi Formula

<Definition term="Ryu-Takayanagi Formula">
The entanglement entropy of a boundary region A in a holographic CFT equals the area of the minimal surface gamma_A in the bulk that is homologous to A:
$$S(A) = \frac{\text{Area}(\gamma_A)}{4 G_N}$$
where G_N is Newton's constant.
</Definition>

This formula reveals that entanglement entropy has a geometric interpretation in the bulk.

<Formula>
$$S(A) = \min_{\gamma_A \sim A} \frac{\text{Area}(\gamma_A)}{4 G_N}$$
</Formula>

```python
import numpy as np

class RyuTakayanagi:
    """Calculations related to the Ryu-Takayanagi formula."""

    def __init__(self, ads_radius: float = 1.0, g_newton: float = 1.0):
        """
        Initialize with AdS parameters.

        Args:
            ads_radius: AdS radius L
            g_newton: Newton's constant in AdS units
        """
        self.L = ads_radius
        self.G_N = g_newton

    def minimal_surface_area_interval(self, theta: float) -> float:
        """
        Calculate area of minimal surface for boundary interval.

        For an interval of angular size theta on the boundary of AdS_3,
        the minimal geodesic has length:
        Length = 2*L * log(sin(theta/2) / epsilon)

        where epsilon is the UV cutoff.

        Args:
            theta: Angular size of boundary interval (radians)

        Returns:
            Regularized length of geodesic
        """
        if theta <= 0 or theta >= 2*np.pi:
            return np.inf

        # Regularized length (dropping UV divergence)
        return 2 * self.L * np.log(np.sin(theta/2) + 1e-10)

    def entanglement_entropy_interval(self, theta: float) -> float:
        """
        Calculate holographic entanglement entropy for boundary interval.

        S = Area / (4 G_N)
        """
        area = self.minimal_surface_area_interval(theta)
        return area / (4 * self.G_N)

    def mutual_information(self, theta_A: float, theta_B: float,
                           separation: float) -> float:
        """
        Calculate holographic mutual information I(A:B).

        I(A:B) = S(A) + S(B) - S(A union B)

        Args:
            theta_A, theta_B: Sizes of regions A and B
            separation: Angular separation between A and B

        Returns:
            Mutual information (can be zero due to phase transition)
        """
        S_A = self.entanglement_entropy_interval(theta_A)
        S_B = self.entanglement_entropy_interval(theta_B)

        # S(A union B) depends on whether the minimal surface is connected
        theta_AB = theta_A + theta_B + 2 * separation

        # Check for "phase transition" in minimal surface
        # Connected: single geodesic spanning A union B
        # Disconnected: two separate geodesics

        S_AB_connected = self.entanglement_entropy_interval(min(theta_AB, 2*np.pi - 1e-6))
        S_AB_disconnected = S_A + S_B

        S_AB = min(S_AB_connected, S_AB_disconnected)

        I_AB = S_A + S_B - S_AB
        return max(0, I_AB)  # Mutual information is non-negative

    def strong_subadditivity_check(self, theta_A: float, theta_B: float,
                                    theta_C: float) -> dict:
        """
        Verify strong subadditivity: S(ABC) + S(B) <= S(AB) + S(BC).

        In holography, this is guaranteed by the geometry.
        """
        # This is simplified - actual calculation depends on geometry
        S_A = self.entanglement_entropy_interval(theta_A)
        S_B = self.entanglement_entropy_interval(theta_B)
        S_C = self.entanglement_entropy_interval(theta_C)

        # For adjacent intervals
        S_AB = self.entanglement_entropy_interval(theta_A + theta_B)
        S_BC = self.entanglement_entropy_interval(theta_B + theta_C)
        S_ABC = self.entanglement_entropy_interval(theta_A + theta_B + theta_C)

        lhs = S_ABC + S_B
        rhs = S_AB + S_BC

        return {
            'S_ABC': S_ABC,
            'S_B': S_B,
            'S_AB': S_AB,
            'S_BC': S_BC,
            'LHS': lhs,
            'RHS': rhs,
            'SSA_satisfied': lhs <= rhs + 1e-10
        }

# Demonstrate RT formula
rt = RyuTakayanagi(ads_radius=1.0, g_newton=1.0)

print("Ryu-Takayanagi Formula Demonstration")
print("=" * 50)

print("\nEntanglement entropy vs interval size:")
for theta_deg in [30, 60, 90, 120, 150, 180]:
    theta = theta_deg * np.pi / 180
    S = rt.entanglement_entropy_interval(theta)
    print(f"  theta = {theta_deg} deg: S = {S:.4f}")

print("\nMutual information between two intervals:")

print(f"  I(A:B) = {result:.4f}")

print("\nStrong subadditivity check:")

print(f"  SSA satisfied: {ssa['SSA_satisfied']}")
```

## 4. Tensor Network Models of Holography

Tensor networks provide concrete realizations of holographic codes.

<Definition term="MERA">
The Multi-scale Entanglement Renormalization Ansatz (MERA) is a tensor network with a hierarchical structure that produces area-law entanglement and has properties similar to AdS/CFT, including a "holographic" direction.
</Definition>

<Definition term="HaPPY Code">
The HaPPY (Harlow-Preskill-Pastawski-Yoshida) code is a holographic quantum error-correcting code built from perfect tensors arranged in a hyperbolic tiling, explicitly realizing many features of AdS/CFT.
</Definition>

```python
import numpy as np
from scipy.linalg import qr

class HolographicTensorNetwork:
    """Build and analyze holographic tensor networks."""

    @staticmethod
    def create_perfect_tensor(legs: int = 6) -> np.ndarray:
        """
        Create a perfect tensor (maximally entangling).

        A perfect tensor T with n legs (each of dimension d) satisfies:
        Contracting any n/2 legs gives a unitary matrix.

        For simplicity, we create an approximate perfect tensor.
        """
        # For 6 legs with dimension 2 each: 2^6 = 64 dim tensor
        d = 2
        total_dim = d ** legs

        # Create a random isometry and reshape
        # This is simplified - true perfect tensors have special structure
        half_legs = legs // 2
        dim_left = d ** half_legs
        dim_right = d ** (legs - half_legs)

        # Random unitary as approximation
        random_matrix = np.random.randn(dim_left, dim_right) + \
                        1j * np.random.randn(dim_left, dim_right)
        U, _ = qr(random_matrix)

        # Pad if needed
        if dim_left < dim_right:
            U_full = np.zeros((dim_right, dim_right), dtype=complex)
            U_full[:dim_left, :] = U
            # Complete to unitary
            U_full[dim_left:, :] = np.random.randn(dim_right - dim_left, dim_right)
            U_full, _ = qr(U_full)
            U = U_full[:dim_left, :]

        return U.reshape([d] * legs)

    @staticmethod
    def happy_code_structure(layers: int = 2):
        """
        Describe the structure of the HaPPY code.

        The HaPPY code tiles hyperbolic space with perfect tensors.
        """
        structure = {
            'geometry': 'Hyperbolic {5,4} tiling',
            'bulk_sites': 5 ** layers,  # Approximate
            'boundary_sites': 5 * 4 ** (layers - 1),  # Approximate
            'tensor_type': '6-leg perfect tensor',
            'properties': [
                'Bulk operators reconstructable from boundary subregions',
                'Area-law entanglement on boundary',
                'Error correction against boundary erasures',
                'Encodes holographic structure of AdS/CFT'
            ]
        }
        return structure

    def mera_layer(self, num_sites: int) -> dict:
        """
        Create one layer of MERA (disentanglers + isometries).

        MERA structure:
        1. Disentanglers: Remove short-range entanglement
        2. Isometries: Coarse-grain (reduce number of sites)
        """
        d = 2  # Qubit dimension

        # Disentanglers (unitary on pairs)
        num_disentanglers = num_sites // 2
        disentanglers = []
        for _ in range(num_disentanglers):
            U = np.random.randn(d**2, d**2) + 1j * np.random.randn(d**2, d**2)
            U, _ = qr(U)
            disentanglers.append(U.reshape(d, d, d, d))

        # Isometries (map 2 sites to 1)
        num_isometries = num_sites // 2
        isometries = []
        for _ in range(num_isometries):
            V = np.random.randn(d, d**2) + 1j * np.random.randn(d, d**2)
            V = V / np.linalg.norm(V, axis=1, keepdims=True)
            isometries.append(V.reshape(d, d, d))

        return {
            'disentanglers': disentanglers,
            'isometries': isometries,
            'input_sites': num_sites,
            'output_sites': num_sites // 2
        }

# Demonstrate holographic tensor networks
htn = HolographicTensorNetwork()

print("Holographic Tensor Networks")
print("=" * 50)

# Perfect tensor
print("\nPerfect Tensor Properties:")

print(f"  Shape: {T.shape}")

# HaPPY code

        print(f"  {key}:")
        for item in value:
            print(f"    - {item}")
    else:
        print(f"  {key}: {value}")

# MERA
```

## 5. Entanglement Wedge Reconstruction

<Definition term="Entanglement Wedge">
The entanglement wedge E(A) of a boundary region A is the bulk domain of dependence of any bulk surface bounded by A and its RT surface gamma_A. Operators in E(A) can be reconstructed from boundary region A alone.
</Definition>

<Formula>
$$\text{Bulk operator } \phi(x) \in E(A) \Rightarrow \phi(x) = \sum_i c_i O_i^{(A)}$$
</Formula>

```python
class EntanglementWedgeReconstruction:
    """Analyze entanglement wedge reconstruction in holographic codes."""

    def __init__(self, num_boundary_sites: int = 10):
        self.n = num_boundary_sites

    def is_in_entanglement_wedge(self, bulk_point: tuple,
                                  boundary_region: set) -> bool:
        """
        Check if a bulk point is in the entanglement wedge of a boundary region.

        Simplified model: bulk point (r, theta) in polar coordinates
        Boundary region: set of angular positions

        Args:
            bulk_point: (r, theta) where r=0 is center, r=1 is boundary
            boundary_region: set of boundary site indices

        Returns:
            True if bulk point is in entanglement wedge
        """
        r, theta = bulk_point

        if len(boundary_region) == 0:
            return False
        if len(boundary_region) == self.n:
            return True

        # Convert boundary region to angular interval
        angles = sorted([2 * np.pi * i / self.n for i in boundary_region])

        # Find the angular extent of the region
        # Check if bulk point's angle is within and radius is appropriate
        # Simplified: entanglement wedge extends to depth ~ log(boundary_size)

        angular_size = len(boundary_region) / self.n * 2 * np.pi
        max_depth = 1 - np.exp(-angular_size)  # Simplified relation

        return r < max_depth

    def complementary_recovery(self, bulk_point: tuple) -> dict:
        """
        Determine which boundary regions can reconstruct a bulk operator.

        A key feature of holographic codes: the same bulk operator
        can be reconstructed from complementary boundary regions.
        """
        r, theta = bulk_point

        # Find minimal boundary region containing bulk point in its wedge
        min_region_size = 1
        for size in range(1, self.n + 1):
            for start in range(self.n):
                region = set(range(start, start + size)) | \
                         set(range(0, max(0, start + size - self.n)))
                region = set(i % self.n for i in region)

                if self.is_in_entanglement_wedge(bulk_point, region):
                    min_region_size = size
                    break
            else:
                continue
            break

        # The complement also works if it's large enough
        complement_works = (self.n - min_region_size) >= min_region_size

        return {
            'bulk_point': bulk_point,
            'min_region_size': min_region_size,
            'min_region_fraction': min_region_size / self.n,
            'complement_recovers': complement_works,
            'interpretation': 'Bulk operator is error-correctable against erasure '
                            f'of up to {self.n - min_region_size} boundary sites'
        }

    def code_properties(self) -> dict:
        """
        Analyze the error-correcting properties of the holographic code.
        """
        # Test reconstruction for bulk points at various depths
        test_points = [
            (0.1, 0),    # Near center
            (0.5, 0),    # Mid-depth
            (0.9, 0),    # Near boundary
        ]

        results = []
        for point in test_points:
            result = self.complementary_recovery(point)
            results.append(result)

        return {
            'boundary_sites': self.n,
            'bulk_points_analyzed': test_points,
            'reconstruction_results': results,
            'summary': 'Deeper bulk points require larger boundary regions, '
                      'providing stronger error protection'
        }

# Demonstrate entanglement wedge reconstruction
ewr = EntanglementWedgeReconstruction(num_boundary_sites=12)

print("Entanglement Wedge Reconstruction")
print("=" * 50)

props = ewr.code_properties()
print(f"Boundary sites: {props['boundary_sites']}")
print(f"\nBulk reconstruction analysis:")

for result in props['reconstruction_results']:
    print(f"\n  Bulk point r={result['bulk_point'][0]}:")
    print(f"    Min boundary region: {result['min_region_size']} sites "
          f"({result['min_region_fraction']*100:.0f}%)")
    print(f"    Complement also recovers: {result['complement_recovers']}")
```

## 6. Spacetime from Entanglement

<Definition term="ER=EPR">
The ER=EPR conjecture (Maldacena-Susskind) proposes that entangled particles are connected by Einstein-Rosen bridges (wormholes). Entanglement IS spacetime connectivity.
</Definition>

<Formula>
$$\text{Entanglement} \Leftrightarrow \text{Spacetime connectivity}$$
</Formula>

The emergence of spacetime from entanglement is captured by:

```python

                                     I_AB: float) -> float:
        """
        Estimate geometric distance from mutual information.

        In holography, mutual information is related to the geometric
        distance between bulk points reconstructable from A and B.

        I(A:B) > 0 implies the regions are "geometrically connected"
        I(A:B) = 0 implies disconnection (phase transition)

        if ratio > 0:
            return -np.log(ratio)
        return np.inf
```

## 7. Quantum Error Correction and Black Holes

<Definition term="Black Hole Interior Reconstruction">
The black hole information paradox is connected to holographic error correction: the interior of a black hole can be reconstructed from the boundary, but only from specific "non-trivial" boundary subregions that include early Hawking radiation.
</Definition>

```python
class BlackHoleQEC:
    """Connection between black holes and quantum error correction."""

    @staticmethod
    def hayden_preskill_protocol():
        """
        Describe the Hayden-Preskill thought experiment.

        After the Page time, information thrown into a black hole
        can be recovered from a few Hawking quanta.
        """
        protocol = {
            'setup': 'Alice throws a diary into an old black hole',
            'question': 'How quickly can Bob recover the diary information?',
            'answer': 'After scrambling time t_* ~ log(S) / T, Bob needs only '
                     'O(1) additional Hawking quanta',
            'connection_to_qec': [
                'Black hole acts as a scrambling quantum channel',
                'Diary is "encoded" into Hawking radiation',
                'Recovery is a decoding operation',
                'Fast scrambling enables efficient decoding'
            ]
        }
        return protocol

    @staticmethod
    def page_curve_from_qec():
        """
        Explain the Page curve using QEC concepts.
        """
        def page_curve(t, t_page, S_BH_max):
            """Model Page curve for entanglement entropy."""
            if t < t_page:
                # Before Page time: entropy increases
                return S_BH_max * (1 - np.exp(-t / t_page))
            else:
                # After Page time: entropy decreases
                return S_BH_max * np.exp(-(t - t_page) / t_page)

        t_page = 1.0
        S_max = 1.0
        times = np.linspace(0, 3, 100)
        entropies = [page_curve(t, t_page, S_max) for t in times]

        return {
            'times': times,
            'entropies': np.array(entropies),
            'page_time': t_page,
            'interpretation': 'Before Page time, radiation entanglement grows. '
                            'After Page time, information starts coming out.'
        }
```
