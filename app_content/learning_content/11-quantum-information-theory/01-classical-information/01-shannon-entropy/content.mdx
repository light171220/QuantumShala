## 1. The Birth of Information Theory

In 1948, Claude Shannon published "A Mathematical Theory of Communication," fundamentally changing how we understand information. Before Shannon, information was an intuitive concept. After Shannon, it became a precise mathematical quantity.

<Definition term="Information">
Information is the resolution of uncertainty. When an uncertain event occurs, we gain information. The amount of information gained depends on how surprising the outcome is.
</Definition>

The key insight is that **rare events carry more information than common events**. Learning that the sun rose today (probability ≈ 1) tells us almost nothing. Learning that you won the lottery (probability ≈ 10⁻⁸) is highly informative.

## 2. Defining Shannon Entropy

Consider a discrete random variable $X$ with possible outcomes $\{x_1, x_2, \ldots, x_n\}$ and probability distribution $p(x_i) = p_i$.

<Definition term="Shannon Entropy">
The Shannon entropy of a discrete random variable $X$ is defined as:

$$H(X) = -\sum_{i=1}^{n} p_i \log_2 p_i$$

with the convention that $0 \log 0 = 0$.
</Definition>

The entropy measures the **average uncertainty** or **expected information content** of the random variable.

<Formula>
$$H(X) = -\mathbb{E}[\log_2 p(X)] = \sum_{x} p(x) \cdot \underbrace{(-\log_2 p(x))}_{\text{self-information}}$$
</Formula>

### The Self-Information

The quantity $I(x) = -\log_2 p(x)$ is called the **self-information** or **surprisal** of outcome $x$.

<Tip>
Self-information $I(x) = -\log_2 p(x)$ tells us how many bits of information we gain when observing outcome $x$. Rare events (small $p(x)$) give more bits.
</Tip>

```python
import numpy as np
import matplotlib.pyplot as plt

def self_information(p):
    """Calculate self-information in bits."""
    if p == 0:
        return np.inf
    return -np.log2(p)

def shannon_entropy(probabilities):
    """Calculate Shannon entropy of a probability distribution."""
    probs = np.array(probabilities)
    # Filter out zero probabilities
    probs = probs[probs > 0]
    return -np.sum(probs * np.log2(probs))

# Example: Fair coin
coin = [0.5, 0.5]
print(f"Fair coin entropy: {shannon_entropy(coin):.4f} bits")

# Example: Biased coin (90% heads)
biased_coin = [0.9, 0.1]
print(f"Biased coin entropy: {shannon_entropy(biased_coin):.4f} bits")

# Example: Certain outcome
certain = [1.0, 0.0]
print(f"Certain outcome entropy: {shannon_entropy(certain):.4f} bits")
```

Output:
```
Fair coin entropy: 1.0000 bits
Biased coin entropy: 0.4690 bits
Certain outcome entropy: 0.0000 bits
```

## 3. Operational Interpretation

Shannon's profound insight was connecting entropy to **data compression**:

<Definition term="Shannon's Source Coding Theorem">
A source with entropy $H(X)$ bits per symbol cannot be compressed to fewer than $H(X)$ bits per symbol on average, and can be compressed arbitrarily close to $H(X)$ bits per symbol.
</Definition>

This means entropy tells us the **fundamental limit of lossless compression**.

<Example>
Consider English text. If we assumed all 26 letters were equally likely, we'd need $\log_2(26) \approx 4.7$ bits per letter. But English has structure:
- 'e' appears ~12.7% of the time
- 'z' appears ~0.07% of the time

The actual entropy of English is about 1.0-1.5 bits per character, explaining why text compresses so well!
</Example>

### Why Logarithms?

The logarithm is the unique function satisfying:
1. **Additivity**: Information from independent events adds: $I(x, y) = I(x) + I(y)$ when $X, Y$ are independent
2. **Continuity**: Small changes in probability yield small changes in information
3. **Normalization**: A fair coin flip gives exactly 1 bit

<Formula>
$$\log_2\left(\frac{1}{p(x) \cdot p(y)}\right) = \log_2\left(\frac{1}{p(x)}\right) + \log_2\left(\frac{1}{p(y)}\right)$$
</Formula>

## 4. Properties of Shannon Entropy

### Property 1: Non-negativity

<Formula>
$$H(X) \geq 0$$
</Formula>

**Proof**: Since $0 \leq p_i \leq 1$, we have $\log_2 p_i \leq 0$, so $-p_i \log_2 p_i \geq 0$ for all $i$.

### Property 2: Maximum Entropy

<Formula>
$$H(X) \leq \log_2 n$$
</Formula>

with equality if and only if $X$ is uniformly distributed: $p_i = 1/n$ for all $i$.

**Proof**: Use the log-sum inequality or Lagrange multipliers. The uniform distribution maximizes uncertainty.

```python
import numpy as np

def max_entropy(n):
    """Maximum entropy for n outcomes."""
    return np.log2(n)

# Verify with uniform distribution
n = 8
uniform = [1/n] * n
print(f"Maximum entropy for {n} outcomes: {max_entropy(n):.4f} bits")
print(f"Uniform distribution entropy: {shannon_entropy(uniform):.4f} bits")

# Any other distribution has less entropy
non_uniform = [0.5, 0.2, 0.1, 0.1, 0.05, 0.03, 0.01, 0.01]
print(f"Non-uniform distribution entropy: {shannon_entropy(non_uniform):.4f} bits")
```

### Property 3: Zero Entropy Condition

<Formula>
$$H(X) = 0 \iff p_i = 1 \text{ for some } i$$
</Formula>

Zero entropy means no uncertainty - we know the outcome with certainty.

### Property 4: Concavity

<Formula>
$$H(\lambda p + (1-\lambda)q) \geq \lambda H(p) + (1-\lambda)H(q)$$
</Formula>

for probability distributions $p, q$ and $0 \leq \lambda \leq 1$.

**Interpretation**: Mixing distributions never decreases average entropy.

### Property 5: Chain Rule

For joint random variables $(X, Y)$:

<Formula>
$$H(X, Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)$$
</Formula>

where $H(Y|X)$ is the conditional entropy.

## 5. Binary Entropy Function

A special case of fundamental importance is the **binary entropy function**:

<Definition term="Binary Entropy">
For a binary random variable with $P(X=1) = p$ and $P(X=0) = 1-p$:

$$h(p) = -p\log_2 p - (1-p)\log_2(1-p)$$
</Definition>

```python
import numpy as np
import matplotlib.pyplot as plt

def binary_entropy(p):
    """Binary entropy function."""
    if p == 0 or p == 1:
        return 0
    return -p * np.log2(p) - (1-p) * np.log2(1-p)

# Plot binary entropy
p_values = np.linspace(0.001, 0.999, 1000)
h_values = [binary_entropy(p) for p in p_values]

plt.figure(figsize=(10, 6))
plt.plot(p_values, h_values, 'b-', linewidth=2)
plt.xlabel('Probability p', fontsize=12)
plt.ylabel('Binary Entropy h(p)', fontsize=12)
plt.title('Binary Entropy Function', fontsize=14)
plt.grid(True, alpha=0.3)
plt.axhline(y=1, color='r', linestyle='--', label='Maximum = 1 bit')
plt.axvline(x=0.5, color='g', linestyle='--', label='p = 0.5')
plt.legend()
plt.show()
```

<Tip>
The binary entropy function appears throughout information theory and quantum information. It's symmetric about $p = 0.5$ and achieves its maximum of 1 bit at $p = 0.5$.
</Tip>

## 6. Joint and Conditional Entropy

### Joint Entropy

For two random variables $X$ and $Y$ with joint distribution $p(x, y)$:

<Formula>
$$H(X, Y) = -\sum_{x,y} p(x, y) \log_2 p(x, y)$$
</Formula>

### Conditional Entropy

<Definition term="Conditional Entropy">
The conditional entropy of $Y$ given $X$ is:

$$H(Y|X) = \sum_x p(x) H(Y|X=x) = -\sum_{x,y} p(x, y) \log_2 p(y|x)$$
</Definition>

**Interpretation**: $H(Y|X)$ is the average uncertainty remaining in $Y$ after learning $X$.

```python
import numpy as np

def joint_entropy(joint_probs):
    """Calculate joint entropy H(X,Y) from joint probability matrix."""
    flat = joint_probs.flatten()
    flat = flat[flat > 0]
    return -np.sum(flat * np.log2(flat))

def conditional_entropy(joint_probs):
    """Calculate H(Y|X) from joint probability matrix."""
    # Rows are X values, columns are Y values
    p_x = joint_probs.sum(axis=1)  # Marginal of X
    h_y_given_x = 0
    for i, p_xi in enumerate(p_x):
        if p_xi > 0:
            # Conditional distribution P(Y|X=xi)
            p_y_given_xi = joint_probs[i, :] / p_xi
            p_y_given_xi = p_y_given_xi[p_y_given_xi > 0]
            h_y_given_x += p_xi * (-np.sum(p_y_given_xi * np.log2(p_y_given_xi)))
    return h_y_given_x

# Example: Joint distribution
joint = np.array([
    [0.25, 0.25],  # X=0: Y=0 or Y=1
    [0.25, 0.25]   # X=1: Y=0 or Y=1
])

print(f"H(X,Y) = {joint_entropy(joint):.4f} bits")
print(f"H(Y|X) = {conditional_entropy(joint):.4f} bits")
```

### Key Inequalities

<Formula>
$$0 \leq H(Y|X) \leq H(Y)$$
$$H(X, Y) \leq H(X) + H(Y)$$
</Formula>

with equality in the second when $X$ and $Y$ are independent.

<Warning>
Conditioning reduces entropy on average: $H(Y|X) \leq H(Y)$. Learning information about $X$ can only decrease (or maintain) our uncertainty about $Y$, never increase it.
</Warning>

## 7. Entropy in Quantum Information

Shannon entropy is the classical precursor to quantum entropy measures. The connection is:

<Formula>
$$H(X) = -\sum_i p_i \log_2 p_i \quad \longrightarrow \quad S(\rho) = -\text{Tr}(\rho \log_2 \rho)$$
</Formula>

where $S(\rho)$ is the von Neumann entropy of a quantum state $\rho$.

For a quantum state diagonal in some basis with eigenvalues $\lambda_i$:

$$S(\rho) = H(\lambda_1, \lambda_2, \ldots) = -\sum_i \lambda_i \log_2 \lambda_i$$

<Tip>
The Shannon entropy of the measurement outcome distribution is always at least as large as the von Neumann entropy. This reflects how measurement disturbs quantum states.
</Tip>

## 8. Applications

### Data Compression
Shannon entropy sets the theoretical limit for lossless compression:
- Huffman coding achieves close to $H(X)$ bits per symbol
- Arithmetic coding can achieve $H(X)$ asymptotically

### Cryptography
Perfect secrecy requires key entropy at least equal to message entropy.

### Machine Learning
- Cross-entropy loss measures difference between predicted and true distributions
- Maximum entropy models make minimal assumptions

### Statistical Mechanics
Thermodynamic entropy and Shannon entropy are related by Boltzmann's constant:
$$S_{\text{thermo}} = k_B \cdot H \cdot \ln 2$$

## Summary

In this lesson, you learned the foundations of information theory:

1. **Shannon entropy** $H(X) = -\sum_i p_i \log_2 p_i$ quantifies average uncertainty
2. **Self-information** $I(x) = -\log_2 p(x)$ measures surprise from individual outcomes
3. **Maximum entropy** is achieved by uniform distributions
4. **Concavity** ensures mixing increases or maintains entropy
5. **Chain rule** connects joint, marginal, and conditional entropies
6. **Source coding theorem** links entropy to optimal compression

These concepts form the foundation for understanding quantum information theory, where Shannon entropy generalizes to von Neumann entropy.

<Tip>
Shannon entropy is to information theory what energy is to physics - a fundamental conserved quantity that constrains what is possible. Mastering entropy is essential for understanding both classical and quantum information.
</Tip>
