## 1. Motivation: Measuring Dependence

How do we quantify how much two random variables "know" about each other? Correlation coefficients capture linear relationships, but mutual information captures **all** statistical dependencies.

<Definition term="Mutual Information">
The mutual information between random variables $X$ and $Y$ is:

$$I(X;Y) = H(X) + H(Y) - H(X,Y)$$

Equivalently:
$$I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)$$
</Definition>

Mutual information measures the **reduction in uncertainty** about one variable given knowledge of the other.

<Tip>
Think of mutual information as the "overlap" in information content between X and Y. It's symmetric: knowing X tells you as much about Y as knowing Y tells you about X.
</Tip>

## 2. Equivalent Definitions

There are several equivalent ways to express mutual information:

<Formula>
$$I(X;Y) = \sum_{x,y} p(x,y) \log_2 \frac{p(x,y)}{p(x)p(y)}$$
</Formula>

This form explicitly measures how much the joint distribution differs from the product of marginals (independence).

```python
import numpy as np

def mutual_information(joint_prob):
    """
    Calculate mutual information from joint probability matrix.

    Args:
        joint_prob: 2D numpy array where joint_prob[i,j] = P(X=i, Y=j)

    Returns:
        Mutual information in bits
    """
    joint = np.array(joint_prob)

    # Marginal distributions
    p_x = joint.sum(axis=1)  # Sum over Y
    p_y = joint.sum(axis=0)  # Sum over X

    # Calculate mutual information
    mi = 0
    for i in range(joint.shape[0]):
        for j in range(joint.shape[1]):
            if joint[i,j] > 0 and p_x[i] > 0 and p_y[j] > 0:
                mi += joint[i,j] * np.log2(joint[i,j] / (p_x[i] * p_y[j]))

    return mi

# Example: Correlated variables
joint_correlated = np.array([
    [0.4, 0.1],
    [0.1, 0.4]
])

# Example: Independent variables
joint_independent = np.array([
    [0.25, 0.25],
    [0.25, 0.25]
])

print(f"Correlated: I(X;Y) = {mutual_information(joint_correlated):.4f} bits")
print(f"Independent: I(X;Y) = {mutual_information(joint_independent):.4f} bits")
```

Output:
```
Correlated: I(X;Y) = 0.2780 bits
Independent: I(X;Y) = 0.0000 bits
```

## 3. Venn Diagram Interpretation

Mutual information can be visualized using a Venn diagram of entropies:

```
    ┌────────────────────────────────────┐
    │              H(X,Y)                │
    │  ┌─────────────┬─────────────┐     │
    │  │             │             │     │
    │  │    H(X|Y)   │   I(X;Y)   │ H(Y|X)│
    │  │             │             │     │
    │  └─────────────┴─────────────┘     │
    │      H(X)           H(Y)           │
    └────────────────────────────────────┘
```

Key relationships:
- $H(X) = H(X|Y) + I(X;Y)$
- $H(Y) = H(Y|X) + I(X;Y)$
- $H(X,Y) = H(X) + H(Y) - I(X;Y)$

<Warning>
Mutual information is always non-negative: $I(X;Y) \geq 0$. Unlike correlation, negative mutual information is impossible - learning about one variable never increases uncertainty about another on average.
</Warning>

## 4. Properties of Mutual Information

### Property 1: Symmetry

<Formula>
$$I(X;Y) = I(Y;X)$$
</Formula>

The information X gives about Y equals the information Y gives about X.

### Property 2: Non-negativity

<Formula>
$$I(X;Y) \geq 0$$
</Formula>

with equality if and only if X and Y are independent.

**Proof**: $I(X;Y) = D_{KL}(p(x,y) \| p(x)p(y)) \geq 0$ by properties of KL divergence.

### Property 3: Upper Bounds

<Formula>
$$I(X;Y) \leq \min\{H(X), H(Y)\}$$
</Formula>

You cannot gain more information about X from Y than X contains.

### Property 4: Relationship to Entropies

<Formula>
$$I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X) + H(Y) - H(X,Y)$$
</Formula>

```python
import numpy as np

def verify_mi_properties(joint_prob):
    """Verify mutual information properties."""
    joint = np.array(joint_prob)

    # Calculate all entropies
    def entropy(p):
        p = p[p > 0]
        return -np.sum(p * np.log2(p))

    p_x = joint.sum(axis=1)
    p_y = joint.sum(axis=0)

    H_X = entropy(p_x)
    H_Y = entropy(p_y)
    H_XY = entropy(joint.flatten())

    # H(Y|X)
    H_Y_given_X = 0
    for i, p_xi in enumerate(p_x):
        if p_xi > 0:
            p_y_given_xi = joint[i, :] / p_xi
            p_y_given_xi = p_y_given_xi[p_y_given_xi > 0]
            H_Y_given_X += p_xi * (-np.sum(p_y_given_xi * np.log2(p_y_given_xi)))

    # H(X|Y)
    H_X_given_Y = 0
    for j, p_yj in enumerate(p_y):
        if p_yj > 0:
            p_x_given_yj = joint[:, j] / p_yj
            p_x_given_yj = p_x_given_yj[p_x_given_yj > 0]
            H_X_given_Y += p_yj * (-np.sum(p_x_given_yj * np.log2(p_x_given_yj)))

    # Three ways to compute I(X;Y)
    I_1 = H_X + H_Y - H_XY
    I_2 = H_X - H_X_given_Y
    I_3 = H_Y - H_Y_given_X

    print(f"H(X) = {H_X:.4f}, H(Y) = {H_Y:.4f}, H(X,Y) = {H_XY:.4f}")
    print(f"H(X|Y) = {H_X_given_Y:.4f}, H(Y|X) = {H_Y_given_X:.4f}")
    print(f"\nMutual Information (3 methods):")
    print(f"  I(X;Y) = H(X) + H(Y) - H(X,Y) = {I_1:.4f}")
    print(f"  I(X;Y) = H(X) - H(X|Y) = {I_2:.4f}")
    print(f"  I(X;Y) = H(Y) - H(Y|X) = {I_3:.4f}")
    print(f"\nProperties:")
    print(f"  Non-negative: {I_1 >= -1e-10}")
    print(f"  ≤ H(X): {I_1 <= H_X + 1e-10}")
    print(f"  ≤ H(Y): {I_1 <= H_Y + 1e-10}")

joint = np.array([[0.4, 0.1], [0.1, 0.4]])
verify_mi_properties(joint)
```

## 5. Data Processing Inequality

One of the most important properties of mutual information:

<Definition term="Data Processing Inequality">
For random variables forming a Markov chain $X \to Y \to Z$:

$$I(X;Z) \leq I(X;Y)$$

Processing data cannot create information about the source.
</Definition>

<Example>
If X is an original image, Y is a compressed version, and Z is a further processed version:

$$I(\text{Original}; \text{Final}) \leq I(\text{Original}; \text{Compressed})$$

Each processing step can only lose information, never gain it.
</Example>

```python
import numpy as np

def demonstrate_data_processing():
    """Demonstrate the data processing inequality."""
    # X -> Y -> Z Markov chain
    # X: Original data (4 states)
    # Y: First processing (2 states)
    # Z: Second processing (2 states)

    # P(Y|X) - lossy compression
    P_Y_given_X = np.array([
        [0.9, 0.1],  # X=0 -> mostly Y=0
        [0.8, 0.2],  # X=1 -> mostly Y=0
        [0.2, 0.8],  # X=2 -> mostly Y=1
        [0.1, 0.9]   # X=3 -> mostly Y=1
    ])

    # P(Z|Y) - further processing
    P_Z_given_Y = np.array([
        [0.95, 0.05],  # Y=0 -> mostly Z=0
        [0.1, 0.9]     # Y=1 -> mostly Z=1
    ])

    # P(X) - uniform source
    P_X = np.array([0.25, 0.25, 0.25, 0.25])

    # Calculate P(X,Y)
    P_XY = np.outer(P_X, np.ones(2)) * P_Y_given_X

    # Calculate P(Y)
    P_Y = P_XY.sum(axis=0)

    # Calculate P(Y,Z)
    P_YZ = np.outer(P_Y, np.ones(2)) * P_Z_given_Y

    # Calculate P(X,Z) via marginalization
    P_Z_given_X = P_Y_given_X @ P_Z_given_Y
    P_XZ = np.outer(P_X, np.ones(2)) * P_Z_given_X

    # Calculate mutual informations
    I_XY = mutual_information(P_XY)
    I_YZ = mutual_information(P_YZ)
    I_XZ = mutual_information(P_XZ)

    print("Data Processing Inequality: X -> Y -> Z")
    print(f"I(X;Y) = {I_XY:.4f} bits")
    print(f"I(Y;Z) = {I_YZ:.4f} bits")
    print(f"I(X;Z) = {I_XZ:.4f} bits")
    print(f"\nI(X;Z) ≤ I(X;Y)? {I_XZ <= I_XY + 1e-10}")
    print(f"I(X;Z) ≤ I(Y;Z)? {I_XZ <= I_YZ + 1e-10}")

demonstrate_data_processing()
```

## 6. KL Divergence Connection

Mutual information is a special case of Kullback-Leibler divergence:

<Definition term="Kullback-Leibler Divergence">
The KL divergence from distribution $q$ to $p$ is:

$$D_{KL}(p \| q) = \sum_x p(x) \log \frac{p(x)}{q(x)}$$
</Definition>

The connection:

<Formula>
$$I(X;Y) = D_{KL}(p(x,y) \| p(x)p(y))$$
</Formula>

This measures how much the joint distribution differs from what it would be under independence.

```python
import numpy as np

def kl_divergence(p, q):
    """Calculate KL divergence D_KL(p || q)."""
    p = np.array(p).flatten()
    q = np.array(q).flatten()

    kl = 0
    for pi, qi in zip(p, q):
        if pi > 0:
            if qi == 0:
                return np.inf
            kl += pi * np.log2(pi / qi)
    return kl

def mi_via_kl(joint_prob):
    """Calculate MI as KL divergence from independence."""
    joint = np.array(joint_prob)
    p_x = joint.sum(axis=1)
    p_y = joint.sum(axis=0)

    # Product of marginals (independence assumption)
    independent = np.outer(p_x, p_y)

    return kl_divergence(joint, independent)

joint = np.array([[0.4, 0.1], [0.1, 0.4]])
print(f"MI via direct formula: {mutual_information(joint):.4f} bits")
print(f"MI via KL divergence: {mi_via_kl(joint):.4f} bits")
```

## 7. Conditional Mutual Information

For three random variables, we define:

<Definition term="Conditional Mutual Information">
$$I(X;Y|Z) = H(X|Z) - H(X|Y,Z) = H(X|Z) + H(Y|Z) - H(X,Y|Z)$$
</Definition>

This measures the mutual information between X and Y given knowledge of Z.

### Chain Rule for Mutual Information

<Formula>
$$I(X;Y,Z) = I(X;Z) + I(X;Y|Z)$$
</Formula>

<Warning>
Unlike unconditional mutual information, $I(X;Y|Z)$ can be larger or smaller than $I(X;Y)$. Knowledge of Z can either reveal or obscure the relationship between X and Y.
</Warning>

## 8. Applications

### Channel Capacity

The capacity of a noisy channel is the maximum mutual information:

<Formula>
$$C = \max_{p(x)} I(X;Y)$$
</Formula>

where X is input, Y is output, and the channel is characterized by $p(y|x)$.

### Feature Selection

In machine learning, mutual information identifies relevant features:
- High $I(\text{Feature}; \text{Label})$ indicates useful features
- Redundant features have high mutual information with each other

### Cryptography

For perfect secrecy:
$$I(\text{Message}; \text{Ciphertext}) = 0$$

The ciphertext reveals nothing about the message.

### Information Bottleneck

A method to find compressed representations:
$$\min_{p(\tilde{x}|x)} I(X;\tilde{X}) - \beta I(\tilde{X};Y)$$

Balance compression ($I(X;\tilde{X})$) against relevance ($I(\tilde{X};Y)$).

## 9. Quantum Generalization

In quantum information theory, mutual information generalizes to:

<Formula>
$$I(A:B) = S(\rho_A) + S(\rho_B) - S(\rho_{AB})$$
</Formula>

where $S(\rho) = -\text{Tr}(\rho \log \rho)$ is the von Neumann entropy.

For quantum systems, this can exceed classical bounds due to entanglement.

<Tip>
Quantum mutual information can be as large as $2 \min\{S(\rho_A), S(\rho_B)\}$ for maximally entangled states - twice the classical maximum! This extra capacity enables superdense coding.
</Tip>

## Summary

In this lesson, you learned about mutual information:

1. **Definition**: $I(X;Y) = H(X) + H(Y) - H(X,Y)$ measures shared information
2. **Symmetry**: $I(X;Y) = I(Y;X)$ - information sharing is symmetric
3. **Non-negativity**: $I(X;Y) \geq 0$ with equality iff independent
4. **Data processing inequality**: Processing cannot create information
5. **KL divergence**: $I(X;Y) = D_{KL}(p(x,y) \| p(x)p(y))$
6. **Applications**: Channel capacity, feature selection, cryptography

Mutual information is the key quantity for understanding information transmission through noisy channels.
