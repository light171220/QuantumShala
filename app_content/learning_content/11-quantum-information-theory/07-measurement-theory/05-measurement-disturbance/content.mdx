## 1. The Quantum Measurement Problem

In classical physics, measurements can be arbitrarily gentle - we can extract information without fundamentally altering the system. Quantum mechanics is radically different.

<Definition term="Measurement Back-action">
Any quantum measurement that gains information about a system necessarily disturbs it. This is not a technological limitation but a fundamental feature of quantum mechanics, encoded in the measurement postulates.
</Definition>

<Warning>
The no-go theorem: There is no measurement that extracts information while leaving all quantum states unchanged. Information gain requires disturbance!
</Warning>

### Why Measurements Disturb

Consider measuring a qubit in the computational basis:

```python
import numpy as np

def projective_measurement(state: np.ndarray, basis: str = 'Z') -> tuple:
    """
    Simulate projective measurement with post-measurement state.

    Args:
        state: State vector |psi>
        basis: 'Z' for computational, 'X' for Hadamard basis

    Returns:
        (outcome, post_measurement_state, disturbance)
    """
    # Measurement operators
    if basis == 'Z':
        P0 = np.array([[1, 0], [0, 0]], dtype=complex)
        P1 = np.array([[0, 0], [0, 1]], dtype=complex)
    else:  # X basis
        plus = np.array([1, 1]) / np.sqrt(2)
        minus = np.array([1, -1]) / np.sqrt(2)
        P0 = np.outer(plus, plus.conj())
        P1 = np.outer(minus, minus.conj())

    # Probabilities
    p0 = np.real(state.conj() @ P0 @ state)
    p1 = np.real(state.conj() @ P1 @ state)

    # Simulate measurement
    outcome = 0 if np.random.random() < p0 else 1
    P_outcome = P0 if outcome == 0 else P1
    p_outcome = p0 if outcome == 0 else p1

    # Post-measurement state
    if p_outcome > 1e-10:
        post_state = P_outcome @ state / np.sqrt(p_outcome)
    else:
        post_state = state

    # Quantify disturbance (fidelity with original)
    fidelity = np.abs(state.conj() @ post_state)**2
    disturbance = 1 - fidelity

    return outcome, post_state, disturbance

# Example: measure |+> in Z basis
plus_state = np.array([1, 1], dtype=complex) / np.sqrt(2)

np.random.seed(42)
print("Measuring |+> in Z basis (10 trials):")
for i in range(10):
    outcome, post, dist = projective_measurement(plus_state.copy(), 'Z')
    print(f"  Trial {i+1}: outcome={outcome}, disturbance={dist:.4f}")
```

## 2. Quantifying Disturbance

Several measures quantify how much a measurement disturbs a quantum state.

<Definition term="State Disturbance">
For an initial state $\rho$ and measurement channel $\mathcal{M}$ with average output $\rho' = \mathcal{M}(\rho)$:

**Fidelity-based**: $D_F = 1 - F(\rho, \rho')$ where $F(\rho, \sigma) = \text{Tr}\sqrt{\sqrt{\rho}\sigma\sqrt{\rho}}$

**Trace distance**: $D_T = \frac{1}{2}\|\rho - \rho'\|_1$
</Definition>

<Formula>
**Average Disturbance:**

For measurement with Kraus operators $\{M_k\}$:

$$\bar{D} = 1 - \sum_k |\text{Tr}(M_k \rho)|^2 / p_k$$

where $p_k = \text{Tr}(M_k \rho M_k^\dagger)$
</Formula>

```python
import numpy as np
from scipy.linalg import sqrtm

def fidelity(rho: np.ndarray, sigma: np.ndarray) -> float:
    """Quantum state fidelity."""
    sqrt_rho = sqrtm(rho)
    inner = sqrtm(sqrt_rho @ sigma @ sqrt_rho)
    return np.real(np.trace(inner))**2

def trace_distance(rho: np.ndarray, sigma: np.ndarray) -> float:
    """Trace distance between states."""
    diff = rho - sigma
    return 0.5 * np.real(np.trace(sqrtm(diff.conj().T @ diff)))

def measurement_disturbance(rho: np.ndarray, kraus_ops: list) -> dict:
    """
    Quantify disturbance from a measurement.

    Args:
        rho: Initial density matrix
        kraus_ops: List of Kraus operators

    Returns:
        Dictionary with disturbance measures
    """
    # Average post-measurement state (unconditional)
    rho_avg = np.zeros_like(rho)
    for M in kraus_ops:
        rho_avg += M @ rho @ M.conj().T

    # Fidelity-based disturbance
    F = fidelity(rho, rho_avg)
    D_fidelity = 1 - F

    # Trace distance
    D_trace = trace_distance(rho, rho_avg)

    # State-specific disturbances for each outcome
    outcome_disturbances = []
    for M in kraus_ops:
        p = np.real(np.trace(M @ rho @ M.conj().T))
        if p > 1e-10:
            rho_post = M @ rho @ M.conj().T / p
            F_k = fidelity(rho, rho_post)
            outcome_disturbances.append((p, 1 - F_k))

    return {
        'fidelity': F,
        'disturbance_fidelity': D_fidelity,
        'disturbance_trace': D_trace,
        'outcome_disturbances': outcome_disturbances,
        'post_state': rho_avg
    }

# Example: Z measurement on various states
P0 = np.array([[1, 0], [0, 0]], dtype=complex)
P1 = np.array([[0, 0], [0, 1]], dtype=complex)
Z_kraus = [sqrtm(P0), sqrtm(P1)]

states = {
    '|0>': np.array([[1, 0], [0, 0]], dtype=complex),
    '|+>': np.array([[0.5, 0.5], [0.5, 0.5]], dtype=complex),
    '|+i>': np.array([[0.5, -0.5j], [0.5j, 0.5]], dtype=complex)
}

print("Z-measurement disturbance:")
for name, rho in states.items():
    result = measurement_disturbance(rho, Z_kraus)
    print(f"  {name}: D_fidelity={result['disturbance_fidelity']:.4f}, D_trace={result['disturbance_trace']:.4f}")
```

## 3. Information-Disturbance Trade-off

<Definition term="Information-Disturbance Trade-off">
There exists a fundamental relationship between the information gained from a measurement and the disturbance it causes:

$$I_{\text{gain}} + D_{\text{residual}} \leq \text{constant}$$

where the constant depends on the specific quantification measures used.
</Definition>

### Fuchs-Peres Trade-off

<Formula>
**Fuchs-Peres Relation:**

For binary outcome measurements distinguishing two states:

$$G + F \leq 1$$

where $G$ is the probability of correctly guessing the state and $F$ is the average fidelity of the post-measurement state with the original.
</Formula>

```python
import numpy as np
from scipy.optimize import minimize

def fuchs_peres_analysis(psi1: np.ndarray, psi2: np.ndarray,
                          strength: float = 1.0) -> dict:
    """
    Analyze Fuchs-Peres trade-off for two pure states.

    Args:
        psi1, psi2: State vectors
        strength: Measurement strength (0=no measurement, 1=projective)

    Returns:
        Dictionary with information gain and disturbance
    """
    # Overlap
    overlap = np.abs(np.vdot(psi1, psi2))

    # Weak measurement with controllable strength
    # Kraus operators interpolate between identity and projective
    s = strength

    # For simplicity, use measurement along optimal direction
    # Optimal distinguishes psi1 and psi2
    diff = psi1 - psi2 * np.vdot(psi1, psi2)
    if np.linalg.norm(diff) > 1e-10:
        diff = diff / np.linalg.norm(diff)
    else:
        diff = np.array([1, 0], dtype=complex)

    # Measurement operators
    M_plus = np.sqrt((1+s)/2) * np.eye(2) + np.sqrt(s*(1-s)/2) * np.outer(diff, diff.conj())
    M_minus = np.sqrt((1-s)/2) * np.eye(2) - np.sqrt(s*(1-s)/2) * np.outer(diff, diff.conj())

    # Actually, let's use a simpler model
    # Weak Z measurement
    c = np.sqrt((1 + strength) / 2)
    s_val = np.sqrt((1 - strength) / 2)

    M0 = np.array([[c, 0], [0, s_val]], dtype=complex)
    M1 = np.array([[s_val, 0], [0, c]], dtype=complex)

    # Information gain: probability of correct guess
    # With equal priors
    rho1 = np.outer(psi1, psi1.conj())
    rho2 = np.outer(psi2, psi2.conj())

    p0_given_1 = np.real(np.trace(M0 @ rho1 @ M0.conj().T))
    p1_given_1 = np.real(np.trace(M1 @ rho1 @ M1.conj().T))
    p0_given_2 = np.real(np.trace(M0 @ rho2 @ M0.conj().T))
    p1_given_2 = np.real(np.trace(M1 @ rho2 @ M1.conj().T))

    # Optimal decoding: guess 1 if p(1|outcome) > p(2|outcome)
    # For equal priors: compare p(outcome|1) vs p(outcome|2)
    p_correct = 0.5 * (max(p0_given_1, p0_given_2) + max(p1_given_1, p1_given_2))

    # Disturbance: average fidelity loss
    def avg_fidelity(psi, M0, M1):
        rho = np.outer(psi, psi.conj())
        fid_total = 0
        for M in [M0, M1]:
            p = np.real(np.trace(M @ rho @ M.conj().T))
            if p > 1e-10:
                post = M @ rho @ M.conj().T / p
                fid_total += p * fidelity(rho, post)
        return fid_total

    F1 = avg_fidelity(psi1, M0, M1)
    F2 = avg_fidelity(psi2, M0, M1)
    avg_fid = (F1 + F2) / 2

    return {
        'overlap': overlap,
        'strength': strength,
        'p_correct': p_correct,
        'avg_fidelity': avg_fid,
        'information_gain': 2 * p_correct - 1,  # Normalized to [0,1]
        'disturbance': 1 - avg_fid,
        'sum': p_correct + avg_fid  # Should be <= 1 + constant
    }

# Analyze trade-off curve
print("Fuchs-Peres Trade-off Analysis")
print("=" * 50)

psi_0 = np.array([1, 0], dtype=complex)
psi_plus = np.array([1, 1], dtype=complex) / np.sqrt(2)

for strength in [0.0, 0.25, 0.5, 0.75, 1.0]:
    result = fuchs_peres_analysis(psi_0, psi_plus, strength)
    print(f"Strength {strength:.2f}: P_correct={result['p_correct']:.3f}, "
          f"Fidelity={result['avg_fidelity']:.3f}, Sum={result['sum']:.3f}")
```

## 4. Measurement Reversibility

Not all measurements are equally disturbing. Some can be partially "undone."

<Definition term="Reversible Measurement">
A measurement is reversible if there exists a recovery operation that restores the original state (possibly conditioned on the outcome). Projective measurements are generally irreversible, while weak measurements can be more reversible.
</Definition>

```python
import numpy as np

def weak_measurement_reversibility(psi: np.ndarray, strength: float) -> dict:
    """
    Analyze reversibility of a weak measurement.

    Args:
        psi: Initial state
        strength: Measurement strength in [0, 1]

    Returns:
        Analysis of reversibility
    """
    # Weak Z measurement Kraus operators
    g = strength
    c = np.sqrt((1 + g) / 2)
    s = np.sqrt((1 - g) / 2)

    M0 = np.array([[c, 0], [0, s]], dtype=complex)
    M1 = np.array([[s, 0], [0, c]], dtype=complex)

    rho = np.outer(psi, psi.conj())

    results = []
    for i, M in enumerate([M0, M1]):
        p = np.real(np.trace(M @ rho @ M.conj().T))
        if p > 1e-10:
            rho_post = M @ rho @ M.conj().T / p

            # Try to reverse: apply M^{-1} (if exists) or pseudo-inverse
            try:
                M_inv = np.linalg.inv(M)
                rho_recovered = M_inv @ rho_post @ M_inv.conj().T
                # Normalize
                rho_recovered = rho_recovered / np.trace(rho_recovered)
                recovery_fidelity = fidelity(rho, rho_recovered)
            except np.linalg.LinAlgError:
                recovery_fidelity = 0

            results.append({
                'outcome': i,
                'probability': p,
                'post_fidelity': fidelity(rho, rho_post),
                'recovery_fidelity': recovery_fidelity
            })

    return {
        'strength': strength,
        'outcomes': results,
        'avg_recovery': sum(r['probability'] * r['recovery_fidelity'] for r in results)
    }

# Analyze reversibility vs strength
print("Measurement Reversibility Analysis")
print("=" * 50)

psi = np.array([1, 1], dtype=complex) / np.sqrt(2)

for strength in [0.1, 0.3, 0.5, 0.7, 0.9]:
    result = weak_measurement_reversibility(psi, strength)
    print(f"Strength {strength:.1f}: Average recovery fidelity = {result['avg_recovery']:.4f}")
```

<Tip>
Weak measurements extract less information but cause less disturbance. This trade-off is exploited in weak value amplification and continuous quantum measurement protocols.
</Tip>

## 5. Error-Disturbance Relations

The uncertainty principle has an operational formulation in terms of measurement errors and disturbance.

<Definition term="Ozawa's Error-Disturbance Relation">
For measuring observable $A$ while observable $B$ is disturbed:

$$\epsilon(A) \eta(B) + \epsilon(A) \sigma(B) + \sigma(A) \eta(B) \geq \frac{1}{2}|\langle[A, B]\rangle|$$

where $\epsilon(A)$ is measurement error, $\eta(B)$ is disturbance to $B$, and $\sigma$ denotes standard deviations.
</Definition>

```python
import numpy as np

def ozawa_analysis(rho: np.ndarray, A: np.ndarray, B: np.ndarray,
                    measurement_kraus: list) -> dict:
    """
    Analyze Ozawa's error-disturbance relation.

    Args:
        rho: Initial state
        A: Observable being measured
        B: Observable potentially disturbed
        measurement_kraus: Kraus operators for A measurement

    Returns:
        Error-disturbance analysis
    """
    # Commutator
    comm = A @ B - B @ A
    commutator_expect = np.abs(np.trace(rho @ comm))

    # Standard deviations in initial state
    A_expect = np.real(np.trace(rho @ A))
    A2_expect = np.real(np.trace(rho @ A @ A))
    sigma_A = np.sqrt(A2_expect - A_expect**2)

    B_expect = np.real(np.trace(rho @ B))
    B2_expect = np.real(np.trace(rho @ B @ B))
    sigma_B = np.sqrt(B2_expect - B_expect**2)

    # Measurement error: deviation from ideal A measurement
    # For POVM {E_k}, error = sqrt(sum_k Tr(rho E_k (a_k - A)^2))
    # Simplified: use outcome variance vs A variance
    outcome_values = []
    for i, M in enumerate(measurement_kraus):
        p = np.real(np.trace(M @ rho @ M.conj().T))
        # Assume outcome value is eigenvalue-like
        # For Z measurement: outcomes are +-1
        outcome_values.append((p, 2*i - 1))  # Simplified

    measured_mean = sum(p * v for p, v in outcome_values)
    measured_var = sum(p * v**2 for p, v in outcome_values) - measured_mean**2
    error_A = np.sqrt(np.abs(measured_var - (A2_expect - A_expect**2)))

    # Disturbance to B: change in B statistics
    rho_post = np.zeros_like(rho)
    for M in measurement_kraus:
        rho_post += M @ rho @ M.conj().T

    B_expect_post = np.real(np.trace(rho_post @ B))
    B2_expect_post = np.real(np.trace(rho_post @ B @ B))
    sigma_B_post = np.sqrt(np.abs(B2_expect_post - B_expect_post**2))

    eta_B = np.abs(sigma_B_post - sigma_B)

    # Ozawa bound
    lhs = error_A * eta_B + error_A * sigma_B + sigma_A * eta_B
    rhs = 0.5 * commutator_expect

    return {
        'error_A': error_A,
        'disturbance_B': eta_B,
        'sigma_A': sigma_A,
        'sigma_B': sigma_B,
        'commutator': commutator_expect,
        'ozawa_lhs': lhs,
        'ozawa_rhs': rhs,
        'satisfied': lhs >= rhs - 1e-10
    }

# Test with X and Z measurements
X = np.array([[0, 1], [1, 0]], dtype=complex)
Z = np.array([[1, 0], [0, -1]], dtype=complex)

# Z measurement Kraus operators
M0 = np.array([[1, 0], [0, 0]], dtype=complex)
M1 = np.array([[0, 0], [0, 1]], dtype=complex)
Z_kraus = [M0, M1]

# Test state
psi = np.array([np.cos(np.pi/8), np.sin(np.pi/8)], dtype=complex)
rho = np.outer(psi, psi.conj())

result = ozawa_analysis(rho, Z, X, Z_kraus)

print("Ozawa Error-Disturbance Analysis")
print("=" * 50)
print(f"Measuring Z, disturbing X")
print(f"Error in Z: {result['error_A']:.4f}")
print(f"Disturbance to X: {result['disturbance_B']:.4f}")
print(f"Ozawa LHS: {result['ozawa_lhs']:.4f}")
print(f"Ozawa RHS: {result['ozawa_rhs']:.4f}")
print(f"Relation satisfied: {result['satisfied']}")
```

## 6. Applications

### Quantum Key Distribution Security

<Definition term="Disturbance-Based Security">
In BB84 QKD, an eavesdropper's presence is detected through disturbance. Any attempt to extract information about the key necessarily disturbs the quantum states, causing detectable errors.
</Definition>

```python
def qkd_intercept_resend_disturbance():
    """
    Analyze disturbance in intercept-resend attack on BB84.
    """
    # BB84 states
    Z_states = [
        np.array([1, 0], dtype=complex),  # |0>
        np.array([0, 1], dtype=complex)   # |1>
    ]
    X_states = [
        np.array([1, 1], dtype=complex) / np.sqrt(2),   # |+>
        np.array([1, -1], dtype=complex) / np.sqrt(2)   # |->
    ]

    all_states = Z_states + X_states

    # Eve measures in Z basis (50% correct basis choice)
    M0 = np.array([[1, 0], [0, 0]], dtype=complex)
    M1 = np.array([[0, 0], [0, 1]], dtype=complex)

    total_disturbance = 0
    n_states = 0

    for psi in all_states:
        rho = np.outer(psi, psi.conj())

        # Eve's measurement disturbs the state
        for M in [M0, M1]:
            p = np.real(np.trace(M @ rho))
            if p > 1e-10:
                rho_post = np.outer(M @ psi, (M @ psi).conj()) / p

                # Disturbance when Bob measures in same basis as Alice
                dist = 1 - fidelity(rho, rho_post)
                total_disturbance += dist

        n_states += 2  # Two outcomes per state

    avg_disturbance = total_disturbance / n_states

    return {
        'avg_disturbance': avg_disturbance,
        'error_rate': avg_disturbance / 2,  # Rough approximation
        'detection_probability': 1 - (1 - avg_disturbance/2)**20  # After 20 bits
    }

result = qkd_intercept_resend_disturbance()
print("QKD Intercept-Resend Attack Analysis")
print("=" * 50)
print(f"Average state disturbance: {result['avg_disturbance']:.4f}")
print(f"Estimated error rate: {result['error_rate']:.4f}")
print(f"Detection probability (20 test bits): {result['detection_probability']:.4f}")
```
