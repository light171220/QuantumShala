## 1. The Ground State Problem

Finding the ground state energy of a quantum system is one of the most fundamental problems in physics and chemistry. Given a Hamiltonian $H$, we want to find:

<Formula>
$$E_0 = \min_{\psi} \frac{\langle\psi|H|\psi\rangle}{\langle\psi|\psi\rangle}$$
</Formula>

<Definition term="Ground State">
The ground state $|\psi_0\rangle$ is the eigenstate of the Hamiltonian $H$ with the lowest eigenvalue $E_0$. It represents the most stable configuration of the quantum system.
</Definition>

For classical computers, finding ground states is exponentially hard in general (QMA-complete). The Variational Quantum Eigensolver (VQE) offers a hybrid approach that leverages quantum computers for state preparation and measurement.

## 2. The Variational Principle

The cornerstone of VQE is the **variational principle**, which states:

<Formula>
$$\langle\psi|H|\psi\rangle \geq E_0 \quad \text{for any normalized state } |\psi\rangle$$
</Formula>

<Definition term="Variational Principle">
For any trial wavefunction $|\psi\rangle$, the expectation value of the Hamiltonian is always greater than or equal to the ground state energy. Equality holds if and only if $|\psi\rangle = |\psi_0\rangle$.
</Definition>

### Proof of the Variational Principle

Let $\{|\phi_n\rangle\}$ be the complete set of eigenstates of $H$ with energies $E_n$ where $E_0 \leq E_1 \leq E_2 \leq \ldots$

Any state can be expanded as:
$$|\psi\rangle = \sum_n c_n |\phi_n\rangle$$

Then:
$$\langle\psi|H|\psi\rangle = \sum_n |c_n|^2 E_n \geq E_0 \sum_n |c_n|^2 = E_0$$

```python
import numpy as np
from typing import Tuple, List

def demonstrate_variational_principle(
    hamiltonian: np.ndarray,
    num_random_states: int = 1000
) -> Tuple[float, List[float]]:
    """
    Demonstrate the variational principle numerically.

    Args:
        hamiltonian: Hermitian matrix representing H
        num_random_states: Number of random trial states

    Returns:
        (ground_state_energy, list_of_expectation_values)
    """
    # Get exact ground state energy
    eigenvalues, eigenvectors = np.linalg.eigh(hamiltonian)
    ground_state_energy = eigenvalues[0]

    expectation_values = []
    dim = hamiltonian.shape[0]

    for _ in range(num_random_states):
        # Generate random normalized state
        state = np.random.randn(dim) + 1j * np.random.randn(dim)
        state = state / np.linalg.norm(state)

        # Compute expectation value
        exp_val = np.real(state.conj() @ hamiltonian @ state)
        expectation_values.append(exp_val)

    # Verify all expectation values >= ground state energy
    all_above_ground = all(e >= ground_state_energy - 1e-10
                          for e in expectation_values)
    print(f"Ground state energy: {ground_state_energy:.6f}")
    print(f"Min expectation value: {min(expectation_values):.6f}")
    print(f"Variational principle holds: {all_above_ground}")

    return ground_state_energy, expectation_values

# Example: 2-qubit Heisenberg Hamiltonian
X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])
I = np.eye(2)

H_heisenberg = (np.kron(X, X) + np.kron(Y, Y) + np.kron(Z, Z))
E0, exp_vals = demonstrate_variational_principle(H_heisenberg)
```

## 3. The Rayleigh-Ritz Quotient

The Rayleigh-Ritz quotient generalizes the variational principle for unnormalized states:

<Formula>
$$R(|\psi\rangle) = \frac{\langle\psi|H|\psi\rangle}{\langle\psi|\psi\rangle}$$
</Formula>

### Properties of the Rayleigh Quotient

1. **Bounded below by $E_0$**: $R(|\psi\rangle) \geq E_0$
2. **Bounded above by $E_{max}$**: $R(|\psi\rangle) \leq E_{max}$
3. **Stationary points are eigenstates**: $\nabla R = 0$ implies $|\psi\rangle$ is an eigenstate
4. **Scale invariance**: $R(c|\psi\rangle) = R(|\psi\rangle)$ for any $c \neq 0$

```python
def rayleigh_quotient(
    state: np.ndarray,
    hamiltonian: np.ndarray
) -> float:
    """
    Compute the Rayleigh-Ritz quotient for a given state.

    Args:
        state: Quantum state vector (not necessarily normalized)
        hamiltonian: Hermitian Hamiltonian matrix

    Returns:
        Rayleigh quotient value
    """
    numerator = np.real(state.conj() @ hamiltonian @ state)
    denominator = np.real(state.conj() @ state)
    return numerator / denominator

def gradient_rayleigh_quotient(
    state: np.ndarray,
    hamiltonian: np.ndarray
) -> np.ndarray:
    """
    Compute the gradient of the Rayleigh quotient.

    The gradient is: 2(H|psi> - R|psi>) / <psi|psi>
    """
    norm_sq = np.real(state.conj() @ state)
    R = rayleigh_quotient(state, hamiltonian)
    gradient = 2 * (hamiltonian @ state - R * state) / norm_sq
    return gradient
```

## 4. VQE Algorithm Structure

VQE uses a parameterized quantum circuit (ansatz) $U(\theta)$ to prepare trial states:

<Formula>
$$|\psi(\theta)\rangle = U(\theta)|0\rangle^{\otimes n}$$
</Formula>

The optimization problem becomes:

<Formula>
$$\theta^* = \arg\min_{\theta} \langle\psi(\theta)|H|\psi(\theta)\rangle$$
</Formula>

### Algorithm Components

```python
from dataclasses import dataclass
from typing import Callable, Optional
import numpy as np

@dataclass
class VQEConfig:
    """Configuration for VQE algorithm."""
    num_qubits: int
    ansatz_depth: int
    hamiltonian: np.ndarray
    optimizer: str = "COBYLA"
    max_iterations: int = 1000
    convergence_threshold: float = 1e-6
    shots: int = 8192

class VQETheoretical:
    """
    Theoretical VQE implementation for understanding core concepts.
    Uses exact state vector simulation.
    """

    def __init__(self, config: VQEConfig):
        self.config = config
        self.iteration_history = []

    def apply_ansatz(self, params: np.ndarray) -> np.ndarray:
        """
        Apply parameterized ansatz to |0...0> state.

        This is a simplified hardware-efficient ansatz with
        Ry rotations and CNOT entangling layers.
        """
        n = self.config.num_qubits
        depth = self.config.ansatz_depth

        # Initialize |0...0> state
        state = np.zeros(2**n, dtype=complex)
        state[0] = 1.0

        param_idx = 0

        for layer in range(depth):
            # Single-qubit Ry rotations
            for qubit in range(n):
                theta = params[param_idx]
                param_idx += 1
                state = self._apply_ry(state, qubit, theta)

            # CNOT entangling layer (linear connectivity)
            for qubit in range(n - 1):
                state = self._apply_cnot(state, qubit, qubit + 1)

        # Final rotation layer
        for qubit in range(n):
            theta = params[param_idx]
            param_idx += 1
            state = self._apply_ry(state, qubit, theta)

        return state

    def _apply_ry(self, state: np.ndarray, qubit: int, theta: float) -> np.ndarray:
        """Apply Ry(theta) gate to specified qubit."""
        n = self.config.num_qubits
        ry = np.array([
            [np.cos(theta/2), -np.sin(theta/2)],
            [np.sin(theta/2), np.cos(theta/2)]
        ])

        # Build full operator
        ops = [np.eye(2)] * n
        ops[qubit] = ry
        full_op = ops[0]
        for op in ops[1:]:
            full_op = np.kron(full_op, op)

        return full_op @ state

    def _apply_cnot(self, state: np.ndarray, control: int, target: int) -> np.ndarray:
        """Apply CNOT gate."""
        n = self.config.num_qubits
        new_state = state.copy()

        for i in range(2**n):
            # Check if control qubit is |1>
            if (i >> (n - 1 - control)) & 1:
                # Flip target qubit
                j = i ^ (1 << (n - 1 - target))
                new_state[i], new_state[j] = state[j], state[i]

        return new_state

    def compute_energy(self, params: np.ndarray) -> float:
        """Compute expectation value <psi(params)|H|psi(params)>."""
        state = self.apply_ansatz(params)
        energy = np.real(state.conj() @ self.config.hamiltonian @ state)
        self.iteration_history.append(energy)
        return energy

    def num_parameters(self) -> int:
        """Return total number of variational parameters."""
        n = self.config.num_qubits
        depth = self.config.ansatz_depth
        return n * (depth + 1)  # depth layers + final layer
```

## 5. Mathematical Guarantees and Bounds

### Approximation Error Bound

The approximation error of VQE is bounded by the expressibility of the ansatz:

<Formula>
$$E_{VQE} - E_0 \leq \min_{|\psi\rangle \in \mathcal{A}} \langle\psi|H - E_0|\psi\rangle$$
</Formula>

where $\mathcal{A}$ is the set of states reachable by the ansatz.

<Tip>
The approximation error depends critically on whether the ansatz can represent the true ground state. This is why ansatz design is crucial for VQE success.
</Tip>

### Statistical Error from Sampling

When estimating energies from measurements, the statistical error scales as:

<Formula>
$$\sigma_E = \frac{\sqrt{\text{Var}(H)}}{\sqrt{N_{shots}}}$$
</Formula>

where $\text{Var}(H) = \langle H^2 \rangle - \langle H \rangle^2$.

```python
def estimate_sampling_error(
    hamiltonian: np.ndarray,
    state: np.ndarray,
    num_shots: int
) -> dict:
    """
    Estimate statistical error from finite sampling.

    Args:
        hamiltonian: Hamiltonian matrix
        state: Quantum state
        num_shots: Number of measurement shots

    Returns:
        Dictionary with energy estimate and error bounds
    """
    # Exact expectation values
    H_expectation = np.real(state.conj() @ hamiltonian @ state)
    H2_expectation = np.real(state.conj() @ (hamiltonian @ hamiltonian) @ state)

    # Variance
    variance = H2_expectation - H_expectation**2

    # Standard error
    std_error = np.sqrt(variance / num_shots)

    return {
        'energy': H_expectation,
        'variance': variance,
        'std_error': std_error,
        'confidence_interval_95': (
            H_expectation - 1.96 * std_error,
            H_expectation + 1.96 * std_error
        )
    }
```

## 6. Hamiltonian Decomposition for Measurement

Real quantum computers measure in the computational basis. To estimate $\langle H \rangle$, we decompose the Hamiltonian into Pauli strings:

<Formula>
$$H = \sum_i c_i P_i$$
</Formula>

where each $P_i$ is a tensor product of Pauli operators.

```python
def decompose_to_pauli_strings(
    hamiltonian: np.ndarray,
    num_qubits: int
) -> List[Tuple[complex, str]]:
    """
    Decompose Hamiltonian into Pauli string representation.

    Args:
        hamiltonian: 2^n x 2^n Hamiltonian matrix
        num_qubits: Number of qubits

    Returns:
        List of (coefficient, pauli_string) tuples
    """
    paulis = {
        'I': np.eye(2),
        'X': np.array([[0, 1], [1, 0]]),
        'Y': np.array([[0, -1j], [1j, 0]]),
        'Z': np.array([[1, 0], [0, -1]])
    }

    pauli_terms = []
    n = num_qubits

    # Iterate over all possible Pauli strings
    for pauli_indices in np.ndindex(*([4] * n)):
        pauli_labels = ['I', 'X', 'Y', 'Z']
        pauli_string = ''.join(pauli_labels[i] for i in pauli_indices)

        # Build Pauli operator
        P = paulis[pauli_labels[pauli_indices[0]]]
        for idx in pauli_indices[1:]:
            P = np.kron(P, paulis[pauli_labels[idx]])

        # Coefficient is Tr(H @ P) / 2^n
        coeff = np.trace(hamiltonian @ P) / (2**n)

        if np.abs(coeff) > 1e-10:
            pauli_terms.append((coeff, pauli_string))

    return pauli_terms

# Example
H = np.array([
    [1, 0, 0, 0],
    [0, -1, 2, 0],
    [0, 2, -1, 0],
    [0, 0, 0, 1]
], dtype=complex)

terms = decompose_to_pauli_strings(H, 2)
print("Pauli decomposition:")
for coeff, pauli in terms:
    if np.abs(coeff.imag) < 1e-10:
        print(f"  {coeff.real:+.4f} {pauli}")
```

## 7. Energy Landscape Analysis

Understanding the optimization landscape is crucial for VQE success:

```python
def analyze_energy_landscape(
    vqe: VQETheoretical,
    param_range: Tuple[float, float] = (-np.pi, np.pi),
    resolution: int = 50
) -> dict:
    """
    Analyze the VQE energy landscape for 2-parameter visualization.

    Returns landscape statistics and critical points.
    """
    num_params = vqe.num_parameters()

    if num_params < 2:
        raise ValueError("Need at least 2 parameters for landscape analysis")

    # Create 2D slice of landscape (vary first two parameters)
    theta1 = np.linspace(param_range[0], param_range[1], resolution)
    theta2 = np.linspace(param_range[0], param_range[1], resolution)

    landscape = np.zeros((resolution, resolution))
    base_params = np.zeros(num_params)

    for i, t1 in enumerate(theta1):
        for j, t2 in enumerate(theta2):
            params = base_params.copy()
            params[0] = t1
            params[1] = t2
            landscape[i, j] = vqe.compute_energy(params)

    # Find critical points
    min_idx = np.unravel_index(np.argmin(landscape), landscape.shape)
    max_idx = np.unravel_index(np.argmax(landscape), landscape.shape)

    return {
        'landscape': landscape,
        'theta1': theta1,
        'theta2': theta2,
        'global_min': landscape[min_idx],
        'global_max': landscape[max_idx],
        'min_location': (theta1[min_idx[0]], theta2[min_idx[1]]),
        'energy_range': landscape.max() - landscape.min()
    }
```

## Summary

In this lesson, you learned the theoretical foundations of VQE:

1. **Variational Principle**: Any trial state gives an upper bound on ground state energy
2. **Rayleigh-Ritz Quotient**: Provides a framework for optimizing over unnormalized states
3. **VQE Structure**: Combines parameterized quantum circuits with classical optimization
4. **Mathematical Guarantees**: Error bounds from ansatz expressibility and finite sampling
5. **Hamiltonian Decomposition**: Converting Hamiltonians to measurable Pauli strings
6. **Landscape Analysis**: Understanding optimization challenges

<Warning>
While VQE provides theoretical guarantees, practical success depends heavily on ansatz design, optimizer choice, and noise resilience. The variational principle guarantees only an upper bound - not that we can efficiently find it!
</Warning>
