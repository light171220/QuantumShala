## 2. Molecular Hamiltonian Construction

The electronic structure Hamiltonian in second quantization:

<Formula>
$$H = \sum_{pq} h_{pq} a_p^\dagger a_q + \frac{1}{2}\sum_{pqrs} h_{pqrs} a_p^\dagger a_q^\dagger a_r a_s$$
</Formula>

```python
    def to_second_quantization(self) -> List[Tuple[complex, str]]:
        """
        Convert to fermionic operator representation.

        Returns list of (coefficient, operator_string) tuples.
        """
        terms = []
        n = self.data.num_orbitals

        # One-body terms: h_pq a_p^dag a_q
        h1 = self.data.one_body_integrals
        for p in range(n):
            for q in range(n):
                if abs(h1[p, q]) > 1e-10:
                    # Both spin-up and spin-down
                    for spin in [0, 1]:
                        p_spin = 2 * p + spin
                        q_spin = 2 * q + spin
                        terms.append((h1[p, q], f"a{p_spin}^dag a{q_spin}"))

        # Two-body terms: (1/2) h_pqrs a_p^dag a_q^dag a_r a_s
        h2 = self.data.two_body_integrals
        for p in range(n):
            for q in range(n):

                    for s in range(n):
```

## 3. VQE for Small Molecules

```python
from typing import Callable, Optional

class MolecularVQE:
    """
    VQE implementation for molecular systems.
    """

    def __init__(
        self,
        molecular_data: MolecularData,
        ansatz_type: str = 'UCCSD',
        optimizer: str = 'COBYLA'
    ):
        self.data = molecular_data
        self.ansatz_type = ansatz_type
        self.optimizer = optimizer
        self.hamiltonian = MolecularHamiltonian(molecular_data)

    def run(
        self,
        initial_params: Optional[np.ndarray] = None,
        max_iterations: int = 1000,
        shots: int = 8192
    ) -> Dict:
        """
        Run VQE optimization.

        Returns dictionary with:
        - 'energy': optimized ground state energy
        - 'params': optimal parameters
        - 'history': energy vs iteration
        - 'chemical_accuracy': bool (if error < 1.6 mHa)
        """
        # Number of parameters depends on ansatz
        num_params = self._get_num_params()

        if initial_params is None:
            # Initialize near Hartree-Fock
            initial_params = np.zeros(num_params)

        # Run optimization
        history = []

        def cost_function(params):
            energy = self._evaluate_energy(params, shots)
            history.append(energy)
            return energy

        from scipy.optimize import minimize
        result = minimize(
            cost_function,
            initial_params,
            method=self.optimizer,
            options={'maxiter': max_iterations}
        )

        # Add nuclear repulsion
        total_energy = result.fun + self.data.nuclear_repulsion

        return {
            'energy': total_energy,
            'electronic_energy': result.fun,
            'params': result.x,
            'history': history,
            'nuclear_repulsion': self.data.nuclear_repulsion,
            'hf_energy': self.data.hf_energy,
            'correlation_energy': total_energy - self.data.hf_energy,
            'converged': result.success
        }

    def _get_num_params(self) -> int:
        """Get number of variational parameters for ansatz."""
        n_occ = self.data.num_electrons // 2
        n_virt = self.data.num_orbitals - n_occ

        if self.ansatz_type == 'UCCSD':
            # Singles + Doubles
            singles = n_occ * n_virt * 2  # alpha and beta
            doubles = (n_occ * (n_occ - 1) // 2) * (n_virt * (n_virt - 1) // 2)
            doubles += n_occ * n_occ * n_virt * n_virt  # mixed spin
            return singles + doubles
        elif self.ansatz_type == 'hardware_efficient':
            layers = 2
            return self.hamiltonian.num_qubits * (layers + 1)
        else:
            raise ValueError(f"Unknown ansatz: {self.ansatz_type}")

    def _evaluate_energy(self, params: np.ndarray, shots: int) -> float:

        return energy

    def compute_properties(self, optimal_params: np.ndarray) -> Dict:
        """
        Compute molecular properties from optimized state.
        """
        return {
            'dipole_moment': self._compute_dipole(optimal_params),
            'one_rdm': self._compute_one_rdm(optimal_params),
            'two_rdm': self._compute_two_rdm(optimal_params)
        }

    def _compute_dipole(self, params):
```

## 4. Potential Energy Surfaces

Compute molecular energy as a function of geometry:

```python
def compute_potential_energy_surface(
    molecule: str,
    parameter_name: str,
    parameter_range: Tuple[float, float],
    num_points: int = 20,
    method: str = 'vqe'
) -> Dict:
    """
    Compute potential energy surface along a coordinate.

    Args:
        molecule: Molecule name (e.g., 'H2', 'LiH')
        parameter_name: Geometric parameter (e.g., 'bond_length')
        parameter_range: (min, max) values
        num_points: Number of points to compute
        method: 'vqe', 'hf', or 'exact'

    Returns:
        Dictionary with parameter values and energies
    """
    params = np.linspace(parameter_range[0], parameter_range[1], num_points)
    energies = []

    for param in params:
        if molecule == 'H2':
            mol_data = create_h2_data(bond_length=param)
        else:
            raise ValueError(f"Unknown molecule: {molecule}")

        if method == 'vqe':
            vqe = MolecularVQE(mol_data)
            result = vqe.run(max_iterations=50)
            energies.append(result['energy'])
        elif method == 'hf':
            energies.append(mol_data.hf_energy + mol_data.nuclear_repulsion)
        elif method == 'exact':
            # Would use exact diagonalization
            energies.append(-1.0 + 0.5 / param)  # Placeholder

    # Find equilibrium
    min_idx = np.argmin(energies)
    equilibrium_param = params[min_idx]
    equilibrium_energy = energies[min_idx]

    return {
        'parameters': params,
        'energies': np.array(energies),
        'equilibrium_parameter': equilibrium_param,
        'equilibrium_energy': equilibrium_energy,
        'method': method
    }
```

## 5. Materials Science: Lattice Models

VQE for condensed matter systems:

### Heisenberg Model

<Formula>
$$H = J \sum_{\langle i,j \rangle} \vec{S}_i \cdot \vec{S}_j = J \sum_{\langle i,j \rangle} (X_i X_j + Y_i Y_j + Z_i Z_j)$$
</Formula>

```python
class HeisenbergModel:
    """
    Heisenberg spin model for VQE.
    """

    def __init__(
        self,
        num_sites: int,
        coupling: float = 1.0,
        topology: str = 'chain'
    ):
        self.num_sites = num_sites
        self.J = coupling
        self.topology = topology

    def get_hamiltonian_terms(self) -> List[Tuple[float, str]]:
        """
        Return Hamiltonian as list of (coefficient, Pauli string) tuples.
        """
        terms = []
        pairs = self._get_neighbor_pairs()

        for i, j in pairs:
            # XX term
            pauli_xx = ['I'] * self.num_sites
            pauli_xx[i] = 'X'
            pauli_xx[j] = 'X'
            terms.append((self.J, ''.join(pauli_xx)))

            # YY term
            pauli_yy = ['I'] * self.num_sites
            pauli_yy[i] = 'Y'
            pauli_yy[j] = 'Y'
            terms.append((self.J, ''.join(pauli_yy)))

            # ZZ term
            pauli_zz = ['I'] * self.num_sites
            pauli_zz[i] = 'Z'
            pauli_zz[j] = 'Z'
            terms.append((self.J, ''.join(pauli_zz)))

        return terms

    def _get_neighbor_pairs(self) -> List[Tuple[int, int]]:
        """Get neighboring site pairs based on topology."""
        if self.topology == 'chain':
            return [(i, i+1) for i in range(self.num_sites - 1)]
        elif self.topology == 'ring':
            pairs = [(i, i+1) for i in range(self.num_sites - 1)]
            pairs.append((self.num_sites - 1, 0))
            return pairs
        elif self.topology == 'ladder':
            # 2-leg ladder
            n = self.num_sites // 2
            pairs = []
            # Rungs
            for i in range(n):
                pairs.append((i, i + n))
            # Legs
            for i in range(n - 1):
                pairs.append((i, i + 1))
                pairs.append((i + n, i + n + 1))
            return pairs
        else:
            raise ValueError(f"Unknown topology: {self.topology}")

    def get_exact_ground_state(self) -> Tuple[float, np.ndarray]:
        """
        Get exact ground state by diagonalization.
        """
        H = self.build_matrix()
        eigenvalues, eigenvectors = np.linalg.eigh(H)
        return eigenvalues[0], eigenvectors[:, 0]

    def build_matrix(self) -> np.ndarray:
        """Build full Hamiltonian matrix."""
        dim = 2 ** self.num_sites
        H = np.zeros((dim, dim), dtype=complex)

        paulis = {
            'I': np.eye(2),
            'X': np.array([[0, 1], [1, 0]]),
            'Y': np.array([[0, -1j], [1j, 0]]),
            'Z': np.array([[1, 0], [0, -1]])
        }

        for coeff, pauli_string in self.get_hamiltonian_terms():
            # Build operator for this term
            op = paulis[pauli_string[0]]
            for p in pauli_string[1:]:
                op = np.kron(op, paulis[p])
            H += coeff * op

        return H

# Example: 4-site Heisenberg chain
model = HeisenbergModel(num_sites=4, coupling=1.0, topology='chain')
exact_energy, ground_state = model.get_exact_ground_state()
print(f"4-site Heisenberg chain:")
print(f"  Exact ground state energy: {exact_energy:.6f}")
print(f"  Number of Pauli terms: {len(model.get_hamiltonian_terms())}")
```

```python
    """
    Estimate resources for VQE calculation.

    Returns estimates for qubits, gates, and measurements.
    """
    estimates = {}

    if problem_type == 'molecular':
        # Molecular simulation
        n_orbitals = problem_size
        estimates = {
            'qubits': 2 * n_orbitals,  # spin orbitals
            'pauli_terms': n_orbitals ** 4,  # two-body integrals
            'uccsd_parameters': n_orbitals ** 4,  # singles + doubles
            'circuit_depth': f'O({n_orbitals}^4)',
            'measurements_per_iteration': n_orbitals ** 4,
            'classical_comparison': {
                'ccsd': f'O({n_orbitals}^6)',
                'fci': f'O({n_orbitals}!)',
                'dmrg': f'O({n_orbitals}^3 * chi^3)'
            }
        }

    elif problem_type == 'heisenberg':
        n_sites = problem_size
        estimates = {
            'qubits': n_sites,
            'pauli_terms': 3 * n_sites,  # XX + YY + ZZ per bond
            'hea_parameters': 3 * n_sites,  # hardware efficient
            'circuit_depth': f'O({n_sites})',
            'measurements_per_iteration': 3 * n_sites,
            'classical_comparison': {
                'exact_diag': f'O(2^{n_sites})',
                'dmrg': f'O({n_sites} * chi^3)',
                'qmc': 'Polynomial (with sign problem)'
            }
        }

    elif problem_type == 'hubbard':
        n_sites = problem_size
        estimates = {
            'qubits': 2 * n_sites,
            'pauli_terms': 4 * n_sites + 2 * n_sites,
            'circuit_depth': f'O({n_sites}^2)',
            'classical_comparison': {
                'exact_diag': f'O(4^{n_sites})',
                'dmrg': f'O({n_sites} * chi^3)',
                'afqmc': 'Polynomial (sign problem free for certain cases)'

        }

    # Estimate feasibility on current hardware
    if 'qubits' in estimates:
        n_qubits = estimates['qubits']
        estimates['hardware_feasibility'] = {

            'google_72q': n_qubits <= 72,
```

## 7. Best Practices for VQE Applications

<Tip>
Start with classical benchmarks. Always compare VQE results against Hartree-Fock, CCSD, and (when feasible) exact diagonalization to validate your implementation.
</Tip>

### Workflow Checklist

1. **Classical preprocessing**
   - Generate molecular integrals with classical chemistry software
   - Verify Hartree-Fock energy as baseline
   - Estimate problem size and feasibility

2. **Ansatz selection**
   - Start with UCCSD for chemistry (chemically motivated)
   - Consider hardware-efficient for larger systems
   - Evaluate expressibility vs. trainability

3. **Optimization strategy**
   - Initialize near Hartree-Fock configuration
   - Use robust optimizer (COBYLA, SPSA)
   - Monitor convergence and restart if stuck

4. **Error mitigation**
   - Always apply measurement error mitigation
   - Use ZNE for improved accuracy
   - Validate against exact results when possible

5. **Result analysis**
   - Report statistical uncertainties
   - Compare to chemical accuracy (1.6 mHartree)
   - Analyze correlation energy recovery
