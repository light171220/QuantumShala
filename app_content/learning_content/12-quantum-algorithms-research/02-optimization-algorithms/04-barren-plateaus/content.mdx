## 1. The Barren Plateau Problem

In deep random circuits, gradient variance vanishes exponentially:

<Formula>
$$\text{Var}\left[\frac{\partial E}{\partial\theta}\right] \leq \frac{C}{2^n}$$
</Formula>

```python
import numpy as np

def detect_barren_plateau(gradient_fn, num_params, num_samples=100):
    """Detect barren plateau by measuring gradient variance."""
    gradients = []
    for _ in range(num_samples):
        params = np.random.uniform(-np.pi, np.pi, num_params)
        grad = gradient_fn(params)
        gradients.append(grad)

    grad_array = np.array(gradients)
    variance = np.var(grad_array, axis=0)
    mean_variance = np.mean(variance)

    return {
        'mean_variance': mean_variance,
        'per_param_variance': variance,
        'is_barren': mean_variance < 1e-6
    }
```

## 2. Causes of Barren Plateaus

1. **Deep random circuits**: Expressibility leads to uniform distribution
2. **Global cost functions**: Measuring all qubits
3. **Hardware noise**: Noise-induced flatness

## 3. Mitigation Strategies

```python
def layerwise_training(energy_fn, ansatz, max_layers, params_per_layer):
    """Train layers incrementally to avoid barren plateaus."""
    from scipy.optimize import minimize

    all_params = []

    for layer in range(1, max_layers + 1):
        # Initialize new layer
        new_params = np.random.uniform(-0.1, 0.1, params_per_layer)
        current_params = np.concatenate([all_params, new_params])

        # Optimize
        def cost(p):
            return energy_fn(p, num_layers=layer)

        result = minimize(cost, current_params, method='COBYLA')
        all_params = list(result.x)

    return np.array(all_params)

def identity_initialization(num_params, epsilon=0.01):
    """Initialize near identity transformation."""
    return epsilon * np.random.randn(num_params)
```

## Summary

Barren plateaus are a fundamental challenge in VQE:
1. **Detection**: Measure gradient variance across random initializations
2. **Causes**: Depth, global costs, noise
3. **Mitigation**: Layerwise training, local costs, shallow circuits, identity initialization
