## 1. Introduction to Random Circuit Sampling

**Random Circuit Sampling (RCS)** is the task of sampling from the output distribution of a random quantum circuit.

<Definition term="Random Circuit Sampling">
Given a random quantum circuit U on n qubits, sample bitstrings x from p(x) = |<x|U|0^n>|^2.
</Definition>

<Formula>
$$p(x) = |\langle x|U|0^n\rangle|^2 \sim \text{Porter-Thomas distribution}$$
</Formula>

```python
import numpy as np
from typing import List, Tuple, Optional

class RandomCircuitGenerator:
    def __init__(self, n_qubits: int, seed: Optional[int] = None):
        self.n_qubits = n_qubits
        self.rng = np.random.default_rng(seed)
        self._init_gates()

    def _init_gates(self):
        self.sqrt_x = np.array([[0.5+0.5j, 0.5-0.5j], [0.5-0.5j, 0.5+0.5j]])
        self.sqrt_y = np.array([[0.5+0.5j, -0.5-0.5j], [0.5+0.5j, 0.5+0.5j]])
        self.cz = np.diag([1, 1, 1, -1])

    def random_su2_gate(self) -> np.ndarray:
        z = self.rng.standard_normal((2, 2)) + 1j * self.rng.standard_normal((2, 2))
        q, r = np.linalg.qr(z)
        d = np.diag(r)
        return q @ np.diag(d / np.abs(d))
```

## 2. The Porter-Thomas Distribution

<Definition term="Porter-Thomas Distribution">
For random circuits, p_x follows exponential distribution with mean 1/2^n: P(p_x) = 2^n * exp(-2^n * p_x).
</Definition>

```python
def analyze_porter_thomas(distribution: np.ndarray) -> dict:
    n_qubits = int(np.log2(len(distribution)))
    scaled = distribution * (2 ** n_qubits)
    from scipy import stats
    ks_stat, ks_pvalue = stats.kstest(scaled, 'expon')
    return {'is_porter_thomas': ks_pvalue > 0.05, 'ks_pvalue': ks_pvalue}
```

## 3. Simulation Complexity

<Formula>
$$\text{State Vector:} \quad O(2^n \cdot d)$$
$$\text{Tensor Network:} \quad O(2^{\text{tw}} \cdot nd)$$
</Formula>

```python
class SimulationAnalyzer:
    def __init__(self, n_qubits: int, depth: int):
        self.n, self.d = n_qubits, depth

    def tree_width_estimate(self) -> int:
        side = int(np.ceil(np.sqrt(self.n)))
        return min(side, self.d) * side
```
