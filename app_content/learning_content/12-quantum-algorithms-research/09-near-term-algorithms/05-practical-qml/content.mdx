## 1. Introduction to Practical QML

Quantum Machine Learning (QML) represents one of the most promising near-term applications of quantum computing. While theoretical speedups remain under investigation, practical QML focuses on building and deploying models that leverage quantum computation.

<Definition term="Quantum Machine Learning">
The intersection of quantum computing and machine learning, where quantum systems are used to enhance learning tasks through quantum feature spaces, quantum sampling, or quantum optimization of classical models.
</Definition>

## 2. Data Encoding Strategies

The first critical step in QML is encoding classical data into quantum states. The choice of encoding dramatically affects the resulting quantum feature space.

### 2.1 Angle Encoding

<Formula>
$$|x\rangle = \bigotimes_{i=1}^{n} R_Y(x_i)|0\rangle$$
</Formula>

Angle encoding maps each feature to a rotation angle on a qubit.

```python
import numpy as np

def angle_encoding(features: np.ndarray) -> np.ndarray:
    """
    Encode classical features as rotation angles.

    Args:
        features: Array of n features (should be normalized to [0, pi])

    Returns:
        Quantum state vector after encoding
    """
    n_qubits = len(features)
    state = np.array([1.0 + 0j])  # Start with |0>

    for feature in features:
        # Create single qubit state |psi> = Ry(feature)|0>
        single_qubit = np.array([
            np.cos(feature / 2),
            np.sin(feature / 2)
        ], dtype=complex)
        # Tensor product with existing state
        state = np.kron(state, single_qubit)

    return state

def normalize_features(X: np.ndarray) -> np.ndarray:
    """Normalize features to [0, pi] for angle encoding."""
    X_min = X.min(axis=0)
    X_max = X.max(axis=0)
    X_normalized = (X - X_min) / (X_max - X_min + 1e-10)
    return X_normalized * np.pi

# Example
features = np.array([0.5, 0.8, 0.3])
normalized = normalize_features(features.reshape(1, -1))[0]
quantum_state = angle_encoding(normalized)
print(f"State dimension: {len(quantum_state)}")
print(f"State norm: {np.linalg.norm(quantum_state):.4f}")
```

### 2.2 Amplitude Encoding

<Formula>
$$|x\rangle = \frac{1}{\|x\|} \sum_{i=0}^{N-1} x_i |i\rangle$$
</Formula>

Amplitude encoding maps features directly to state amplitudes, allowing $2^n$ features with $n$ qubits.

```python
def amplitude_encoding(features: np.ndarray) -> np.ndarray:
    """
    Encode features as amplitudes of quantum state.

    Args:
        features: Array of 2^n features

    Returns:
        Normalized quantum state
    """
    # Pad to power of 2 if necessary
    n = len(features)
    next_power_of_2 = 2 ** int(np.ceil(np.log2(n)))

    if n < next_power_of_2:
        features = np.pad(features, (0, next_power_of_2 - n))

    # Normalize
    norm = np.linalg.norm(features)
    if norm < 1e-10:
        raise ValueError("Cannot encode zero vector")

    return features / norm

# Example: Encode 8 features in 3 qubits
data = np.array([1.0, 2.0, 3.0, 4.0, 3.0, 2.0, 1.0, 0.5])
state = amplitude_encoding(data)
print(f"Encoded {len(data)} features in {int(np.log2(len(state)))} qubits")
print(f"Amplitudes: {state}")
```

### 2.3 IQP (Instantaneous Quantum Polynomial) Encoding

<Formula>
$$U_{IQP}(x) = \prod_{i<j} e^{i x_i x_j Z_i Z_j} \prod_i e^{i x_i Z_i} H^{\otimes n}$$
</Formula>

IQP encoding creates entanglement based on feature products, capturing correlations.

```python
def iqp_encoding_circuit(
    features: np.ndarray,
    reps: int = 2
) -> List[dict]:
    """
    Generate IQP encoding circuit description.

    Args:
        features: Input features
        reps: Number of repetitions

    Returns:
        List of gate operations
    """
    n = len(features)
    circuit = []

    for rep in range(reps):
        # Hadamard layer
        for i in range(n):
            circuit.append({'gate': 'H', 'qubit': i})

        # Single qubit Z rotations
        for i in range(n):
            circuit.append({
                'gate': 'RZ',
                'qubit': i,
                'angle': 2 * features[i]
            })

        # Entangling ZZ interactions
        for i in range(n):
            for j in range(i + 1, n):
                circuit.append({
                    'gate': 'RZZ',
                    'qubits': [i, j],
                    'angle': 2 * features[i] * features[j]
                })

    return circuit

# Example
features = np.array([0.3, 0.7, 0.5])
circuit = iqp_encoding_circuit(features)
print(f"IQP circuit has {len(circuit)} operations")
for op in circuit[:5]:
    print(f"  {op}")
```

<Tip>
IQP encoding is particularly useful when feature correlations (products $x_i \cdot x_j$) are important for classification. The ZZ interactions create entanglement that represents these correlations in the quantum feature space.
</Tip>

## 3. Variational Quantum Classifier (VQC)

The VQC combines data encoding with a trainable parameterized circuit to learn classification boundaries.

### 3.1 VQC Architecture

```python
from typing import Optional
import numpy as np

class VariationalQuantumClassifier:
    """
    Implementation of a Variational Quantum Classifier.
    """

    def __init__(
        self,
        n_qubits: int,
        n_layers: int,
        encoding: str = 'angle'
    ):
        self.n_qubits = n_qubits
        self.n_layers = n_layers
        self.encoding = encoding
        self.n_params = n_qubits * n_layers * 3  # Ry, Rz, Ry per qubit per layer
        self.params = np.random.randn(self.n_params) * 0.1

    def encode_data(self, x: np.ndarray) -> np.ndarray:
        """Encode classical data into quantum state."""
        if self.encoding == 'angle':
            # Pad or truncate to n_qubits
            x_padded = np.zeros(self.n_qubits)
            x_padded[:min(len(x), self.n_qubits)] = x[:self.n_qubits]
            return angle_encoding(x_padded * np.pi)
        else:
            return amplitude_encoding(x)

    def apply_variational_layer(
        self,
        state: np.ndarray,
        params: np.ndarray
    ) -> np.ndarray:
        """Apply one variational layer."""
        n = self.n_qubits
        param_idx = 0

        # Single-qubit rotations
        for qubit in range(n):
            # Apply Ry-Rz-Ry sequence
            for gate_type in ['Ry', 'Rz', 'Ry']:
                angle = params[param_idx]
                param_idx += 1
                state = self._apply_rotation(state, qubit, gate_type, angle)

        # Entangling layer (CNOT ladder)
        for qubit in range(n - 1):
            state = self._apply_cnot(state, qubit, qubit + 1)

        return state

    def _apply_rotation(
        self,
        state: np.ndarray,
        qubit: int,
        gate_type: str,
        angle: float
    ) -> np.ndarray:
        """Apply single-qubit rotation."""
        if gate_type == 'Ry':
            gate = np.array([
                [np.cos(angle/2), -np.sin(angle/2)],
                [np.sin(angle/2), np.cos(angle/2)]
            ])
        else:  # Rz
            gate = np.array([
                [np.exp(-1j * angle/2), 0],
                [0, np.exp(1j * angle/2)]
            ])

        return self._apply_single_qubit_gate(state, gate, qubit)

    def _apply_single_qubit_gate(
        self,
        state: np.ndarray,
        gate: np.ndarray,
        qubit: int
    ) -> np.ndarray:
        """Apply gate to specific qubit."""
        n = self.n_qubits
        dim = 2 ** n
        new_state = np.zeros(dim, dtype=complex)

        for i in range(dim):
            qubit_val = (i >> (n - 1 - qubit)) & 1
            i_flipped = i ^ (1 << (n - 1 - qubit))

            if qubit_val == 0:
                new_state[i] += gate[0, 0] * state[i] + gate[0, 1] * state[i_flipped]
            else:
                new_state[i] += gate[1, 0] * state[i_flipped] + gate[1, 1] * state[i]

        return new_state

    def _apply_cnot(
        self,
        state: np.ndarray,
        control: int,
        target: int
    ) -> np.ndarray:
        """Apply CNOT gate."""
        n = self.n_qubits
        new_state = state.copy()

        for i in range(2 ** n):
            control_val = (i >> (n - 1 - control)) & 1
            if control_val == 1:
                j = i ^ (1 << (n - 1 - target))
                new_state[i], new_state[j] = state[j], state[i]

        return new_state

    def forward(self, x: np.ndarray) -> float:
        """
        Forward pass: encode data and apply variational circuit.

        Returns:
            Expectation value of Z on first qubit
        """
        # Encode data
        state = self.encode_data(x)

        # Apply variational layers
        params_per_layer = self.n_qubits * 3
        for layer in range(self.n_layers):
            layer_params = self.params[layer * params_per_layer:(layer + 1) * params_per_layer]
            state = self.apply_variational_layer(state, layer_params)

        # Measure expectation of Z on first qubit
        return self._measure_z_expectation(state, 0)

    def _measure_z_expectation(self, state: np.ndarray, qubit: int) -> float:
        """Measure <Z> expectation on specified qubit."""
        n = self.n_qubits
        expectation = 0.0

        for i in range(2 ** n):
            qubit_val = (i >> (n - 1 - qubit)) & 1
            sign = 1 if qubit_val == 0 else -1
            expectation += sign * np.abs(state[i]) ** 2

        return expectation

    def predict(self, x: np.ndarray) -> int:
        """Predict class label."""
        exp_val = self.forward(x)
        return 0 if exp_val > 0 else 1

    def predict_proba(self, x: np.ndarray) -> np.ndarray:
        """Predict class probabilities."""
        exp_val = self.forward(x)
        # Map [-1, 1] to [0, 1]
        prob_class_0 = (exp_val + 1) / 2
        return np.array([prob_class_0, 1 - prob_class_0])

# Example usage
vqc = VariationalQuantumClassifier(n_qubits=4, n_layers=2)
x = np.array([0.5, 0.3, 0.7, 0.2])
prediction = vqc.predict(x)
probs = vqc.predict_proba(x)
print(f"Prediction: {prediction}")
print(f"Probabilities: {probs}")
```

### 3.2 Training the VQC

```python
from scipy.optimize import minimize

def train_vqc(
    vqc: VariationalQuantumClassifier,
    X_train: np.ndarray,
    y_train: np.ndarray,
    max_iterations: int = 100,
    verbose: bool = True
) -> dict:
    """
    Train VQC using gradient-free optimization.

    Args:
        vqc: VQC model
        X_train: Training features
        y_train: Training labels (0 or 1)
        max_iterations: Maximum optimizer iterations
        verbose: Print progress

    Returns:
        Training history
    """
    history = {'loss': [], 'accuracy': []}

    def loss_function(params):
        vqc.params = params
        total_loss = 0.0
        correct = 0

        for x, y in zip(X_train, y_train):
            exp_val = vqc.forward(x)
            # Hinge-like loss: want exp_val > 0 for y=0, exp_val < 0 for y=1
            target = 1 if y == 0 else -1
            loss = max(0, 1 - target * exp_val)
            total_loss += loss

            pred = 0 if exp_val > 0 else 1
            if pred == y:
                correct += 1

        avg_loss = total_loss / len(X_train)
        accuracy = correct / len(X_train)

        history['loss'].append(avg_loss)
        history['accuracy'].append(accuracy)

        if verbose and len(history['loss']) % 10 == 0:
            print(f"Iteration {len(history['loss'])}: Loss={avg_loss:.4f}, Acc={accuracy:.4f}")

        return avg_loss

    # Optimize
    result = minimize(
        loss_function,
        vqc.params,
        method='COBYLA',
        options={'maxiter': max_iterations}
    )

    vqc.params = result.x
    return history

# Example with synthetic data
np.random.seed(42)
n_samples = 50
X_train = np.random.randn(n_samples, 4) * 0.5
y_train = (X_train[:, 0] + X_train[:, 1] > 0).astype(int)

vqc = VariationalQuantumClassifier(n_qubits=4, n_layers=2)
history = train_vqc(vqc, X_train, y_train, max_iterations=50, verbose=True)
```

## 4. Quantum Kernel Methods

Quantum kernels provide an alternative approach where the quantum computer computes similarity measures in a quantum feature space.

<Definition term="Quantum Kernel">
A kernel function $k(x, x') = |\langle\phi(x)|\phi(x')\rangle|^2$ where $|\phi(x)\rangle = U_\phi(x)|0\rangle$ is a quantum feature map.
</Definition>

### 4.1 Computing Quantum Kernels

```python
class QuantumKernel:
    """
    Quantum kernel computation using various feature maps.
    """

    def __init__(
        self,
        n_qubits: int,
        feature_map: str = 'iqp',
        reps: int = 2
    ):
        self.n_qubits = n_qubits
        self.feature_map = feature_map
        self.reps = reps

    def _apply_feature_map(self, x: np.ndarray) -> np.ndarray:
        """Apply feature map to get quantum state."""
        n = self.n_qubits

        # Start with |+>^n state (after Hadamards)
        state = np.ones(2**n, dtype=complex) / np.sqrt(2**n)

        for rep in range(self.reps):
            # Z rotations
            for i in range(n):
                if i < len(x):
                    state = self._apply_rz(state, i, 2 * x[i])

            # ZZ interactions for IQP
            if self.feature_map == 'iqp':
                for i in range(n):
                    for j in range(i + 1, n):
                        if i < len(x) and j < len(x):
                            state = self._apply_rzz(state, i, j, 2 * x[i] * x[j])

            # Hadamard layer (except last rep)
            if rep < self.reps - 1:
                for i in range(n):
                    state = self._apply_hadamard(state, i)

        return state

    def _apply_rz(self, state: np.ndarray, qubit: int, angle: float) -> np.ndarray:
        """Apply RZ gate."""
        n = self.n_qubits
        new_state = state.copy()

        for i in range(2**n):
            qubit_val = (i >> (n - 1 - qubit)) & 1
            phase = -angle/2 if qubit_val == 0 else angle/2
            new_state[i] *= np.exp(1j * phase)

        return new_state

    def _apply_rzz(
        self,
        state: np.ndarray,
        q1: int,
        q2: int,
        angle: float
    ) -> np.ndarray:
        """Apply RZZ gate."""
        n = self.n_qubits
        new_state = state.copy()

        for i in range(2**n):
            q1_val = (i >> (n - 1 - q1)) & 1
            q2_val = (i >> (n - 1 - q2)) & 1
            parity = q1_val ^ q2_val
            phase = angle/2 if parity == 0 else -angle/2
            new_state[i] *= np.exp(1j * phase)

        return new_state

    def _apply_hadamard(self, state: np.ndarray, qubit: int) -> np.ndarray:
        """Apply Hadamard gate."""
        n = self.n_qubits
        new_state = np.zeros(2**n, dtype=complex)

        for i in range(2**n):
            qubit_val = (i >> (n - 1 - qubit)) & 1
            partner = i ^ (1 << (n - 1 - qubit))

            if qubit_val == 0:
                new_state[i] += (state[i] + state[partner]) / np.sqrt(2)
            else:
                new_state[i] += (state[partner] - state[i]) / np.sqrt(2)

        return new_state

    def kernel(self, x1: np.ndarray, x2: np.ndarray) -> float:
        """Compute kernel value between two data points."""
        state1 = self._apply_feature_map(x1)
        state2 = self._apply_feature_map(x2)

        overlap = np.abs(np.vdot(state1, state2)) ** 2
        return overlap

    def compute_kernel_matrix(self, X: np.ndarray) -> np.ndarray:
        """Compute full kernel matrix."""
        n_samples = len(X)
        K = np.zeros((n_samples, n_samples))

        for i in range(n_samples):
            for j in range(i, n_samples):
                K[i, j] = self.kernel(X[i], X[j])
                K[j, i] = K[i, j]

        return K

# Example
qk = QuantumKernel(n_qubits=3, feature_map='iqp', reps=2)
X = np.random.randn(10, 3) * 0.5
K = qk.compute_kernel_matrix(X)
print(f"Kernel matrix shape: {K.shape}")
print(f"Kernel matrix (first 5x5):\n{K[:5, :5]}")
```

### 4.2 Quantum Kernel SVM

```python
from scipy.linalg import solve

class QuantumSVM:
    """
    Support Vector Machine with quantum kernel.
    """

    def __init__(self, quantum_kernel: QuantumKernel, C: float = 1.0):
        self.kernel = quantum_kernel
        self.C = C
        self.alpha = None
        self.X_train = None
        self.y_train = None
        self.b = 0

    def fit(self, X: np.ndarray, y: np.ndarray):
        """
        Train the quantum SVM.

        Uses simplified dual formulation.
        """
        self.X_train = X
        self.y_train = y
        n = len(X)

        # Compute kernel matrix
        K = self.kernel.compute_kernel_matrix(X)

        # Simplified: solve for alpha using kernel ridge regression
        # (A proper SVM would use quadratic programming)
        y_converted = 2 * y - 1  # Convert to +1/-1

        # Add regularization
        regularized_K = K + np.eye(n) / self.C
        self.alpha = solve(regularized_K, y_converted)

        # Compute bias
        predictions = K @ self.alpha
        self.b = np.mean(y_converted - predictions)

    def predict(self, X: np.ndarray) -> np.ndarray:
        """Predict class labels."""
        n_test = len(X)
        predictions = np.zeros(n_test)

        for i in range(n_test):
            score = self.b
            for j in range(len(self.X_train)):
                k_val = self.kernel.kernel(X[i], self.X_train[j])
                score += self.alpha[j] * k_val
            predictions[i] = 1 if score > 0 else 0

        return predictions.astype(int)

    def score(self, X: np.ndarray, y: np.ndarray) -> float:
        """Compute accuracy."""
        predictions = self.predict(X)
        return np.mean(predictions == y)

# Example
qk = QuantumKernel(n_qubits=4, feature_map='iqp')
qsvm = QuantumSVM(qk, C=1.0)
qsvm.fit(X_train, y_train)
print(f"Training accuracy: {qsvm.score(X_train, y_train):.4f}")
```

## 5. Circuit Expressibility and Trainability

Understanding the balance between expressibility and trainability is crucial for practical QML.

<Formula>
$$\text{Expressibility} = D_{KL}(\hat{P}_{circuit} || P_{Haar})$$
</Formula>

```python
def compute_expressibility(
    circuit_sampler: Callable,
    n_qubits: int,
    n_samples: int = 1000,
    n_bins: int = 50
) -> dict:
    """
    Estimate circuit expressibility via fidelity distribution.

    Args:
        circuit_sampler: Function that returns random circuit output state
        n_qubits: Number of qubits
        n_samples: Number of sample pairs
        n_bins: Histogram bins

    Returns:
        Expressibility metrics
    """
    fidelities = []

    for _ in range(n_samples):
        # Generate two random parameter sets
        state1 = circuit_sampler()
        state2 = circuit_sampler()

        # Compute fidelity
        fidelity = np.abs(np.vdot(state1, state2)) ** 2
        fidelities.append(fidelity)

    fidelities = np.array(fidelities)

    # Haar-random fidelity distribution: P(F) = (d-1)(1-F)^{d-2}
    d = 2 ** n_qubits

    # Compare histograms
    hist, edges = np.histogram(fidelities, bins=n_bins, range=(0, 1), density=True)
    centers = (edges[:-1] + edges[1:]) / 2

    # Theoretical Haar distribution
    haar_dist = (d - 1) * (1 - centers) ** (d - 2)

    # KL divergence (approximate)
    hist_safe = np.maximum(hist, 1e-10)
    haar_safe = np.maximum(haar_dist, 1e-10)
    kl_div = np.sum(hist_safe * np.log(hist_safe / haar_safe)) * (edges[1] - edges[0])

    return {
        'kl_divergence': kl_div,
        'mean_fidelity': np.mean(fidelities),
        'std_fidelity': np.std(fidelities),
        'expressibility_score': 1 / (1 + kl_div)  # Higher is better
    }
```

## 6. Best Practices for NISQ QML

<Warning>
NISQ devices have significant noise and limited qubits. Following best practices is essential for obtaining meaningful results.
</Warning>

### 6.1 Feature Preprocessing

```python
def preprocess_for_qml(
    X: np.ndarray,
    method: str = 'standardize',
    target_range: Tuple[float, float] = (0, np.pi)
) -> np.ndarray:
    """
    Preprocess features for QML encoding.

    Args:
        X: Feature matrix (n_samples, n_features)
        method: 'standardize', 'normalize', or 'pca'
        target_range: Range to scale features to

    Returns:
        Preprocessed features
    """
    X_processed = X.copy()

    if method == 'standardize':
        # Zero mean, unit variance
        mean = X_processed.mean(axis=0)
        std = X_processed.std(axis=0) + 1e-10
        X_processed = (X_processed - mean) / std

    elif method == 'normalize':
        # Min-max normalization
        X_min = X_processed.min(axis=0)
        X_max = X_processed.max(axis=0)
        X_processed = (X_processed - X_min) / (X_max - X_min + 1e-10)

    elif method == 'pca':
        # PCA for dimensionality reduction
        mean = X_processed.mean(axis=0)
        X_centered = X_processed - mean
        cov = X_centered.T @ X_centered / len(X)
        eigenvalues, eigenvectors = np.linalg.eigh(cov)
        X_processed = X_centered @ eigenvectors[:, ::-1]

    # Scale to target range
    X_min = X_processed.min()
    X_max = X_processed.max()
    X_processed = (X_processed - X_min) / (X_max - X_min + 1e-10)
    X_processed = X_processed * (target_range[1] - target_range[0]) + target_range[0]

    return X_processed

# Example

print(f"Raw range: [{X_raw.min():.2f}, {X_raw.max():.2f}]")
print(f"Preprocessed range: [{X_preprocessed.min():.2f}, {X_preprocessed.max():.2f}]")
```

### 6.2 Noise Mitigation Strategies

```python
def apply_error_mitigation(
    expectation_values: List[float],
    method: str = 'zero_noise_extrapolation'
) -> float:
    """
    Apply error mitigation to expectation value estimates.

    Args:
        expectation_values: List of values at different noise levels
        method: Mitigation method

    Returns:
        Mitigated expectation value
    """
    if method == 'zero_noise_extrapolation':
        # Assume linear noise dependence
        # expectation_values[0] = 1x noise
        # expectation_values[1] = 2x noise (stretched gates)
        if len(expectation_values) >= 2:
            # Linear extrapolation to zero noise
            e1, e2 = expectation_values[0], expectation_values[1]
            # e1 = e0 + noise * c
            # e2 = e0 + 2*noise * c
            # => e0 = 2*e1 - e2
            return 2 * e1 - e2
        else:
            return expectation_values[0]

    elif method == 'measurement_error_mitigation':
        # Would apply calibration matrix

        return expectation_values[0]

    return expectation_values[0]
```

```python
def qml_training_pipeline(
    X_train: np.ndarray,
    y_train: np.ndarray,
    X_test: np.ndarray,
    y_test: np.ndarray,
    n_qubits: int = 4,
    n_layers: int = 2,
    max_iter: int = 100
) -> dict:
    """
    Complete QML training pipeline with best practices.
    """
    results = {}

    # 1. Preprocess data
    X_train_processed = preprocess_for_qml(X_train)
    X_test_processed = preprocess_for_qml(X_test)
    results['preprocessing'] = 'standardize'

    # 2. Dimensionality reduction if needed
    if X_train.shape[1] > n_qubits:
        print(f"Reducing dimensions from {X_train.shape[1]} to {n_qubits}")
        # Simple PCA
        mean = X_train_processed.mean(axis=0)
        X_centered = X_train_processed - mean
        cov = X_centered.T @ X_centered
        eigenvalues, eigenvectors = np.linalg.eigh(cov)
        top_components = eigenvectors[:, -n_qubits:]

        X_train_processed = X_centered @ top_components
        X_test_processed = (X_test_processed - mean) @ top_components

    # 3. Initialize VQC
    vqc = VariationalQuantumClassifier(
        n_qubits=n_qubits,
        n_layers=n_layers
    )

    # 4. Train
    print("Training VQC...")
    history = train_vqc(
        vqc,
        X_train_processed[:, :n_qubits],
        y_train,
        max_iterations=max_iter,
        verbose=False
    )
    results['training_history'] = history

    # 5. Evaluate
    train_preds = [vqc.predict(x) for x in X_train_processed[:, :n_qubits]]
    test_preds = [vqc.predict(x) for x in X_test_processed[:, :n_qubits]]

    results['train_accuracy'] = np.mean(train_preds == y_train)
    results['test_accuracy'] = np.mean(test_preds == y_test)

    print(f"Train accuracy: {results['train_accuracy']:.4f}")
    print(f"Test accuracy: {results['test_accuracy']:.4f}")

    return results

# Example usage
np.random.seed(42)
```

## 7. Comparing with Classical Baselines

Always compare QML models with classical baselines to assess practical value.

```python
def compare_with_classical(
    X_train: np.ndarray,
    y_train: np.ndarray,
    X_test: np.ndarray,
    y_test: np.ndarray,
    qml_accuracy: float
) -> dict:
    """
    Compare QML results with classical baselines.
    """
    results = {'qml': qml_accuracy}

    # Logistic regression (simple linear)
    X_mean = X_train.mean(axis=0)
    X_std = X_train.std(axis=0) + 1e-10
    X_train_norm = (X_train - X_mean) / X_std
    X_test_norm = (X_test - X_mean) / X_std

    # Simple gradient descent
    w = np.zeros(X_train_norm.shape[1])
    b = 0
    lr = 0.1

    for _ in range(100):
        y_pred = 1 / (1 + np.exp(-(X_train_norm @ w + b)))
        grad_w = X_train_norm.T @ (y_pred - y_train) / len(y_train)
        grad_b = np.mean(y_pred - y_train)
        w -= lr * grad_w
        b -= lr * grad_b

    test_pred = (1 / (1 + np.exp(-(X_test_norm @ w + b))) > 0.5).astype(int)
    results['logistic_regression'] = np.mean(test_pred == y_test)

    # Random baseline
    results['random'] = 0.5

    # Most frequent class
    most_common = np.argmax(np.bincount(y_train))
    results['most_frequent'] = np.mean(y_test == most_common)

    print("\n=== Model Comparison ===")
    for model, acc in results.items():
        better = " (best)" if acc == max(results.values()) else ""
        print(f"{model}: {acc:.4f}{better}")

    return results
```
