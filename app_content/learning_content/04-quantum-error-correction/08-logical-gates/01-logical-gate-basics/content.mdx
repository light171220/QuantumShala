# Logical Gate Fundamentals

## Introduction

Quantum error correction protects information, but useful computation requires applying gates to that information. This lesson introduces the fundamental concepts of logical gatesвҖ”operations on encoded qubits that must be implemented without breaking fault tolerance.

## What Are Logical Gates?

```
Logical Gates: The Basic Concept
в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

PHYSICAL GATE:
    Acts on physical qubits directly
    U: |ПҲвҹ© вҶ’ U|ПҲвҹ©

LOGICAL GATE:
    Acts on encoded (logical) qubits
    U_L: |ПҲ_Lвҹ© вҶ’ U|ПҲвҹ©_L

    The gate must:
    1. Transform the logical state correctly
    2. Keep the state in the code space
    3. Maintain fault tolerance

EXAMPLE (Steane Code):
    Physical X gate: Flips one physical qubit
    Logical X gate: Flips the logical qubit

    |0_Lвҹ© в”Җв”ҖX_Lв”Җв”Җв–ә |1_Lвҹ©

    But X_L is NOT just X on one qubit!
    X_L = XвӮҖXвӮҒXвӮӮXвӮғXвӮ„XвӮ…XвӮҶ (on all 7 qubits)
```

```python
PAULI GATES: X, Y, Z

        """Describe Clifford gates."""

CLIFFORD GATES: H, S, CNOT, (and all Paulis)
в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

    - All Pauli gates

    Surface: Limited transversal Cliffords

NON-CLIFFORD GATES: T, Toffoli, and general rotations

    Clifford + T = Universal gate set
    T is the "missing piece" for universality

IMPLEMENTATION CHALLENGE:
```

## Code Structure and Gate Support

```
Which Gates Does a Code Support?
в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

Different codes support different transversal gates:

CODE                    TRANSVERSAL GATES
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
[[7,1,3]] Steane        X, Z, H, S, CNOT (full Clifford!)
[[9,1,3]] Shor          X, Z, CNOT
[[5,1,3]] Perfect       X, Z (limited)
Surface Code            X, Z, CNOT (limited Clifford)
[[15,1,3]] Reed-Muller  X, Z, H, T (!), but not S
Color Codes             X, Z, H, S, CNOT (Clifford)

KEY INSIGHT: No single code supports ALL gates transversally!

EASTIN-KNILL THEOREM:
    For any stabilizer code:
    Cannot have transversal universal gate set

    Why? Transversal gates form a finite group
    Universal gates require continuous rotations


STRATEGIES TO OVERCOME:

1. CODE SWITCHING
    Use different codes for different gates
    Switch between codes as needed

2. MAGIC STATE INJECTION
    Use special "magic states" to implement T
    Inject noisy states, distill to high fidelity

3. NON-TRANSVERSAL IMPLEMENTATIONS
    Lattice surgery, code deformation
    Still fault-tolerant, but not transversal
```

## Summary

Logical gates are the key to computation on error-corrected qubits:

1. **Logical vs Physical**: Logical gates transform encoded information
2. **Fault Tolerance**: Gates must not spread errors beyond correction capacity
3. **Gate Classification**: Paulis (easy) вҶ’ Clifford (medium) вҶ’ non-Clifford (hard)
4. **Eastin-Knill Limit**: No code has a transversal universal gate set
5. **Implementation Methods**: Transversal, magic states, lattice surgery
