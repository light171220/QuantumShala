# Magic State Injection

## Introduction

Magic states are special quantum states that, combined with Clifford operations, enable universal quantum computation. This lesson covers how magic states work, how to inject them into computations, and how to purify noisy magic states through distillation.

## What Are Magic States?

```
Magic States: The Key to Universality
═════════════════════════════════════

PROBLEM:
    Clifford gates are not universal
    T gate is needed, but NOT transversal

SOLUTION:
    Pre-prepare a special state |T⟩
    Use Clifford operations to "inject" the T gate

MAGIC STATES:

|T⟩ = T|+⟩ = (|0⟩ + e^(iπ/4)|1⟩)/√2

    This state "contains" the T gate!

|H⟩ = (cos(π/8)|0⟩ + sin(π/8)|1⟩)

    Also known as |A⟩, enables π/8 rotations

|Y⟩ = (|0⟩ + i|1⟩)/√2 = S|+⟩

    For S gate (if not transversal)


VISUALIZATION on Bloch sphere:

         Z
         │
    |0⟩  ●
         │\
         │ \  ← |T⟩ is here
         │  \    (45° from X axis in XY plane)
    ─────┼───●── Y
         │  /
         │ /
    |1⟩  ●
         │
         X

|T⟩ points at angle π/4 from the +X axis in the XY plane.
```

## Gate Teleportation

```python
class GateTeleportation:
    """
    Implement gate teleportation using magic states.
    """

    def __init__(self):
        pass

    def t_gate_teleportation(self):
        """T gate via teleportation."""
        print("""
T Gate Teleportation
════════════════════

CIRCUIT:

    |ψ⟩ ─────────●────────M──►──┬── S†Z ─── T|ψ⟩
                 │              │
    |T⟩ = T|+⟩ ──X────────────  └── if m=1

STEPS:
    1. Start with data qubit |ψ⟩ and magic state |T⟩
    2. Apply CNOT: data controls, magic is target
    3. Measure data qubit in X basis (apply H then measure Z)
    4. If measurement = 1, apply S†Z correction to magic qubit
    5. Result: magic qubit now holds T|ψ⟩

WHY IT WORKS:

    Initial: |ψ⟩ ⊗ |T⟩ = (α|0⟩ + β|1⟩) ⊗ (|0⟩ + e^(iπ/4)|1⟩)/√2

    After CNOT:
    α|0⟩(|0⟩ + e^(iπ/4)|1⟩)/√2 + β|1⟩(|1⟩ + e^(iπ/4)|0⟩)/√2

    After H on data:
    |+⟩ ⊗ (α|0⟩ + βe^(iπ/4)|1⟩)/√2
  + |−⟩ ⊗ (α|1⟩ + βe^(iπ/4)|0⟩)/√2

    Measure data:
    m=0: magic = α|0⟩ + βe^(iπ/4)|1⟩ = T|ψ⟩ ✓
    m=1: magic = α|1⟩ + βe^(iπ/4)|0⟩ = S†Z T|ψ⟩
         Apply S†Z: get T|ψ⟩ ✓
        """)

    def s_gate_teleportation(self):
        """S gate via teleportation."""
        print("""
S Gate Teleportation
════════════════════

For codes where S is not transversal, use |Y⟩ = S|+⟩:

CIRCUIT:

    |ψ⟩ ──────●────────M──►──┬── Z ─── S|ψ⟩
              │              │
    |Y⟩ ──────X──────────────└── if m=1

CORRECTION:
    m=0: result is S|ψ⟩
    m=1: apply Z to get S|ψ⟩

SIMPLER than T gate (Pauli correction only)!
        """)

    def general_rotation(self):
        """Arbitrary rotations from T gates."""
        print("""
Arbitrary Rotations from T Gates
════════════════════════════════

Any single-qubit rotation can be approximated using Clifford + T.

SOLOVAY-KITAEV THEOREM:
    Any rotation can be ε-approximated with O(log^c(1/ε)) gates
    (c ≈ 3.97, improved algorithms give c ≈ 1-2)

EXAMPLE: Approximate R_z(θ)
    R_z(θ) ≈ Clifford₁ · T · Clifford₂ · T · ... · Cliffordₖ · T

    For θ = π/16:
    R_z(π/16) ≈ HTH·T·HTH  (approximate)

RESOURCE COST:
    Target precision: ε
    T gates needed: O(log(1/ε))
    Each T needs one magic state

    For ε = 10⁻¹⁰: need ~100-200 T gates per rotation
        """)


teleport = GateTeleportation()
teleport.t_gate_teleportation()
teleport.s_gate_teleportation()
teleport.general_rotation()
```

## Noisy Magic States

```
The Noise Problem
═════════════════

IDEAL:
    |T⟩ = T|+⟩ = (|0⟩ + e^(iπ/4)|1⟩)/√2

REALITY:
    Preparation is noisy!
    ρ_noisy = (1-p)|T⟩⟨T| + p·noise

PROBLEM:
    Noisy |T⟩ → noisy T gate
    If |T⟩ has error p, T|ψ⟩ has error ~p
    Error propagates into computation!


SOURCES OF NOISE:

1. Direct Preparation
    Prepare physical |T⟩, encode into code
    Physical prep error: ~1-10%
    This is TOO NOISY for fault tolerance

2. Non-transversal T (if available)
    Apply T directly to encoded qubit
    But T is not transversal → spreads errors
    Also too noisy!

SOLUTION: MAGIC STATE DISTILLATION
    Take many noisy magic states
    Use Clifford operations to produce fewer, cleaner states
    Repeat until desired purity achieved
```

## Magic State Distillation

```python
class MagicStateDistillation:
    """
    Implement magic state distillation protocols.
    """

    def __init__(self):
        pass

    def fifteen_to_one(self):
        """The 15-to-1 distillation protocol."""
        print("""
15-to-1 Magic State Distillation
════════════════════════════════

INPUT: 15 noisy |T⟩ states with error rate p_in
OUTPUT: 1 cleaner |T⟩ state with error rate p_out

PROTOCOL:
    1. Encode the 15 states in [[15,1,3]] Reed-Muller code
    2. Apply transversal T to get logical |T_L⟩
    3. Measure syndromes
    4. If syndrome = 0: SUCCESS (decoded state is clean)
       If syndrome ≠ 0: FAILURE (discard and retry)

WHY [[15,1,3]]?
    Special code where T IS transversal!
    T^⊗15 = T_L (logical T)

ERROR ANALYSIS:
    p_out ≈ 35 · p_in³

    If p_in = 1%:   p_out ≈ 35 × (0.01)³ = 3.5 × 10⁻⁵
    If p_in = 0.1%: p_out ≈ 35 × (0.001)³ = 3.5 × 10⁻⁸

SUCCESS PROBABILITY:
    P_success ≈ (1 - p_in)^15 ≈ 1 - 15p_in

    For p_in = 1%: P_success ≈ 85%

CIRCUIT OVERVIEW:
    ┌─────────────────────────────────────────────────┐
    │  15 noisy |T⟩ states                            │
    │  |T̃₁⟩, |T̃₂⟩, ..., |T̃₁₅⟩                         │
    └─────────────────────────────────────────────────┘
                           │
                           ▼
    ┌─────────────────────────────────────────────────┐
    │  Encode in [[15,1,3]] Reed-Muller               │
    │  Apply stabilizer checks                        │
    └─────────────────────────────────────────────────┘
                           │
                           ▼
    ┌────────────────┐     ┌───────────────────────────┐
    │  Syndrome = 0  │────►│ Output: clean |T_L⟩       │
    └────────────────┘     │ Error rate: 35p_in³       │
            │              └───────────────────────────┘
            │ Syndrome ≠ 0
            ▼
    ┌────────────────┐
    │ DISCARD, retry │
    └────────────────┘
        """)

    def concatenated_distillation(self):
        """Multi-level distillation for high purity."""
        print("""
Concatenated Distillation
═════════════════════════

For very low error rates, repeat distillation:

LEVEL 1:
    Input: 15 states at error p
    Output: 1 state at error 35p³

LEVEL 2:
    Input: 15 states at error 35p³ (need 15 × 15 = 225 raw states)
    Output: 1 state at error 35(35p³)³ = 35⁴ p⁹

LEVEL k:
    Error: O(p^(3^k))
    Raw states needed: 15^k

EXAMPLE (p = 1%, target 10⁻¹⁵):

    Level 1: 3.5 × 10⁻⁵
    Level 2: 1.5 × 10⁻¹²
    Level 3: 1.2 × 10⁻³⁴ ✓

    States needed: 15³ = 3,375 raw states per output


OPTIMIZATIONS:
    Different protocols at different levels
    Use 7-to-1 for higher error rates
    Use 15-to-1 for lower error rates
        """)

    def resource_analysis(self):
        """Analyze distillation resource costs."""
        print("""
Distillation Resource Costs
═══════════════════════════

The cost of T gates dominates fault-tolerant computation!

METRICS:
    Raw magic states per distilled state
    Qubits × time = space-time volume
    Success probability

15-TO-1 COSTS:
    Raw states: 15 per output
    Qubits: ~15n (where n is code block size)
    Time: ~d syndrome rounds
    Success: ~85% at p=1%

FULL COMPUTATION ESTIMATE:
    Algorithm needs: 10⁶ T gates
    Distillation ratio: ~10³ raw per output
    Total raw states: ~10⁹

    This is often the dominant cost!

IMPROVEMENTS:
    - Better codes (e.g., [[7,1,3]] for early rounds)
    - Parallel distillation factories
    - Optimized scheduling
    - State injection without full distillation
        """)


distillation = MagicStateDistillation()
distillation.fifteen_to_one()
distillation.concatenated_distillation()
distillation.resource_analysis()
```

## Summary

Magic state injection is the key to universal fault-tolerant computation:

1. **Magic States**: Special states like |T⟩ = T|+⟩ that encode non-Clifford gates
2. **Gate Teleportation**: Use Clifford operations to extract the gate from the state
3. **Distillation**: Purify noisy magic states using protocols like 15-to-1
4. **Factories**: Continuous production pipeline for high-purity magic states
5. **Injection**: Integrate distilled states into encoded computation

Magic states are the most expensive resource in fault-tolerant quantum computing.
