## 1. What is Decoherence?

Decoherence is the process by which a quantum system loses its quantum properties - particularly superposition and entanglement - through interactions with its environment.

<Definition term="Decoherence">
Decoherence is the irreversible process by which quantum superpositions evolve into classical statistical mixtures due to entanglement with environmental degrees of freedom. It explains why we don't observe quantum effects in everyday macroscopic objects.
</Definition>

Consider a qubit in superposition that interacts with environment:

<Formula>
$$|\psi\rangle_{sys} \otimes |E_0\rangle_{env} \rightarrow \alpha|0\rangle|E_0\rangle + \beta|1\rangle|E_1\rangle$$
</Formula>

If we trace out the environment (because we can't measure it), the system appears mixed:

```python
import numpy as np

def demonstrate_decoherence(alpha, beta, env_overlap):
    """
    Demonstrate how environmental entanglement causes decoherence.

    Args:
        alpha, beta: Superposition amplitudes
        env_overlap: <E_0|E_1> - overlap of environment states

    Returns:
        Reduced density matrix of system
    """
    # Full system-environment state
    # |psi> = alpha|0>|E_0> + beta|1>|E_1>

    # Reduced density matrix after tracing out environment
    # rho_sys = |alpha|^2 |0><0| + |beta|^2 |1><1|
    #         + alpha*beta^* <E_1|E_0> |0><1| + c.c.

    rho = np.array([
        [np.abs(alpha)**2, alpha * np.conj(beta) * env_overlap],
        [np.conj(alpha) * beta * np.conj(env_overlap), np.abs(beta)**2]
    ])

    return rho

def purity(rho):
    return np.real(np.trace(rho @ rho))

# Initially orthogonal environment states (full decoherence)
alpha = 1/np.sqrt(2)
beta = 1/np.sqrt(2)

print("Effect of environmental distinguishability on coherence:")
print("-" * 55)
for overlap in [1.0, 0.8, 0.5, 0.2, 0.0]:
    rho = demonstrate_decoherence(alpha, beta, overlap)
    off_diag = np.abs(rho[0, 1])
    p = purity(rho)
    print(f"<E_0|E_1> = {overlap:.1f}: Off-diagonal = {off_diag:.3f}, Purity = {p:.3f}")
```

<Tip>
The key insight is that decoherence is not about energy loss, but about information leaking into the environment. When the environment "records" which state the system is in, quantum interference is destroyed.
</Tip>

## 2. T1: Energy Relaxation

T1 characterizes how quickly an excited qubit decays to its ground state, releasing energy to the environment.

<Definition term="T1 Relaxation Time">
T1 (also called longitudinal relaxation time) is the characteristic time for an excited qubit state to decay to the ground state. The probability of finding the qubit in the excited state decays as $e^{-t/T_1}$.
</Definition>

```python
import numpy as np

def amplitude_damping_channel(rho, gamma):
    """
    Apply amplitude damping channel.

    Args:
        rho: Input density matrix
        gamma: Damping parameter (0 to 1)
              gamma = 1 - exp(-t/T1)

    Returns:
        Output density matrix
    """
    # Kraus operators for amplitude damping
    K0 = np.array([[1, 0], [0, np.sqrt(1 - gamma)]])
    K1 = np.array([[0, np.sqrt(gamma)], [0, 0]])

    rho_out = K0 @ rho @ K0.conj().T + K1 @ rho @ K1.conj().T
    return rho_out

def t1_decay(t, T1):
    """Calculate damping parameter from time and T1."""
    return 1 - np.exp(-t / T1)

# Simulate T1 decay of excited state
T1 = 100e-6  # 100 microseconds
times = np.linspace(0, 5*T1, 100)

# Start in |1> state
rho_excited = np.array([[0, 0], [0, 1]], dtype=complex)

print("T1 relaxation of |1> state:")
print("-" * 45)
for t in [0, T1, 2*T1, 3*T1, 5*T1]:
    gamma = t1_decay(t, T1)
    rho_t = amplitude_damping_channel(rho_excited, gamma)
    p_excited = np.real(rho_t[1, 1])
    print(f"t = {t/T1:.0f}*T1: P(excited) = {p_excited:.4f}")
```

### Physical Mechanisms

T1 decay occurs through several mechanisms:

1. **Spontaneous emission**: Qubit emits a photon
2. **Purcell effect**: Coupling to resonator modes
3. **Quasiparticle tunneling**: In superconducting qubits
4. **Phonon emission**: In solid-state qubits

```python
import numpy as np

def purcell_limit(qubit_freq, cavity_freq, coupling, cavity_decay):
    """
    Calculate Purcell-limited T1.

    Args:
        qubit_freq: Qubit frequency (Hz)
        cavity_freq: Cavity frequency (Hz)
        coupling: Qubit-cavity coupling (Hz)
        cavity_decay: Cavity decay rate (Hz)

    Returns:
        Purcell-limited T1 in seconds
    """
    detuning = qubit_freq - cavity_freq

    # Purcell decay rate
    gamma_purcell = (coupling / detuning)**2 * cavity_decay

    T1_purcell = 1 / gamma_purcell
    return T1_purcell

# Example: Superconducting transmon
qubit_freq = 5e9  # 5 GHz
cavity_freq = 7e9  # 7 GHz
coupling = 100e6  # 100 MHz
cavity_decay = 1e6  # 1 MHz (Q ~ 7000)

T1_purcell = purcell_limit(qubit_freq, cavity_freq, coupling, cavity_decay)
print(f"Purcell-limited T1: {T1_purcell*1e6:.1f} microseconds")
```

## 3. T2: Phase Coherence

T2 characterizes how quickly the phase relationship in a superposition is lost.

<Definition term="T2 Dephasing Time">
T2 (also called transverse relaxation time) is the characteristic time for phase coherence to decay. The off-diagonal elements of the density matrix decay as $e^{-t/T_2}$.
</Definition>

<Formula>
$$\frac{1}{T_2} = \frac{1}{2T_1} + \frac{1}{T_\phi}$$
</Formula>

where $T_\phi$ is the pure dephasing time.

```python
import numpy as np

def phase_damping_channel(rho, gamma):
    """
    Apply phase damping (dephasing) channel.

    Args:
        rho: Input density matrix
        gamma: Dephasing parameter (0 to 1)
              gamma = 1 - exp(-t/T_phi)

    Returns:
        Output density matrix
    """
    # Kraus operators for phase damping
    K0 = np.array([[1, 0], [0, np.sqrt(1 - gamma)]])
    K1 = np.array([[0, 0], [0, np.sqrt(gamma)]])

    rho_out = K0 @ rho @ K0.conj().T + K1 @ rho @ K1.conj().T
    return rho_out

def combined_t1_t2_decay(rho, t, T1, T2):
    """
    Apply combined T1 and T2 decay.

    Uses the generalized amplitude damping model.
    """
    # Amplitude damping from T1
    gamma_1 = 1 - np.exp(-t / T1)

    # Pure dephasing contribution to T2
    # 1/T2 = 1/(2*T1) + 1/T_phi
    # So T_phi = 1 / (1/T2 - 1/(2*T1))
    T_phi = 1 / (1/T2 - 1/(2*T1))
    gamma_phi = 1 - np.exp(-t / T_phi)

    # Apply amplitude damping first
    rho_1 = amplitude_damping_channel(rho, gamma_1)

    # Then pure dephasing
    rho_out = phase_damping_channel(rho_1, gamma_phi)

    return rho_out

# Compare T1 and T2 effects on |+> state
plus = np.array([1, 1]) / np.sqrt(2)
rho_plus = np.outer(plus, plus.conj())

T1 = 100e-6  # 100 us
T2 = 50e-6   # 50 us (T2 <= 2*T1)

print("Combined T1/T2 decay of |+> state:")
print("-" * 55)
print(f"{'t/T2':^10} | {'Re(rho_01)':^12} | {'P(0)':^10} | {'P(1)':^10}")
print("-" * 55)

for t in [0, 0.5*T2, T2, 2*T2, 5*T2]:
    rho_t = combined_t1_t2_decay(rho_plus, t, T1, T2)
    print(f"{t/T2:^10.1f} | {np.real(rho_t[0,1]):^12.4f} | {np.real(rho_t[0,0]):^10.4f} | {np.real(rho_t[1,1]):^10.4f}")
```

<Warning>
T2 is always bounded: T2 <= 2*T1. This is because T1 processes also cause phase decoherence. If T2 = 2*T1, there is no pure dephasing (T_phi = infinity).
</Warning>

## 4. Noise Spectroscopy

Understanding the noise spectrum helps identify decoherence sources and design better qubits.

```python
import numpy as np

def noise_spectrum_1_over_f(frequencies, amplitude, alpha=1):
    """
    Generate 1/f^alpha noise spectrum.

    Common in solid-state systems.
    """
    return amplitude / (frequencies ** alpha)

def noise_spectrum_white(frequencies, amplitude):
    """
    White noise spectrum (constant).

    Quantum limit from zero-point fluctuations.
    """
    return amplitude * np.ones_like(frequencies)

def noise_spectrum_lorentzian(frequencies, amplitude, f_c):
    """
    Lorentzian noise spectrum.

    Characteristic of two-level fluctuators.
    """
    return amplitude / (1 + (frequencies / f_c)**2)

def t2_from_noise_spectrum(S_f, f_qubit):
    """
    Estimate T2 from noise spectrum at qubit frequency.

    Simplified model: T2 ~ 1/sqrt(S(f_qubit))
    """
    noise_at_qubit = S_f
    return 1 / np.sqrt(noise_at_qubit)

# Plot different noise spectra (conceptually)
frequencies = np.logspace(-3, 6, 1000)  # mHz to MHz

print("Noise spectrum comparison at 1 kHz:")
print("-" * 40)
f_test = 1e3  # 1 kHz

S_1f = noise_spectrum_1_over_f(f_test, amplitude=1e-6)
S_white = noise_spectrum_white(f_test, amplitude=1e-12)
S_lorentz = noise_spectrum_lorentzian(f_test, amplitude=1e-9, f_c=100)

print(f"1/f noise:      S(1kHz) = {S_1f:.2e}")
print(f"White noise:    S(1kHz) = {S_white:.2e}")
print(f"Lorentzian:     S(1kHz) = {S_lorentz:.2e}")
```

## 5. Measuring Coherence Times

Several experimental protocols are used to measure T1 and T2:

### T1 Measurement

```python
import numpy as np

def simulate_t1_measurement(T1, wait_times, n_shots=1000):
    """
    Simulate T1 measurement experiment.

    Protocol:
    1. Prepare |1>
    2. Wait time t
    3. Measure in computational basis
    4. Repeat for statistics
    """
    results = []

    for t in wait_times:
        # Probability of still being in |1>
        p_excited = np.exp(-t / T1)

        # Simulate measurement outcomes
        outcomes = np.random.random(n_shots) < p_excited
        measured_prob = np.mean(outcomes)
        results.append(measured_prob)

    return np.array(results)

def fit_t1(wait_times, probabilities):
    """
    Fit T1 from measurement data.

    Uses linear regression on log(P).
    """
    # P(1) = exp(-t/T1)
    # log(P) = -t/T1

    log_p = np.log(probabilities + 1e-10)  # Avoid log(0)

    # Linear fit: log_p = -t/T1
    slope = np.polyfit(wait_times, log_p, 1)[0]
    T1_fit = -1 / slope

    return T1_fit

# Simulate experiment
T1_true = 100e-6  # 100 us
wait_times = np.linspace(0, 500e-6, 20)

measured_probs = simulate_t1_measurement(T1_true, wait_times)
T1_measured = fit_t1(wait_times, measured_probs)

print(f"True T1: {T1_true*1e6:.1f} us")
print(f"Measured T1: {T1_measured*1e6:.1f} us")
```

### T2 Measurement (Ramsey Experiment)

```python
import numpy as np

def simulate_ramsey_experiment(T2, wait_times, detuning=0, n_shots=1000):
    """
    Simulate Ramsey fringe experiment for T2 measurement.

    Protocol:
    1. Apply pi/2 pulse (create |+>)
    2. Wait time t (free evolution with detuning)
    3. Apply pi/2 pulse
    4. Measure
    """
    results = []

    for t in wait_times:
        # Ramsey oscillation with decay
        # P(0) = 0.5 * (1 + exp(-t/T2) * cos(detuning * t))
        visibility = np.exp(-t / T2)
        p_zero = 0.5 * (1 + visibility * np.cos(detuning * t))

        outcomes = np.random.random(n_shots) < p_zero
        results.append(np.mean(outcomes))

    return np.array(results)

# Simulate Ramsey experiment
T2_true = 50e-6  # 50 us
detuning = 2 * np.pi * 100e3  # 100 kHz detuning
wait_times = np.linspace(0, 200e-6, 50)

ramsey_data = simulate_ramsey_experiment(T2_true, wait_times, detuning)

print("Ramsey experiment simulation:")
print("-" * 40)
for i in range(0, len(wait_times), 10):
    t = wait_times[i]
    p = ramsey_data[i]
    print(f"t = {t*1e6:6.1f} us: P(0) = {p:.3f}")
```

### T2* vs T2 Echo

<Definition term="T2* (Free Induction Decay)">
T2* is the apparent dephasing time measured in a simple Ramsey experiment. It includes both intrinsic dephasing and inhomogeneous broadening from low-frequency noise.
</Definition>

```python
import numpy as np

def simulate_echo_experiment(T2, T2_star, wait_time, n_shots=1000):
    """
    Compare Ramsey (T2*) and Echo (T2) experiments.

    Echo refocuses low-frequency noise.
    """
    # Ramsey: sensitive to all noise
    visibility_ramsey = np.exp(-wait_time / T2_star)

    # Echo: refocuses slow noise, measures true T2
    visibility_echo = np.exp(-wait_time / T2)

    return visibility_ramsey, visibility_echo

# Demonstrate echo advantage
T2 = 100e-6       # True T2: 100 us
T2_star = 20e-6   # T2* limited by slow noise: 20 us

print("Echo vs Ramsey comparison:")
print("-" * 50)
for t in [10e-6, 30e-6, 50e-6, 100e-6]:
    v_ramsey, v_echo = simulate_echo_experiment(T2, T2_star, t)
    print(f"t = {t*1e6:5.0f} us: Ramsey visibility = {v_ramsey:.3f}, Echo visibility = {v_echo:.3f}")
```

## 6. Decoherence in Multi-Qubit Systems

For multiple qubits, decoherence affects entanglement and correlations:

```python
import numpy as np

def two_qubit_decoherence(rho, T1_1, T2_1, T1_2, T2_2, t):
    """
    Apply independent decoherence to two qubits.

    Simplified model using effective decay rates.
    """
    # Decay factors
    gamma_1_q1 = np.exp(-t / T1_1)
    gamma_2_q1 = np.exp(-t / T2_1)
    gamma_1_q2 = np.exp(-t / T1_2)
    gamma_2_q2 = np.exp(-t / T2_2)

    # Effective two-qubit decay matrix
    # Diagonal elements decay with T1 factors
    # Off-diagonal elements decay with T2 factors

    rho_out = rho.copy().astype(complex)

    # Population decay (T1 effects)
    # |11> decays to |01>, |10>, |00>
    # Simplified: just apply damping to excited state populations

    # Off-diagonal decay (T2 effects)
    for i in range(4):
        for j in range(4):
            if i != j:
                # Get qubit states
                q1_i, q2_i = i // 2, i % 2
                q1_j, q2_j = j // 2, j % 2

                # Apply dephasing based on state differences
                if q1_i != q1_j:
                    rho_out[i, j] *= gamma_2_q1
                if q2_i != q2_j:
                    rho_out[i, j] *= gamma_2_q2

    return rho_out

# Bell state decoherence
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
rho_bell = np.outer(bell, bell.conj())

# Different T2 times for two qubits
T1_1, T2_1 = 100e-6, 50e-6
T1_2, T2_2 = 80e-6, 40e-6

print("Bell state decoherence with different qubit T2 times:")
print("-" * 55)
for t in [0, 10e-6, 25e-6, 50e-6, 100e-6]:
    rho_t = two_qubit_decoherence(rho_bell, T1_1, T2_1, T1_2, T2_2, t)
    fidelity = np.real(bell.conj() @ rho_t @ bell)
    off_diag = np.abs(rho_t[0, 3])  # |00><11| coherence
    print(f"t = {t*1e6:5.0f} us: Fidelity = {fidelity:.4f}, |rho_03| = {off_diag:.4f}")
```

## 7. Engineering Longer Coherence

Several strategies extend coherence times:

```python
import numpy as np

def dynamical_decoupling_effectiveness(T2_star, T2, n_pulses, total_time):
    """
    Estimate coherence improvement from dynamical decoupling.

    Args:
        T2_star: Free induction decay time
        T2: Intrinsic T2 (echo limit)
        n_pulses: Number of refocusing pulses
        total_time: Total evolution time

    Returns:
        Effective T2 under DD
    """
    # More pulses refocus more of the noise spectrum
    # Effective T2 approaches intrinsic T2 as n_pulses increases

    # Simplified model
    improvement_factor = min(n_pulses, T2 / T2_star)
    T2_effective = T2_star * improvement_factor

    return min(T2_effective, T2)

# Demonstrate DD improvement
T2_star = 10e-6   # Limited by low-frequency noise
T2 = 100e-6       # Intrinsic T2

print("Dynamical decoupling improvement:")
print("-" * 45)
for n in [1, 2, 4, 8, 16, 32]:
    T2_dd = dynamical_decoupling_effectiveness(T2_star, T2, n, 100e-6)
    improvement = T2_dd / T2_star
    print(f"n = {n:2d} pulses: T2_eff = {T2_dd*1e6:.1f} us ({improvement:.1f}x improvement)")
```

<Tip>
Modern superconducting qubits achieve T1 times of 100+ microseconds and T2 times of 50-100 microseconds. Gate times are typically 20-50 nanoseconds, allowing hundreds to thousands of gates before decoherence dominates.
</Tip>

## Summary

In this lesson, you learned about decoherence:

1. **Decoherence mechanism**: Environmental entanglement destroys quantum coherence
2. **T1 relaxation**: Energy decay from excited to ground state
3. **T2 dephasing**: Loss of phase coherence, bounded by 2*T1
4. **Noise spectroscopy**: Different noise types affect coherence differently
5. **Measurement protocols**: Ramsey and echo experiments characterize coherence
6. **Multi-qubit effects**: Entangled states are particularly vulnerable
7. **Coherence engineering**: Dynamical decoupling and better materials extend T1, T2

Understanding decoherence timescales is essential for designing quantum error correction codes that can protect quantum information on timescales relevant for computation.
