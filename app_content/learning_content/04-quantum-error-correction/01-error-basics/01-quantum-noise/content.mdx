## 1. Introduction to Quantum Noise

Unlike classical computers where bits are robust and errors are rare, quantum computers are extraordinarily sensitive to their environment. Every interaction between a qubit and its surroundings can potentially corrupt the delicate quantum information being processed.

<Definition term="Quantum Noise">
Quantum noise refers to any unwanted interaction between a quantum system and its environment that causes degradation of quantum information. This includes both systematic errors and random fluctuations that affect qubit states.
</Definition>

The fragility of quantum information is not a bug but a fundamental feature of quantum mechanics. The same properties that give quantum computers their power - superposition and entanglement - also make them susceptible to noise.

<Tip>
Think of a quantum state as a soap bubble: beautiful and capable of encoding complex patterns, but extremely delicate. Classical bits are more like marbles - robust but limited in what patterns they can represent.
</Tip>

## 2. Sources of Physical Noise

Real quantum computers face numerous noise sources depending on their physical implementation:

### Thermal Fluctuations

At any temperature above absolute zero, thermal energy can excite qubits, causing unwanted transitions between states.

```python
import numpy as np

def thermal_excitation_probability(energy_gap, temperature):
    """
    Calculate probability of thermal excitation.

    Args:
        energy_gap: Energy difference between qubit states (in Joules)
        temperature: Temperature in Kelvin

    Returns:
        Probability of finding qubit in excited state
    """
    k_B = 1.380649e-23  # Boltzmann constant

    if temperature == 0:
        return 0.0

    # Boltzmann distribution
    ratio = energy_gap / (k_B * temperature)
    p_excited = 1 / (1 + np.exp(ratio))

    return p_excited

# Example: Superconducting qubit at different temperatures
energy_gap = 1e-23  # Typical qubit energy gap in Joules

temps = [0.01, 0.02, 0.05, 0.1, 0.5, 1.0]  # Kelvin
print("Thermal excitation probability vs temperature:")
print("-" * 45)
for T in temps:
    p = thermal_excitation_probability(energy_gap, T)
    print(f"T = {T:.2f} K: P(excited) = {p:.6f}")
```

### Electromagnetic Interference

Stray electromagnetic fields can couple to qubits, causing unwanted rotations of the quantum state.

```python
import numpy as np

def magnetic_field_phase_error(field_strength, gyromagnetic_ratio, time):
    """
    Calculate phase error from magnetic field fluctuations.

    Args:
        field_strength: Magnetic field strength in Tesla
        gyromagnetic_ratio: Gyromagnetic ratio of qubit (rad/s/T)
        time: Exposure time in seconds

    Returns:
        Phase error in radians
    """
    # Larmor precession frequency
    omega = gyromagnetic_ratio * field_strength
    phase_error = omega * time

    return phase_error

# Example: Electron spin qubit
gamma_electron = 1.76e11  # rad/s/T (electron gyromagnetic ratio)
stray_field = 1e-9  # 1 nanotesla stray field
gate_time = 1e-6  # 1 microsecond gate

phase_error = magnetic_field_phase_error(stray_field, gamma_electron, gate_time)
print(f"Phase error from 1 nT stray field: {np.degrees(phase_error):.4f} degrees")
```

### Control Errors

Imperfect control pulses lead to gates that don't perform the exact intended operation.

```python
import numpy as np

def imperfect_rotation(theta_target, epsilon):
    """
    Model an imperfect rotation gate.

    Args:
        theta_target: Intended rotation angle
        epsilon: Relative error in rotation

    Returns:
        Actual rotation matrix
    """
    theta_actual = theta_target * (1 + epsilon)

    # Rotation around X-axis
    Rx = np.array([
        [np.cos(theta_actual/2), -1j*np.sin(theta_actual/2)],
        [-1j*np.sin(theta_actual/2), np.cos(theta_actual/2)]
    ])

    return Rx, theta_actual - theta_target

# Example: X gate with calibration error
epsilon = 0.01  # 1% rotation error
target = np.pi  # X gate is pi rotation

R, error = imperfect_rotation(target, epsilon)
print(f"Target rotation: {np.degrees(target):.1f} degrees")
print(f"Rotation error: {np.degrees(error):.4f} degrees")
print(f"Fidelity loss: {1 - np.abs(np.cos(error/2))**2:.6f}")
```

## 3. The Density Matrix Formalism

To properly describe noisy quantum systems, we need the density matrix formalism, which can represent both pure and mixed quantum states.

<Definition term="Density Matrix">
A density matrix (or density operator) $\rho$ is a positive semi-definite Hermitian operator with trace 1 that provides a complete statistical description of a quantum state, including classical uncertainty about which quantum state the system is in.
</Definition>

For a pure state $|\psi\rangle$:
<Formula>
$$\rho = |\psi\rangle\langle\psi|$$
</Formula>

For a mixed state (classical mixture of pure states):
<Formula>
$$\rho = \sum_i p_i |\psi_i\rangle\langle\psi_i|$$
</Formula>

```python
import numpy as np

def create_density_matrix(state_vector):
    """
    Create density matrix from pure state vector.

    Args:
        state_vector: Complex numpy array representing |psi>

    Returns:
        Density matrix rho = |psi><psi|
    """
    state = np.array(state_vector).reshape(-1, 1)
    rho = state @ state.conj().T
    return rho

def create_mixed_state(states, probabilities):
    """
    Create density matrix for mixed state.

    Args:
        states: List of pure state vectors
        probabilities: Classical probabilities for each state

    Returns:
        Mixed state density matrix
    """
    rho = np.zeros((len(states[0]), len(states[0])), dtype=complex)

    for state, prob in zip(states, probabilities):
        rho += prob * create_density_matrix(state)

    return rho

# Example: Pure state |+>
plus_state = np.array([1, 1]) / np.sqrt(2)
rho_pure = create_density_matrix(plus_state)
print("Pure state |+> density matrix:")
print(rho_pure)
print(f"Purity (Tr(rho^2)): {np.real(np.trace(rho_pure @ rho_pure)):.4f}")

# Example: Mixed state (50% |0>, 50% |1>)
zero = np.array([1, 0])
one = np.array([0, 1])
rho_mixed = create_mixed_state([zero, one], [0.5, 0.5])
print("\nMixed state (50% |0>, 50% |1>) density matrix:")
print(rho_mixed)
print(f"Purity (Tr(rho^2)): {np.real(np.trace(rho_mixed @ rho_mixed)):.4f}")
```

## 4. Types of Quantum Errors

Quantum errors can be categorized into several fundamental types:

### Bit-Flip Errors (X Errors)

A bit-flip error flips $|0\rangle$ to $|1\rangle$ and vice versa, analogous to classical bit errors.

```python
import numpy as np

# Pauli X (bit-flip) operator
X = np.array([[0, 1], [1, 0]])

def apply_bit_flip(rho, probability):
    """
    Apply bit-flip channel to density matrix.

    Args:
        rho: Input density matrix
        probability: Probability of bit flip

    Returns:
        Output density matrix after noise
    """
    I = np.eye(2)

    # Kraus operators for bit-flip channel
    # rho' = (1-p) * rho + p * X @ rho @ X
    rho_out = (1 - probability) * rho + probability * (X @ rho @ X)

    return rho_out

# Apply to |+> state
plus = np.array([1, 1]) / np.sqrt(2)
rho_plus = create_density_matrix(plus)

print("Bit-flip channel on |+> state:")
for p in [0, 0.1, 0.5, 1.0]:
    rho_out = apply_bit_flip(rho_plus, p)
    fidelity = np.real(plus.conj() @ rho_out @ plus)
    print(f"p = {p:.1f}: Fidelity = {fidelity:.4f}")
```

### Phase-Flip Errors (Z Errors)

A phase-flip error changes the relative phase between $|0\rangle$ and $|1\rangle$.

```python
import numpy as np

# Pauli Z (phase-flip) operator
Z = np.array([[1, 0], [0, -1]])

def apply_phase_flip(rho, probability):
    """
    Apply phase-flip channel to density matrix.

    Args:
        rho: Input density matrix
        probability: Probability of phase flip

    Returns:
        Output density matrix after noise
    """
    rho_out = (1 - probability) * rho + probability * (Z @ rho @ Z)
    return rho_out

# Compare bit-flip vs phase-flip on different states
zero = np.array([1, 0])
plus = np.array([1, 1]) / np.sqrt(2)

p = 0.1  # 10% error probability

print("Effect of 10% bit-flip vs phase-flip:")
print("-" * 50)

for name, state in [("zero", zero), ("|+>", plus)]:
    rho = create_density_matrix(state)

    rho_bf = apply_bit_flip(rho, p)
    rho_pf = apply_phase_flip(rho, p)

    fid_bf = np.real(state.conj() @ rho_bf @ state)
    fid_pf = np.real(state.conj() @ rho_pf @ state)

    print(f"|{name}>: Bit-flip fidelity = {fid_bf:.4f}, Phase-flip fidelity = {fid_pf:.4f}")
```

<Warning>
Bit-flip errors affect superposition states in the computational basis, while phase-flip errors affect superposition states in the Hadamard basis. A complete error correction code must protect against both types!
</Warning>

### Bit-Phase Flip Errors (Y Errors)

The Y error combines both bit-flip and phase-flip effects.

```python
import numpy as np

# Pauli Y operator
Y = np.array([[0, -1j], [1j, 0]])

def apply_y_error(rho, probability):
    """
    Apply Y error channel to density matrix.
    """
    rho_out = (1 - probability) * rho + probability * (Y @ rho @ Y)
    return rho_out

# Demonstrate that Y = iXZ
X = np.array([[0, 1], [1, 0]])
Z = np.array([[1, 0], [0, -1]])
Y_check = 1j * X @ Z

print("Verifying Y = iXZ:")
print(f"Y = \n{Y}")
print(f"iXZ = \n{Y_check}")
print(f"Equal: {np.allclose(Y, Y_check)}")
```

## 5. The Depolarizing Channel

The depolarizing channel is a important noise model that applies all three types of Pauli errors with equal probability.

<Definition term="Depolarizing Channel">
The depolarizing channel with parameter $p$ replaces the input state with the maximally mixed state with probability $p$:
$$\mathcal{E}(\rho) = (1-p)\rho + \frac{p}{3}(X\rho X + Y\rho Y + Z\rho Z)$$
</Definition>

```python
import numpy as np

def depolarizing_channel(rho, p):
    """
    Apply depolarizing channel to a single qubit.

    Args:
        rho: Input density matrix
        p: Depolarizing parameter (probability of error)

    Returns:
        Output density matrix
    """
    I = np.eye(2)
    X = np.array([[0, 1], [1, 0]])
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]])

    # Apply depolarizing channel
    rho_out = (1 - p) * rho
    rho_out += (p/3) * (X @ rho @ X)
    rho_out += (p/3) * (Y @ rho @ Y)
    rho_out += (p/3) * (Z @ rho @ Z)

    return rho_out

def calculate_purity(rho):
    """Calculate purity Tr(rho^2)."""
    return np.real(np.trace(rho @ rho))

# Study depolarization of a pure state
plus = np.array([1, 1]) / np.sqrt(2)
rho_initial = create_density_matrix(plus)

print("Depolarizing channel effect on |+> state:")
print("-" * 45)
print(f"{'p':^8} | {'Purity':^10} | {'Fidelity':^10}")
print("-" * 45)

for p in np.linspace(0, 1, 11):
    rho_out = depolarizing_channel(rho_initial, p)
    purity = calculate_purity(rho_out)
    fidelity = np.real(plus.conj() @ rho_out @ plus)
    print(f"{p:^8.2f} | {purity:^10.4f} | {fidelity:^10.4f}")
```

## 6. Coherent vs Incoherent Errors

Understanding the distinction between coherent and incoherent errors is crucial for error correction strategies.

<Definition term="Coherent Errors">
Coherent errors are systematic, unitary rotations that affect all qubits in a predictable way. They can accumulate constructively and are often caused by calibration errors or control imperfections.
</Definition>

<Definition term="Incoherent Errors">
Incoherent errors are random, non-unitary processes that reduce the purity of quantum states. They arise from environmental interactions and do not accumulate in a predictable direction.
</Definition>

```python
import numpy as np

def coherent_over_rotation(rho, epsilon, num_gates):
    """
    Model accumulated coherent error from over-rotation.

    Args:
        rho: Initial density matrix
        epsilon: Small rotation error per gate
        num_gates: Number of gates applied

    Returns:
        Final density matrix
    """
    # Total accumulated rotation error
    total_error = epsilon * num_gates

    # Rotation around Z axis
    Rz = np.array([
        [np.exp(-1j * total_error / 2), 0],
        [0, np.exp(1j * total_error / 2)]
    ])

    return Rz @ rho @ Rz.conj().T

def incoherent_dephasing(rho, gamma, num_gates):
    """
    Model accumulated incoherent dephasing.

    Args:
        rho: Initial density matrix
        gamma: Dephasing probability per gate
        num_gates: Number of gates applied

    Returns:
        Final density matrix
    """
    # Dephasing channel repeated n times
    # Equivalent to single channel with p = 1 - (1-gamma)^n
    p_total = 1 - (1 - gamma)**num_gates

    Z = np.array([[1, 0], [0, -1]])
    rho_out = (1 - p_total) * rho + p_total * Z @ rho @ Z

    return rho_out

# Compare coherent vs incoherent error accumulation
plus = np.array([1, 1]) / np.sqrt(2)
rho_initial = create_density_matrix(plus)

epsilon = 0.01  # 1% rotation error per gate
gamma = 0.01    # 1% dephasing per gate

print("Error accumulation: Coherent vs Incoherent")
print("-" * 55)
print(f"{'Gates':^8} | {'Coherent Fid':^14} | {'Incoherent Fid':^14}")
print("-" * 55)

for n in [1, 10, 50, 100, 200]:
    rho_coh = coherent_over_rotation(rho_initial, epsilon, n)
    rho_inc = incoherent_dephasing(rho_initial, gamma, n)

    fid_coh = np.real(plus.conj() @ rho_coh @ plus)
    fid_inc = np.real(plus.conj() @ rho_inc @ plus)

    print(f"{n:^8} | {fid_coh:^14.4f} | {fid_inc:^14.4f}")
```

<Tip>
Coherent errors can actually be worse than incoherent errors because they accumulate linearly rather than quadratically. However, coherent errors are often easier to characterize and correct through calibration.
</Tip>

## 7. Noise in Multi-Qubit Systems

When dealing with multiple qubits, noise can affect individual qubits independently or create correlated errors.

```python
import numpy as np

def tensor_product(A, B):
    """Compute tensor product of two matrices."""
    return np.kron(A, B)

def independent_depolarizing(rho_2qubit, p):
    """
    Apply independent depolarizing noise to each qubit.

    Args:
        rho_2qubit: 4x4 density matrix for 2 qubits
        p: Single-qubit depolarizing probability

    Returns:
        Noisy 2-qubit density matrix
    """
    I = np.eye(2)
    X = np.array([[0, 1], [1, 0]])
    Y = np.array([[0, -1j], [1j, 0]])
    Z = np.array([[1, 0], [0, -1]])

    paulis = [I, X, Y, Z]

    rho_out = np.zeros((4, 4), dtype=complex)

    # Apply noise to qubit 1
    for i, P1 in enumerate(paulis):
        for j, P2 in enumerate(paulis):
            coeff = 1.0
            if i > 0:  # Non-identity on qubit 1
                coeff *= p/3
            else:
                coeff *= (1 - p)
            if j > 0:  # Non-identity on qubit 2
                coeff *= p/3
            else:
                coeff *= (1 - p)

            operator = tensor_product(P1, P2)
            rho_out += coeff * (operator @ rho_2qubit @ operator.conj().T)

    return rho_out

# Create Bell state |00> + |11>
bell = np.array([1, 0, 0, 1]) / np.sqrt(2)
rho_bell = np.outer(bell, bell.conj())

print("Independent depolarizing noise on Bell state:")
print("-" * 40)
for p in [0, 0.05, 0.1, 0.2]:
    rho_noisy = independent_depolarizing(rho_bell, p)
    fidelity = np.real(bell.conj() @ rho_noisy @ bell)
    purity = np.real(np.trace(rho_noisy @ rho_noisy))
    print(f"p = {p:.2f}: Fidelity = {fidelity:.4f}, Purity = {purity:.4f}")
```
