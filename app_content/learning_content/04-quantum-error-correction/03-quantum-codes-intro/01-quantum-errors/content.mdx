# Quantum Errors and Their Nature

## Introduction

Quantum error correction faces challenges that have no classical analog. Unlike classical bits, qubits exist in superpositions, cannot be copied, and collapse when measured. This lesson explores why protecting quantum information requires fundamentally new techniques.

## The No-Cloning Theorem

```python
import numpy as np
from typing import Tuple, Optional

class NoCloning:
    """
    Demonstrates why quantum states cannot be copied.

    The no-cloning theorem: There is no unitary operation U such that
    U|ψ⟩|0⟩ = |ψ⟩|ψ⟩ for all states |ψ⟩.
    """

    def __init__(self):
        # Basis states
        self.ket_0 = np.array([1, 0], dtype=complex)
        self.ket_1 = np.array([0, 1], dtype=complex)

    def attempt_cloning(self, state: np.ndarray) -> str:
        """
        Show why cloning fails.

        Proof by contradiction:
        If cloning worked, U|ψ⟩|0⟩ = |ψ⟩|ψ⟩

        For |0⟩: U|0⟩|0⟩ = |0⟩|0⟩ = |00⟩
        For |1⟩: U|1⟩|0⟩ = |1⟩|1⟩ = |11⟩

        For |+⟩ = (|0⟩ + |1⟩)/√2:
        By linearity: U|+⟩|0⟩ = (U|0⟩|0⟩ + U|1⟩|0⟩)/√2
                              = (|00⟩ + |11⟩)/√2

        But we wanted: |+⟩|+⟩ = (|0⟩ + |1⟩)(|0⟩ + |1⟩)/2
                              = (|00⟩ + |01⟩ + |10⟩ + |11⟩)/2

        CONTRADICTION! These are different states.
        """
        ket_plus = (self.ket_0 + self.ket_1) / np.sqrt(2)

        # What linearity would give
        linear_result = self._tensor([self.ket_0, self.ket_0]) + \
                       self._tensor([self.ket_1, self.ket_1])
        linear_result = linear_result / np.sqrt(2)

        # What cloning should give
        clone_result = self._tensor([ket_plus, ket_plus])

        # Check if they're equal
        overlap = abs(np.vdot(linear_result, clone_result))

        return f"""
        No-Cloning Proof for |+⟩:

        Linearity gives: (|00⟩ + |11⟩)/√2  [Bell state]
        Cloning needs:   (|00⟩ + |01⟩ + |10⟩ + |11⟩)/2  [Product state]

        Overlap: {overlap:.4f} (would be 1.0 if equal)

        ╔═══════════════════════════════════════════════╗
        ║  CLONING IS IMPOSSIBLE by quantum mechanics   ║
        ╚═══════════════════════════════════════════════╝

        Implication for error correction:
        We cannot copy a qubit to create redundancy!
        """

    def _tensor(self, states: list) -> np.ndarray:
        """Tensor product of states."""
        result = states[0]
        for s in states[1:]:
            result = np.kron(result, s)
        return result


# Classical copying vs quantum no-cloning
cloner = NoCloning()
print(cloner.attempt_cloning(None))
```

## The Pauli Error Basis

```python
import numpy as np
from typing import Dict, Tuple

class PauliErrors:
    """
    Any single-qubit error can be expressed in the Pauli basis.

    Error Decomposition Theorem:
    Any operator E on a qubit can be written as:
    E = e₀I + e₁X + e₂Y + e₃Z

    This means: If we can correct X, Y, and Z errors separately,
    we can correct ANY error!
    """

    def __init__(self):
        self.I = np.array([[1, 0], [0, 1]], dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

        self.paulis = {'I': self.I, 'X': self.X, 'Y': self.Y, 'Z': self.Z}

    def decompose_error(self, E: np.ndarray) -> Dict[str, complex]:
        """
        Decompose arbitrary error into Pauli basis.

        Using: eₚ = Tr(P·E)/2 for each Pauli P
        """
        coefficients = {}
        for name, P in self.paulis.items():
            coeff = np.trace(P @ E) / 2
            coefficients[name] = coeff
        return coefficients

    def reconstruct_error(self, coeffs: Dict[str, complex]) -> np.ndarray:
        """Reconstruct error from Pauli coefficients."""
        result = np.zeros((2, 2), dtype=complex)
        for name, coeff in coeffs.items():
            result += coeff * self.paulis[name]
        return result

    def demonstrate_error_discretization(self):
        """
        Show how continuous errors become discrete after measurement.

        Key insight for QEC:
        Even though errors are continuous, syndrome measurement
        "collapses" them to discrete Pauli errors!
        """
        print("Error Discretization in Quantum Error Correction")
        print("=" * 55)

        # Continuous rotation error around X-axis
        theta = np.pi / 6  # 30 degree rotation
        R_x = np.array([
            [np.cos(theta/2), -1j*np.sin(theta/2)],
            [-1j*np.sin(theta/2), np.cos(theta/2)]
        ])

        print(f"\nContinuous error: R_x({np.degrees(theta):.1f}°)")

        # Decompose into Paulis
        coeffs = self.decompose_error(R_x)

        print("\nPauli decomposition:")
        for name, coeff in coeffs.items():
            if abs(coeff) > 1e-10:
                print(f"  {name}: {coeff:.4f}")

        print(f"""

        ┌─────────────────────────────────────────────────┐
        │  Before syndrome measurement:                   │
        │    |ψ'⟩ = {coeffs['I']:.3f}|ψ⟩ + {coeffs['X']:.3f}·i·X|ψ⟩      │
        │                                                 │
        │  After syndrome measurement:                    │
        │    Probability of no error: {abs(coeffs['I'])**2:.3f}          │
        │    Probability of X error:  {abs(coeffs['X'])**2:.3f}          │
        │                                                 │
        │  The continuous error becomes DISCRETE!         │
        └─────────────────────────────────────────────────┘
        """)

        return coeffs

    def pauli_error_effects(self):
        """Visualize how each Pauli affects quantum states."""
        ket_0 = np.array([1, 0], dtype=complex)
        ket_1 = np.array([0, 1], dtype=complex)
        ket_plus = (ket_0 + ket_1) / np.sqrt(2)

        print("\nPauli Error Effects on Quantum States")
        print("=" * 45)

        states = {'|0⟩': ket_0, '|1⟩': ket_1, '|+⟩': ket_plus}

        for state_name, state in states.items():
            print(f"\n{state_name}:")
            for pauli_name, P in self.paulis.items():
                result = P @ state
                # Format result as ket notation
                if np.allclose(result, ket_0):
                    res_str = "|0⟩"
                elif np.allclose(result, ket_1):
                    res_str = "|1⟩"
                elif np.allclose(result, -ket_0):
                    res_str = "-|0⟩"
                elif np.allclose(result, -ket_1):
                    res_str = "-|1⟩"
                elif np.allclose(result, 1j*ket_0):
                    res_str = "i|0⟩"
                elif np.allclose(result, 1j*ket_1):
                    res_str = "i|1⟩"
                elif np.allclose(result, ket_plus):
                    res_str = "|+⟩"
                elif np.allclose(result, (ket_0 - ket_1)/np.sqrt(2)):
                    res_str = "|-⟩"
                else:
                    res_str = f"[{result[0]:.2f}, {result[1]:.2f}]"
                print(f"  {pauli_name}{state_name} = {res_str}")


# Demonstrate Pauli error basis
pauli = PauliErrors()
pauli.demonstrate_error_discretization()
pauli.pauli_error_effects()
```

## The Measurement Problem

```python
import numpy as np

class MeasurementProblem:
    """
    Demonstrates why direct measurement destroys quantum information.
    """

    def __init__(self):
        self.ket_0 = np.array([1, 0], dtype=complex)
        self.ket_1 = np.array([0, 1], dtype=complex)

    def measure_state(self, state: np.ndarray, num_trials: int = 1000) -> dict:
        """
        Simulate measurement of quantum state.

        Problem: Measurement gives classical bit, losing phase information!
        """
        prob_0 = abs(state[0])**2
        prob_1 = abs(state[1])**2

        results = {'0': 0, '1': 0}
        np.random.seed(42)

        for _ in range(num_trials):
            if np.random.random() < prob_0:
                results['0'] += 1
            else:
                results['1'] += 1

        return {k: v/num_trials for k, v in results.items()}

    def show_information_loss(self):
        """Demonstrate information loss from measurement."""
        print("The Measurement Problem in Error Correction")
        print("=" * 50)

        # These states look identical under Z-measurement
        states = {
            '|+⟩ = (|0⟩+|1⟩)/√2': (self.ket_0 + self.ket_1) / np.sqrt(2),
            '|-⟩ = (|0⟩-|1⟩)/√2': (self.ket_0 - self.ket_1) / np.sqrt(2),
            '|+i⟩ = (|0⟩+i|1⟩)/√2': (self.ket_0 + 1j*self.ket_1) / np.sqrt(2),
        }

        print("\nThese states give IDENTICAL Z-measurement statistics:")
        for name, state in states.items():
            results = self.measure_state(state)
            print(f"  {name}: P(0)={results['0']:.2f}, P(1)={results['1']:.2f}")

        print("""
        ┌───────────────────────────────────────────────────┐
        │  ALL these states give 50% |0⟩ and 50% |1⟩!      │
        │                                                   │
        │  The phase information (±, ±i) is LOST!          │
        │                                                   │
        │  We cannot check if our qubit is correct by      │
        │  simply measuring it - that would destroy it.    │
        └───────────────────────────────────────────────────┘
        """)

    def quantum_vs_classical_checking(self):
        """Compare classical and quantum error checking."""
        print("\nClassical vs Quantum Error Checking")
        print("=" * 45)

        print("""
        CLASSICAL:
        ┌─────────────────────────────────────────────┐
        │  bit = 1                                    │
        │  ↓                                          │
        │  READ: bit == 1? ✓                          │
        │  ↓                                          │
        │  bit = 1 (unchanged)                        │
        │                                             │
        │  We can check without disturbing!           │
        └─────────────────────────────────────────────┘

        QUANTUM:
        ┌─────────────────────────────────────────────┐
        │  |ψ⟩ = α|0⟩ + β|1⟩                          │
        │  ↓                                          │
        │  MEASURE: Get 0 or 1 randomly               │
        │  ↓                                          │
        │  |0⟩ or |1⟩ (superposition destroyed!)      │
        │                                             │
        │  We cannot check without destroying!        │
        │                                             │
        │  QEC Solution: Measure SYNDROME, not DATA   │
        │  - Syndrome reveals error type              │
        │  - Doesn't reveal or disturb data          │
        └─────────────────────────────────────────────┘
        """)


# Demonstrate measurement problem

measurement.show_information_loss()
```

## The Key Insight: Entanglement to the Rescue

```
How Quantum Error Correction Works
══════════════════════════════════

Classical Strategy (fails for quantum):
    Copy the bit → Use majority voting → Recover original

Quantum Strategy:
    Entangle qubit with ancillas → Measure ancillas → Learn error without measuring data

    |ψ⟩ ─────●─────●─────   Data qubit (protected)
             │     │
    |0⟩ ────╳─────│───── ─ Ancilla 1  ─┐
                  │                     ├→ Measure ancillas
    |0⟩ ──────────╳───── ─ Ancilla 2  ─┘   to get syndrome

Key Properties:
┌─────────────────────────────────────────────────────────────────┐
│ 1. No Cloning:    We don't copy |ψ⟩, we ENTANGLE it             │
│                                                                 │
│ 2. Indirect Info: Syndrome tells us WHICH error, not WHAT data  │
│                                                                 │
│ 3. Discretization: Continuous errors collapse to discrete       │
│                                                                 │
│ 4. Redundancy:    Information spread across multiple qubits     │
└─────────────────────────────────────────────────────────────────┘
```

## Summary

The key insights from this lesson:

1. **No-cloning theorem** prevents simple redundancy—we must use entanglement instead
2. **Measurement collapse** means we cannot check qubit values directly—we measure syndromes
3. **Continuous errors** discretize when we measure syndromes—Pauli basis is sufficient
4. **Phase errors** are uniquely quantum—classical codes only handle bit flips
5. **Syndrome measurement** extracts error information without disturbing encoded data

These challenges motivated the development of the first quantum error-correcting codes, which we explore in the next lessons.
