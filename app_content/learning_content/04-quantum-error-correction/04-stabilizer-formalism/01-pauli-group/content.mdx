# The Pauli Group

## Introduction

The Pauli group is the mathematical backbone of stabilizer codes. Understanding its structure enables efficient manipulation of quantum states and systematic construction of error-correcting codes. This lesson explores the group's properties and their applications to quantum error correction.

## Single-Qubit Pauli Operators

```python
import numpy as np
from typing import Tuple, List, Dict

class PauliOperators:
    """
    The four Pauli operators form the basis for all single-qubit operations.
    """

    def __init__(self):
        # The four Pauli matrices
        self.I = np.array([[1, 0], [0, 1]], dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

        self.paulis = {'I': self.I, 'X': self.X, 'Y': self.Y, 'Z': self.Z}

    def show_properties(self):
        """Display key properties of Pauli operators."""
        print("""
Pauli Operator Properties
═══════════════════════════

The Pauli matrices:

    I = ⎛ 1  0 ⎞       X = ⎛ 0  1 ⎞
        ⎝ 0  1 ⎠           ⎝ 1  0 ⎠

    Y = ⎛ 0  -i ⎞      Z = ⎛ 1   0 ⎞
        ⎝ i   0 ⎠          ⎝ 0  -1 ⎠

Key Properties:
┌───────────────────────────────────────────────────────────┐
│ 1. Hermitian: P† = P  (self-adjoint)                     │
│ 2. Unitary: P†P = I  (preserves inner products)          │
│ 3. Involutory: P² = I  (applying twice gives identity)   │
│ 4. Eigenvalues: ±1 only                                  │
│ 5. Trace: Tr(X) = Tr(Y) = Tr(Z) = 0, Tr(I) = 2          │
└───────────────────────────────────────────────────────────┘
        """)

    def verify_properties(self):
        """Numerically verify Pauli properties."""
        print("\nVerifying Properties")
        print("─" * 45)

        for name, P in self.paulis.items():
            # Hermitian
            is_hermitian = np.allclose(P, P.conj().T)

            # Involutory (P² = I)
            is_involutory = np.allclose(P @ P, self.I)

            # Eigenvalues
            eigenvalues = np.linalg.eigvals(P)

            print(f"{name}: Hermitian={is_hermitian}, "
                  f"Involutory={is_involutory}, "
                  f"Eigenvals={np.round(eigenvalues.real, 2)}")


paulis = PauliOperators()
paulis.show_properties()
paulis.verify_properties()
```

## The Single-Qubit Pauli Group G₁

```python
class SingleQubitPauliGroup:
    """
    The single-qubit Pauli group G₁ = {±I, ±X, ±Y, ±Z, ±iI, ±iX, ±iY, ±iZ}.

    This is a group of 16 elements under matrix multiplication.
    """

    def __init__(self):
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

        # All 16 group elements
        self.elements = []
        for phase in [1, -1, 1j, -1j]:
            for pauli in [self.I, self.X, self.Y, self.Z]:
                self.elements.append(phase * pauli)

    def show_group_structure(self):
        """Display the group structure."""
        print("""
The Single-Qubit Pauli Group G₁
═══════════════════════════════

G₁ = ⟨i, X, Y, Z⟩ = {±1, ±i} × {I, X, Y, Z}

Elements (16 total):
┌─────────────────────────────────────────────────────────┐
│  +I,  +X,  +Y,  +Z                                      │
│  -I,  -X,  -Y,  -Z                                      │
│  +iI, +iX, +iY, +iZ                                     │
│  -iI, -iX, -iY, -iZ                                     │
└─────────────────────────────────────────────────────────┘

Group properties:
    - Order: |G₁| = 16
    - Center: Z(G₁) = {±I, ±iI} (elements that commute with all)
    - G₁/Z(G₁) ≅ Z₂ × Z₂ (the quotient is abelian)

Multiplication rules:
    X² = Y² = Z² = I
    XY = iZ,  YX = -iZ  (anticommute)
    YZ = iX,  ZY = -iX  (anticommute)
    ZX = iY,  XZ = -iY  (anticommute)
        """)

    def multiplication_table(self):
        """Show products of Pauli operators."""
        print("\nPauli Multiplication Table (ignoring phase)")
        print("─" * 45)

        labels = ['I', 'X', 'Y', 'Z']
        ops = [self.I, self.X, self.Y, self.Z]

        print("    │  I    X    Y    Z")
        print("────┼────────────────────")

        for i, (l1, o1) in enumerate(zip(labels, ops)):
            row = f" {l1}  │ "
            for j, (l2, o2) in enumerate(zip(labels, ops)):
                product = o1 @ o2

                # Find which Pauli this is (up to phase)
                for k, (l3, o3) in enumerate(zip(labels, ops)):
                    if np.allclose(np.abs(product), np.abs(o3)):
                        # Get phase
                        if np.allclose(product, o3):
                            phase = ""
                        elif np.allclose(product, -o3):
                            phase = "-"
                        elif np.allclose(product, 1j * o3):
                            phase = "i"
                        else:
                            phase = "-i"
                        row += f"{phase}{l3}".rjust(4) + " "
                        break
            print(row)


g1 = SingleQubitPauliGroup()
g1.show_group_structure()
g1.multiplication_table()
```

## Commutation and Anticommutation

```python
class PauliCommutation:
    """
    Understanding when Pauli operators commute or anticommute.
    """

    def __init__(self):
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

    def show_commutation_relations(self):
        """Display commutation and anticommutation relations."""
        print("""
Pauli Commutation Relations
═══════════════════════════

Two operators A, B:
    - COMMUTE if AB = BA           (can measure simultaneously)
    - ANTICOMMUTE if AB = -BA      (cannot measure simultaneously)

Commutation table ([A,B] = AB - BA):
┌────────────────────────────────────────────────────────────────┐
│         I      X      Y      Z                                 │
│    I    0      0      0      0       (I commutes with all)     │
│    X    0      0     2iZ   -2iY                                │
│    Y    0    -2iZ    0     2iX                                 │
│    Z    0     2iY   -2iX    0                                  │
└────────────────────────────────────────────────────────────────┘

Anticommutation table ({A,B} = AB + BA):
┌────────────────────────────────────────────────────────────────┐
│         I      X      Y      Z                                 │
│    I   2I     2X     2Y     2Z      (I anticommutes with none) │
│    X   2X     2I     0      0                                  │
│    Y   2Y     0      2I     0                                  │
│    Z   2Z     0      0      2I                                 │
└────────────────────────────────────────────────────────────────┘

Key insight:
╔═════════════════════════════════════════════════════════════════╗
║  Different non-identity Paulis ANTICOMMUTE: XY = -YX, etc.     ║
║  Same or identity Paulis COMMUTE: XX = XX, IX = XI, etc.       ║
╚═════════════════════════════════════════════════════════════════╝
        """)

    def check_commutation(self, A: np.ndarray, B: np.ndarray) -> str:
        """Check if two operators commute or anticommute."""
        AB = A @ B
        BA = B @ A

        if np.allclose(AB, BA):
            return "commute"
        elif np.allclose(AB, -BA):
            return "anticommute"
        else:
            return "neither"

    def verify_relations(self):
        """Verify all commutation relations."""
        print("\nVerifying Commutation Relations")
        print("─" * 45)

        ops = {'I': self.I, 'X': self.X, 'Y': self.Y, 'Z': self.Z}

        for n1, o1 in ops.items():
            for n2, o2 in ops.items():
                result = self.check_commutation(o1, o2)
                print(f"[{n1}, {n2}]: {result}")


comm = PauliCommutation()
comm.show_commutation_relations()
```

## The n-Qubit Pauli Group Gₙ

```python
class NQubitPauliGroup:
    """
    The n-qubit Pauli group Gₙ consists of tensor products of single-qubit Paulis.
    """

    def __init__(self, n: int):
        self.n = n
        self.I = np.eye(2, dtype=complex)
        self.X = np.array([[0, 1], [1, 0]], dtype=complex)
        self.Y = np.array([[0, -1j], [1j, 0]], dtype=complex)
        self.Z = np.array([[1, 0], [0, -1]], dtype=complex)

        self.paulis = {'I': self.I, 'X': self.X, 'Y': self.Y, 'Z': self.Z}

    def string_to_operator(self, pauli_string: str) -> np.ndarray:
        """
        Convert Pauli string like 'XIZI' to tensor product operator.
        """
        assert len(pauli_string) == self.n

        result = self.paulis[pauli_string[0]]
        for p in pauli_string[1:]:
            result = np.kron(result, self.paulis[p])
        return result

    def check_commutation(self, s1: str, s2: str) -> bool:
        """
        Check if two Pauli strings commute using efficient method.

        Two n-qubit Paulis commute iff the number of positions
        where they have different non-identity operators is EVEN.
        """
        anticommuting_positions = 0
        for c1, c2 in zip(s1, s2):
            if c1 != 'I' and c2 != 'I' and c1 != c2:
                anticommuting_positions += 1

        return (anticommuting_positions % 2) == 0

    def show_structure(self):
        """Display the n-qubit Pauli group structure."""
        print(f"""
The {self.n}-Qubit Pauli Group G_{self.n}
{'═' * 35}

G_n = {{±1, ±i}} × {{I, X, Y, Z}}^⊗n

Properties:
┌───────────────────────────────────────────────────────────────┐
│ Order: |G_n| = 4 × 4^n = 4^(n+1)                              │
│ For n={self.n}: |G_{self.n}| = {4**(self.n+1)} elements                              │
│                                                               │
│ Ignoring phase: |G_n / Z(G_n)| = 4^n                         │
│ For n={self.n}: {4**self.n} distinct Pauli operators (up to phase)          │
└───────────────────────────────────────────────────────────────┘

Elements are tensor products:
    P = p₁ ⊗ p₂ ⊗ ... ⊗ pₙ  where each pᵢ ∈ {{I, X, Y, Z}}

Notation examples for n={self.n}:
    {'I' * self.n} = I⊗I⊗...⊗I (identity)
    X{'I' * (self.n-1)} = X⊗I⊗...⊗I (X on first qubit)
    {'I' * (self.n-1)}Z = I⊗...⊗I⊗Z (Z on last qubit)
        """)

    def demonstrate_commutation(self):
        """Show commutation checking for multi-qubit Paulis."""
        print("\nCommutation in Multi-Qubit Paulis")
        print("─" * 50)

        examples = [
            ('XIZI', 'IZXI'),
            ('XXII', 'ZZII'),
            ('XYZI', 'XZYI'),
            ('ZZZZ', 'XXXX'),
        ]

        for s1, s2 in examples:
            if len(s1) == self.n and len(s2) == self.n:
                commutes = self.check_commutation(s1, s2)
                result = "commute" if commutes else "anticommute"
                print(f"{s1} and {s2}: {result}")


# Example with 4 qubits
g4 = NQubitPauliGroup(4)
g4.show_structure()
g4.demonstrate_commutation()
```

## Symplectic Representation

```python
class SymplecticRepresentation:
    """
    Efficient binary representation of Pauli operators.

    Each n-qubit Pauli (ignoring phase) is encoded as two n-bit vectors:
        x = (x₁, x₂, ..., xₙ)  - indicates X component
        z = (z₁, z₂, ..., zₙ)  - indicates Z component

    Pauli at position i: x[i]=0,z[i]=0 → I
                         x[i]=1,z[i]=0 → X
                         x[i]=1,z[i]=1 → Y (since Y = iXZ)
                         x[i]=0,z[i]=1 → Z
    """

    def __init__(self, n: int):
        self.n = n

    def string_to_binary(self, pauli_string: str) -> Tuple[np.ndarray, np.ndarray]:
        """Convert Pauli string to binary (x, z) vectors."""
        x = np.zeros(self.n, dtype=int)
        z = np.zeros(self.n, dtype=int)

        for i, p in enumerate(pauli_string):
            if p == 'X':
                x[i] = 1
            elif p == 'Y':
                x[i] = 1
                z[i] = 1
            elif p == 'Z':
                z[i] = 1
            # I: both remain 0

        return (x, z)

    def binary_to_string(self, x: np.ndarray, z: np.ndarray) -> str:
        """Convert binary vectors back to Pauli string."""
        result = ""
        for i in range(self.n):
            if x[i] == 0 and z[i] == 0:
                result += 'I'
            elif x[i] == 1 and z[i] == 0:
                result += 'X'
            elif x[i] == 1 and z[i] == 1:
                result += 'Y'
            else:  # x[i] == 0 and z[i] == 1
                result += 'Z'
        return result

    def symplectic_inner_product(self, v1: Tuple[np.ndarray, np.ndarray],
                                  v2: Tuple[np.ndarray, np.ndarray]) -> int:
        """
        Compute symplectic inner product.

        ⟨v₁, v₂⟩_s = x₁·z₂ + z₁·x₂ (mod 2)

        This determines commutation:
            Result = 0 → operators commute
            Result = 1 → operators anticommute
        """
        x1, z1 = v1
        x2, z2 = v2

        return (np.dot(x1, z2) + np.dot(z1, x2)) % 2

    def show_representation(self):
        """Display the symplectic representation."""
        print("""
Symplectic Representation of Pauli Operators
═════════════════════════════════════════════

Binary encoding (x, z) for each qubit:
┌─────────────────────────────────────────────┐
│  Pauli  │  x  │  z  │  Meaning             │
├─────────┼─────┼─────┼──────────────────────┤
│    I    │  0  │  0  │  No operator         │
│    X    │  1  │  0  │  X only              │
│    Y    │  1  │  1  │  XZ (= iY)           │
│    Z    │  0  │  1  │  Z only              │
└─────────────────────────────────────────────┘

For n qubits: (x, z) = (x₁...xₙ, z₁...zₙ) ∈ Z₂^(2n)

Symplectic inner product:
    ⟨(x₁,z₁), (x₂,z₂)⟩_s = x₁·z₂ + z₁·x₂  (mod 2)

Commutation rule:
╔═════════════════════════════════════════════════════════════╗
║  ⟨v₁, v₂⟩_s = 0  ⟺  Paulis commute                        ║
║  ⟨v₁, v₂⟩_s = 1  ⟺  Paulis anticommute                    ║
╚═════════════════════════════════════════════════════════════╝
        """)

    def demonstrate(self):
        """Demonstrate symplectic representation."""
        print("\nSymplectic Representation Examples")
        print("─" * 50)

        examples = ['XIZI', 'ZZXX', 'YYYY', 'XIII']

        for s in examples:
            if len(s) <= self.n:
                s_padded = s + 'I' * (self.n - len(s))
                x, z = self.string_to_binary(s_padded)
                print(f"{s_padded}: x={x}, z={z}")

        print("\nCommutation via Symplectic Inner Product:")
        pairs = [('XIZI', 'IZXI'), ('XXII', 'ZZII')]
        for s1, s2 in pairs:
            s1 = s1 + 'I' * (self.n - len(s1))
            s2 = s2 + 'I' * (self.n - len(s2))
            v1 = self.string_to_binary(s1)
            v2 = self.string_to_binary(s2)
            inner = self.symplectic_inner_product(v1, v2)
            result = "commute" if inner == 0 else "anticommute"
            print(f"⟨{s1}, {s2}⟩_s = {inner} → {result}")


symplectic = SymplecticRepresentation(4)
symplectic.show_representation()
symplectic.demonstrate()
```

## Application to Error Correction

```
Pauli Group in Quantum Error Correction
═══════════════════════════════════════

1. ERROR MODEL
   Any error can be decomposed into the Pauli basis:
       E = Σ eₚ·P   where P ∈ Gₙ

   If we can correct each Pauli error, we can correct any error!

2. STABILIZERS
   A stabilizer code is defined by a subgroup S ⊆ Gₙ where:
   - All elements of S commute with each other
   - The code space is the +1 eigenspace of all S

3. ERROR DETECTION
   Error E is detectable if E anticommutes with some stabilizer.
   E anticommutes with S ⟺ SE = -ES ⟺ ES†S = -E ⟺ syndrome ≠ 0

4. CORRECTABLE ERRORS
   Error E is correctable if either:
   - E ∈ S (error is a stabilizer - does nothing to code space)
   - E anticommutes with some stabilizer (detectable, unique syndrome)

5. LOGICAL OPERATORS
   Operators that commute with all stabilizers but are NOT in S:
   These act non-trivially on encoded information.

┌─────────────────────────────────────────────────────────────────┐
│  The entire structure of stabilizer codes is built on          │
│  understanding which Paulis commute and anticommute!           │
└─────────────────────────────────────────────────────────────────┘
```

## Summary

The Pauli group provides the mathematical foundation for stabilizer codes:

1. **Group structure**: 16 elements per qubit (±1, ±i phases × 4 Paulis)
2. **Commutation rules**: Non-identity Paulis anticommute; this determines detectability
3. **Tensor products**: n-qubit Paulis are products of single-qubit Paulis
4. **Symplectic representation**: Efficient binary encoding for computation
5. **Error correction**: Stabilizer commutation determines error detection
