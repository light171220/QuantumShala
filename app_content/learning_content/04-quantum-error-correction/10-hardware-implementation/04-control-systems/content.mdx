# Classical Control Systems for QEC

## Introduction

Quantum error correction requires tight integration between quantum hardware and classical control electronics. Real-time syndrome measurement, decoding, and feedback must occur faster than errors accumulate—pushing classical engineering to its limits.

DECODER IMPLEMENTATIONS ON FPGA:

1. LOOKUP TABLE (small codes)
    - Pre-compute all syndrome→correction mappings
    - O(1) lookup time
    - Memory: 2^(num_syndromes) entries
    - Works for d≤5 codes

2. MINIMUM WEIGHT MATCHING (FPGA)
    - Parallel graph construction
    - Pipelined matching
    - ~100 cycles for d=5
    - Challenging for d>7

3. UNION-FIND (FPGA)
    - Naturally parallel
    - Near-linear time
    - Good scaling to d=20+
    - Preferred for large codes

## Pauli Frame Tracking

```python
class PauliFrameTracker:
    """
    Software-based Pauli frame tracking.
    """

    def __init__(self, num_logical_qubits):
        self.num_qubits = num_logical_qubits
        # Track X and Z components for each qubit
        self.x_frame = [False] * num_logical_qubits
        self.z_frame = [False] * num_logical_qubits

    def apply_correction(self, qubit, pauli):
        """Update frame instead of applying physical gate."""
        if pauli in ['X', 'Y']:
            self.x_frame[qubit] = not self.x_frame[qubit]
        if pauli in ['Z', 'Y']:
            self.z_frame[qubit] = not self.z_frame[qubit]

    def propagate_through_clifford(self, gate, qubits):
        """
        Update frame when Clifford gate is applied.

        Paulis conjugate through Cliffords:
        H: X↔Z, Y→-Y
        S: X→Y, Y→-X, Z→Z
        CNOT: X⊗I→X⊗X, I⊗X→I⊗X, Z⊗I→Z⊗I, I⊗Z→Z⊗Z
        """
        if gate == 'H':
            q = qubits[0]
            # Swap X and Z frames
            self.x_frame[q], self.z_frame[q] = \
                self.z_frame[q], self.x_frame[q]

        elif gate == 'S':
            q = qubits[0]
            # X→Y means X frame adds to Z frame
            if self.x_frame[q]:
                self.z_frame[q] = not self.z_frame[q]

        elif gate == 'CNOT':
            ctrl, targ = qubits
            # X on control propagates to target
            if self.x_frame[ctrl]:
                self.x_frame[targ] = not self.x_frame[targ]
            # Z on target propagates to control
            if self.z_frame[targ]:
                self.z_frame[ctrl] = not self.z_frame[ctrl]

    def get_measurement_correction(self, qubit, basis='Z'):
        """Determine if measurement outcome should be flipped."""
        if basis == 'Z':
            return self.x_frame[qubit]
        elif basis == 'X':
            return self.z_frame[qubit]
        return False

    def demonstrate(self):
        """Show Pauli frame tracking in action."""
        print("""
Pauli Frame Tracking
════════════════════

WHY TRACK INSTEAD OF APPLY?

    Physical correction: Apply X gate (~20 ns)
    Frame tracking: Update bit in software (~1 ns)

    For Clifford circuits: corrections commute to end!

EXAMPLE:

    Initial: |ψ⟩, Frame = I

    1. Error detected: X error
       Option A: Apply X gate (slow)
       Option B: Update frame: Frame = X (fast)

    2. Apply H gate to qubit
       Frame propagates: X → Z (through H)
       New frame: Frame = Z

    3. Apply CNOT (this qubit is control)
       Frame propagates: Z → ZZ (Z on both qubits)

    4. Measure in Z basis
       If frame has X component: flip result
       Otherwise: result is correct

WHEN PHYSICAL GATES NEEDED:
    - Non-Clifford gates (T gates)
    - Frame would become non-Pauli
    - Must "cash out" frame before T gate
        """)


# Demonstration
tracker = PauliFrameTracker(2)
tracker.demonstrate()
```

## Summary

Classical control systems are essential for QEC:

1. **Architecture**: Hierarchical from host to FPGA to quantum
2. **Latency**: Must complete decode cycle < 1 μs for superconducting
3. **FPGAs**: Enable real-time decoding with deterministic timing
4. **Decoders**: Lookup tables for small codes, Union-Find for large
5. **Scaling**: Hierarchical control for millions of qubits
6. **Pauli Frame**: Software tracking avoids physical correction gates

The classical control system is as important as the quantum hardware for achieving fault tolerance.
