## 1. MBQC Overview

Measurement-based QC uses a highly entangled cluster state as a resource:

```python
import numpy as np

def create_cluster_state(n_qubits):
    """Create 1D cluster state |+>^n with CZ gates."""
    dim = 2**n_qubits
    # Start with |+>^n
    state = np.ones(dim) / np.sqrt(dim)

    # Apply CZ between neighbors
    for i in range(n_qubits - 1):
        state = apply_cz(state, i, i+1, n_qubits)

    return state

def apply_cz(state, q1, q2, n):
    """Apply CZ gate."""
    new_state = state.copy()
    for idx in range(len(state)):
        b1 = (idx >> (n-1-q1)) & 1
        b2 = (idx >> (n-1-q2)) & 1
        if b1 == 1 and b2 == 1:
            new_state[idx] *= -1
    return new_state

def measure_in_basis(state, qubit, angle, n_qubits):
    """Measure qubit in rotated basis."""
    # Measurement basis: cos(angle)|0> + sin(angle)|1>
    prob_0 = 0
    for idx in range(len(state)):
        bit = (idx >> (n_qubits-1-qubit)) & 1
        if bit == 0:
            prob_0 += np.abs(state[idx])**2

    outcome = 0 if np.random.random() < prob_0 else 1
    return outcome
```

## 2. Gate Teleportation

Single-qubit gates via measurement angles:
- X gate: measure at angle 0
- Z gate: measure at angle pi/2
- General rotation: adjust angle

## Summary

MBQC offers an alternative computation model:
1. Prepare entangled cluster state
2. Compute via adaptive single-qubit measurements
3. Feed-forward classical results
