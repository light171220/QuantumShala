## 1. Adiabatic Theorem

Slowly varying Hamiltonian keeps system in ground state:

<Formula>
$$H(s) = (1-s)H_0 + s H_P, \quad s \in [0,1]$$
</Formula>

```python
import numpy as np
from scipy.linalg import expm

def adiabatic_evolution(H_initial, H_final, T, dt=0.01):
    """Simulate adiabatic evolution."""
    steps = int(T / dt)
    dim = H_initial.shape[0]

    # Start in ground state of H_initial
    eigenvalues, eigenvectors = np.linalg.eigh(H_initial)
    state = eigenvectors[:, 0]

    for step in range(steps):
        s = step / steps
        H = (1 - s) * H_initial + s * H_final
        U = expm(-1j * H * dt)
        state = U @ state
        state = state / np.linalg.norm(state)

    return state

def check_adiabatic_condition(H_initial, H_final, T):
    """Check if evolution is adiabatic."""
    # Gap must be large compared to rate of change
    min_gap = float('inf')
    for s in np.linspace(0, 1, 100):
        H = (1 - s) * H_initial + s * H_final
        eigs = np.linalg.eigvalsh(H)
        gap = eigs[1] - eigs[0]
        min_gap = min(min_gap, gap)

    # Adiabatic condition: T >> 1/gap^2
    return T > 10 / min_gap**2, min_gap
```

## Summary

Adiabatic QC encodes solutions as ground states:
1. Start with easy ground state
2. Slowly evolve to problem Hamiltonian
3. Final state encodes solution
