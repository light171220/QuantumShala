## 1. The Importance of Circuit Visualization

Circuit visualization is the first line of defense in quantum debugging. Visual representations help identify structural issues, gate sequences, and potential sources of errors before execution.

```
Why Visualization Matters for Debugging
═══════════════════════════════════════

        Written Code                    Visual Circuit
        ─────────────                   ──────────────

        qc.h(0)                         q0: ──H──●──────
        qc.cx(0, 1)                              │
        qc.h(0)             ───>        q1: ─────X──H───
        qc.h(1)

        Can you spot the         Easy to see gate order
        gate sequence?           and qubit interactions!

Common Issues Caught by Visualization:
├── Incorrect gate order
├── Missing entanglement operations
├── Wrong qubit indices
├── Redundant gate sequences
└── Depth/parallelization issues
```

## 3. Visualization Output Formats

```python
class CircuitVisualizer:
    """Comprehensive circuit visualization toolkit."""

    def __init__(self, circuit: QuantumCircuit):
        self.circuit = circuit

    def text_diagram(self, fold: int = 80,
                     show_idle: bool = True) -> str:
        """Generate ASCII text diagram for terminal debugging."""
        return self.circuit.draw(
            output='text',
            fold=fold,
            idle_wires=show_idle,
            initial_state=True,
            cregbundle=True
        )

    def mpl_figure(self, style: str = 'iqp',
                   scale: float = 1.0,
                   filename: str = None):
        """Generate matplotlib figure for detailed analysis."""
        fig = self.circuit.draw(
            output='mpl',
            style=style,
            scale=scale,
            initial_state=True,
            plot_barriers=True
        )

        if filename:
            fig.savefig(filename, dpi=150, bbox_inches='tight')
        return fig

    def latex_source(self) -> str:
        """Generate LaTeX source for documentation."""
        return self.circuit.draw(output='latex_source')

    def get_circuit_stats(self) -> dict:
        """Extract circuit statistics for debugging."""
        return {
            'depth': self.circuit.depth(),
            'width': self.circuit.width(),
            'num_qubits': self.circuit.num_qubits,
            'num_clbits': self.circuit.num_clbits,
            'gate_counts': dict(self.circuit.count_ops()),
            'num_parameters': self.circuit.num_parameters,
            'global_phase': float(self.circuit.global_phase)
        }

# Usage example
visualizer = CircuitVisualizer(circuit)
print("Circuit Statistics:")
stats = visualizer.get_circuit_stats()
for key, value in stats.items():
    print(f"  {key}: {value}")
```

## 4. Layer-by-Layer Visualization

Breaking circuits into layers helps identify issues at specific points.

```python
from typing import List, Tuple

class LayerAnalyzer:
    """Analyze circuits layer by layer for debugging."""

    def __init__(self, circuit: QuantumCircuit):
        self.circuit = circuit
        self.layers = self._extract_layers()

    def _extract_layers(self) -> List[List[Tuple]]:
        """Extract circuit operations into layers."""
        layers = []
        current_layer = []
        qubits_used = set()

        for instruction, qargs, cargs in self.circuit.data:
            qubit_indices = [q._index for q in qargs]

            # Check if any qubit is already used in current layer
            if qubits_used.intersection(qubit_indices):
                # Start new layer
                if current_layer:
                    layers.append(current_layer)
                current_layer = [(instruction, qargs, cargs)]
                qubits_used = set(qubit_indices)
            else:
                current_layer.append((instruction, qargs, cargs))
                qubits_used.update(qubit_indices)

        if current_layer:
            layers.append(current_layer)

        return layers

    def visualize_layer(self, layer_idx: int) -> QuantumCircuit:
        """Create circuit showing only specified layer."""
        if layer_idx >= len(self.layers):
            raise IndexError(f"Layer {layer_idx} does not exist")

        layer_circuit = QuantumCircuit(self.circuit.num_qubits)
        for instruction, qargs, cargs in self.layers[layer_idx]:
            qubit_indices = [q._index for q in qargs]
            layer_circuit.append(instruction, qubit_indices)

        return layer_circuit

    def visualize_up_to_layer(self, layer_idx: int) -> QuantumCircuit:
        """Create circuit up to and including specified layer."""
        partial_circuit = QuantumCircuit(self.circuit.num_qubits)

        for i in range(min(layer_idx + 1, len(self.layers))):
            for instruction, qargs, cargs in self.layers[i]:
                qubit_indices = [q._index for q in qargs]
                partial_circuit.append(instruction, qubit_indices)

        return partial_circuit

    def print_layer_summary(self):
        """Print summary of all layers."""
        print(f"Circuit has {len(self.layers)} layers:")
        for i, layer in enumerate(self.layers):
            gates = [inst.name for inst, _, _ in layer]
            print(f"  Layer {i}: {', '.join(gates)}")

# Example usage
analyzer = LayerAnalyzer(circuit)
analyzer.print_layer_summary()

# Visualize specific layers
print("\nLayer 0 (initial operations):")
layer0 = analyzer.visualize_layer(0)
print(layer0.draw())
```

## 5. Cirq Circuit Visualization

Google's Cirq provides different visualization approaches.

```python
import cirq
import numpy as np

def create_cirq_debugging_circuit():
    """Create a Cirq circuit for visualization debugging."""

    # Define qubits
    qubits = cirq.LineQubit.range(3)

    # Build circuit with moments
    circuit = cirq.Circuit([
        cirq.Moment([cirq.H(qubits[0])]),
        cirq.Moment([cirq.CNOT(qubits[0], qubits[1])]),
        cirq.Moment([
            cirq.X(qubits[2]),
            cirq.Z(qubits[0])
        ]),
        cirq.Moment([cirq.CNOT(qubits[2], qubits[1])]),
        cirq.Moment([
            cirq.measure(qubits[0], key='q0'),
            cirq.measure(qubits[1], key='q1'),
            cirq.measure(qubits[2], key='q2')
        ])
    ])

    return circuit, qubits

class CirqVisualizer:
    """Visualization tools for Cirq circuits."""

    def __init__(self, circuit: cirq.Circuit):
        self.circuit = circuit

    def text_diagram(self,
                     transpose: bool = False,
                     include_tags: bool = True) -> str:
        """Generate text diagram."""
        return self.circuit.to_text_diagram(
            transpose=transpose,
            include_tags=include_tags
        )

    def svg_diagram(self) -> str:
        """Generate SVG representation."""
        return cirq.contrib.svg.SVGCircuit(self.circuit)

    def get_moment_details(self) -> List[dict]:
        """Get detailed information about each moment."""
        details = []
        for i, moment in enumerate(self.circuit):
            moment_info = {
                'moment_index': i,
                'operations': [],
                'qubits_used': set()
            }
            for op in moment:
                moment_info['operations'].append({
                    'gate': str(op.gate),
                    'qubits': [str(q) for q in op.qubits]
                })
                moment_info['qubits_used'].update(op.qubits)
            moment_info['qubits_used'] = [str(q) for q in moment_info['qubits_used']]
            details.append(moment_info)
        return details

    def check_circuit_structure(self) -> dict:
        """Analyze circuit structure for potential issues."""
        analysis = {
            'total_moments': len(self.circuit),
            'unique_gates': set(),
            'two_qubit_gates': 0,
            'measurements': 0,
            'empty_moments': 0
        }

        for moment in self.circuit:
            if len(moment) == 0:
                analysis['empty_moments'] += 1
            for op in moment:
                analysis['unique_gates'].add(type(op.gate).__name__)
                if len(op.qubits) == 2:
                    analysis['two_qubit_gates'] += 1
                if isinstance(op.gate, cirq.MeasurementGate):
                    analysis['measurements'] += 1

        analysis['unique_gates'] = list(analysis['unique_gates'])
        return analysis

# Example usage
cirq_circuit, qubits = create_cirq_debugging_circuit()
cirq_viz = CirqVisualizer(cirq_circuit)

print("Cirq Circuit Diagram:")
print(cirq_viz.text_diagram())

print("\nCircuit Structure Analysis:")
structure = cirq_viz.check_circuit_structure()
for key, value in structure.items():
    print(f"  {key}: {value}")
```

## 6. State Vector Visualization

Visualizing quantum states helps verify circuit correctness.

```python
from qiskit.quantum_info import Statevector
from qiskit.visualization import plot_bloch_multivector, plot_state_qsphere
import numpy as np

class StateVisualizer:
    """Visualize quantum states for debugging."""

    def __init__(self, circuit: QuantumCircuit):
        self.circuit = circuit
        # Remove measurements for statevector simulation
        self.sim_circuit = circuit.remove_final_measurements(inplace=False)

    def get_statevector(self) -> Statevector:
        """Get the statevector of the circuit."""
        return Statevector.from_instruction(self.sim_circuit)

    def print_state_amplitudes(self, threshold: float = 1e-10):
        """Print non-zero state amplitudes."""
        sv = self.get_statevector()
        data = sv.data
        n_qubits = int(np.log2(len(data)))

        print(f"State Vector ({n_qubits} qubits):")
        print("-" * 40)

        for i, amplitude in enumerate(data):
            if abs(amplitude) > threshold:
                # Convert index to binary string
                basis_state = format(i, f'0{n_qubits}b')
                magnitude = abs(amplitude)
                phase = np.angle(amplitude)

                print(f"|{basis_state}⟩: "
                      f"amplitude = {amplitude:.4f}, "
                      f"|amp| = {magnitude:.4f}, "
                      f"phase = {phase:.4f} rad")

    def compare_states(self, expected: np.ndarray,
                       tolerance: float = 1e-6) -> dict:
        """Compare actual state with expected state."""
        actual = self.get_statevector().data

        if len(actual) != len(expected):
            return {
                'match': False,
                'error': 'State dimensions do not match',
                'actual_dim': len(actual),
                'expected_dim': len(expected)
            }

        # Calculate fidelity
        fidelity = abs(np.vdot(actual, expected)) ** 2

        # Element-wise comparison
        differences = []
        for i, (a, e) in enumerate(zip(actual, expected)):
            if abs(a - e) > tolerance:
                n_qubits = int(np.log2(len(actual)))
                differences.append({
                    'index': i,
                    'basis_state': format(i, f'0{n_qubits}b'),
                    'actual': complex(a),
                    'expected': complex(e),
                    'difference': abs(a - e)
                })

        return {
            'match': len(differences) == 0,
            'fidelity': fidelity,
            'differences': differences
        }

    def visualize_evolution(self, steps: List[int] = None):
        """Visualize state evolution at different circuit depths."""
        if steps is None:
            steps = list(range(self.sim_circuit.depth() + 1))

        evolution_data = []

        for step in steps:
            # Create partial circuit up to this step
            partial = QuantumCircuit(self.sim_circuit.num_qubits)
            ops_added = 0

            for inst, qargs, cargs in self.sim_circuit.data:
                if ops_added >= step:
                    break
                qubit_indices = [q._index for q in qargs]
                partial.append(inst, qubit_indices)
                ops_added += 1

            sv = Statevector.from_instruction(partial)
            evolution_data.append({
                'step': step,
                'state': sv.data,
                'probabilities': sv.probabilities()
            })

        return evolution_data

# Example usage
state_viz = StateVisualizer(circuit)
state_viz.print_state_amplitudes()
```

## 7. Bloch Sphere Visualization

The Bloch sphere provides intuitive single-qubit state visualization.

```python
class BlochSphereDebugger:
    """Debug single-qubit states using Bloch sphere."""

    def __init__(self):
        self.states_history = []

    def qubit_to_bloch_coords(self, statevector: np.ndarray,
                               qubit_idx: int) -> Tuple[float, float, float]:
        """Extract Bloch coordinates for a specific qubit."""
        from qiskit.quantum_info import partial_trace, DensityMatrix

        n_qubits = int(np.log2(len(statevector)))
        dm = DensityMatrix(statevector)

        # Trace out all other qubits
        trace_qubits = [i for i in range(n_qubits) if i != qubit_idx]
        if trace_qubits:
            reduced_dm = partial_trace(dm, trace_qubits)
        else:
            reduced_dm = dm

        # Convert to Bloch vector
        rho = reduced_dm.data

        # Pauli matrices
        x = 2 * np.real(rho[0, 1])
        y = 2 * np.imag(rho[1, 0])
        z = np.real(rho[0, 0] - rho[1, 1])

        return (x, y, z)

    def track_qubit_evolution(self, circuit: QuantumCircuit,
                               qubit_idx: int) -> List[dict]:
        """Track a qubit's evolution through the circuit."""
        evolution = []
        sim_circuit = circuit.remove_final_measurements(inplace=False)

        # Track state after each gate on this qubit
        partial = QuantumCircuit(sim_circuit.num_qubits)
        gate_count = 0

        # Initial state
        sv = Statevector.from_instruction(partial)
        coords = self.qubit_to_bloch_coords(sv.data, qubit_idx)
        evolution.append({
            'step': 0,
            'gate': 'initial',
            'bloch_coords': coords,
            'purity': self._calculate_purity(sv.data, qubit_idx)
        })

        for inst, qargs, cargs in sim_circuit.data:
            qubit_indices = [q._index for q in qargs]
            partial.append(inst, qubit_indices)
            gate_count += 1

            # Only record when this qubit is involved
            if qubit_idx in qubit_indices:
                sv = Statevector.from_instruction(partial)
                coords = self.qubit_to_bloch_coords(sv.data, qubit_idx)
                evolution.append({
                    'step': gate_count,
                    'gate': inst.name,
                    'bloch_coords': coords,
                    'purity': self._calculate_purity(sv.data, qubit_idx)
                })

        return evolution

    def _calculate_purity(self, statevector: np.ndarray,
                          qubit_idx: int) -> float:
        """Calculate purity of reduced density matrix."""
        from qiskit.quantum_info import partial_trace, DensityMatrix

        n_qubits = int(np.log2(len(statevector)))
        dm = DensityMatrix(statevector)

        trace_qubits = [i for i in range(n_qubits) if i != qubit_idx]
        if trace_qubits:
            reduced_dm = partial_trace(dm, trace_qubits)
        else:
            reduced_dm = dm

        return np.real(np.trace(reduced_dm.data @ reduced_dm.data))

    def print_evolution_report(self, evolution: List[dict], qubit_idx: int):
        """Print formatted evolution report."""
        print(f"\nBloch Sphere Evolution for Qubit {qubit_idx}")
        print("=" * 60)
        print(f"{'Step':<6} {'Gate':<12} {'X':<10} {'Y':<10} {'Z':<10} {'Purity':<8}")
        print("-" * 60)

        for state in evolution:
            x, y, z = state['bloch_coords']
            print(f"{state['step']:<6} {state['gate']:<12} "
                  f"{x:<10.4f} {y:<10.4f} {z:<10.4f} "
                  f"{state['purity']:<8.4f}")

# Example usage
bloch_debugger = BlochSphereDebugger()
evolution = bloch_debugger.track_qubit_evolution(circuit, qubit_idx=0)
bloch_debugger.print_evolution_report(evolution, qubit_idx=0)
```

## 8. Interactive Visualization Tools

Modern tools support interactive debugging visualizations.

```python
class InteractiveCircuitExplorer:
    """Interactive tools for circuit exploration."""

    def __init__(self, circuit: QuantumCircuit):
        self.circuit = circuit
        self.checkpoints = {}

    def add_checkpoint(self, name: str, after_gate: int):
        """Add a checkpoint for state inspection."""
        self.checkpoints[name] = after_gate

    def inspect_checkpoint(self, name: str) -> dict:
        """Get state information at a checkpoint."""
        if name not in self.checkpoints:
            raise ValueError(f"Checkpoint '{name}' not found")

        gate_idx = self.checkpoints[name]
        partial = self._build_partial_circuit(gate_idx)

        sim_circuit = partial.remove_final_measurements(inplace=False)
        sv = Statevector.from_instruction(sim_circuit)

        return {
            'checkpoint': name,
            'gate_index': gate_idx,
            'statevector': sv.data,
            'probabilities': sv.probabilities_dict(),
            'circuit': partial.draw(output='text')
        }

    def _build_partial_circuit(self, up_to_gate: int) -> QuantumCircuit:
        """Build circuit up to specified gate index."""
        partial = QuantumCircuit(
            self.circuit.num_qubits,
            self.circuit.num_clbits
        )

        for i, (inst, qargs, cargs) in enumerate(self.circuit.data):
            if i >= up_to_gate:
                break
            qubit_indices = [q._index for q in qargs]
            clbit_indices = [c._index for c in cargs]
            partial.append(inst, qubit_indices, clbit_indices)

        return partial

    def compare_checkpoints(self, name1: str, name2: str) -> dict:
        """Compare states at two checkpoints."""
        state1 = self.inspect_checkpoint(name1)
        state2 = self.inspect_checkpoint(name2)

        sv1 = state1['statevector']
        sv2 = state2['statevector']

        # Calculate overlap
        overlap = abs(np.vdot(sv1, sv2)) ** 2

        return {
            'checkpoint_1': name1,
            'checkpoint_2': name2,
            'state_overlap': overlap,
            'probability_diff': {
                k: state2['probabilities'].get(k, 0) - state1['probabilities'].get(k, 0)
                for k in set(state1['probabilities']) | set(state2['probabilities'])
            }
        }

    def generate_debug_report(self) -> str:
        """Generate comprehensive debug report."""
        report = []
        report.append("=" * 60)
        report.append("CIRCUIT DEBUG REPORT")
        report.append("=" * 60)

        # Circuit statistics
        report.append("\n1. CIRCUIT STATISTICS")
        report.append("-" * 40)
        report.append(f"   Qubits: {self.circuit.num_qubits}")
        report.append(f"   Classical bits: {self.circuit.num_clbits}")
        report.append(f"   Depth: {self.circuit.depth()}")
        report.append(f"   Gate count: {sum(self.circuit.count_ops().values())}")

        # Gate breakdown
        report.append("\n2. GATE BREAKDOWN")
        report.append("-" * 40)
        for gate, count in sorted(self.circuit.count_ops().items()):
            report.append(f"   {gate}: {count}")

        # Checkpoints
        if self.checkpoints:
            report.append("\n3. CHECKPOINT STATES")
            report.append("-" * 40)
            for name, idx in sorted(self.checkpoints.items(),
                                     key=lambda x: x[1]):
                state = self.inspect_checkpoint(name)
                probs = state['probabilities']
                top_states = sorted(probs.items(),
                                   key=lambda x: x[1],
                                   reverse=True)[:3]
                report.append(f"\n   {name} (after gate {idx}):")
                for basis, prob in top_states:
                    report.append(f"      |{basis}⟩: {prob:.4f}")

        return "\n".join(report)

# Example usage
explorer = InteractiveCircuitExplorer(circuit)
explorer.add_checkpoint("after_bell", 2)
explorer.add_checkpoint("before_measure", 5)

print(explorer.generate_debug_report())
```

## 9. Debugging Visualization Best Practices

```
Best Practices for Circuit Visualization Debugging
═══════════════════════════════════════════════════

1. USE BARRIERS STRATEGICALLY
   ├── Separate logical sections
   ├── Mark checkpoint locations
   └── Prevent unwanted optimizations

2. CHOOSE APPROPRIATE OUTPUT FORMAT
   ├── text: Quick terminal debugging
   ├── mpl: Detailed analysis with colors
   ├── latex: Documentation and papers
   └── svg: Web integration

3. LAYER-BY-LAYER ANALYSIS
   ├── Verify each transformation step
   ├── Check gate parallelization
   └── Identify serialization bottlenecks

4. STATE VISUALIZATION AT CHECKPOINTS
   ├── Verify expected superpositions
   ├── Check entanglement creation
   └── Validate phase relationships

5. COMPARE EXPECTED VS ACTUAL
   ├── Build reference circuits
   ├── Use fidelity metrics
   └── Track probability distributions
```

## Summary

Circuit visualization is an essential debugging skill for quantum software engineers:

1. **Multiple Formats**: Use text for quick checks, matplotlib for detailed analysis
2. **Layer Analysis**: Break circuits into layers to isolate issues
3. **State Visualization**: Verify quantum states match expectations
4. **Bloch Sphere**: Intuitive single-qubit state debugging
5. **Checkpoints**: Mark and inspect intermediate states
6. **Cross-Platform**: Qiskit, Cirq, and other tools each have visualization strengths

Effective visualization catches errors early and provides insight into circuit behavior that code inspection alone cannot reveal.
