# Communication Efficiency in Quantum Federated Learning

Communication is often the bottleneck in federated learning. This lesson explores techniques to reduce bandwidth requirements while maintaining model quality.

## Learning Objectives

After completing this lesson, you will be able to:

- Understand communication bottlenecks in QFL
- Implement gradient compression techniques
- Apply parameter quantization
- Use local update accumulation
- Design communication-efficient protocols
- Balance communication and convergence trade-offs

## 1. Communication Bottlenecks

In federated learning, clients must transmit model updates to the server each round, creating significant bandwidth demands.

<Definition term="Communication Bottleneck">
The limitation on federated learning performance caused by the bandwidth required to transmit model parameters or gradients between clients and server.
</Definition>

```python
import numpy as np

def analyze_communication_cost():
    """
    Analyze communication costs in federated QML.
    """
    print("Communication Cost Analysis:")
    print("=" * 50)

    scenarios = [

        {'name': 'Medium VQC', 'qubits': 10, 'layers': 5, 'clients': 20},

    ]

    bytes_per_float = 4  # 32-bit floats

    for scenario in scenarios:
        # Parameters: 3 rotations per qubit per layer

        # Per round: clients upload gradients, download global model
        upload = scenario['clients'] * n_params * bytes_per_float
        download = scenario['clients'] * n_params * bytes_per_float
        total = upload + download

        # For 100 rounds
        n_rounds = 100
        total_100 = total * n_rounds

        print(f"\n{scenario['name']}:")
        print(f"  Qubits: {scenario['qubits']}, Layers: {scenario['layers']}")

    print("-" * 40)
```

</Tip>

## 2. Gradient Sparsification

Send only the largest gradient values to reduce communication.

<Formula>
Top-k sparsification:
$$\tilde{g} = \text{Top}_k(g) = \{g_i : |g_i| \geq |g_{(k)}|\}$$
where $g_{(k)}$ is the k-th largest absolute value.
</Formula>

```python
import numpy as np

class GradientSparsifier:
    """
    Sparsify gradients to reduce communication.
    """

    def __init__(self, compression_ratio=0.1):
        """
        Args:
            compression_ratio: Fraction of values to keep (0-1)
        """
        self.compression_ratio = compression_ratio
        self.residuals = {}  # Error feedback per client

    def sparsify(self, gradient, client_id=None):
        """
        Keep only top-k% of gradient values.

        Returns sparse representation (indices, values).
        """
        n = len(gradient)
        k = max(1, int(n * self.compression_ratio))

        # Add residual from previous round
        if client_id is not None and client_id in self.residuals:
            gradient = gradient + self.residuals[client_id]

        # Find top-k by absolute value
        abs_grad = np.abs(gradient)
        threshold_idx = np.argsort(abs_grad)[-k:]

        # Create sparse gradient
        sparse_indices = threshold_idx
        sparse_values = gradient[threshold_idx]

        # Store residual (values not sent)
        if client_id is not None:
            residual = gradient.copy()
            residual[threshold_idx] = 0
            self.residuals[client_id] = residual

        return {
            'indices': sparse_indices,
            'values': sparse_values,
            'original_size': n,
            'compressed_size': k,
            'ratio': k / n
        }

    def reconstruct(self, sparse_data):
        """Reconstruct full gradient from sparse representation."""
        gradient = np.zeros(sparse_data['original_size'])
        gradient[sparse_data['indices']] = sparse_data['values']
        return gradient

    def compute_savings(self, sparse_data):
        """Compute communication savings."""
        # Original: n floats
        # Compressed: k indices + k values
        original = sparse_data['original_size'] * 4  # bytes
        compressed = sparse_data['compressed_size'] * (4 + 4)  # value + index
        return 1 - compressed / original

# Demonstration
print("Gradient Sparsification:")
print("=" * 50)

np.random.seed(42)

# Create gradient with some large values
gradient = np.random.randn(100) * 0.1
gradient[10:15] = np.random.randn(5) * 2.0  # Some large values

print(f"\nOriginal gradient: {len(gradient)} values")
print(f"Gradient norm: {np.linalg.norm(gradient):.4f}")

sparsifier = GradientSparsifier(compression_ratio=0.1)

sparse = sparsifier.sparsify(gradient, client_id='client_0')
reconstructed = sparsifier.reconstruct(sparse)

print(f"\nSparsified:")
print(f"  Values kept: {sparse['compressed_size']}/{sparse['original_size']}")
print(f"  Compression ratio: {sparse['ratio']:.1%}")
print(f"  Savings: {sparsifier.compute_savings(sparse):.1%}")

error = np.linalg.norm(reconstructed - gradient)
print(f"\nReconstruction error: {error:.4f}")

# Error feedback helps over rounds
print("\n\nError Feedback Effect:")
print("-" * 40)
total_sent = np.zeros(100)
for round_num in range(5):
    gradient = np.random.randn(100) * 0.1
    sparse = sparsifier.sparsify(gradient, client_id='client_0')
    total_sent[sparse['indices']] += sparse['values']

print(f"After 5 rounds, residual norm: {np.linalg.norm(sparsifier.residuals['client_0']):.4f}")
```

<Warning>
Without error feedback, sparsification can cause some parameters to never update. Always accumulate residuals for the next round.
</Warning>

## 3. Gradient Quantization

Reduce precision of gradient values to use fewer bits.

```python
import numpy as np

class GradientQuantizer:
    """
    Quantize gradients to reduce bit-width.
    """

    def __init__(self, n_bits=8):
        self.n_bits = n_bits
        self.n_levels = 2 ** n_bits

    def quantize_uniform(self, gradient):
        """
        Uniform quantization to n_levels.
        """
        min_val = np.min(gradient)
        max_val = np.max(gradient)

        # Scale to [0, n_levels-1]
        if max_val > min_val:
            scaled = (gradient - min_val) / (max_val - min_val)
            quantized = np.round(scaled * (self.n_levels - 1)).astype(int)
        else:
            quantized = np.zeros_like(gradient, dtype=int)

        return {
            'values': quantized,
            'min': min_val,
            'max': max_val,
            'bits': self.n_bits
        }

    def dequantize(self, quantized_data):
        """Reconstruct gradient from quantized values."""
        min_val = quantized_data['min']
        max_val = quantized_data['max']
        values = quantized_data['values']

        if max_val > min_val:
            scaled = values / (self.n_levels - 1)
            gradient = scaled * (max_val - min_val) + min_val
        else:
            gradient = np.full_like(values, min_val, dtype=float)

        return gradient

    def stochastic_quantize(self, gradient):
        """
        Stochastic quantization (unbiased).

        Each value rounds up or down with probability
        proportional to distance.
        """
        min_val = np.min(gradient)
        max_val = np.max(gradient)

        if max_val <= min_val:
            return self.quantize_uniform(gradient)

        scaled = (gradient - min_val) / (max_val - min_val) * (self.n_levels - 1)

        # Stochastic rounding
        floor_val = np.floor(scaled).astype(int)
        ceil_val = np.ceil(scaled).astype(int)
        prob_ceil = scaled - floor_val

        random_vals = np.random.rand(len(gradient))
        quantized = np.where(random_vals < prob_ceil, ceil_val, floor_val)

        return {
            'values': quantized,
            'min': min_val,
            'max': max_val,
            'bits': self.n_bits
        }

    def compute_savings(self):
        """Compute bit savings vs 32-bit floats."""
        return 1 - self.n_bits / 32

# Demonstration
print("Gradient Quantization:")
print("=" * 50)

np.random.seed(42)
gradient = np.random.randn(100) * 0.5

print(f"\nOriginal: 32-bit floats, {len(gradient)} values")
print(f"Original size: {len(gradient) * 4} bytes")

for n_bits in [8, 4, 2, 1]:
    quantizer = GradientQuantizer(n_bits=n_bits)

    # Quantize and dequantize
    quantized = quantizer.quantize_uniform(gradient)
    reconstructed = quantizer.dequantize(quantized)

    error = np.linalg.norm(reconstructed - gradient)
    max_error = np.max(np.abs(reconstructed - gradient))

    print(f"\n{n_bits}-bit quantization:")
    print(f"  Levels: {quantizer.n_levels}")
    print(f"  Savings: {quantizer.compute_savings():.1%}")
    print(f"  RMS error: {error/len(gradient):.6f}")
    print(f"  Max error: {max_error:.6f}")
```

<Tip>
Stochastic quantization is unbiased in expectation, which can help convergence compared to deterministic rounding.
</Tip>

## 4. Combined Compression

Combine sparsification and quantization for maximum efficiency.

```python
import numpy as np

class CombinedCompressor:
    """
    Combined sparsification + quantization.
    """

    def __init__(self, sparsity=0.1, n_bits=8):
        self.sparsifier = GradientSparsifier(compression_ratio=sparsity)
        self.quantizer = GradientQuantizer(n_bits=n_bits)
        self.sparsity = sparsity
        self.n_bits = n_bits

    def compress(self, gradient, client_id=None):
        """Apply both sparsification and quantization."""
        # Step 1: Sparsify
        sparse = self.sparsifier.sparsify(gradient, client_id)

        # Step 2: Quantize the sparse values
        quantized = self.quantizer.quantize_uniform(sparse['values'])

        return {
            'indices': sparse['indices'],
            'quantized_values': quantized['values'],
            'min': quantized['min'],
            'max': quantized['max'],
            'original_size': sparse['original_size'],
            'compressed_size': sparse['compressed_size']
        }

    def decompress(self, compressed_data):
        """Reconstruct gradient."""
        # Dequantize values
        quantized_struct = {
            'values': compressed_data['quantized_values'],
            'min': compressed_data['min'],
            'max': compressed_data['max']
        }
        values = self.quantizer.dequantize(quantized_struct)

        # Reconstruct full gradient
        gradient = np.zeros(compressed_data['original_size'])
        gradient[compressed_data['indices']] = values

        return gradient

    def compute_compression_ratio(self, original_size):
        """Compute overall compression ratio."""
        # Original: original_size * 32 bits
        original_bits = original_size * 32

        # Compressed: k indices (32-bit) + k values (n_bits) + 2 floats (min/max)
        k = int(original_size * self.sparsity)
        compressed_bits = k * 32 + k * self.n_bits + 64

        return original_bits / compressed_bits

# Demonstration
print("Combined Compression:")
print("=" * 50)

np.random.seed(42)
gradient = np.random.randn(1000) * 0.5

print(f"\nOriginal gradient: {len(gradient)} parameters")
print(f"Original size: {len(gradient) * 4} bytes")

configs = [
    {'sparsity': 0.1, 'bits': 8},
    {'sparsity': 0.1, 'bits': 4},
    {'sparsity': 0.05, 'bits': 8},
    {'sparsity': 0.05, 'bits': 4},
]

for config in configs:
    compressor = CombinedCompressor(
        sparsity=config['sparsity'],
        n_bits=config['bits']
    )

    compressed = compressor.compress(gradient, client_id='test')
    reconstructed = compressor.decompress(compressed)

    error = np.linalg.norm(reconstructed - gradient) / len(gradient)
    ratio = compressor.compute_compression_ratio(len(gradient))

    print(f"\nSparsity={config['sparsity']:.0%}, Bits={config['bits']}:")
    print(f"  Compression ratio: {ratio:.1f}x")
    print(f"  RMS error: {error:.6f}")
```

## 5. Local Update Accumulation

Reduce communication frequency by accumulating multiple local updates.

<Formula>
Local SGD:
Perform $\tau$ local updates before communicating:
$$\theta_k^{t+\tau} = \theta_k^t - \eta \sum_{i=0}^{\tau-1} g_k^{t+i}$$
</Formula>

```python
import numpy as np

class LocalSGDClient:
    """
    Client with local update accumulation.
    """

    def __init__(self, n_params, sync_frequency=10):
        self.n_params = n_params
        self.sync_frequency = sync_frequency
        self.local_params = np.random.randn(n_params) * 0.1
        self.accumulated_updates = np.zeros(n_params)
        self.local_steps = 0

    def local_update(self, gradient, lr=0.01):
        """Perform local gradient step."""
        self.local_params -= lr * gradient
        self.accumulated_updates += gradient
        self.local_steps += 1

        return self.local_steps >= self.sync_frequency

    def get_update_for_sync(self):
        """Get accumulated update for synchronization."""
        update = self.accumulated_updates.copy()
        self.accumulated_updates = np.zeros(self.n_params)
        steps = self.local_steps
        self.local_steps = 0
        return update, steps

    def receive_global_model(self, global_params):
        """Update local model from global."""
        self.local_params = global_params.copy()

def local_sgd_simulation():
    """
    Simulate Local SGD vs standard FedAvg.
    """
    print("Local SGD vs Standard FedAvg:")
    print("=" * 50)

    np.random.seed(42)
    n_params = 100
    n_clients = 5
    n_total_updates = 100

    results = {}

    for sync_freq in [1, 5, 10, 20]:
        # Initialize clients
        clients = [LocalSGDClient(n_params, sync_frequency=sync_freq)
                  for _ in range(n_clients)]
        global_params = np.random.randn(n_params) * 0.1

        sync_count = 0
        total_updates = 0

        while total_updates < n_total_updates:
            # Local updates
            ready_clients = []
            for client in clients:
                gradient = np.random.randn(n_params) * 0.1  # Simulated
                should_sync = client.local_update(gradient)
                total_updates += 1
                if should_sync:
                    ready_clients.append(client)

            # Synchronize when clients are ready
            if ready_clients:
                updates = []
                for client in ready_clients:
                    update, steps = client.get_update_for_sync()
                    updates.append(update / steps)  # Average gradient

                # Aggregate
                avg_update = np.mean(updates, axis=0)
                global_params -= 0.01 * avg_update

                # Distribute
                for client in clients:
                    client.receive_global_model(global_params)

                sync_count += 1

        results[sync_freq] = {
            'syncs': sync_count,
            'updates': total_updates,
            'ratio': total_updates / sync_count
        }

    print("\n  Sync Freq | Syncs | Updates | Ratio")
    print("  " + "-" * 40)
    for freq, r in results.items():
        print(f"  {freq:9d} | {r['syncs']:5d} | {r['updates']:7d} | {r['ratio']:.1f}:1")

    print("\n\nCommunication savings:")
    baseline = results[1]['syncs']
    for freq in [5, 10, 20]:
        savings = 1 - results[freq]['syncs'] / baseline
        print(f"  Sync every {freq}: {savings:.1%} reduction")

local_sgd_simulation()
```

<Warning>
Longer sync intervals reduce communication but may slow convergence or cause client drift. Monitor convergence and adjust frequency as needed.
</Warning>

## 6. Communication-Efficient Protocols

Complete protocols combining multiple efficiency techniques.

```python
import numpy as np

class EfficientQFL:
    """
    Communication-efficient quantum federated learning.
    """

    def __init__(self, n_params, n_clients, config):
        self.n_params = n_params
        self.n_clients = n_clients
        self.config = config

        self.global_params = np.random.randn(n_params) * 0.1

        # Compression components
        self.compressor = CombinedCompressor(
            sparsity=config.get('sparsity', 0.1),
            n_bits=config.get('bits', 8)
        )

        # Statistics
        self.stats = {
            'rounds': 0,
            'bytes_sent': 0,
            'bytes_baseline': 0
        }

    def client_update(self, client_id, data_size):
        """Simulate client local training and compression."""
        # Simulate gradient computation
        gradient = np.random.randn(self.n_params) * 0.1

        # Compress
        compressed = self.compressor.compress(gradient, client_id)

        # Track communication
        bytes_sent = (len(compressed['indices']) * 4 +  # indices
                     len(compressed['quantized_values']) *
                         (self.config.get('bits', 8) / 8) +  # values
                     8)  # min/max
        self.stats['bytes_sent'] += bytes_sent
        self.stats['bytes_baseline'] += self.n_params * 4

        return compressed, data_size

    def aggregate(self, client_updates):
        """Aggregate compressed updates."""
        # Decompress and weight by data size
        total_data = sum(size for _, size in client_updates)
        aggregated = np.zeros(self.n_params)

        for compressed, data_size in client_updates:
            gradient = self.compressor.decompress(compressed)
            aggregated += (data_size / total_data) * gradient

        # Update global model
        lr = self.config.get('lr', 0.1)
        self.global_params -= lr * aggregated

        self.stats['rounds'] += 1

    def run_round(self, data_sizes):
        """Run one federated round."""
        updates = []
        for client_id, data_size in enumerate(data_sizes):
            compressed, size = self.client_update(client_id, data_size)
            updates.append((compressed, size))

        self.aggregate(updates)

    def get_stats(self):
        """Get communication statistics."""
        return {
            'rounds': self.stats['rounds'],
            'bytes_sent': self.stats['bytes_sent'],
            'bytes_baseline': self.stats['bytes_baseline'],
            'compression_ratio': (self.stats['bytes_baseline'] /
                                 self.stats['bytes_sent']
                                 if self.stats['bytes_sent'] > 0 else 0),
            'savings': (1 - self.stats['bytes_sent'] / self.stats['bytes_baseline']
                       if self.stats['bytes_baseline'] > 0 else 0)
        }

# Demonstration
print("Communication-Efficient QFL Protocol:")
print("=" * 50)

np.random.seed(42)

configs = [
    {'name': 'Baseline', 'sparsity': 1.0, 'bits': 32},
    {'name': 'Sparse', 'sparsity': 0.1, 'bits': 32},
    {'name': 'Quantized', 'sparsity': 1.0, 'bits': 8},
    {'name': 'Both', 'sparsity': 0.1, 'bits': 8},
    {'name': 'Aggressive', 'sparsity': 0.05, 'bits': 4},
]

n_params = 1000
n_clients = 10
n_rounds = 50
data_sizes = [100 + i * 10 for i in range(n_clients)]

print(f"\nSetup: {n_params} params, {n_clients} clients, {n_rounds} rounds")
print("\nResults:")
print("-" * 60)
print(f"{'Config':<12} {'Bytes Sent':<15} {'Compression':<12} {'Savings':<10}")
print("-" * 60)

for config in configs:
    qfl = EfficientQFL(n_params, n_clients, config)

    for _ in range(n_rounds):
        qfl.run_round(data_sizes)

    stats = qfl.get_stats()
    print(f"{config['name']:<12} {stats['bytes_sent']/1024:.1f} KB"
          f"{'':>7} {stats['compression_ratio']:.1f}x"
          f"{'':>8} {stats['savings']:.1%}")
```

<Tip>
Start with moderate compression and monitor model quality. Increase compression if convergence is maintained, or reduce if accuracy degrades.
</Tip>

## Summary

Communication efficiency in quantum federated learning can be improved through:

1. **Gradient Sparsification**: Send only top-k% largest values
2. **Quantization**: Reduce bit-width of transmitted values
3. **Combined Compression**: Apply both techniques together
4. **Local Accumulation**: Synchronize less frequently
5. **Protocol Design**: Balance compression with convergence

These techniques can reduce communication by 10-100x with minimal impact on model quality.
