# Distributed VQE

The Variational Quantum Eigensolver (VQE) can be distributed across multiple quantum devices to speed up computation and handle larger problems. This lesson explores federated approaches to VQE.

## Learning Objectives

After completing this lesson, you will be able to:

- Understand distributed VQE architectures
- Partition Hamiltonians across devices
- Implement parallel expectation value estimation
- Aggregate results from multiple quantum backends
- Handle device-specific calibration in VQE
- Optimize convergence in distributed settings

## 1. VQE Review and Distribution Motivation

VQE finds the ground state energy of a Hamiltonian by optimizing a parameterized quantum circuit.

<Definition term="Variational Quantum Eigensolver">
A hybrid quantum-classical algorithm that prepares a parameterized quantum state and optimizes parameters to minimize the expectation value of a Hamiltonian.
</Definition>

<Formula>
VQE objective:
$$E(\theta) = \langle \psi(\theta) | H | \psi(\theta) \rangle = \sum_i c_i \langle \psi(\theta) | P_i | \psi(\theta) \rangle$$
where $H = \sum_i c_i P_i$ is the Hamiltonian as a sum of Pauli terms.
</Formula>

```python
import numpy as np

class VQESimulator:
    """
    Basic VQE simulator for demonstration.
    """

    def __init__(self, n_qubits):
        self.n_qubits = n_qubits

    def prepare_state(self, params):
        """
        Simulate parameterized state preparation.
        Returns state vector.
        """
        # Simple variational ansatz simulation
        dim = 2 ** self.n_qubits
        state = np.zeros(dim, dtype=complex)
        state[0] = 1.0  # Start in |0...0⟩

        # Apply rotations (simplified)
        for i, p in enumerate(params[:self.n_qubits]):
            # Ry rotation on qubit i
            idx = 2 ** i
            new_state = state.copy()
            c, s = np.cos(p/2), np.sin(p/2)
            for j in range(0, dim, 2*idx):
                for k in range(idx):
                    a, b = state[j+k], state[j+k+idx]
                    new_state[j+k] = c*a - s*b
                    new_state[j+k+idx] = s*a + c*b
            state = new_state

        return state

    def measure_pauli_z(self, state, qubit):
        """Measure Z expectation on a qubit."""
        dim = len(state)
        expectation = 0.0
        idx = 2 ** qubit
        for i in range(dim):
            # +1 if bit is 0, -1 if bit is 1
            sign = 1 - 2 * ((i >> qubit) & 1)
            expectation += sign * np.abs(state[i])**2
        return expectation

    def compute_energy(self, params, hamiltonian):
        """
        Compute energy expectation value.

        Args:
            params: Circuit parameters
            hamiltonian: List of (coefficient, pauli_string) tuples
        """
        state = self.prepare_state(params)
        energy = 0.0

        for coeff, pauli in hamiltonian:
            # Simplified: only handle Z terms
            if pauli == 'I' * self.n_qubits:
                energy += coeff
            elif all(p in ['I', 'Z'] for p in pauli):
                term_exp = 1.0
                for q, p in enumerate(pauli):
                    if p == 'Z':
                        term_exp *= self.measure_pauli_z(state, q)
                energy += coeff * term_exp

        return energy
```

## 2. Hamiltonian Partitioning

Large Hamiltonians can be partitioned across devices for parallel evaluation.

```python
import numpy as np
from collections import defaultdict

class HamiltonianPartitioner:
    """
    Partition Hamiltonians for distributed evaluation.
    """

    def __init__(self, hamiltonian):
        """
        Args:
            hamiltonian: List of (coefficient, pauli_string) tuples
        """
        self.hamiltonian = hamiltonian
        self.n_terms = len(hamiltonian)

    def partition_by_count(self, n_partitions):
        """
        Simple partition by term count.
        """
        partition_size = (self.n_terms + n_partitions - 1) // n_partitions
        partitions = []

        for i in range(0, self.n_terms, partition_size):
            partitions.append(self.hamiltonian[i:i+partition_size])

        return partitions

    def partition_by_weight(self, n_partitions):
        """
        Partition to balance coefficient magnitudes.
        """
        # Sort by magnitude
        sorted_terms = sorted(self.hamiltonian,
                            key=lambda x: abs(x[0]), reverse=True)

        partitions = [[] for _ in range(n_partitions)]
        weights = [0.0] * n_partitions

        # Greedy assignment to balance weights
        for coeff, pauli in sorted_terms:
            # Assign to partition with smallest weight
            min_idx = np.argmin(weights)
            partitions[min_idx].append((coeff, pauli))
            weights[min_idx] += abs(coeff)

        return partitions, weights

    def partition_by_commutation(self, n_partitions):
        """
        Partition based on commuting groups.

        Terms that commute can be measured simultaneously.
        """
        def commutes(p1, p2):
            """Check if two Pauli strings commute."""
            anticommute_count = 0
            for a, b in zip(p1, p2):
                if a != 'I' and b != 'I' and a != b:
                    anticommute_count += 1
            return anticommute_count % 2 == 0

        # Greedy grouping
        groups = []
        for coeff, pauli in self.hamiltonian:
            placed = False
            for group in groups:
                # Check if commutes with all in group
                if all(commutes(pauli, p) for _, p in group):
                    group.append((coeff, pauli))
                    placed = True
                    break
            if not placed:
                groups.append([(coeff, pauli)])

        # Merge groups if too many
        while len(groups) > n_partitions:
            # Merge two smallest groups
            groups.sort(key=len)
            groups[0].extend(groups[1])
            groups.pop(1)

        return groups
```

## 3. Parallel Expectation Value Estimation

Different quantum devices evaluate different Hamiltonian partitions in parallel.

```python
import numpy as np
from concurrent.futures import ThreadPoolExecutor
import time

class DistributedVQE:
    """
    Distributed VQE across multiple simulated devices.
    """

    def __init__(self, n_qubits, n_devices):
        self.n_qubits = n_qubits
        self.n_devices = n_devices
        self.devices = [VQESimulator(n_qubits) for _ in range(n_devices)]
        self.device_times = [0.0] * n_devices

    def evaluate_partition(self, device_id, params, partition):
        """
        Evaluate Hamiltonian partition on one device.
        """
        start = time.time()

        partial_energy = 0.0
        for coeff, pauli in partition:
            # Simulate measurement
            state = self.devices[device_id].prepare_state(params)

            term_exp = 1.0
            for q, p in enumerate(pauli):
                if p == 'Z':
                    term_exp *= self.devices[device_id].measure_pauli_z(state, q)
                elif p == 'I':
                    pass  # Identity contributes 1
                else:
                    # X and Y need basis rotation (simplified)
                    term_exp *= np.random.uniform(-1, 1)

            partial_energy += coeff * term_exp

        elapsed = time.time() - start
        self.device_times[device_id] += elapsed

        return partial_energy

    def compute_energy_distributed(self, params, partitions):
        """
        Compute total energy using all devices in parallel.
        """
        with ThreadPoolExecutor(max_workers=self.n_devices) as executor:
            futures = []
            for i, partition in enumerate(partitions):
                device_id = i % self.n_devices
                future = executor.submit(
                    self.evaluate_partition, device_id, params, partition
                )
                futures.append(future)

            partial_energies = [f.result() for f in futures]

        return sum(partial_energies)

    def optimize(self, partitions, n_steps=20, lr=0.1):
        """
        Optimize VQE parameters.
        """
        params = np.random.randn(self.n_qubits) * 0.1
        energies = []

        for step in range(n_steps):
            # Compute gradient via parameter shift
            grad = np.zeros_like(params)
            shift = np.pi / 2

            for i in range(len(params)):
                params[i] += shift
                e_plus = self.compute_energy_distributed(params, partitions)
                params[i] -= 2 * shift
                e_minus = self.compute_energy_distributed(params, partitions)
                params[i] += shift

                grad[i] = (e_plus - e_minus) / 2

            params -= lr * grad
            energy = self.compute_energy_distributed(params, partitions)
            energies.append(energy)

        return params, energies
```

## 4. Result Aggregation and Error Handling

Combining results from different devices requires handling varying noise levels.

```python
import numpy as np

class DistributedResultAggregator:
    """
    Aggregate VQE results from multiple devices.
    """

    def __init__(self, n_devices):
        self.n_devices = n_devices
        self.device_errors = {}  # Track error estimates per device

    def estimate_variance(self, device_id, measurements, n_shots):
        """
        Estimate measurement variance for a device.
        """
        # Shot noise variance: var = (1 - <O>^2) / n_shots
        mean_val = np.mean(measurements)
        variance = (1 - mean_val**2) / n_shots
        self.device_errors[device_id] = np.sqrt(variance)
        return variance

    def weighted_aggregate(self, partial_energies, variances):
        """
        Combine partial energies with inverse variance weighting.
        """
        variances = np.array(variances)
        weights = 1.0 / (variances + 1e-10)
        weights /= np.sum(weights)

        total_energy = np.sum(weights * partial_energies)

        # Combined variance
        combined_var = 1.0 / np.sum(1.0 / (variances + 1e-10))

        return total_energy, combined_var, weights

    def detect_outliers(self, partial_energies, threshold=3.0):
        """
        Detect outlier results using z-scores.
        """
        mean = np.mean(partial_energies)
        std = np.std(partial_energies)

        z_scores = np.abs((partial_energies - mean) / (std + 1e-10))
        outliers = z_scores > threshold

        return outliers, z_scores

    def robust_aggregate(self, partial_energies, variances):
        """
        Aggregate with outlier detection.
        """
        # Detect outliers
        outliers, z_scores = self.detect_outliers(partial_energies)

        if np.any(outliers):
            # Exclude outliers
            valid = ~outliers
            partial_energies = np.array(partial_energies)[valid]
            variances = np.array(variances)[valid]

        return self.weighted_aggregate(partial_energies, variances)
```

## 5. Synchronization Strategies

Coordinating parameter updates across devices affects convergence.

```python
import numpy as np

class SynchronizedVQE:
    """
    VQE with different synchronization strategies.
    """

    def __init__(self, n_qubits, n_devices):
        self.n_qubits = n_qubits
        self.n_devices = n_devices
        self.global_params = np.random.randn(n_qubits) * 0.1

    def synchronous_update(self, device_gradients, lr=0.1):
        """
        Wait for all devices, then update.
        """
        # Average gradients from all devices
        avg_grad = np.mean(device_gradients, axis=0)
        self.global_params -= lr * avg_grad
        return self.global_params.copy()

    def asynchronous_update(self, device_id, gradient, lr=0.1,
                            staleness_weight=1.0):
        """
        Update immediately from any device.
        """
        self.global_params -= lr * staleness_weight * gradient
        return self.global_params.copy()

    def periodic_averaging(self, local_params_list, weights=None):
        """
        Periodically average parameters from all devices.
        """
        if weights is None:
            weights = np.ones(len(local_params_list)) / len(local_params_list)

        self.global_params = np.sum([w * p for w, p in
                                     zip(weights, local_params_list)], axis=0)
        return self.global_params.copy()

    def compare_strategies(self, hamiltonian, n_iterations=20):
        """
        Compare convergence of different strategies.
        """
        results = {}

        for strategy in ['synchronous', 'async', 'periodic']:
            np.random.seed(42)
            self.global_params = np.random.randn(self.n_qubits) * 0.1

            energies = []

            for it in range(n_iterations):
                # Simulate device computations
                device_gradients = []
                for d in range(self.n_devices):
                    # Add device-specific noise
                    noise = np.random.randn(self.n_qubits) * 0.1
                    grad = self.global_params + noise  # Simplified gradient
                    device_gradients.append(grad)

                if strategy == 'synchronous':
                    self.synchronous_update(device_gradients)
                elif strategy == 'async':
                    # Simulate random arrival order
                    order = np.random.permutation(self.n_devices)
                    for d in order:
                        staleness = np.random.uniform(0.5, 1.0)
                        self.asynchronous_update(d, device_gradients[d],
                                                staleness_weight=staleness)
                else:  # periodic
                    local_params = [self.global_params + g * 0.1
                                   for g in device_gradients]
                    if it % 5 == 4:  # Sync every 5 iterations
                        self.periodic_averaging(local_params)

                # Compute energy (simplified)
                energy = np.sum(self.global_params ** 2)
                energies.append(energy)

            results[strategy] = energies

        return results
```

## Summary

Distributed VQE enables solving larger quantum chemistry problems by:

1. **Partitioning**: Divide Hamiltonian terms across devices
2. **Parallel Evaluation**: Measure terms simultaneously on multiple backends
3. **Weighted Aggregation**: Combine results accounting for device quality
4. **Synchronization**: Choose appropriate update strategies
5. **Error Handling**: Detect and handle outlier results

This federated approach scales VQE beyond single-device limitations.
