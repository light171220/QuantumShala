## 1. Introduction to Integrated Photonics

Integrated photonics brings optical circuits onto chips, analogous to electronic integrated circuits.

<Definition term="Integrated Photonic Circuit">
An integrated photonic circuit is a device that confines and manipulates light within waveguides on a chip, enabling compact, stable, and scalable optical systems.
</Definition>

### Advantages for Quantum Computing

| Advantage | Description |
|-----------|-------------|
| Stability | No free-space alignment drift |
| Scalability | Lithographic fabrication scales to many components |
| Compactness | Thousands of components per cm^2 |
| Reproducibility | CMOS-compatible fabrication |
| Integration | On-chip sources, circuits, detectors |

## 2. Waveguide Physics

### Total Internal Reflection

Light is confined in waveguides through total internal reflection:

<Formula>
$$n_{core} \sin\theta > n_{cladding}$$
</Formula>

For single-mode operation, the waveguide dimensions must satisfy:

<Formula>
$$V = \frac{2\pi a}{\lambda}\sqrt{n_{core}^2 - n_{clad}^2} < 2.405$$
</Formula>

### Mode Confinement

The effective refractive index $n_{eff}$ satisfies:

<Formula>
$$n_{clad} < n_{eff} < n_{core}$$
</Formula>

```python
import numpy as np

def effective_index_slab(wavelength_um: float, width_um: float,
                         n_core: float, n_clad: float) -> float:
    """
    Estimate effective index for a slab waveguide (approximate).

    Args:
        wavelength_um: Wavelength in micrometers
        width_um: Waveguide width in micrometers
        n_core: Core refractive index
        n_clad: Cladding refractive index

    Returns:
        Approximate effective refractive index
    """
    # V-parameter
    V = (2 * np.pi * width_um / wavelength_um) * np.sqrt(n_core**2 - n_clad**2)

    # Approximate solution for fundamental mode
    # Using empirical fit for single-mode regime
    b = (1 - np.exp(-V/2)) * (1 - (1.1 + V/4) * np.exp(-V/1.5))
    b = max(0, min(b, 1))

    n_eff = np.sqrt(n_clad**2 + b * (n_core**2 - n_clad**2))
    return n_eff

# Silicon waveguide at 1550 nm
n_eff = effective_index_slab(1.55, 0.5, n_core=3.48, n_clad=1.44)
print(f"Effective index (Si at 1550 nm): {n_eff:.3f}")
```

## 3. Silicon Photonics

Silicon photonics leverages CMOS fabrication infrastructure.

<Definition term="Silicon Photonics">
Silicon photonics uses silicon-on-insulator (SOI) wafers to create optical waveguides and devices, benefiting from mature semiconductor manufacturing technology.
</Definition>

### Material Properties

| Property | Silicon | SiO2 (Cladding) |
|----------|---------|-----------------|
| Refractive index (1550 nm) | 3.48 | 1.44 |
| Transparency range | 1.1-8 um | 0.2-4 um |
| Two-photon absorption | Yes (at high power) | No |
| Thermo-optic coefficient | 1.8 x 10^-4 /K | 1 x 10^-5 /K |

### Key Components

#### Strip Waveguides
- Dimensions: ~500 nm x 220 nm
- High confinement, small bend radius (~5 um)
- Loss: 1-3 dB/cm

#### Directional Couplers (Beam Splitters)
Two waveguides in close proximity couple light:

<Formula>
$$\kappa = \frac{\pi \Delta n_{eff}}{wavelength}$$
</Formula>

Coupling ratio depends on interaction length.

#### Mach-Zehnder Interferometers
- Thermo-optic phase shifters
- Typical power: ~25 mW for pi shift
- Speed: ~100 kHz (thermal limited)

<Warning>
Silicon has strong two-photon absorption at telecom wavelengths, which can cause loss and noise for high-power applications. Single-photon experiments typically avoid this issue.
</Warning>

## 4. Lithium Niobate on Insulator (LNOI)

LNOI offers unique advantages for quantum photonics.

<Definition term="LNOI">
Lithium niobate on insulator combines thin-film lithium niobate (LN) with silica substrate, providing strong electro-optic modulation and nonlinear optical effects in a compact waveguide platform.
</Definition>

### Material Properties

| Property | LiNbO3 |
|----------|--------|
| Refractive index (1550 nm) | 2.21 (ordinary) |
| Electro-optic coefficient | r33 = 31 pm/V |
| Nonlinear coefficient | d33 = 25 pm/V |
| Transparency range | 0.4-5 um |

### Key Advantages

1. **Fast Modulation**: Electro-optic effect enables GHz+ switching
2. **Low Loss**: <0.3 dB/cm achievable
3. **SPDC Sources**: Native chi-2 nonlinearity for photon pair generation
4. **No TPA**: No two-photon absorption issues

```python
import numpy as np

def eo_phase_shift(voltage: float, electrode_gap: float,
                   length_mm: float, r33: float = 31e-12,
                   n: float = 2.21, wavelength: float = 1.55e-6) -> float:
    """
    Calculate electro-optic phase shift in LNOI modulator.

    Args:
        voltage: Applied voltage (V)
        electrode_gap: Electrode separation (m)
        length_mm: Interaction length (mm)
        r33: Electro-optic coefficient (m/V)
        n: Refractive index
        wavelength: Wavelength (m)

    Returns:
        Phase shift in radians
    """
    E_field = voltage / electrode_gap
    delta_n = 0.5 * n**3 * r33 * E_field
    phase = 2 * np.pi * delta_n * (length_mm * 1e-3) / wavelength
    return phase

# Example: V_pi calculation
for V in [1, 3, 5]:
    phase = eo_phase_shift(V, 10e-6, 10)
    print(f"V={V}V: phase shift = {phase:.3f} rad ({phase/np.pi:.3f} pi)")
```

## 5. Integrated Photon Sources

### SFWM in Silicon

Spontaneous four-wave mixing in silicon waveguides:

<Formula>
$$2\omega_{pump} = \omega_{signal} + \omega_{idler}$$
</Formula>

- Compact ring resonators enhance efficiency
- Generates photon pairs at telecom wavelengths

### SPDC in LNOI

Periodically poled lithium niobate (PPLN) enables efficient SPDC:

<Formula>
$$\omega_{pump} = \omega_{signal} + \omega_{idler}$$
$$k_{pump} = k_{signal} + k_{idler} + \frac{2\pi}{\Lambda}$$
</Formula>

where $\Lambda$ is the poling period.

<Tip>
LNOI SPDC sources can achieve >10^8 pairs/s/mW with high spectral purity, making them promising for integrated quantum photonics.
</Tip>

## 6. Integrated Detectors

### Waveguide-Integrated SNSPDs

SNSPDs can be fabricated directly on waveguides:

- NbN or NbTiN nanowires on top of waveguide
- Evanescent coupling absorption
- >90% on-chip detection efficiency demonstrated

### Ge-on-Si Photodetectors

For classical monitoring and feedback:
- Responsivity: ~1 A/W at 1550 nm
- Bandwidth: >40 GHz
- Dark current: <10 nA

## 7. Platform Comparison

| Feature | Silicon Photonics | LNOI |
|---------|-------------------|------|
| Modulation speed | ~100 kHz (thermal) | >40 GHz (EO) |
| Propagation loss | 1-3 dB/cm | 0.1-0.5 dB/cm |
| Nonlinearity | chi-3 (weak) | chi-2 (strong) |
| CMOS compatible | Yes | Improving |
| On-chip sources | SFWM | SPDC |
| Maturity | High | Medium |
| Cost | Low | Medium |

## 8. Programmable Photonic Processors

Large-scale integrated circuits enable programmable quantum operations.

### Mesh Architectures

Rectangular (Clements) and triangular (Reck) meshes of MZIs can implement arbitrary unitaries:

```python
import numpy as np

def mesh_component_count(n_modes: int, architecture: str = 'clements') -> dict:
    """
    Calculate components needed for programmable unitary mesh.

    Args:
        n_modes: Number of optical modes
        architecture: 'clements' or 'reck'

    Returns:
        Component counts
    """
    n_mzis = n_modes * (n_modes - 1) // 2

    if architecture == 'clements':
        depth = n_modes
    else:  # reck
        depth = 2 * n_modes - 3

    n_phase_shifters = n_mzis * 2 + n_modes  # Internal + external

    return {
        'n_modes': n_modes,
        'mzis': n_mzis,
        'phase_shifters': n_phase_shifters,
        'depth': depth
    }

# Compare for 12-mode processor
print("12-mode programmable processor:")
for arch in ['clements', 'reck']:
    counts = mesh_component_count(12, arch)
    print(f"  {arch}: {counts['mzis']} MZIs, depth = {counts['depth']}")
```

## 9. Challenges and Solutions

### Fabrication Variability

<Warning>
Waveguide dimensions vary across the chip, causing phase errors. Active calibration and trimming are essential for high-fidelity operations.
</Warning>

Solutions:
- Post-fabrication trimming (UV, thermal)
- Active phase correction with feedback
- Error-corrected encoding

### Coupling Losses

Fiber-to-chip coupling is a major loss source:
- Edge coupling: 1-3 dB/facet
- Grating coupling: 2-5 dB/facet
- Improving with inverse-design optimized couplers

### Thermal Management

Thermo-optic effects cause crosstalk:

<Formula>
$$\Delta\phi = \frac{2\pi}{\lambda} \frac{dn}{dT} \Delta T \cdot L$$
</Formula>

## 10. Hybrid Integration

Combining multiple platforms:

### III-V on Silicon
- InP lasers bonded to silicon
- Enables on-chip light sources

### SNSPD Integration
- NbN detectors on silicon waveguides
- Near-unity on-chip detection

### Heterogeneous Assemblies
- Silicon circuits + LNOI modulators + SNSPD detectors
- Best of each platform

## 11. Summary

Integrated photonics is essential for scalable photonic quantum computing:

1. **Silicon photonics** offers mature CMOS fabrication but thermal-limited modulation
2. **LNOI** provides fast electro-optic modulation and native nonlinearity
3. **Waveguide confinement** enables compact, stable circuits
4. **Programmable meshes** can implement arbitrary unitaries
5. **Hybrid integration** combines strengths of multiple platforms

The field is rapidly advancing toward fully integrated quantum photonic processors with on-chip sources, circuits, and detectors.
