## 1. The Single-Photon Source Challenge

Photonic quantum computing requires sources that emit exactly one photon at a time, on demand. This is surprisingly difficult to achieve.

<Definition term="Ideal Single-Photon Source">
An ideal single-photon source produces exactly one photon in a well-defined spatiotemporal mode with unity probability upon triggering, with all photons being perfectly indistinguishable.
</Definition>

### Key Figures of Merit

| Parameter | Definition | Ideal Value |
|-----------|------------|-------------|
| Efficiency | P(at least one photon) | 1.0 |
| Purity | P(exactly one photon given photon detected) | 1.0 |
| Indistinguishability | HOM visibility between consecutive photons | 1.0 |
| Brightness | Photon rate into useful mode | Maximum |

<Tip>
Classical light sources (lasers, LEDs) follow Poissonian statistics and cannot produce exactly one photon. Quantum sources are fundamentally different.
</Tip>

## 2. Spontaneous Parametric Down-Conversion (SPDC)

SPDC is the most widely used source of photon pairs and entangled photons.

### Physical Process

In SPDC, a pump photon interacting with a nonlinear crystal spontaneously splits into two lower-energy photons called **signal** and **idler**:

<Formula>
$$\omega_p = \omega_s + \omega_i$$
$$\vec{k}_p = \vec{k}_s + \vec{k}_i$$
</Formula>

These are the energy and momentum (phase-matching) conservation conditions.

<Definition term="Spontaneous Parametric Down-Conversion">
SPDC is a quantum nonlinear optical process where a photon from a pump beam spontaneously converts into a pair of correlated photons (signal and idler) in a chi-2 nonlinear crystal.
</Definition>

### Types of Phase Matching

| Type | Signal/Idler Polarizations | Crystal Examples |
|------|---------------------------|------------------|
| Type-0 | Same as pump | PPLN, PPKTP |
| Type-I | Same as each other, opposite to pump | BBO, LBO |
| Type-II | Orthogonal to each other | BBO, BiBO |

### SPDC State

The output state from SPDC (to first order) is:

<Formula>
$$|\psi\rangle = |0\rangle + \lambda |1_s, 1_i\rangle + O(\lambda^2)$$
</Formula>

where $\lambda \ll 1$ is the pair generation probability amplitude.

```python
import numpy as np

def spdc_statistics(mean_pair_number: float, max_pairs: int = 5) -> dict:
    """
    Calculate photon number statistics from SPDC source.

    SPDC produces thermal (geometric) statistics for the reduced state
    of signal or idler alone.

    Args:
        mean_pair_number: Mean number of pairs per pulse (lambda^2)
        max_pairs: Maximum number of pairs to consider

    Returns:
        Dictionary with probabilities for different pair numbers
    """
    mu = mean_pair_number
    probabilities = {}

    for n in range(max_pairs + 1):
        # Geometric distribution for pair number
        p_n = (mu ** n) / ((1 + mu) ** (n + 1))
        probabilities[n] = p_n

    return probabilities

# Example: Low-gain regime
stats = spdc_statistics(0.01)
print("SPDC pair statistics (mu = 0.01):")
for n, p in stats.items():
    print(f"  P({n} pairs) = {p:.6f}")
```

## 3. Heralded Single-Photon Generation

Detection of the idler photon "heralds" the presence of the signal photon.

<Definition term="Heralded Single Photon">
A heralded single photon is generated when detection of one photon from a pair (the herald) projects the other photon into a single-photon state.
</Definition>

### Heralding Efficiency

<Formula>
$$\eta_H = \frac{P(\text{signal detected} | \text{idler detected})}{1}$$
</Formula>

The heralding efficiency depends on:
- Collection efficiency
- Detector efficiency
- Spectral/spatial filtering

### Multi-Pair Contamination

<Warning>
Higher pump power increases brightness but also increases multi-pair emission, degrading the single-photon purity:

$$g^{(2)}(0) \approx 2\mu + O(\mu^2)$$

where $\mu$ is the mean pair number per pulse.
</Warning>

```python
import numpy as np

def heralded_source_metrics(mean_pairs: float, detection_efficiency: float,
                            herald_efficiency: float) -> dict:
    """
    Calculate metrics for a heralded single-photon source.

    Args:
        mean_pairs: Mean photon pairs per pulse
        detection_efficiency: Detector efficiency
        herald_efficiency: Probability signal is collected given idler detected

    Returns:
        Dictionary with source metrics
    """
    mu = mean_pairs

    # Herald rate (probability of detecting idler)
    herald_rate = 1 - np.exp(-mu * detection_efficiency)

    # Conditional single-photon state purity
    # P(1 signal | herald) vs P(>1 signal | herald)
    p_one = np.exp(-mu) * mu
    p_more = 1 - np.exp(-mu) * (1 + mu)
    purity = p_one / (p_one + p_more) if (p_one + p_more) > 0 else 0

    # g^(2)(0) approximation
    g2 = 2 * mu

    return {
        'herald_rate': herald_rate,
        'single_photon_purity': purity,
        'g2_zero': g2,
        'heralding_efficiency': herald_efficiency
    }

# Typical SPDC source
metrics = heralded_source_metrics(0.01, 0.9, 0.7)
print("Heralded source metrics:")
for key, value in metrics.items():
    print(f"  {key}: {value:.4f}")
```

## 4. Quantum Dot Single-Photon Sources

Quantum dots (QDs) are semiconductor nanostructures that act as artificial atoms, providing deterministic single-photon emission.

<Definition term="Quantum Dot">
A quantum dot is a nanoscale semiconductor structure (typically 2-20 nm) that confines electrons and holes in all three dimensions, creating discrete energy levels similar to atoms.
</Definition>

### Self-Assembled InAs/GaAs Quantum Dots

The most common QD platform uses InAs islands grown on GaAs substrates:

- **Emission wavelength**: 900-1000 nm (tunable to telecom via metamorphic growth)
- **Radiative lifetime**: ~1 ns
- **Operating temperature**: 4-20 K (typically)

### Two-Level System Dynamics

The QD ground-to-exciton transition behaves as a two-level system:

<Formula>
$$\frac{d\rho_{ee}}{dt} = -\gamma \rho_{ee} + \Omega \rho_{eg} + \Omega^* \rho_{ge}$$
</Formula>

where $\gamma$ is the spontaneous emission rate and $\Omega$ is the driving Rabi frequency.

### Antibunching

QDs naturally produce antibunched light with $g^{(2)}(0) \approx 0$:

<Formula>
$$g^{(2)}(\tau) = 1 - e^{-|\tau|/\tau_0}$$
</Formula>

```python
import numpy as np
import matplotlib.pyplot as plt

def g2_quantum_dot(tau: np.ndarray, lifetime_ns: float) -> np.ndarray:
    """
    Calculate g^(2)(tau) for an ideal quantum dot.

    Args:
        tau: Time delay array in nanoseconds
        lifetime_ns: Radiative lifetime in nanoseconds

    Returns:
        g^(2)(tau) values
    """
    return 1 - np.exp(-np.abs(tau) / lifetime_ns)

# Plot g^(2)(tau) for a QD
tau = np.linspace(-5, 5, 200)
g2 = g2_quantum_dot(tau, lifetime_ns=1.0)

plt.figure(figsize=(8, 5))
plt.plot(tau, g2, 'b-', linewidth=2)
plt.axhline(y=1, color='r', linestyle='--', label='Classical limit')
plt.axhline(y=0.5, color='g', linestyle='--', label='Non-classical threshold')
plt.xlabel('Time delay (ns)')
plt.ylabel('g^(2)(tau)')
plt.title('Photon Antibunching from Quantum Dot')
plt.legend()
plt.grid(True)
plt.ylim(-0.1, 1.5)
plt.show()
```

## 5. Resonance Fluorescence

The highest-quality single photons from QDs use **resonance fluorescence** - direct excitation at the emission wavelength.

### Advantages of Resonance Fluorescence

1. **No timing jitter**: Photon emission is synchronized to excitation pulse
2. **High indistinguishability**: Minimal dephasing from phonons
3. **Transform-limited linewidth**: Near-ideal spectral purity

### Experimental Challenge

<Warning>
Resonance fluorescence requires separating the weak single-photon signal from the strong excitation laser. This is typically done using:
- Cross-polarization filtering
- Spatial mode filtering
- Time-gating
</Warning>

## 6. Photon Indistinguishability

For quantum computing, photons must be indistinguishable to enable interference (e.g., HOM effect).

<Definition term="Photon Indistinguishability">
Photon indistinguishability quantifies how identical two photons are in all degrees of freedom (frequency, timing, polarization, spatial mode). It is measured by the HOM interference visibility.
</Definition>

### Sources of Distinguishability

| Source | Effect | Mitigation |
|--------|--------|------------|
| Spectral wandering | Random emission frequency | Charge stabilization |
| Pure dephasing | Broadened linewidth | Resonant excitation |
| Timing jitter | Variable emission time | Pulsed excitation |
| Phonon sidebands | Extra spectral features | Spectral filtering |

### Measuring Indistinguishability

The HOM visibility directly measures indistinguishability:

<Formula>
$$V_{HOM} = \frac{P_{classical} - P_{coincidence}}{P_{classical}}$$
</Formula>

For perfectly indistinguishable photons, $V_{HOM} = 1$ (no coincidences at zero delay).

```python
import numpy as np

def hom_visibility_from_spectra(spectrum1: np.ndarray, spectrum2: np.ndarray,
                                wavelengths: np.ndarray) -> float:
    """
    Estimate HOM visibility from spectral overlap.

    Args:
        spectrum1, spectrum2: Normalized spectral intensity distributions
        wavelengths: Wavelength array

    Returns:
        Estimated visibility from spectral overlap
    """
    # Spectral overlap integral
    overlap = np.trapz(np.sqrt(spectrum1 * spectrum2), wavelengths)
    norm1 = np.trapz(spectrum1, wavelengths)
    norm2 = np.trapz(spectrum2, wavelengths)

    spectral_indist = overlap**2 / (norm1 * norm2)
    return spectral_indist

# Example: Two Gaussian spectra with slight frequency difference
wavelengths = np.linspace(900, 910, 1000)
center1 = 905.0
center2 = 905.1  # 0.1 nm detuning
width = 0.2  # nm

spectrum1 = np.exp(-((wavelengths - center1)**2) / (2 * width**2))
spectrum2 = np.exp(-((wavelengths - center2)**2) / (2 * width**2))

visibility = hom_visibility_from_spectra(spectrum1, spectrum2, wavelengths)
print(f"HOM visibility estimate: {visibility:.4f}")
```

## 7. Multiplexed Photon Sources

To overcome the probabilistic nature of SPDC, multiplexing combines many sources.

### Spatial Multiplexing

Multiple SPDC sources are combined with active switching:

<Formula>
$$P_{\text{output}}(1) = 1 - (1-p)^N$$
</Formula>

where $p$ is single-source probability and $N$ is the number of sources.

### Temporal Multiplexing

Photons from different time bins are routed to a single output:

```python
import numpy as np

def multiplexed_efficiency(single_source_prob: float, num_sources: int,
                           switch_loss: float = 0.9) -> dict:
    """
    Calculate effective efficiency of a multiplexed source.

    Args:
        single_source_prob: Probability of photon from each source
        num_sources: Number of multiplexed sources
        switch_loss: Transmission through switching network

    Returns:
        Dictionary with efficiency metrics
    """
    p = single_source_prob

    # Probability of at least one photon
    p_any = 1 - (1 - p) ** num_sources

    # After switch losses (approximation for active switching)
    switch_stages = int(np.ceil(np.log2(num_sources)))
    total_switch_transmission = switch_loss ** switch_stages

    effective_efficiency = p_any * total_switch_transmission

    # Multi-photon probability
    p_multi = 1 - (1-p)**num_sources - num_sources * p * (1-p)**(num_sources-1)

    return {
        'raw_probability': p_any,
        'switch_transmission': total_switch_transmission,
        'effective_efficiency': effective_efficiency,
        'multi_photon_prob': p_multi / p_any if p_any > 0 else 0
    }

# Compare different multiplexing levels
for N in [1, 4, 16, 64]:
    result = multiplexed_efficiency(0.01, N)
    print(f"N={N}: efficiency={result['effective_efficiency']:.4f}")
```

## 8. Comparison of Source Technologies

| Property | SPDC (Heralded) | Quantum Dot | Multiplexed SPDC |
|----------|-----------------|-------------|------------------|
| Deterministic | No | Yes | Quasi |
| g^(2)(0) | ~0.01 | <0.01 | ~0.01 |
| Indistinguishability | 90-99% | 95-99.9% | 90-95% |
| Wavelength | Flexible | Fixed (tunable) | Flexible |
| Temperature | Room temp | 4-20 K | Room temp |
| Scalability | Challenging | Promising | Moderate |

<Tip>
Quantum dots currently hold records for indistinguishability (>99%) and brightness, but require cryogenic operation. SPDC is more accessible but fundamentally probabilistic.
</Tip>

## 9. Emerging Technologies

### Defect Centers in Diamond

NV centers and SiV centers provide room-temperature single-photon emission with spin-photon interfaces.

### Color Centers in Silicon Carbide

T centers in SiC emit at telecom wavelengths (1278 nm) with good spin properties.

### Organic Molecules

Single molecules in solid matrices provide bright, stable single-photon sources at low temperatures.

## 10. Summary

Single-photon sources are critical components for photonic quantum computing:

1. **SPDC** provides probabilistic but wavelength-flexible photon pairs
2. **Heralding** projects one photon of a pair into a single-photon state
3. **Quantum dots** offer deterministic emission with excellent properties
4. **Indistinguishability** is essential for multi-photon interference
5. **Multiplexing** can improve effective efficiency of probabilistic sources

The choice of source depends on application requirements: QDs for highest quality, SPDC for flexibility and room-temperature operation.
