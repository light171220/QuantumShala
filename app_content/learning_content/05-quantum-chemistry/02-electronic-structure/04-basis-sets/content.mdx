## 1. Why We Need Basis Sets

The molecular orbitals in the Schrödinger equation are unknown functions. We approximate them as linear combinations of known functions:

<Formula>
$$\phi_i(\mathbf{r}) = \sum_{\mu=1}^{K} c_{\mu i}\chi_\mu(\mathbf{r})$$
</Formula>

<Definition term="Basis Set">
A set of mathematical functions used to represent molecular orbitals. The quality of the basis set determines how accurately we can describe the electronic wavefunction.
</Definition>

```python
def basis_set_concept():
    """Illustrate the basis set expansion concept."""

    print("Basis Set Expansion")
    print("=" * 60)

    print("\nProblem: Molecular orbitals φᵢ(r) are unknown functions")
    print()
    print("Solution: Expand in known functions (basis functions)")
    print()
    print("  φᵢ(r) = Σμ cμᵢ χμ(r)")
    print()
    print("Where:")
    print("  φᵢ   = molecular orbital i")
    print("  χμ   = basis function μ (known)")
    print("  cμᵢ  = expansion coefficient (unknown, to be optimized)")
    print("  K    = number of basis functions")

    print("\nThis transforms the problem from:")
    print("  Finding unknown functions → Finding unknown coefficients")
    print()
    print("The Schrödinger equation becomes a matrix equation!")
    print("  HC = SCE  (Roothaan equations)")

basis_set_concept()
```

<Tip>
The basis set expansion converts a differential equation into a linear algebra problem—finding matrix eigenvectors instead of solving for unknown functions.
</Tip>

## 2. Slater-Type Orbitals (STOs)

The natural choice for atomic basis functions:

<Formula>
$$\chi^{STO}_{nlm}(r,\theta,\phi) = N r^{n-1} e^{-\zeta r} Y_l^m(\theta,\phi)$$
</Formula>

<Definition term="Slater-Type Orbital (STO)">
Basis functions that mimic the exact hydrogen atom solutions, with an exponential radial decay e^(-ζr). They have correct nuclear cusp behavior but are difficult to integrate.
</Definition>

```python
import numpy as np

def slater_orbital(r, n, zeta):
    """Radial part of a Slater-type orbital."""
    N = (2*zeta)**n * np.sqrt(2*zeta / np.math.factorial(2*n))
    return N * r**(n-1) * np.exp(-zeta * r)

def sto_properties():
    """Show properties of Slater-type orbitals."""

    print("Slater-Type Orbitals (STOs)")
    print("=" * 55)

    print("\nGeneral form: χ(r) = N r^(n-1) exp(-ζr) Y_l^m(θ,φ)")

    print("\nAdvantages:")
    print("  ✓ Correct cusp at nucleus (derivative condition)")
    print("  ✓ Correct exponential decay at large r")
    print("  ✓ Fewer functions needed for accuracy")

    print("\nDisadvantages:")
    print("  ✗ No analytical formula for 2-center integrals")
    print("  ✗ Very expensive numerical integration")
    print("  ✗ Not practical for large molecules")

    print("\nTypical exponents ζ:")
    exponents = {
        'H (1s)': 1.0,
        'C (1s)': 5.67,
        'C (2s)': 1.72,
        'C (2p)': 1.72,
        'O (2p)': 2.25,
    }
    for orbital, zeta in exponents.items():
        print(f"  {orbital}: ζ = {zeta}")

sto_properties()
```

## 3. Gaussian-Type Orbitals (GTOs)

The practical workhorse of quantum chemistry:

<Formula>
$$\chi^{GTO}(r,\theta,\phi) = N x^l y^m z^n e^{-\alpha r^2}$$
</Formula>

<Definition term="Gaussian-Type Orbital (GTO)">
Basis functions with Gaussian radial decay e^(-αr²). While individually less accurate than STOs, they have analytical integral formulas and can be combined (contracted) to approximate STOs efficiently.
</Definition>

```python
import numpy as np

def gaussian_orbital(r, alpha, l=0):
    """Radial part of a Gaussian-type orbital."""
    return r**l * np.exp(-alpha * r**2)

def gto_properties():
    """Show properties of Gaussian-type orbitals."""

    print("Gaussian-Type Orbitals (GTOs)")
    print("=" * 55)

    print("\nGeneral form: χ(r) = N x^l y^m z^n exp(-αr²)")

    print("\nAdvantages:")
    print("  ✓ Product of two Gaussians is a Gaussian!")
    print("    exp(-α|r-A|²) × exp(-β|r-B|²) = K exp(-γ|r-P|²)")
    print("  ✓ All integrals have analytical formulas")
    print("  ✓ Very efficient computation")

    print("\nDisadvantages:")
    print("  ✗ Wrong cusp at nucleus (derivative = 0)")
    print("  ✗ Decays too fast at large r")
    print("  ✗ Need more functions than STOs")

    print("\nSolution: Use CONTRACTED Gaussians")
    print("  Combine multiple GTOs to approximate one STO")

gto_properties()
```

## 4. Contracted Gaussian Functions

Multiple primitive GTOs are combined to better approximate STOs:

<Formula>
$$\chi^{CGTO}_\mu(\mathbf{r}) = \sum_{p=1}^{L} d_{p\mu} g_p(\alpha_p, \mathbf{r})$$
</Formula>

```python
def contracted_gaussians():
    """Demonstrate contracted Gaussian basis functions."""

    print("Contracted Gaussian Functions")
    print("=" * 60)

    print("\nIdea: Sum of Gaussians ≈ Slater orbital")
    print()
    print("  χ_contracted(r) = Σₚ dₚ gₚ(αₚ, r)")
    print()
    print("Where dₚ are contraction coefficients")

    # STO-3G example for hydrogen 1s
    sto3g_h = [
        (0.1688554, 3.42525091),   # (coefficient, exponent)
        (0.6239137, 0.62391373),
        (0.4446345, 0.16885540),
    ]

    print("\nExample: STO-3G for H 1s orbital")
    print(f"{'Coefficient':<15} {'Exponent α':<15}")
    print("-" * 30)
    for coef, exp in sto3g_h:
        print(f"{coef:<15.6f} {exp:<15.6f}")

    print("\nNaming convention:")
    print("  STO-3G: 3 Gaussians per STO")
    print("  STO-6G: 6 Gaussians per STO")
    print("  More Gaussians → Better STO approximation")

contracted_gaussians()
```

## 6. Polarization and Diffuse Functions

Basis sets can be augmented for better accuracy:

<Definition term="Polarization Functions">
Higher angular momentum functions added to describe how electron density distorts (polarizes) during bonding. For example, adding p-functions to hydrogen or d-functions to carbon.
</Definition>

<Definition term="Diffuse Functions">
Basis functions with small exponents that extend far from the nucleus. Essential for anions, weak interactions, and excited states.
</Definition>

```python
def augmented_basis_sets():
    """Explain polarization and diffuse functions."""

    print("Augmenting Basis Sets")
    print("=" * 60)

    print("\n1. POLARIZATION FUNCTIONS (higher angular momentum)")
    print("-" * 50)

    polarization = [
        ('H, He', 's', 'p', 'Allows H to polarize in bonds'),
        ('Li-Ne', 'sp', 'd', 'Describes bonding distortion'),
        ('Na-Ar', 'spd', 'f', 'Important for hypervalent'),
    ]

    print(f"{'Atoms':<10} {'Valence':<8} {'Add':<8} {'Purpose':<30}")
    for atoms, valence, add, purpose in polarization:
        print(f"{atoms:<10} {valence:<8} {add:<8} {purpose:<30}")

    print("\n2. DIFFUSE FUNCTIONS (small exponents, extended)")
    print("-" * 50)
    print("Notation: '+' or 'aug-' prefix")
    print()
    print("Essential for:")
    print("  • Anions (extra electron is loosely bound)")
    print("  • Rydberg excited states")
    print("  • Weak interactions (van der Waals)")
    print("  • Molecular polarizabilities")

    print("\n3. COMMON NOTATION:")
    print("-" * 50)

    notations = [
        ('6-31G', 'Double-ζ valence'),
        ('6-31G*', 'Add d to heavy atoms'),
        ('6-31G**', 'Add d to heavy, p to H'),
        ('6-31+G*', 'Add diffuse to heavy atoms'),
        ('6-31++G**', 'Diffuse on all atoms'),
        ('aug-cc-pVDZ', 'Dunning DZ with diffuse'),
    ]

    for basis, description in notations:
        print(f"  {basis:<15} {description}")

augmented_basis_sets()
```

## 7. Basis Set Convergence

As basis sets grow, energies approach the complete basis set (CBS) limit:

<Formula>
$$E_{CBS} = \lim_{X\to\infty} E_X$$
</Formula>

```python
import numpy as np

def basis_set_convergence():
    """Demonstrate convergence to CBS limit."""

    print("Basis Set Convergence")
    print("=" * 60)

    # Example: H2O HF energy (approximate values)
    data = {
        'STO-3G':     (-74.96, 7),
        '3-21G':      (-75.58, 13),
        '6-31G*':     (-76.01, 19),
        'cc-pVDZ':    (-76.02, 24),
        'cc-pVTZ':    (-76.05, 58),
        'cc-pVQZ':    (-76.06, 115),
        'cc-pV5Z':    (-76.07, 201),
        'CBS limit':  (-76.07, float('inf'))
    }

    print("\nH₂O Hartree-Fock Energy Convergence:")
    print(f"{'Basis Set':<15} {'Energy (Eh)':<15} {'# Functions':<15}")
    print("-" * 45)

    for basis, (energy, n_funcs) in data.items():
        n_str = str(n_funcs) if n_funcs != float('inf') else '∞'
        print(f"{basis:<15} {energy:<15.3f} {n_str:<15}")

    print("\nKey insight: Diminishing returns as basis grows")
    print("Cost scales as O(N⁴) for HF, O(N⁵-N⁷) for correlation!")

basis_set_convergence()
```

<Warning>
Larger basis sets improve accuracy but increase computational cost dramatically. The number of two-electron integrals scales as O(N⁴), where N is the number of basis functions.
</Warning>

## Summary

In this lesson, you learned:

1. **Basis sets** represent unknown orbitals as linear combinations of known functions
2. **STOs** are physically correct but computationally expensive
3. **GTOs** have efficient integral formulas and are universally used
4. **Contracted functions** combine multiple primitives
5. **Polarization and diffuse** functions improve accuracy for specific applications
6. **Larger basis sets** converge to the CBS limit at increased cost
7. **Quantum computing** qubit counts depend on basis set size

The choice of basis set balances accuracy against computational resources.
