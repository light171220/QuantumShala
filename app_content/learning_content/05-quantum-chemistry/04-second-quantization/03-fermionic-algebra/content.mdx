## 1. Fermionic Anticommutation Review

<Definition term="Anticommutation Relations">
The fundamental algebraic rules for fermionic operators: {cᵢ, c†ⱼ} = δᵢⱼ, {cᵢ, cⱼ} = 0, {c†ᵢ, c†ⱼ} = 0, where {A,B} = AB + BA.
</Definition>

```python
def anticommutation_review():
    """Review fermionic anticommutation relations."""

    print("Fermionic Anticommutation Relations")
    print("=" * 60)

    print("\nFundamental Relations:")
    print("  {cᵢ, c†ⱼ} = cᵢc†ⱼ + c†ⱼcᵢ = δᵢⱼ")
    print("  {cᵢ, cⱼ} = cᵢcⱼ + cⱼcᵢ = 0")
    print("  {c†ᵢ, c†ⱼ} = c†ᵢc†ⱼ + c†ⱼc†ᵢ = 0")

    print("\nKey Consequences:")
    print("-" * 45)

    print("\n1. Operator Swapping:")
    print("   cᵢcⱼ = -cⱼcᵢ  (i ≠ j)")
    print("   c†ᵢc†ⱼ = -c†ⱼc†ᵢ  (i ≠ j)")
    print("   cᵢc†ⱼ = δᵢⱼ - c†ⱼcᵢ")

    print("\n2. Pauli Exclusion:")
    print("   cᵢcᵢ = 0  (can't annihilate twice)")
    print("   c†ᵢc†ᵢ = 0  (can't create twice)")

    print("\n3. Nilpotency:")
    print("   (cᵢ)² = 0")
    print("   (c†ᵢ)² = 0")

    print("\n4. Number Operator Idempotence:")
    print("   n̂ᵢ² = (c†ᵢcᵢ)² = c†ᵢcᵢc†ᵢcᵢ")
    print("       = c†ᵢ(1 - c†ᵢcᵢ)cᵢ")
    print("       = c†ᵢcᵢ - (c†ᵢ)²(cᵢ)²")
    print("       = n̂ᵢ - 0 = n̂ᵢ")

anticommutation_review()
```

## 2. Sign Conventions and Ordering

<Warning>
Fermionic sign conventions are a common source of errors! Always track the parity of operator permutations carefully.
</Warning>

```python
def sign_conventions():
    """Explain fermionic sign conventions."""

    print("Fermionic Sign Conventions")
    print("=" * 60)

    print("\nOperator Ordering Convention:")
    print("  Standard: Larger orbital indices to the left")
    print("  |i < j < k⟩ = c†ₖc†ⱼc†ᵢ|0⟩")

    print("\nSign from Permutation:")
    print("-" * 45)
    print("  Each swap of adjacent operators: factor of -1")
    print()
    print("  c†₁c†₂c†₃|0⟩ = |1,2,3⟩  (reference ordering)")
    print("  c†₂c†₁c†₃|0⟩ = -|1,2,3⟩  (one swap)")
    print("  c†₂c†₃c†₁|0⟩ = +|1,2,3⟩  (two swaps)")

    print("\nParity of Permutation:")
    print("  P(σ) = number of transpositions")
    print("  Sign = (-1)^P(σ)")

    print("\nExample: Reordering c†₃c†₁c†₂")
    print("-" * 45)
    print("  Start: c†₃c†₁c†₂")
    print("  Swap 3,1: -c†₁c†₃c†₂")
    print("  Swap 3,2: +c†₁c†₂c†₃")
    print()
    print("  Result: c†₃c†₁c†₂ = +c†₁c†₂c†₃")
    print("  (Two swaps = even parity)")

sign_conventions()
```

## 3. Normal Ordering

<Definition term="Normal Ordering">
An arrangement of operators where all creation operators are to the left of all annihilation operators. Denoted by colons: :c†c: or N[c†c].
</Definition>

```python
def normal_ordering():
    """Explain normal ordering of fermionic operators."""

    print("Normal Ordering")
    print("=" * 60)

    print("\nDefinition:")
    print("  Normal order: All c† to the LEFT of all c")
    print("  Notation: :ABC: or N[ABC]")

    print("\nRules:")
    print("-" * 45)
    print("  1. Move all c† to the left of all c")
    print("  2. Track signs from anticommutation")
    print("  3. Ignore delta functions (contractions)")

    print("\nExamples:")
    print("  :c†ᵢcⱼ: = c†ᵢcⱼ  (already normal)")
    print("  :cⱼc†ᵢ: = -c†ᵢcⱼ  (swap needed)")

    print("\nVacuum Expectation:")
    print("-" * 45)
    print("  ⟨0|:anything:|0⟩ = 0")
    print()
    print("  (Unless 'anything' is a c-number)")
    print()
    print("  Proof: Either c|0⟩ = 0 or ⟨0|c† = 0")

    print("\nNormal Ordering vs Regular Product:")
    print("  cᵢc†ⱼ = δᵢⱼ - c†ⱼcᵢ = δᵢⱼ + :cᵢc†ⱼ:")
    print()
    print("  Regular product = contraction + normal ordered")

normal_ordering()
```

## 4. Contractions

<Formula>
$$\overline{c_i c_j^\dagger} = c_i c_j^\dagger - {:}c_i c_j^\dagger{:} = \delta_{ij}$$
</Formula>

```python
def contractions():
    """Explain fermionic contractions."""

    print("Fermionic Contractions")
    print("=" * 60)

    print("\nDefinition:")
    print("  Contraction of AB = AB - :AB:")
    print("  (What's 'left over' after normal ordering)")

    print("\nNotation:")
    print("  Contraction shown by connecting operators")
    print("  Written as overline or connected by line")

    print("\nElementary Contractions:")
    print("-" * 45)
    contractions_list = [
        ('cᵢc†ⱼ', 'δᵢⱼ'),
        ('c†ᵢcⱼ', '0'),
        ('cᵢcⱼ', '0'),
        ('c†ᵢc†ⱼ', '0'),
    ]

    for ops, result in contractions_list:
        print(f"  {ops:15} → {result}")

    print("\nWhy only cᵢc†ⱼ gives δᵢⱼ:")
    print("  cᵢc†ⱼ = δᵢⱼ - c†ⱼcᵢ")
    print("  :cᵢc†ⱼ: = -c†ⱼcᵢ")
    print("  Contraction = cᵢc†ⱼ - :cᵢc†ⱼ: = δᵢⱼ - c†ⱼcᵢ + c†ⱼcᵢ = δᵢⱼ")

    print("\nWith Particle-Hole Formalism:")
    print("-" * 45)
    print("  Define operators relative to Fermi vacuum:")
    print("  Occupied orbitals: hole operators hᵢ = c†ᵢ")
    print("  Virtual orbitals: particle operators pₐ = cₐ")
    print()
    print("  Additional non-zero contractions appear!")

contractions()
```

## 5. Wick's Theorem

<Definition term="Wick's Theorem">
A theorem stating that a product of creation and annihilation operators equals the sum of all possible normal-ordered terms with all possible contractions.
</Definition>

```python
def wicks_theorem():
    """Explain Wick's theorem."""

    print("Wick's Theorem")
    print("=" * 60)

    print("\nStatement:")
    print("  ABC...Z = :ABC...Z:")
    print("          + Σ (all single contractions)")
    print("          + Σ (all double contractions)")
    print("          + ...")
    print("          + (all fully contracted)")

    print("\nFor Two Operators:")
    print("-" * 45)
    print("  AB = :AB: + contraction(A,B)")
    print()
    print("  cᵢc†ⱼ = :cᵢc†ⱼ: + δᵢⱼ")
    print("       = -c†ⱼcᵢ + δᵢⱼ ✓")

    print("\nFor Four Operators:")
    print("-" * 45)
    print("  ABCD = :ABCD:")
    print("       + :AB:·contr(C,D) + :AC:·contr(B,D) + ...")
    print("       + contr(A,B)·contr(C,D) + ...")

    print("\nSign Rule for Contractions:")
    print("  (-1)^P where P = # of fermion operators")
    print("  crossed by the contraction line")

    print("\nVacuum Expectation Value:")
    print("-" * 45)
    print("  ⟨0|ABC...Z|0⟩ = Σ (fully contracted terms)")
    print()
    print("  Only fully contracted terms survive!")

wicks_theorem()
```

## 6. Applying Wick's Theorem

```python
def wick_examples():
    """Examples of Wick's theorem."""

    print("Wick's Theorem Examples")
    print("=" * 60)

    print("\nExample 1: ⟨0|cₐc†ᵦcᵧc†ᵨ|0⟩")
    print("-" * 45)
    print("  Two possible full contractions:")
    print()
    print("  1. Contract (a,b) and (c,d):")
    print("     δₐᵦ · δᵧᵨ · (sign from crossings)")
    print("     No crossings: +δₐᵦδᵧᵨ")
    print()
    print("  2. Contract (a,d) and (c,b):")
    print("     δₐᵨ · δᵧᵦ · (sign from crossings)")
    print("     One crossing: -δₐᵨδᵧᵦ")
    print()
    print("  Result: ⟨0|cₐc†ᵦcᵧc†ᵨ|0⟩ = δₐᵦδᵧᵨ - δₐᵨδᵧᵦ")

    print("\nExample 2: ⟨0|c†₁c†₂c₃c₄|0⟩")
    print("-" * 45)
    print("  Need to contract c† with c only!")
    print()
    print("  Possible contractions:")
    print("  1. (c†₁,c₃) and (c†₂,c₄): δ₁₃δ₂₄ · (+1)")
    print("  2. (c†₁,c₄) and (c†₂,c₃): δ₁₄δ₂₃ · (-1)")
    print()
    print("  Result: δ₁₃δ₂₄ - δ₁₄δ₂₃")

    print("\nPhysical Interpretation:")
    print("-" * 45)
    print("  This is the overlap of two 2-particle states!")
    print("  ⟨1,2|3,4⟩ = δ₁₃δ₂₄ - δ₁₄δ₂₃")
    print("  (Determinant structure of Slater determinants)")

wick_examples()
```

<Tip>
When counting crossings for signs, draw the contractions as arcs above the operators. Each time an arc crosses another arc or an operator, include a factor of -1.
</Tip>

## 7. Matrix Elements

```python
def matrix_elements():
    """Calculate matrix elements using second quantization."""

    print("Matrix Elements in Second Quantization")
    print("=" * 60)

    print("\nGeneral Formula:")
    print("  ⟨Ψ'|Ô|Ψ⟩ where |Ψ⟩, |Ψ'⟩ are Slater determinants")
    print()
    print("  Use Wick's theorem with modified contractions!")

    print("\nOne-Body Operator:")
    print("-" * 45)
    print("  Ô = Σᵢⱼ hᵢⱼ c†ᵢcⱼ")
    print()
    print("  ⟨Φ|Ô|Φ⟩ = Σᵢⱼ hᵢⱼ ⟨Φ|c†ᵢcⱼ|Φ⟩")
    print("         = Σᵢⱼ hᵢⱼ δᵢⱼ nᵢ  (occupied i)")
    print("         = Σᵢ∈occ hᵢᵢ")

    print("\nTwo-Body Operator:")
    print("-" * 45)
    print("  Ô = ½Σᵢⱼₖₗ ⟨ij|kl⟩ c†ᵢc†ⱼcₗcₖ")
    print()
    print("  ⟨Φ|Ô|Φ⟩ = ½Σᵢⱼ∈occ (⟨ij|ij⟩ - ⟨ij|ji⟩)")
    print("         = ½Σᵢⱼ∈occ (Jᵢⱼ - Kᵢⱼ)")

    print("\nHF Energy from Second Quantization:")
    print("-" * 45)
    print("  E_HF = Σᵢ hᵢᵢ + ½Σᵢⱼ (Jᵢⱼ - Kᵢⱼ) + V_nn")

matrix_elements()
```

## 8. Particle-Hole Formalism

```python
def particle_hole():
    """The particle-hole formalism."""

    print("Particle-Hole Formalism")
    print("=" * 60)

    print("\nIdea:")
    print("  Redefine 'vacuum' as the HF ground state |Φ₀⟩")
    print("  instead of the true vacuum |0⟩")

    print("\nNew Operators:")
    print("-" * 45)
    print("  Occupied orbitals (i, j, k, ...):")
    print("    bᵢ = c†ᵢ  (hole annihilation)")
    print("    b†ᵢ = cᵢ  (hole creation)")
    print()
    print("  Virtual orbitals (a, b, c, ...):")
    print("    pₐ = cₐ  (particle annihilation)")
    print("    p†ₐ = c†ₐ  (particle creation)")

    print("\nNew 'Vacuum':")
    print("  |Φ₀⟩ = c†₁c†₂...c†ₙ|0⟩  (HF ground state)")
    print()
    print("  bᵢ|Φ₀⟩ = 0  (no holes)")
    print("  pₐ|Φ₀⟩ = 0  (no particles)")

    print("\nExcited States:")
    print("-" * 45)
    print("  Single excitation: p†ₐbᵢ|Φ₀⟩ = c†ₐcᵢ|Φ₀⟩")
    print("  Double excitation: p†ₐp†ᵦbⱼbᵢ|Φ₀⟩ = c†ₐc†ᵦcⱼcᵢ|Φ₀⟩")

    print("\nAdvantage:")
    print("  Normal ordering relative to |Φ₀⟩ simplifies")
    print("  the calculation of correlation effects!")

particle_hole()
```

## Summary

In this lesson, you learned:

1. **Anticommutation relations** {c, c†} = δ, {c, c} = {c†, c†} = 0
2. **Sign conventions** require tracking permutation parity
3. **Normal ordering** puts all c† left of all c
4. **Contractions** give non-trivial contributions: c_ic†_j = δ_ij
5. **Wick's theorem** expands products into contractions
6. **Matrix elements** computed via fully-contracted terms
7. **Particle-hole formalism** redefines vacuum as HF state
