## 1. Fock Matrix Decomposition

<Formula>
$$F_{\mu\nu} = h_{\mu\nu} + G_{\mu\nu}$$
</Formula>

where $G_{\mu\nu} = \sum_{\lambda\sigma} P_{\lambda\sigma}\left[(\mu\nu|\lambda\sigma) - \frac{1}{2}(\mu\sigma|\lambda\nu)\right]$

```python
def fock_matrix_structure():
    """Detail the components of the Fock matrix."""

    print("Fock Matrix Components")
    print("=" * 60)

    print("\n1. ONE-ELECTRON PART (h):")
    print("-" * 45)
    print("   hμν = ⟨χμ|ĥ|χν⟩")
    print()
    print("   ĥ = -½∇² - Σₐ Zₐ/rₐ")
    print()
    print("   Components:")
    print("   • Kinetic energy: Tμν = -½⟨χμ|∇²|χν⟩")
    print("   • Nuclear attraction: Vμν = -⟨χμ|Σₐ Zₐ/rₐ|χν⟩")
    print()
    print("   Properties:")
    print("   • Computed once (doesn't depend on density)")
    print("   • Scales as O(N²)")

    print("\n2. TWO-ELECTRON PART (G):")
    print("-" * 45)
    print("   Gμν = Σλσ Pλσ [(μν|λσ) - ½(μσ|λν)]")
    print("              └─Coulomb──┘  └Exchange─┘")
    print()
    print("   Properties:")
    print("   • Depends on density matrix P")
    print("   • Must be rebuilt each SCF iteration")
    print("   • Naive scaling: O(N⁴)")

fock_matrix_structure()
```

## 2. Two-Electron Integral Notation

<Definition term="Mulliken Notation">
The notation (μν|λσ) for two-electron integrals: ∫∫ χμ*(r₁)χν(r₁) (1/r₁₂) χλ*(r₂)χσ(r₂) dr₁dr₂
</Definition>

```python
def integral_notation():
    """Explain two-electron integral notation."""

    print("Two-Electron Integral Notation")
    print("=" * 60)

    print("\nMulliken (Chemist's) Notation: (μν|λσ)")
    print("-" * 45)
    print("  (μν|λσ) = ∫∫ χμ*(1)χν(1) (1/r₁₂) χλ*(2)χσ(2) dr₁dr₂")
    print()
    print("  Mnemonic: 'bra' indices together, 'ket' indices together")
    print("  (μν| ← electron 1,  |λσ) ← electron 2")

    print("\nDirac (Physicist's) Notation: ⟨μλ|νσ⟩")
    print("-" * 45)
    print("  ⟨μλ|νσ⟩ = ∫∫ χμ*(1)χλ*(2) (1/r₁₂) χν(1)χσ(2) dr₁dr₂")
    print()
    print("  Relationship: (μν|λσ) = ⟨μλ|νσ⟩")

    print("\nSymmetries (real orbitals):")
    print("-" * 45)
    print("  (μν|λσ) = (νμ|λσ) = (μν|σλ) = (νμ|σλ)")
    print("          = (λσ|μν) = (σλ|μν) = (λσ|νμ) = (σλ|νμ)")
    print()
    print("  8-fold symmetry → only ~N⁴/8 unique integrals")

integral_notation()
```

## 4. Density Fitting / Resolution of Identity

<Definition term="Density Fitting (DF) / Resolution of Identity (RI)">
An approximation that expands products of basis functions in an auxiliary basis, reducing the scaling of Coulomb integrals from O(N⁴) to O(N³).
</Definition>

```python
def density_fitting():
    """Explain density fitting approximation."""

    print("Density Fitting / Resolution of Identity")
    print("=" * 60)

    print("\nThe Idea:")
    print("-" * 45)
    print("  χμ(r)χν(r) ≈ Σₚ dᵖμν Pₚ(r)")
    print()
    print("  Product of AOs expanded in auxiliary basis {Pₚ}")
    print("  Usually atom-centered Gaussians")

    print("\nConsequence for Integrals:")
    print("-" * 45)
    print("  (μν|λσ) ≈ Σₚₚ dᵖμν dᵍλσ (P|Q)")
    print()
    print("  = Σₚₚ (μν|P)(P|Q)⁻¹(Q|λσ)")
    print()
    print("  Two-index metric (P|Q): O(N_aux²)")
    print("  Three-index (μν|P): O(N² × N_aux)")

    print("\nScaling Improvement:")
    print("-" * 45)
    print("  Exact:  O(N⁴) for (μν|λσ)")
    print("  DF/RI:  O(N³) or O(N² × N_aux)")
    print()
    print("  Error: ~0.1 mH/atom (much smaller than basis set error)")

    print("\nWhen to Use:")
    print("-" * 45)
    print("  • Almost always for MP2, CC2")
    print("  • Increasingly for HF/DFT")
    print("  • Essential for large molecules")

density_fitting()
```

## 8. Code: Complete Fock Matrix Builder

```python
def fock_builder_example():
    """Complete Fock matrix construction code."""

    code = '''
import numpy as np

def build_fock(h, eri, P, screening_thresh=1e-10):
    """Build Fock matrix with Schwarz screening.

    Args:
        h: Core Hamiltonian (n_ao × n_ao)
        eri: Two-electron integrals (n_ao × n_ao × n_ao × n_ao)
        P: Density matrix (n_ao × n_ao)
        screening_thresh: Integral screening threshold

    Returns:
        F: Fock matrix (n_ao × n_ao)
    """
    n_ao = h.shape[0]

    # Pre-compute Schwarz bounds
    schwarz = np.zeros((n_ao, n_ao))
    for mu in range(n_ao):
        for nu in range(n_ao):
            schwarz[mu, nu] = np.sqrt(abs(eri[mu, nu, mu, nu]))

    # Build G matrix
    G = np.zeros((n_ao, n_ao))

    for mu in range(n_ao):
        for nu in range(n_ao):
            # Screen based on (μν|μν)
            if schwarz[mu, nu] < screening_thresh:
                continue

            for lam in range(n_ao):
                for sig in range(n_ao):
                    # Schwarz screening
                    bound = schwarz[mu, nu] * schwarz[lam, sig]
                    if bound * abs(P[lam, sig]) < screening_thresh:
                        continue

                    # Coulomb
                    G[mu, nu] += P[lam, sig] * eri[mu, nu, lam, sig]
                    # Exchange
                    G[mu, nu] -= 0.5 * P[lam, sig] * eri[mu, sig, lam, nu]

    return h + G
'''
    print("Complete Fock Matrix Builder")
    print("=" * 60)
    print(code)

fock_builder_example()
```

## Summary

In this lesson, you learned:

1. **Fock matrix** = one-electron (h) + two-electron (G)
2. **Two-electron integrals** follow 8-fold permutation symmetry
3. **Schwarz screening** eliminates negligible integrals
4. **Density fitting** reduces scaling from O(N⁴) to O(N³)
5. **Sparsity** enables linear scaling for large molecules
6. **Diagonal elements** approximate orbital energies
7. **Open-shell systems** require modified Fock matrices

Efficient Fock matrix construction is the key to practical HF calculations.
