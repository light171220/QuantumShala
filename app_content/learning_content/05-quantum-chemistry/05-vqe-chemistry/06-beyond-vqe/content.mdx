## 1. VQE Limitations Revisited

<Definition term="Variational Quantum Eigensolver Limitations">

</Definition>

## 2. Quantum Phase Estimation (QPE)

<Formula>
$$U|E_k\rangle = e^{-iE_k t}|E_k\rangle \xrightarrow{\text{QPE}} |E_k\rangle|\tilde{E}_k\rangle$$
</Formula>

```python
def quantum_phase_estimation():
    """Quantum phase estimation for eigenvalues."""

    print("Quantum Phase Estimation (QPE)")
    print("=" * 60)

    print("\nKey Idea:")
    print("  Extract eigenvalue by measuring phase of U = e^{-iHt}")
    print("  Deterministic, not variational!")

    print("\nAlgorithm Overview:")
    print("-" * 45)
    steps = [
        "1. Prepare eigenstate |E⟩ (or approximation)",
        "2. Apply controlled-U^(2^k) operations",
        "3. Perform inverse QFT on ancilla register",
        "4. Measure ancilla to get binary representation of E",
    ]
    for step in steps:
        print(f"  {step}")

    print("\nCircuit Structure:")
    print("-" * 45)
    print("  |0⟩ ─[H]─●─────────────────[QFT†]─ Measure")
    print("  |0⟩ ─[H]───●───────────────[QFT†]─ Measure")
    print("  |0⟩ ─[H]─────●─────────────[QFT†]─ Measure")
    print("   ⋮           ⋮")
    print("  |ψ⟩ ────[U¹]─[U²]─[U⁴]─...──────── Keep")

    print("\nResource Requirements:")
    print("-" * 45)
    resources = [
        ("Ancilla qubits", "n_bits", "Precision bits"),
        ("Time evolution", "U = e^{-iHt}", "Main cost"),
        ("Circuit depth", "O(n⁴) per Trotter", "Deep!"),
        ("Total depth", "2^n_bits × depth(U)", "Very deep"),
    ]
    print(f"  {'Resource':<18} {'Requirement':<15} {'Note'}")
    print("  " + "-" * 50)
    for r, req, note in resources:
        print(f"  {r:<18} {req:<15} {note}")

    print("\nComparison with VQE:")
    print("-" * 45)
    print(f"  {'Aspect':<20} {'VQE':<20} {'QPE'}")
    print("  " + "-" * 55)
    comparison = [
        ("Accuracy", "Ansatz-limited", "Exponential precision"),
        ("Circuit depth", "Moderate", "Very deep"),
        ("Optimization", "Required", "None"),
        ("NISQ-ready", "Yes", "No (needs FT)"),
        ("Excited states", "Difficult", "Natural"),
    ]
    for aspect, vqe, qpe in comparison:
        print(f"  {aspect:<20} {vqe:<20} {qpe}")

quantum_phase_estimation()
```

<Tip>
QPE provides exponentially precise eigenvalues but requires fault-tolerant quantum computers. Current research focuses on combining VQE's NISQ compatibility with QPE's accuracy guarantees.
</Tip>

## 5. Quantum Imaginary Time Evolution

<Formula>
$$|\psi(\tau)\rangle = \frac{e^{-H\tau}|\psi_0\rangle}{\|e^{-H\tau}|\psi_0\rangle\|}$$
</Formula>

```python
def imaginary_time_evolution():
    """Quantum imaginary time evolution."""

    print("Quantum Imaginary Time Evolution (QITE)")
    print("=" * 60)

    print("\nClassical ITE:")
    print("-" * 45)
    print("  |ψ(τ)⟩ = e^{-Hτ}|ψ₀⟩ / ||...||")
    print()
    print("  As τ → ∞, projects onto ground state")
    print("  (Lowest eigenvalue dominates exponential)")

    print("\nChallenge for Quantum:")
    print("-" * 45)
    print("  e^{-Hτ} is NOT unitary!")
    print("  Cannot directly implement on quantum computer")

    print("\nQITE Approach:")
    print("-" * 45)
    steps = [
        "1. Approximate e^{-Hδτ} by unitary e^{-iAδτ}",
        "2. A chosen to match change in |ψ⟩",
        "3. Determine A by solving linear system",
        "4. Iterate: |ψₖ₊₁⟩ = e^{-iAₖδτ}|ψₖ⟩",
    ]
    for step in steps:
        print(f"  {step}")

    print("\nMcLachlan Variational Principle:")
    print("-" * 45)
    print("  Minimize: ||d|ψ⟩/dτ + H|ψ⟩||²")
    print()
    print("  This determines optimal A for each step")

    print("\nAdvantages:")
    print("-" * 45)
    print("  • No optimization required (systematic)")
    print("  • Guaranteed convergence to ground state")
    print("  • Works with any initial state")

    print("\nCosts:")
    print("-" * 45)
    print("  • Many measurements per step (for A)")
    print("  • Many time steps needed")
    print("  • Matrix inversion at each step")

    print("\nExample Convergence:")
    print("-" * 45)
    taus = [0, 0.5, 1.0, 2.0, 5.0]
    energies = [-1.05, -1.10, -1.12, -1.135, -1.137]
    print(f"  {'τ':<10} {'Energy (Eh)'}")
    print("  " + "-" * 25)
    for tau, e in zip(taus, energies):
        print(f"  {tau:<10} {e:.3f}")

imaginary_time_evolution()
```

## 6. Variational Quantum Dynamics

```python
def variational_dynamics():
    """Variational methods for quantum dynamics."""

    print("Variational Quantum Dynamics")
    print("=" * 60)

    print("\nReal-Time Evolution:")
    print("-" * 45)
    print("  |ψ(t)⟩ = e^{-iHt}|ψ₀⟩")
    print()
    print("  Applications:")
    print("    • Spectroscopy (absorption spectra)")
    print("    • Reaction dynamics")
    print("    • Transport properties")

    print("\nVariational Approach:")
    print("-" * 45)
    print("  Represent |ψ(t)⟩ = U(θ(t))|0⟩")
    print("  Find θ(t) that best follows exact evolution")

    print("\nTime-Dependent Variational Principle:")
    print("-" * 45)
    print("  McLachlan: minimize ||∂|ψ⟩/∂t - (-iH)|ψ⟩||²")
    print()
    print("  Leads to: M θ̇ = V")
    print("    M_ij = Re⟨∂ψ/∂θᵢ|∂ψ/∂θⱼ⟩")
    print("    V_i = Im⟨∂ψ/∂θᵢ|H|ψ⟩")

    print("\nAlgorithm:")
    print("-" * 45)
    steps = [
        "1. Initialize θ(0) from initial state",
        "2. Measure M and V matrices",
        "3. Solve M θ̇ = V for parameter velocities",
        "4. Update θ(t+δt) = θ(t) + θ̇ δt",
        "5. Repeat for desired time range",
    ]
    for step in steps:
        print(f"  {step}")

    print("\nApplications:")
    print("-" * 45)
    apps = [
        ("Spectroscopy", "Compute absorption spectra"),
        ("Dynamics", "Follow reaction coordinates"),
        ("Correlation", "Time-dependent properties"),
        ("Thermalization", "Approach to equilibrium"),
    ]
    for app, desc in apps:
        print(f"  {app}:")
        print(f"    {desc}")
        print()

variational_dynamics()
```

<Warning>
Variational dynamics can suffer from accumulating errors over long times. The accuracy of the ansatz limits how well dynamics can be captured, especially for complex evolutions.
</Warning>
