## 1. The VQE Optimization Problem

<Definition term="VQE Optimization">
The task of finding parameters θ* that minimize the energy expectation value: θ* = argmin_θ E(θ) = argmin_θ ⟨ψ(θ)|Ĥ|ψ(θ)⟩.
</Definition>

## 3. SPSA Optimizer

<Formula>
$$\theta_{k+1} = \theta_k - a_k \frac{E(\theta_k + c_k\Delta_k) - E(\theta_k - c_k\Delta_k)}{2c_k} \Delta_k$$
</Formula>

```python
import numpy as np

def spsa_optimizer():
    """SPSA: Simultaneous Perturbation Stochastic Approximation."""

    print("SPSA Optimizer")
    print("=" * 60)

    print("\nKey Idea:")
    print("  Approximate gradient using only 2 function evaluations")
    print("  regardless of parameter dimension!")

    print("\nAlgorithm:")
    print("-" * 45)
    print("  1. Generate random perturbation Δₖ ∈ {±1}ⁿ")
    print("  2. Evaluate E(θ + cΔ) and E(θ - cΔ)")
    print("  3. Estimate gradient:")
    print("     g ≈ [E(θ+cΔ) - E(θ-cΔ)]/(2c) × Δ")
    print("  4. Update: θ ← θ - a × g")

    print("\nParameter Schedules:")
    print("-" * 45)
    print("  Learning rate: aₖ = a / (k + A)^α")
    print("  Perturbation:  cₖ = c / k^γ")
    print()
    print("  Typical values:")
    print("    a = 0.1, A = 10, α = 0.602")
    print("    c = 0.1, γ = 0.101")

    print("\nExample Implementation:")
    print("-" * 45)

    def spsa_step(cost_fn, theta, k, a=0.1, c=0.1, A=10):
        """One SPSA iteration."""
        n = len(theta)

        # Decaying parameters
        ak = a / (k + A) ** 0.602
        ck = c / (k + 1) ** 0.101

        # Random perturbation
        delta = 2 * np.random.randint(0, 2, n) - 1

        # Two-sided evaluation
        e_plus = cost_fn(theta + ck * delta)
        e_minus = cost_fn(theta - ck * delta)

        # Gradient estimate
        grad = (e_plus - e_minus) / (2 * ck) * delta

        # Update
        theta_new = theta - ak * grad
        return theta_new

    print("  def spsa_step(cost_fn, theta, k):")
    print("      delta = random ±1 vector")
    print("      grad = (E(θ+cΔ) - E(θ-cΔ))/(2c) × Δ")
    print("      return theta - a × grad")

    print("\nAdvantages for VQE:")
    print("  • Only 2 circuit evaluations per iteration")
    print("  • Robust to noise (stochastic by design)")
    print("  • Works well with shot noise")

spsa_optimizer()
```

<Tip>
SPSA is particularly well-suited for VQE on noisy hardware because it only requires two circuit evaluations per iteration and naturally handles noise through its stochastic gradient estimation.
</Tip>

## 4. Parameter-Shift Rule

```python
def parameter_shift_rule():
    """Exact gradients via parameter-shift rule."""

    print("Parameter-Shift Rule")
    print("=" * 60)

    print("\nThe Key Result:")
    print("-" * 45)
    print("  For gates of the form U(θ) = exp(-iθG/2):")
    print()
    print("  ∂E/∂θ = [E(θ + π/2) - E(θ - π/2)] / 2")
    print()
    print("  EXACT gradient from 2 circuit evaluations!")

    print("\nWhy Does This Work?")
    print("-" * 45)
    print("  For Pauli rotations (Rx, Ry, Rz):")
    print("  U(θ) = cos(θ/2)I - i sin(θ/2)P")
    print()
    print("  Energy is sinusoidal in θ:")
    print("  E(θ) = A + B cos(θ) + C sin(θ)")
    print()
    print("  ∂E/∂θ = -B sin(θ) + C cos(θ)")
    print("        = [E(θ+π/2) - E(θ-π/2)] / 2")

    print("\nImplementation:")
    print("-" * 45)

    def compute_gradient(cost_fn, theta, idx):
        """Compute gradient for parameter idx using parameter-shift."""
        theta_plus = theta.copy()
        theta_minus = theta.copy()

        theta_plus[idx] += np.pi / 2
        theta_minus[idx] -= np.pi / 2

        grad = (cost_fn(theta_plus) - cost_fn(theta_minus)) / 2
        return grad

    print("  theta_plus = theta.copy()")
    print("  theta_plus[i] += π/2")
    print("  theta_minus = theta.copy()")
    print("  theta_minus[i] -= π/2")
    print("  grad_i = (E(theta_plus) - E(theta_minus)) / 2")

    print("\nCost Analysis:")
    print("-" * 45)
    print("  n parameters → 2n circuit evaluations")
    print("  More expensive than SPSA per iteration")
    print("  But: exact gradients → better convergence")

    print("\nGeneralized Parameter-Shift:")
    print("-" * 45)
    print("  For generators with eigenvalues ±r:")
    print("  ∂E/∂θ = r[E(θ + π/4r) - E(θ - π/4r)]")

parameter_shift_rule()
```

## 6. Handling Noise in Optimization

```python
def noise_handling():
    """Strategies for optimizing with noisy evaluations."""

    print("Handling Noise in VQE Optimization")
    print("=" * 60)

    print("\nSources of Noise:")
    print("-" * 45)
    print("  1. Shot noise: σ ∝ 1/√N_shots")
    print("  2. Gate errors: coherent and incoherent")
    print("  3. Readout errors: bit-flip probabilities")

    print("\nNoise Impact on Optimization:")
    print("-" * 45)
    print("  • Gradients become noisy: ∇E → ∇E + noise")
    print("  • Optimization may oscillate near minimum")
    print("  • May converge to wrong value")

    print("\nStrategies:")
    print("-" * 45)

    print("\n  1. Increase Shots Adaptively:")
    print("     • Start with few shots (fast exploration)")
    print("     • Increase shots near convergence (precision)")
    print("     • Example: N = N₀ × (1 + k/k_scale)")

    print("\n  2. Running Averages:")
    print("     • Average multiple evaluations at same θ")
    print("     • Reduces variance by factor √n_avg")

    print("\n  3. Noise-Resilient Optimizers:")
    print("     • SPSA: designed for stochastic settings")
    print("     • iCANS: noise-aware step sizes")
    print("     • QNSPSA: quantum natural SPSA")

    print("\n  4. Regularization:")
    print("     • Add penalty term: E(θ) + λ||θ||²")
    print("     • Smooths landscape, prevents overfitting to noise")

    print("\nAdaptive Shot Example:")
    print("-" * 45)

    def adaptive_shots(iteration, base_shots=1000, max_shots=100000):
        """Adaptive shot allocation."""
        shots = min(base_shots * (1 + iteration // 10), max_shots)
        return int(shots)

    iters = [0, 10, 50, 100, 200]
    print(f"  {'Iteration':<12} {'Shots'}")
    for i in iters:
        print(f"  {i:<12} {adaptive_shots(i)}")

noise_handling()
```

<Warning>
Never trust a VQE result without understanding the noise level. Always report error bars and validate convergence by restarting from different initial points.
</Warning>
